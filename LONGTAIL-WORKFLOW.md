# 롱테일 키워드 워크플로우

## 트리거
사용자가 "롱테일:" 또는 키워드 목록(PAA 질문)을 제공하면 실행

## 입력 형식
```
롱테일:
퇴직금을 IRP에 이체하면 세금은 어떻게 되나요?
개인형 IRP는 어떻게 수령하나요?
IRP는 언제 출금할 수 있나요?
```

## 머니위키 작성 공식 (필수)

### URL/제목(H1): 명사형 키워드 (짧고 굵게)
```
PAA: "퇴직금을 IRP에 이체하면 세금은 어떻게 되나요?"
  ↓ 변환
URL: /w/퇴직금-IRP-이체-세금
H1: 퇴직금 IRP 이체 세금
```

### 소제목(H2): PAA 질문 그대로 (문장형)
```
## 퇴직금을 IRP에 이체하면 세금은 어떻게 되나요?
```

### 본문: 질문에 대한 답변
- 첫 문장에 핵심 답변
- 위키 톤 (간결, 정확)
- 블로그 X (주저리주저리 X)

## 처리 단계

### 1단계: 분석
- PAA 질문 → 명사형 키워드 변환
- 유사 키워드 그룹핑
- 생성 문서 수 산출

### 2단계: 구조 제안
```
📊 분석 결과
- 입력 질문: X개
- 생성 문서: Y개

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📄 문서 1: /w/퇴직금-IRP-이체-세금
   H1: 퇴직금 IRP 이체 세금
├─ H2: 퇴직금을 IRP에 이체하면 세금은 어떻게 되나요?
├─ H2: IRP 이체 시 과세이연이란?
└─ H2: IRP 해지하면 세금은?

📄 문서 2: /w/개인형-IRP-수령-방법
   H1: 개인형 IRP 수령 방법
├─ H2: 개인형 IRP는 어떻게 수령하나요?
└─ H2: IRP 수령 시 필요 서류

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

진행할까요?
```

### 3단계: 글 작성
사용자 확인 후 순차 작성

## 변환 규칙
| PAA 질문 (문장형) | 제목/URL (명사형) |
|------------------|------------------|
| 퇴직금을 IRP에 이체하면 세금은 어떻게 되나요? | 퇴직금 IRP 이체 세금 |
| 개인형 IRP는 어떻게 수령하나요? | 개인형 IRP 수령 방법 |
| IRP는 언제 출금할 수 있나요? | IRP 중도인출 조건 |
| 퇴직연금 IRP를 해지할 때 세금은? | IRP 해지 세금 계산 |

## 출력 규칙
- URL: 명사형-하이픈-연결 (짧게)
- H1: URL과 동일 (띄어쓰기)
- H2: PAA 질문 그대로
- 본문: 위키 톤, 간결, 정확

## SEO 메타 태그 규칙

### Meta Title (검색 결과 제목)
```
{H1} | 머니위키
```
예: `퇴직금 IRP 이체 세금 | 머니위키`

- 블로그 스타일 X (총정리, 완벽가이드, 꿀팁 등 금지)
- 위키피디아 형식 유지
- 브랜드 일관성 = 장기 신뢰도

### Meta Description (검색 결과 설명)
```
{PAA 질문}? {핵심 답변 1줄}
```
예: `퇴직금을 IRP에 이체하면 세금은 어떻게 될까요? 과세이연 조건과 해지 시 세금까지 정리했습니다.`

- 질문 포함 → 검색 의도 매칭
- 답변 힌트 → 클릭 유도
- 150자 이내

## 복합 키워드 타이틀 전략 (핵심!)

### 원칙: 타이틀 하나에 3~5개 키워드 자연스럽게 포함

타이틀을 설계할 때 **여러 키워드 조합**이 자연스럽게 녹아들도록 구성

### 예시: 미지급 실업급여

**키워드 목록**:
- 미지급 실업급여
- 미지급 실업급여 수급
- 미지급 실업급여 신청
- 미지급 실업급여 수급 방법
- 미지급 실업급여 수급권의 소멸시효

**타이틀 설계**:
```
"미지급 실업급여 수급 순위 및 신청 방법"
         ├─ 미지급 실업급여 (기본)
         ├─ 미지급 실업급여 수급 (기본+수급)
         ├─ 미지급 실업급여 신청 (기본+신청)
         └─ 미지급 실업급여 수급 방법 (기본+수급+방법)
```

### 타이틀 설계 공식

```
{기본키워드} + {수식어1} + {연결어} + {수식어2}
```

**연결어 예시**: 및, 와, 방법, 조건, 기준, 계산, 신청

### 적용 체크리스트

- [ ] 기본 키워드가 타이틀 앞에 있는가?
- [ ] 2~3개 수식어가 자연스럽게 연결되는가?
- [ ] 3~5개 키워드 조합이 타이틀에서 추출 가능한가?
- [ ] 한국어로 자연스러운가? (어색하면 실패)
