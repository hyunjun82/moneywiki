# 실업급여 수급 조건

실업급여는 비자발적으로 실직한 근로자의 생활 안정을 위한 제도입니다. [고용보험법](https://www.law.go.kr/법령/고용보험법)에 따른 수급 조건을 정리합니다.

## 1. 기본 수급 조건

### 1.1 4가지 필수 조건

| 조건 | 내용 |
|------|------|
| ① 고용보험 가입 | 퇴직 전 18개월 중 **180일 이상** |
| ② 비자발적 퇴사 | 자기 사정이 아닌 퇴직 |
| ③ 근로 의사 | 적극적 구직활동 의사 |
| ④ 능력 | 근로 능력 보유 |

### 1.2 고용보험 가입 기간

| 항목 | 내용 |
|------|------|
| 기준 | 퇴직 전 **18개월** 중 |
| 최소 기간 | **180일** (약 6개월) |
| 산정 | 유급휴일 포함 |

## 2. 비자발적 퇴사

### 2.1 실업급여 대상 퇴사

| 퇴사 사유 | 대상 여부 |
|----------|----------|
| 권고사직 | ✅ 대상 |
| 계약 만료 | ✅ 대상 |
| 정리해고 | ✅ 대상 |
| 폐업 | ✅ 대상 |
| 정당한 자진퇴사 | ✅ 대상 |

### 2.2 정당한 자진퇴사 사유

| 사유 | 인정 여부 |
|------|----------|
| 임금 체불 | ✅ 인정 |
| 최저임금 위반 | ✅ 인정 |
| 직장 내 괴롭힘 | ✅ 인정 |
| 통근 어려움 (2시간 이상) | ✅ 인정 |
| 배우자 전근 | ✅ 인정 |
| 본인 건강 | ✅ 인정 |
| 가족 돌봄 | ✅ 인정 |

### 2.3 실업급여 대상 아닌 퇴사

| 퇴사 사유 | 대상 여부 |
|----------|----------|
| 단순 자발적 퇴사 | ❌ 미대상 |
| 이직 목적 퇴사 | ❌ 미대상 |
| 징계해고 (중대사유) | ❌ 미대상 |

## 3. 수급 기간과 금액

### 3.1 수급 기간

| 연령/가입기간 | 1년 미만 | 1~3년 | 3~5년 | 5~10년 | 10년 이상 |
|--------------|---------|-------|-------|--------|----------|
| 50세 미만 | 120일 | 150일 | 180일 | 210일 | 240일 |
| 50세 이상/장애인 | 120일 | 180일 | 210일 | 240일 | 270일 |

### 3.2 수급액 계산

```
1일 실업급여 = 퇴직 전 평균임금 × 60%
```

| 항목 | 금액 (2026년) |
|------|--------------|
| 상한액 | 1일 **66,000원** |
| 하한액 | 최저임금 × 80% × 1일 |

### 3.3 월 수급액 예시

| 퇴직 전 월급 | 1일 급여 | 월 수급액 (30일) |
|-------------|---------|-----------------|
| 250만원 | 약 50,000원 | 약 150만원 |
| 300만원 | 약 60,000원 | 약 180만원 |
| 400만원 | 66,000원 (상한) | 약 198만원 |

## 4. 신청 절차

### 4.1 신청 흐름

```
퇴직 → 이직확인서 발급 → 워크넷 구직등록 → 수급자격 신청 → 교육 → 실업인정
```

### 4.2 단계별 절차

| 단계 | 내용 | 기한 |
|------|------|------|
| 1 | 회사에서 이직확인서 제출 | 퇴직 후 10일 내 |
| 2 | 워크넷 구직등록 | - |
| 3 | 고용센터 수급자격 신청 | - |
| 4 | 수급자격인정자 교육 | 신청 후 |
| 5 | 실업인정 (4주마다) | 지정일 |

### 4.3 필요 서류

| 서류 | 발급처 |
|------|--------|
| 이직확인서 | 전 회사 |
| 신분증 | - |
| 통장 사본 | - |

## 5. 실업인정

### 5.1 실업인정이란

| 항목 | 내용 |
|------|------|
| 목적 | 구직활동 확인 |
| 주기 | **4주마다** 1회 |
| 방법 | 고용센터 방문 또는 온라인 |

### 5.2 구직활동 요건

| 활동 | 인정 여부 |
|------|----------|
| 입사지원 | ✅ |
| 면접 | ✅ |
| 취업박람회 참가 | ✅ |
| 직업훈련 | ✅ |

## 6. 수급 제한

### 6.1 수급 중지 사유

| 사유 | 결과 |
|------|------|
| 재취업 | 수급 중지 (조기재취업수당 가능) |
| 허위 구직활동 | 수급 중지 + 반환 |
| 부정수급 | 3배 반환 + 형사처벌 |

### 6.2 수급 불가 사유

| 사유 | 불가 |
|------|------|
| 자영업 시작 | ❌ |
| 해외 체류 | ❌ |
| 군 입대 | ❌ |

## 7. 특수 상황

### 7.1 자영업자 실업급여

| 조건 | 내용 |
|------|------|
| 가입 | 임의가입 (자영업자 고용보험) |
| 기간 | 1년 이상 |
| 폐업 | 비자발적 폐업 |

### 7.2 예술인 실업급여

| 조건 | 내용 |
|------|------|
| 가입 | 예술인 고용보험 |
| 기간 | 9개월 이상 |
| 수급 | 구직급여 수급 |

## 8. 주의사항

### 8.1 신청 기한

| 항목 | 기한 |
|------|------|
| 수급자격 신청 | 퇴직 후 **12개월** 이내 |
| 이직확인서 | 회사가 10일 내 제출 |

### 8.2 재취업 후

| 상황 | 처리 |
|------|------|
| 재취업 | 즉시 신고 |
| 단기 취업 | 취업일 제외 후 수급 |
| 미신고 | 부정수급 처리 |
