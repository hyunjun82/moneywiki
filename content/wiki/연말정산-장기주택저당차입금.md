# 연말정산 장기주택저당차입금

장기주택저당차입금 이자 소득공제를 정리합니다.

## 1. 공제 요건

| 요건 | 기준 |
|------|------|
| 상환기간 | **10년 이상** |
| 주택 | 무주택/1주택 세대주 |
| 기준시가 | 취득 시 **5억원 이하** |
| 담보 | 해당 주택 |

## 2. 공제 한도

| 조건 | 한도 |
|------|------|
| 15년+, 고정금리+비거치 | **1,800만원** |
| 15년+, 고정 또는 비거치 | **1,500만원** |
| 15년+, 기타 | **500만원** |
| 10~15년 | **300만원** |

## 3. 금리 조건

| 금리 | 인정 |
|------|------|
| 고정금리 | 전체 기간 고정 |
| 변동금리 | 추가 한도 미적용 |

## 4. 상환 조건

| 방식 | 인정 |
|------|------|
| 비거치식 | 원금 분할상환 |
| 거치식 | 추가 한도 미적용 |

## 5. 취득 시기

| 요건 | 기준 |
|------|------|
| 차입일 | 주택 취득 전후 3개월 |
