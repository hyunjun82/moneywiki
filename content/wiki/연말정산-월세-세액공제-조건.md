# 연말정산 월세 세액공제 조건

월세 세액공제는 무주택 근로자가 월세를 낼 경우 세금에서 일정 금액을 직접 차감해주는 제도입니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=40613&cntntsId=239025)에 따른 정확한 조건을 정리합니다.

## 1. 공제 대상자 조건

[조세특례제한법 제95조의2](https://www.law.go.kr/법령/조세특례제한법)에 따르면, 다음 조건을 모두 충족해야 합니다.

### 1.1 소득 요건

| 구분 | 기준 |
|------|------|
| 총급여 | **8,000만원 이하** |
| 종합소득금액 | **7,000만원 이하** |

### 1.2 무주택 요건

- 과세연도 종료일(12월 31일) 기준 **무주택 세대의 세대주**
- 세대주가 주택 관련 공제를 받지 않는 경우 **세대원도 가능**
- 거주자와 배우자는 생계를 달리해도 **동일 세대로 간주**

### 1.3 전입신고 요건

- **주민등록상 전입신고 필수**
- 임대차계약서 주소와 주민등록등본 주소가 **일치**해야 함
- 전입신고 없이는 다른 요건을 충족해도 **공제 불가**

## 2. 공제 대상 주택

[국세청 국세상담센터](https://call.nts.go.kr/call/qna/selectQnaInfo.do?ctgId=CTG11906&mi=1318)에서 안내하는 대상 주택입니다.

| 조건 | 기준 |
|------|------|
| 주택 규모 | 국민주택규모(전용면적 85㎡) 이하 |
| 기준시가 | **4억원 이하** |
| 포함 대상 | 주거용 오피스텔, 고시원 포함 |

- 다가구주택의 경우 **가구당 전용면적** 기준 적용
- 국민주택규모 또는 기준시가 4억원 이하 중 **하나만 충족**하면 됨

## 3. 공제율과 한도

### 3.1 소득별 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **17%** |
| 5,500만원 초과 ~ 8,000만원 이하 | **15%** |

### 3.2 공제 한도

- 연간 월세액 **1,000만원**까지 공제 가능
- 최대 공제액: 1,000만원 × 17% = **170만원**

### 3.3 계산 예시

**연봉 4,000만원, 월세 60만원인 경우**
- 연간 월세: 60만원 × 12개월 = 720만원
- 공제율: 17% (총급여 5,500만원 이하)
- 세액공제액: 720만원 × 17% = **122만 4천원**

## 4. 필요 서류

연말정산 시 회사에 제출해야 하는 서류입니다.

| 서류 | 용도 |
|------|------|
| 주민등록표등본 | 전입신고 확인 |
| 임대차계약서 사본 | 임차 사실 증명 |
| 계좌이체 영수증 | 월세 납부 증빙 |

- 현금 납부 시 **무통장입금증** 등 지급 증빙 필요
- 간소화서비스에서 조회되지 않으면 **직접 제출** 필요

## 5. 주의사항

### 5.1 공제 불가 상황

- 과세연도 중 주택을 취득하여 **12월 31일 기준 유주택자**인 경우
- 주민등록상 **전입신고가 안 된** 경우
- 총급여 **8,000만원 초과**인 경우

### 5.2 배우자 공제 (2026년 시행)

2026년 1월 1일부터 세대주와 주소를 달리하는 배우자도 요건 충족 시 **별도로 월세 세액공제**를 받을 수 있습니다.
