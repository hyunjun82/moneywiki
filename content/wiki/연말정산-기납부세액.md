# 연말정산 기납부세액

기납부세액은 연말정산의 핵심 개념으로, 환급 또는 추가 납부를 결정하는 기준이 됩니다. [소득세법 제137조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 기납부세액이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 의미 | 1년간 **매월 급여에서 원천징수된** 소득세 합계 |
| 납부 시기 | 매월 급여 지급 시 |
| 납부 방법 | 회사가 급여에서 공제 후 국세청에 납부 |

### 1.2 원천징수 개념

```
매월 급여 → 간이세액표에 따른 세금 계산 → 원천징수 → 회사가 국세청에 납부
```

## 2. 연말정산 세금 계산 흐름

### 2.1 세금 계산 순서

```
총급여
  ↓
근로소득금액 (근로소득공제 적용)
  ↓
과세표준 (소득공제 적용)
  ↓
산출세액 (세율 적용)
  ↓
결정세액 (세액공제 적용)
  ↓
기납부세액과 비교
  ↓
환급 또는 추가납부
```

### 2.2 환급/추가납부 결정

| 상황 | 결과 |
|------|------|
| 기납부세액 > 결정세액 | **환급** (돌려받음) |
| 기납부세액 < 결정세액 | **추가납부** (더 내야 함) |
| 기납부세액 = 결정세액 | 환급/추가납부 없음 |

## 3. 원천징수 비율 선택

### 3.1 선택 가능 비율

[소득세법]에 따라 근로자가 선택 가능:

| 비율 | 효과 |
|------|------|
| **80%** | 매월 세금 적게, 연말 환급 적음/추가납부 가능 |
| **100%** | 기본 (간이세액표 그대로) |
| **120%** | 매월 세금 많이, 연말 환급 많음 |

### 3.2 비율 선택 전략

| 상황 | 권장 비율 |
|------|----------|
| 공제 항목 많은 경우 | 80~100% |
| 공제 항목 적은 경우 | 100~120% |
| 연말 환급 원하는 경우 | 120% |
| 매월 실수령 높이고 싶은 경우 | 80% |

## 4. 기납부세액 확인 방법

### 4.1 급여명세서 확인

| 항목 | 확인 내용 |
|------|----------|
| 소득세 | 매월 원천징수 금액 |
| 지방소득세 | 소득세의 10% |

### 4.2 원천징수영수증

| 항목 | 위치 |
|------|------|
| 기납부소득세 | [결정세액] 란 아래 |
| 기납부지방소득세 | 별도 표기 |

### 4.3 홈택스 조회

1. 홈택스 로그인
2. My홈택스 → 연말정산 → 지급명세서 조회

## 5. 계산 예시

### 5.1 환급 사례

**연봉 5,000만원, 매월 원천징수 25만원**

| 항목 | 금액 |
|------|------|
| 기납부세액 | 25만원 × 12개월 = **300만원** |
| 결정세액 (공제 적용 후) | **200만원** |
| **환급액** | 300만원 - 200만원 = **100만원** |

### 5.2 추가납부 사례

**연봉 5,000만원, 80% 원천징수 선택**

| 항목 | 금액 |
|------|------|
| 기납부세액 (80% 적용) | **240만원** |
| 결정세액 | **280만원** |
| **추가납부** | 280만원 - 240만원 = **40만원** |

## 6. 간이세액표

### 6.1 간이세액표란

| 항목 | 내용 |
|------|------|
| 목적 | 매월 원천징수 세액 계산 |
| 기준 | 월급여, 부양가족 수 |
| 적용 | 연말정산 전 **예정 세금** 성격 |

### 6.2 간이세액표 조회

- [국세청 간이세액표](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7878)
- 홈택스 → 조회/발급 → 간이세액표

## 7. 특수 상황

### 7.1 중도입사자

| 상황 | 기납부세액 |
|------|----------|
| 3월 입사 | 3~12월 원천징수 합계 |
| 전 직장 소득 | 합산하여 정산 |

### 7.2 중도퇴사자

| 상황 | 처리 |
|------|------|
| 퇴사 시 정산 | 퇴사 시점까지 연말정산 |
| 재취업 | 새 회사에서 합산 정산 |

### 7.3 휴직자

| 상황 | 기납부세액 |
|------|----------|
| 유급 휴직 | 휴직급여에서 원천징수 |
| 무급 휴직 | 해당 기간 기납부세액 없음 |

## 8. 주의사항

### 8.1 기납부세액 ≠ 실제 세금

- 기납부세액은 **추정치**
- 연말정산 후 확정된 세금 = 결정세액

### 8.2 지방소득세

- 소득세의 **10%**
- 기납부세액, 결정세액 모두 별도 계산
- 환급/추가납부도 별도

### 8.3 연봉 변동 시

- 승진, 인센티브 등으로 연봉 증가 시
- 추가납부 가능성 ↑
- 원천징수 비율 조정 고려
