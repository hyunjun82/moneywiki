# 연말정산 공제신고서

소득세액공제신고서 작성 방법을 정리합니다.

## 1. 신고서 개요

| 항목 | 내용 |
|------|------|
| 정식명칭 | 소득·세액 공제신고서 |
| 제출처 | 회사 (원천징수의무자) |
| 제출시기 | 1월~2월 |

## 2. 주요 기재 항목

| 구분 | 항목 |
|------|------|
| 인적공제 | 본인, 배우자, 부양가족 |
| 연금보험료 | 국민연금, 건강보험 |
| 특별소득공제 | 보험료, 주택자금 |
| 그 밖의 공제 | 신용카드, 기부금 |
| 세액공제 | 의료비, 교육비 |

## 3. 작성 방법

| 단계 | 내용 |
|------|------|
| 1 | 간소화자료 PDF 다운로드 |
| 2 | 공제신고서 양식 작성 |
| 3 | 추가 증빙 첨부 |
| 4 | 회사 제출 |

## 4. 첨부 서류

| 서류 | 발급처 |
|------|--------|
| 간소화자료 | 홈택스 |
| 주민등록등본 | 주민센터 |
| 기부금영수증 | 기부처 |
| 월세 계약서 | 본인 보관 |
