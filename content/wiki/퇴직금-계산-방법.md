# 퇴직금 계산 방법

퇴직금은 근로자의 노후를 보장하기 위한 법정 급여입니다. [근로자퇴직급여보장법](https://www.law.go.kr/법령/근로자퇴직급여보장법)에 따른 계산 방법을 정리합니다.

## 1. 퇴직금 지급 조건

### 1.1 기본 조건

| 조건 | 내용 |
|------|------|
| 근무 기간 | **1년 이상** 계속 근무 |
| 근로 형태 | 주 15시간 이상 |
| 사업장 | 모든 사업장 |

### 1.2 대상자

| 대상 | 퇴직금 지급 |
|------|------------|
| 정규직 | ✅ 대상 |
| 계약직 | ✅ 대상 (1년 이상) |
| 파트타임 | ✅ 대상 (주 15시간 이상) |
| 일용직 | ⚠️ 조건 충족 시 |

## 2. 퇴직금 계산 공식

### 2.1 기본 공식

```
퇴직금 = 평균임금 × 30일 × (재직일수 / 365)
```

또는 간단히:

```
퇴직금 = 1일 평균임금 × 30 × 근속연수
```

### 2.2 평균임금 계산

```
평균임금 = 퇴직 전 3개월 급여 총액 / 3개월 총 일수
```

| 항목 | 포함 여부 |
|------|----------|
| 기본급 | ✅ 포함 |
| 정기 상여금 | ✅ 포함 |
| 연차수당 | ✅ 포함 |
| 야근수당 | ✅ 포함 |
| 비과세 식대 | ❌ 제외 |

## 3. 계산 예시

### 3.1 기본 예시

**월급 300만원, 3년 근무**

| 항목 | 계산 |
|------|------|
| 3개월 급여 | 300만원 × 3 = 900만원 |
| 3개월 일수 | 91일 (31+28+31) |
| 1일 평균임금 | 900만원 / 91일 = 약 98,901원 |
| 퇴직금 | 98,901원 × 30일 × 3년 = **약 890만원** |

### 3.2 상여금 포함 예시

**월급 300만원, 연 상여금 400만원, 5년 근무**

| 항목 | 계산 |
|------|------|
| 3개월 급여 | 900만원 |
| 상여금 (3개월분) | 400만원 × 3/12 = 100만원 |
| 3개월 총액 | 1,000만원 |
| 1일 평균임금 | 1,000만원 / 91일 = 약 109,890원 |
| 퇴직금 | 109,890원 × 30일 × 5년 = **약 1,648만원** |

## 4. 평균임금 계산 상세

### 4.1 포함 항목

| 항목 | 포함 |
|------|------|
| 기본급 | ✅ |
| 직책수당 | ✅ |
| 근속수당 | ✅ |
| 정기 상여금 | ✅ (연간 지급액 × 3/12) |
| 연차수당 | ✅ (연간 지급액 × 3/12) |
| 연장근로수당 | ✅ |
| 야간근로수당 | ✅ |

### 4.2 제외 항목

| 항목 | 제외 |
|------|------|
| 비과세 식대 | ❌ |
| 경조사비 | ❌ |
| 실비변상 | ❌ |
| 일시적 상여 | ❌ |

## 5. 근속연수 계산

### 5.1 재직일수 산정

| 포함 | 제외 |
|------|------|
| 실 근무일 | - |
| 유급휴일 | - |
| 연차휴가 | - |
| 휴직기간 | ⚠️ 회사 규정에 따름 |

### 5.2 일수 → 연수 변환

| 재직일수 | 근속연수 |
|----------|----------|
| 365일 | 1년 |
| 730일 | 2년 |
| 548일 | 1.5년 |

## 6. 퇴직금 지급

### 6.1 지급 기한

| 항목 | 기한 |
|------|------|
| 원칙 | 퇴직일로부터 **14일 이내** |
| 예외 | 당사자 합의 시 연장 가능 |

### 6.2 미지급 시

| 상황 | 조치 |
|------|------|
| 14일 초과 | 지연이자 발생 (연 20%) |
| 미지급 | 노동부 진정, 민사소송 |

## 7. 퇴직연금 전환

### 7.1 퇴직연금 유형

| 유형 | 특징 |
|------|------|
| DB형 | 회사가 운용, 확정급여 |
| DC형 | 근로자가 운용, 확정기여 |
| IRP | 개인형 퇴직연금 |

### 7.2 퇴직연금 수령

| 항목 | 내용 |
|------|------|
| IRP 이체 | 퇴직금을 IRP로 이체 |
| 연금 수령 | 55세 이후 연금으로 수령 |
| 일시금 | 일시금 수령 가능 (세금 더 많음) |

## 8. 주의사항

### 8.1 1년 미만 근무

| 근무 기간 | 퇴직금 |
|----------|--------|
| 1년 미만 | ❌ 법적 의무 없음 |
| 11개월 29일 | ❌ |
| 1년 | ✅ |

### 8.2 주 15시간 미만

- 주 15시간 미만 근무 시 퇴직금 대상 아님
- 소정근로시간 기준

### 8.3 퇴직금 중간정산

| 조건 | 가능 여부 |
|------|----------|
| 무주택자 주택 구입 | ✅ 가능 |
| 6개월 이상 요양 | ✅ 가능 |
| 파산·회생 | ✅ 가능 |
| 일반적 요청 | ❌ 불가 (원칙) |
