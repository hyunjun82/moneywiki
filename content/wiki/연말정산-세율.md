소득세는 누진세율이 적용되어 소득이 높을수록 높은 세율이 적용돼요. [소득세법 제55조](https://www.law.go.kr/법령/소득세법)에 따른 세율을 정리해 볼게요.

## 소득세율표 (2026년)

과세표준별 세율이에요.

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | 6% | - |
| 1,400만원 초과 ~ 5,000만원 | 15% | 126만원 |
| 5,000만원 초과 ~ 8,800만원 | 24% | 576만원 |
| 8,800만원 초과 ~ 1.5억원 | 35% | 1,544만원 |
| 1.5억원 초과 ~ 3억원 | 38% | 1,994만원 |
| 3억원 초과 ~ 5억원 | 40% | 2,594만원 |
| 5억원 초과 ~ 10억원 | 42% | 3,594만원 |
| 10억원 초과 | 45% | 6,594만원 |

누진세율이라서 전체 소득에 단일 세율이 적용되는 게 아니라 각 구간별로 다른 세율이 적용돼요.

## 세금 계산 방법

산출세액은 (과세표준 x 세율) - 누진공제로 계산해요.

과세표준 4,000만원인 경우를 예로 들어볼게요.

구간별로 계산하면 1,400만원 x 6% = 84만원이고, 나머지 2,600만원 x 15% = 390만원이에요. 합계는 474만원이에요.

누진공제를 적용하면 4,000만원 x 15% - 126만원 = 474만원으로 같은 결과가 나와요.

## 연봉별 예상 세율

실효세율 예시예요.

| 연봉 | 예상 과세표준 | 실효세율 |
|------|--------------|----------|
| 3,000만원 | 약 1,500만원 | 약 6~7% |
| 5,000만원 | 약 2,500만원 | 약 8~10% |
| 7,000만원 | 약 4,000만원 | 약 10~12% |
| 1억원 | 약 6,500만원 | 약 15~18% |

실효세율은 실제 납부 세금을 총급여로 나눈 값이에요.

| 구분 | 설명 |
|------|------|
| 명목세율 | 과세표준 구간별 세율 (6~45%) |
| 실효세율 | 실제 납부 세금 / 소득 |

## 과세표준 계산

총급여에서 과세표준이 나오는 과정이에요.

| 단계 | 내용 |
|------|------|
| 총급여 | 비과세 제외한 급여 |
| 근로소득공제 | 총급여 구간별 공제 |
| 근로소득금액 | 총급여 - 근로소득공제 |
| 소득공제 | 인적공제, 특별소득공제 등 |
| 과세표준 | 근로소득금액 - 소득공제 |

총급여 5,000만원인 경우 예시예요.

| 항목 | 금액 |
|------|------|
| 총급여 | 5,000만원 |
| 근로소득공제 | 약 1,175만원 |
| 근로소득금액 | 3,825만원 |
| 인적공제 | 150만원 |
| 국민연금 | 225만원 |
| 기타 공제 | 200만원 |
| 과세표준 | 약 3,250만원 |

## 세율 적용 예시

과세표준 3,000만원인 경우예요.

| 계산 | 금액 |
|------|------|
| 3,000만원 x 15% | 450만원 |
| 누진공제 | -126만원 |
| 산출세액 | 324만원 |

과세표준 6,000만원인 경우예요.

| 계산 | 금액 |
|------|------|
| 6,000만원 x 24% | 1,440만원 |
| 누진공제 | -576만원 |
| 산출세액 | 864만원 |

과세표준 1억원인 경우예요.

| 계산 | 금액 |
|------|------|
| 1억원 x 35% | 3,500만원 |
| 누진공제 | -1,544만원 |
| 산출세액 | 1,956만원 |

## 지방소득세 포함

소득세에 지방소득세가 추가돼요.

| 항목 | 세율 |
|------|------|
| 소득세 | 6~45% |
| 지방소득세 | 소득세의 10% |
| 합계 | 6.6~49.5% |

산출세액 500만원인 경우 소득세 500만원 + 지방소득세 50만원 = 총 550만원이에요.

## 세율 절감 방법

과세표준을 낮추는 방법이에요.

| 방법 | 효과 |
|------|------|
| 인적공제 | 1인당 150만원 공제 |
| 연금저축 | 최대 600만원 공제 |
| 신용카드 | 최대 300만원 공제 |

세액공제를 활용하는 방법이에요.

| 방법 | 효과 |
|------|------|
| 연금저축 세액공제 | 16.5% 또는 13.2% |
| 의료비 세액공제 | 15% |
| 교육비 세액공제 | 15% |

## 주의사항

연봉 1억이라고 35% 세금을 내는 게 아니에요. 과세표준 8,800만원 초과분에만 35%가 적용돼요.

과세표준은 연봉보다 훨씬 낮아요. 각종 공제 적용 후 금액이기 때문이에요.
