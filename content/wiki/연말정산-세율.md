# 연말정산 세율 (소득세율)

소득세는 누진세율이 적용되어 소득이 높을수록 높은 세율이 적용됩니다. [소득세법 제55조](https://www.law.go.kr/법령/소득세법)에 따른 세율을 정리합니다.

## 1. 소득세율표 (2026년)

### 1.1 과세표준별 세율

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | **6%** | - |
| 1,400만원 초과 ~ 5,000만원 | **15%** | 126만원 |
| 5,000만원 초과 ~ 8,800만원 | **24%** | 576만원 |
| 8,800만원 초과 ~ 1.5억원 | **35%** | 1,544만원 |
| 1.5억원 초과 ~ 3억원 | **38%** | 1,994만원 |
| 3억원 초과 ~ 5억원 | **40%** | 2,594만원 |
| 5억원 초과 ~ 10억원 | **42%** | 3,594만원 |
| 10억원 초과 | **45%** | 6,594만원 |

### 1.2 누진세율의 의미

- 전체 소득에 단일 세율 적용 ❌
- 각 구간별로 **다른 세율** 적용 ✅

## 2. 세금 계산 방법

### 2.1 계산 공식

```
산출세액 = (과세표준 × 세율) - 누진공제
```

### 2.2 계산 예시

**과세표준 4,000만원인 경우**

| 방법 1: 구간별 계산 |
|-------------------|
| 1,400만원 × 6% = 84만원 |
| (4,000만원 - 1,400만원) × 15% = 390만원 |
| **합계: 474만원** |

| 방법 2: 누진공제 적용 |
|---------------------|
| 4,000만원 × 15% - 126만원 |
| = 600만원 - 126만원 |
| **= 474만원** |

## 3. 연봉별 예상 세율

### 3.1 실효세율 예시

| 연봉 | 예상 과세표준 | 실효세율 |
|------|--------------|----------|
| 3,000만원 | ~1,500만원 | 약 6~7% |
| 5,000만원 | ~2,500만원 | 약 8~10% |
| 7,000만원 | ~4,000만원 | 약 10~12% |
| 1억원 | ~6,500만원 | 약 15~18% |

※ 실효세율 = 실제 납부 세금 / 총급여

### 3.2 실효세율 vs 명목세율

| 구분 | 설명 |
|------|------|
| 명목세율 | 과세표준 구간별 세율 (6~45%) |
| 실효세율 | 실제 납부 세금 / 소득 |

## 4. 과세표준 계산

### 4.1 과세표준 산출 흐름

```
총급여
  ↓ 근로소득공제
근로소득금액
  ↓ 인적공제, 특별소득공제 등
과세표준
  ↓ 세율 적용
산출세액
```

### 4.2 총급여 → 과세표준 예시

**총급여 5,000만원**

| 항목 | 금액 |
|------|------|
| 총급여 | 5,000만원 |
| 근로소득공제 | 약 1,175만원 |
| 근로소득금액 | 3,825만원 |
| 인적공제 | 150만원 |
| 국민연금 | 225만원 |
| 기타 공제 | 200만원 |
| **과세표준** | **약 3,250만원** |

## 5. 세율 적용 예시

### 5.1 과세표준 3,000만원

| 계산 | 금액 |
|------|------|
| 3,000만원 × 15% | 450만원 |
| 누진공제 | -126만원 |
| **산출세액** | **324만원** |

### 5.2 과세표준 6,000만원

| 계산 | 금액 |
|------|------|
| 6,000만원 × 24% | 1,440만원 |
| 누진공제 | -576만원 |
| **산출세액** | **864만원** |

### 5.3 과세표준 1억원

| 계산 | 금액 |
|------|------|
| 1억원 × 35% | 3,500만원 |
| 누진공제 | -1,544만원 |
| **산출세액** | **1,956만원** |

## 6. 지방소득세 포함

### 6.1 총 세금

| 항목 | 세율 |
|------|------|
| 소득세 | 6~45% |
| 지방소득세 | 소득세의 10% |
| **합계** | 6.6~49.5% |

### 6.2 실제 세금 계산

**산출세액 500만원인 경우**

| 항목 | 금액 |
|------|------|
| 소득세 | 500만원 |
| 지방소득세 | 50만원 |
| **총 세금** | **550만원** |

## 7. 세율 절감 방법

### 7.1 과세표준 낮추기

| 방법 | 효과 |
|------|------|
| 인적공제 | 1인당 150만원 공제 |
| 연금저축 | 최대 600만원 공제 |
| 신용카드 | 최대 300만원 공제 |

### 7.2 세액공제 활용

| 방법 | 효과 |
|------|------|
| 연금저축 세액공제 | 16.5% 또는 13.2% |
| 의료비 세액공제 | 15% |
| 교육비 세액공제 | 15% |

## 8. 주의사항

### 8.1 세율 오해

| 오해 | 실제 |
|------|------|
| 연봉 1억이면 35% 세금 | ❌ |
| 과세표준 8,800만원 초과분만 35% | ✅ |

### 8.2 과세표준 ≠ 연봉

- 과세표준은 연봉보다 **훨씬 낮음**
- 각종 공제 적용 후 금액
