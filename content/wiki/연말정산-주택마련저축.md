# 연말정산 주택마련저축

주택마련저축 소득공제를 정리합니다.

## 1. 공제 요건

| 요건 | 기준 |
|------|------|
| 총급여 | **7천만원 이하** |
| 세대 | 무주택 세대주 |

## 2. 공제율과 한도

| 항목 | 기준 |
|------|------|
| 공제율 | **40%** |
| 한도 | 연 **300만원** |

## 3. 대상 상품

| 상품 | 공제 |
|------|------|
| 주택청약종합저축 | ✅ |
| 청년우대형청약 | ✅ |
| 청약저축 | ✅ |
| 청약예금 | ✅ |

## 4. 공제 계산

| 납입액 | 공제액(40%) |
|--------|-------------|
| 300만원 | 120만원 |
| 500만원 | 200만원 |
| 750만원 | 300만원 (한도) |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 7천만원 초과 | 공제 불가 |
| 세대원 | 공제 불가 |
