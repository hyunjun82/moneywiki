# 권리금 보호

상가 임차인이 영업하며 쌓은 권리금을 보호하는 제도입니다. 임대인은 정당한 사유 없이 권리금 회수를 방해할 수 없습니다.

## 1. 권리금이란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 영업시설, 거래처, 신용 등의 가치 |
| 대상 | 상가건물 임대차 |
| 근거 | 상가건물임대차보호법 제10조의3 |

### 1.2 권리금 종류

| 종류 | 내용 |
|------|------|
| 바닥권리금 | 상권, 입지 프리미엄 |
| 시설권리금 | 인테리어, 시설 비용 |
| 영업권리금 | 거래처, 고객, 노하우 |

### 1.3 권리금 특성

| 특성 | 설명 |
|------|------|
| 관행 | 상관행에서 발전 |
| 법정 | 법으로 보호 |
| 양도 | 후임 임차인에게 양도 |

## 2. 권리금 보호 제도

### 2.1 보호 대상

| 구분 | 기준 (환산보증금) |
|------|-----------------|
| 서울 | **9억원 이하** |
| 수도권 과밀억제권역 | 6.9억원 이하 |
| 광역시 등 | 5.4억원 이하 |
| 그 외 | 3.7억원 이하 |

※ 환산보증금 = 보증금 + (월세 × 100)

### 2.2 임대인 의무

| 의무 | 내용 |
|------|------|
| 방해 금지 | 권리금 회수 방해 금지 |
| 정보 제공 | 신규 임차인에게 정보 제공 |

### 2.3 방해 행위 예시

| 행위 | 해당 |
|------|------|
| 신규 임차인과 계약 거절 | ✅ |
| 현저히 높은 차임 요구 | ✅ |
| 현저히 높은 보증금 요구 | ✅ |
| 정당한 사유 없이 조건 거부 | ✅ |

## 3. 정당한 사유

### 3.1 방해해도 되는 경우

| 사유 | 설명 |
|------|------|
| 신규 임차인 자격 부족 | 자금력, 영업능력 |
| 임차인 의무 위반 | 3기 연체, 무단전대 등 |
| 철거·재건축 | 1년 6개월 이내 착공 |
| 직접 영업 | 임대인이 직접 영업 |

### 3.2 자격 부족 기준

| 기준 | 내용 |
|------|------|
| 자금력 | 차임 지급 능력 |
| 신용 | 신용 불량 여부 |
| 영업능력 | 동종 영업 경험 |

## 4. 권리금 회수

### 4.1 회수 절차

| 단계 | 내용 |
|------|------|
| 1 | 신규 임차인 물색 |
| 2 | 권리금 협상 |
| 3 | 임대인에게 주선 |
| 4 | 신규 임차인과 임대차 계약 |
| 5 | 권리금 수령 |

### 4.2 주선 의무

| 항목 | 내용 |
|------|------|
| 시기 | 임대차 종료 3개월 전~ |
| 내용 | 신규 임차인 주선 |
| 정보 | 영업정보 등 제공 |

### 4.3 권리금 계약

| 항목 | 기재 |
|------|------|
| 금액 | 권리금 총액 |
| 구성 | 시설, 영업, 바닥 |
| 지급 | 지급 시기, 방법 |
| 조건 | 임대차 계약 성립 조건 등 |

## 5. 손해배상

### 5.1 배상 청구

| 요건 | 내용 |
|------|------|
| 방해 행위 | 임대인의 방해 |
| 손해 발생 | 권리금 회수 불가 |
| 인과관계 | 방해로 인한 손해 |

### 5.2 배상 범위

| 항목 | 내용 |
|------|------|
| 권리금 | 신규 임차인이 지급하려던 금액 |
| 감정 | 감정평가 권리금 |
| 손해 | 실제 손해 |

### 5.3 청구 기한

| 기한 | 내용 |
|------|------|
| 소멸시효 | 임대차 종료 후 **3년** |
| 주의 | 기한 내 청구 필요 |

## 6. 권리금 계약서

### 6.1 필수 기재 사항

| 항목 | 내용 |
|------|------|
| 당사자 | 양도인, 양수인 |
| 목적물 | 상가 표시 |
| 권리금 | 금액, 구성 |
| 지급 | 시기, 방법 |
| 인수 항목 | 시설, 재고 등 |

### 6.2 계약서 예시

```
[권리금 계약서]

1. 상가 표시: ○○시 ○○구 ○○로 ○○호
2. 권리금 총액: 금 ○○원

   - 시설권리금: ○○원
   - 영업권리금: ○○원
   - 바닥권리금: ○○원

3. 지급 방법
   - 계약금: ○○원 (계약 시)
   - 잔금: ○○원 (임대차계약 성립 시)

4. 인수 항목
   - 인테리어, 집기류, 영업장비
   - 기존 거래처 정보

5. 조건
   - 본 계약은 양수인과 임대인 간 임대차계약
     성립을 조건으로 한다.

2026년 ○월 ○일
양도인: ○○○ (인)
양수인: ○○○ (인)
```

| 방법 | 내용 |
|------|------|
| 당사자 협의 | 임대인과 직접 협의 |
| 조정 협의 | 제3자 중재 |

### 7.2 분쟁조정

| 기관 | 연락처 |
|------|--------|
| 상가건물임대차분쟁조정위원회 | 1588-5500 |

### 7.3 소송

| 종류 | 내용 |
|------|------|
