연말정산 환급금 확인 방법을 정리해 볼게요. [국세청 홈택스](https://www.hometax.go.kr)에서 조회할 수 있어요.

## 조회 방법

홈택스에서 조회하는 방법이에요.

| 단계 | 절차 |
|------|------|
| 1 | 홈택스 로그인 |
| 2 | My홈택스 클릭 |
| 3 | 연말정산/장려금 메뉴 |
| 4 | 지급명세서 조회 |

손택스(모바일)에서 조회하는 방법이에요.

| 단계 | 절차 |
|------|------|
| 1 | 손택스 앱 실행 |
| 2 | 로그인 |
| 3 | 조회/발급 > 연말정산 |

## 환급 시기

환급금 지급 시기예요.

| 구분 | 시기 |
|------|------|
| 회사 정산 | 2~3월 급여 |
| 종소세 신고 | 6~7월 |

## 환급금 계산

환급금 계산 기준이에요.

| 항목 | 계산 |
|------|------|
| 환급 | 기납부세액 > 결정세액 |
| 추가납부 | 기납부세액 < 결정세액 |

## 조회 가능 시점

조회 가능 시점이에요.

| 시점 | 조회 |
|------|------|
| 회사 제출 후 | 3월~ |
| 지급명세서 | 3월 10일 이후 |
