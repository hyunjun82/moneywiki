# 기초연금

기초연금은 노후 소득 보장을 위해 만 65세 이상 어르신에게 지급되는 연금입니다. [기초연금법](https://www.law.go.kr/법령/기초연금법)에 따른 내용을 정리합니다.

## 1. 기초연금 개요

### 1.1 수급 자격

| 요건 | 기준 |
|------|------|
| 나이 | 만 **65세 이상** |
| 국적 | 대한민국 국적 |
| 거주 | 국내 거주 |
| 소득 | 소득인정액 **선정기준액 이하** |

### 1.2 선정기준액 (2026년)

| 가구 유형 | 선정기준액 |
|----------|-----------|
| 단독가구 | 약 **213만원** |
| 부부가구 | 약 **340만원** |

## 2. 소득인정액 계산

### 2.1 계산 공식

```
소득인정액 = 소득평가액 + 재산의 소득환산액
```

### 2.2 소득평가액

| 소득 종류 | 산정 |
|----------|------|
| 근로소득 | (근로소득 - 110만원) × 70% |
| 사업소득 | 사업소득 전액 |
| 재산소득 | 이자·배당·임대소득 |
| 공적이전소득 | 국민연금 등 |

### 2.3 재산의 소득환산

```
소득환산액 = (재산 - 기본재산액 - 부채) × 소득환산율 ÷ 12
```

| 지역 | 기본재산액 |
|------|-----------|
| 대도시 | 1억3,500만원 |
| 중소도시 | 8,500만원 |
| 농어촌 | 7,250만원 |

| 재산 유형 | 환산율 |
|----------|--------|
| 일반재산 | 연 4% |
| 금융재산 | 연 4% |
| 고급자동차 | 연 100% |

## 3. 기초연금 금액

### 3.1 기준연금액 (2026년)

| 가구 유형 | 최대 금액 |
|----------|----------|
| 단독가구 | 약 **34만원** |
| 부부가구 | 약 **54만원** (각 27만원) |

### 3.2 부부 감액

| 상황 | 감액 |
|------|------|
| 부부 모두 수급 | 각 **20%** 감액 |

### 3.3 소득역전방지 감액

| 상황 | 감액 |
|------|------|
| 소득인정액 + 기초연금 > 선정기준액 | 초과분만큼 감액 |

## 4. 국민연금 연계 감액

### 4.1 감액 기준

| 국민연금 수령액 | 감액 |
|---------------|------|
| 기준연금액의 150% 이하 | 감액 없음 |
| 기준연금액의 150% 초과 | 최대 50% 감액 |

### 4.2 감액 예시

**국민연금 월 80만원 수령**

| 항목 | 금액 |
|------|------|
| 기준연금액 | 34만원 |
| 150% 기준 | 51만원 |
| 국민연금 | 80만원 (기준 초과) |
| 감액 적용 | 약 17만원 감액 |
| **지급액** | **약 17만원** |

## 5. 신청 방법

### 5.1 신청 시기

| 시기 | 내용 |
|------|------|
| 신청 가능 | 만 65세 생일 **1개월 전**부터 |
| 소급 지급 | 신청일이 속한 달부터 지급 |

### 5.2 신청 장소

| 장소 | 방법 |
|------|------|
| 주민센터 | 방문 신청 |
| 국민연금공단 | 방문 신청 |
| 복지로 | 온라인 신청 |

### 5.3 필요 서류

| 서류 | 용도 |
|------|------|
| 신분증 | 본인 확인 |
| 통장 사본 | 연금 입금용 |
| 금융정보 제공 동의서 | 소득·재산 조회 |
| 임대차계약서 | 임차인의 경우 |

## 6. 지급

### 6.1 지급일

| 항목 | 내용 |
|------|------|
| 지급일 | 매월 **25일** |
| 휴일 시 | 전일 지급 |

### 6.2 지급 방법

| 방법 | 내용 |
|------|------|
| 본인 계좌 | 지정 계좌 입금 |
| 대리수령 | 제한적 허용 |

## 7. 수급 자격 유지

### 7.1 정기 확인

| 항목 | 주기 |
|------|------|
| 소득·재산 조사 | **매년** |
| 수급자격 확인 | 매년 |

### 7.2 변동 신고

| 변동 사항 | 신고 |
|----------|------|
| 주소 변경 | 30일 이내 |
| 소득·재산 변동 | 30일 이내 |
| 가구원 변동 | 30일 이내 |

## 8. 수급 제한

### 8.1 수급 제한 사유

| 사유 | 제한 |
|------|------|
| 공무원연금 수급 | ❌ 수급 불가 |
| 사학연금 수급 | ❌ 수급 불가 |
| 군인연금 수급 | ❌ 수급 불가 |
| 별정우체국연금 수급 | ❌ 수급 불가 |

### 8.2 예외

| 상황 | 수급 |
|------|------|
| 직역연금 수급권자의 배우자 | ⚠️ 개별 판단 |
| 유족연금 수급 | ⚠️ 개별 판단 |

## 9. 다른 제도와의 관계

### 9.1 병급 가능

| 제도 | 병급 |
|------|------|
| 국민연금 | ✅ (감액 가능) |
| 기초생활보장 | ✅ (소득 산입) |
| 장애인연금 | ✅ (선택) |
