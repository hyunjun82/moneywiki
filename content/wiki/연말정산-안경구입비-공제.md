# 연말정산 안경구입비 공제

안경·콘택트렌즈 구입비는 의료비 세액공제의 일부로, **1인당 50만원 한도**가 적용됩니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리합니다.

## 1. 공제 기본 조건

[소득세법 시행령 제118조의5](https://www.law.go.kr/법령/소득세법시행령)에 따른 공제 요건입니다.

### 1.1 공제 대상

| 조건 | 내용 |
|------|------|
| 용도 | **시력보정용**에 한함 |
| 구입처 | 안경점 (사업자등록 사업장) |
| 대상자 | 근로자 본인 + 기본공제 대상자 |

### 1.2 공제율과 한도

| 항목 | 내용 |
|------|------|
| 공제율 | **15%** (의료비 세액공제) |
| 한도 | **1인당 연간 50만원** |
| 최대 공제액 | 50만원 × 15% = **7만 5천원** (1인 기준) |

## 2. 공제 대상 항목

### 2.1 공제 가능

| 항목 | 공제 여부 |
|------|----------|
| 시력보정용 안경 | ✅ 가능 |
| 시력보정용 콘택트렌즈 | ✅ 가능 |
| 도수 있는 선글라스 | ✅ 가능 |
| 돋보기 (노안용) | ✅ 가능 |

### 2.2 공제 불가

| 항목 | 공제 여부 |
|------|----------|
| 미용 목적 선글라스 | ❌ 불가 |
| 컬러렌즈 (미용) | ❌ 불가 |
| 서클렌즈 | ❌ 불가 |
| 안경테만 교체 | ❌ 불가 |
| 안경 수리비 | ❌ 불가 |

## 3. 한도 적용

### 3.1 1인당 50만원

가족 구성원 **각각** 50만원 한도 적용

| 가족 | 안경 구입비 | 공제 대상 |
|------|------------|----------|
| 본인 | 60만원 | 50만원 |
| 배우자 | 30만원 | 30만원 |
| 자녀1 | 40만원 | 40만원 |
| **합계** | 130만원 | **120만원** |

### 3.2 의료비 전체 한도와 별도

안경 50만원 한도는 의료비 세액공제 내에서 **별도 관리**

| 구분 | 한도 |
|------|------|
| 본인·장애인·65세 이상 의료비 | 한도 없음 |
| 그 외 부양가족 의료비 | 700만원 |
| **안경·콘택트렌즈** | **1인당 50만원** |

## 4. 공제 계산

### 4.1 의료비 3% 기준

안경 구입비도 의료비에 포함되어 **총급여 3% 초과분**부터 공제

### 4.2 계산 예시

**연봉 4,000만원, 안경 40만원 + 병원비 200만원 지출**

1. 3% 기준액: 4,000만원 × 3% = 120만원
2. 총 의료비: 240만원
3. 공제 대상: 240만원 - 120만원 = 120만원
4. 안경 공제: 40만원 × 15% = 6만원
5. 병원비 공제: 80만원 × 15% = 12만원
6. **총 세액공제: 18만원**

## 5. 증빙서류

### 5.1 필수 제출 서류

안경 구입비는 **연말정산 간소화에서 조회되지 않음**

| 서류 | 내용 |
|------|------|
| **안경 구입 영수증** | 안경점에서 발급 |
| 기재 사항 | 구입자 성명, 시력보정용 명시 |

### 5.2 영수증 발급

안경점에서 **'시력보정용 안경(콘택트렌즈) 구입 영수증'** 요청

필수 기재 사항:
- 구입자 성명
- 주민등록번호 (앞 6자리)
- 구입 일자
- 금액
- **"시력보정용"** 문구

## 6. 라식·라섹과 구분

### 6.1 별도 공제 항목

| 항목 | 분류 | 한도 |
|------|------|------|
| 안경·콘택트렌즈 | 안경 구입비 | 50만원 |
| 라식·라섹 수술 | 일반 의료비 | 700만원 (본인은 한도 없음) |

### 6.2 중복 공제 가능

- 라식 수술: 일반 의료비로 공제
- 수술 후 안경 구입: 안경 구입비로 별도 공제
- **두 가지 모두 공제 가능**

## 7. 주의사항

### 7.1 사업자 안경점

- **사업자등록된 안경점**에서 구입해야 공제 가능
- 해외 구입, 직구 안경은 공제 불가

### 7.2 영수증 보관

- 연말정산 간소화에 안 나오므로 **영수증 필수 보관**
- 분실 시 안경점에서 재발급 요청

### 7.3 부양가족 확인

- 안경을 구입한 부양가족이 **기본공제 대상자**여야 함
- 소득이 있는 성인 자녀는 기본공제 대상이 아닐 수 있음
