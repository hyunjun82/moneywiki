안경과 콘택트렌즈 구입비는 의료비 세액공제의 일부로 1인당 50만원 한도가 적용돼요. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리해 볼게요.

## 공제 기본 조건

[소득세법 시행령 제118조의5](https://www.law.go.kr/법령/소득세법시행령)에 따른 공제 요건이에요.

| 항목 | 내용 |
|------|------|
| 용도 | 시력보정용에 한함 |
| 구입처 | 안경점 (사업자등록 사업장) |
| 대상자 | 근로자 본인 + 기본공제 대상자 |

## 공제율과 한도

의료비 세액공제의 일부로 적용돼요.

| 항목 | 내용 |
|------|------|
| 공제율 | 15% |
| 한도 | 1인당 연간 50만원 |
| 최대 공제액 | 50만원 x 15% = 7만 5천원 (1인 기준) |

## 공제 대상 항목

시력보정용인지에 따라 공제 가능 여부가 달라요.

### 공제 가능

| 항목 | 공제 가능 |
|------|----------|
| 시력보정용 안경 | 가능 |
| 시력보정용 콘택트렌즈 | 가능 |
| 도수 있는 선글라스 | 가능 |
| 돋보기 (노안용) | 가능 |

### 공제 불가

| 항목 | 공제 가능 |
|------|----------|
| 미용 목적 선글라스 | 불가 |
| 컬러렌즈 (미용) | 불가 |
| 서클렌즈 | 불가 |
| 안경테만 교체 | 불가 |
| 안경 수리비 | 불가 |

## 한도 적용

가족 구성원 각각 50만원 한도가 적용돼요.

| 가족 | 안경 구입비 | 공제 대상 |
|------|------------|----------|
| 본인 | 60만원 | 50만원 |
| 배우자 | 30만원 | 30만원 |
| 자녀1 | 40만원 | 40만원 |
| 합계 | 130만원 | 120만원 |

안경 50만원 한도는 의료비 세액공제 내에서 별도 관리돼요.

| 구분 | 한도 |
|------|------|
| 본인·장애인·65세 이상 의료비 | 한도 없음 |
| 그 외 부양가족 의료비 | 700만원 |
| 안경·콘택트렌즈 | 1인당 50만원 |

## 공제 계산

안경 구입비도 의료비에 포함되어 총급여 3% 초과분부터 공제돼요.

연봉 4,000만원, 안경 40만원 + 병원비 200만원 지출한 경우예요.

| 항목 | 금액 |
|------|------|
| 3% 기준액 | 4,000만원 x 3% = 120만원 |
| 총 의료비 | 240만원 |
| 공제 대상 | 240만원 - 120만원 = 120만원 |
| 안경 공제 | 40만원 x 15% = 6만원 |
| 병원비 공제 | 80만원 x 15% = 12만원 |
| 총 세액공제 | 18만원 |

## 증빙서류

안경 구입비는 연말정산 간소화에서 조회되지 않아요.

| 서류 | 내용 |
|------|------|
| 안경 구입 영수증 | 안경점에서 발급 |
| 기재 사항 | 구입자 성명, 시력보정용 명시 |

영수증에 필수로 들어가야 할 내용이에요.

- 구입자 성명
- 주민등록번호 (앞 6자리)
- 구입 일자
- 금액
- 시력보정용 문구

## 라식·라섹과 구분

안경과 라식·라섹은 별도 공제 항목이에요.

| 항목 | 분류 | 한도 |
|------|------|------|
| 안경·콘택트렌즈 | 안경 구입비 | 50만원 |
| 라식·라섹 수술 | 일반 의료비 | 700만원 (본인은 한도 없음) |

라식 수술과 수술 후 안경 구입을 모두 공제받을 수 있어요.

## 주의사항

사업자등록된 안경점에서 구입해야 공제 가능해요. 해외 구입이나 직구 안경은 공제 안 돼요.

영수증을 분실하면 안경점에서 재발급받을 수 있어요. 연말정산 간소화에 안 나오므로 꼭 보관하세요.

안경을 구입한 부양가족이 기본공제 대상자여야 해요. 소득이 있는 성인 자녀는 기본공제 대상이 아닐 수 있어요.
