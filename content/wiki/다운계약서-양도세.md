"어차피 비과세인데 계약서 금액 좀 낮춰도 상관없지 않아요?"

솔직히 이런 생각 한 번쯤 해보셨을 거예요. 하지만 **절대 하시면 안 돼요**. 비과세 요건 다 갖춰도 다운계약서 쓰면 비과세 못 받고, 가산세에 과태료까지 폭탄 맞아요. 왜 그런지 설명드릴게요.

## 핵심 요약

| 항목 | 내용 |
|------|------|
| 비과세 | **적용 불가** (요건 충족해도) |
| 양도세 | **전액 과세** |
| 가산세 | 최대 **40%** |
| 과태료 | 취득가액의 **10%** |
| 매수자 | **똑같이 불이익** |

## 다운계약서가 뭐예

### 정의

| 용어 | 설명 |
|------|------|
| **다운계약서** | 실제 거래가액보다 **낮게** 작성한 계약서 |
| **업계약서** | 실제 거래가액보다 **높게** 작성한 계약서 |

둘 다 **허위계약서**예요. 법적으로 동일한 불이익이 적용돼요.

## 다운계약서 # 왜 쓰나

```
매도자: 양도세 줄이려고 (다운)
매수자: 취득세 줄이려고 (다운), 대출 늘리려고 (업)
```

**하지만 적발되면 오히려 더 큰 손해!**

## 실제 사례

### 김나라씨 사례

김나라씨는 1세대1주택 비과세 요건을 **완벽하게 충족**했어요.

```
'17.1월           →    '23.10월                    →    '25.10월
A주택 취득            A주택 양도                        국세청 확인
(4억원)              실제: 10억원                      다운계약 적발
                    계약서: 9억원 (1억 다운)
```

김나라씨는 이렇게 생각했어요.

> "어차피 비과세인데 계약서 금액은 상관없지 않나?"

**결과: 비과세 부인 + 양도세 + 가산세 폭탄!**

## # 왜 이렇게 됐을

소득세법 제91조 제2항에서 명확하게 규정하고 있어요.

> "매매계약서의 거래가액을 실지거래가액과 다르게 적은 경우 비과세 또는 감면이 부인된다"

**비과세 요건 충족 여부와 관계없이**, 허위계약서 작성 자체가 비과세 배제 사유예요.

## 다운계약서 불이익 총정리

### 1. 비과세·감면 적용 배제

| 상황 | 결과 |
|------|------|
| 1세대1주택 비과세 요건 충족 | **비과세 부인** |
| 장기보유특별공제 요건 충족 | **공제 부인 가능** |
| 기타 세금 감면 요건 충족 | **감면 부인** |

### 2. 가산세 부과

| 가산세 종류 | 금액 |
|------------|------|
| 무(과소)신고가산세 | 납부세액의 **최대 40%** |
| 납부지연가산세 | 미납세액 × 0.022% × 지연일수 |

### 3. 과태료 부과

| 항목 | 금액 |
|------|------|
| 부동산거래신고법 위반 | 취득가액의 **최대 10%** |

### 4. 매수자도 불이익

| 시점 | 불이익 |
|------|--------|
| 나중에 해당 주택 양도 시 | **비과세·감면 적용 부인** |

매수자도 양도자와 **동일한 불이익**을 받아요. 나중에 그 집 팔 때 비과세 못 받을 수 있어요.

## 비과세 배제 금액 계산

### 계산 방법

```
비과세 배제 금액 = Min(①, ②)

① 비과세 적용하지 않았을 경우의 양도소득 산출세액
② 계약서 거래가액과 실거래가액의 차액
```

### 예시

```
실제 거래가액: 10억원
계약서 금액: 9억원
차액: 1억원

① 양도소득 산출세액: 3,000만원 (가정)
② 차액: 1억원

→ 비과세 배제 금액 = Min(3,000만원, 1억원) = 3,000만원
→ 최소 3,000만원의 양도세 + 가산세 + 과태료
```

## 국세청은 어떻게 알아내

### 적발 방법

| 방법 | 내용 |
|------|------|
| 실거래가 신고 | 부동산거래신고 내용 확인 |
| 금융 거래 추적 | 계좌 입출금 내역 확인 |
| 국토부 연계 | 실거래가 시스템 데이터 |
| 제보 | 거래 당사자, 중개인 등 |

### 적발 시점

```
양도 후 ~5년 이내 (부과제척기간)
- 일반: 5년
- 사기 등 부정행위: 10년
```

## 다운계약서 절대 하지 마세

### 리스크 요약

| 생각 | 현실 |
|------|------|
| "어차피 비과세인데..." | 비과세 **부인** |
| "금액 차이 적으니까..." | 차액만큼 **세금 추징** |
| "안 걸리겠지..." | 금융 추적으로 **적발** |
| "매수자만 이득인데..." | 매수자도 **불이익** |

### 올바른 방법

```
1. 실제 거래금액으로 계약서 작성
   ↓
2. 실거래가 정확히 신고
   ↓
3. 비과세 요건 충족하면 → 비과세 적용
   ↓
4. 비과세 요건 미충족이면 → 정상 세금 납부
```

정상적으로 비과세 요건만 갖추면 양도세 없이 팔 수 있어요. **굳이 위험을 감수할 이유가 없어요.**

## 관련 법령

### 소득세법 제91조 제2항

> "거래당사자가 매매계약서의 거래가액을 실지거래가액과 다르게 적은 경우에는 해당 자산에 대하여 양도소득세의 비과세 또는 감면에 관한 규정을 적용할 때 비과세 또는 감면받았거나 받을 세액에서 금액을 빼는 것"

### 국기법 제47조의3, 제47조의4

- 무(과소)신고가산세: 납부세액의 최대 40%
- 납부지연가산세: 미납세액 × 0.022% × 지연일수

### 부동산거래신고법 제28조

- 허위신고 과태료: 취득가액의 10% 이하
