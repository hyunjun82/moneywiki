# 연말정산 결정세액

결정세액 계산을 정리합니다.

## 1. 계산 순서

```
총급여 - 근로소득공제 = 근로소득금액
근로소득금액 - 소득공제 = 과세표준
과세표준 × 세율 = 산출세액
산출세액 - 세액공제 = 결정세액
```

## 2. 환급/추가납부

| 비교 | 결과 |
|------|------|
| 기납부세액 > 결정세액 | **환급** |
| 기납부세액 < 결정세액 | **추가납부** |
| 기납부세액 = 결정세액 | 없음 |

## 3. 기납부세액

| 항목 | 내용 |
|------|------|
| 원천징수 | 매월 급여에서 공제된 세금 |
| 납세조합 | 납세조합 납부세액 |

## 4. 계산 예시

| 항목 | 금액 |
|------|------|
| 산출세액 | 500만원 |
| 세액공제 | -150만원 |
| **결정세액** | **350만원** |
| 기납부세액 | 400만원 |
| **환급액** | **50만원** |

## 5. 최저한세

| 항목 | 내용 |
|------|------|
| 결정세액 | 0원 이하 불가 |
| 최저 | 0원 |
