# 아동수당

아동수당은 아동 양육에 따른 경제적 부담을 줄이기 위한 제도입니다. [아동수당법](https://www.law.go.kr/법령/아동수당법)에 따른 내용을 정리합니다.

## 1. 아동수당 개요

### 1.1 지급 대상

| 조건 | 기준 |
|------|------|
| 연령 | 만 **8세 미만** (0~95개월) |
| 국적 | 대한민국 국적 |
| 거주 | 국내 거주 |

### 1.2 지급 금액

| 항목 | 금액 |
|------|------|
| 월 지급액 | **10만원** |
| 연 지급액 | **120만원** |

### 1.3 지급 방식

| 항목 | 내용 |
|------|------|
| 지급일 | 매월 **25일** |
| 지급 방법 | 계좌 입금 |

## 2. 신청 방법

### 2.1 신청 시기

| 시기 | 방법 |
|------|------|
| 출생 시 | 출생신고와 함께 자동 신청 |
| 이후 | 수시 신청 가능 |

### 2.2 신청 방법

| 방법 | 내용 |
|------|------|
| 행복출산 원스톱 | 출생신고 시 동시 신청 |
| 주민센터 | 방문 신청 |
| 복지로 | 온라인 신청 |
| 정부24 | 온라인 신청 |

### 2.3 필요 서류

| 서류 | 용도 |
|------|------|
| 신분증 | 신청인 확인 |
| 통장 사본 | 수당 입금 계좌 |
| 가족관계증명서 | 아동 확인 (자동 조회 가능) |

## 3. 소급 지급

### 3.1 소급 기준

| 조건 | 소급 기간 |
|------|----------|
| 출생 후 60일 이내 신청 | 출생월부터 지급 |
| 60일 이후 신청 | 신청월부터 지급 |

### 3.2 예시

| 상황 | 지급 시작 |
|------|----------|
| 1월 출생, 2월 신청 | 1월분부터 |
| 1월 출생, 5월 신청 | 5월분부터 |

## 4. 지급 기간

### 4.1 시작과 종료

| 항목 | 시점 |
|------|------|
| 시작 | 출생월 (60일 내 신청 시) |
| 종료 | 만 8세 생일 전달 |

### 4.2 연령 계산 예시

**2020년 7월 15일 출생**

| 항목 | 날짜 |
|------|------|
| 만 8세 | 2028년 7월 15일 |
| 마지막 지급 | 2028년 6월 25일 |

## 5. 특수 상황

### 5.1 해외 체류

| 기간 | 지급 |
|------|------|
| 90일 미만 | 정상 지급 |
| 90일 이상 | 지급 정지 |

### 5.2 시설 보호 아동

| 대상 | 지급 |
|------|------|
| 아동복지시설 | 시설장 계좌로 지급 |
| 가정위탁 | 위탁부모 계좌로 지급 |

### 5.3 입양 아동

| 대상 | 지급 |
|------|------|
| 입양 아동 | 입양부모 신청 시 지급 |
| 입양 전 | 친생부모 또는 시설장 |

## 6. 변경·정지

### 6.1 변경 신고

| 변경 사항 | 신고 |
|----------|------|
| 주소 변경 | 신고 필요 |
| 계좌 변경 | 신고 필요 |
| 수급자 변경 | 신고 필요 |

### 6.2 지급 정지

| 사유 | 정지 |
|------|------|
| 해외 90일 이상 | 정지 |
| 사망 | 정지 |
| 연령 초과 | 자동 종료 |

## 7. 다른 수당과의 관계

### 7.1 중복 수급 가능

| 수당 | 중복 |
|------|------|
| 양육수당 | ✅ 가능 |
| 아이돌봄서비스 | ✅ 가능 |
| 부모급여 | ✅ 가능 |

### 7.2 연관 제도

| 제도 | 대상 |
|------|------|
| 부모급여 | 0~1세 (2023년~) |
| 양육수당 | 어린이집 미이용 아동 |
| 가정양육수당 | 어린이집 미이용 |

## 8. 부모급여

### 8.1 부모급여 금액 (2026년)

| 연령 | 금액 |
|------|------|
| 0세 | 월 **100만원** |
| 1세 | 월 **50만원** |

### 8.2 아동수당 + 부모급여

| 연령 | 월 총액 |
|------|--------|
| 0세 | 100만원 + 10만원 = **110만원** |
| 1세 | 50만원 + 10만원 = **60만원** |
| 2~7세 | **10만원** |

## 9. 주의사항

### 9.1 신청 기한

| 주의 | 내용 |
|------|------|
| 60일 경과 | 소급 지급 불가 |
| 조기 신청 | 출생신고와 함께 권장 |

### 9.2 계좌 관리

| 주의 | 내용 |
|------|------|
| 아동 명의 계좌 | 가능 |
| 부모 명의 계좌 | 가능 |
| 계좌 변경 | 복지로에서 변경 |
