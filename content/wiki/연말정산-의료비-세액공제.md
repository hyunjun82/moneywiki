# 연말정산 의료비 세액공제

의료비 세액공제는 총급여의 3%를 초과하는 의료비에 대해 세금을 직접 줄여주는 제도입니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리합니다.

## 1. 의료비 세액공제 기본 구조

### 1.1 공제 대상 금액

**총급여의 3% 초과분**부터 공제

| 총급여 | 3% 기준액 | 공제 시작점 |
|--------|-----------|-------------|
| 3,000만원 | 90만원 | 90만원 초과분 |
| 4,000만원 | 120만원 | 120만원 초과분 |
| 5,000만원 | 150만원 | 150만원 초과분 |
| 6,000만원 | 180만원 | 180만원 초과분 |

### 1.2 공제율

| 의료비 유형 | 공제율 |
|-------------|--------|
| 일반 의료비 | **15%** |
| 난임시술비 | **20%** |
| 미숙아·선천성이상아 의료비 | **20%** |

## 2. 공제 한도

### 2.1 한도 없는 항목

다음 대상의 의료비는 **한도 제한 없이** 전액 공제:

| 대상 | 한도 |
|------|------|
| **본인** | 한도 없음 |
| **장애인** | 한도 없음 |
| **65세 이상** | 한도 없음 |
| **6세 이하** | 한도 없음 |
| 건강보험 산정특례자 | 한도 없음 |
| 난임시술비 | 한도 없음 |
| 미숙아·선천성이상아 | 한도 없음 |

### 2.2 한도 있는 항목

| 대상 | 연간 한도 |
|------|-----------|
| 그 외 부양가족 | **700만원** |
| 산후조리원 | **200만원** (출산 1회당) |
| 안경·콘택트렌즈 | **50만원** (1인당) |

## 3. 공제 대상 의료비

### 3.1 공제 가능 항목

| 항목 | 공제 여부 |
|------|----------|
| 병원 진료비 | ✅ 가능 |
| 약국 약제비 | ✅ 가능 |
| 치과 치료비 (보철, 틀니, 스케일링) | ✅ 가능 |
| 한방 치료비 | ✅ 가능 |
| 안경·콘택트렌즈 (시력보정용) | ✅ 가능 (50만원 한도) |
| 보청기 | ✅ 가능 |
| 의료기기 구입·임차비 | ✅ 가능 |
| 산후조리원 | ✅ 가능 (200만원 한도) |
| 장애인활동지원 본인부담금 | ✅ 가능 (2024년~) |

### 3.2 공제 불가 항목

| 항목 | 공제 여부 |
|------|----------|
| 미용·성형 수술비 | ❌ 불가 |
| 건강검진비 (일부 예외) | ❌ 불가 |
| 간병인비 | ❌ 불가 |
| 실손보험 보전금액 | ❌ 불가 |
| 미용 목적 치열교정 | ❌ 불가 |

## 4. 부양가족 의료비

### 4.1 특별 규정

의료비는 부양가족의 **나이·소득 제한 없이** 공제 가능

| 조건 | 공제 가능 여부 |
|------|----------------|
| 60세 미만 부모님 | ✅ 가능 |
| 소득 100만원 초과 부모님 | ✅ 가능 |
| 별거 중인 부모님 (생계 같이) | ✅ 가능 |

### 4.2 주의사항

- 다른 형제가 부모님을 기본공제 대상자로 등록한 경우 → 그 형제가 의료비 공제
- 의료비를 실제 **지출한 본인**만 공제 가능

## 5. 공제 계산 예시

**연봉 5,000만원, 의료비 400만원 지출한 경우**

1. 3% 기준액: 5,000만원 × 3% = 150만원
2. 공제 대상: 400만원 - 150만원 = 250만원
3. 세액공제액: 250만원 × 15% = **37만 5천원**

## 6. 증빙서류

### 6.1 연말정산 간소화

대부분의 의료비는 **간소화 서비스에서 자동 조회**

### 6.2 별도 제출 필요

| 항목 | 제출 서류 |
|------|----------|
| 안경·콘택트렌즈 | 안경점 영수증 |
| 장애인 보장구 | 구입 영수증 |
| 해외 의료비 | 외국 의료기관 영수증 |

## 7. 치과 치료비 공제

### 7.1 공제 가능

- 충치 치료, 신경 치료
- 스케일링
- 보철, 틀니, 임플란트

### 7.2 조건부 공제

- **치열교정**: 저작기능장애 진단서 필요

### 7.3 공제 불가

- 미용 목적 치아 미백
- 미용 목적 라미네이트
