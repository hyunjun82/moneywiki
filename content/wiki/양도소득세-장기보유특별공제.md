# 양도소득세 장기보유특별공제

## 기본 개념
부동산을 장기간 보유하면 양도소득세를 추가로 공제해주는 제도입니다.

## 공제율

### 1세대 1주택 (보유+거주)
| 보유기간 | 공제율 |
|----------|-------|
| 3년 이상 | 24% (12%+12%) |
| 4년 이상 | 32% (16%+16%) |
| 5년 이상 | 40% (20%+20%) |
| 6년 이상 | 48% (24%+24%) |
| 7년 이상 | 56% (28%+28%) |
| 8년 이상 | 64% (32%+32%) |
| 9년 이상 | 72% (36%+36%) |
| 10년 이상 | **80% (40%+40%)** |

- 보유기간 공제 + 거주기간 공제

### 다주택자 (보유만)
| 보유기간 | 공제율 |
|----------|-------|
| 3년 이상 | 6% |
| 4년 이상 | 8% |
| 5년 이상 | 10% |
| 6년 이상 | 12% |
| 7년 이상 | 14% |
| 8년 이상 | 16% |
| 9년 이상 | 18% |
| 10년 이상 | **20%** |

### 조정대상지역 2주택 이상
- 2024년 5월 10일 이후 양도분
- 장기보유특별공제 **배제**

## 계산 방법

### 기본 계산식
```
장기보유특별공제액 = 양도차익 × 공제율
```

### 1세대 1주택 (10년 보유·거주)
```
양도차익: 5억원
보유기간: 10년 → 40%
거주기간: 10년 → 40%
총 공제율: 80%
공제액: 5억 × 80% = 4억원
과세표준: 1억원
```

### 다주택자 (10년 보유)
```
양도차익: 5억원
보유기간: 10년 → 20%
공제액: 5억 × 20% = 1억원
과세표준: 4억원
```

## 보유·거주기간 계산

### 보유기간
- 취득일부터 양도일까지
- 실제 보유한 전체 기간

### 거주기간 (1세대 1주택만)
- 전입신고일부터 전출신고일까지
- 실제 거주 확인 가능
- **최대 인정**: 보유기간까지

### 예시 1: 보유 10년, 거주 5년
```
보유기간 공제: 40%
거주기간 공제: 20% (5년)
총 공제율: 60%
```

### 예시 2: 보유 10년, 거주 12년 (불가능)
```
보유기간 공제: 40%
거주기간 공제: 40% (10년까지만 인정)
총 공제율: 80%
```

## 실무 적용 사례

### 사례 1: 1세대 1주택 (10년 보유·거주)
```
취득가액: 3억원
양도가액: 8억원
양도차익: 5억원
장기보유특별공제: 5억 × 80% = 4억원
과세표준: 1억원
양도세: 약 1,500만원 (세율 15% 적용 시)
```

### 사례 2: 1세대 1주택 (5년 보유·거주)
```
양도차익: 5억원
장기보유특별공제: 5억 × 40% = 2억원
과세표준: 3억원
양도세: 약 6,300만원
```

### 사례 3: 다주택자 (10년 보유)
```
양도차익: 5억원
장기보유특별공제: 5억 × 20% = 1억원
과세표준: 4억원
양도세: 약 1억 1,200만원
```

## 공제 배제 사례

### 조정대상지역 2주택 이상
- 2024년 5월 10일 이후 양도
- 장기보유특별공제 적용 안 됨
- 중과세율 적용

### 단기 보유 (3년 미만)
- 장기보유특별공제 없음
- 기본세율만 적용

### 분양권·조합원입주권
- 장기보유특별공제 적용 안 됨

## 특례

### 상속·증여 주택
- 피상속인·증여자 보유기간 합산 가능
- 거주기간은 본인 거주만 인정

### 농어촌주택
- 9억원 이하 농어촌주택
- 장기보유특별공제 적용 가능

### 일시적 2주택
- 기존 주택 양도 시
- 장기보유특별공제 적용 가능

## 주의사항

### 조정대상지역 확인
- 양도 시점 조정대상지역 여부 확인
- 2주택 이상 시 공제 배제

### 거주 증명
- 전입신고 필수
- 실제 거주 확인 자료 보관
- 세무조사 대비

### 보유기간 증명
- 매매계약서
- 등기부등본
- 취득·양도 시기 명확히

## 절세 전략

### 장기 보유·거주
- 10년 이상 보유·거주로 80% 공제
- 양도세 대폭 절감

### 비과세 vs 장기보유특별공제
- 12억원 이하: 비과세 (양도세 0원)
- 12억원 초과: 초과분에 장기보유특별공제 적용

### 거주기간 확보
- 전입신고 철저히
- 실제 거주 증명 자료 보관
- 거주기간 = 보유기간 만들기

## 계산 도구

### 간단 계산
```
양도차익 × 공제율 = 공제액
양도차익 - 공제액 = 과세표준
```

### 국세청 계산기
- 홈택스 양도소득세 계산
- 자동 공제율 적용
- 세액 시뮬레이션
