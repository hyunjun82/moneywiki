# 연말정산 주택임차차입금 공제

주택임차차입금 원리금상환액 공제는 전세자금대출 상환금에 대해 **40% 소득공제**를 해주는 제도입니다. [국세청 주택자금 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 조건을 정리합니다.

## 1. 공제 기본 조건

[소득세법 제52조](https://www.law.go.kr/법령/소득세법)에 따른 공제 요건입니다.

| 요건 | 조건 |
|------|------|
| 주택 | **무주택 세대주** |
| 주택 규모 | 국민주택규모(85㎡) 이하 |
| 주택 가격 | 기준시가 **4억원 이하** |
| 대출 용도 | 전세보증금 또는 월세보증금 차입 |

## 2. 공제율과 한도

### 2.1 공제 내용

| 항목 | 내용 |
|------|------|
| 공제율 | **40%** (소득공제) |
| 연간 한도 | **400만원** (주택청약 합산) |
| 최대 공제액 | 원리금 1,000만원 상환 시 400만원 |

### 2.2 주택청약과 합산

| 공제 항목 | 개별 한도 | 합산 한도 |
|----------|----------|----------|
| 주택청약저축 | 96만원 | **400만원** |
| 주택임차차입금 | 400만원 | (합산) |

## 3. 대출 요건

### 3.1 대출 유형

| 대출처 | 공제 가능 |
|--------|----------|
| 금융기관 (은행, 보험사) | ✅ 가능 |
| 국민주택기금 | ✅ 가능 |
| 대부업체 | ✅ 가능 (증빙 필요) |
| 개인 차입 | ✅ 가능 (증빙 필요) |

### 3.2 대출 조건

| 요건 | 조건 |
|------|------|
| 차입일 | 입주일 전후 **3개월 이내** |
| 대출금 사용 | 임대차계약서상 보증금 지급에 사용 |
| 임대차계약 | 임대차계약서 작성 필수 |

## 4. 무주택 세대주 요건

### 4.1 무주택 판정

| 기준 | 조건 |
|------|------|
| 기준일 | 12월 31일 |
| 대상 | 세대 전원 무주택 |

### 4.2 세대 구성

| 세대원 | 포함 여부 |
|--------|----------|
| 본인 | ✅ |
| 배우자 | ✅ |
| 동일 주소 직계존비속 | ✅ |

## 5. 공제 계산

### 5.1 계산 구조

| 순서 | 내용 |
|------|------|
| 1단계 | 연간 원리금상환액 확인 |
| 2단계 | 상환액 × 40% 계산 |
| 3단계 | 400만원 한도 적용 |

### 5.2 계산 예시

**전세대출 원리금 1,200만원 상환**

1. 원리금상환액: 1,200만원
2. 공제 대상: 1,200만원 × 40% = 480만원
3. 한도 적용: **400만원** (한도 초과)

## 6. 증빙서류

### 6.1 필요 서류

| 서류 | 용도 |
|------|------|
| 주민등록표등본 | 무주택 세대주 확인 |
| 임대차계약서 사본 | 임차 사실 증명 |
| 금융기관 대출증명서 | 차입 사실 증명 |
| 원리금상환증명서 | 상환액 확인 |

### 6.2 개인 차입 시 추가 서류

| 서류 | 용도 |
|------|------|
| 금전소비대차계약서 | 차입 계약 증명 |
| 이체 증빙 (계좌이체) | 상환 사실 증명 |
| 차입금 사용 증빙 | 보증금 지급 증명 |

## 7. 월세와 구분

### 7.1 월세 세액공제

| 항목 | 월세 세액공제 | 주택임차차입금 |
|------|-------------|----------------|
| 공제 유형 | 세액공제 | 소득공제 |
| 대상 | 월세 납부액 | 전세대출 상환액 |
| 공제율 | 15~17% | 40% |

### 7.2 중복 적용

- 월세 + 전세대출 보증금 있는 경우 → **각각 공제 가능**
- 월세보증금 차입도 주택임차차입금 공제 가능

## 8. 주택청약과 관계

### 8.1 합산 한도

| 구분 | 금액 |
|------|------|
| 주택청약 공제액 | 96만원 |
| 주택임차차입금 공제액 | 304만원 |
| **합계** | **400만원** (한도) |

### 8.2 우선순위

- 주택임차차입금 상환액이 크면 → 임차차입금 우선 적용
- 남은 한도로 주택청약 공제

## 9. 주의사항

### 9.1 입주 시기

- 대출일이 입주일 전후 **3개월** 이내여야 함
- 입주 전 대출 → 입주 후 상환 시작 → **공제 가능**

### 9.2 대출 상환

- **원금 + 이자** 모두 공제 대상
- 중도상환 수수료는 제외

### 9.3 대환대출

- 기존 대출을 갈아탄 경우
- 신규 대출이 요건 충족하면 **계속 공제 가능**

### 9.4 연말정산 간소화

- 금융기관 대출은 **간소화 서비스 조회**
- 개인 차입은 서류 **직접 제출**
