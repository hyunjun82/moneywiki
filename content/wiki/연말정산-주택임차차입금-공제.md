주택임차차입금 원리금상환액 공제는 전세자금대출 상환금에 대해 40% 소득공제를 해주는 제도예요. [국세청 주택자금 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 조건을 정리해 볼게요.

## 공제 기본 조건

[소득세법 제52조](https://www.law.go.kr/법령/소득세법)에 따른 공제 요건이에요.

| 요건 | 조건 |
|------|------|
| 주택 | 무주택 세대주 |
| 주택 규모 | 국민주택규모(85㎡) 이하 |
| 주택 가격 | 기준시가 4억원 이하 |
| 대출 용도 | 전세보증금 또는 월세보증금 차입 |

## 공제율과 한도

공제 내용이에요.

| 항목 | 내용 |
|------|------|
| 공제율 | 40% (소득공제) |
| 연간 한도 | 400만원 (주택청약 합산) |
| 최대 공제액 | 원리금 1,000만원 상환 시 400만원 |

주택청약과 합산 한도예요.

| 공제 항목 | 개별 한도 | 합산 한도 |
|----------|----------|----------|
| 주택청약저축 | 96만원 | 400만원 |
| 주택임차차입금 | 400만원 | (합산) |

## 대출 요건

대출 유형별 공제 가능 여부예요.

| 대출처 | 공제 가능 |
|--------|----------|
| 금융기관 (은행, 보험사) | 가능 |
| 국민주택기금 | 가능 |
| 대부업체 | 가능 (증빙 필요) |
| 개인 차입 | 가능 (증빙 필요) |

대출 조건이에요.

| 요건 | 조건 |
|------|------|
| 차입일 | 입주일 전후 3개월 이내 |
| 대출금 사용 | 임대차계약서상 보증금 지급에 사용 |
| 임대차계약 | 임대차계약서 작성 필수 |

## 무주택 세대주 요건

무주택 판정 기준이에요.

| 기준 | 조건 |
|------|------|
| 기준일 | 12월 31일 |
| 대상 | 세대 전원 무주택 |

세대 구성이에요.

| 세대원 | 포함 여부 |
|--------|----------|
| 본인 | 포함 |
| 배우자 | 포함 |
| 동일 주소 직계존비속 | 포함 |

## 공제 계산

계산 구조예요.

| 순서 | 내용 |
|------|------|
| 1단계 | 연간 원리금상환액 확인 |
| 2단계 | 상환액 × 40% 계산 |
| 3단계 | 400만원 한도 적용 |

계산 예시예요. 전세대출 원리금 1,200만원 상환한 경우예요.

| 항목 | 금액 |
|------|------|
| 원리금상환액 | 1,200만원 |
| 공제 대상 | 1,200만원 × 40% = 480만원 |
| 한도 적용 | 400만원 (한도 초과) |

## 증빙서류

필요 서류예요.

| 서류 | 용도 |
|------|------|
| 주민등록표등본 | 무주택 세대주 확인 |
| 임대차계약서 사본 | 임차 사실 증명 |
| 금융기관 대출증명서 | 차입 사실 증명 |
| 원리금상환증명서 | 상환액 확인 |

개인 차입 시 추가 서류예요.

| 서류 | 용도 |
|------|------|
| 금전소비대차계약서 | 차입 계약 증명 |
| 이체 증빙 (계좌이체) | 상환 사실 증명 |
| 차입금 사용 증빙 | 보증금 지급 증명 |

## 월세와 구분

월세 세액공제와 비교예요.

| 항목 | 월세 세액공제 | 주택임차차입금 |
|------|-------------|----------------|
| 공제 유형 | 세액공제 | 소득공제 |
| 대상 | 월세 납부액 | 전세대출 상환액 |
| 공제율 | 15~17% | 40% |

월세 + 전세대출 보증금이 있는 경우 각각 공제받을 수 있어요. 월세보증금 차입도 주택임차차입금 공제가 가능해요.

## 주택청약과 관계

합산 한도예요.

| 구분 | 금액 |
|------|------|
| 주택청약 공제액 | 96만원 |
| 주택임차차입금 공제액 | 304만원 |
| 합계 | 400만원 (한도) |

주택임차차입금 상환액이 크면 임차차입금 우선 적용되고, 남은 한도로 주택청약 공제가 적용돼요.

## 주의사항

대출일이 입주일 전후 3개월 이내여야 해요. 입주 전 대출 후 입주 후 상환 시작하면 공제받을 수 있어요.

원금과 이자 모두 공제 대상이에요. 중도상환 수수료는 제외돼요.

기존 대출을 갈아탄 대환대출의 경우 신규 대출이 요건 충족하면 계속 공제받을 수 있어요.

금융기관 대출은 간소화 서비스에서 조회되고, 개인 차입은 서류를 직접 제출해야 해요.
