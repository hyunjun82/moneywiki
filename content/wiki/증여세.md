# 증여세

증여세는 타인으로부터 무상으로 재산을 받을 때 부과되는 세금입니다. [상속세 및 증여세법](https://www.law.go.kr/법령/상속세및증여세법)에 따른 내용을 정리합니다.

## 증여세 개요

### 1.1 과세 대상

| 증여 유형 | 과세 |
|----------|------|
| 현금 증여 |  |
| 부동산 증여 |  |
| 주식 증여 |  |
| 채무 면제 |  |
| 저가 양수 |  (시가와 차액) |

### 1.2 납세 의무자

| 구분 | 납세자 |
|------|--------|
| 원칙 | **수증자** (받는 사람) |
| 연대납세 | 증여자 |

## 증여 공제

### 2.1 증여재산 공제 (10년 합산)

| 증여자 | 공제 한도 |
|--------|----------|
| 배우자 | **6억원** |
| 직계존속 (부모) | **5,000만원** |
| 직계비속 (자녀) | **5,000만원** |
| 미성년 자녀 | **2,000만원** |
| 기타 친족 | **1,000만원** |

### 2.2 10년 합산 규정

```
10년 이내 동일인 증여 → 합산하여 공제 적용
```

**예시**: 부모님께 5년 전 3천만원, 올해 4천만원 증여받은 경우
- 10년 합산: 7천만원
- 공제: 5천만원
- 과세표준: 2천만원

## 세율

### 3.1 증여세율표

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1억원 이하 | **10%** | - |
| 1억~5억원 | **20%** | 1,000만원 |
| 5억~10억원 | **30%** | 6,000만원 |
| 10억~30억원 | **40%** | 1억6,000만원 |
| 30억원 초과 | **50%** | 4억6,000만원 |

### 3.2 세대생략 할증

| 수증자 | 할증 |
|--------|------|
| 손자녀 (조부모→손자녀) | 세액의 **30%** 할증 |
| 미성년자가 20억 초과 | 세액의 **40%** 할증 |

## 계산 예시

### 4.1 성인 자녀 5억원 증여

| 항목 | 금액 |
|------|------|
| 증여가액 | 5억원 |
| 증여공제 | -5,000만원 |
| 과세표준 | 4억5,000만원 |
| 세율 | 20% |
| 산출세액 | 4.5억 × 20% - 1,000만원 = 8,000만원 |
| 신고세액공제 (3%) | -240만원 |
| **납부세액** | **7,760만원** |

### 4.2 배우자 10억원 증여

| 항목 | 금액 |
|------|------|
| 증여가액 | 10억원 |
| 증여공제 | -6억원 |
| 과세표준 | 4억원 |
| 세율 | 20% |
| 산출세액 | 4억 × 20% - 1,000만원 = 7,000만원 |
| 신고세액공제 (3%) | -210만원 |
| **납부세액** | **6,790만원** |

### 4.3 손자녀 증여 (세대생략)

**조부모 → 손자녀 2억원**

| 항목 | 금액 |
|------|------|
| 과세표준 | 2억원 - 5,000만원 = 1.5억원 |
| 산출세액 | 2,000만원 |
| 세대생략 할증 | × 1.3 |
| **납부세액** | **약 2,600만원** |

## 신고·납부

### 5.1 신고 기한

| 항목 | 기한 |
|------|------|
| 신고 기한 | 증여일이 속한 달 말일부터 **3개월** |
| 신고세액공제 | **3%** |

### 5.2 납부 방법

| 방법 | 내용 |
|------|------|
| 일시납부 | 전액 한 번에 |
| 분납 | 세액 1천만원 초과 시 2개월 분납 |
| 연부연납 | 세액 2천만원 초과 시 5년 분할 |

### 5.3 신고 방법

| 방법 | 내용 |
|------|------|
| 홈택스 | 온라인 신고 |
| 세무서 | 방문 신고 |

## 부담부증여

### 6.1 개념

```
부담부증여 = 증여 + 채무 인수
```

**예시**: 5억원 아파트 증여 + 대출 2억원 인수

### 6.2 세금 계산

| 구분 | 과세 |
|------|------|
| 증여 부분 (3억원) | **증여세** |
| 채무 인수 (2억원) | **양도소득세** (증여자) |

### 6.3 유불리 판단

| 상황 | 유리한 방식 |
|------|------------|
| 양도차익 적음 | 부담부증여 |
| 양도차익 많음 | 일반증여 |

## 증여 추정

### 7.1 증여 추정 규정

| 상황 | 추정 |
|------|------|
| 재산 취득 자금 출처 불분명 | 증여 추정 |
| 채무 상환 자금 불분명 | 증여 추정 |

### 7.2 자금출처 입증

| 입증 방법 | 예시 |
|----------|------|
| 근로소득 | 원천징수영수증 |
| 사업소득 | 소득금액증명 |
| 대출 | 금융기관 서류 |
| 기존 재산 처분 | 계약서, 입금 내역 |

## 주의사항

### 9.1 차용증 작성
