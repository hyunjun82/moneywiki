# 연말정산 신용카드 공제 한도

신용카드 소득공제는 연말정산에서 가장 많이 활용되는 공제 항목 중 하나입니다. 2026년 기준으로 공제 조건, 한도, 절세 전략을 정리합니다.

## 1. 공제 시작 조건

신용카드 등 소득공제는 **총급여의 25% 초과 사용분**부터 적용됩니다.

| 총급여 | 25% 기준액 | 공제 시작점 |
|--------|------------|-------------|
| 3,000만원 | 750만원 | 750만원 초과분 |
| 4,000만원 | 1,000만원 | 1,000만원 초과분 |
| 5,000만원 | 1,250만원 | 1,250만원 초과분 |
| 6,000만원 | 1,500만원 | 1,500만원 초과분 |

예를 들어 연봉 4,000만원인 사람이 연간 2,000만원을 카드로 사용했다면, 1,000만원(25%)을 초과한 **1,000만원에 대해서만** 공제율이 적용됩니다.

## 2. 결제 수단별 공제율

| 결제 수단 | 공제율 | 비고 |
|-----------|--------|------|
| 신용카드 | **15%** | 가장 낮음 |
| 체크카드 | **30%** | 신용카드의 2배 |
| 현금영수증 | **30%** | 체크카드와 동일 |
| 전통시장 | **40%** | 추가 공제 대상 |
| 대중교통 | **40%** | 추가 공제 대상 |
| 도서·공연·영화 | **30%** | 총급여 7천만원 이하만 |

## 3. 공제 한도

### 기본 공제 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 ~ 1.2억원 이하 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

### 추가 공제 한도

기본 한도를 다 채운 후에도 추가 공제가 가능합니다.

| 항목 | 추가 한도 |
|------|-----------|
| 전통시장 | 100만원 |
| 대중교통 | 100만원 |
| 도서·공연·영화 (7천만원 이하) | 100만원 |

**총급여 7천만원 이하**: 기본 300만원 + 추가 300만원 = **최대 600만원**

## 4. 2025년 7월부터 신설된 항목

### 체육시설 이용료 공제

2025년 7월 1일 이후 결제분부터 **수영장, 헬스장** 이용료도 소득공제 대상입니다.

- **공제율**: 30%
- **대상**: 총급여 7천만원 이하 근로자
- **제외**: PT 강습비, 회원권 양도

## 5. 공제 제외 항목

다음 항목은 신용카드로 결제해도 **공제 대상이 아닙니다**.

- 세금, 공과금 (전기료, 수도료 등)
- 통신비, 인터넷 사용료
- 신차 구매 비용
- 자동차 리스료
- 해외 결제 금액
- 면세점 구매
- 보험료
- 학교 등록금

## 6. 절세 전략

### 25% 기준 전후 전략

```
[25% 미달 구간]
→ 어차피 공제 안됨
→ 혜택 좋은 신용카드(포인트, 캐시백) 사용

[25% 초과 구간]
→ 공제 시작
→ 체크카드·현금영수증으로 전환 (공제율 2배)
```

### 맞벌이 부부 전략

- 소득이 **높은 쪽**에 카드 사용 몰아주기
- 25% 기준 빠르게 넘기고 공제 극대화

### 월별 사용 체크

연말에 몰아서 쓰면 25% 못 넘길 수 있음. **매월 균등하게** 사용하는 것이 유리.
