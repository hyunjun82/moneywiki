# 분양권 전매

분양권 전매는 아파트 분양 당첨권을 제3자에게 양도하는 것입니다. 제한 조건과 세금을 정리합니다.

## 1. 분양권 전매 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 분양권 | 아파트 분양 당첨 권리 |
| 전매 | 제3자에게 양도 |
| 규제 | 전매제한 기간 적용 |

### 1.2 전매 대상

| 대상 | 전매 |
|------|------|
| 분양권 | ✅ (제한 내) |
| 입주권 | ✅ |
| 조합원 지위 | 일부 제한 |

## 2. 전매제한 기간 (2026년)

### 2.1 규제지역별

| 지역 | 전매제한 |
|------|----------|
| 투기과열지구 | **소유권이전등기까지** |
| 조정대상지역 | **3년** |
| 비규제지역 | **6개월~1년** |

### 2.2 주택 유형별

| 유형 | 전매제한 |
|------|----------|
| 공공분양 | 5~10년 |
| 민간분양 | 지역별 상이 |
| 오피스텔 | 제한 없음 (일부 지역 제외) |

### 2.3 수도권 비규제지역

| 주택규모 | 전매제한 |
|----------|----------|
| 전용 85㎡ 이하 | 1년 |
| 전용 85㎡ 초과 | 6개월 |

### 2.4 지방 비규제지역

| 주택규모 | 전매제한 |
|----------|----------|
| 전용 85㎡ 이하 | 6개월 |
| 전용 85㎡ 초과 | 제한 없음 |

## 3. 전매제한 완화

### 3.1 완화 현황 (2024~)

| 지역 | 변경 전 | 변경 후 |
|------|---------|---------|
| 투기과열지구 | 소유권등기까지 | 3년 (일부) |
| 조정지역 | 3년 | 1년 (일부) |
| 비규제 | 6개월~1년 | 단축 |

### 3.2 예외 허용

| 사유 | 전매 허용 |
|------|----------|
| 세대원 사망 | ✅ |
| 해외이주 | ✅ |
| 이혼 | ✅ |
| 채무불이행 | ✅ |
| 근무지 이전 | ✅ |

## 4. 전매 절차

### 4.1 전매 가능 확인

| 단계 | 내용 |
|------|------|
| 1 | 전매제한 기간 확인 |
| 2 | 분양 계약서 확인 |
| 3 | 시행사 전매 승인 |

### 4.2 매매 절차

| 단계 | 내용 |
|------|------|
| 1 | 매수인 확인 |
| 2 | 분양권 양수도 계약 |
| 3 | 시행사에 명의변경 신청 |
| 4 | 잔금·중도금 승계 |
| 5 | 명의변경 완료 |

### 4.3 필요 서류

| 서류 | 용도 |
|------|------|
| 분양권 양수도계약서 | 매매 계약 |
| 분양계약서 | 원계약 확인 |
| 인감증명서 | 양도인 날인 |
| 신분증 | 본인 확인 |

## 5. 세금

### 5.1 양도소득세

| 보유 기간 | 세율 |
|----------|------|
| 1년 미만 | **70%** |
| 1~2년 | **60%** |
| 2년 이상 | **기본세율** (6~45%) |

### 5.2 조정지역 중과

| 지역 | 추가 세율 |
|------|----------|
| 조정대상지역 | **+20%** (2주택) |
| 투기과열지구 | **+30%** (3주택 이상) |

### 5.3 계산 예시

**분양권 프리미엄 5,000만원, 1년 6개월 보유**

| 항목 | 금액 |
|------|------|
| 양도차익 | 5,000만원 |
| 세율 | 60% |
| **양도세** | **3,000만원** |

### 5.4 취득세 (매수인)

| 항목 | 세율 |
|------|------|
| 분양권 취득 | 프리미엄의 **4%** |
| 입주 시 | 취득세 추가 (1~12%) |

## 6. 비용

### 6.1 양도인 비용

| 비용 | 내용 |
|------|------|
| 양도소득세 | 프리미엄에 과세 |
| 중개수수료 | 협의 |

### 6.2 양수인 비용

| 비용 | 내용 |
|------|------|
| 프리미엄 | 매매가 - 분양가 |
| 취득세 | 프리미엄의 4% |
| 중개수수료 | 협의 |
| 잔금·중도금 | 승계 |

### 6.3 비용 예시

**분양가 5억, 프리미엄 1억**

| 양수인 비용 | 금액 |
|------------|------|
| 프리미엄 | 1억원 |
| 취득세 (프리미엄) | 400만원 |
| 잔금 | 분양가 잔액 |
| 입주 시 취득세 | 약 1,000만원 (1주택) |

## 7. 중도금 대출

### 7.1 승계 조건

| 조건 | 내용 |
|------|------|
| 대출 승계 | 매수인 자격 필요 |
| DSR 심사 | 매수인 재심사 |

### 7.2 승계 불가 시

| 상황 | 대응 |
|------|------|
| 대출 불가 | 기존 대출 상환 필요 |
| 자금 준비 | 현금 필요 |

## 8. 위반 시 제재

### 8.1 전매제한 위반

| 제재 | 내용 |
|------|------|
| 계약 해제 | 분양 무효 |
| 형사 처벌 | 3년 이하 징역, 3천만원 이하 벌금 |

### 8.2 불법 전매

| 유형 | 제재 |
|------|------|
| 명의신탁 | 계약 무효 + 형사 처벌 |
| 다운계약서 | 과태료 + 형사 처벌 |

## 9. 투자 주의사항

### 9.1 리스크

| 리스크 | 내용 |
|--------|------|
| 분양가 하락 | 프리미엄 손실 |
| 입주 포기 | 계약금 포기 |
| 대출 불가 | 자금 부족 |
| 세금 부담 | 양도세 60~70% |

### 9.2 확인 사항

| 확인 | 내용 |
|------|------|
| 전매제한 | 기간 확인 |
| 중도금 대출 | 승계 가능 여부 |
| 시세 확인 | 주변 시세 비교 |
| 세금 계산 | 양도세 미리 계산 |

## 10. 자주 묻는 질문

### 10.1 가족 간 전매는?

| 상황 | 가능 |
|------|------|
| 직계존비속 | 전매제한 중에도 가능 |
| 단, 증여세 | 시가 기준 부과 |

### 10.2 법인 분양권은?

| 항목 | 내용 |
|------|------|
| 전매 가능 | 지분 양도 형태 |
| 세금 | 법인세 적용 |
