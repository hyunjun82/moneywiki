# 퇴직연금

퇴직연금은 퇴직금을 체계적으로 적립하고 운용하는 제도입니다. 종류와 특징을 정리합니다.

## 1. 퇴직연금 개요

### 1.1 퇴직연금 종류

| 종류 | 특징 |
|------|------|
| DB형 | 확정급여형 (회사 운용) |
| DC형 | 확정기여형 (근로자 운용) |
| IRP | 개인형퇴직연금 |

### 1.2 도입 의무

| 대상 | 의무 |
|------|------|
| 상시 근로자 1인 이상 | 퇴직급여 제도 의무 |
| 퇴직연금 or 퇴직금 | 선택 가능 |

## 2. DB형 (확정급여형)

### 2.1 특징

| 항목 | 내용 |
|------|------|
| 급여 확정 | 퇴직 시 급여 기준 확정 |
| 운용 주체 | **회사** |
| 운용 리스크 | 회사 부담 |

### 2.2 퇴직급여 계산

```
퇴직급여 = 퇴직 직전 3개월 평균임금 × 근속연수
```

### 2.3 장단점

| 장점 | 단점 |
|------|------|
| 급여 확정 (안정성) | 운용 참여 불가 |
| 리스크 없음 | 회사 파산 시 위험 |

## 3. DC형 (확정기여형)

### 3.1 특징

| 항목 | 내용 |
|------|------|
| 기여금 확정 | 매년 임금의 1/12 이상 |
| 운용 주체 | **근로자** |
| 운용 리스크 | 근로자 부담 |

### 3.2 적립금

```
연간 적립금 = 연봉 × (1/12) 이상
```

### 3.3 장단점

| 장점 | 단점 |
|------|------|
| 운용 자율성 | 손실 가능성 |
| 고수익 가능 | 운용 책임 본인 |
| 이직 시 유리 | 투자 지식 필요 |

### 3.4 운용 상품

| 상품 | 특징 |
|------|------|
| 원리금보장 | 예금, ELB 등 |
| 실적배당 | 펀드, ETF 등 |
| 혼합형 | 원리금+실적배당 |

## 4. IRP (개인형퇴직연금)

### 4.1 가입 대상

| 대상 | 가입 |
|------|------|
| 근로자 | ✅ |
| 자영업자 | ✅ |
| 퇴직자 | ✅ (퇴직금 이전) |
| 공무원 | ✅ |

### 4.2 납입 한도

| 항목 | 한도 |
|------|------|
| 연간 납입 | **1,800만원** |
| 세액공제 한도 | **900만원** (연금저축 포함) |

### 4.3 세액공제

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **16.5%** |
| 5,500만원 초과 | **13.2%** |

### 4.4 세액공제 한도

| 연령 | 한도 |
|------|------|
| 50세 미만 | 900만원 |
| 50세 이상 | 900만원 |

## 5. 수령 방법

### 5.1 수령 시기

| 조건 | 수령 |
|------|------|
| 55세 이상 | 연금 수령 가능 |
| 퇴직 시 | 일시금 또는 IRP 이전 |

### 5.2 수령 방식

| 방식 | 세금 |
|------|------|
| 연금 수령 | 연금소득세 (3.3~5.5%) |
| 일시금 수령 | 퇴직소득세 |

### 5.3 연금 수령 조건

| 조건 | 내용 |
|------|------|
| 최소 기간 | **10년 이상** |
| 수령 개시 | **55세 이후** |
| 연간 한도 | 연금계좌 평가액/11-연금수령연차 |

## 6. 세금 비교

### 6.1 수령 시 세금

| 구분 | 연금 수령 | 일시금 수령 |
|------|----------|------------|
| 세율 | 3.3~5.5% | 퇴직소득세 |
| 분리과세 | ✅ | ✅ |

### 6.2 연금소득세

| 수령 나이 | 세율 |
|----------|------|
| 55~69세 | **5.5%** |
| 70~79세 | **4.4%** |
| 80세 이상 | **3.3%** |

### 6.3 중도 해지 시

| 상황 | 세금 |
|------|------|
| 중도 해지 | 기타소득세 **16.5%** |
| 부득이한 사유 | 연금소득세 적용 |

## 7. 퇴직 시 처리

### 7.1 퇴직금 처리 옵션

| 옵션 | 내용 |
|------|------|
| IRP 이전 | 세금 이연 |
| 일시금 수령 | 퇴직소득세 납부 |

### 7.2 IRP 이전 장점

| 장점 | 내용 |
|------|------|
| 과세 이연 | 수령 시까지 세금 유예 |
| 세율 우대 | 연금 수령 시 저세율 |
| 추가 납입 | 세액공제 혜택 |

## 8. 운용 전략

### 8.1 연령별 전략

| 연령 | 전략 |
|------|------|
| 20~30대 | 공격적 운용 (주식 비중↑) |
| 40대 | 균형 운용 |
| 50대~ | 안정적 운용 (채권 비중↑) |

### 8.2 운용 상품 비중

| 상품 | 젊은 층 | 중년 | 은퇴 근접 |
|------|---------|------|----------|
| 주식형 | 70% | 50% | 30% |
| 채권형 | 20% | 30% | 40% |
| 원리금보장 | 10% | 20% | 30% |

## 9. 주의사항

### 9.1 중도 인출 제한

| 사유 | 인출 |
|------|------|
| 무주택자 주택 구입 | ✅ |
| 6개월 이상 요양 | ✅ |
| 파산·개인회생 | ✅ |
| 천재지변 | ✅ |
| 그 외 | ❌ |

### 9.2 유의점

| 주의 | 내용 |
|------|------|
| 운용 손실 | DC형·IRP 본인 책임 |
| 중도 해지 | 불이익 세율 적용 |
| 연금 수령 | 10년 이상 필수 |
