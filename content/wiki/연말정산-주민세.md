지방소득세는 소득세에 부가되는 지방세예요. [지방세법 제89조](https://www.law.go.kr/법령/지방세법)에 따른 내용을 정리해 볼게요.

## 지방소득세 개요

기본 정보예요.

| 항목 | 내용 |
|------|------|
| 정식 명칭 | 지방소득세 |
| 과거 명칭 | 주민세 (소득분) |
| 세율 | 소득세의 10% |
| 납부처 | 지방자치단체 |

세금 구조예요.

| 항목 | 비율 |
|------|------|
| 소득세 | 100% |
| 지방소득세 | 10% |
| 실제 납부 세금 | 110% |

## 계산 방법

기본 계산이에요.

| 항목 | 계산 |
|------|------|
| 소득세 결정세액 | 100만원 |
| 지방소득세 | 100만원 × 10% = 10만원 |
| 총 세금 | 110만원 |

환급 계산이에요.

| 항목 | 계산 |
|------|------|
| 소득세 환급 | 50만원 |
| 지방소득세 환급 | 5만원 |
| 총 환급 | 55만원 |

추가납부 계산이에요.

| 항목 | 계산 |
|------|------|
| 소득세 추가납부 | 30만원 |
| 지방소득세 추가납부 | 3만원 |
| 총 추가납부 | 33만원 |

## 원천징수

매월 원천징수 방식이에요.

| 항목 | 내용 |
|------|------|
| 소득세 | 간이세액표 적용 |
| 지방소득세 | 소득세의 10% |
| 공제 | 급여에서 함께 공제 |

급여명세서 예시예요.

| 항목 | 예시 |
|------|------|
| 소득세 | 20만원 |
| 지방소득세 | 2만원 |
| 총 공제 | 22만원 |

## 연말정산 반영

정산 절차예요.

| 단계 | 내용 |
|------|------|
| 1 | 소득세 결정세액 확정 |
| 2 | 지방소득세 = 소득세 × 10% |
| 3 | 기납부 지방소득세 비교 |
| 4 | 환급 또는 추가납부 |

환급 시점이에요.

| 항목 | 시점 |
|------|------|
| 소득세 환급 | 3월 급여 |
| 지방소득세 환급 | 3월 급여 (함께) |

## 세액공제와 지방소득세

세액공제 적용 방식이에요.

| 공제 | 적용 |
|------|------|
| 자녀세액공제 | 소득세에서 공제 |
| 지방소득세 | 공제 후 소득세의 10% |

계산 예시예요. 세액공제 적용 전후예요.

| 항목 | 금액 |
|------|------|
| 산출세액 | 200만원 |
| 세액공제 | 50만원 |
| 결정세액 (소득세) | 150만원 |
| 지방소득세 | 15만원 |
| 총 세금 | 165만원 |

## 특수 상황

이직자의 경우예요.

| 상황 | 처리 |
|------|------|
| 전 직장 | 기납부 지방소득세 합산 |
| 새 직장 | 연말정산 시 정산 |

퇴사자의 경우예요.

| 상황 | 처리 |
|------|------|
| 중도 퇴사 | 퇴사 시 정산 |
| 미취업 | 5월 종소세 신고 |

프리랜서의 경우예요.

| 상황 | 처리 |
|------|------|
| 3.3% 원천징수 | 소득세 3% + 지방소득세 0.3% |
| 종합소득세 신고 | 정산 후 환급/납부 |

## 지방소득세 활용

주민등록지 기준이에요.

| 항목 | 내용 |
|------|------|
| 납부 지역 | 주민등록상 주소지 |
| 사용처 | 해당 지자체 재원 |

지역별 차이가 없어요.

| 항목 | 내용 |
|------|------|
| 세율 | 전국 동일 10% |
| 차이 | 없음 |

## 주의사항

소득세 신고 시 자동 신고되므로 별도 신고가 필요 없어요.

원천징수영수증에서 기납부 소득세와 기납부 지방소득세를 확인하세요.

소득세와 지방소득세는 함께 환급되고 동일 계좌로 입금돼요.

## 세금 계산 예시

연봉 5,000만원, 결정세액 150만원인 경우예요.

| 항목 | 금액 |
|------|------|
| 소득세 결정세액 | 150만원 |
| 지방소득세 | 15만원 |
| 기납부 소득세 | 180만원 |
| 기납부 지방소득세 | 18만원 |
| 소득세 환급 | 30만원 |
| 지방소득세 환급 | 3만원 |
| 총 환급 | 33만원 |
