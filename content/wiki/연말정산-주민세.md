# 연말정산 주민세 (지방소득세)

지방소득세는 소득세에 부가되는 지방세입니다. [지방세법 제89조](https://www.law.go.kr/법령/지방세법)에 따른 내용을 정리합니다.

## 1. 지방소득세 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 정식 명칭 | **지방소득세** |
| 과거 명칭 | 주민세 (소득분) |
| 세율 | 소득세의 **10%** |
| 납부처 | 지방자치단체 |

### 1.2 세금 구조

```
소득세 + 지방소득세 = 실제 납부 세금
   100%   +   10%    =    110%
```

## 2. 계산 방법

### 2.1 기본 계산

| 항목 | 계산 |
|------|------|
| 소득세 결정세액 | 100만원 |
| 지방소득세 | 100만원 × 10% = **10만원** |
| 총 세금 | 110만원 |

### 2.2 환급 계산

| 항목 | 계산 |
|------|------|
| 소득세 환급 | 50만원 |
| 지방소득세 환급 | 5만원 |
| 총 환급 | **55만원** |

### 2.3 추가납부 계산

| 항목 | 계산 |
|------|------|
| 소득세 추가납부 | 30만원 |
| 지방소득세 추가납부 | 3만원 |
| 총 추가납부 | **33만원** |

## 3. 원천징수

### 3.1 매월 원천징수

| 항목 | 내용 |
|------|------|
| 소득세 | 간이세액표 적용 |
| 지방소득세 | 소득세의 10% |
| 공제 | 급여에서 함께 공제 |

### 3.2 급여명세서

| 항목 | 예시 |
|------|------|
| 소득세 | 20만원 |
| 지방소득세 | 2만원 |
| 총 공제 | 22만원 |

## 4. 연말정산 반영

### 4.1 정산 절차

| 단계 | 내용 |
|------|------|
| 1 | 소득세 결정세액 확정 |
| 2 | 지방소득세 = 소득세 × 10% |
| 3 | 기납부 지방소득세 비교 |
| 4 | 환급 또는 추가납부 |

### 4.2 환급 시점

| 항목 | 시점 |
|------|------|
| 소득세 환급 | 3월 급여 |
| 지방소득세 환급 | 3월 급여 (함께) |

## 5. 세액공제와 지방소득세

### 5.1 세액공제 적용

| 공제 | 적용 |
|------|------|
| 자녀세액공제 | 소득세에서 공제 |
| 지방소득세 | 공제 후 소득세의 10% |

### 5.2 계산 예시

**세액공제 적용 전후**

| 항목 | 금액 |
|------|------|
| 산출세액 | 200만원 |
| 세액공제 | 50만원 |
| 결정세액 (소득세) | 150만원 |
| 지방소득세 | 15만원 |
| **총 세금** | **165만원** |

## 6. 특수 상황

### 6.1 이직자

| 상황 | 처리 |
|------|------|
| 전 직장 | 기납부 지방소득세 합산 |
| 새 직장 | 연말정산 시 정산 |

### 6.2 퇴사자

| 상황 | 처리 |
|------|------|
| 중도 퇴사 | 퇴사 시 정산 |
| 미취업 | 5월 종소세 신고 |

### 6.3 프리랜서

| 상황 | 처리 |
|------|------|
| 3.3% 원천징수 | 소득세 3% + 지방소득세 0.3% |
| 종합소득세 신고 | 정산 후 환급/납부 |

## 7. 지방소득세 활용

### 7.1 주민등록지 기준

| 항목 | 내용 |
|------|------|
| 납부 지역 | 주민등록상 주소지 |
| 사용처 | 해당 지자체 재원 |

### 7.2 지자체별 동일

| 항목 | 내용 |
|------|------|
| 세율 | 전국 동일 10% |
| 차이 | 없음 |

## 8. 주의사항

### 8.1 별도 신고 불필요

- 소득세 신고 시 **자동 신고**
- 별도 절차 없음

### 8.2 원천징수영수증 확인

| 항목 | 확인 |
|------|------|
| 기납부 소득세 | 확인 필요 |
| 기납부 지방소득세 | 확인 필요 |

### 8.3 환급 계좌

- 소득세와 지방소득세 **함께 환급**
- 동일 계좌로 입금

## 9. 세금 계산 예시

### 9.1 연말정산 결과

**연봉 5,000만원, 결정세액 150만원**

| 항목 | 금액 |
|------|------|
| 소득세 결정세액 | 150만원 |
| 지방소득세 | 15만원 |
| 기납부 소득세 | 180만원 |
| 기납부 지방소득세 | 18만원 |
| 소득세 환급 | 30만원 |
| 지방소득세 환급 | 3만원 |
| **총 환급** | **33만원** |
