# 연말정산 산후조리원

산후조리원 비용 의료비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 출산 1회당 **200만원** |

## 2. 공제 조건

| 요건 | 기준 |
|------|------|
| 총급여 | **7천만원 이하** |
| 이용자 | 산모 본인 |

## 3. 공제 예시

| 산후조리원 비용 | 한도 적용 | 공제액(15%) |
|----------------|----------|-------------|
| 150만원 | 150만원 | 22.5만원 |
| 300만원 | 200만원 | 30만원 |

## 4. 필요서류

| 서류 | 발급처 |
|------|--------|
| 산후조리원 영수증 | 산후조리원 |
| 간소화 자료 | 자동 반영 |
