# 연말정산 건강검진

건강검진 비용 의료비 공제를 정리합니다.

## 1. 공제 여부

| 검진 유형 | 공제 |
|----------|------|
| 종합검진 | ✅ |
| 정밀검진 | ✅ |
| 국가건강검진(무료) | ❌ |

## 2. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 대상자별 상이 |

## 3. 공제 한도

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 | **한도 없음** |
| 65세 이상 | **한도 없음** |
| 그 외 | 700만원 |

## 4. 공제 예시

| 검진비 | 대상 | 공제액(15%) |
|--------|------|-------------|
| 50만원 | 본인 | 7.5만원 |
| 100만원 | 부모 | 15만원 |

## 5. 필요서류

| 서류 | 발급처 |
|------|--------|
| 의료비 영수증 | 검진센터 |
| 간소화 자료 | 자동 반영 |
