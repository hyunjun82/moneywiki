# 연말정산 표준세액공제

표준세액공제는 특별소득공제와 특별세액공제를 신청하지 않는 근로자를 위한 공제입니다. [소득세법 제59조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 공제 금액 | **13만원** |
| 공제 유형 | 세액공제 |
| 적용 조건 | 특별공제 미신청 시 |

### 1.2 표준세액공제 적용 상황

| 상황 | 적용 |
|------|------|
| 소득공제신고서 미제출 | ✅ 자동 적용 |
| 특별공제 포기 선택 | ✅ 적용 |
| 특별공제 신청 | ❌ 미적용 |

## 2. 표준세액공제와 특별공제 비교

### 2.1 특별소득공제 항목

표준세액공제 선택 시 **포기하는** 소득공제:

| 항목 | 내용 |
|------|------|
| 보험료 공제 | 건강보험, 고용보험 등 |
| 주택자금 공제 | 주택임차차입금, 장기주택저당차입금 |

### 2.2 특별세액공제 항목

표준세액공제 선택 시 **포기하는** 세액공제:

| 항목 | 공제율 |
|------|--------|
| 보장성보험료 | 12% |
| 의료비 | 15% |
| 교육비 | 15% |
| 기부금 | 15~25% |
| 월세 | 15~17% |

### 2.3 유지되는 공제

표준세액공제를 선택해도 **유지되는** 공제:

| 공제 | 유지 여부 |
|------|----------|
| 인적공제 (기본공제, 추가공제) | ✅ 유지 |
| 연금보험료 공제 (국민연금) | ✅ 유지 |
| 근로소득공제 | ✅ 유지 |
| 연금저축 세액공제 | ✅ 유지 |
| 자녀 세액공제 | ✅ 유지 |
| 신용카드 소득공제 | ✅ 유지 |

## 3. 표준세액공제 선택 판단

### 3.1 표준세액공제가 유리한 경우

| 상황 | 이유 |
|------|------|
| 특별공제 합계 13만원 미만 | 표준 13만원이 더 큼 |
| 1인 가구, 공제 항목 없음 | 서류 준비 번거로움 회피 |
| 증빙서류 준비 불가 | 차선책으로 선택 |

### 3.2 특별공제가 유리한 경우 (대부분)

| 항목 | 예시 금액 |
|------|----------|
| 보장성보험료 (100만원 한도) | 12만원 |
| 의료비 | 가변 |
| 교육비 | 가변 |
| 기부금 | 가변 |

**대부분의 근로자**: 특별공제 합계 > 13만원

## 4. 계산 예시

### 4.1 표준세액공제 선택

**특별공제 항목이 거의 없는 경우**

| 항목 | 금액 |
|------|------|
| 보장성보험 | 50만원 납입 → 6만원 공제 |
| 의료비 | 3% 미달 → 0원 |
| 교육비 | 없음 → 0원 |
| 기부금 | 없음 → 0원 |
| **특별공제 합계** | **6만원** |
| **표준세액공제** | **13만원** ← 선택 |

### 4.2 특별공제 선택 (일반적)

**일반적인 근로자**

| 항목 | 공제액 |
|------|--------|
| 보장성보험 | 12만원 |
| 의료비 | 30만원 |
| 교육비 | 45만원 |
| 기부금 | 10만원 |
| **특별공제 합계** | **97만원** ← 선택 |

## 5. 신청 방법

### 5.1 표준세액공제 선택

| 방법 | 절차 |
|------|------|
| 소득공제신고서 미제출 | 자동 적용 |
| 신고서에 표기 | "표준세액공제 적용" 체크 |

### 5.2 특별공제 선택 (권장)

| 방법 | 절차 |
|------|------|
| 간소화 자료 제출 | 공제 항목 자동 반영 |
| 증빙서류 제출 | 추가 공제 신청 |

## 6. 주의사항

### 6.1 선택의 영향

| 선택 | 영향 |
|------|------|
| 표준세액공제 | 특별공제 전부 포기 |
| 부분 선택 불가 | 전부 또는 전무 |

### 6.2 경정청구 가능

| 상황 | 대응 |
|------|------|
| 표준세액공제 선택 후 후회 | 경정청구로 변경 가능 |
| 기한 | 법정신고기한 5년 이내 |

### 6.3 일용근로자

| 대상 | 적용 |
|------|------|
| 일용근로자 | 표준세액공제 자동 적용 |
| 일반 근로자 | 선택 가능 |

## 7. 표준세액공제 13만원의 의미

### 7.1 산정 기준

특별공제를 받지 않는 근로자의 **최소한의 공제**:
- 과거 근로소득세액공제와 통합
- 2014년 세법 개정으로 현행 체계

### 7.2 물가 반영 없음

- 13만원은 **고정 금액**
- 물가 상승 반영 없음 (2014년 이후 동일)

## 8. 실무 팁

### 8.1 대부분은 특별공제 선택

- 보장성보험 하나만 있어도 비슷
- 의료비, 교육비 있으면 확실히 유리

### 8.2 간소화 서비스 활용

- 홈택스에서 간편하게 자료 조회
- 회사에 PDF 제출로 간단히 처리
