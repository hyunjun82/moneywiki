도서·공연·영화비는 일반 신용카드보다 높은 30% 공제율이 적용돼요. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따르면 총급여 7천만원 이하 근로자만 대상이에요.

## 공제 대상자

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 소득 요건이에요.

| 조건 | 기준 |
|------|------|
| 총급여 | 7천만원 이하 |
| 종합소득금액 | 6천만원 이하 |

총급여 7천만원을 넘으면 도서·공연·영화 공제 대상에서 빠져요. 일반 신용카드 공제(15%)로만 처리돼요.

## 공제율과 한도

공제율을 비교해 볼게요.

| 항목 | 공제율 | 비교 |
|------|--------|------|
| 도서·공연·영화 | 30% | 신용카드의 2배 |
| 체크카드·현금영수증 | 30% | 동일 |
| 신용카드 | 15% | 기본 |

추가한도도 있어요.

| 구분 | 한도 |
|------|------|
| 기본 한도 | 300만원 (신용카드 전체) |
| 도서·공연·영화 추가 한도 | 100만원 |

기본한도 300만원을 다 채워도 도서·공연·영화에서 100만원 추가 공제받을 수 있어요. 전통시장 100만원, 대중교통 100만원과 합치면 최대 600만원까지 공제 가능해요.

## 공제 대상 항목

도서 구입 항목이에요.

| 항목 | 공제 여부 |
|------|----------|
| 종이책 구입 | 가능 |
| 전자책 구입 | 가능 |
| 오디오북 구입 | 가능 |
| 온라인 서점 구매 | 가능 |
| 중고 서적 | 가능 (사업자 등록 서점) |
| 신문·잡지 구독료 | 불가 |
| 학습지 | 불가 |

공연 관람 항목이에요.

| 항목 | 공제 여부 |
|------|----------|
| 뮤지컬 | 가능 |
| 연극 | 가능 |
| 콘서트 | 가능 |
| 오페라·발레 | 가능 |
| 클래식 음악회 | 가능 |
| 전시회 (박물관, 미술관) | 가능 |
| 스포츠 경기 | 불가 |

영화 관람 항목이에요.

| 항목 | 공제 여부 | 시행일 |
|------|----------|--------|
| 영화관 관람 | 가능 | 2023년 7월부터 |
| OTT 구독료 | 불가 | |
| VOD 개별 결제 | 불가 | |

## 사용처 확인

도서·공연·영화로 인정받으려면 문화비 소득공제 가맹점에서 결제해야 해요. [문화비 소득공제 홈페이지](https://www.culture.go.kr/deduction)에서 가맹점을 검색할 수 있어요.

연말정산 간소화 서비스에서는 1월 15일부터 도서·공연·영화 사용금액이 자동 구분돼요.

## 공제 계산 예시

연봉 5,000만원이고 도서·공연 200만원 사용한 경우예요.

25% 기준액은 1,250만원이에요. 전체 카드 사용액이 1,500만원이라면 공제 대상은 250만원(1,500만원 - 1,250만원)이에요. 도서·공연 200만원 × 30% = 60만원 소득공제예요.

기본 한도를 다 채운 경우도 볼게요. 신용카드 공제로 기본 한도 300만원을 채웠어도 도서·공연 200만원을 추가로 쓰면 200만원 × 30% = 60만원을 추가 공제받을 수 있어요. 총 360만원 공제가 가능해요.

## 절세 전략

같은 금액 사용 시 공제 효과를 비교해 볼게요.

| 사용 항목 | 100만원 사용 시 공제액 |
|----------|----------------------|
| 도서·공연·영화 | 30만원 |
| 신용카드 | 15만원 |
| 차이 | 15만원 |

기본 한도(300만원)를 신용카드로 채운 후에는 도서·공연·영화에서 추가로 쓰면 별도 100만원 한도가 적용돼요. 결제 전에 문화비 소득공제 가맹점인지 확인하세요.

## 주의사항

총급여 7천만원을 초과하면 도서·공연·영화 공제 대상에서 빠져요. 전통시장, 대중교통 추가 한도는 소득 제한이 없어요.

영화 관람료는 2023년 7월 1일 이후 사용분부터 적용돼요. 도서·공연은 2018년 7월 1일 이후 사용분부터 적용됐어요.
