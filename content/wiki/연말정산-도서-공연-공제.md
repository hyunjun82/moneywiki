# 연말정산 도서 공연 공제

도서·공연·영화 사용금액은 일반 신용카드보다 높은 **30% 공제율**이 적용됩니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 조건을 정리합니다.

## 1. 공제 대상자

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 소득 요건입니다.

| 조건 | 기준 |
|------|------|
| 총급여 | **7,000만원 이하** |
| 종합소득금액 | **6,000만원 이하** |

**주의**: 총급여 7,000만원 초과 시 도서·공연·영화 공제 **대상 제외**

## 2. 공제율과 한도

### 2.1 공제율

| 항목 | 공제율 | 비교 |
|------|--------|------|
| 도서·공연·영화 | **30%** | 신용카드(15%)의 2배 |
| 체크카드·현금영수증 | 30% | 동일 |
| 신용카드 | 15% | 기본 |

### 2.2 추가 한도

| 구분 | 한도 |
|------|------|
| 기본 한도 | 300만원 (신용카드 전체) |
| **도서·공연·영화 추가 한도** | **100만원** |

**최대 공제**: 기본 300만원 + 전통시장 100만원 + 대중교통 100만원 + **도서·공연·영화 100만원** = **600만원**

## 3. 공제 대상 항목

### 3.1 도서 구입

| 항목 | 공제 여부 |
|------|----------|
| 종이책 구입 | ✅ 가능 |
| 전자책 구입 | ✅ 가능 |
| 오디오북 구입 | ✅ 가능 |
| 온라인 서점 구매 | ✅ 가능 |
| 중고 서적 | ✅ 가능 (사업자 등록 서점) |
| 신문·잡지 구독료 | ❌ 불가 |
| 학습지 | ❌ 불가 |

### 3.2 공연 관람

| 항목 | 공제 여부 |
|------|----------|
| 뮤지컬 | ✅ 가능 |
| 연극 | ✅ 가능 |
| 콘서트 | ✅ 가능 |
| 오페라·발레 | ✅ 가능 |
| 클래식 음악회 | ✅ 가능 |
| 전시회 (박물관, 미술관) | ✅ 가능 |
| 스포츠 경기 | ❌ 불가 |

### 3.3 영화 관람

| 항목 | 공제 여부 | 시행일 |
|------|----------|--------|
| 영화관 관람 | ✅ 가능 | 2023년 7월~ |
| OTT 구독료 (넷플릭스 등) | ❌ 불가 | |
| VOD 개별 결제 | ❌ 불가 | |

## 4. 사용처 확인

### 4.1 확인 방법

도서·공연·영화로 인정받으려면 **문화비 소득공제 가맹점**에서 결제해야 합니다.

1. [문화비 소득공제 홈페이지](https://www.culture.go.kr/deduction) 접속
2. 가맹점 검색
3. 해당 사업장 등록 여부 확인

### 4.2 연말정산 간소화

1월 15일부터 간소화 서비스에서 **도서·공연·영화 사용금액 자동 구분**

## 5. 공제 계산 예시

**연봉 5,000만원, 도서·공연 200만원 사용한 경우**

1. 25% 기준액: 1,250만원
2. 전체 카드 사용액이 1,500만원이라면:
   - 공제 대상: 250만원 (1,500만원 - 1,250만원)
3. 도서·공연 200만원 × 30% = **60만원 소득공제**

**기본 한도를 다 채운 경우**

- 신용카드 공제로 기본 한도 300만원 채움
- 도서·공연 200만원 추가 사용
- 추가 공제: 200만원 × 30% = 60만원 (추가 한도 100만원 이내)
- **총 360만원 공제**

## 6. 절세 전략

### 6.1 문화비 적극 활용

같은 금액 사용 시 공제 효과:
- 도서·공연·영화: 100만원 × 30% = **30만원**
- 신용카드: 100만원 × 15% = 15만원
- **차이: 15만원**

### 6.2 추가 한도 활용

기본 한도(300만원)를 신용카드로 채운 후:
- 도서·공연·영화에서 추가 사용 → **별도 100만원 한도** 적용

### 6.3 가맹점 확인

- 결제 전 **문화비 소득공제 가맹점** 여부 확인
- 가맹점 아닌 곳은 일반 신용카드 공제(15%)만 적용

## 7. 주의사항

### 7.1 소득 제한

- 총급여 **7,000만원 초과** 시 도서·공연·영화 공제 대상 제외
- 전통시장, 대중교통 추가 한도는 소득 제한 없음

### 7.2 적용 시점

- **영화 관람료**: 2023년 7월 1일 이후 사용분부터 적용
- **도서·공연**: 2018년 7월 1일 이후 사용분부터 적용
