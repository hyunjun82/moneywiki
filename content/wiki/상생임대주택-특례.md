"집은 임대 주고 있는데, 비과세 받으려면 내가 직접 살아야 한대요. 방법 없을까요?"

조정대상지역 주택이라 2년 거주해야 비과세인데, 현실적으로 어려운 분들 많으시죠. **상생임대주택 특례**를 활용하면 거주 안 해도 비과세 받을 수 있어요. 요건이 좀 까다롭긴 한데, 차근차근 설명드릴게요.

## 핵심 요약

| 항목 | 내용 |
|------|------|
| 특례 혜택 | 거주요건 **면제** |
| 직전임대차계약 | **1년6개월** 이상 |
| 상생임대차계약 | **2년** 이상, 임대료 5% 이내 인상 |
| 계약 체결 기간 | '21.12.20 ~ '26.12.31 |

## 상생임대주택 특례란

## 상생임대주택 # 제도 개

조정대상지역 주택은 2년 이상 **거주**해야 비과세인데요.

상생임대주택 특례를 적용하면 **거주하지 않아도** 비과세 받을 수 있어요.

### 특례 혜택

| 혜택 | 적용 조건 |
|------|----------|
| 1세대1주택 비과세 | 거주요건 **면제** |
| 장기보유특별공제 | 거주요건 **면제** |
| 거주주택 비과세 | 거주요건 **면제** |

## 특례 요건 4가지

### 요건 총정리

| 요건 | 내용 |
|------|------|
| **직전계약** | 주택 취득 후 임대차계약 체결 |
| **상생계약** | '21.12.20~'26.12.31 중 체결 |
| **임대기간** | 직전 1년6개월 이상 + 상생 2년 이상 |
| **임대료** | 5% 이내 인상 |

### 요건 1: 직전임대차계약

| 항목 | 내용 |
|------|------|
| 시점 | 주택 **취득 후** 체결 |
| 기간 | **1년6개월** 이상 |

**주의**: 분양권 상태에서 체결한 계약은 직전임대차계약이 **아니에요**.

### 요건 2: 상생임대차계약

| 항목 | 내용 |
|------|------|
| 체결 기간 | **'21.12.20 ~ '26.12.31** |
| 임대료 인상 | 직전 대비 **5% 이내** |
| 임대기간 | **2년** 이상 |

### 요건 3: 임대기간

```
직전임대차계약: 1년6개월 이상
     +
상생임대차계약: 2년 이상
     =
총 3년6개월 이상 임대
```

### 요건 4: 임대료 인상 제한

```
상생임대차계약 임대료 ≤ 직전임대차계약 임대료 × 105%
```

## 흔한 실수 사례

### 사례 1: 4년 단일 계약

#### 상황

이대한씨는 4년짜리 임대차계약을 1회 체결했어요.

```
'21.6월           '21.12월          '25.12월        '26.1월
A주택 취득        임대차 계약 시작    임대차 계약 종료   A주택 양도
(조정대상지역)     (4년 단일 계약)
```

이대한씨는 이렇게 생각했어요.

> "4년이면 직전 1년6개월 + 상생 2년보다 긴데, 당연히 특례 받겠지?"

**결과: 특례 적용 불가!**

## 상생임대주택 ## 왜 안 될

**하나의 임대차계약을 임의로 2개로 나눌 수 없어요.**

| 필요한 것 | 이대한씨 상황 |
|----------|-------------|
| 직전계약 **별도** 체결 | **없음** |
| 상생계약 **별도** 체결 | **없음** |
| 2개의 **별도 계약** | 1개의 계약만 있음 |

#### 올바른 방법

```
1차 계약: 직전임대차계약 (1년6개월 이상)
    ↓
계약 종료 후
    ↓
2차 계약: 상생임대차계약 (2년 이상, 5% 이내 인상)
```

**반드시 2개의 별도 계약이 필요해요.**

### 사례 2: 다른 임차인과 기간 합산

#### 상황

```
1차 계약: 임차인 A와 1년 (임차인 요청으로 단기)
2차 계약: 임차인 B와 6개월
→ 합산 1년6개월
```

이렇게 해서 직전임대차계약 1년6개월 충족했다고 볼 수 있을까요?

**결과: 직전임대차계약으로 인정 불가!**

## 상생임대주택 ## 왜 안 될

서로 다른 임차인과 순차적으로 체결한 계약은 **합산 불가**예요.

| 가능 | 불가 |
|------|------|
| **같은** 임차인과 연장 | **다른** 임차인과 순차 체결 |

## 올바른 특례 적용 방법

## 상생임대주택 # 타임라

```
1. 주택 취득 (조정대상지역)
   ↓
2. 직전임대차계약 체결 (1년6개월 이상)
   ↓
3. 직전계약 종료 후 상생임대차계약 체결
   - 같은 임차인
   - 임대료 5% 이내 인상
   - 2년 이상
   ↓
4. 상생계약 종료 후 양도
   ↓
5. 비과세 또는 장기보유특별공제 적용
```

### 체크리스트

```
□ 주택 취득 후 첫 임대차계약인가요? (직전계약)
□ 직전계약 기간이 1년6개월 이상인가요?
□ 상생계약을 '21.12.20~'26.12.31 사이에 체결했나요?
□ 상생계약 임대료가 5% 이내 인상인가요?
□ 상생계약 기간이 2년 이상인가요?
□ 다가구주택이면 모든 호가 요건 충족했나요?
□ 별도의 2개 계약인가요? (단일 계약 불가)
```

## 다가구주택 특례 적용

다가구주택은 **모든 호가 요건을 충족해야** 전체 양도 시 특례를 받을 수 있어요.

자세한 내용은 [다가구주택 상생임대주택](/w/다가구주택-상생임대주택) 문서를 참고하세요.
