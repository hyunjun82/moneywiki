# 재산세

재산세는 부동산 소유에 대해 매년 부과되는 지방세입니다. [지방세법](https://www.law.go.kr/법령/지방세법)에 따른 내용을 정리합니다.

## 재산세 개요

### 1.1 과세 대상

| 대상 | 과세 기준일 |
|------|------------|
| 주택 | 매년 **6월 1일** |
| 토지 | 매년 **6월 1일** |
| 건물 | 매년 **6월 1일** |

### 1.2 납세 의무자

| 기준 | 납세자 |
|------|--------|
| 6월 1일 현재 | **소유자** |
| 공유 부동산 | 지분별 납부 |

## 주택 재산세

### 2.1 세율

| 과세표준 | 세율 |
|----------|------|
| 6,000만원 이하 | **0.1%** |
| 6,000만~1.5억원 | **0.15%** |
| 1.5억~3억원 | **0.25%** |
| 3억원 초과 | **0.4%** |

### 2.2 1세대 1주택 특례

**공시가격 9억원 이하 1세대 1주택**

| 과세표준 | 일반 세율 | 특례 세율 |
|----------|----------|----------|
| 6,000만원 이하 | 0.1% | **0.05%** |
| 6,000만~1.5억원 | 0.15% | **0.1%** |
| 1.5억~3억원 | 0.25% | **0.2%** |
| 3억원 초과 | 0.4% | **0.35%** |

### 2.3 과세표준 계산

```
과세표준 = 공시가격 × 공정시장가액비율(60%)
```

### 2.4 계산 예시

**공시가격 5억원 아파트**

| 항목 | 계산 |
|------|------|
| 공시가격 | 5억원 |
| 공정시장가액비율 | × 60% |
| 과세표준 | 3억원 |
| 세율 | 0.4% (3억원 초과) |
| 재산세 | 약 **75만원** |

| 5,000만~1억원 | 0.3% |
| 1억원 초과 | **0.5%** |

### 3.2 별도합산 토지

| 과세표준 | 세율 |
|----------|------|
| 2억원 이하 | 0.2% |
| 2억~10억원 | 0.3% |
| 10억원 초과 | **0.4%** |

### 3.3 분리과세 토지

| 토지 유형 | 세율 |
|----------|------|
| 전·답·과수원 | **0.07%** |

## 납부 기한

### 6.1 납부 일정

| 대상 | 납부 기한 | 납부액 |
|------|----------|--------|
| 주택 (1차) | **7월 16~31일** | 1/2 |
| 주택 (2차) | **9월 16~30일** | 1/2 |
| 토지 | **9월 16~30일** | 전액 |
| 건물 | **7월 16~31일** | 전액 |

### 6.2 납부 방법

| 방법 | 내용 |
|------|------|
