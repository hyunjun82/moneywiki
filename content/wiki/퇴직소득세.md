# 퇴직소득세

퇴직소득세는 퇴직금에 부과되는 소득세입니다. [소득세법](https://www.law.go.kr/법령/소득세법)에 따른 계산 방법을 정리합니다.

## 1. 퇴직소득세 계산 구조

### 1.1 계산 흐름

```
퇴직급여액
  ↓ 근속연수공제
퇴직소득금액
  ↓ 환산급여 계산
환산급여
  ↓ 환산급여공제
과세표준
  ↓ 세율 적용
환산산출세액
  ↓ 실제 세액 환산
퇴직소득세
```

### 1.2 계산 공식

```
퇴직소득세 = 환산산출세액 × (근속연수 / 12)
```

## 2. 근속연수공제

### 2.1 공제액 계산

| 근속연수 | 공제액 |
|----------|--------|
| 5년 이하 | 30만원 × 근속연수 |
| 5~10년 | 150만원 + 50만원 × (근속연수-5) |
| 10~20년 | 400만원 + 80만원 × (근속연수-10) |
| 20년 초과 | 1,200만원 + 120만원 × (근속연수-20) |

### 2.2 근속연수별 공제액 예시

| 근속연수 | 공제액 |
|----------|--------|
| 5년 | 150만원 |
| 10년 | 400만원 |
| 15년 | 800만원 |
| 20년 | 1,200만원 |
| 25년 | 1,800만원 |
| 30년 | 2,400만원 |

## 3. 환산급여 계산

### 3.1 환산급여 공식

```
환산급여 = (퇴직급여 - 근속연수공제) × 12 / 근속연수
```

### 3.2 환산급여공제

| 환산급여 | 공제액 |
|----------|--------|
| 800만원 이하 | 환산급여 전액 |
| 800만~7,000만원 | 800만원 + (환산급여-800만원) × 60% |
| 7,000만~1억원 | 4,520만원 + (환산급여-7,000만원) × 55% |
| 1억~3억원 | 6,170만원 + (환산급여-1억원) × 45% |
| 3억원 초과 | 1억5,170만원 + (환산급여-3억원) × 35% |

## 4. 세율 적용

### 4.1 퇴직소득 세율표

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | 6% | - |
| 1,400만~5,000만원 | 15% | 126만원 |
| 5,000만~8,800만원 | 24% | 576만원 |
| 8,800만~1.5억원 | 35% | 1,544만원 |
| 1.5억원 초과 | 38% | 1,994만원 |

### 4.2 환산산출세액 계산

```
환산산출세액 = 과세표준 × 세율 - 누진공제
```

## 5. 계산 예시

### 5.1 예시 1: 10년 근속, 퇴직금 1억원

| 단계 | 계산 |
|------|------|
| 퇴직급여 | 1억원 |
| 근속연수공제 | 400만원 |
| 퇴직소득금액 | 9,600만원 |
| 환산급여 | 9,600만원 × 12/10 = 1억1,520만원 |
| 환산급여공제 | 6,884만원 |
| 과세표준 | 4,636만원 |
| 환산산출세액 | 4,636만원 × 15% - 126만원 = 569만원 |
| **퇴직소득세** | 569만원 × 10/12 = **약 474만원** |

### 5.2 예시 2: 20년 근속, 퇴직금 2억원

| 단계 | 계산 |
|------|------|
| 퇴직급여 | 2억원 |
| 근속연수공제 | 1,200만원 |
| 퇴직소득금액 | 1억8,800만원 |
| 환산급여 | 1억8,800만원 × 12/20 = 1억1,280만원 |
| 환산급여공제 | 6,776만원 |
| 과세표준 | 4,504만원 |
| 환산산출세액 | 4,504만원 × 15% - 126만원 = 550만원 |
| **퇴직소득세** | 550만원 × 20/12 = **약 917만원** |

## 6. 퇴직소득세 감면

### 6.1 IRP 이체 시 세금 이연

| 항목 | 내용 |
|------|------|
| 퇴직금 → IRP | 퇴직소득세 **이연** |
| 55세 이후 연금 수령 | 퇴직소득세 **30~40% 감면** |
| 일시금 수령 | 이연된 세금 납부 |

### 6.2 연금 수령 시 세율

| 연금 수령 나이 | 세율 감면 |
|--------------|----------|
| 55~69세 | 퇴직소득세 × 70% |
| 70~79세 | 퇴직소득세 × 60% |
| 80세 이상 | 퇴직소득세 × 60% |

## 7. 납부 방법

### 7.1 원천징수

| 항목 | 내용 |
|------|------|
| 원천징수 의무 | 회사 |
| 시기 | 퇴직금 지급 시 |
| 신고 | 퇴직 다음 달 10일까지 |

### 7.2 지방소득세

| 항목 | 세율 |
|------|------|
| 퇴직소득세 | 계산된 세액 |
| 지방소득세 | 퇴직소득세의 10% |
| **총 세금** | 퇴직소득세 × 1.1 |

## 8. 주의사항

### 8.1 퇴직금 vs 퇴직연금

| 구분 | 퇴직금 | 퇴직연금 DC |
|------|--------|------------|
| 세금 시점 | 수령 시 | 인출 시 |
| 운용 수익 | 없음 | 과세 (연금소득세) |

### 8.2 중간정산 시

| 항목 | 내용 |
|------|------|
| 근속연수 | 중간정산 시점까지로 계산 |
| 세금 | 중간정산 시 납부 |
| 최종 퇴직 | 남은 기간으로 별도 계산 |
