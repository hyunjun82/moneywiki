# 임대인 수리의무

임대인은 임대물을 사용할 수 있는 상태로 유지해야 합니다. 수리의무 범위를 정확히 알고 권리를 행사하세요.

## 1. 수리의무란

### 1.1 법적 근거

| 항목 | 내용 |
|------|------|
| 근거 | 민법 제623조 |
| 의무 | 임대물의 사용·수익에 필요한 상태 유지 |
| 대상 | 목적물의 **수선** |

### 1.2 수선의무 원칙

| 원칙 | 설명 |
|------|------|
| 임대인 의무 | 사용에 필요한 상태 유지 |
| 범위 | 파손·장해 수선 |
| 정도 | 통상 사용에 지장 없도록 |

## 2. 수리 책임 구분

### 2.1 임대인 책임

| 항목 | 설명 |
|------|------|
| 대규모 수리 | 구조적 수리 |
| 시설 노후 | 경년 열화로 인한 고장 |
| 하자 | 입주 전 하자 |
| 비용 부담 | 상당한 비용 소요 |

### 2.2 임차인 책임

| 항목 | 설명 |
|------|------|
| 소규모 수리 | 간단한 수리 |
| 과실 손상 | 부주의로 인한 고장 |
| 소모품 교체 | 전구, 건전지 등 |

### 2.3 구분 기준

| 기준 | 내용 |
|------|------|
| 수리 규모 | 대규모 vs 소규모 |
| 비용 규모 | 상당한 비용 vs 소액 |
| 원인 | 노후 vs 과실 |
| 목적물 유지 | 필수적 vs 편의적 |

## 3. 항목별 책임

### 3.1 구조·설비 (임대인 책임)

| 항목 | 책임 |
|------|------|
| 지붕 누수 | **임대인** |
| 외벽 균열 | **임대인** |
| 창호 노후 | **임대인** |
| 배관 파손 | **임대인** |
| 보일러 노후 | **임대인** |
| 에어컨 노후 | **임대인** (기존 설치 시) |

### 3.2 설비 고장 (상황에 따라)

| 상황 | 책임 |
|------|------|
| 보일러 노후 고장 | **임대인** |
| 보일러 과실 고장 | **임차인** |
| 배수구 노후 막힘 | **임대인** |
| 배수구 관리 소홀 | **임차인** |

### 3.3 소규모 수리 (임차인 책임)

| 항목 | 책임 |
|------|------|
| 전구 교체 | **임차인** |
| 수도꼭지 패킹 | **임차인** |
| 배수구 청소 | **임차인** |
| 도어락 건전지 | **임차인** |

### 3.4 구체적 판단

| 항목 | 노후·하자 | 과실 |
|------|----------|------|
| 보일러 | 임대인 | 임차인 |
| 에어컨 | 임대인 | 임차인 |
| 싱크대 | 임대인 | 임차인 |
| 변기 | 임대인 | 임차인 |
| 샤워기 | 임대인 | 임차인 |
| 세탁기 연결 | 임대인 | 임차인 |

## 4. 수리 요청 절차

### 4.1 요청 방법

| 단계 | 방법 |
|------|------|
| 1단계 | 임대인에게 연락 |
| 2단계 | 증상 설명 |
| 3단계 | 수리 요청 |
| 4단계 | 수리 일정 협의 |

### 4.2 연락 방법

| 방법 | 권장 |
|------|------|
| 문자·카톡 | ⭐ 증거 남음 |
| 전화 | ○ 긴급 시 |
| 내용증명 | 응하지 않을 때 |

### 4.3 요청 예시

```
[수리 요청]

안녕하세요, ○○동 ○○호 임차인 ○○○입니다.

보일러가 작동하지 않아 난방이 안 됩니다.
사용한 지 10년 이상 된 것으로 노후로 인한
고장으로 보입니다.

빠른 수리를 부탁드립니다.

2026년 1월 15일
임차인 ○○○
```

## 5. 수리 거부 시 대응

### 5.1 임차인 권리

| 권리 | 내용 |
|------|------|
| 수리 후 비용 청구 | 직접 수리 후 비용 청구 |
| 월세 공제 | 수리비를 월세에서 공제 |
| 계약 해지 | 중대한 하자 시 |
| 손해배상 | 하자로 인한 손해 |

### 5.2 비용 청구

| 절차 | 내용 |
|------|------|
| 1 | 임대인에게 수리 요청 (증거 보관) |
| 2 | 상당 기간 대기 |
| 3 | 직접 수리 |
| 4 | 영수증 보관 |
| 5 | 비용 청구 |

### 5.3 월세 공제

| 절차 | 내용 |
|------|------|
| 요건 | 수리 요청 후 미이행 |
| 방법 | 수리비를 월세에서 공제 |
| 통지 | 공제 사실 통지 |
| 주의 | 분쟁 대비 증거 확보 |

### 5.4 계약 해지

| 요건 | 내용 |
|------|------|
| 중대한 하자 | 거주 불가능 수준 |
| 상당 기간 미수리 | 요청 후 상당 기간 경과 |
| 효과 | 계약 해지, 보증금 반환 |

## 6. 특약

### 6.1 유효한 특약

| 특약 | 효력 |
|------|------|
| 소규모 수리 임차인 | ✅ 유효 |
| 특정 항목 지정 | ✅ 유효 |
| 비용 분담 기준 | ✅ 유효 |

### 6.2 무효 가능 특약

| 특약 | 효력 |
|------|------|
| 모든 수리 임차인 | ⚠️ 무효 가능 |
| 대규모 수리 임차인 | ⚠️ 무효 가능 |
| 구조적 수리 임차인 | ⚠️ 무효 가능 |

### 6.3 특약 예시

```
[수리 특약]

1. 대규모 수리(보일러, 배관, 누수 등)는
   임대인이 부담한다.
2. 소규모 수리(전구, 패킹, 청소 등)는
   임차인이 부담한다.
3. 고장 발생 시 임차인은 즉시 임대인에게
   통보하고, 임대인은 신속히 조치한다.
```

| 손해 | ✅ (임대인 과실 시) |

## 10. 자주 묻는 질문

### 10.1 도배·장판도 수리의무?

| 답변 | 설명 |
|------|------|
| ⚠️ 일반적으로 아님 | 소규모 수리로 분류 |
| 예외 | 누수 등 임대인 원인 시 |

### 10.2 옵션 가전도 수리해줘야?

| 답변 | 설명 |
|------|------|
| ⚠️ 계약 조건에 따라 | 제공 조건 확인 |
| 노후 | 임대인 |
| 과실 | 임차인 |

### 10.3 퇴거 후 하자 발견되면?

| 답변 | 설명 |
|------|------|
| 원인 규명 | 기존 하자 vs 임차인 과실 |
| 입증 책임 | 청구자가 입증 |
| 대응 | 인수인계서 확인 |
