# 연말정산 추가납부 분할

연말정산 결과 추가 납부 세금이 발생하면 분할 납부를 신청할 수 있습니다. [소득세법 제137조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 추가납부 발생 원인

### 1.1 주요 원인

| 원인 | 설명 |
|------|------|
| 원천징수 비율 80% 선택 | 매월 세금 적게 납부 |
| 성과급, 인센티브 | 추가 소득 발생 |
| 공제 항목 감소 | 전년 대비 공제 줄어듦 |
| 부양가족 변동 | 부양가족 공제 감소 |

### 1.2 추가납부 세액 계산

```
결정세액 - 기납부세액 = 추가납부 세액
```

| 항목 | 금액 |
|------|------|
| 결정세액 | 350만원 |
| 기납부세액 | 300만원 |
| **추가납부** | **50만원** |

## 2. 분할 납부 제도

### 2.1 분할 납부 조건

[소득세법]에 따른 분할 납부 조건:

| 조건 | 내용 |
|------|------|
| 대상 | 추가 납부 세액 발생한 근로자 |
| 기간 | **최대 3개월** 분할 |
| 이자 | **없음** (원금만 납부) |
| 신청 | 근로자가 회사에 신청 |

### 2.2 분할 납부 장점

| 장점 | 내용 |
|------|------|
| 일시 부담 감소 | 큰 금액도 나눠서 납부 |
| 이자 없음 | 분할해도 추가 비용 없음 |
| 급여에서 공제 | 별도 납부 절차 불필요 |

## 3. 분할 납부 방법

### 3.1 신청 절차

| 단계 | 내용 |
|------|------|
| 1 | 연말정산 결과 확인 |
| 2 | 추가납부 세액 확인 |
| 3 | 회사 인사/경리팀에 분할 신청 |
| 4 | 분할 계획 확정 |
| 5 | 급여에서 분할 공제 |

### 3.2 분할 예시

**추가납부 세액 90만원인 경우**

| 월 | 공제액 | 잔액 |
|----|--------|------|
| 3월 | 30만원 | 60만원 |
| 4월 | 30만원 | 30만원 |
| 5월 | 30만원 | 0원 |

## 4. 회사 입장

### 4.1 회사의 의무

| 항목 | 내용 |
|------|------|
| 분할 허용 | 근로자 신청 시 허용 의무 |
| 납부 대행 | 분할 금액 원천징수 후 납부 |
| 기록 관리 | 분할 납부 내역 기록 |

### 4.2 실무 처리

| 처리 | 내용 |
|------|------|
| 급여 공제 | 매월 급여에서 분할액 공제 |
| 원천세 신고 | 분할 납부 내역 반영 |

## 5. 분할 불가 상황

### 5.1 분할 불가

| 상황 | 이유 |
|------|------|
| 퇴사 예정 | 남은 급여로 정산 필요 |
| 급여 없음 | 공제할 급여 없음 |

### 5.2 대안

| 상황 | 대안 |
|------|------|
| 퇴사 시 | 마지막 급여에서 전액 공제 |
| 급여 부족 | 퇴직금에서 공제 가능 |

## 6. 분할 납부 시 주의사항

### 6.1 회사 정책 확인

| 확인 사항 | 내용 |
|----------|------|
| 분할 가능 기간 | 회사별 상이 가능 |
| 신청 방법 | 서면, 이메일 등 |
| 신청 기한 | 연말정산 완료 전 |

### 6.2 이직 시

| 상황 | 처리 |
|------|------|
| 분할 중 이직 | 퇴사 시 잔액 일시 정산 |
| 새 회사 | 정산 완료 후 입사 |

## 7. 추가납부 줄이는 방법

### 7.1 원천징수 비율 조정

| 비율 | 효과 |
|------|------|
| 80% | 추가납부 가능성 ↑ |
| 100% | 기본 |
| 120% | 환급 가능성 ↑ |

### 7.2 공제 항목 확대

| 방법 | 내용 |
|------|------|
| 연금저축 납입 | 세액공제 |
| 기부금 | 세액공제 |
| 신용카드 → 체크카드 | 공제율 상향 |

## 8. 분할 납부 vs 일시 납부

### 8.1 비교

| 구분 | 분할 납부 | 일시 납부 |
|------|----------|----------|
| 부담 | 낮음 | 높음 |
| 이자 | 없음 | 없음 |
| 절차 | 신청 필요 | 자동 |

### 8.2 선택 기준

| 상황 | 권장 |
|------|------|
| 추가납부 10만원 미만 | 일시 납부 |
| 추가납부 30만원 이상 | 분할 납부 고려 |
| 추가납부 50만원 이상 | 분할 납부 권장 |
