# 재해손실 세액공제

## 기본 개념
천재지변이나 재해로 인한 자산 손실에 대해 세액을 공제해주는 제도입니다.

## 대상 재해

### 천재지변
- 태풍, 홍수, 해일
- 지진, 화산 폭발
- 낙뢰, 폭설, 가뭄

### 인재
- 화재
- 붕괴, 폭발
- 교통사고 (일부)

### 기타
- 전염병으로 인한 재산 피해
- 정부 재난 선포 지역

## 공제 대상 자산

### 주택 및 건물
- 본인 거주 주택
- 사업용 건물
- 부속 건물

### 가재도구
- 가전제품
- 가구
- 생활용품

### 차량
- 자가용 승용차
- 화물차, 영업용 차량

### 기타 자산
- 농작물, 가축
- 사업용 자산

## 공제 금액 계산

### 기본 계산식
```
공제 금액 = (재해손실액 - 보험금 등 보전액) × 15%
```

### 공제 한도
- **한도 없음** (실제 손실액 기준)
- 다만, 해당 연도 결정세액 한도 내

### 최저 손실액
- 특별한 최저 기준 없음
- 모든 재해손실 공제 가능

## 계산 사례

### 사례 1: 화재로 주택 손실
```
주택 손실액: 5,000만원
보험금 수령: 3,000만원
실제 손실: 2,000만원
→ 세액공제: 2,000 × 15% = 300만원
```

### 사례 2: 태풍 피해
```
주택 수리비: 1,500만원
가재도구 손실: 500만원
보험금: 500만원
실제 손실: 1,500만원
→ 세액공제: 1,500 × 15% = 225만원
```

### 사례 3: 홍수 피해
```
차량 손실: 2,000만원
보험금: 1,800만원
실제 손실: 200만원
→ 세액공제: 200 × 15% = 30만원
```

## 신청 절차

### 피해 확인
- 소방서 화재증명서
- 경찰서 사고확인서
- 지방자치단체 재난확인서
- 기상청 기상증명서

### 손실액 평가
- 손해사정사 감정
- 세무사 확인
- 사진·영상 증빙

### 서류 준비
- 재해사실 확인서
- 손실액 산정 명세서
- 보험금 수령 내역
- 수리비 영수증

### 신고
- 연말정산 시 회사에 제출
- 또는 종합소득세 확정신고 시 신청

## 주의사항

### 보전액 제외
- 보험금
- 재해지원금
- 손해배상금
- 복구비 지원금

### 중복 공제 불가
- 사업소득 필요경비와 중복 불가
- 한쪽만 선택

### 시기
- 재해 발생 연도에 공제
- 다음 연도 이월 불가

### 증빙 보관
- 모든 증빙서류 5년간 보관
- 사진, 영상 자료 보관 권장

## 관련 지원 제도

### 재난지원금
- 정부·지자체 재난지원금
- 별도 신청 필요
- 세액공제와 별개

### 긴급복구 지원
- 주택 응급복구비 지원
- 생활안정 지원금

### 특별재난지역 추가 혜택
- 지방세 감면
- 건강보험료 감면
- 전기·가스 요금 감면

## 실무 팁

1. **즉시 증빙 확보**: 피해 즉시 사진·영상 촬영
2. **공식 확인서 발급**: 관련 기관에서 확인서 발급 필수
3. **보험 처리**: 보험 청구와 별도로 세액공제 신청
4. **전문가 상담**: 손실액이 큰 경우 세무사 상담
