부모님께 상속받은 집을 팔려고 하는데 세금이 걱정되시죠? 함께 알아보시죠.

**상속주택은 "부모님과 같이 살았느냐, 따로 살았느냐"에 따라 거주요건이 완전히 달라요. 이거 모르면 비과세 못 받고 세금 폭탄 맞을 수 있어요.**

## 핵심 요약

| 상속 유형 | 거주·보유 기간 | 비과세 요건 |
|----------|--------------|-----------|
| **동일세대** | 피상속인 기간 **합산** | 합산해서 2년 보유+거주 |
| **별도세대** | 상속인이 **별도 충족** | 상속 후 2년 보유+거주 |

## 별도세대 상속 - 이게 함정이에요

### 실제 사례

고요한씨는 무주택자로 **별도세대**인 부친으로부터 A주택을 상속받았어요.

```
'98.12월    →    '21.10월    →    '25.5월
 아버지          고요한씨         고요한씨
 A주택 취득     A주택 상속       A주택 양도
               (조정대상지역)    (거주 안 함)
```

고요한씨는 이렇게 생각했어요.

> "아버지가 오래 보유하셨으니까 비과세 받을 수 있겠지?"

**결과: 양도소득세 과세!**

왜냐하면 **별도세대**에서 상속받은 경우에는 피상속인(아버지)의 거주기간을 합산할 수 없어요.

### 별도세대 상속이 뭐예요?

| 구분 | 설명 |
|------|------|
| **별도세대** | 상속 당시 부모님과 **따로 살던** 경우 |
| **동일세대** | 상속 당시 부모님과 **같이 살던** 경우 |

쉽게 말해서 **주민등록이 따로 되어 있었으면** 대부분 별도세대예요.

## 동일세대 상속 - 기간 합산 가능

### 동일세대면 유리해요

부모님과 같이 살다가 상속받은 경우에는 **피상속인의 거주·보유 기간을 합산**할 수 있어요.

```
예: 부모님 20년 보유·거주 + 상속인 1년 보유·거주
= 총 21년으로 계산

→ 2년 거주요건 충족!
→ 1세대1주택 비과세 가능!
```

### 비교 정리

| 항목 | 동일세대 상속 | 별도세대 상속 |
|------|-------------|-------------|
| 기간 합산 | **가능** | **불가** |
| 거주요건 | 합산해서 판단 | 상속인이 별도 충족 |
| 주의사항 | 같이 살았음을 증명 | 상속 후 거주 필수 |

## 조정대상지역이면 거주요건 필수

### 조정대상지역 상속주택

상속개시일(사망일) 기준으로 주택이 **조정대상지역**에 있으면 거주요건이 적용돼요.

| 요건 | 조정대상지역 | 비조정대상지역 |
|------|------------|--------------|
| 보유 | 2년 이상 | 2년 이상 |
| 거주 | **2년 이상** | 없음 |

### 취득시기는 상속개시일

상속주택의 취득시기는 **상속개시일**(피상속인 사망일)이에요.

```
아버지 사망: 2021.10.15
→ 상속주택 취득시기: 2021.10.15

2021.10.15 당시 조정대상지역이면
→ 상속인이 2년 이상 거주해야 비과세
```

## 비과세 받으려면 이렇게 하세요

### 별도세대 상속인 경우

```
1. 상속개시일 확인 (취득시기)
   ↓
2. 조정대상지역 여부 확인
   ↓
3. 조정대상지역이면 2년 이상 거주 필수
   ↓
4. 2년 보유 + 2년 거주 후 양도
   ↓
5. 1세대1주택 비과세 적용
```

### 동일세대 상속인 경우

```
1. 동일세대였음을 증명 (주민등록 등)
   ↓
2. 피상속인 + 상속인 거주·보유 기간 합산
   ↓
3. 합산 2년 이상이면 거주요건 충족
   ↓
4. 양도 후 비과세 신고
```

## 주의사항

### 흔한 실수

| 실수 | 결과 |
|------|------|
| 별도세대인데 거주 안 하고 양도 | 비과세 불가, 양도세 과세 |
| 부모님 보유기간만 믿고 양도 | 별도세대면 합산 불가 |
| 조정대상지역 확인 안 함 | 거주요건 누락 |

### 이것만 기억하세요

> **별도세대 상속 = 상속인이 처음부터 다시 거주요건 충족해야 함**

동일세대였으면 합산 가능하지만, 별도세대였으면 상속개시일부터 새로 시작이에요.

## 관련 법령 해석

### 기획재정부 해석 (2023.08.09)

> "상속인과 피상속인이 상속개시 당시 동일세대가 아닌 경우에는 피상속인의 거주기간을 통산할 수 없는 것으로, 조정대상지역 내의 주택을 별도세대인 자가 상속받아 양도하는 경우에는 거주기간의 제한을 받는 것입니다."
