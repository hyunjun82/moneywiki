# 전입신고 방법

전입신고는 임차인의 대항력 확보에 필수입니다. 잔금일 당일 전입신고를 해야 보증금을 안전하게 보호받습니다.

## 1. 전입신고란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 새 주소지로 주민등록 이전 |
| 근거 | 주민등록법 제16조 |
| 기한 | 이사 후 **14일 이내** |

### 1.2 왜 중요한가

| 이유 | 설명 |
|------|------|
| 대항력 | 제3자에게 임대차 주장 |
| 우선변제권 | 경매 시 보증금 보호 |
| 행정 | 각종 행정 서비스 이용 |

### 1.3 대항력 발생 시점

| 시점 | 내용 |
|------|------|
| 전입신고 | 신고일 |
| 대항력 발생 | **다음날 0시** |
| 주의 | 잔금일 당일 신고 필수 |

## 2. 전입신고 방법

### 2.1 방문 신고

| 절차 | 내용 |
|------|------|
| 장소 | 새 주소지 관할 주민센터 |
| 시간 | 평일 9시~18시 |
| 처리 | 즉시 처리 |

### 2.2 온라인 신고

| 절차 | 내용 |
|------|------|
| 사이트 | 정부24 (gov.kr) |
| 인증 | 공인인증서, 간편인증 |
| 시간 | 24시간 가능 |
| 처리 | 즉시~당일 |

### 2.3 무인민원발급기

| 가능 | 내용 |
|------|------|
| 전입신고 | ❌ 불가 |
| 등본 발급 | ✅ 가능 |

## 3. 온라인 전입신고 절차

### 3.1 정부24 접속

| 단계 | 내용 |
|------|------|
| 1 | gov.kr 접속 |
| 2 | 로그인 (인증서/간편인증) |
| 3 | "전입신고" 검색 |
| 4 | "전입신고 신청" 선택 |

### 3.2 신청서 작성

| 항목 | 입력 |
|------|------|
| 전입자 정보 | 본인 및 동반 가족 |
| 새 주소 | 전입할 주소 |
| 전입 일자 | 이사한 날짜 |
| 세대주 관계 | 세대주/세대원 |

### 3.3 제출 및 확인

| 단계 | 내용 |
|------|------|
| 제출 | 신청서 제출 |
| 확인 | 처리 상태 확인 |
| 완료 | 등본에서 주소 확인 |

## 4. 필요 서류

### 4.1 방문 신고 시

| 서류 | 필수 |
|------|------|
| 신분증 | ✅ 필수 |
| 전입신고서 | 현장 작성 가능 |
| 임대차계약서 | 확정일자 동시 신청 시 |

### 4.2 세대원 포함 시

| 서류 | 필수 |
|------|------|
| 세대원 신분증 | 동반 시 |
| 위임장 | 대리 신고 시 |
| 가족관계증명서 | 필요 시 |

### 4.3 대리 신고 시

| 서류 | 필수 |
|------|------|
| 위임장 | ✅ 필수 |
| 신청인 신분증 | ✅ 필수 |
| 대리인 신분증 | ✅ 필수 |

## 5. 확정일자 동시 신청

### 5.1 동시 신청 방법

| 방법 | 내용 |
|------|------|
| 방문 | 전입신고 시 함께 신청 |
| 준비물 | 임대차계약서 원본 |
| 수수료 | 600원 |

### 5.2 온라인 확정일자

| 방법 | 내용 |
|------|------|
| 사이트 | 등기소 인터넷등기소 |
| 조건 | 전자계약 시 자동 부여 |

### 5.3 확정일자 효력

| 효력 | 내용 |
|------|------|
| 우선변제권 | 경매 시 보증금 우선 배당 |
| 순위 | 확정일자 받은 날 기준 |

## 6. 전입신고 시기

### 6.1 권장 시기

| 시기 | 권장 |
|------|------|
| 잔금일 당일 | ⭐ 최우선 권장 |
| 이사 당일 | ⭐ 권장 |
| 14일 이내 | 법정 기한 |

### 6.2 잔금일 전입신고

| 이유 | 설명 |
|------|------|
| 대항력 | 다음날 0시 발생 |
| 순위 | 근저당 설정 전 확보 |
| 보호 | 보증금 보호 |

### 6.3 주의사항

| 주의 | 내용 |
|------|------|
| 늦은 전입 | 후순위 임차인 될 수 있음 |
| 등기 확인 | 전입 전 등기부 확인 |

## 7. 전입신고 확인

### 7.1 확인 방법

| 방법 | 내용 |
|------|------|
| 주민등록등본 | 새 주소 확인 |
| 정부24 | 온라인 발급 |
| 주민센터 | 방문 발급 |

### 7.2 확인 시점

| 시점 | 확인 |
|------|------|
| 신고 직후 | 등본 발급 가능 |
| 온라인 | 처리 완료 후 |

## 8. 특수 상황

### 8.1 오피스텔 전입

| 확인 | 내용 |
|------|------|
| 용도 | 주거용인지 확인 |
| 건축물대장 | 용도 확인 |
| 전입 가능 | 주거용만 가능 |

### 8.2 다가구주택

| 확인 | 내용 |
|------|------|
| 호수 | 정확한 호수 기재 |
| 동 | 동·호수 정확히 |

### 8.3 위반건축물

| 상황 | 대응 |
|------|------|
| 전입 불가 | 위반건축물은 제한 |
| 확인 | 건축물대장 확인 |
| 주의 | 계약 전 확인 필수 |

## 9. 과태료

### 9.1 신고 지연 시

| 기간 | 과태료 |
|------|--------|
| 14일 초과 | 최대 5만원 |
| 상습 위반 | 가중 |

### 9.2 면제 사유

| 사유 | 면제 |
|------|------|
| 질병 | 검토 |
| 해외 체류 | 검토 |
| 천재지변 | 면제 |

## 10. 자주 묻는 질문

### 10.1 주말에도 전입신고 가능?

| 답변 | 방법 |
|------|------|
| 방문 | ❌ 불가 (평일만) |
| 온라인 | ✅ 가능 (정부24) |

### 10.2 실거주 안 해도 전입 가능?

| 답변 | 설명 |
|------|------|
| 원칙 | 실거주지에 전입 |
| 위반 | 허위 전입 시 처벌 |
| 임대차 | 실거주 필수 |

### 10.3 세대 분리와 전입신고?

| 답변 | 설명 |
|------|------|
| 가능 | 같은 주소 세대 분리 |
| 요건 | 독립 세대 요건 충족 |
| 신청 | 주민센터 상담 |
