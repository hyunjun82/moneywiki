# 전입신고 방법

전입신고는 보증금 보호의 첫 단계예요. [대항력](/w/대항력-발생-시점)을 얻으려면 반드시 해야 해요.

## 1. 전입신고란

새 주소지로 주민등록을 옮기는 거예요. 이걸 해야 [대항력](/w/대항력-발생-시점)이 생기고, 나중에 집이 팔리거나 경매에 넘어가도 보증금을 보호받을 수 있어요.

| 항목 | 내용 |
|------|------|
| 기한 | 이사 후 **14일 이내** |
| 근거 | [주민등록법 제16조](https://www.law.go.kr/법령/주민등록법) |
| 효과 | 대항력 발생 (다음날 0시) |

**핵심**: 대항력은 전입신고한 **다음 날 0시**에 생겨요. 그래서 잔금일 당일 바로 신고해야 해요.

## 2. 온라인 전입신고 (추천)

[정부24](https://www.gov.kr)에서 24시간 신청 가능해요. 주민센터 갈 필요 없어요.

| 단계 | 방법 |
|------|------|
| 1 | [정부24](https://www.gov.kr) 접속 |
| 2 | 로그인 (공인인증서 또는 간편인증) |
| 3 | "전입신고" 검색 → 신청 |
| 4 | 새 주소, 전입일자, 동반가족 입력 |
| 5 | 제출 → 처리 완료 확인 |

## 3. 주민센터 방문 신고

온라인이 어려우면 새 주소지 관할 주민센터에 직접 가면 돼요.

| 항목 | 내용 |
|------|------|
| 장소 | 새 주소지 관할 주민센터 |
| 시간 | 평일 9시~18시 |
| 필요서류 | 신분증 (대리 시 위임장 추가) |
| 처리 | 즉시 |

## 4. 확정일자 동시에 받기

전입신고할 때 [확정일자](/w/확정일자)도 같이 받으세요. 임대차계약서 원본 가져가면 돼요. 수수료 600원.

[확정일자](/w/확정일자)까지 받으면 [우선변제권](/w/우선변제권-요건)이 생겨서 경매 시 보증금을 먼저 받을 수 있어요.

| 확정일자 효과 | 설명 |
|--------------|------|
| 우선변제권 | 경매 시 보증금 먼저 배당 |
| 순위 | 확정일자 받은 날짜 기준 |

## 5. 특수 상황

### 오피스텔
주거용 오피스텔만 전입신고 가능해요. 업무용은 안 돼요. 건축물대장에서 용도 확인하세요.

### 다가구주택
정확한 동·호수를 기재해야 해요. 틀리면 [대항력](/w/대항력-발생-시점)에 문제 생길 수 있어요.

### 위반건축물
전입신고가 제한될 수 있어요. 계약 전에 건축물대장 확인 필수.

## 6. 과태료

14일 넘기면 최대 5만원 과태료가 나올 수 있어요.

| 위반 | 과태료 |
|------|--------|
| 14일 초과 | 최대 5만원 |
| 상습 | 가중 |

## 7. 자주 묻는 질문

### 주말에도 가능해요?
방문은 안 되고, [정부24](https://www.gov.kr) 온라인은 24시간 가능해요.

### 실거주 안 하면?
실제 거주지에 전입해야 해요. 허위 전입은 처벌받아요. [임대차보호법](/w/임대차보호법)도 실거주가 조건이에요.

### 세대 분리도 가능?
같은 주소에서 세대 분리 가능해요. 주민센터에서 상담받으세요.
