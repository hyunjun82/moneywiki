장기주택저당차입금 이자상환액 공제는 주택담보대출 이자에 대해 소득공제를 받는 제도예요. [국세청 주택자금 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 내용을 정리해 볼게요.

## 공제 대상자 요건

[소득세법 제52조](https://www.law.go.kr/법령/소득세법)에 따른 대상자 요건이에요.

| 요건 | 조건 |
|------|------|
| 주택 수 | 무주택 또는 1주택 세대주 |
| 세대주 여부 | 세대주 (세대원 불가) |
| 주택 가격 | 취득 당시 기준시가 5억원 이하 |
| 주택 규모 | 국민주택규모(85㎡) 이하 또는 기준시가 5억원 이하 |

## 대출 요건

상환 기간 요건이에요.

| 상환 기간 | 공제 가능 |
|----------|----------|
| 15년 이상 | 가능 |
| 10~15년 미만 | 2014년 이전 차입분만 |

차입 시기 요건이에요.

| 요건 | 조건 |
|------|------|
| 신규 취득 | 취득일로부터 3개월 이내 차입 |
| 저당권 설정 | 해당 주택에 저당권 설정 필수 |

## 공제 한도

2015년 이후 차입분 한도예요.

| 조건 | 공제 한도 |
|------|----------|
| 15년 이상, 고정금리 또는 비거치식 | 500만원 |
| 15년 이상, 고정금리 및 비거치식 | 1,500만원 |
| 15년 이상, 고정금리 및 비거치식 + 상환기간 15년 이상 | 1,800만원 |

고정금리는 대출 기간 전체 금리가 고정된 경우예요. 비거치식은 거치기간 없이 원금을 분할상환하는 방식이에요.

## 공제 대상 금액

공제 대상 여부예요.

| 항목 | 공제 여부 |
|------|----------|
| 이자상환액 | 공제 대상 |
| 원금상환액 | 공제 불가 |
| 중도상환수수료 | 공제 불가 |

연간 실제 상환한 이자 금액만 공제 대상이에요.

## 계산 예시

취득 주택 기준시가 4억원(85㎡), 대출 3억원(15년 상환, 고정금리, 비거치식), 연간 이자상환액 1,200만원인 경우예요.

| 항목 | 금액 |
|------|------|
| 이자상환액 | 1,200만원 |
| 공제 한도 | 1,500만원 |
| 소득공제액 | 1,200만원 |

세율 24% 적용 시 1,200만원 × 24% = 288만원 절세 효과가 있어요.

## 대환대출 공제

기존 대출을 새 대출로 갈아탄 경우 공제 조건이에요.

| 조건 | 내용 |
|------|------|
| 기존 대출 상환용 | 기존 장기주택저당차입금 대환 |
| 대환 시점 | 기존 대출 상환과 동시 |
| 상환 기간 | 대환 후 잔여 상환기간 유지 |

대환 후 상환 기간이 단축되면 공제받을 수 없어요.

## 2주택 이상 소유 시

2주택 이상인 경우 공제 여부예요.

| 상황 | 공제 |
|------|------|
| 2주택 이상 | 불가 |
| 연도 중 2주택 된 경우 | 해당 연도 공제 불가 |
| 다시 1주택 된 경우 | 1주택 된 연도부터 다시 가능 |

## 증빙서류

필요한 서류예요.

| 서류 | 용도 |
|------|------|
| 주민등록표등본 | 세대주, 1주택 확인 |
| 등기사항전부증명서 | 저당권 설정 확인 |
| 대출상환증명서 | 이자상환액 확인 |
| 분양계약서 (신축) | 취득 시기 확인 |

금융기관 대출은 간소화 서비스에서 자동 조회돼요.

## 주의사항

세대주만 공제받을 수 있어요. 세대원은 공제 대상이 아니에요.

기준시가는 취득 당시 기준이에요. 현재 시가가 아니라 취득할 때 5억원 이하였어야 해요.

대출 실행 시 상환 기간이 15년 이상으로 설정되어 있어야 하고, 중도상환으로 15년 미만이 되면 공제받을 수 없어요.
