# 연말정산 장기주택저당차입금 공제

장기주택저당차입금 이자상환액 공제는 주택담보대출 이자에 대해 **소득공제**를 받는 제도입니다. [국세청 주택자금 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 조건을 정리합니다.

## 1. 공제 대상자 요건

[소득세법 제52조](https://www.law.go.kr/법령/소득세법)에 따른 대상자 요건입니다.

| 요건 | 조건 |
|------|------|
| 주택 수 | 무주택 또는 **1주택** 세대주 |
| 세대주 여부 | **세대주** (세대원 불가) |
| 주택 가격 | 취득 당시 기준시가 **5억원 이하** |
| 주택 규모 | 국민주택규모(85㎡) 이하 또는 기준시가 5억원 이하 |

## 2. 대출 요건

### 2.1 상환 기간

| 상환 기간 | 공제 한도 |
|----------|----------|
| **15년 이상** | 공제 가능 |
| 10~15년 미만 | 2014년 이전 차입분만 |

### 2.2 차입 시기

| 요건 | 조건 |
|------|------|
| 신규 취득 | 취득일로부터 **3개월 이내** 차입 |
| 소유권 이전등기 | 등기 전 차입 가능 |

### 2.3 저당권 설정

해당 주택에 **저당권** 설정 필수

## 3. 공제 한도

### 3.1 2015년 이후 차입분

| 조건 | 공제 한도 |
|------|----------|
| 15년 이상, 고정금리 또는 비거치식 | **500만원** |
| 15년 이상, 고정금리 **및** 비거치식 | **1,500만원** |
| 15년 이상, 고정금리 및 비거치식 + 상환기간 15년 이상 | **1,800만원** |

### 3.2 세부 기준

| 구분 | 내용 |
|------|------|
| 고정금리 | 대출 기간 전체 금리 고정 |
| 비거치식 | 거치기간 없이 원금 분할상환 |

### 3.3 연도별 한도 비교

| 차입 시기 | 10~15년 | 15년 이상 |
|----------|---------|----------|
| 2012~2014년 | 300~500만원 | 500~1,500만원 |
| 2015년 이후 | 공제 불가 | 500~1,800만원 |

## 4. 공제 대상 금액

### 4.1 이자상환액만 공제

| 항목 | 공제 여부 |
|------|----------|
| **이자상환액** | ✅ 공제 대상 |
| 원금상환액 | ❌ 공제 불가 |
| 중도상환수수료 | ❌ 공제 불가 |

### 4.2 연간 이자상환액

연간 실제 상환한 **이자 금액** 기준

## 5. 계산 예시

### 5.1 예시 상황

- 취득 주택: 기준시가 4억원, 85㎡
- 대출: 3억원, 15년 상환, 고정금리, 비거치식
- 연간 이자상환액: 1,200만원

### 5.2 계산

| 항목 | 금액 |
|------|------|
| 이자상환액 | 1,200만원 |
| 공제 한도 | 1,500만원 |
| **소득공제액** | **1,200만원** |

세율 24% 적용 시: 1,200만원 × 24% = **288만원** 절세

## 6. 대환대출

### 6.1 공제 가능 조건

| 조건 | 내용 |
|------|------|
| 기존 대출 상환용 | 기존 장기주택저당차입금 대환 |
| 대환 시점 | 기존 대출 상환과 동시 |
| 상환 기간 | 대환 후 잔여 상환기간 유지 |

### 6.2 주의사항

- 대환 후 **상환 기간이 단축**되면 공제 불가
- 대환 시 **한도 재계산** 필요

## 7. 2주택 이상 소유 시

### 7.1 공제 불가

| 상황 | 공제 |
|------|------|
| 2주택 이상 | ❌ 불가 |
| 연도 중 2주택 된 경우 | 해당 연도 공제 불가 |

### 7.2 다시 1주택 된 경우

- 1주택이 된 연도부터 **다시 공제 가능**
- 기존 대출이 요건 충족해야 함

## 8. 증빙서류

### 8.1 연말정산 간소화

금융기관 대출은 **간소화 서비스에서 조회**

### 8.2 필요 서류

| 서류 | 용도 |
|------|------|
| 주민등록표등본 | 세대주, 1주택 확인 |
| 등기사항전부증명서 | 저당권 설정 확인 |
| 대출상환증명서 | 이자상환액 확인 |
| 분양계약서 (신축) | 취득 시기 확인 |

## 9. 주의사항

### 9.1 세대주 요건

- **세대주만** 공제 가능
- 세대원은 공제 불가

### 9.2 기준시가 확인

- **취득 당시** 기준시가 5억원 이하
- 현재 시가 아님

### 9.3 상환 기간

- 대출 실행 시 **상환 기간 15년 이상** 설정
- 중도상환으로 15년 미만 되면 공제 불가
