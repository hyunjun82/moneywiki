# 연말정산 공제율

항목별 공제율을 정리합니다.

## 1. 세액공제율

| 항목 | 공제율 |
|------|--------|
| 보험료 | **12%** |
| 의료비 | **15%** (난임 20%) |
| 교육비 | **15%** |
| 연금저축 | **13.2~16.5%** |
| 기부금 | **15~30%** |
| 월세 | **15~17%** |

## 2. 소득공제율

| 항목 | 공제율 |
|------|--------|
| 신용카드 | **15%** |
| 체크카드 | **30%** |
| 현금영수증 | **30%** |
| 전통시장 | **40%** |
| 대중교통 | **40%** |
| 주택청약 | **40%** |
| 주택임차 | **40%** |

## 3. 연금저축 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **16.5%** |
| 5,500만원 초과 | **13.2%** |

## 4. 월세 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **17%** |
| 5,500만원 초과 | **15%** |

## 5. 기부금 공제율

| 금액 | 공제율 |
|------|--------|
| 1천만원 이하 | **15%** |
| 1천만원 초과 | **30%** |
