직업훈련비 교육비 공제를 정리해 볼게요. 본인 직업훈련비는 전액 세액공제받을 수 있어요.

## 공제 개요

직업훈련비 공제 개요예요.

| 항목 | 내용 |
|------|------|
| 공제 유형 | 교육비 세액공제 |
| 공제율 | 15% |
| 한도 | 본인 전액 |

## 공제 대상

공제받을 수 있는 훈련이에요.

| 대상 | 공제 |
|------|------|
| 직업능력개발훈련 | 가능 |
| 학점은행제 | 가능 |
| 사이버대학 | 가능 |
| 자격증 학원 (직훈시설) | 가능 |

## 직업훈련시설

인정되는 시설과 안 되는 시설이에요.

| 시설 | 공제 |
|------|------|
| 고용노동부 인정 | 가능 |
| 직업전문학교 | 가능 |
| 일반 학원 | 불가 |

## 필요서류

공제받으려면 필요한 서류예요.

| 서류 | 발급처 |
|------|--------|
| 교육비납입증명서 | 훈련기관 |
| 수강증 | 훈련기관 |

## 주의사항

공제받을 때 주의할 점이에요.

| 주의 | 내용 |
|------|------|
| 시설 확인 | 직업훈련시설 인정 여부를 확인해야 해요 |
| 일반학원 | 일반 학원은 공제받을 수 없어요 |
