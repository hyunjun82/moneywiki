# 연말정산 직업훈련비

직업훈련비 교육비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 교육비 세액공제 |
| 공제율 | **15%** |
| 한도 | 본인 **전액** |

## 2. 공제 대상

| 대상 | 공제 |
|------|------|
| 직업능력개발훈련 | ✅ |
| 학점은행제 | ✅ |
| 사이버대학 | ✅ |
| 자격증 학원 | ✅ (직훈시설) |

## 3. 직업훈련시설

| 시설 | 공제 |
|------|------|
| 고용노동부 인정 | ✅ |
| 직업전문학교 | ✅ |
| 일반 학원 | ❌ |

## 4. 필요서류

| 서류 | 발급처 |
|------|--------|
| 교육비납입증명서 | 훈련기관 |
| 수강증 | 훈련기관 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 시설 확인 | 직업훈련시설 인정 여부 |
| 일반학원 | 공제 불가 |
