# 연말정산 라식 라섹

라식, 라섹 수술비 의료비 공제를 정리합니다.

## 1. 공제 여부

| 수술 | 공제 |
|------|------|
| 라식 | ✅ 가능 |
| 라섹 | ✅ 가능 |
| 스마일라식 | ✅ 가능 |
| ICL삽입술 | ✅ 가능 |

## 2. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 대상자별 상이 |

## 3. 공제 한도

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 | **한도 없음** |
| 65세 이상 | **한도 없음** |
| 그 외 | 700만원 |

## 4. 공제 예시

| 수술비 | 대상 | 공제액(15%) |
|--------|------|-------------|
| 200만원 | 본인 | 30만원 |
| 300만원 | 배우자 | 45만원 (한도 내) |

## 5. 필요서류

| 서류 | 발급처 |
|------|--------|
| 의료비 영수증 | 안과 |
| 간소화 자료 | 자동 반영 |
