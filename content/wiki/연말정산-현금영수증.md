# 연말정산 현금영수증

현금영수증 소득공제를 정리합니다.

## 1. 공제율

| 결제수단 | 공제율 |
|----------|--------|
| 현금영수증 | **30%** |
| 체크카드 | 30% |
| 신용카드 | 15% |

## 2. 현금영수증 종류

| 종류 | 설명 |
|------|------|
| 소득공제용 | 개인 소득공제 |
| 지출증빙용 | 사업자 경비 처리 |

## 3. 등록 방법

| 방법 | 절차 |
|------|------|
| 홈택스 | 조회/발급 → 현금영수증 → 소비자 등록 |
| 휴대폰 | 휴대폰 번호 등록 |
| 카드 | 현금영수증 전용카드 |

## 4. 발급 받기

| 방법 | 설명 |
|------|------|
| 휴대폰 번호 | 결제 시 번호 제시 |
| 전용카드 | 카드 제시 |
| 자진발급 | 국세청 자동 귀속 |

## 5. 자진발급 조회

| 절차 | 내용 |
|------|------|
| 1 | 홈택스 로그인 |
| 2 | 조회/발급 → 현금영수증 |
| 3 | 자진발급 내역 확인 |
| 4 | 본인 귀속 신청 |
