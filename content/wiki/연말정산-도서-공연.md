# 연말정산 도서 공연

도서·공연비 소득공제를 정리합니다.

## 1. 공제 대상

| 대상 | 포함 |
|------|------|
| 도서 구입 | ✅ |
| 공연 관람 | ✅ |
| 박물관 입장 | ✅ |
| 미술관 입장 | ✅ |
| 영화 관람 | ✅ |

## 2. 공제 조건

| 요건 | 기준 |
|------|------|
| 총급여 | **7천만원 이하** |
| 공제율 | **30%** |
| 추가한도 | **100만원** |

## 3. 공제 제외

| 제외 | 내용 |
|------|------|
| 총급여 7천만원 초과 | 일반 카드로 공제 |
| 전자책 | 일부 제외 |

## 4. 결제 방법

| 방법 | 인정 |
|------|------|
| 신용카드 | ✅ |
| 체크카드 | ✅ |
| 현금영수증 | ✅ |

## 5. 간소화 자료

| 항목 | 내용 |
|------|------|
| 표시 | 도서공연 별도 표시 |
| 확인 | 간소화 자료에서 확인 |
