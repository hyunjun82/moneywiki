도서·공연·영화비는 일반 신용카드보다 높은 30% 공제율이 적용돼요. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따르면 총급여 7천만원 이하 근로자만 대상이에요.

## 공제 대상

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따라 문화비 소득공제 대상이에요.

| 대상 | 공제 |
|------|------|
| 도서 구입 | 가능 |
| 공연 관람 | 가능 |
| 박물관 입장 | 가능 |
| 미술관 입장 | 가능 |
| 영화 관람 | 가능 |

종이책, 전자책, 오디오북 모두 도서 구입으로 공제돼요. 뮤지컬, 연극, 콘서트, 오페라, 발레도 공연 관람으로 공제받을 수 있어요. 영화는 2023년 7월부터 공제 대상에 포함됐어요.

## 공제 조건

| 요건 | 기준 |
|------|------|
| 총급여 | 7천만원 이하 |
| 공제율 | 30% |
| 추가한도 | 100만원 |

총급여 7천만원을 넘으면 도서·공연·영화 공제를 못 받아요. 이 경우 일반 신용카드 공제(15%)로만 처리돼요.

## 공제율 비교

같은 금액을 써도 공제 효과가 달라요.

| 항목 | 공제율 |
|------|--------|
| 도서·공연·영화 | 30% |
| 신용카드 | 15% |
| 체크카드 | 30% |
| 현금영수증 | 30% |

100만원 사용 시 도서·공연·영화는 30만원 공제, 신용카드는 15만원 공제예요. 같은 돈을 쓰면 문화비로 쓰는 게 2배 유리해요.

## 공제 제외

| 제외 대상 | 이유 |
|----------|------|
| 총급여 7천만원 초과 | 소득 기준 미충족 |
| 신문·잡지 구독 | 도서 아님 |
| OTT 구독료 | 공연·영화 아님 |
| 스포츠 경기 | 공연 아님 |

## 결제 방법

| 방법 | 인정 |
|------|------|
| 신용카드 | 가능 |
| 체크카드 | 가능 |
| 현금영수증 | 가능 |

어떤 결제 수단이든 30% 공제율이 동일하게 적용돼요. 문화비 소득공제 가맹점에서 결제해야 해요. [문화비 소득공제 홈페이지](https://www.culture.go.kr/deduction)에서 가맹점을 확인할 수 있어요.

## 간소화 자료

연말정산 간소화 서비스에서 도서·공연 사용금액이 별도로 표시돼요. 1월 15일부터 자동 구분되니까 별도 증빙 없이 확인하면 돼요.
