# 연말정산 정치자금 기부금

정치자금 기부금은 가장 높은 세액공제율이 적용되는 기부금입니다. [국세청 기부금 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 구조

### 1.1 공제율 체계

[정치자금법](https://www.law.go.kr/법령/정치자금법)에 따른 공제율입니다.

| 기부 금액 | 공제율 | 비고 |
|----------|--------|------|
| **10만원 이하** | **100/110** (약 90.9%) | 거의 전액 환급 |
| 10만원 초과 ~ 3천만원 | **15%** | |
| 3천만원 초과 | **25%** | |

### 1.2 10만원 기부 시 실질 부담

| 항목 | 금액 |
|------|------|
| 기부금 | 10만원 |
| 세액공제 | 10만원 × 100/110 = **9만 909원** |
| 실질 부담 | **약 9,091원** |

## 2. 공제 대상

### 2.1 기부 대상

| 대상 | 공제 여부 |
|------|----------|
| 정당 (당비 포함) | ✅ 가능 |
| 후원회 | ✅ 가능 |
| 선거관리위원회 기탁금 | ✅ 가능 |

### 2.2 공제 불가 대상

| 대상 | 공제 여부 |
|------|----------|
| 후보자에게 직접 전달 | ❌ 불가 |
| 정치인 개인 계좌 입금 | ❌ 불가 |
| 불법 정치자금 | ❌ 불가 |

## 3. 기부 한도

### 3.1 연간 기부 한도

[정치자금법]에 따른 개인 기부 한도:

| 대상 | 연간 한도 |
|------|-----------|
| 정당 (당비) | 제한 없음 |
| 후원회 1개당 | **연 500만원** |
| 후원회 전체 | **연 2,000만원** |
| 선거비용 기탁 | **선거비용 한도 내** |

### 3.2 세액공제 한도

| 구분 | 한도 |
|------|------|
| 근로소득금액 | 100% 한도 |

※ 정치자금 기부금은 소득금액의 100%까지 공제 가능

## 4. 공제 계산 예시

### 4.1 10만원 기부

**정치후원금 10만원 기부**

- 공제율: 100/110
- 세액공제액: 10만원 × 100/110 = **9만 909원**
- 실질 부담: 9,091원

### 4.2 50만원 기부

**정치후원금 50만원 기부**

| 구간 | 금액 | 공제율 | 공제액 |
|------|------|--------|--------|
| 10만원 이하 | 10만원 | 100/110 | 9만 909원 |
| 10만원 초과 | 40만원 | 15% | 6만원 |
| **합계** | | | **15만 909원** |

### 4.3 100만원 기부

**정치후원금 100만원 기부**

| 구간 | 금액 | 공제율 | 공제액 |
|------|------|--------|--------|
| 10만원 이하 | 10만원 | 100/110 | 9만 909원 |
| 10만원 초과 | 90만원 | 15% | 13.5만원 |
| **합계** | | | **22만 5,909원** |

## 5. 정치자금 유형별 안내

### 5.1 정당 당비

| 항목 | 내용 |
|------|------|
| 납부처 | 가입한 정당 |
| 공제 대상 | 전액 공제 |
| 증빙 | 정당 납부 영수증 |

### 5.2 후원회 후원금

| 항목 | 내용 |
|------|------|
| 대상 | 국회의원, 지방의원, 당대표 등 |
| 납부처 | 각 정치인 후원회 |
| 한도 | 1인당 500만원, 전체 2,000만원 |

### 5.3 선거관리위원회 기탁금

| 항목 | 내용 |
|------|------|
| 납부처 | 중앙선거관리위원회 |
| 배분 | 정당에 지지율 비례 배분 |
| 용도 | 특정 정당/후보 지정 불가 |

## 6. 증빙서류

### 6.1 연말정산 간소화

대부분 **자동 조회** 가능 (1월 15일부터)

### 6.2 별도 제출 필요 시

| 서류 | 발급처 |
|------|--------|
| 정치자금 기부금 영수증 | 정당, 후원회 |
| 기탁금 영수증 | 선거관리위원회 |

## 7. 다른 기부금과 비교

### 7.1 기부금 공제율 비교

| 유형 | 10만원 이하 | 초과분 |
|------|------------|--------|
| **정치자금** | **100/110** | 15~25% |
| 법정기부금 | 15~25% | 15~25% |
| 지정기부금 | 15~25% | 15~25% |
| 종교단체 | 15~25% | 15~25% |

### 7.2 정치자금의 장점

- **10만원까지 거의 전액 환급**
- 정치 참여와 절세를 동시에
- 소득금액 100% 한도 적용

## 8. 주의사항

### 8.1 불법 정치자금

| 상황 | 결과 |
|------|------|
| 불법 정치자금 | 공제 불가 + 처벌 가능 |
| 대리 기부 | 공제 불가 |
| 기부금 영수증 허위 발급 | 처벌 대상 |

### 8.2 증빙 보관

- 기부금 영수증 **5년** 보관 권장
- 경정청구 시 증빙 필요

### 8.3 맞벌이 부부

- 각자 기부하면 **각자 공제**
- 한 사람이 10만원 기부: 9만원 환급
- 부부 각각 10만원: 18만원 환급

## 9. 절세 전략

### 9.1 최적 기부 금액

**최소 부담으로 최대 효과**: 10만원 기부

- 부담: 약 9,000원
- 효과: 정치 참여 + 절세

### 9.2 고액 기부 시

50만원 기부 시 세액공제: 15만원
→ 실질 부담 35만원으로 정치 참여
