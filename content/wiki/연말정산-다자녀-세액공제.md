# 연말정산 다자녀 세액공제

다자녀 가구의 세액공제 혜택을 정리합니다.

## 1. 공제 금액

| 자녀 수 | 공제액 |
|---------|--------|
| 1명 | 15만원 |
| 2명 | 35만원 |
| 3명 | 65만원 |
| 4명 | 95만원 |
| 5명 | 125만원 |

## 2. 계산 공식

```
3명 이상: 35만원 + 30만원 × (자녀수 - 2)
```

## 3. 대상 요건

| 요건 | 기준 |
|------|------|
| 나이 | 만 7세 이상 ~ 만 18세 이하 |
| 소득 | 연 100만원 이하 |
| 기준일 | 12월 31일 |
