"갱신하는데 부동산에서 복비 내라고 해요. 내야 해요?"

부동산 안 거쳤으면 낼 이유 없어요.

## 계약갱신 시 복비(중개수수료)

### 기본 원칙

복비는 **중개 서비스에 대한 대가**예요.

| 상황 | 복비 |
|------|------|
| 부동산 통해서 갱신 | O |
| 당사자끼리 갱신 | X |

부동산이 중개해줬으면 복비 내야 해요. 집주인이랑 세입자가 직접 갱신하면 복비 없어요.

### 계약갱신청구권 행사하면

[계약갱신청구권](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-행사방법)은 세입자가 집주인한테 **직접** 행사하는 거예요.

"저 2년 더 살게요" 문자 보내면 끝이에요. 부동산 필요 없어요.

부동산 안 거쳤으니 **복비 없어요.**

## 부동산에서 복비 요구하면

### 갱신 중개를 의뢰했나요?

"부동산에서 갱신 처리해달라고 부탁했어요?"

**의뢰 안 했으면** 복비 낼 이유 없어요.

처음 계약할 때 복비 냈잖아요. 갱신은 별개 계약이에요. 갱신 중개를 따로 의뢰한 게 아니면 복비 안 내도 돼요.

### 부동산이 연락해왔다면

"부동산에서 먼저 연락해서 갱신 도와준다고 했어요."

요청하지 않은 서비스예요. 복비 낼 의무 없어요.

> "저는 갱신 중개를 의뢰한 적 없어요."

이렇게 말하면 돼요.

## 복비 내야 하는 경우

### 부동산 통해서 조건 협의

"보증금 얼마 올릴지 부동산이 중간에서 협의해줬어요."

이러면 중개 서비스 받은 거예요. 복비 나올 수 있어요.

| 서비스 | 복비 |
|--------|------|
| 조건 협의 대행 | O |
| 계약서 작성 대행 | O |
| 단순 갱신 통보 | X |

### 재계약인 경우

[재계약](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-재계약)은 새로운 계약이에요.

신규 계약처럼 부동산 통하면 복비 나와요. 갱신이랑 재계약은 달라요.

| 구분 | 계약갱신청구권 | 재계약 |
|------|---------------|--------|
| 성격 | 기존 계약 연장 | 새 계약 |
| 5% 상한 | O | X |
| 복비 | 보통 없음 | 있을 수 있음 |

## 갱신 복비 얼마나 되나요

만약 부동산 통해서 갱신한다면요.

### 복비 요율

| 거래금액 | 상한 요율 |
|----------|----------|
| 5천만원 미만 | 0.5% |
| 5천만원~1억 미만 | 0.4% |
| 1억~6억 미만 | 0.3% |
| 6억~12억 미만 | 0.4% |
| 12억~15억 미만 | 0.5% |
| 15억 이상 | 0.6% |

예를 들어 보증금 3억 전세 갱신이면요.

3억 × 0.3% = 90만원 (상한)

실제로는 협의 가능해요.

### 월세 환산

월세 있으면 환산해서 계산해요.

> 거래금액 = 보증금 + (월세 × 100)

보증금 1,000만원 / 월세 50만원이면요.
1,000만원 + (50만원 × 100) = 6,000만원

6,000만원 기준으로 복비 계산해요.

## 복비 아끼는 방법

### 직접 갱신하기

집주인한테 직접 연락하세요.

1. 문자나 카톡으로 갱신 의사 전달
2. 인상 금액 협의 (5% 이내)
3. 계약서 수정 또는 새로 작성

이러면 복비 0원이에요.

### 계약서 작성만 부탁하기

"협의는 다 됐는데 계약서만 작성해주세요."

이러면 복비 대신 **계약서 작성 비용**만 낼 수도 있어요. 보통 10~20만원 정도. 부동산마다 달라요.

## 자주 묻는 질문

### 집주인이 부동산 통하자고 하면요?

세입자 입장에서는 직접 해도 돼요. 집주인이 부동산 통하고 싶으면 집주인이 복비 내면 돼요.

강제로 세입자한테 복비 부담시킬 수 없어요.

### 갱신 계약서 꼭 써야 해요?

법적으로 필수는 아니에요. 근데 나중 분쟁 대비해서 **서면으로 남기는 게 좋아요.**

기존 계약서에 "갱신 합의" 내용 추가하거나, 새 계약서 작성하거나요.

### 확정일자 다시 받아야 해요?

갱신이면 기존 확정일자 유지돼요. 다시 안 받아도 돼요.

근데 조건이 바뀌었으면 (보증금 인상 등) 새로 받아두는 게 안전해요.

더 자세한 내용은 [공인중개사법](https://www.law.go.kr/법령/공인중개사법)에서 확인하세요.
