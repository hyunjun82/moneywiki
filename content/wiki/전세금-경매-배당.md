# 전세금 경매 배당

집이 경매로 넘어가면 보증금을 받기 위해 배당요구를 해야 합니다. 배당 순위와 절차를 정확히 알아야 보증금을 지킬 수 있습니다.

## 1. 경매 배당이란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 배당 | 경매대금을 채권자에게 분배 |
| 배당요구 | 배당 참여 신청 |
| 배당순위 | 배당 우선순위 |
| 배당표 | 분배 계획표 |

### 1.2 임차인 관련 용어

| 용어 | 설명 |
|------|------|
| 대항력 | 제3자에게 임대차 주장 |
| 우선변제권 | 다른 채권자보다 먼저 변제 |
| 최우선변제권 | 소액임차인 최우선 배당 |
| 배당요구 종기 | 배당요구 마감일 |

## 2. 배당 순위

### 2.1 기본 배당 순서

| 순위 | 채권 |
|------|------|
| 1순위 | 경매비용 |
| 2순위 | 소액임차인 최우선변제금 |
| 3순위 | 당해세 (재산세, 종부세) |
| 4순위 | 선순위 담보권, 임차권 |
| 5순위 | 일반채권 |

### 2.2 임차인 배당 순위

| 상황 | 순위 |
|------|------|
| 소액임차인 (대항력) | **최우선변제** |
| 근저당보다 선순위 | **담보권과 동순위** |
| 근저당보다 후순위 | 일반채권 순위 |

### 2.3 순위 결정 기준

| 기준 | 비교 |
|------|------|
| 대항력 취득일 | 전입신고 다음날 0시 |
| 확정일자 | 확정일자 받은 날 |
| 근저당 설정일 | 등기부등본 확인 |

## 3. 소액임차인 최우선변제

### 3.1 2026년 기준 (지역별)

| 지역 | 보증금 기준 | 최우선변제금 |
|------|-----------|------------|
| 서울 | **1.65억 이하** | **5,500만원** |
| 수도권 과밀억제권역 | **1.45억 이하** | **4,800만원** |
| 광역시, 세종, 안산, 김포 등 | **8,500만원 이하** | **2,800만원** |
| 그 외 지역 | **7,500만원 이하** | **2,500만원** |

### 3.2 최우선변제 요건

| 요건 | 내용 |
|------|------|
| 보증금 | 지역별 기준 이하 |
| 대항력 | 전입신고 + 점유 |
| 시기 | 경매개시결정 등기 전 대항력 |
| 배당요구 | 배당요구 종기 전 신청 |

### 3.3 주의사항

| 주의 | 내용 |
|------|------|
| 주택가액 1/2 한도 | 전체 1/2 범위 내 |
| 복수 임차인 | 안분 배당 |
| 선순위 확인 | 대항력 취득 시점 중요 |

## 4. 배당요구 절차

### 4.1 배당요구 방법

| 방법 | 내용 |
|------|------|
| 제출처 | 관할 법원 경매계 |
| 서류 | 배당요구서, 증빙서류 |
| 기한 | 배당요구 종기 전 |

### 4.2 필요 서류

| 서류 | 용도 |
|------|------|
| 배당요구서 | 신청서 (법원 양식) |
| 임대차계약서 | 계약 증명 |
| 주민등록등본 | 전입일자 확인 |
| 확정일자 증명 | 우선변제권 증명 |

### 4.3 배당요구 종기

| 항목 | 내용 |
|------|------|
| 결정 | 법원이 공고 |
| 확인 | 대법원 경매정보 사이트 |
| 주의 | **종기 후 요구 불가** |

### 4.4 온라인 확인

| 사이트 | 주소 |
|--------|------|
| 대법원 경매정보 | courtauction.go.kr |
| 확인 내용 | 경매 일정, 배당요구 종기 |

|------|------|
| 정의 | 배당 계획 문서 |

| 열람 | 배당기일 전 열람 가능 |

### 5.2 배당표 확인

| 순위 | 본인 배당 순위 |

### 5.3 배당기일

| 절차 | 내용 |
|------|------|
| 출석 | 배당기일에 출석 |

| 수령 | 배당금 수령 |

## 6. 배당 이의

### 6.1 이의 사유

| 사유 | 예시 |
|------|------|
| 순위 오류 | 선순위인데 후순위로 기재 |
| 금액 오류 | 채권액 계산 착오 |
| 채권 부인 | 허위 채권 주장 |

### 6.2 이의 절차

| 절차 | 내용 |
|------|------|
| 이의 신청 | 배당기일에 이의 |
| 소송 제기 | 배당이의의 소 제기 |
| 기한 | 배당기일로부터 7일 |

### 6.3 배당이의의 소

| 항목 | 내용 |
|------|------|
| 관할 | 경매 법원 |
| 피고 | 이의 대상 채권자 |
| 소명 | 본인 주장 입증 |

## 7. 배당금 수령

### 7.1 수령 방법

| 방법 | 내용 |
|------|------|
| 출석 | 배당기일에 직접 수령 |
| 계좌 | 법원에 계좌 신고 후 입금 |

### 7.2 필요 서류

| 서류 | 용도 |
|------|------|
| 신분증 | 본인 확인 |
| 통장 | 입금 계좌 |
| 위임장 | 대리인 수령 시 |

### 7.3 미수령 시

| 상황 | 조치 |
|------|------|
| 배당기일 불참 | 추후 공탁금 수령 |
| 공탁 | 법원 공탁소에서 수령 |

## 8. 낙찰 후 임차인

### 8.1 대항력 있는 임차인

|------|------|
| 선순위 | 낙찰자에게 대항 가능 |
| 후순위 | 배당으로 회수, 퇴거 |

### 8.2 선순위 임차인

|------|------|
| 거주 | 계약 기간까지 거주 |
| 보증금 | 낙찰자에게 청구 |
| 퇴거 협의 | 보상 협의 가능 |

### 8.3 후순위 임차인

| 조치 | 내용 |
|------|------|
| 배당 | 배당요구로 회수 |
| 부족분 | 전 임대인에게 청구 |
| 퇴거 | 인도명령 시 퇴거 |

## 10. 자주 묻는 질문

### 10.1 배당요구 안 하면?

| 답변 | 설명 |
|------|------|
| 배당 제외 | 배당받지 못함 |
| 구제 | 전 소유자에게 청구 (어려움) |

### 10.2 선순위인데 배당 부족하면?

| 답변 | 설명 |
|------|------|
| 가능 | 낙찰대금 부족 시 발생 |
| 대응 | 전 소유자에게 부족분 청구 |

### 10.3 경매 취소되면?

| 답변 | 설명 |
|------|------|
| 원상복귀 | 임대차관계 유지 |
| 보증금 | 기존대로 임대인에게 청구 |
