# 연말정산 교복 구입비

교복 구입비 교육비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 교육비 세액공제 |
| 공제율 | **15%** |
| 한도 | 1인당 **50만원** |

## 2. 공제 대상

| 대상 | 공제 |
|------|------|
| 초등학생 | ❌ |
| 중학생 | ✅ |
| 고등학생 | ✅ |

## 3. 공제 항목

| 항목 | 공제 |
|------|------|
| 교복 | ✅ |
| 체육복 | ✅ |
| 교복 부속품 | ✅ |

## 4. 공제 예시

| 교복비 | 한도 | 공제액(15%) |
|--------|------|-------------|
| 30만원 | 30만원 | 4.5만원 |
| 70만원 | 50만원 | 7.5만원 |

## 5. 필요서류

| 서류 | 발급처 |
|------|--------|
| 교복 구입영수증 | 교복 업체 |
| 간소화 자료 | 일부 반영 |
