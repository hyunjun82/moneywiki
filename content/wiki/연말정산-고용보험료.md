# 연말정산 고용보험료

고용보험료 소득공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 소득공제 |
| 공제율 | **100%** |
| 한도 | **없음** |

## 2. 고용보험료율

| 항목 | 요율 |
|------|------|
| 본인 부담 | **0.9%** |
| 회사 부담 | 1.15~1.75% |

## 3. 공제 예시

| 월급여 | 본인 부담 | 연간 공제 |
|--------|----------|----------|
| 300만원 | 2.7만원 | 32.4만원 |
| 400만원 | 3.6만원 | 43.2만원 |
| 500만원 | 4.5만원 | 54만원 |

## 4. 4대보험 공제 합계

| 보험 | 본인 부담률 |
|------|------------|
| 국민연금 | 4.5% |
| 건강보험 | 3.545% |
| 장기요양 | 0.46% |
| 고용보험 | 0.9% |
| **합계** | **약 9.4%** |

## 5. 자동 공제

| 항목 | 내용 |
|------|------|
| 간소화 자료 | 자동 반영 |
| 별도 신청 | 불필요 |
