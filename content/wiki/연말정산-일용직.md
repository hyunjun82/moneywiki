# 연말정산 일용직

일용직 근로자 세금을 정리합니다.

## 1. 일용직 과세 방식

| 항목 | 내용 |
|------|------|
| 과세 방식 | 원천징수 |
| 연말정산 | ❌ 없음 |
| 과세 종결 | ✅ 원천징수로 종결 |

## 2. 세금 계산

| 항목 | 기준 |
|------|------|
| 일당 공제 | 15만원 |
| 과세 대상 | 15만원 초과분 |
| 세율 | 6% × 55% = **2.7%** |

## 3. 계산 예시

| 일당 | 과세분 | 세금 |
|------|--------|------|
| 15만원 | 0 | 0원 |
| 20만원 | 5만원 | 1,350원 |
| 30만원 | 15만원 | 4,050원 |

## 4. 일용직 기준

| 요건 | 기준 |
|------|------|
| 근로기간 | 3개월 미만 |
| 고용 형태 | 일정 장소에서 계속 근무 아님 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 3개월 이상 | 일반 근로소득 |
| 상용직 전환 | 연말정산 대상 |
