# 노인장기요양보험

노인장기요양보험은 고령이나 노인성 질환으로 일상생활이 어려운 분께 요양 서비스를 지원하는 제도입니다.

## 1. 제도 개요

### 1.1 대상

| 대상 | 조건 |
|------|------|
| 65세 이상 | 장기요양이 필요한 경우 |
| 65세 미만 | 노인성 질환 (치매, 뇌혈관 등) |

### 1.2 노인성 질환

| 질환 | 예시 |
|------|------|
| 치매 | 알츠하이머, 혈관성 치매 |
| 뇌혈관질환 | 뇌졸중, 뇌경색 |
| 파킨슨병 | 파킨슨 관련 질환 |

### 1.3 보험료

| 항목 | 요율 |
|------|------|
| 장기요양보험료 | 건강보험료의 **12.95%** |
| 부담 | 근로자·사업주 각 50% |

## 2. 장기요양등급

### 2.1 등급 기준

| 등급 | 장기요양인정점수 | 상태 |
|------|----------------|------|
| 1등급 | 95점 이상 | 전적 도움 필요 |
| 2등급 | 75~94점 | 상당 부분 도움 |
| 3등급 | 60~74점 | 부분적 도움 |
| 4등급 | 51~59점 | 일정 부분 도움 |
| 5등급 | 45~50점 | 치매 (경증) |
| 인지지원등급 | 45점 미만 | 치매 (경증) |

### 2.2 등급별 특징

| 등급 | 대상 |
|------|------|
| 1~2등급 | 거동 불편, 전적 도움 필요 |
| 3~4등급 | 일부 도움 필요 |
| 5등급 | 치매 환자 |
| 인지지원등급 | 경증 치매 |

## 3. 등급 신청

### 3.1 신청 자격

| 자격 | 조건 |
|------|------|
| 65세 이상 | 건강보험 가입자 |
| 65세 미만 | 노인성 질환자 |

### 3.2 신청 방법

| 방법 | 내용 |
|------|------|
| 방문 | 국민건강보험공단 지사 |
| 온라인 | 장기요양보험 홈페이지 |
| 우편·팩스 | 신청서 발송 |

### 3.3 필요 서류

| 서류 | 용도 |
|------|------|
| 장기요양인정 신청서 | 기본 서류 |
| 의사 소견서 | 건강 상태 확인 |
| 신분증 | 본인 확인 |

### 3.4 판정 절차

| 단계 | 내용 | 기간 |
|------|------|------|
| 1 | 신청 접수 | - |
| 2 | 방문조사 | 신청 후 |
| 3 | 등급판정위원회 | 조사 후 |
| 4 | 결과 통보 | **30일 이내** |

## 4. 서비스 종류

### 4.1 재가급여 (집에서)

| 서비스 | 내용 |
|--------|------|
| 방문요양 | 요양보호사 가정 방문 |
| 방문목욕 | 목욕 서비스 |
| 방문간호 | 간호사 방문 간호 |
| 주야간보호 | 주간·야간 시설 이용 |
| 단기보호 | 일시적 시설 보호 |
| 복지용구 | 휠체어, 침대 등 대여·구입 |

### 4.2 시설급여 (시설에서)

| 시설 | 대상 |
|------|------|
| 노인요양시설 | 1~2등급 (주로) |
| 노인요양공동생활가정 | 1~4등급 |

### 4.3 특별현금급여

| 급여 | 내용 |
|------|------|
| 가족요양비 | 가족이 요양 시 지급 |
| 특례요양비 | 시설 부족 지역 |

## 5. 급여 한도

### 5.1 월 한도액 (2026년)

| 등급 | 월 한도액 |
|------|----------|
| 1등급 | 약 204만원 |
| 2등급 | 약 182만원 |
| 3등급 | 약 141만원 |
| 4등급 | 약 130만원 |
| 5등급 | 약 108만원 |
| 인지지원등급 | 약 62만원 |

### 5.2 초과 시

| 상황 | 부담 |
|------|------|
| 한도 초과 | **전액 본인 부담** |

## 6. 본인부담금

### 6.1 일반 본인부담

| 급여 유형 | 본인부담률 |
|----------|----------|
| 재가급여 | **15%** |
| 시설급여 | **20%** |

### 6.2 감경 대상

| 대상 | 본인부담률 |
|------|----------|
| 기초생활수급자 | **면제** |
| 의료급여 수급자 | 면제~7.5% |
| 차상위 계층 | 6~9% |
| 천재지변 피해 | 감경 |

### 6.3 비급여 항목

| 항목 | 부담 |
|------|------|
| 식사 재료비 | 본인 부담 |
| 이미용비 | 본인 부담 |
| 상급 침실 | 본인 부담 |

## 7. 복지용구

### 7.1 대여 품목

| 품목 | 예시 |
|------|------|
| 수동휠체어 | 일반형, 리클라이닝형 |
| 전동침대 | 욕창예방 매트리스 포함 |
| 이동욕조 | 목욕 보조 |
| 욕창예방방석 | 에어쿠션 등 |

### 7.2 구입 품목

| 품목 | 예시 |
|------|------|
| 이동변기 | 좌변기, 간이변기 |
| 목욕의자 | 샤워의자 |
| 보행기 | 보행 보조 |
| 안전손잡이 | 화장실 손잡이 |

### 7.3 연간 한도

| 항목 | 한도 |
|------|------|
| 구입 품목 | 연 **160만원** |
| 대여 품목 | 월 한도 별도 |

## 8. 요양보호사

### 8.1 서비스 내용

| 서비스 | 내용 |
|--------|------|
| 신체활동 | 세면, 식사, 배설 보조 |
| 일상생활 | 취사, 청소, 세탁 |
| 정서지원 | 말벗, 외출 동행 |
| 인지활동 | 치매예방 활동 |

### 8.2 이용 시간

| 등급 | 권장 시간 |
|------|----------|
| 1등급 | 하루 4시간 이상 |
| 2등급 | 하루 3~4시간 |
| 3~4등급 | 하루 2~3시간 |

## 9. 주의사항

### 9.1 등급 갱신

| 항목 | 내용 |
|------|------|
| 유효기간 | 1~2년 (등급별 상이) |
| 갱신 신청 | 만료 **90일 전**부터 |

### 9.2 부정수급 주의

| 주의 | 내용 |
|------|------|
| 허위 신청 | 환수 및 제재 |
| 부당 청구 | 급여 중지 |
