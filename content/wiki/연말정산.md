# 연말정산

연말정산은 근로자가 1년간 납부한 근로소득세를 정산하는 절차입니다. 쉽게 말해, 세금을 더 냈으면 돌려받고, 덜 냈으면 더 내는 과정입니다.

## 1. 연말정산이란?

### 1.1 기본 개념
- 매월 급여에서 **원천징수**된 세금을 연간 기준으로 다시 계산
- 각종 **소득공제**와 **세액공제**를 적용해 최종 세금 확정
- 차액을 **환급**받거나 **추가 납부**

### 1.2 왜 '13월의 월급'이라 부를까?
실제 납부해야 할 세금보다 더 많이 원천징수된 경우, 그 차액을 돌려받기 때문입니다. 공제 항목을 잘 챙기면 수십만 원에서 수백만 원까지 환급받을 수 있습니다.

## 2. 소득공제 vs 세액공제

### 2.1 소득공제
**과세표준**(세금을 계산하는 기준 금액)을 낮춰주는 방식

| 항목 | 한도 |
|------|------|
| 인적공제 | 본인·배우자·부양가족 1인당 150만원 |
| 신용카드 등 | 총급여 25% 초과분의 15~40% |
| 주택청약저축 | 연 240만원 한도 (40% 공제) |
| 주택담보대출 이자 | 연 300~1,800만원 |

### 2.2 세액공제
계산된 **세금에서 직접 차감**하는 방식 (더 직접적인 혜택)

| 항목 | 공제율/한도 |
|------|------------|
| 의료비 | 총급여 3% 초과분의 15% |
| 교육비 | 본인 전액, 자녀 1인당 연 300만원 |
| 기부금 | 15~30% |
| 연금저축 | 납입액의 12~15% (최대 400만원) |
| 월세 | 10~12% (최대 750만원) |

## 3. 주요 공제 항목

### 3.1 신용카드 등 소득공제
- **공제 시작점**: 총급여의 25% 초과 사용분
- **공제율**: 신용카드 15%, 체크카드/현금영수증 30%, 전통시장 40%
- **한도**: 총급여별 200~300만원

### 3.2 의료비 세액공제
- 총급여의 3% 초과분에 대해 15% 공제
- 난임시술비, 미숙아 등은 20% 공제
- 안경·콘택트렌즈(50만원 한도), 보청기 포함

### 3.3 교육비 세액공제
- 본인: 대학원 등록금 전액
- 초·중·고: 1인당 연 300만원
- 대학생: 1인당 연 900만원
- 학원비(취학 전 아동만), 교복구입비(50만원) 포함

### 3.4 주거 관련 공제
- **주택청약저축**: 무주택 세대주, 연 240만원 한도의 40%
- **주택담보대출 이자**: 무주택/1주택자, 상환기간별 다름
- **월세 세액공제**: 무주택 세대주, 총급여 7천만원 이하

## 4. 연말정산 일정

| 시기 | 내용 |
|------|------|
| 1월 15일경 | 홈택스 간소화서비스 오픈 |
| 1월 중순~2월 | 회사에 서류 제출 |
| 2~3월 | 환급/추가납부 반영 |

## 5. 절세 팁

1. **체크카드와 현금영수증** 적극 활용 (공제율 30%)
2. **IRP/연금저축** 활용 (세액공제 + 노후 대비)
3. **기부금** 영수증 꼼꼼히 챙기기
4. **맞벌이 부부**는 공제 항목 배분 전략 수립
