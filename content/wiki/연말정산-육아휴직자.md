# 연말정산 육아휴직자

육아휴직자 연말정산을 정리합니다.

## 1. 육아휴직급여 과세

| 항목 | 과세 |
|------|------|
| 육아휴직급여 | **비과세** |
| 출산휴가급여 | **비과세** |
| 회사 급여 | 과세 |

## 2. 연말정산 대상

| 대상 | 정산 |
|------|------|
| 휴직 전 급여 | ✅ 연말정산 |
| 육아휴직급여 | ❌ 제외 |

## 3. 배우자 공제

| 상황 | 공제 |
|------|------|
| 소득금액 100만원 이하 | 배우자가 공제 가능 |
| 소득금액 100만원 초과 | 공제 불가 |

## 4. 소득금액 계산

| 항목 | 계산 |
|------|------|
| 총급여 | 근로소득만 |
| 육아휴직급여 | 제외 |
| 소득금액 | 총급여 - 근로소득공제 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 신용카드 | 휴직 중 사용분도 공제 가능 |
| 의료비 | 본인 또는 배우자가 공제 |
| 보험료 | 계약자가 공제 |
