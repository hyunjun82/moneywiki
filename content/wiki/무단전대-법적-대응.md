# 무단전대 법적 대응

임대인 동의 없이 집을 다른 사람에게 빌려주면 무단전대입니다. 임대인과 전차인 모두에게 심각한 법적 문제가 생길 수 있습니다.

## 1. 무단전대란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 전대 | 임차인이 제3자에게 다시 임대 |
| 무단전대 | 임대인 동의 없이 전대 |
| 근거 | 민법 제629조 |

### 1.2 전대 vs 양도

| 구분 | 전대 | 양도 |
|------|------|------|
| 관계 | 3자 관계 유지 | 권리 이전 |
| 책임 | 원임차인도 책임 | 양수인만 책임 |
| 계약 | 전대차계약 추가 | 임차권 이전 |

### 1.3 무단전대 성립 요건

| 요건 | 내용 |
|------|------|
| 임대인 동의 없음 | 사전 동의 미취득 |
| 제3자 사용 | 임차인 외 다른 사람이 사용 |
| 대가 수수 | 금전 등 대가 지급 |

## 2. 무단전대 해당 여부

### 2.1 무단전대에 해당

| 상황 | 해당 |
|------|------|
| 돈 받고 다른 사람에게 빌려줌 | ✅ |
| 에어비앤비 운영 | ✅ |
| 월세 받고 타인 거주 | ✅ |

### 2.2 무단전대 아닌 경우

| 상황 | 해당 |
|------|------|
| 가족과 함께 거주 | ❌ |
| 일시적 손님 숙박 | ❌ |
| 무상으로 일시 거주 | ❌ (일반적으로) |

### 2.3 판단 기준

| 기준 | 내용 |
|------|------|
| 대가 | 금전 수수 여부 |
| 기간 | 일시적 vs 지속적 |
| 사용 형태 | 전체 vs 일부 |

## 3. 임대인 권리

### 3.1 계약 해지권

| 권리 | 내용 |
|------|------|
| 해지 사유 | 무단전대는 계약 해지 사유 |
| 해지 통지 | 해지 의사 통지 |
| 효력 | 통지 도달 시 해지 |

### 3.2 해지 절차

| 단계 | 내용 |
|------|------|
| 1 | 무단전대 확인 |
| 2 | 내용증명으로 해지 통지 |
| 3 | 원상복구 및 명도 요청 |
| 4 | 불응 시 명도소송 |

### 3.3 해지 통지 예시

```
[계약해지 통지서]

임차인 ○○○ 귀하

귀하가 임대인의 동의 없이 ○○시 ○○구 ○○동
○○호를 제3자에게 전대한 사실을 확인하였습니다.

이는 민법 제629조 위반으로, 본 통지로써
임대차계약을 해지합니다.

1. 해지 효력 발생일: 본 통지 도달일
2. 원상복구 및 명도: 14일 내 이행
3. 미이행 시 법적 조치 예정

2026년 ○월 ○일
임대인: ○○○ (인)
```

## 4. 임차인 책임

### 4.1 계약 해지

| 결과 | 내용 |
|------|------|
| 해지 | 임대인 해지 통지 시 계약 종료 |
| 명도 | 주택 반환 의무 |
| 보증금 | 정산 후 반환 |

### 4.2 손해배상

| 책임 | 내용 |
|------|------|
| 임대인에게 | 손해가 있으면 배상 |
| 전차인에게 | 전차인 손해 배상 |

### 4.3 전차인에 대한 책임

| 책임 | 내용 |
|------|------|
| 보증금 반환 | 전차인 보증금 반환 |
| 손해배상 | 이사비 등 손해 |

## 5. 전차인 지위

### 5.1 무단전대 시 전차인

| 상황 | 결과 |
|------|------|
| 대항력 | ❌ 임대인에게 대항 불가 |
| 보증금 | 원임차인에게 청구 |
| 퇴거 | 임대인 요구 시 퇴거 |

### 5.2 전차인 대응

| 대응 | 방법 |
|------|------|
| 원임차인 확인 | 계약 전 임대인 동의 확인 |
| 보증금 반환 | 원임차인에게 청구 |
| 법률 상담 | 손해배상 검토 |

### 5.3 전차인 보호 방법

| 방법 | 내용 |
|------|------|
| 사전 확인 | 임대인 동의서 확인 |
| 계약서 | 동의서 사본 첨부 |
| 등기 확인 | 등기부등본 확인 |

## 6. 동의 있는 전대

### 6.1 적법한 전대

| 요건 | 내용 |
|------|------|
| 임대인 동의 | 서면 동의 |
| 전대차계약 | 전대인과 전차인 계약 |
| 통지 | 전대 사실 통지 |

### 6.2 전차인 보호

| 보호 | 내용 |
|------|------|
| 대항력 | 원임차인 승계 |
| 우선변제권 | 확정일자 필요 |
| 계약 갱신 | 원임대차 범위 내 |

### 6.3 동의서 양식

```
[전대 동의서]

임대인 ○○○는 아래 부동산에 대하여
임차인 ○○○가 제3자에게 전대하는 것을
동의합니다.

1. 부동산 표시: ○○시 ○○구 ○○동 ○○호
2. 전대 허용 기간: 원임대차 기간 내
3. 전대 조건: 보증금 ○○원, 월세 ○○원 이내
4. 기타: 전차인 인적사항 사전 통보

2026년 ○월 ○일
임대인: ○○○ (인)
```

## 10. 자주 묻는 질문
