# 연말정산 세액감면

세액감면 종류를 정리합니다.

## 1. 세액감면 vs 세액공제

| 구분 | 세액감면 | 세액공제 |
|------|----------|----------|
| 방식 | 세금 자체 감면 | 세금에서 차감 |
| 효과 | 더 유리 | 일정 금액 차감 |

## 2. 감면 종류

| 대상 | 감면율 | 기간 |
|------|--------|------|
| 중소기업 청년 | **90%** | 5년 |
| 60세 이상 | **70%** | 3년 |
| 장애인 | **70%** | 3년 |
| 경력단절여성 | **70%** | 3년 |

## 3. 중소기업 청년

| 요건 | 기준 |
|------|------|
| 나이 | 만 15~34세 |
| 기업 | 중소기업 |
| 한도 | 연 200만원 |

## 4. 신청 방법

| 절차 | 내용 |
|------|------|
| 서류 | 감면신청서 |
| 제출처 | 회사 |
| 시기 | 취업 시 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 중복 | 감면 간 중복 불가 |
| 제외 업종 | 금융, 전문서비스 등 |
