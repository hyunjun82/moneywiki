2026년에 건강보험료가 얼마나 오르는지 궁금하셨죠?

**2026년 건강보험료율이 7.09%에서 7.19%로 0.1%p 인상됐어요. 월급 300만원 기준으로 본인 부담이 약 122,000원이에요.**

## 2026년 건강보험료 요약

| 구분 | 2025년 | 2026년 | 변동 |
|------|--------|--------|------|
| 건강보험료율 | 7.09% | **7.19%** | +0.1%p |
| 본인 부담 | 3.595% | **3.595%** | +0.05%p |
| 장기요양보험료율 | 13.14% | **13.14%** | +0.19%p |
| 보험료 상한액 | 9,008,340원 | **9,183,480원** | +175,140원 |

## 건강보험료 계산 방법

### 직장가입자

직장인은 회사와 반반 부담해요.

```
건강보험료 = 월급 × 7.19% ÷ 2
           = 월급 × 3.595%

장기요양보험료 = 건강보험료 × 13.14%

총 부담 = 건강보험료 + 장기요양보험료
```

## 장기요양보험료

### 장기요양보험이란?

65세 이상 노인이나 치매·중풍 등으로 일상생활이 어려운 분들을 위한 보험이에요.

### 장기요양보험료 계산

```
장기요양보험료 = 건강보험료 × 13.14%

예: 건강보험료 107,850원
   = 107,850원 × 13.14%
   = 약 14,170원
```

## 건강보험료 상한액

### 2026년 상한액

| 구분 | 금액 |
|------|------|
| 월 상한액 | **9,183,480원** |
| 연 상한액 | **110,201,760원** |

아무리 돈을 많이 벌어도 월 9,183,480원 이상은 안 내요.

## 지역가입자 건강보험료

### 계산 방법

지역가입자는 소득, 재산, 자동차를 합산해서 보험료를 내요.

```
보험료 = (소득점수 + 재산점수 + 자동차점수) × 점수당 금액
```

### 2026년 점수당 금액

| 연도 | 점수당 금액 |
|------|-----------|
| 2025년 | 208.4원 |
| **2026년** | **약 211원** (예상) |

### 지역가입자 보험료 예시

| 조건 | 월 보험료 (예상) |
|------|-----------------|
| 소득 300만원, 재산 1억 | 약 20~25만원 |
| 소득 200만원, 재산 5천만원 | 약 12~15만원 |
| 소득 100만원, 재산 3천만원 | 약 6~8만원 |

정확한 금액은 [건강보험공단](https://www.nhis.or.kr)에서 확인하세요.

## 건강보험 혜택

### 받을 수 있는 것

- **진료비 지원**: 병원비의 30~60%만 본인 부담
- **건강검진**: 2년마다 무료 검진
- **출산급여**: 출산 시 50만원 지원
- **요양급여**: 입원·외래 진료비 지원

### 본인부담금

| 구분 | 본인 부담 |
|------|----------|
| 의원 | 30% |
| 병원 | 40% |
| 종합병원 | 50% |
| 상급종합병원 | 60% |

## 피부양자 자격

### 피부양자란?

직장가입자의 가족 중 별도 보험료 없이 건강보험 혜택을 받는 사람이에요.

### 피부양자 조건

| 조건 | 기준 |
|------|------|
| 소득 | 연 소득 2,000만원 이하 |
| 재산 | 재산세 과표 5.4억원 이하 |
| 관계 | 배우자, 직계존비속, 형제자매 |

소득·재산 기준을 초과하면 지역가입자로 전환돼요.

## 건강보험료 조회 방법

### 온라인 조회

1. [국민건강보험공단](https://www.nhis.or.kr) 접속
2. '개인민원' → '보험료 조회' 클릭
3. 공동인증서/간편인증 로그인
4. 보험료 확인

### 고객센터

- **전화**: 1577-1000
- **영업시간**: 평일 09:00~18:00
