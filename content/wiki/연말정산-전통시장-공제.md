# 연말정산 전통시장 공제

전통시장 사용금액은 **40% 공제율**로 가장 높은 혜택을 받습니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 공제 조건을 정리합니다.

## 1. 전통시장 공제율

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제율 비교입니다.

| 사용처 | 공제율 | 비고 |
|--------|--------|------|
| **전통시장** | **40%** | 최고 공제율 |
| 대중교통 | 40% | 전통시장과 동일 |
| 체크카드/현금영수증 | 30% | |
| 신용카드 | 15% | 기본 |

**결제 수단 무관**: 전통시장에서는 신용카드, 체크카드, 현금영수증 모두 **40%** 적용

## 2. 공제 한도

### 2.1 기본 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

### 2.2 추가 한도 (전통시장 전용)

기본 한도를 다 채운 후에도 **추가 공제** 가능:

| 항목 | 추가 한도 |
|------|-----------|
| 전통시장 | **100만원** |
| 대중교통 | 100만원 |
| 도서·공연·영화 (7천만원 이하) | 100만원 |

**최대 공제**: 기본 300만원 + 추가 300만원 = **600만원**

## 3. 공제 조건

### 3.1 최저 사용 금액

**총급여의 25% 초과 사용분**부터 공제 적용

| 총급여 | 25% 기준액 |
|--------|------------|
| 4,000만원 | 1,000만원 초과분부터 |
| 5,000만원 | 1,250만원 초과분부터 |
| 6,000만원 | 1,500만원 초과분부터 |

### 3.2 전통시장 정의

[전통시장 및 상점가 육성을 위한 특별법]에 따라 등록된 시장

## 4. 전통시장 확인 방법

### 4.1 홈택스 조회

1. [홈택스](https://www.hometax.go.kr) 접속
2. 상담·불복·제보 → 기타
3. **전통시장 정보 조회**
4. 시장명 또는 사업자번호로 검색

### 4.2 연말정산 간소화

1월 15일부터 간소화 서비스에서 **전통시장 사용금액 자동 구분**

### 4.3 누락 시 문의

전통시장인데 조회 안 되면:
- 해당 지방자치단체
- 전통시장 관할 세무서

## 5. 공제 계산 예시

**연봉 5,000만원, 전통시장 500만원 사용한 경우**

1. 25% 기준액: 1,250만원
2. 전체 카드 사용액이 1,750만원이라면:
   - 공제 대상: 500만원 (1,750만원 - 1,250만원)
3. 전통시장 500만원 × 40% = **200만원 소득공제**

## 6. 절세 전략

### 6.1 전통시장 적극 활용

같은 금액 사용 시 공제 효과:
- 전통시장: 100만원 × 40% = **40만원**
- 신용카드: 100만원 × 15% = 15만원
- **차이: 25만원**

### 6.2 추가 한도 활용

기본 한도(300만원)를 신용카드로 채운 후:
- 전통시장에서 추가 사용 → **별도 100만원 한도** 적용

### 6.3 전통시장 찾기

- 전통시장, 재래시장, 골목상권 포함
- 대형마트 내 전통시장 입점 매장도 해당될 수 있음

## 7. 주의사항

### 7.1 일반 매장과 구분

모든 재래시장이 전통시장은 아님
→ 반드시 **홈택스에서 확인** 필요

### 7.2 영수증 보관

전통시장 확인이 어려운 경우를 대비해 **영수증 보관** 권장
