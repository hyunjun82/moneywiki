# 임대차계약 해제 해지 차이

해제와 해지, 비슷해 보이지만 법적 효력이 달라요. 언제 뭘 쓰는지 정확히 알아야 해요.

## 핵심 차이

| 구분 | 해제 | 해지 |
|------|------|------|
| 효력 | 소급 (처음부터 없던 것) | 장래 (앞으로만) |
| 원상회복 | 전부 반환 | 기간분 정산 |
| 적용 | 일시적 계약 (매매 등) | 계속적 계약 (임대차 등) |
| 근거 | [민법 제543조](https://www.law.go.kr/법령/민법) | [민법 제550조](https://www.law.go.kr/법령/민법) |

**쉽게 말해**: 해제는 "처음부터 없던 일", 해지는 "지금부터 끝".

## 언제 해제, 언제 해지?

### 해제가 적용되는 경우

| 시기 | 상황 |
|------|------|
| 잔금 전 | 계약금만 지급한 상태 |
| 입주 전 | 아직 이사 안 함 |
| 이행 전 | 계약 이행 시작 전 |

### 해지가 적용되는 경우

| 시기 | 상황 |
|------|------|
| 입주 후 | 실제 거주 중 |
| 이행 중 | 월세 납부 등 진행 중 |
| 기간 중 | 계약 기간 중 종료 |

**핵심**: 이행 시작 전 = 해제, 이행 진행 중 = 해지

## 잔금 전 해제

잔금 지급 전이면 계약금만으로 해제할 수 있어요.

| 당사자 | 방법 |
|--------|------|
| 임차인 | 계약금 포기 |
| 임대인 | 계약금 배액 배상 |

### 이행 착수 여부

| 상황 | 해제 가능 |
|------|----------|
| 잔금 준비만 |  가능 |
| 중도금 지급 | ️ 제한적 |
| 인도, [전입신고](/w/전입신고-방법) |  불가 |

중도금 이상 지급하거나 [전입신고](/w/전입신고-방법)하면 이행 착수로 봐서 해제가 어려워요.

## 입주 후 해지

입주 후엔 해지를 해야 해요. 해지해도 이전 기간은 유효하게 남아요.

### 법정 해지 사유

| 사유 | 해지 권리자 |
|------|------------|
| 2기 월세 연체 | 임대인 |
| [무단전대](/w/무단전대-법적-대응) | 임대인 |
| 목적물 하자 | 임차인 |
| 수리의무 불이행 | 임차인 |

### 해지 절차

| 단계 | 내용 |
|------|------|
| 1 | 해지 통지 (내용증명 권장) |
| 2 | 상대방 수령 |
| 3 | 해지 효력 발생 |
| 4 | 원상회복, 보증금 정산 |

## 묵시적 갱신 후 해지

[묵시적갱신](/w/묵시적갱신) 후에는 언제든 해지할 수 있어요.

### 임차인 해지

| 항목 | 내용 |
|------|------|
| 통지 | 해지 의사 통지 |
| 효력 | 통지 후 **3개월** 경과 시 |

### 임대인 해지

| 항목 | 내용 |
|------|------|
| 제한 | 법정 해지 사유 필요 |
| 사유 | 2기 연체, [무단전대](/w/무단전대-법적-대응) 등 |

## 보증금 처리

### 해제 시
계약금만 처리하면 돼요. 보증금 전액 반환.

### 해지 시
정산 후 반환해요.

| 공제 항목 | 설명 |
|----------|------|
| 연체 월세 | 미납분 |
| 원상복구비 | 임차인 과실분 |
| 공과금 | 미납분 |

자세한 건 [보증금 일부반환](/w/보증금-일부반환) 참고하세요.

## 위약금

### 해제 시

| 상황 | 위약금 |
|------|--------|
| 계약금 해제 | 계약금 상당 |
| 약정 위약금 | 약정 금액 |

### 해지 시

| 상황 | 위약금 |
|------|--------|
| 합의 해지 | 합의 금액 |
| 약정 해지 | 특약 금액 |
| 법정 해지 | 손해배상 |

## 자주 묻는 질문

### 해제와 해지 잘못 쓰면?
실제 상황에 맞게 해석해요. 법적으로 큰 문제는 없어요.

### 법정해지 사유 없으면?
상대방 동의(합의해지) 필요해요. 위약금이 발생할 수 있어요.

### 해지 후 낸 월세는?
사용 기간만큼 공제하고 미사용분은 반환받아요.
