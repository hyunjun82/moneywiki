# 임대차계약 해제 해지 차이

해제와 해지는 비슷하지만 법적 효력이 다릅니다. 임대차계약에서는 주로 해지를 사용합니다.

## 1. 해제와 해지 기본 개념

### 1.1 계약 해제

| 항목 | 내용 |
|------|------|
| 정의 | 계약을 처음부터 없던 것으로 |
| 효력 | **소급적 효력** (과거로 소급) |
| 근거 | 민법 제543조 |
| 적용 | 일시적 계약 (매매 등) |

### 1.2 계약 해지

| 항목 | 내용 |
|------|------|
| 정의 | 장래를 향해 계약 종료 |
| 효력 | **장래적 효력** (앞으로만) |
| 근거 | 민법 제550조 |
| 적용 | 계속적 계약 (임대차 등) |

### 1.3 핵심 차이

| 구분 | 해제 | 해지 |
|------|------|------|
| 효력 | 소급 (처음부터) | 장래 (앞으로) |
| 원상회복 | 전부 반환 | 기간분 정산 |
| 적용 계약 | 일시적 | 계속적 |

## 2. 임대차에서의 적용

### 2.1 해제가 적용되는 경우

| 시기 | 상황 |
|------|------|
| 잔금 전 | 계약금만 지급 상태 |
| 입주 전 | 인도 전 상태 |
| 이행 전 | 계약 이행 시작 전 |

### 2.2 해지가 적용되는 경우

| 시기 | 상황 |
|------|------|
| 입주 후 | 실제 거주 중 |
| 계약 진행 중 | 월세 납부 등 이행 중 |
| 기간 중 | 계약 기간 중 종료 |

### 2.3 판단 기준

| 기준 | 적용 |
|------|------|
| 이행 시작 전 | 해제 |
| 이행 진행 중 | 해지 |

## 3. 해제의 효과

### 3.1 법적 효과

| 효과 | 내용 |
|------|------|
| 소급 효력 | 계약 처음부터 무효 |
| 원상회복 | 받은 것 반환 |
| 손해배상 | 상대방 손해 배상 |

### 3.2 원상회복 의무

| 항목 | 반환 |
|------|------|
| 계약금 | 받은 쪽이 반환 |
| 보증금 | 받은 쪽이 반환 |
| 사용이익 | 정산 가능 |

### 3.3 해제 방법

| 유형 | 방법 |
|------|------|
| 합의해제 | 당사자 합의 |
| 약정해제 | 계약서 해제조항 |
| 법정해제 | 채무불이행 등 |

## 4. 해지의 효과

### 4.1 법적 효과

| 효과 | 내용 |
|------|------|
| 장래 효력 | 해지 시점부터 종료 |
| 기존 유효 | 이전 기간은 유효 |
| 정산 | 미납분, 원상복구 정산 |

### 4.2 정산 범위

| 항목 | 정산 |
|------|------|
| 보증금 | 공제 후 반환 |
| 연체 월세 | 공제 |
| 원상복구 | 필요시 공제 |
| 공과금 | 미납분 공제 |

### 4.3 해지 방법

| 유형 | 방법 |
|------|------|
| 합의해지 | 당사자 합의 |
| 약정해지 | 계약서 해지조항 |
| 법정해지 | 2기 연체 등 |

## 5. 잔금 전 해제

### 5.1 계약금만 지급 시

| 권리 | 내용 |
|------|------|
| 계약금 해제 | 계약금 포기/배액배상 |
| 기한 | 이행에 착수하기 전 |

### 5.2 해제 방법

| 당사자 | 방법 |
|--------|------|
| 매수인 | 계약금 포기 |
| 매도인 | 계약금 배액 배상 |

### 5.3 이행 착수

| 상황 | 해제 가능 |
|------|----------|
| 잔금 준비만 | ✅ 가능 |
| 중도금 지급 | ⚠️ 제한적 |
| 인도, 전입 | ❌ 불가 |

## 6. 입주 후 해지

### 6.1 법정 해지 사유

| 사유 | 권리자 |
|------|--------|
| 2기 월세 연체 | 임대인 |
| 무단 전대 | 임대인 |
| 목적물 하자 | 임차인 |
| 수리의무 불이행 | 임차인 |

### 6.2 해지 절차

| 단계 | 내용 |
|------|------|
| 1 | 해지 통지 (내용증명) |
| 2 | 상대방 수령 |
| 3 | 해지 효력 발생 |
| 4 | 원상회복, 정산 |

### 6.3 해지 통지 효력

| 방식 | 효력 |
|------|------|
| 내용증명 | 도달 시 효력 |
| 문자 | 도달 시 효력 |
| 구두 | 증거 확보 어려움 |
