# 전자기부금 영수증

## 기본 개념
국세청 연말정산 간소화 서비스에서 자동 조회되는 전자 형태의 기부금 영수증입니다.

## 전자기부금 영수증 제도

### 도입 목적
- 납세자 편의 증진
- 영수증 분실 방지
- 허위 공제 방지
- 행정 효율성 향상

### 적용 대상
- 법정기부금
- 지정기부금
- 우리사주조합 기부금
- 종교단체 기부금

## 조회 방법

### 국세청 홈택스
1. 홈택스 접속 (www.hometax.go.kr)
2. 로그인 (공동인증서, 간편인증)
3. 연말정산 간소화 서비스
4. 기부금 항목 선택
5. 자동 조회 및 다운로드

### 조회 기간
- **매년 1월 15일부터** 조회 가능
- 전년도 1월 1일 ~ 12월 31일 기부 내역

### 모바일 조회
- 손택스 앱 이용
- 홈택스와 동일한 기능
- PDF 다운로드 및 공유 가능

## 전자 발급 의무 기관

### 발급 의무 대상
- 국가, 지방자치단체
- 법정기부금 단체
- 지정기부금 단체 (종교단체 제외)
- 우리사주조합

### 발급 시기
- 기부 받은 날부터 **2개월 이내**
- 국세청에 전자 전송

### 미발급 시 불이익
- 과태료 부과 (건당 5만원)
- 기부금 단체 지정 취소 가능

## 종이영수증과의 차이

### 전자영수증
- 국세청 자동 조회
- 분실 걱정 없음
- 회사 제출 간편
- 위변조 불가

### 종이영수증
- 직접 발급 받아 보관
- 분실 시 재발급 필요
- 회사에 직접 제출
- 간소화 서비스 미등록 시 필요

## 미조회 시 대처 방법

### 1. 기부 단체 확인
- 전자발급 의무 단체인지 확인
- 종교단체는 전자발급 의무 없음

### 2. 기부 단체에 연락
- 전자 전송 요청
- 또는 종이영수증 발급 요청

### 3. 수기 제출
- 종이영수증 수령
- 회사에 직접 제출

## 기부금 공제 확인 사항

### 영수증 확인 항목
- 기부자 성명, 주민등록번호
- 기부 금액, 기부 날짜
- 기부금 단체명, 사업자번호
- 기부금 유형 (법정/지정)

### 공제 가능 여부
- 기부금 단체 지정 여부 확인
- 정치자금 기부금은 별도 조회
- 종교단체 10% 한도 확인

## 실무 팁

### 1. 기부 시점 확인
- 12월 기부분 → 다음 해 1월 조회
- 신용카드 결제일 기준

### 2. 정기 기부
- 자동이체 설정 시 매월 자동 발급
- 연말에 한 번에 조회 가능

### 3. 중복 확인
- 같은 기부금 중복 제출 주의
- 간소화 조회분과 종이영수증 중복 불가

### 4. 가족 기부금
- 본인 명의 기부금만 공제 가능
- 배우자, 자녀 명의는 공제 불가

## 주의사항

### 조회 시기
- 1월 15일 이전: 조회 불가
- 1월 15일 이후: 전년도 기부금 조회 가능

### 기부 단체 확인
- 지정기부금 단체 목록 확인
- 국세청 홈페이지에서 검색 가능

### 허위 기부금
- 실제 기부하지 않은 영수증 제출 시 가산세
- 형사처벌 가능

## 문제 발생 시 대처

### 영수증 누락
1. 기부 단체에 연락
2. 전자 전송 요청 또는 종이영수증 발급
3. 국세청 신고센터 문의 (126)

### 금액 오류
1. 기부 단체에 정정 요청
2. 정정 영수증 재발급
3. 국세청에 전자 재전송

### 중복 발급
1. 하나만 선택하여 제출
2. 중복 제출 시 과다 공제로 추징
