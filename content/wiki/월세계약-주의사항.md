# 월세계약 주의사항

월세계약은 매월 임대료가 발생하므로 꼼꼼한 확인이 필요합니다. 주요 주의사항을 정리합니다.

## 1. 월세 종류

### 1.1 월세 유형

| 유형 | 설명 |
|------|------|
| 순수 월세 | 보증금 적음, 월세 높음 |
| 반전세 | 보증금 중간, 월세 중간 |
| 보증부 월세 | 일반적인 형태 |

### 1.2 전환 계산

| 계산 | 기준 |
|------|------|
| 전환율 | 한국은행 기준금리 + 2% |
| 상한 | 연 10% |

**예시: 보증금 1억 → 월세 전환 (전환율 5%)**

| 계산 | 금액 |
|------|------|
| 보증금 5천만원 전환 | 5천만 × 5% ÷ 12 |
| 월세 | **약 21만원** |
| 결과 | 보증금 5천만 + 월세 21만원 |

## 2. 계약 전 확인

### 2.1 필수 확인 사항

| 확인 | 내용 |
|------|------|
| 등기부등본 | 소유자, 근저당, 가압류 |
| 건축물대장 | 용도, 위반건축물 여부 |
| 시세 확인 | 주변 월세 시세 비교 |

### 2.2 권리 분석

| 항목 | 안전 기준 |
|------|----------|
| 근저당 + 보증금 | 시세 70% 이하 |
| 선순위 보증금 | 확정일자부 열람 |

### 2.3 현장 확인

| 확인 | 내용 |
|------|------|
| 시설 상태 | 하자, 고장 여부 |
| 관리비 | 실제 관리비 확인 |
| 옵션 | 가전, 가구 상태 |

## 3. 계약서 작성

### 3.1 필수 기재 사항

| 항목 | 기재 내용 |
|------|----------|
| 보증금 | 총 보증금 금액 |
| 월세 | 월 임대료 |
| 관리비 | 포함/별도, 금액 |
| 납부일 | 매월 납부일 |
| 납부 방법 | 계좌이체 등 |

### 3.2 관리비 특약

**필수 명시 사항**

| 항목 | 내용 |
|------|------|
| 관리비 금액 | 월 ○○만원 |
| 포함 항목 | 수도, 전기, 가스, 인터넷 등 |
| 별도 항목 | 별도 납부 항목 |
| 변동 기준 | 실비 정산 등 |

### 3.3 특약 예시

```
[관리비 특약]
1. 관리비는 월 10만원으로 하며, 공용전기료,
   청소비, 승강기유지비를 포함한다.
2. 수도료, 전기료, 가스료, 인터넷비용은
   임차인이 별도 부담한다.
3. 난방비는 개별 계량하여 임차인이 부담한다.
```

## 4. 월세 납부

### 4.1 납부 방법

| 방법 | 권장 |
|------|------|
| 계좌이체 | ⭐ 권장 (증빙) |
| 자동이체 | ⭐ 권장 |
| 현금 | △ (영수증 필수) |

### 4.2 납부일

| 통상 | 내용 |
|------|------|
| 매월 말일 | 선불 방식 |
| 다음 달 초 | 후불 방식 |
| 계약서 확인 | 명시된 날짜 |

### 4.3 연체 시

| 상황 | 결과 |
|------|------|
| 1~2회 연체 | 독촉, 연체료 |
| 2기 연체 | 계약 해지 사유 |
| 연체료 | 약정 이율 (보통 연 5~12%) |

## 5. 월세 세액공제

### 5.1 공제 조건

| 조건 | 기준 |
|------|------|
| 소득 | 총급여 **7천만원 이하** |
| 주택 | 무주택 세대주 |
| 주택 규모 | 전용 85㎡ 이하 또는 기준시가 4억 이하 |
| 임대차 | 전입신고 완료 |

### 5.2 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **17%** |
| 5,500~7,000만원 | **15%** |

### 5.3 공제 한도

| 항목 | 한도 |
|------|------|
| 연간 월세 한도 | **750만원** |
| 최대 공제액 | 750만 × 17% = **127.5만원** |

### 5.4 계산 예시

**연간 월세 600만원 (월 50만원), 총급여 4천만원**

| 항목 | 금액 |
|------|------|
| 연간 월세 | 600만원 |
| 공제율 | 17% |
| 세액공제 | 600만 × 17% = **102만원** |

## 6. 월세 영수증

### 6.1 현금영수증 발급

| 방법 | 내용 |
|------|------|
| 홈택스 신청 | 임대인 사업자번호로 |
| 자동 발급 | 계좌이체 시 증빙 |

### 6.2 세액공제 증빙

| 서류 | 용도 |
|------|------|
| 임대차계약서 | 계약 증명 |
| 주민등록등본 | 전입 증명 |
| 월세 이체 내역 | 납부 증명 |
| 집주인 동의서 | 현금영수증 발급 거부 시 |

## 7. 관리비 주의사항

### 7.1 관리비 구성

| 항목 | 포함 여부 확인 |
|------|--------------|
| 공용전기료 | 포함/별도 |
| 청소비 | 포함/별도 |
| 승강기유지비 | 포함/별도 |
| 주차비 | 포함/별도 |
| 인터넷 | 포함/별도 |

### 7.2 관리비 분쟁 예방

| 방법 | 내용 |
|------|------|
| 계약서 명시 | 포함 항목 상세히 |
| 별도 확인서 | 관리비 내역서 받기 |
| 영수증 보관 | 납부 증빙 |

### 7.3 관리비 인상

| 상황 | 대응 |
|------|------|
| 일방적 인상 | 계약서 근거 확인 |
| 합리적 인상 | 협의 |
| 과도한 인상 | 분쟁조정 신청 |

## 8. 보증금 보호

### 8.1 대항력 확보

| 절차 | 시기 |
|------|------|
| 입주 | 잔금일 |
| 전입신고 | 잔금일 당일 |
| 확정일자 | 잔금일 당일 |

### 8.2 전세보증보험

| 대상 | 가입 |
|------|------|
| 보증금 있는 월세 | ✅ 가능 |
| 보증금 금액 | 보증금에 대해 가입 |

### 8.3 소액임차인

| 지역 | 보증금 기준 | 최우선변제 |
|------|-----------|----------|
| 서울 | 1.65억 이하 | 5,500만원 |
| 수도권 | 1.45억 이하 | 4,800만원 |
