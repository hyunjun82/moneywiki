# 연말정산 난임시술비 공제

난임시술비는 일반 의료비보다 높은 **20% 세액공제율**이 적용됩니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 공제 조건을 정리합니다.

## 1. 공제율과 한도

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 난임시술비 공제 기준입니다.

### 1.1 공제율 비교

| 의료비 유형 | 공제율 |
|-------------|--------|
| **난임시술비** | **20%** |
| 미숙아·선천성이상아 | 20% |
| 일반 의료비 | 15% |

### 1.2 공제 한도

| 대상 | 한도 |
|------|------|
| **난임시술비** | **한도 없음** |
| 본인·장애인·65세 이상 | 한도 없음 |
| 그 외 부양가족 | 700만원 |

## 2. 공제 대상 시술

### 2.1 공제 가능 항목

| 시술 | 공제 여부 |
|------|----------|
| 체외수정 (시험관아기) | ✅ 가능 |
| 인공수정 | ✅ 가능 |
| 배아이식 | ✅ 가능 |
| 난자 채취 | ✅ 가능 |
| 정자 채취 | ✅ 가능 |
| 난임 관련 검사비 | ✅ 가능 |
| 난임 치료 약제비 | ✅ 가능 |
| 동결배아 보관료 | ✅ 가능 |

### 2.2 공제 불가 항목

| 항목 | 공제 여부 |
|------|----------|
| 난자·정자 기증자 보상금 | ❌ 불가 |
| 대리모 비용 | ❌ 불가 |
| 성별 선택 목적 시술 | ❌ 불가 |

## 3. 공제 대상자

### 3.1 본인 및 배우자

근로자 본인과 배우자의 난임시술비 모두 공제 가능

### 3.2 다른 부양가족

- 기본공제 대상자인 부양가족의 난임시술비도 공제 가능
- 단, 실무상 난임시술은 주로 본인·배우자에게 적용

## 4. 공제 계산 방식

### 4.1 의료비 공제 구조

난임시술비도 의료비 세액공제의 일부로, **총급여 3% 초과분**부터 공제

| 순서 | 내용 |
|------|------|
| 1단계 | 총급여의 3% 계산 |
| 2단계 | 총 의료비에서 3% 차감 |
| 3단계 | 난임시술비 × 20% 계산 |
| 4단계 | 일반 의료비 × 15% 계산 |
| 5단계 | 합산하여 세액공제 적용 |

### 4.2 계산 예시

**연봉 5,000만원, 난임시술비 1,000만원 + 일반 의료비 200만원 지출**

1. 3% 기준액: 5,000만원 × 3% = 150만원
2. 총 의료비: 1,200만원
3. 공제 대상: 1,200만원 - 150만원 = 1,050만원
4. 난임시술비 공제: (1,000만원 - 150만원) × 20% = 170만원
5. 일반 의료비 공제: 200만원 × 15% = 30만원
6. **총 세액공제: 200만원**

※ 3% 기준액은 난임시술비에서 먼저 차감하는 것이 유리

## 5. 증빙서류

### 5.1 연말정산 간소화

대부분의 난임시술비는 **간소화 서비스에서 자동 조회**

### 5.2 별도 확인 필요

| 항목 | 확인 사항 |
|------|----------|
| 난임시술 확인 | 의료기관에서 난임시술로 분류 확인 |
| 누락 시 | 진료비 영수증 직접 제출 |

## 6. 정부 지원과 병행

### 6.1 건강보험 지원

난임시술 건강보험 적용 (본인부담 30~50%)
→ **본인부담금**에 대해 세액공제

### 6.2 지자체 지원금

지자체 난임시술 지원금 수령 시
→ **지원금 제외한 본인 부담분**만 공제

| 구분 | 공제 대상 |
|------|----------|
| 총 시술비 | 500만원 |
| 지자체 지원 | -100만원 |
| 본인부담금 | **400만원** → 공제 대상 |

## 7. 주의사항

### 7.1 실손보험 보전금액

실손보험으로 돌려받은 금액은 공제 대상에서 **제외**

### 7.2 공제 적용 순서

의료비 세액공제 계산 시 **3% 기준액 차감**은 일반 의료비보다 **난임시술비에서 먼저** 차감하는 것이 유리 (20% > 15%)

### 7.3 세대원 중복

- 맞벌이 부부 중 **한 명만** 공제 가능
- 의료비를 **지출한 근로자**가 공제 신청
