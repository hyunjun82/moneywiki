"난임시술비 공제율이 더 높다는데요?" 맞아요. 20%예요. 일반 의료비 15%보다 5%p 더 받아요.

## 의료비 유형별 공제율

| 유형 | 공제율 | 한도 |
|-----|-------|-----|
| **난임시술비** | **20%** | **없음** |
| 미숙아·선천성이상아 | 20% | 없음 |
| 본인·65세 이상·장애인 | 15% | 없음 |
| 그 외 부양가족 | 15% | 700만원 |

## 공제 대상 시술

| 공제 가능 | 공제 불가 |
|---------|---------|
| 체외수정(시험관아기) | 난자·정자 기증 보상금 |
| 인공수정 | 대리모 비용 |
| 배아이식 | 성별 선택 시술 |
| 난자·정자 채취 | |
| 난임 검사비·약제비 | |
| 동결배아 보관료 | |

## 공제 대상자

| 대상 | 공제 가능 |
|-----|---------|
| 본인 | ⭕ |
| 배우자 | ⭕ |
| 기본공제 부양가족 | ⭕ |

맞벌이 부부는 의료비를 지출한 근로자가 공제 신청해요.

## 공제 계산 예시

연봉 5,000만원, 난임시술비 1,000만원 + 일반 의료비 200만원:

| 단계 | 계산 |
|-----|------|
| 3% 기준액 | 5,000만원 × 3% = 150만원 |
| 난임시술비 공제 | (1,000만원 - 150만원) × 20% = **170만원** |
| 일반 의료비 공제 | 200만원 × 15% = **30만원** |
| **총 세액공제** | **200만원** |

## 증빙서류

| 항목 | 확인사항 |
|-----|---------|
| 간소화서비스 | 난임시술로 분류됐는지 확인 |
| 누락 시 | 진료비 영수증 직접 제출 |
| 지원금 | 지자체·실손보험 환급액 제외 |
