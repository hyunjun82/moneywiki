산후조리원 비용은 의료비 세액공제 항목으로 출산 1회당 200만원 한도가 적용돼요. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리해 볼게요.

## 공제 기본 조건

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 산후조리원 공제 요건이에요.

| 조건 | 내용 |
|------|------|
| 총급여 제한 | 없음 (2019년 폐지) |
| 이용 대상 | 근로자 본인 또는 배우자 |
| 시설 요건 | 모자보건법에 따른 산후조리원 |

## 공제율과 한도

산후조리원 공제액 계산 기준이에요.

| 항목 | 내용 |
|------|------|
| 공제율 | 15% (의료비 세액공제) |
| 한도 | 출산 1회당 200만원 |
| 최대 공제액 | 200만원 x 15% = 30만원 |

## 한도 적용 방식

출산 횟수에 따라 한도가 달라져요.

| 상황 | 한도 |
|------|------|
| 단태아 출산 | 200만원 |
| 쌍둥이 출산 | 200만원 (1회 출산) |
| 세쌍둥이 출산 | 200만원 (1회 출산) |
| 연중 2회 출산 | 400만원 (200만원 x 2) |

맞벌이 부부의 경우 출산 1회당 부부 합산 200만원 한도예요. 부부가 각각 100만원씩 공제하거나 한 명이 200만원 공제할 수 있어요.

## 공제 대상 비용

산후조리원 이용과 관련된 비용이 공제돼요.

| 항목 | 공제 여부 |
|------|----------|
| 산후조리원 이용료 | 가능 |
| 산모 케어 비용 | 가능 |
| 신생아 케어 비용 | 가능 |
| 추가 프로그램 (산후조리원 내) | 가능 |

공제되지 않는 비용도 있어요.

| 항목 | 공제 여부 |
|------|----------|
| 산후도우미 비용 | 불가 |
| 자택 산후관리 서비스 | 불가 |
| 산후조리원 외 추가 서비스 | 불가 |

## 의료비 3% 기준 적용

산후조리원 비용도 의료비에 포함되어 총급여 3% 초과분부터 공제돼요.

| 순서 | 내용 |
|------|------|
| 1단계 | 총급여의 3% 계산 |
| 2단계 | 총 의료비에서 3% 차감 |
| 3단계 | 산후조리원 비용 x 15% (200만원 한도) |

연봉 5,000만원, 산후조리원 300만원 + 병원비 100만원 지출한 경우예요. 3% 기준액은 150만원이고, 총 의료비 400만원에서 150만원을 빼면 250만원이 공제 대상이에요. 산후조리원은 200만원 한도로 30만원 공제, 병원비 50만원에서 7.5만원 공제해서 총 37.5만원 세액공제를 받아요.

## 증빙서류

산후조리원 비용은 간소화 서비스에서 자동 조회돼요.

| 서류 | 내용 |
|------|------|
| 산후조리원 영수증 | 이용 기간, 금액 기재 |
| 결제 증빙 | 카드 영수증 또는 이체 내역 |

누락됐으면 산후조리원에서 직접 영수증을 받아야 해요.

## 시설 요건

모자보건법에 따라 등록된 산후조리원만 공제 대상이에요.

| 구분 | 공제 여부 |
|------|----------|
| 등록 산후조리원 | 가능 |
| 미등록 산후조리원 | 불가 |
| 산후조리 전문 병원 | 가능 (의료기관) |

영수증에 사업자등록번호가 있는지 확인하세요.

## 주의사항

200만원 초과 지출 시 초과분은 공제받을 수 없어요. 다른 의료비와 합산해도 초과분은 공제 안 돼요.

12월 출산 후 1월에 산후조리원을 이용하면 이용한 연도(1월) 기준으로 공제받아요.

산후조리원 200만원은 별도 한도예요. 부양가족 의료비 700만원 한도와 별개로 적용돼요.
