# 연말정산 산후조리원 공제

산후조리원 비용은 의료비 세액공제 항목으로, **출산 1회당 200만원 한도**가 적용됩니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리합니다.

## 1. 공제 기본 조건

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 산후조리원 공제 요건입니다.

### 1.1 공제 대상자

| 조건 | 내용 |
|------|------|
| 총급여 제한 | **없음** (2019년 폐지) |
| 이용 대상 | 근로자 본인 또는 배우자 |
| 시설 요건 | 모자보건법에 따른 산후조리원 |

### 1.2 공제율과 한도

| 항목 | 내용 |
|------|------|
| 공제율 | **15%** (의료비 세액공제) |
| 한도 | **출산 1회당 200만원** |
| 최대 공제액 | 200만원 × 15% = **30만원** |

## 2. 한도 적용 방식

### 2.1 출산 1회당 한도

| 상황 | 한도 |
|------|------|
| 단태아 출산 | 200만원 |
| 쌍둥이 출산 | 200만원 (1회 출산) |
| 세쌍둥이 출산 | 200만원 (1회 출산) |
| 연중 2회 출산 | 400만원 (200만원 × 2) |

### 2.2 부부 합산

맞벌이 부부의 경우:
- 출산 1회당 **부부 합산 200만원** 한도
- 부부가 각각 100만원씩 공제하거나 한 명이 200만원 공제 가능

## 3. 공제 대상 비용

### 3.1 공제 가능

| 항목 | 공제 여부 |
|------|----------|
| 산후조리원 이용료 | ✅ 가능 |
| 산모 케어 비용 | ✅ 가능 |
| 신생아 케어 비용 | ✅ 가능 |
| 추가 프로그램 (산후조리원 내) | ✅ 가능 |

### 3.2 공제 불가

| 항목 | 공제 여부 |
|------|----------|
| 산후도우미 비용 | ❌ 불가 |
| 자택 산후관리 서비스 | ❌ 불가 |
| 산후조리원 외 추가 서비스 | ❌ 불가 |

## 4. 의료비 3% 기준 적용

### 4.1 공제 구조

산후조리원 비용도 의료비에 포함되어 **총급여 3% 초과분**부터 공제

| 순서 | 내용 |
|------|------|
| 1단계 | 총급여의 3% 계산 |
| 2단계 | 총 의료비에서 3% 차감 |
| 3단계 | 산후조리원 비용 × 15% (200만원 한도) |

### 4.2 계산 예시

**연봉 5,000만원, 산후조리원 300만원 + 병원비 100만원 지출**

1. 3% 기준액: 5,000만원 × 3% = 150만원
2. 총 의료비: 400만원
3. 공제 대상: 400만원 - 150만원 = 250만원
4. 산후조리원 공제: 200만원 × 15% = 30만원 (300만원 중 200만원 한도)
5. 병원비 공제: 50만원 × 15% = 7.5만원
6. **총 세액공제: 37.5만원**

## 5. 증빙서류

### 5.1 연말정산 간소화

산후조리원 비용은 **간소화 서비스에서 자동 조회**

### 5.2 누락 시

| 서류 | 내용 |
|------|------|
| 산후조리원 영수증 | 이용 기간, 금액 기재 |
| 결제 증빙 | 카드 영수증 또는 이체 내역 |

## 6. 시설 요건

### 6.1 적격 시설

**모자보건법**에 따라 등록된 산후조리원만 공제 대상

| 구분 | 공제 여부 |
|------|----------|
| 등록 산후조리원 | ✅ 가능 |
| 미등록 산후조리원 | ❌ 불가 |
| 산후조리 전문 병원 | ✅ 가능 (의료기관) |

### 6.2 확인 방법

1. 산후조리원 인허가 사항 확인
2. 모자보건법 등록 여부 문의
3. 영수증에 사업자등록번호 확인

## 7. 주의사항

### 7.1 한도 초과분

- 200만원 초과 지출 시 **초과분은 공제 불가**
- 다른 의료비와 합산해도 초과분 공제 안 됨

### 7.2 연도 구분

- 12월 출산 → 1월 산후조리원 이용
- **이용한 연도 기준**으로 공제 (1월에 공제)

### 7.3 부양가족 의료비 한도

- 산후조리원 200만원은 **별도 한도**
- 부양가족 의료비 700만원 한도와 별개로 적용
