# 집주인 변경 시 임차인 권리

집이 매매되어 집주인이 바뀌면 세입자는 불안해집니다. 대항력이 있으면 권리가 보호됩니다.

## 1. 집주인 변경 상황

### 1.1 변경 사유

| 사유 | 설명 |
|------|------|
| 매매 | 집주인이 집을 팔 때 |
| 상속 | 집주인 사망 시 |
| 증여 | 집주인이 양도 시 |
| 경매 | 경매로 낙찰될 때 |

### 1.2 임차인 걱정

| 걱정 | 내용 |
|------|------|
| 퇴거 요구 | 나가라고 하면? |
| 보증금 | 누가 돌려주나? |
| 계약 | 계약이 유효한가? |

## 2. 대항력이 있는 경우

### 2.1 대항력 효과

| 효과 | 내용 |
|------|------|
| 계약 유지 | 기존 계약 그대로 |
| 거주 보장 | 계약 기간까지 거주 |
| 보증금 보호 | 새 집주인이 반환 의무 |

### 2.2 새 집주인의 의무

| 의무 | 내용 |
|------|------|
| 임대인 지위 승계 | 기존 임대인 권리·의무 승계 |
| 보증금 반환 | 계약 종료 시 반환 |
| 계약 조건 준수 | 기존 조건 유지 |

### 2.3 임차인 권리

| 권리 | 내용 |
|------|------|
| 계약갱신청구권 | 1회 행사 가능 (미사용 시) |
| 묵시적 갱신 | 적용됨 |
| 월세 | 기존 금액 유지 |

## 3. 대항력이 없는 경우

### 3.1 위험 상황

| 상황 | 결과 |
|------|------|
| 전입신고 미완료 | 대항력 없음 |
| 점유 없음 | 대항력 없음 |
| 대항력 취득 전 매매 | 새 집주인에게 주장 불가 |

### 3.2 결과

| 결과 | 내용 |
|------|------|
| 퇴거 요구 가능 | 새 집주인이 퇴거 요구 |
| 보증금 | 전 집주인에게 청구 |
| 대책 | 전 집주인 상대 소송 |

## 4. 매매 vs 경매

### 4.1 일반 매매

| 항목 | 내용 |
|------|------|
| 대항력 있으면 | 새 집주인에게 대항 |
| 대항력 없으면 | 전 집주인에게 청구 |
| 계약 승계 | 자동 승계 |

### 4.2 경매 낙찰

| 항목 | 내용 |
|------|------|
| 선순위 임차인 | 낙찰자에게 대항 |
| 후순위 임차인 | 대항 불가, 배당 신청 |
| 소액임차인 | 최우선변제 가능 |

### 4.3 경매 시 임차인 분류

| 유형 | 권리 |
|------|------|
| 대항력 선순위 | 낙찰자에게 대항, 계속 거주 |
| 대항력 후순위 | 배당요구, 퇴거 |
| 대항력 없음 | 전 소유자에게 청구 |

## 5. 보증금 반환

### 5.1 일반 매매 시

| 책임자 | 내용 |
|--------|------|
| 새 집주인 | 보증금 반환 의무 승계 |
| 전 집주인 | 원칙적으로 면책 |

### 5.2 경매 낙찰 시

| 상황 | 보증금 |
|------|--------|
| 선순위 | 낙찰자가 인수 또는 변제 |
| 후순위 | 배당으로 회수 |
| 배당 부족 | 전 소유자 상대 청구 |

### 5.3 보증금 승계 협의

| 방법 | 내용 |
|------|------|
| 정산 협의 | 전·현 집주인 간 정산 |
| 임차인 확인 | 승계 사실 확인 |
| 서면 작성 | 변경 확인서 |

## 6. 임차인 대응

### 6.1 매매 통보 받았을 때

| 절차 | 내용 |
|------|------|
| 1 | 대항력 확인 (전입일, 확정일자) |
| 2 | 등기부등본 확인 |
| 3 | 새 집주인 정보 확인 |
| 4 | 연락처 교환 |

### 6.2 새 집주인과 협의

| 협의 | 내용 |
|------|------|
| 계약 확인 | 기존 계약 조건 |
| 보증금 | 반환 방법 |
| 연락처 | 수리, 문의 등 |

### 6.3 서류 확보

| 서류 | 용도 |
|------|------|
| 기존 계약서 | 계약 증명 |
| 등기부등본 | 소유자 변경 확인 |
| 전입세대 열람 | 전입 확인 |
| 변경 확인서 | 새 집주인과 |

## 7. 새 집주인이 퇴거 요구 시

### 7.1 대항력 있는 경우

| 대응 | 내용 |
|------|------|
| 거부 가능 | 계약 기간까지 거주 권리 |
| 근거 | 대항력 + 계약 |
| 단 | 실거주 사유면 갱신 거절 가능 |

### 7.2 대응 방법

| 단계 | 조치 |
|------|------|
| 1 | 대항력 확인서류 준비 |
| 2 | 계약 기간 안내 |
| 3 | 서면 거부 |
| 4 | 필요시 법률 상담 |

### 7.3 협의 가능 사항

| 협의 | 조건 |
|------|------|
| 조기 퇴거 | 합리적 보상 |
| 이사비 | 협의 |
| 보증금 조기 반환 | 협의 |

## 8. 계약서 재작성

### 8.1 필요 여부

| 상황 | 필요 |
|------|------|
| 기존 계약 유효 | 불필요 |
| 확인 차원 | 권장 |
| 조건 변경 | 필요 |

### 8.2 변경 확인서

| 내용 | 기재 |
|------|------|
| 변경 사유 | 매매로 인한 임대인 변경 |
| 새 임대인 | 정보 |
| 기존 조건 | 유지 확인 |
| 서명 | 양측 서명 |

### 8.3 확인서 예시

```
[임대인 변경 확인서]

1. 부동산 표시: ○○시 ○○구 ○○동 ○○호
2. 변경 사유: 매매로 인한 소유권 이전
3. 전 임대인: 홍길동
4. 신 임대인: 김철수
5. 기존 계약: 2024.7.1.~2026.6.30. 보증금 3억

위 임대차계약의 임대인 지위가 신 임대인에게
승계되었음을 확인합니다.

2026년 1월 15일
신 임대인: 김철수 (인)
임차인: 박영희 (인)
```

## 9. 주의사항

### 9.1 등기부등본 확인

| 확인 | 내용 |
|------|------|
| 소유권 이전 | 새 집주인 명의 확인 |
| 근저당 | 신규 설정 여부 |
| 권리관계 | 변동 확인 |

### 9.2 전세보증보험

| 상황 | 조치 |
|------|------|
| 가입 중 | 보증기관에 변경 신고 |
| 미가입 | 가입 검토 |

### 9.3 월세 입금 계좌

| 주의 | 내용 |
|------|------|
| 계좌 변경 | 새 집주인 계좌로 |
| 확인 | 소유자 명의 확인 |

## 10. 자주 묻는 질문

### 10.1 전 집주인이 보증금 안 주면?

| 답변 | 설명 |
|------|------|
| 새 집주인 책임 | 대항력 있으면 |
| 전 집주인 청구 | 대항력 없으면 |

### 10.2 새 집주인이 월세 올리겠다면?

| 답변 | 설명 |
|------|------|
| 계약 기간 중 | 불가 |
| 갱신 시 | 5% 상한 적용 |

### 10.3 상속으로 변경되면?

| 답변 | 설명 |
|------|------|
| 동일 | 상속인이 임대인 지위 승계 |
| 권리 | 그대로 유지 |
