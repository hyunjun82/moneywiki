# 사업자등록

사업자등록은 사업을 시작할 때 세무서에 신고하는 절차입니다. 등록 방법과 유형을 정리합니다.

## 사업자등록 개요

### 1.1 등록 의무

| 대상 | 등록 |
|------|------|
| 모든 사업자 | 의무 |
| 프리랜서 (계속적) | 의무 |
| 일시적 용역 | 미등록 가능 |

### 1.2 등록 기한

| 항목 | 기한 |
|------|------|
| 원칙 | 사업개시일로부터 **20일** |
| 권장 | 사업 개시 전 |

## 사업자 유형

### 2.1 과세 유형

| 유형 | 기준 |
|------|------|
| 일반과세자 | 연 매출 8,000만원 이상 |
| 간이과세자 | 연 매출 **8,000만원 미만** |
| 면세사업자 | 면세 재화·용역만 공급 |

### 2.2 사업 형태

| 유형 | 특징 |
|------|------|
| 개인사업자 | 개인 명의 사업 |
| 법인사업자 | 법인 설립 후 사업 |

### 2.3 일반 vs 간이과세자

| 구분 | 일반과세자 | 간이과세자 |
|------|----------|-----------|
| 매출 기준 | 8,000만원 이상 | 8,000만원 미만 |
| 부가세 신고 | 연 4회 | **연 1회** |
| 세금계산서 | 발급 의무 | 영수증 발급 |
| 매입세액 공제 | 전액 | 업종별 차등 |

## 등록 신청

### 3.1 신청 방법

| 방법 | 내용 |
|------|------|
| 온라인 | 홈택스 (www.hometax.go.kr) |
| 방문 | 관할 세무서 |

### 3.2 필요 서류 (개인)

| 서류 | 용도 |
|------|------|
| 사업자등록 신청서 | 기본 서류 |
| 신분증 | 본인 확인 |
| 임대차계약서 | 사업장 확인 |
| 허가증·등록증 | 허가 업종의 경우 |

### 3.3 필요 서류 (법인)

| 서류 | 용도 |
|------|------|
| 사업자등록 신청서 | 기본 서류 |
| 법인등기부등본 | 법인 확인 |
| 정관 사본 | 사업 목적 |
| 임대차계약서 | 사업장 확인 |
| 주주명부 | 주주 현황 |

### 3.4 처리 기간

| 방법 | 기간 |
|------|------|
| 온라인 | 즉시~3일 |
| 방문 | 당일~3일 |

|------|------|

| 상호 | 사업체 이름 |
| 대표자 | 사업 대표자 |
| 사업장 소재지 | 사업장 주소 |
| 업태·종목 | 사업 분야 |
| 개업일 | 사업 시작일 |

## 간이과세자 특례

### 6.1 적용 대상

| 조건 | 기준 |
|------|------|
| 연 매출 | **8,000만원 미만** |
| 업종 | 일부 제외 (광업, 제조업 등) |

### 6.2 혜택
