# 연말정산 퇴사자

퇴사자의 연말정산은 일반 근로자와 다른 절차가 적용됩니다. [소득세법 제137조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 퇴사 시 연말정산

### 1.1 중도 정산

| 항목 | 내용 |
|------|------|
| 시기 | **퇴사월 급여 지급 시** |
| 주체 | 퇴직하는 회사 |
| 범위 | 해당 연도 1월~퇴사월 소득 |

### 1.2 중도 정산 절차

```
퇴사 → 마지막 급여 지급 → 중도 연말정산 → 원천징수영수증 발급
```

## 2. 재취업 여부에 따른 처리

### 2.1 같은 해 재취업

| 단계 | 내용 |
|------|------|
| 1 | 전 직장에서 원천징수영수증 수령 |
| 2 | 새 직장에 원천징수영수증 제출 |
| 3 | 새 직장에서 **합산하여 연말정산** |

### 2.2 재취업하지 않음

| 단계 | 내용 |
|------|------|
| 1 | 전 직장에서 원천징수영수증 수령 |
| 2 | 다음 해 **5월 종합소득세 신고** |
| 3 | 누락 공제 반영, 최종 세금 확정 |

### 2.3 다음 해 취업

| 단계 | 내용 |
|------|------|
| 1 | 퇴사 연도: 5월 종합소득세 신고 |
| 2 | 새 직장: 해당 연도 소득만 연말정산 |

## 3. 원천징수영수증

### 3.1 필수 수령

| 항목 | 내용 |
|------|------|
| 명칭 | 근로소득 원천징수영수증 |
| 발급 주체 | 퇴직하는 회사 |
| 발급 시기 | 퇴사 시 또는 요청 시 |

### 3.2 기재 내용

| 항목 | 내용 |
|------|------|
| 총급여 | 1월~퇴사월 급여 합계 |
| 기납부세액 | 원천징수한 세금 합계 |
| 공제 내역 | 중도정산 시 적용한 공제 |

### 3.3 미수령 시

| 방법 | 절차 |
|------|------|
| 회사 요청 | 퇴직 회사에 발급 요청 |
| 홈택스 조회 | 다음 해 3월 이후 조회 가능 |

## 4. 중도 퇴사 연말정산

### 4.1 적용 공제

| 공제 | 적용 |
|------|------|
| 인적공제 | 월할 ❌ (전액) |
| 신용카드 | 1~퇴사월 사용분 |
| 의료비 | 1~퇴사월 지출분 |
| 교육비 | 1~퇴사월 납입분 |

### 4.2 적용 세율

- 연간 소득으로 **환산** 후 세율 적용
- 실제 근무 기간 소득만으로 계산 ❌

## 5. 종합소득세 신고

### 5.1 신고 대상

| 상황 | 신고 필요 |
|------|----------|
| 퇴사 후 미취업 | ✅ 필요 |
| 프리랜서로 전환 | ✅ 필요 |
| 다른 소득 있음 | ✅ 필요 |

### 5.2 신고 기간

| 항목 | 기간 |
|------|------|
| 신고 기간 | 다음 해 **5월 1일~31일** |
| 납부 기한 | 5월 31일 |

### 5.3 신고 방법

| 방법 | 절차 |
|------|------|
| 홈택스 | 온라인 셀프 신고 |
| 세무사 | 대행 신고 |
| 세무서 | 방문 신고 |

## 6. 공제 누락 처리

### 6.1 중도정산 시 누락

| 방법 | 내용 |
|------|------|
| 재취업 시 | 새 회사 연말정산에 반영 |
| 미취업 시 | 5월 종합소득세 신고에 반영 |

### 6.2 경정청구

| 조건 | 내용 |
|------|------|
| 기한 | 법정신고기한 5년 이내 |
| 방법 | 홈택스 경정청구 |

## 7. 환급금 수령

### 7.1 중도정산 환급

| 시점 | 방법 |
|------|------|
| 퇴사 시 | 마지막 급여에 포함 |
| 별도 지급 | 회사 정책에 따라 |

### 7.2 종합소득세 환급

| 시점 | 방법 |
|------|------|
| 신고 후 | 신고한 계좌로 입금 |
| 기간 | 신고 후 2~3개월 내 |

## 8. 주의사항

### 8.1 원천징수영수증 보관

- **5년간** 보관 권장
- 경정청구, 대출 등에 필요

### 8.2 연말정산 간소화 자료

| 상황 | 조회 범위 |
|------|----------|
| 퇴사 후 재취업 | 1~12월 전체 |
| 미취업 | 1~퇴사월 (근무 기간) |

### 8.3 퇴직금과 구분

| 항목 | 적용 세금 |
|------|----------|
| 급여 | 근로소득세 (연말정산) |
| 퇴직금 | 퇴직소득세 (별도 정산) |

## 9. 계산 예시

### 9.1 6월 퇴사, 미취업

**1~6월 총급여 2,500만원, 기납부세액 50만원**

1. 중도정산: 회사에서 진행
2. 5월 신고: 누락 공제 반영
3. 결정세액 30만원 → 환급 20만원

### 9.2 6월 퇴사, 9월 재취업

**전 직장 2,500만원 + 새 직장 1,500만원 = 4,000만원**

1. 원천징수영수증 새 직장 제출
2. 새 직장에서 합산 연말정산
3. 1~12월 전체 공제 적용
