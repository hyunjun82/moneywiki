# 부가가치세

부가가치세(VAT)는 재화나 용역의 거래 과정에서 발생하는 부가가치에 부과되는 세금입니다. [부가가치세법](https://www.law.go.kr/법령/부가가치세법)에 따른 내용을 정리합니다.

## 부가가치세 개요

### 1.1 기본 구조

```
부가가치세 = 매출세액 - 매입세액
```

| 항목 | 계산 |
|------|------|
| 매출세액 | 매출액 × 10% |
| 매입세액 | 매입액 × 10% |
| 납부세액 | 매출세액 - 매입세액 |

### 1.2 과세 대상

| 거래 유형 | 과세 |
|----------|------|
| 재화 공급 |  과세 |
| 용역 공급 |  과세 |
| 재화 수입 |  과세 |
| 면세 품목 |  면세 |

### 1.3 면세 품목

| 품목 | 면세 |
|------|------|
| 미가공 농산물 |  |
| 의료·교육 서비스 |  |
| 도서·신문 |  |
| 대중교통 |  |

## 과세 유형

### 2.1 일반과세자

| 항목 | 내용 |
|------|------|
| 기준 | 연 매출 8,000만원 이상 |
| 세율 | **10%** |
| 신고 | 연 4회 (확정 2회 + 예정 2회) |
| 세금계산서 | 발급 의무 |

### 2.2 간이과세자

| 항목 | 내용 |
|------|------|
| 기준 | 연 매출 **8,000만원 미만** |
| 세율 | 업종별 1.5~4% |
| 신고 | **연 1회** (1월) |
| 세금계산서 | 영수증 발급 |

### 2.3 간이과세자 부가가치율

| 업종 | 부가가치율 |
|------|-----------|
| 소매업 | 15% |
| 음식점업 | 20% |
| 제조업 | 20% |
| 숙박업 | 25% |
| 건설업 | 30% |
| 부동산임대 | 40% |

## 신고·납부

### 3.1 일반과세자 신고 기간

| 과세기간 | 신고 기간 | 유형 |
|----------|----------|------|
| 1~3월 | 4월 1~25일 | 예정신고 |
| 1~6월 | **7월 1~25일** | 확정신고 |
| 7~9월 | 10월 1~25일 | 예정신고 |
| 7~12월 | **1월 1~25일** | 확정신고 |

### 3.2 간이과세자 신고 기간

| 과세기간 | 신고 기간 |
|----------|----------|
| 1~12월 | **1월 1~25일** |

### 3.3 납부 방법

| 방법 | 내용 |
|------|------|
| 계좌이체 | 홈택스에서 납부 |
| 카드 납부 | 수수료 0.8% |
| 분납 | 세액 1천만원 초과 시 |

## 세금계산서

### 4.1 발급 의무

| 과세자 유형 | 발급 |
|------------|------|
| 일반과세자 | **세금계산서** |
| 간이과세자 (4,800만원 이상) | 세금계산서 |
| 간이과세자 (4,800만원 미만) | 영수증 |

### 4.2 전자세금계산서

| 의무 발급 대상 | 기준 |
|--------------|------|
| 법인사업자 | 전부 |
| 개인사업자 | 연 매출 8,000만원 이상 |

### 4.3 발급 기한

| 시점 | 기한 |
|------|------|
| 재화 공급일 | **다음 달 10일까지** |
| 용역 제공일 | 다음 달 10일까지 |

## 계산 예시

### 6.1 일반과세자

**분기 매출 1억원, 매입 6,000만원**

| 항목 | 금액 |
|------|------|
| 매출세액 | 1억원 × 10% = 1,000만원 |
| 매입세액 | 6,000만원 × 10% = 600만원 |
| **납부세액** | **400만원** |

### 6.2 간이과세자 (음식점)

**연 매출 5,000만원**

| 항목 | 금액 |
|------|------|
| 매출액 | 5,000만원 |
| 부가가치율 | 20% |
| 과세표준 | 1,000만원 |
| 세율 | 10% |
| **납부세액** | **100만원** |

## 영세율

### 7.1 적용 대상

| 거래 유형 | 세율 |
|----------|------|
| 수출 | **0%** |
| 외국항행 용역 | 0% |
| 외화 획득 용역 | 0% |

### 7.2 영세율 vs 면세

| 구분 | 영세율 | 면세 |
|------|--------|------|
| 세율 | 0% | 없음 |
| 매입세액 공제 |  가능 |  불가 |
| 세금계산서 | 발급 | 발급 안 함 |
