# 연말정산 보청기

보청기 구입비 의료비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 대상자별 상이 |

## 2. 공제 한도

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 | **한도 없음** |
| 65세 이상 | **한도 없음** |
| 그 외 | 700만원 |

## 3. 필요서류

| 서류 | 발급처 |
|------|--------|
| 보청기 구입영수증 | 판매업소 |
| 의사 소견서 | 병원 (권장) |

## 4. 공제 예시

| 보청기 비용 | 대상 | 공제액(15%) |
|-------------|------|-------------|
| 200만원 | 본인 | 30만원 |
| 300만원 | 65세 부모 | 45만원 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 판매업소 | 의료기기 판매업 등록 |
| 간소화 | 별도 제출 필요할 수 있음 |
