# 연말정산 부녀자 공제

부녀자 공제는 여성 근로자에게 추가로 소득공제를 해주는 제도입니다. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 조건을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 공제 유형 | **추가 소득공제** |
| 공제 금액 | 연 **50만원** |
| 적용 대상 | 여성 근로자 |

### 1.2 세금 절감 효과

| 과세표준 구간 | 세율 | 절감 세액 |
|--------------|------|----------|
| 1,400만원 이하 | 6% | 3만원 |
| 5,000만원 이하 | 15% | 7.5만원 |
| 8,800만원 이하 | 24% | 12만원 |

## 2. 공제 조건

[소득세법 제51조](https://www.law.go.kr/법령/소득세법)에 따른 조건입니다.

### 2.1 소득 요건

| 조건 | 기준 |
|------|------|
| 총급여 | **3,000만원 이하** |
| 종합소득금액 | **3,000만원 이하** |

### 2.2 대상자 유형

**두 가지 유형 중 하나에 해당**:

| 유형 | 조건 |
|------|------|
| **유형 1** | 배우자가 있는 여성 |
| **유형 2** | 부양가족이 있는 여성 세대주 |

### 2.3 유형별 상세 조건

**유형 1: 배우자 있는 여성**

| 조건 | 내용 |
|------|------|
| 성별 | 여성 |
| 혼인 | 배우자 있음 (법적 혼인) |
| 소득 | 총급여 3,000만원 이하 |
| 세대주 여부 | 무관 |

**유형 2: 세대주 여성**

| 조건 | 내용 |
|------|------|
| 성별 | 여성 |
| 세대주 | 세대주여야 함 |
| 부양가족 | 기본공제 대상 부양가족 있어야 함 |
| 소득 | 총급여 3,000만원 이하 |

## 3. 공제 대상 예시

### 3.1 공제 가능

| 사례 | 적용 |
|------|------|
| 기혼 여성, 연봉 2,800만원 | ✅ 부녀자 공제 |
| 미혼 여성 세대주, 부모 부양 | ✅ 부녀자 공제 |
| 여성 세대주, 자녀 부양 | ✅ 부녀자 공제 |

### 3.2 공제 불가

| 사례 | 적용 |
|------|------|
| 미혼 여성, 세대원 | ❌ 불가 |
| 미혼 여성 세대주, 부양가족 없음 | ❌ 불가 |
| 기혼 여성, 연봉 3,500만원 | ❌ 불가 (소득 초과) |
| 남성 | ❌ 불가 (여성만 해당) |

## 4. 한부모 공제와 관계

### 4.1 중복 적용 불가

한부모 공제와 부녀자 공제 모두 해당 시:
→ **한부모 공제(100만원)만 적용**

| 공제 | 금액 | 중복 시 |
|------|------|---------|
| 부녀자 공제 | 50만원 | ❌ 적용 안 됨 |
| 한부모 공제 | 100만원 | ✅ 이것만 적용 |

### 4.2 판단 기준

| 상황 | 적용되는 공제 |
|------|--------------|
| 이혼 여성 + 자녀 양육 | 한부모 공제 100만원 |
| 사별 여성 + 자녀 양육 | 한부모 공제 100만원 |
| 기혼 여성 | 부녀자 공제 50만원 |

## 5. 공제 계산 예시

### 5.1 기혼 여성 근로자

**기혼 여성, 연봉 2,500만원**

| 공제 항목 | 금액 |
|----------|------|
| 본인 기본공제 | 150만원 |
| **부녀자 공제** | **50만원** |
| 배우자 공제 (해당 시) | 150만원 |

### 5.2 세대주 미혼 여성

**미혼 여성 세대주, 부모 부양, 연봉 2,800만원**

| 공제 항목 | 금액 |
|----------|------|
| 본인 기본공제 | 150만원 |
| 부모 기본공제 | 150만원 |
| 경로우대 공제 | 100만원 |
| **부녀자 공제** | **50만원** |

## 6. 맞벌이 부부의 경우

### 6.1 부녀자 공제 적용

| 아내 연봉 | 공제 가능 |
|----------|----------|
| 2,500만원 | ✅ 가능 |
| 3,000만원 | ✅ 가능 |
| 3,500만원 | ❌ 불가 |

### 6.2 절세 전략

- 아내 연봉 3,000만원 이하 → **부녀자 공제 50만원**
- 남편에게 부양가족 공제 몰아주기 고려

## 7. 증빙서류

### 7.1 기본 서류

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 세대주 여부 확인 |
| 가족관계증명서 | 배우자 또는 부양가족 확인 |

### 7.2 특별한 경우

별도 증빙 없이 **회사에 신고**하면 적용

## 8. 주의사항

### 8.1 소득 기준

- **총급여 3,000만원 초과 시 불가**
- 비과세 소득 제외한 총급여 기준

### 8.2 세대주 판단

- **주민등록상 세대주**여야 함
- 실제 생계 책임과 무관하게 등록 기준

### 8.3 부양가족 요건 (세대주 미혼 여성)

- **기본공제 대상** 부양가족이 있어야 함
- 소득 요건, 나이 요건 충족 필요

## 9. 다른 공제와 병행

### 9.1 함께 적용 가능

| 공제 | 병행 가능 |
|------|----------|
| 경로우대 공제 | ✅ 가능 |
| 장애인 공제 | ✅ 가능 |
| 부양가족 공제 | ✅ 가능 |

### 9.2 중복 불가

| 공제 | 병행 |
|------|------|
| 한부모 공제 | ❌ 한부모 공제만 적용 |
