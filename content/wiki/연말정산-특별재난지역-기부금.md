# 연말정산 특별재난지역 기부금

특별재난지역에 고향사랑기부금을 기부하면 더 높은 세액공제 혜택을 받을 수 있습니다.

## 공제율 비교

| 구분 | 10만원 이하 | 10만원 초과 |
|------|------------|------------|
| 일반 지역 | 100% | 16.5% |
| **특별재난지역** | 100% | **30%** |

## 계산 예시

50만원을 특별재난지역에 기부한 경우:

| 항목 | 금액 |
|------|------|
| 10만원 이하 공제 | 10만원 (100%) |
| 10만원 초과분 40만원 | 12만원 (30%) |
| **총 세액공제** | **22만원** |
| 답례품 (30%) | 15만원 |
| **실질 혜택** | **37만원** |

> 일반 지역 기부 시 세액공제 16.6만원 vs 특별재난지역 22만원

## 특별재난지역 지정 기준

- 자연재해 (태풍, 홍수, 지진 등)
- 대규모 피해 발생 지역
- 행정안전부 장관 지정

## 확인 방법

1. **고향사랑e음** 사이트 접속
2. 기부 가능 지자체 목록에서 '특별재난지역' 표시 확인
3. 또는 행정안전부 재난안전포털 확인

## 기부 시 유의사항

- 특별재난지역 지정 기간 내 기부해야 적용
- 지정 해제 후 기부 시 일반 공제율(16.5%) 적용
- 기부 연도 기준으로 공제 적용
