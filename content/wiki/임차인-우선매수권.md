# 임차인 우선매수권

경매로 살던 집이 넘어갈 때, 대항력 있는 임차인은 최고가 매수인과 같은 조건으로 집을 살 수 있습니다.

## 1. 우선매수권이란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 최고가와 같은 조건으로 우선 매수할 권리 |
| 대상 | 대항력 있는 임차인 |
| 근거 | 민사집행법 제148조 |

### 1.2 도입 취지

| 취지 | 설명 |
|------|------|
| 주거 안정 | 경매로 인한 주거 불안 해소 |
| 임차인 보호 | 살던 집 매수 기회 제공 |
| 생활 유지 | 이사 부담 경감 |

## 2. 우선매수권 요건

### 2.1 권리 요건

| 요건 | 내용 |
|------|------|
| 대항력 | 전입신고 + 점유 |
| 시기 | 경매개시결정 전 대항력 취득 |
| 대상 | 주거용 건물 임차인 |

### 2.2 적용 대상

| 대상 | 적용 |
|------|------|
| 주택 임차인 | ✅ |
| 상가 임차인 | ✅ (상가임대차보호법) |
| 전세권자 | ❌ (별도 규정) |

### 2.3 제외 대상

| 대상 | 제외 |
|------|------|
| 대항력 없는 임차인 | ❌ |
| 경매개시 후 전입 | ❌ |
| 점유 없는 임차인 | ❌ |

## 3. 우선매수 신고

### 3.1 신고 기한

| 기한 | 내용 |
|------|------|
| 마감 | **매각기일** 끝나기 전 |
| 장소 | 매각 법원 |
| 방법 | 우선매수신고서 제출 |

### 3.2 신고 방법

| 절차 | 내용 |
|------|------|
| 1 | 우선매수신고서 작성 |
| 2 | 증빙서류 첨부 |
| 3 | 법원 경매계 제출 |
| 4 | 신고 접수 확인 |

### 3.3 필요 서류

| 서류 | 용도 |
|------|------|
| 우선매수신고서 | 신청서 양식 |
| 임대차계약서 | 임차 증명 |
| 주민등록등본 | 전입 증명 |
| 신분증 사본 | 본인 확인 |

## 4. 우선매수 절차

### 4.1 절차 흐름

```
경매 공고 → 입찰 진행 → 최고가 결정 →
우선매수 신고 → 동일 조건 매수 의사 →
우선매수인 결정 → 대금 납부 → 소유권 취득
```

### 4.2 단계별 내용

| 단계 | 내용 |
|------|------|
| 1 | 경매 입찰 진행 |
| 2 | 최고가 매수인 결정 |
| 3 | 우선매수 신고 확인 |
| 4 | 최고가와 동일 조건 제시 |
| 5 | 우선매수인으로 결정 |
| 6 | 매각대금 납부 |
| 7 | 소유권 취득 |

### 4.3 대금 납부

| 항목 | 내용 |
|------|------|
| 금액 | 최고가 매수 금액 |
| 기한 | 매각허가결정 확정 후 1개월 |
| 방법 | 법원 지정 계좌 입금 |

## 5. 보증금과의 관계

### 5.1 보증금 충당 여부

| 질문 | 답변 |
|------|------|
| 보증금으로 매수대금 충당? | **❌ 불가** |
| 별도 준비 | 매수대금 전액 별도 준비 |
| 보증금 | 배당 절차로 돌려받음 |

### 5.2 자금 계획

| 항목 | 내용 |
|------|------|
| 매수대금 | 100% 준비 필요 |
| 보증금 | 배당으로 별도 수령 |
| 대출 | 주택담보대출 활용 가능 |

### 5.3 예시

| 항목 | 금액 |
|------|------|
| 낙찰가 | 3억원 |
| 보증금 | 2억원 |
| 필요 자금 | **3억원** (보증금 충당 안 됨) |
| 보증금 수령 | 배당으로 2억 수령 (선순위 시) |

## 6. 우선매수 효과

### 6.1 낙찰자 지위

| 효과 | 내용 |
|------|------|
| 소유권 | 소유권 취득 |
| 점유 | 기존 점유 유지 |
| 권리 | 최고가 매수인과 동일 |

### 6.2 기존 권리

| 권리 | 처리 |
|------|------|
| 임차권 | 소멸 (혼동) |
| 대항력 | 불필요 |
| 보증금 | 배당 절차로 수령 |

## 7. 우선매수 포기·철회

### 7.1 포기

| 항목 | 내용 |
|------|------|
| 시기 | 매각기일 전까지 |
| 방법 | 신고 철회 또는 불신고 |
| 효과 | 최고가 매수인 낙찰 |

### 7.2 대금 미납부

| 상황 | 결과 |
|------|------|
| 미납부 | 우선매수 효력 상실 |
| 차순위 | 차순위 매수인에게 기회 |
| 제재 | 매각보증금 몰수 |

## 8. 복수 임차인

### 8.1 복수 신고 시

| 상황 | 처리 |
|------|------|
| 여러 임차인 신고 | 추첨 또는 지분 매수 |
| 협의 | 협의로 결정 가능 |

### 8.2 지분 매수

| 방법 | 내용 |
|------|------|
| 지분 분할 | 여러 명이 지분으로 매수 |
| 단독 매수 | 협의로 한 명이 매수 |

## 9. 주의사항

### 9.1 신고 기한

| 주의 | 내용 |
|------|------|
| 마감 | **매각기일 종료 전** |
| 늦으면 | 권리 상실 |

### 9.2 자금 준비

| 주의 | 내용 |
|------|------|
| 전액 준비 | 보증금 충당 안 됨 |
| 대출 확인 | 사전 대출 승인 |
| 기한 엄수 | 대금 납부 기한 준수 |

### 9.3 권리 분석

| 확인 | 내용 |
|------|------|
| 인수 부담 | 선순위 권리 확인 |
| 추가 비용 | 취득세, 등기 비용 |

## 10. 자주 묻는 질문

### 10.1 우선매수하면 유리?

| 답변 | 설명 |
|------|------|
| 상황에 따라 | 자금 여력, 거주 의사 고려 |
| 장점 | 이사 불필요, 주거 유지 |
| 단점 | 대금 전액 필요 |

### 10.2 대출 가능?

| 답변 | 설명 |
|------|------|
| ✅ 가능 | 주택담보대출 활용 |
| 주의 | 사전 심사 필수 |
| 기한 | 대금 납부일 전 실행 |

### 10.3 보증금 못 받으면?

| 답변 | 설명 |
|------|------|
| 배당 | 배당 절차로 수령 |
| 선순위 | 선순위면 전액 가능 |
| 후순위 | 남은 금액 범위 |
