# 자녀세액공제

자녀세액공제는 자녀를 양육하는 근로자의 세금 부담을 줄여주는 제도입니다.

## 자녀세액공제 개요

### 1.1 공제 유형

| 유형 | 내용 |
|------|------|
| 자녀세액공제 | 자녀 수 기준 공제 |
| 출생·입양 세액공제 | 출생·입양 시 추가 공제 |

### 1.2 공제 대상

| 대상 | 조건 |
|------|------|
| 자녀 | 만 7세 이상 ~ 만 18세 이하 |
| 손자녀 | 부양하는 경우 |
| 위탁아동 | 부양하는 경우 |

## 자녀세액공제 금액

### 2.1 자녀 수별 공제

| 자녀 수 | 공제액 |
|---------|--------|
| 1명 | **15만원** |
| 2명 | **35만원** |
| 3명 | 65만원 |
| 4명 | 95만원 |
| 5명 | 125만원 |

### 2.2 계산 공식

```
3명 이상: 35만원 + 30만원 × (자녀 수 - 2)
```

### 2.3 계산 예시

**자녀 4명**

| 항목 | 금액 |
|------|------|
| 기본 (2명분) | 35만원 |
| 추가 (2명분) | 30만원 × 2 = 60만원 |
| **합계** | **95만원** |

## 대상 자녀 요건

### 3.1 나이 요건

| 구분 | 나이 |
|------|------|
| 자녀세액공제 | 만 7세 이상 ~ 만 18세 이하 |
| 만 7세 미만 | 아동수당 지급 (별도) |

### 3.2 기준일

| 항목 | 기준 |
|------|------|
| 나이 판정 | 해당 연도 **12월 31일** |

### 3.3 나이 계산 예시

**2026년 기준**

| 출생연도 | 나이 | 대상 여부 |
|----------|------|----------|
| 2019년생 | 만 7세 |  |
| 2020년생 | 만 6세 |  (아동수당) |
| 2008년생 | 만 18세 |  |
| 2007년생 | 만 19세 |  |

## 출생·입양 세액공제

### 4.1 공제 금액

| 순서 | 공제액 |
|------|--------|
| 첫째 | **30만원** |
| 둘째 | **50만원** |
| 셋째 이상 | **70만원** |

### 4.2 적용 시기

| 항목 | 시기 |
|------|------|
| 적용 | 출생·입양한 해 |
| 1회성 | 출생·입양 연도만 적용 |

### 4.3 계산 예시

**2026년 셋째 출생**

| 항목 | 금액 |
|------|------|
| 자녀세액공제 (3명) | 65만원 |
| 출생 공제 (셋째) | 70만원 |
| **합계** | **135만원** |

## 손자녀 공제

### 5.1 대상

| 대상 | 조건 |
|------|------|
| 손자녀 | 기본공제 대상인 경우 |
| 나이 | 만 7세 이상 ~ 만 18세 이하 |

### 5.2 요건

| 요건 | 기준 |
|------|------|
| 소득 | 100만원 이하 |
| 부양 | 생계를 같이 하는 경우 |

## 중복 적용

### 6.1 중복 불가

| 상황 | 처리 |
|------|------|
| 맞벌이 부부 | 한 사람만 공제 |
| 부모·조부모 중복 | 한 사람만 공제 |

### 6.2 기본공제와 관계

| 공제 | 관계 |
|------|------|
| 기본공제 | 별도 적용 (150만원) |
| 자녀세액공제 | 세액에서 차감 |

## 다른 공제와 비교

### 7.1 자녀 관련 공제 비교

| 공제 | 내용 | 금액 |
|------|------|------|
| 기본공제 | 소득공제 | 150만원/인 |
| 자녀세액공제 | 세액공제 | 15~만원/인 |
| 교육비공제 | 세액공제 | 지출액 15% |

### 7.2 아동수당과 비교

| 구분 | 자녀세액공제 | 아동수당 |
|------|-------------|---------|
| 대상 | 만 7세 이상 | 만 7세 미만 |
| 형태 | 세금 감면 | 현금 지급 |
| 금액 | 15만원~/년 | 10만원/월 |

### 8.1 신청 방법

| 방법 | 절차 |
|------|------|
| 간소화 서비스 | 자동 반영 |
| 회사 제출 | 연말정산 시 |

### 8.2 필요 서류

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 가족관계 확인 |
| 가족관계증명서 | 자녀 확인 |
| 출생증명서 | 출생 공제 시 |
| 입양관계증명서 | 입양 공제 시 |

## 특수 상황

### 9.1 이혼 시

|------|--------|

| 양육자 | 실제 양육하는 부모 |

### 9.2 재혼 시

| 상황 | 공제 |
|------|------|
| 배우자의 자녀 | 부양 시 공제 가능 |
| 입양 시 | 입양 공제 가능 |

|------|------|

### 10.2 신고 방법

| 방법 | 절차 |

## 주의사항

### 11.1 나이 확인

| 주의 | 내용 |
|------|------|
| 만 나이 | 12월 31일 기준 |
| 만 7세 미만 | 아동수당 대상 |
| 만 19세 이상 | 공제 대상 아님 |

### 11.2 중복 주의

| 주의 | 내용 |
|------|------|
| 맞벌이 | 한 사람만 신청 |
