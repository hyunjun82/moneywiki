# 연말정산 연금소득

연금소득 과세를 정리합니다.

## 1. 연금소득 종류

| 종류 | 과세 |
|------|------|
| 국민연금 | 종합과세 |
| 공무원연금 | 종합과세 |
| 퇴직연금 | 분리/종합 |
| 개인연금 | 분리/종합 |

## 2. 연금소득공제

| 총연금액 | 공제액 |
|----------|--------|
| 350만원 이하 | 전액 |
| 350~700만원 | 350만원+초과분×40% |
| 700~1,400만원 | 490만원+초과분×20% |
| 1,400만원 초과 | 630만원+초과분×10% |

## 3. 분리과세

| 조건 | 세율 |
|------|------|
| 사적연금 1,200만원 이하 | 3.3~5.5% |
| 종합과세 | 6~45% |

## 4. 과세 제외

| 연금 | 과세 |
|------|------|
| 유족연금 | 비과세 |
| 장애연금 | 비과세 |
| 반환일시금 | 퇴직소득 |
