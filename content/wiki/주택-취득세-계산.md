# 주택 취득세 계산

## 기본 개념
주택을 취득할 때 납부하는 지방세로, 주택 수와 가격에 따라 세율이 달라집니다.

## 기본 세율

### 1주택자
| 주택 가격 | 취득세율 |
|----------|---------|
| 6억원 이하 | **1%** |
| 6억원~9억원 | **1~3%** (누진) |
| 9억원 초과 | **3%** |

### 2주택자
- 기본세율 + 조정대상지역 중과세
- 조정지역: **8%**
- 비조정지역: **1~3%**

### 3주택자
- 조정지역: **12%**
- 비조정지역: **8%**

### 법인
- **12%** (일반)
- 조정지역: **12%**

## 세부 계산

### 6억~9억원 누진 계산
```
취득세 = (6억원 × 1%) + (초과액 × 2/3억 × 1%)
```

### 예시: 8억원 주택 (1주택)
```
6억원까지: 6억 × 1% = 600만원
초과 2억원: 2억 × (2/3억) × 1% = 1.33% 적용
           2억 × 1.33% = 266.7만원
합계: 866.7만원
```

## 조정대상지역 중과세

### 조정지역 2주택
- 취득세율: **8%**
- 지방교육세: 0.4%
- 합계: **8.4%**

### 조정지역 3주택
- 취득세율: **12%**
- 지방교육세: 0.4%
- 합계: **12.4%**

### 현재 조정대상지역 (2024년 기준)
- 서울 전역
- 경기: 과천, 성남(분당·수정), 광명, 하남
- 기타 일부 지역

## 실무 계산 사례

### 사례 1: 1주택, 4억원 (비조정지역)
```
취득가액: 4억원
취득세율: 1%
취득세: 400만원
지방교육세: 16만원 (400 × 4%)
농어촌특별세: 면제
합계: 416만원
```

### 사례 2: 1주택, 10억원
```
취득가액: 10억원
6억원까지: 600만원 (1%)
6억~9억: 400만원 (2억×2%)
9억~10억: 300만원 (1억×3%)
취득세 합계: 1,300만원
지방교육세: 52만원
합계: 1,352만원
```

### 사례 3: 조정지역 2주택, 5억원
```
취득가액: 5억원
취득세율: 8%
취득세: 4,000만원
지방교육세: 160만원
합계: 4,160만원
```

### 사례 4: 조정지역 3주택, 7억원
```
취득가액: 7억원
취득세율: 12%
취득세: 8,400만원
지방교육세: 336만원
합계: 8,736만원
```

## 부대 세금

### 지방교육세
- 취득세의 **10%** (6억 이하)
- 취득세의 **10%** (6억 초과)
- 중과세 시: 취득세의 **10%**

### 농어촌특별세
- 1주택자 6억 이하: **면제**
- 2주택 이상 또는 6억 초과: 취득세의 **10%**

### 계산 예시 (1주택, 10억원)
```
취득세: 1,300만원
지방교육세: 130만원 (1,300 × 10%)
농어촌특별세: 130만원 (1,300 × 10%)
합계: 1,560만원
```

## 감면 제도

### 생애최초 주택 구입
- 취득가액 **12억원 이하**
- 취득세 **200만원 한도 면제**
- 또는 **50% 감면** (지역별)

### 신혼부부 감면
- 혼인일로부터 5년 이내
- 12억원 이하
- 50% 감면 (최대 200만원)

### 다자녀 가구
- 자녀 3명 이상
- 취득세 감면 (지자체별)

## 주택 수 계산

### 1세대 기준
- 배우자 포함
- 동거 가족 포함 (30세 미만 미혼 자녀)

### 주택 수 포함
- 아파트, 단독주택, 다가구
- 분양권 (일부 포함)
- 상속주택 (일부 제외)

### 주택 수 제외
- 60㎡ 이하 소형주택 (가액 1억원 이하)
- 3년 이상 보유 지방 주택
- 상속주택 (일시적)

## 일시적 2주택 특례

### 요건
1. 기존 주택 보유 중 신규 주택 취득
2. 기존 주택을 **3년 이내** 처분
3. 신규 주택으로 전입

### 혜택
- 신규 주택 취득 시 1주택 세율 적용
- 기존 주택 미처분 시 추징

## 납부 방법

### 납부 기한
- 잔금일로부터 **60일 이내**
- 등기 전까지 납부 권장

### 납부 방법
1. 위택스 (www.wetax.go.kr)
2. 관할 구청·시청
3. 은행 납부

### 분할 납부
- 1,000만원 초과 시
- 2개월 이내 분할 가능

## 주의사항

### 무신고 가산세
- 신고 안 하면 **20%** 가산세
- 납부 지연 시 이자 가산

### 주택 수 변동
- 취득 시점 기준 주택 수
- 잔금일 기준 판단

### 조정지역 변동
- 계약일이 아닌 잔금일 기준
- 지역 지정·해제 수시 확인

## 절세 전략

### 1. 생애최초 감면 활용
- 12억원 이하 주택
- 200만원 한도 면제

### 2. 일시적 2주택 활용
- 3년 내 기존 주택 처분
- 1주택 세율 적용

### 3. 가족 분산
- 주택 수 분산 (증여세 주의)
- 별도 세대 구성

### 4. 비조정지역 우선
- 조정지역 해제 지역
- 중과세 회피

## 실무 체크리스트

### 취득 전 확인
- [ ] 현재 주택 수 확인
- [ ] 조정대상지역 여부 확인
- [ ] 취득세율 계산
- [ ] 감면 대상 여부 확인

### 취득 후
- [ ] 60일 이내 신고·납부
- [ ] 일시적 2주택 시 3년 내 처분
- [ ] 생애최초 감면 신청
