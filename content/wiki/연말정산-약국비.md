# 연말정산 약국비

약국 약제비 의료비 공제를 정리합니다.

## 1. 공제 구분

| 구분 | 공제 유형 |
|------|----------|
| 처방약 | 의료비 세액공제 |
| 비처방약 | 신용카드 소득공제 |

## 2. 의료비 공제 대상

| 대상 | 공제 |
|------|------|
| 의사 처방약 | ✅ |
| 조제약 | ✅ |
| 비처방약(약국 구매) | ❌ |
| 영양제 | ❌ |
| 건강식품 | ❌ |

## 3. 공제율

| 공제 유형 | 공제율 |
|----------|--------|
| 의료비 | **15%** |
| 신용카드 | 15~30% |

## 4. 간소화 자료

| 항목 | 반영 |
|------|------|
| 처방약 | 자동 반영 |
| 비처방약 | 카드 사용분에 포함 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 처방전 | 의사 처방전 필요 |
| 비처방약 | 의료비 아닌 카드로 공제 |
