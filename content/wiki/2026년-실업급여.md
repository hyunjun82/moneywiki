2026년에 실업급여가 얼마나 오르는지 궁금하셨죠?

**2026년 실업급여 상한액이 68,100원으로 인상됐어요. 월로 치면 최대 약 204만원을 받을 수 있어요.**

## 2026년 실업급여 요약

| 구분 | 2025년 | 2026년 | 변동 |
|------|--------|--------|------|
| 1일 상한액 |66,048원 | **68,100원** | +2,100원 |
| 1일 하한액 | 64,192원 | **66,048원** | +1,856원 |
| 월 최대 | 약 198만원 | **약 204만원** | +6만원 |
| 월 최소 | 약 192만원 | **약 198만원** | +6만원 |

## 실업급여 상한액·하한액

### 상한액 (1일 68,100원)

평균임금의 60%가 68,100원을 넘으면 68,100원만 받아요.

```
월급 450만원 기준
1일 평균임금 = 4,500,000원 ÷ 30일 = 150,000원
60% = 90,000원

하지만 상한액 적용 → 68,100원만 지급
```

### 하한액 (1일 66,048원)

평균임금의 60%가 하한액보다 적으면 하한액을 받아요.

```
하한액 = 최저임금 × 80% × 8시간
      = 10,320원 × 0.8 × 8시간
      = 66,048원
```

### 상한액·하한액 차이

| 구분 | 1일 | 월 (30일) | 연간 (270일) |
|------|-----|----------|-------------|
| 상한액 | 68,100원 | 2,043,000원 | 18,387,000원 |
| 하한액 | 66,048원 | 1,981,440원 | 17,832,960원 |
| **차이** | **2,052원** | **61,560원** | **554,040원** |

상한액과 하한액 차이가 2,052원밖에 안 되기 때문에 사실상 대부분 비슷한 금액을 받아요.

## 실업급여 수급기간

### 가입기간 + 나이별 수급일수

| 가입기간 | 50세 미만 | 50세 이상·장애인 |
|----------|----------|-----------------|
| 1년 미만 | 120일 | 120일 |
| 1~3년 | 150일 | 180일 |
| 3~5년 | 180일 | 210일 |
| 5~10년 | 210일 | 240일 |
| 10년 이상 | 240일 | **270일** |

### 총 수령액 계산

```
총 수령액 = 1일 금액 × 수급일수

예: 50세 미만, 5년 가입, 상한액 적용
   = 68,100원 × 210일
   = 14,301,000원
```

## 실업급여 계산 예시

### 월급 300만원 (50세 미만, 3년 가입)

```
1일 평균임금 = 3,000,000원 ÷ 30일 = 100,000원
60% = 60,000원

하한액(66,048원)보다 적음 → 하한액 적용

총 수령액 = 66,048원 × 180일 = 11,888,640원
```

### 월급 500만원 (50세 이상, 10년 가입)

```
1일 평균임금 = 5,000,000원 ÷ 30일 = 166,667원
60% = 100,000원

상한액(68,100원)보다 많음 → 상한액 적용

총 수령액 = 68,100원 × 270일 = 18,387,000원
```

## 실업급여 받는 조건

### 기본 조건

| 조건 | 내용 |
|------|------|
| 고용보험 가입 | 퇴사 전 18개월 중 **180일 이상** |
| 퇴사 사유 | **비자발적** (권고사직, 계약만료 등) |
| 구직 의사 | 재취업 의지가 있어야 함 |
| 구직 능력 | 일할 수 있는 상태여야 함 |

### 자발적 퇴사도 받는 경우

- 임금 체불
- 직장 내 괴롭힘
- 사업장 이전 (통근 3시간 이상)
- 근로조건 변경
- 가족 돌봄

## 실업급여 신청 방법

### 신청 절차

1. **워크넷 구직등록**: [워크넷](https://www.work.go.kr)에서 구직등록
2. **수급자격 신청**: 퇴사 후 고용센터 방문 또는 온라인
3. **교육 이수**: 수급자격 인정 교육 (온라인 가능)
4. **실업인정**: 1-4주마다 구직활동 보고

### 준비 서류

- 신분증
- 이직확인서 (회사에서 발급)
- 통장 사본
- 구직등록 확인증

### 온라인 신청

1. [고용24](https://www.ei.go.kr) 접속
2. '실업급여' 메뉴 클릭
3. 수급자격 신청
