# 연말정산 경정청구

경정청구는 연말정산에서 **공제를 누락**했을 때 추가로 환급받는 제도입니다. [국세청 경정청구 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2443&cntntsId=7786)에 따른 방법을 정리합니다.

## 1. 경정청구란

### 1.1 개념

| 항목 | 내용 |
|------|------|
| 정의 | 세금을 **과다 납부**한 경우 환급 요청 |
| 대상 | 공제 누락, 잘못 신고한 경우 |
| 결과 | 추가 환급금 수령 |

### 1.2 경정청구 vs 수정신고

| 구분 | 경정청구 | 수정신고 |
|------|---------|---------|
| 상황 | 세금 많이 낸 경우 | 세금 적게 낸 경우 |
| 결과 | **환급** | **추가 납부** |

## 2. 경정청구 기한

[국세기본법 제45조의2](https://www.law.go.kr/법령/국세기본법)에 따른 기한입니다.

### 2.1 기본 기한

| 기준 | 기한 |
|------|------|
| 기산일 | 법정신고기한 다음 날 |
| 기간 | **5년 이내** |

### 2.2 연도별 기한 (2026년 기준)

| 귀속연도 | 법정신고기한 | 경정청구 기한 |
|---------|-------------|--------------|
| 2020년 | 2021.5.31 | **2026.5.31** |
| 2021년 | 2022.5.31 | 2027.5.31 |
| 2022년 | 2023.5.31 | 2028.5.31 |
| 2023년 | 2024.5.31 | 2029.5.31 |
| 2024년 | 2025.5.31 | 2030.5.31 |

## 3. 경정청구 대상

### 3.1 공제 누락 사례

| 항목 | 사례 |
|------|------|
| 부양가족 | 부모님 공제 누락 |
| 의료비 | 간소화 미조회 의료비 |
| 교육비 | 학원비, 교복비 누락 |
| 기부금 | 종교단체 기부금 누락 |
| 안경구입비 | 영수증 미제출 |
| 월세 | 월세 세액공제 누락 |

### 3.2 잘못된 공제

| 상황 | 경정청구 |
|------|---------|
| 소득공제 과소 계상 | ✅ 가능 |
| 세액공제 누락 | ✅ 가능 |
| 세율 잘못 적용 | ✅ 가능 |

## 4. 경정청구 방법

### 4.1 홈택스 신청

1. [홈택스](https://www.hometax.go.kr) 접속
2. **신청/제출 → 경정청구** 선택
3. 귀속연도 선택
4. 원천징수영수증 불러오기
5. 추가 공제 항목 입력
6. 증빙서류 첨부
7. 신청서 제출

### 4.2 필요 서류

| 서류 | 용도 |
|------|------|
| 근로소득 원천징수영수증 | 기존 신고 내역 |
| 공제 증빙서류 | 추가 공제 증명 |
| 가족관계증명서 | 부양가족 공제 |

### 4.3 세무서 방문

- 홈택스 이용 어려우면 **관할 세무서 방문**
- 서류 지참 후 경정청구서 작성

## 5. 처리 절차

### 5.1 처리 기간

| 단계 | 기간 |
|------|------|
| 접수 | 즉시 |
| 심사 | 1~2개월 |
| 결정 | 심사 후 결정 |
| **환급** | 결정 후 **약 2주** |

### 5.2 총 소요 기간

일반적으로 신청 후 **2~3개월** 내 환급

## 6. 환급금 수령

### 6.1 환급 방법

| 방법 | 내용 |
|------|------|
| 계좌 입금 | 신청 시 계좌 등록 |
| 우편환 | 계좌 미등록 시 |

### 6.2 환급 금액

| 항목 | 금액 |
|------|------|
| 추가 환급 세액 | 공제 누락분 |
| **환급가산금** | 이자 성격 (연 2.4%) |

## 7. 주의사항

### 7.1 증빙 필수

- 모든 공제 항목에 **증빙서류** 첨부
- 증빙 없으면 공제 불인정

### 7.2 기한 엄수

- 5년 지나면 **권리 소멸**
- 오래된 귀속연도부터 먼저 신청

### 7.3 반려 시

- 경정청구 반려 시 **이의신청** 가능
- 반려 사유 확인 후 보완 신청

## 8. 자주 하는 질문

### Q. 퇴사 후에도 경정청구 가능한가요?

**A.** 네. 개인이 직접 홈택스에서 신청 가능합니다.

### Q. 여러 해 한꺼번에 신청 가능한가요?

**A.** 네. 각 귀속연도별로 별도 신청합니다.

### Q. 경정청구 후 추가로 더 찾을 게 있으면?

**A.** 재경정청구가 가능합니다. 단, 같은 항목 중복 불가.
