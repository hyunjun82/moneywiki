# 연말정산 장애인 특수교육비

장애인 특수교육비는 장애인의 재활교육을 위해 지출한 비용에 대해 세액공제를 받을 수 있습니다. [국세청 교육비 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 공제 유형 | **교육비 세액공제** |
| 공제율 | **15%** |
| 한도 | **없음** (전액 공제) |

### 1.2 일반 교육비와 비교

| 구분 | 한도 |
|------|------|
| 초·중·고 | 300만원 |
| 대학생 | 900만원 |
| **장애인 특수교육비** | **한도 없음** |

## 2. 공제 대상

### 2.1 대상자

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 대상입니다.

| 대상 | 요건 |
|------|------|
| 본인 | 장애인 해당 |
| 배우자 | 장애인 해당 + 소득 요건 |
| 직계존속 | 장애인 해당 + 나이·소득 요건 |
| 직계비속 | 장애인 해당 + 나이·소득 요건 |
| 형제자매 | 장애인 해당 + 나이·소득 요건 |

### 2.2 장애인 범위

| 유형 | 내용 |
|------|------|
| 장애인복지법상 장애인 | 등록 장애인 |
| 국가유공자 상이자 | 상이등급 |
| 중증환자 | 항시 치료 필요 (세법상 장애인) |

## 3. 공제 대상 시설

### 3.1 사회복지시설

| 시설 | 근거법 |
|------|--------|
| 장애인복지관 | 장애인복지법 |
| 장애인거주시설 | 장애인복지법 |
| 장애인직업재활시설 | 장애인복지법 |
| 장애아전담어린이집 | 영유아보육법 |

### 3.2 특수교육기관

| 시설 | 내용 |
|------|------|
| 특수학교 | 교육과정 운영 |
| 특수학급 | 일반학교 내 |
| 특수교육지원센터 | 교육청 소속 |

### 3.3 기타 인정 시설

| 시설 | 비고 |
|------|------|
| 발달재활서비스 제공기관 | 바우처 기관 |
| 언어치료실 | 인가된 기관 |
| 감각통합치료실 | 인가된 기관 |
| 인지치료기관 | 인가된 기관 |

## 4. 공제 대상 비용

### 4.1 공제 가능

| 비용 | 공제 여부 |
|------|----------|
| 재활교육비 | ✅ 가능 |
| 치료교육비 | ✅ 가능 |
| 언어치료비 | ✅ 가능 |
| 감각통합치료비 | ✅ 가능 |
| 직업훈련비 | ✅ 가능 |
| 발달재활서비스 본인부담금 | ✅ 가능 |

### 4.2 공제 불가

| 비용 | 공제 여부 |
|------|----------|
| 일반 학원비 | ❌ 불가 |
| 의료비 (별도 공제) | ❌ 중복 불가 |
| 보조기기 구입비 | ❌ (의료비 공제) |

## 5. 공제 계산 예시

### 5.1 장애 자녀 재활교육

**장애 자녀 재활교육비 연 1,200만원**

- 한도: 없음
- 공제율: 15%
- 세액공제: 1,200만원 × 15% = **180만원**

### 5.2 복합 교육비

**장애 자녀 특수학교 + 재활치료**

| 항목 | 금액 | 한도 |
|------|------|------|
| 특수학교 등록금 | 200만원 | 300만원 (초·중·고) |
| 재활치료비 | 800만원 | 없음 |
| **총 공제 대상** | **1,000만원** | |
| **세액공제** | **150만원** | |

## 6. 증빙서류

### 6.1 필수 서류

| 서류 | 용도 |
|------|------|
| 교육비 납입증명서 | 금액 확인 |
| 장애인증명서 | 장애인 자격 확인 |

### 6.2 간소화 조회

- 일부 시설은 **간소화 자동 조회**
- 조회 안 되면 **직접 제출**

## 7. 다른 공제와 병행

### 7.1 함께 적용 가능

| 공제 | 병행 |
|------|------|
| 장애인 추가공제 (200만원) | ✅ 가능 |
| 장애인 의료비 (한도 없음) | ✅ 가능 |

### 7.2 중복 불가

| 항목 | 적용 |
|------|------|
| 동일 비용 의료비+교육비 | ❌ 택1 |

## 8. 세법상 장애인

### 8.1 중증환자 (세법상 장애인)

| 조건 | 내용 |
|------|------|
| 항시 치료 필요 | 의료기관 발급 확인서 |
| 암, 난치병 등 | 중증질환 |
| 취학, 취업 곤란 | 지속적 치료 필요 |

### 8.2 증빙

- **장애인증명서** (의료기관 발급)
- 세법상 장애인으로 인정

## 9. 주의사항

### 9.1 시설 인가 확인

- **인가된 시설**만 공제 가능
- 무인가 시설 비용은 불가

### 9.2 의료비와 구분

| 비용 성격 | 적용 공제 |
|----------|----------|
| 치료 목적 | 의료비 공제 |
| 교육 목적 | 교육비 공제 |

### 9.3 서류 보관

- 증빙서류 **5년** 보관 권장
- 경정청구 시 필요
