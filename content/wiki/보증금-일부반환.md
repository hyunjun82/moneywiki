# 보증금 일부반환

퇴거 시 보증금 전액이 아닌 일부만 돌려받는 경우가 있습니다. 적법한 공제와 부당한 공제를 구분해야 합니다.

## 1. 보증금 공제란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 보증금에서 비용 차감 |
| 근거 | 임대차계약, 민법 |
| 목적 | 채무 변제, 원상복구 |

### 1.2 공제 성격

| 성격 | 내용 |
|------|------|
| 담보 | 보증금은 채무 담보 |
| 정산 | 계약 종료 시 정산 |
| 반환 | 공제 후 잔액 반환 |

## 2. 적법한 공제 사유

### 2.1 공제 가능 항목

| 항목 | 공제 |
|------|------|
| 연체 월세 | ✅ |
| 연체 관리비 | ✅ |
| 미납 공과금 | ✅ |
| 원상복구비 (과실) | ✅ |
| 약정 위약금 | ✅ |

### 2.2 공제 불가 항목

| 항목 | 공제 |
|------|------|
| 자연 노후 수선 | ❌ |
| 통상 손모 | ❌ |
| 임대인 책임 수리 | ❌ |
| 과도한 원상복구 | ❌ |

### 2.3 공제 기준

| 기준 | 내용 |
|------|------|
| 정당 사유 | 공제 사유 존재 |
| 합리적 금액 | 실비 또는 시가 |
| 증빙 | 견적서, 영수증 |

## 3. 원상복구비 공제

### 3.1 공제 가능 범위

| 손상 | 공제 |
|------|------|
| 임차인 과실 | ✅ |
| 비정상 사용 | ✅ |
| 무단 변경 | ✅ |

### 3.2 공제 불가 범위

| 손상 | 공제 |
|------|------|
| 자연 노후 | ❌ |
| 통상 마모 | ❌ |
| 경년 열화 | ❌ |

### 3.3 감가상각 적용

| 항목 | 내용연수 |
|------|---------|
| 도배 | 3~5년 |
| 장판 | 5~8년 |
| 보일러 | 10~15년 |

**계산 예시**

```
장판 5년 사용, 임차인 과실 파손
내용연수 8년

잔존가치 = (8-5)/8 = 37.5%
교체비용 50만원 × 37.5% = 18.75만원 공제
```

## 4. 공제 절차

### 4.1 정산 협의

| 단계 | 내용 |
|------|------|
| 1 | 시설물 점검 (함께) |
| 2 | 공제 항목 확인 |
| 3 | 금액 협의 |
| 4 | 정산서 작성 |
| 5 | 잔액 반환 |

### 4.2 정산서 내용

| 항목 | 기재 |
|------|------|
| 보증금 | 총액 |
| 공제 항목 | 각 항목별 금액 |
| 공제 합계 | 총 공제액 |
| 반환 금액 | 잔액 |
| 서명 | 양측 서명 |

### 4.3 정산서 예시

```
[보증금 정산서]

1. 보증금 총액: 50,000,000원

2. 공제 항목
   - 연체 월세 (1개월): 500,000원
   - 장판 수리 (과실): 200,000원
   - 청소비: 100,000원
   - 공제 합계: 800,000원

3. 반환 금액: 49,200,000원

위 내용으로 정산에 합의합니다.

2026년 ○월 ○일
임대인: ○○○ (인)
임차인: ○○○ (인)
```
