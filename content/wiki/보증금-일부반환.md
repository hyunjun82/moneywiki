# 보증금 일부반환

퇴거할 때 보증금 전액이 아닌 일부만 돌려받는 경우가 있어요. 적법한 공제와 부당한 공제를 구분해야 해요.

## 1. 공제 가능한 것

| 항목 | 공제 |
|------|------|
| 연체 월세 | ✅ |
| 연체 관리비 | ✅ |
| 미납 공과금 | ✅ |
| 원상복구비 (임차인 과실) | ✅ |
| 약정 위약금 | ✅ |

## 2. 공제 불가능한 것

| 항목 | 공제 |
|------|------|
| 자연 노후 (경년열화) | ❌ |
| 통상 손모 (일반 사용) | ❌ |
| 임대인 책임 수리 | ❌ |
| 과도한 원상복구비 | ❌ |

**핵심**: 임차인 잘못이 아닌 자연스러운 닳음은 공제할 수 없어요.

## 3. 원상복구비 공제 기준

### 공제 가능
- 임차인 과실로 인한 손상
- 비정상적 사용으로 인한 파손
- 무단 구조 변경

### 공제 불가
- 벽지 변색 (자연 노후)
- 장판 마모 (통상 사용)
- 가전 노후 (경년열화)

### 감가상각 적용

오래 사용했으면 그만큼 감가상각해야 해요.

| 항목 | 내용연수 |
|------|---------|
| 도배 | 3~5년 |
| 장판 | 5~8년 |
| 보일러 | 10~15년 |

**예시**: 장판 5년 사용, 임차인 과실로 파손
- 내용연수 8년 기준
- 잔존가치 = (8-5)/8 = 37.5%
- 교체비 50만원 × 37.5% = **18.75만원만 공제**

## 4. 정산 절차

| 단계 | 내용 |
|------|------|
| 1 | 시설물 함께 점검 |
| 2 | 공제 항목 확인 |
| 3 | 금액 협의 |
| 4 | 정산서 작성 |
| 5 | 잔액 반환 |

**팁**: 입주할 때 사진 찍어두면 퇴거 시 분쟁 예방돼요.

## 5. 과도한 공제 대응

임대인이 터무니없는 금액을 요구하면:

| 대응 | 방법 |
|------|------|
| 증빙 요청 | 견적서, 영수증 요청 |
| 협의 | 합리적 금액 협의 |
| 분쟁조정 | [임대차분쟁조정](/w/임대차분쟁조정-신청) 신청 |
| 소송 | 보증금반환청구소송 |

## 6. 분쟁 해결

| 기관 | 연락처 |
|------|--------|
| 임대차분쟁조정위원회 | **132** |
| 한국소비자원 | 1372 |

## 7. 증거 확보

분쟁 대비 이것들을 보관하세요:

| 증거 | 내용 |
|------|------|
| 인수인계서 | 입주 시 상태 기록 |
| 사진 | 입주/퇴거 시 촬영 |
| 정산 내역 | 공제 내역 서면 |
| 대화 기록 | 협의 내용 |

## 8. 특수 상황

### 임대인 연락 두절
내용증명 보내고 [임차권등기](/w/임차권등기명령)로 대항력 유지하세요.

### 명도와 반환 동시
보증금 반환과 명도는 동시이행이에요. 보증금 안 주면 열쇠 안 줘도 돼요. (특약 있으면 예외)

## 9. 자주 묻는 질문

### 영수증 없이 공제?
거부할 수 있어요. 증빙 요청하세요.

### 퇴거 후 추가 청구?
정산 완료 후엔 제한적이에요. 정산서에 "정산 완료" 문구 넣으세요.

### 청소비 공제?
전문청소가 필요한 수준이면 가능하지만, 기본 청소는 임차인이 하면 돼요.
