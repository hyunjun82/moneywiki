# 전세계약 주의사항

전세계약은 큰 금액이 오가는 거래입니다. 전세사기 피해를 예방하기 위한 필수 확인 사항을 정리합니다.

## 1. 전세사기 유형

### 1.1 주요 사기 유형

| 유형 | 설명 |
|------|------|
| 깡통전세 | 대출+보증금 > 집값 |
| 이중계약 | 동일 주택 복수 계약 |
| 위조서류 | 가짜 등기부등본 |
| 대리인 사칭 | 소유자 아닌 자 계약 |
| 신탁 물건 | 신탁회사 동의 없는 계약 |

### 1.2 깡통전세 위험

| 상황 | 위험도 |
|------|--------|
| (근저당+보증금) > 시세 100% | 🚨 매우 위험 |
| (근저당+보증금) > 시세 80% | ⚠️ 위험 |
| (근저당+보증금) ≤ 시세 70% | ✅ 비교적 안전 |

## 2. 계약 전 필수 확인

### 2.1 등기부등본 확인

| 확인 항목 | 확인 내용 |
|----------|----------|
| 갑구 | 소유자, 가압류, 가처분 |
| 을구 | 근저당권, 전세권 |
| 신탁 여부 | 신탁원부 확인 필요 |
| 발급일 | **당일 발급본** 확인 |

### 2.2 소유자 확인

| 확인 방법 | 내용 |
|----------|------|
| 신분증 대조 | 등기부 소유자와 일치 |
| 실제 거주 | 집주인 거주 여부 |
| 대리인 | 위임장 + 인감증명 |

### 2.3 권리 분석

| 항목 | 안전 기준 |
|------|----------|
| 근저당 채권최고액 | 시세 60% 이하 |
| 선순위 보증금 | 확정일자부 열람 |
| 체납 세금 | 국세·지방세 완납증명 |

## 3. 시세 확인 방법

### 3.1 시세 조회 사이트

| 사이트 | 용도 |
|--------|------|
| 국토부 실거래가 | 실제 거래가 확인 |
| KB부동산 | KB시세 확인 |
| 네이버부동산 | 호가·시세 비교 |
| 부동산114 | 시세 정보 |

### 3.2 안전 보증금 기준

| 계산 방식 | 기준 |
|----------|------|
| 안전 보증금 | 시세 × 70% - 근저당 |
| 최대 보증금 | 시세 × 80% - 근저당 |

### 3.3 계산 예시

**시세 5억원, 근저당 1억원인 경우**

| 항목 | 금액 |
|------|------|
| 시세 | 5억원 |
| 근저당 | 1억원 |
| 안전 보증금 | 5억 × 70% - 1억 = **2.5억원** |
| 최대 보증금 | 5억 × 80% - 1억 = **3억원** |

## 4. 전세보증보험

### 4.1 보증 기관

| 기관 | 상품 |
|------|------|
| 주택도시보증공사(HUG) | 전세보증금반환보증 |
| SGI서울보증 | 전세금보장신용보험 |
| 한국주택금융공사 | 전세자금보증 |

### 4.2 HUG 전세보증금반환보증

| 항목 | 내용 |
|------|------|
| 보증 내용 | 보증금 미반환 시 대신 지급 |
| 가입 조건 | 전입신고 + 확정일자 |
| 보증료 | 보증금의 0.115~0.154%/년 |
| 보증 한도 | 수도권 7억, 기타 5억 |

### 4.3 보증보험 가입 조건

| 조건 | 기준 |
|------|------|
| 보증금 | 시세 90% 이하 |
| 선순위 | 근저당 등 포함 90% 이하 |
| 다가구 | 선순위 보증금 합산 |

### 4.4 보증료 계산

**보증금 3억원, 2년 계약**

| 항목 | 금액 |
|------|------|
| 보증금 | 3억원 |
| 보증료율 | 0.128%/년 (일반) |
| 2년 보증료 | 3억 × 0.128% × 2 = **76.8만원** |

## 5. 신탁 부동산 주의

### 5.1 신탁 물건 확인

| 확인 방법 | 내용 |
|----------|------|
| 등기부 갑구 | "신탁" 기재 확인 |
| 신탁원부 | 신탁회사에서 발급 |
| 동의서 | 신탁회사 임대 동의서 |

### 5.2 신탁 물건 위험

| 위험 | 내용 |
|------|------|
| 무효 계약 | 신탁회사 동의 없으면 무효 |
| 보호 불가 | 대항력·우선변제권 없음 |

### 5.3 안전 계약 방법

| 방법 | 내용 |
|------|------|
| 신탁회사 확인 | 임대 동의서 수령 |
| 수익자 확인 | 실제 권리자 확인 |
| 계약서 날인 | 신탁회사 또는 수익자 |

| 요건 | 내용 |
|------|------|

| 효력 | 다음 날 0시 발생 |

### 7.2 우선변제권 확보

|------|------|

## 8. 위험 신호

### 8.1 피해야 할 매물

| 위험 신호 | 설명 |
|----------|------|
| 시세 대비 저렴 | 급매 아닌데 저렴 |
| 대리인 계약 | 소유자 불참 |
| 급한 계약 요구 | 급히 결정 종용 |
| 다수 세입자 | 선순위 많음 |
| 신탁 물건 | 동의서 없는 계약 |

### 8.2 추가 확인 필요 상황

## 10. 전세사기 예방 요약

### 10.1 핵심 수칙

| 수칙 | 내용 |
|------|------|
| 1 | 등기부등본 **당일** 확인 |
| 2 | 보증금 ≤ 시세 70% |
| 3 | 전세보증보험 **필수** 가입 |
| 4 | 신탁 물건 **동의서** 확인 |
| 5 | 잔금일 **등기부 재확인** |

### 10.2 안전 거래 공식

```
안전한 전세 =
  적정 보증금(시세 70% 이하) +
  권리 확인(등기부등본) +
  보험 가입(전세보증보험) +
  대항력 확보(전입+확정일자)
```
