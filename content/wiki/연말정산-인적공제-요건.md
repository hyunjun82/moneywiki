# 인적공제 요건

## 기본 개념
본인 및 부양가족에 대해 1인당 150만원씩 소득에서 공제하는 제도입니다.

## 기본공제 대상자별 요건

### 1. 본인
- **공제액**: 150만원
- **요건**: 없음 (무조건 공제)

### 2. 배우자
- **공제액**: 150만원
- **소득요건**: 연간 소득 100만원 이하
  - 근로소득만 있는 경우: 총급여 500만원 이하
  - 사업소득: 필요경비 제외 후 100만원 이하

### 3. 직계존속 (부모님, 조부모님)
- **공제액**: 150만원
- **나이요건**: 만 60세 이상
- **소득요건**: 연간 소득 100만원 이하
- **생계요건**: 동거 불요 (따로 살아도 가능)

### 4. 직계비속 (자녀, 손자녀)
- **공제액**: 150만원
- **나이요건**: 만 20세 이하
- **소득요건**: 연간 소득 100만원 이하
- **예외**: 장애인은 나이 제한 없음

### 5. 형제자매
- **공제액**: 150만원
- **나이요건**: 만 20세 이하 또는 만 60세 이상
- **소득요건**: 연간 소득 100만원 이하
- **생계요건**: 생계를 같이해야 함

### 6. 위탁아동
- **공제액**: 150만원
- **요건**: 아동복지법상 위탁아동으로 6개월 이상 양육
- **소득요건**: 연간 소득 100만원 이하

## 소득금액 계산 기준

### 근로소득
```
총급여 500만원 이하
= 근로소득금액 100만원 이하
(근로소득공제 400만원)
```

### 사업소득
```
사업소득금액 = 총수입 - 필요경비
→ 100만원 이하여야 공제 가능
```

### 기타소득
```
기타소득금액 300만원 이하
(필요경비 60% 인정 시)
```

### 종합소득
```
근로소득 + 사업소득 + 기타소득 등
→ 합계 100만원 이하
```

## 추가공제

### 경로우대 (100만원)
- 만 70세 이상
- 기본공제 대상자에 추가

### 장애인 (200만원)
- 장애인복지법상 장애인
- 국가유공자 등 상이자
- 나이 제한 없음

### 부녀자 (50만원)
- 종합소득 3천만원 이하
- 배우자 있는 여성 근로자
- 또는 세대주인 여성 근로자

### 한부모 (100만원)
- 배우자 없는 근로자
- 기본공제대상 직계비속 또는 입양자 있는 경우

## 실무 체크리스트

### 부모님 공제 시
- [ ] 만 60세 이상 확인
- [ ] 연간 소득 100만원 이하 확인
- [ ] 형제자매 중복 공제 방지
- [ ] 국민연금 수령액 확인 (소득에 포함 안 됨)

### 자녀 공제 시
- [ ] 만 20세 이하 확인
- [ ] 아르바이트 소득 확인 (총급여 500만원 이하)
- [ ] 맞벌이 부부 중 누가 공제받을지 결정

### 장애인 공제 시
- [ ] 장애인증명서 발급
- [ ] 나이 제한 없음 확인
- [ ] 기본공제 + 장애인공제 중복 가능

## 주의사항

### 중복 공제 금지
- 형제자매가 동시에 부모님 공제 불가
- 매년 1명만 공제 가능
- 부모님이 맞벌이인 경우 자녀는 1명만 공제

### 소득 확인 방법
1. 국세청 홈택스 소득확인
2. 4대보험 가입내역 조회
3. 금융소득 확인 (이자·배당 2천만원 초과 시 종합과세)
