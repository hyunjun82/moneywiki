# 교육비공제

교육비 세액공제는 본인과 부양가족의 교육비 지출에 대해 세금을 감면받는 제도입니다.

## 교육비공제 개요

### 1.1 공제 방식

| 항목 | 내용 |
|------|------|
| 공제 유형 | **세액공제** |
| 공제율 | **15%** |
| 대상 | 본인 및 부양가족 |

### 1.2 공제 대상자 요건

| 대상 | 소득 요건 | 나이 요건 |
|------|----------|----------|
| 본인 | 없음 | 없음 |
| 배우자 | 100만원 이하 | 없음 |
| 직계비속 | 100만원 이하 | 없음 |
| 형제자매 | 100만원 이하 | 없음 |

## 공제 한도

### 2.1 대상별 한도

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 특수교육비 | **한도 없음** |
| 대학생 | **900만원** |
| 초·중·고등학생 | **300만원** |
| 취학전 아동 | **300만원** |

### 2.2 본인 교육비

| 항목 | 한도 |
|------|------|
| 대학 등록금 | 한도 없음 |
| 대학원 등록금 | 한도 없음 |
| 직업훈련비 | 한도 없음 |

|------|------|

## 계산 방법

### 5.1 계산 공식

```
교육비 세액공제 = 교육비 지출액 × 15%
```

### 5.2 계산 예시

**대학생 자녀 등록금 800만원**

| 항목 | 금액 |
|------|------|
| 등록금 지출 | 800만원 |
| 한도 | 900만원 |
| 공제 대상 | 800만원 |
| 공제율 | 15% |
| **세액공제** | **120만원** |

### 5.3 복수 자녀 예시

**초등학생 1명, 대학생 1명**

| 자녀 | 교육비 | 한도 | 공제 대상 | 세액공제 |
|------|--------|------|----------|----------|
| 초등학생 | 200만원 | 300만원 | 200만원 | 30만원 |
| 대학생 | 1,000만원 | 900만원 | 900만원 | 135만원 |
| **합계** | - | - | - | **165만원** |
