# 4대보험

4대보험(사대보험)은 국가가 운영하는 사회보험 제도입니다. 종류와 보험료를 정리합니다.

## 1. 4대보험 종류

### 1.1 보험별 목적

| 보험 | 목적 |
|------|------|
| 국민연금 | 노후 소득 보장 |
| 건강보험 | 의료비 보장 |
| 고용보험 | 실업·육아휴직 지원 |
| 산재보험 | 업무상 재해 보상 |

### 1.2 관리 기관

| 보험 | 관리 기관 |
|------|----------|
| 국민연금 | 국민연금공단 |
| 건강보험 | 국민건강보험공단 |
| 고용보험 | 근로복지공단 |
| 산재보험 | 근로복지공단 |

## 2. 보험료율 (2026년)

### 2.1 보험료율 표

| 보험 | 총 요율 | 근로자 | 사업주 |
|------|--------|--------|--------|
| 국민연금 | 9.5% | **4.75%** | **4.75%** |
| 건강보험 | 7.19% | **3.595%** | **3.595%** |
| 장기요양 | 건보의 13.14% | **6.57%** | **6.57%** |
| 고용보험 | 1.8% | **0.9%** | **0.9%+α** |
| 산재보험 | 업종별 상이 | **0%** | **전액** |

### 2.2 근로자 총 부담

| 항목 | 요율 |
|------|------|
| 국민연금 | 4.75% |
| 건강보험 | 3.595% |
| 장기요양보험 | 약 0.47% |
| 고용보험 | 0.9% |
| **합계** | **약 9.7%** |

### 2.3 월급별 보험료 예시

**월급 300만원 기준**

| 보험 | 근로자 부담 |
|------|------------|
| 국민연금 | 142,500원 |
| 건강보험 | 107,850원 |
| 장기요양 | 14,170원 |
| 고용보험 | 27,000원 |
| **합계** | **약 29.2만원** |

## 3. 국민연금

### 3.1 가입 대상

| 대상 | 가입 |
|------|------|
| 18~60세 미만 근로자 | 의무 |
| 자영업자 | 의무 |
| 전업주부 등 | 임의 |

### 3.2 보험료

| 항목 | 내용 |
|------|------|
| 총 요율 | **9.5%** |
| 본인 | 4.75% |
| 회사 | 4.75% |

2026년 연금개혁으로 9%에서 9.5%로 인상됐어요. 2033년까지 매년 0.5%p씩 올라 최종 13%가 돼요.

### 3.3 기준소득월액

| 구분 | 금액 (2026년) |
|------|--------------|
| 하한 | 37만원 |
| 상한 | 617만원 |

## 4. 건강보험

### 4.1 가입 대상

| 대상 | 가입 |
|------|------|
| 직장 근로자 | 직장가입 |
| 자영업자 | 지역가입 |
| 피부양자 | 직장가입자 피부양 |

### 4.2 보험료

| 항목 | 요율 |
|------|------|
| 건강보험 | 7.19% (각 3.595%) |
| 장기요양보험 | 건보의 13.14% |

### 4.3 급여 혜택

- 진료비 지원
- 건강검진
- 출산급여
- 요양급여

## 5. 고용보험

| 대상 | 가입 |
|------|------|
| 근로자 전원 | 의무 |
| 65세 이상 신규 | 실업급여 제외 |
| 주 15시간 미만 | 제외 가능 |

### 5.2 보험료

| 항목 | 근로자 | 사업주 |
|------|--------|--------|
| 실업급여 | 0.9% | 0.9% |
| 고용안정·직능 | - | 0.25~0.85% |

### 5.3 급여 혜택

| 급여 | 내용 |
|------|------|
| 실업급여 | 구직급여, 취업촉진수당 |

| 출산휴가급여 | 출산휴가 시 지원 |
| 직업능력개발 | 훈련비 지원 |
