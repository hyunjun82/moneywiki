퇴직금을 받았는데 제대로 계산된 건지 어떻게 알 수 있을까요. 회사에서 주는 정산서를 봐도 뭐가 뭔지 모르겠어요.

퇴직금 정산서는 **퇴직금 계산 과정**을 보여주는 서류예요. 근속연수, 평균임금, 세금 공제 내역까지 확인할 수 있어요.

정산서에 뭐가 적혀있는지, 어떻게 검토하는지 알려드릴게요.

## 퇴직금 정산서란

회사에서 퇴직금을 계산한 내역을 보여주는 서류예요.

### 정산서에 들어있는 내용

**기본 정보:**
- 성명, 사번
- 입사일, 퇴직일
- 근속연수, 재직일수

**임금 정보:**
- 퇴직 전 3개월 임금 내역
- 기본급, 수당, 상여금 등
- 평균임금 계산 내역

**퇴직금 계산:**
- 1일 평균임금
- 퇴직금 총액
- 세금 공제 내역
- 실수령액

### 정산서 vs 원천징수영수증

| 구분 | 퇴직금 정산서 | 퇴직소득원천징수영수증 |
|------|-------------|---------------------|
| 발급 주체 | 회사 | 회사 (세무서 양식) |
| 용도 | 퇴직금 계산 확인 | 세금 신고/증빙 |
| 양식 | 회사마다 다름 | 법정 양식 |
| 필수 여부 | 선택 | 필수 |

원천징수영수증은 연말정산이나 세금 신고에 필요한 공식 서류예요.

## 정산서 항목별 확인 방법이에요

정산서의 각 항목을 어떻게 확인하는지 알려드릴게요.

### 근속연수 확인

**입사일**과 **퇴직일**이 맞는지 확인하세요.

근속연수 = 퇴직일 - 입사일

**주의:** 휴직 기간이 빠져있으면 안 돼요. 대부분의 휴직은 근속연수에 포함돼요.

### 평균임금 확인

퇴직 전 3개월 임금이 정확한지 확인하세요.

**포함되어야 하는 항목:**
- 기본급
- 고정 수당 (직책수당, 자격수당 등)
- 연장근로수당, 야간수당
- 정기 상여금 (연간 상여금 ÷ 12 × 3)
- 연차수당 (연간 ÷ 12 × 3)

**빠져있으면 안 되는 것:**
- 정기적으로 받던 수당
- 통상임금에 포함되는 항목

### 퇴직금 계산 확인

**퇴직금 = 1일 평균임금 × 30일 × (재직일수 ÷ 365)**

정산서의 계산이 이 공식과 맞는지 확인하세요.

### 세금 확인

**퇴직소득세 + 지방소득세**가 공제돼요.

퇴직소득세가 너무 많이 빠진 것 같으면 계산해보세요.

## 정산서 검토 체크리스트예요

꼼꼼히 확인할 항목이에요.

### 기본 정보 체크

- [ ] 입사일이 정확한가?
- [ ] 퇴직일이 정확한가?
- [ ] 근속연수가 맞는가?
- [ ] 휴직 기간이 빠지지 않았는가?

### 임금 정보 체크

- [ ] 기본급이 맞는가?
- [ ] 받던 수당이 모두 포함됐는가?
- [ ] 상여금이 포함됐는가?
- [ ] 연차수당이 포함됐는가?

### 계산 결과 체크

- [ ] 평균임금 계산이 맞는가?
- [ ] 퇴직금 계산이 맞는가?
- [ ] 세금 공제가 적절한가?
- [ ] 실수령액이 맞는가?

## 정산서가 틀린 것 같으면요

계산이 잘못된 것 같으면 대응하세요.

### 1단계: 회사에 문의

먼저 회사 인사/총무 담당자에게 문의하세요.

**"정산서를 확인해봤는데, OO 항목이 빠진 것 같아요. 확인 부탁드려요."**

단순 실수인 경우 금방 정정해줘요.

### 2단계: 서면 요청

구두로 안 되면 서면(이메일)으로 요청하세요.

증거로 남기는 게 중요해요.

### 3단계: 노동부 문의

회사가 정정해주지 않으면 **고용노동부 1350**에 문의하세요.

무료 상담받을 수 있어요.

### 4단계: 진정 신청

그래도 안 되면 **노동부 진정**을 넣을 수 있어요.

[고용노동부 민원마당](https://minwon.moel.go.kr)에서 온라인으로 신청해요.

## 자주 빠지는 항목이에요

회사에서 자주 누락하는 항목이에요.

### 상여금

정기 상여금은 평균임금에 포함돼요.

**연간 상여금 ÷ 12 × 3** 만큼 퇴직 전 3개월 임금에 더해야 해요.

### 연차수당

미사용 연차수당도 평균임금에 포함돼요.

**연차수당 ÷ 12 × 3** 만큼 포함돼야 해요.

### 식대, 교통비

실비 변상이 아니라 고정 지급되는 식대, 교통비는 임금이에요.

평균임금에 포함돼야 해요.

### 야간수당, 연장수당

실제로 받은 야간수당, 연장수당도 평균임금에 포함돼요.

## 퇴직소득원천징수영수증도 받으세요

정산서와 별도로 **퇴직소득원천징수영수증**을 받아야 해요.

### 용도

**연말정산:** 다음 해 연말정산 시 필요해요.

**종합소득세 신고:** 다른 소득이 있으면 신고 시 필요해요.

**소득 증빙:** 대출 등 소득 증빙 시 사용해요.

### 받는 방법

퇴직금 지급 시 회사에서 발급해줘요.

안 받았으면 회사에 요청하세요.

### 홈택스에서도 확인 가능

[국세청 홈택스](https://www.hometax.go.kr) → My 홈택스 → 연말정산/지급명세서 → 지급명세서 조회

다음 해 3월 이후에 조회 가능해요.

## 정산서 양식 예시예요

일반적인 퇴직금 정산서 양식이에요.

### 기본 정보 영역

| 항목 | 내용 |
|------|------|
| 성명 | 홍길동 |
| 사번 | 12345 |
| 입사일 | 2020.01.01 |
| 퇴직일 | 2025.12.31 |
| 근속연수 | 5년 11개월 30일 |
| 재직일수 | 2,191일 |

### 임금 내역 영역

| 항목 | 1차월 | 2차월 | 3차월 | 합계 |
|------|-------|-------|-------|------|
| 기본급 | 300만원 | 300만원 | 300만원 | 900만원 |
| 식대 | 20만원 | 20만원 | 20만원 | 60만원 |
| 교통비 | 10만원 | 10만원 | 10만원 | 30만원 |
| 상여금(3/12) | - | - | - | 150만원 |
| **합계** | - | - | - | **1,140만원** |

### 퇴직금 계산 영역

| 항목 | 계산 |
|------|------|
| 3개월 임금 합계 | 1,140만원 |
| 3개월 총 일수 | 91일 |
| 1일 평균임금 | 125,275원 |
| 퇴직금 | 125,275 × 30 × (2,191÷365) = **22,556,000원** |

### 세금 공제 영역

| 항목 | 금액 |
|------|------|
| 퇴직금 | 22,556,000원 |
| 퇴직소득세 | 450,000원 |
| 지방소득세 | 45,000원 |
| **실수령액** | **22,061,000원** |

## 놓치기 쉬운 체크포인트

- 퇴직금 정산서는 **계산 내역 확인용** 서류
- **입사일, 퇴직일, 근속연수** 정확한지 확인
- **상여금, 연차수당, 고정 수당** 빠지지 않았는지 확인
- 틀리면 먼저 **회사에 정정 요청**
- **퇴직소득원천징수영수증**도 꼭 받아두기
- 홈택스에서 **다음 해 3월 이후** 조회 가능
