# 연말정산 근로소득공제

근로소득공제 계산을 정리합니다.

## 1. 공제율

| 총급여 | 공제금액 |
|--------|----------|
| 500만원 이하 | 총급여의 70% |
| 500~1,500만원 | 350만원 + 500만원 초과분의 40% |
| 1,500~4,500만원 | 750만원 + 1,500만원 초과분의 15% |
| 4,500~1억원 | 1,200만원 + 4,500만원 초과분의 5% |
| 1억원 초과 | 1,475만원 + 1억원 초과분의 2% |

## 2. 계산 예시

| 총급여 | 근로소득공제 | 근로소득금액 |
|--------|--------------|--------------|
| 3,000만원 | 975만원 | 2,025만원 |
| 5,000만원 | 1,225만원 | 3,775만원 |
| 7,000만원 | 1,325만원 | 5,675만원 |
| 1억원 | 1,475만원 | 8,525만원 |

## 3. 한도

| 총급여 | 공제 한도 |
|--------|----------|
| 1억원 초과 | 공제 한도 있음 |
| 연봉 높을수록 | 공제율 낮아짐 |

## 4. 자동 적용

| 항목 | 내용 |
|------|------|
| 신청 | 불필요 |
| 적용 | 자동 계산 |
