# 연말정산 공제한도

항목별 공제한도를 정리합니다.

## 1. 소득공제 한도

| 항목 | 한도 |
|------|------|
| 신용카드 | 200~300만원 |
| 주택청약 | 300만원 |
| 주택임차차입금 | 400만원 |
| 장기주택저당차입금 | 300~1,800만원 |
| 연금보험료 | 전액 |

## 2. 세액공제 한도

| 항목 | 한도 |
|------|------|
| 보험료 | 100만원 |
| 의료비 | 700만원 (본인 무제한) |
| 교육비 | 300~900만원 |
| 연금저축 | 600만원 |
| IRP 포함 | 900만원 |
| 월세 | 1,000만원 |

## 3. 신용카드 한도

| 총급여 | 한도 |
|--------|------|
| 7천만원 이하 | 300만원 |
| 7천만원~1.2억 | 250만원 |
| 1.2억 초과 | 200만원 |

## 4. 교육비 한도

| 대상 | 한도 |
|------|------|
| 취학 전 | 300만원 |
| 초중고 | 300만원 |
| 대학 | 900만원 |
| 본인 | 전액 |

## 5. 기부금 한도

| 유형 | 한도 |
|------|------|
| 법정기부금 | 소득 100% |
| 지정기부금 | 소득 30% |
| 종교단체 | 소득 10% |
