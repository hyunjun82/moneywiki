# 배당소득세

배당소득세는 주식 배당금에 부과되는 세금입니다. 세율과 과세 방법을 정리합니다.

## 배당소득세 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 배당금에 대한 소득세 |
| 과세 대상 | 주식 배당, 펀드 배당 등 |
| 과세 방식 | 원천징수 또는 종합과세 |

### 1.2 배당소득 종류

| 종류 | 예시 |
|------|------|
| 주식 배당금 | 국내·해외 주식 배당 |
| 펀드 분배금 | 펀드 이익 분배 |
| ETF 분배금 | ETF 배당 |
| 집합투자 분배금 | 신탁 이익 |

## 세율

### 2.1 원천징수 세율

| 항목 | 세율 |
|------|------|
| 배당소득세 | **14%** |
| 지방소득세 | **1.4%** |
| **합계** | **15.4%** |

### 2.2 원천징수 예시

**배당금 100만원 수령 시**

| 항목 | 금액 |
|------|------|
| 배당금 | 1,000,000원 |
| 배당소득세 (14%) | 140,000원 |
| 지방소득세 (1.4%) | 14,000원 |
| **실수령액** | **846,000원** |

## 금융소득종합과세

### 3.1 기준

| 항목 | 기준 |
|------|------|
| 금융소득 | 이자 + 배당 |
| 기준 금액 | 연간 **2,000만원** |
| 초과 시 | **종합과세** |

### 3.2 분리과세 vs 종합과세

| 금융소득 | 과세 방식 |
|----------|----------|
| 2,000만원 이하 | **분리과세** (15.4%) |
| 2,000만원 초과 | **종합과세** (6~45%) |

### 3.3 종합과세 계산

```
금융소득 = 이자소득 + 배당소득
2,000만원 이하 → 분리과세 15.4%
2,000만원 초과분 → 종합과세 (다른 소득과 합산)
```

### 3.4 종합과세 예시

**금융소득 3,000만원, 근로소득 5,000만원**

| 항목 | 과세 |
|------|------|
| 금융소득 2,000만원 | 분리과세 15.4% |
| 금융소득 1,000만원 | 종합과세 |
| 근로소득 5,000만원 | 종합과세 |
| 과세표준 | 6,000만원 (합산) |
| 세율 | 24% (누진) |

## Gross-up (배당세액공제)

### 4.1 Gross-up 제도

| 항목 | 내용 |
|------|------|
| 목적 | 법인세와 이중과세 조정 |
| 방법 | 배당금에 11%를 더해 소득 산정 |
| 공제 | 산출세액에서 배당세액공제 |

### 4.2 계산 방식

```
배당소득금액 = 배당금 × 111%
배당세액공제 = 배당금 × 11%
```

### 4.3 예시

**배당금 1,000만원 (종합과세 대상)**

| 항목 | 금액 |
|------|------|
| 배당금 | 1,000만원 |
| Gross-up (11%) | 110만원 |
| 배당소득금액 | 1,110만원 |
| 산출세액 계산 후 | 배당세액공제 110만원 적용 |

## 해외 배당소득

### 5.1 과세 방법

| 항목 | 내용 |
|------|------|
| 현지 원천징수 | 국가별 상이 (미국 15%) |
| 국내 과세 | 종합과세 또는 분리과세 |
| 외국납부세액공제 | 이중과세 조정 |

### 5.2 미국 배당 예시

**배당금 $1,000 (환율 1,300원)**

| 항목 | 금액 |
|------|------|
| 배당금 | $1,000 (130만원) |
| 미국 원천징수 (15%) | $150 (19.5만원) |
| 실수령 | $850 (110.5만원) |
| 한국 세금 | 종합과세 시 추가 가능 |

### 5.3 외국납부세액공제

| 항목 | 내용 |
|------|------|
| 공제 방법 | 외국에서 낸 세금 공제 |
| 한도 | 국내 세액 범위 내 |

## 절세 방법

### 6.1 비과세·저율과세 활용

| 계좌 | 혜택 |
|------|------|
| ISA | 비과세 (200~400만원) |
| 연금저축 | 과세이연 |
| IRP | 과세이연 |

### 6.2 분리과세 유지

| 방법 | 내용 |
|------|------|
| 금융소득 관리 | 2,000만원 이하 유지 |
| 부부 분산 | 배우자 명의 활용 |
| 수령 시기 조절 | 연도별 분산 |

### 6.3 주식 양도 vs 배당

| 구분 | 세금 |
|------|------|
| 배당 | 15.4% (또는 종합과세) |
| 국내주식 양도 | 대주주만 과세 |
| 해외주식 양도 | 22% (250만원 공제) |

## 신고 방법

### 7.1 분리과세 (2,000만원 이하)

| 신고 | 필요 |
|------|------|
| 별도 신고 |  불필요 |
| 원천징수 | 완료됨 |

### 7.2 종합과세 (2,000만원 초과)

| 신고 | 필요 |
|------|------|
| 종합소득세 신고 |  필수 |
| 시기 | 5월 |
| 방법 | 홈택스 |

### 7.3 필요 서류

| 서류 | 용도 |
|------|------|
| 금융소득 명세서 | 소득 확인 |
| 원천징수영수증 | 기납부 세액 |

## 배당주 투자 세금

### 8.1 국내 배당주

| 항목 | 세금 |
|------|------|
| 배당금 | 15.4% 원천징수 |
| 매매차익 | 비과세 (대주주 제외) |

### 8.2 해외 배당주

| 항목 | 세금 |
|------|------|
| 배당금 | 현지 원천징수 + 국내 종합과세 |
| 매매차익 | 22% (양도세) |

### 8.3 배당 ETF

| 유형 | 세금 |
|------|------|
| 국내상장 배당ETF | 15.4% |
| 해외상장 배당ETF | 현지 원천징수 + 종합과세 |

## 주의사항

### 9.1 금융소득 관리

| 주의 | 내용 |
|------|------|
| 2,000만원 기준 | 이자+배당 합산 |
| 종합과세 시 | 건강보험료 상승 |
| 피부양자 탈락 | 금융소득 기준 |

### 9.2 건강보험 영향

| 금융소득 | 영향 |
|----------|------|
| 1,000만원 이하 | 없음 |
| 1,000만원 초과 | 건보료 부과 |

### 9.3 기타 영향

| 항목 | 영향 |
|------|------|
| 기초연금 | 소득 산정 포함 |
| 국민건강보험 | 피부양자 자격 |
