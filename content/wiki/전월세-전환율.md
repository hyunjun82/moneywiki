# 전월세 전환율

보증금을 월세로 바꾸거나 월세를 보증금으로 바꿀 때 적용하는 비율입니다. 법정 상한을 초과할 수 없습니다.

## 1. 전월세 전환율이란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 보증금 ↔ 월세 전환 시 적용 비율 |
| 근거 | 주택임대차보호법 제7조의2 |
| 목적 | 임차인 보호, 과도한 월세 방지 |

### 1.2 적용 상황

| 상황 | 예시 |
|------|------|
| 보증금 → 월세 | 보증금 1억 줄이고 월세 추가 |
| 월세 → 보증금 | 월세 줄이고 보증금 증액 |
| 반전세 계약 | 보증금 + 월세 조합 |

## 2. 전환율 상한

### 2.1 법정 상한

| 항목 | 기준 |
|------|------|
| 상한 | 기준금리 + **2%** |
| 최대 | **연 10%** 초과 불가 |
| 근거 | 주택임대차보호법 제7조의2 |

### 2.2 2026년 적용 예시

| 기준금리 | 전환율 상한 |
|---------|-----------|
| 3.0% | 3.0% + 2% = **5.0%** |
| 3.5% | 3.5% + 2% = **5.5%** |
| 4.0% | 4.0% + 2% = **6.0%** |

※ 한국은행 기준금리는 변동될 수 있으니 계약 시점에 확인하세요.

### 2.3 상한 초과 시

| 결과 | 내용 |
|------|------|
| 초과분 무효 | 상한 초과분 효력 없음 |
| 반환 청구 | 초과 납부분 반환 가능 |

## 3. 전환 계산법

### 3.1 보증금 → 월세

**계산 공식**

```
월세 = 감액 보증금 × 전환율 ÷ 12

예시: 보증금 5천만원 감액, 전환율 5%
월세 = 5,000만원 × 5% ÷ 12 = 약 20.8만원
```

### 3.2 월세 → 보증금

**계산 공식**

```
보증금 증가액 = 감소 월세 × 12 ÷ 전환율

예시: 월세 20만원 감소, 전환율 5%
보증금 = 20만원 × 12 ÷ 5% = 4,800만원
```

### 3.3 계산 예시

**사례: 전세 3억 → 반전세 전환**

| 변경 전 | 변경 후 |
|--------|--------|
| 보증금 3억 | 보증금 2억 |
| 월세 0원 | 월세 ? |

| 계산 | 내용 |
|------|------|
| 감액 보증금 | 1억원 |
| 전환율 (5%) | 1억 × 5% ÷ 12 |
| 월세 | **약 41.7만원** |

## 4. 전환율 적용 범위

### 4.1 적용 대상

| 적용 | 대상 |
|------|------|
| 주택 | ✅ 모든 주거용 건물 |
| 오피스텔 (주거용) | ✅ 적용 |
| 상가 | ❌ 별도 규정 |

### 4.2 적용 시점

| 시점 | 적용 |
|------|------|
| 신규 계약 | 자유 합의 (상한 내) |
| 갱신 계약 | 상한 적용 |
| 증액 청구 | 상한 적용 |

### 4.3 비적용 상황

| 상황 | 적용 |
|------|------|
| 순수 보증금 인상 | 전월세상한제 적용 |
| 순수 월세 인상 | 전월세상한제 적용 |
| 신규 계약 | 상한 내 자유 합의 |

## 5. 실무 활용

### 5.1 계약서 확인

| 확인 | 내용 |
|------|------|
| 전환율 명시 | 계약서에 기재 여부 |
| 상한 준수 | 법정 상한 이내 |
| 계산 검증 | 월세 금액 적정 여부 |

### 5.2 협상 포인트

| 항목 | 협상 |
|------|------|
| 전환율 | 상한보다 낮게 협상 |
| 보증금/월세 비율 | 자금 상황에 맞게 |
| 관리비 | 포함 범위 확인 |

### 5.3 유리한 선택

| 상황 | 유리한 선택 |
|------|-----------|
| 자금 여유 | 보증금 높게 (월세 절감) |
| 자금 부족 | 보증금 낮게 (초기비용 절감) |
| 금리 하락기 | 보증금 높게 |
| 금리 상승기 | 월세 높게 (상대적) |

## 6. 주의사항

| 주의 | 대응 |
|------|------|
| 상한 초과 제안 | 거절 또는 재협상 |
| 모르고 계약 | 초과분 반환 청구 |

### 6.2 관리비 포함 여부

|------|------|

### 6.3 전환율 계약서 명시

| 권장 | 내용 |
|------|------|
| 명시 | 전환율 계약서에 기재 |
