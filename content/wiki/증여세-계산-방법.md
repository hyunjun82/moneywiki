# 증여세 계산 방법

## 기본 개념
타인으로부터 재산을 무상으로 받을 때 납부하는 세금입니다.

## 세율

### 증여세 세율표
| 과세표준 | 세율 | 누진공제 |
|----------|------|---------|
| 1억원 이하 | 10% | - |
| 1억원 ~ 5억원 | 20% | 1,000만원 |
| 5억원 ~ 10억원 | 30% | 6,000만원 |
| 10억원 ~ 30억원 | 40% | 1억 6,000만원 |
| 30억원 초과 | 50% | 4억 6,000만원 |

## 증여재산공제

### 배우자
- **6억원** 공제
- 10년간 합산

### 직계존속 (부모→자녀)
- 성인: **5,000만원**
- 미성년자: **2,000만원**
- 10년간 합산

### 직계비속 (자녀→부모)
- **5,000만원**
- 10년간 합산

### 기타 친족
- **1,000만원**
- 10년간 합산

## 계산 절차

### 1단계: 증여재산가액
```
증여받은 재산의 시가
```

### 2단계: 증여재산공제
```
증여재산가액 - 증여재산공제 = 과세표준
```

### 3단계: 산출세액
```
과세표준 × 세율 - 누진공제 = 산출세액
```

### 4단계: 납부세액
```
산출세액 - 세액공제 = 납부세액
```

## 실무 계산 사례

### 사례 1: 부모→성인 자녀 1억원
```
증여재산: 1억원
증여재산공제: 5,000만원
과세표준: 5,000만원
산출세액: 5,000만원 × 10% = 500만원
납부세액: 500만원
```

### 사례 2: 부모→성인 자녀 3억원
```
증여재산: 3억원
증여재산공제: 5,000만원
과세표준: 2억 5,000만원
산출세액: 2.5억 × 20% - 1,000만원 = 4,000만원
납부세액: 4,000만원
```

### 사례 3: 배우자 간 10억원
```
증여재산: 10억원
증여재산공제: 6억원
과세표준: 4억원
산출세액: 4억 × 20% - 1,000만원 = 7,000만원
납부세액: 7,000만원
```

### 사례 4: 10년 내 2회 증여
```
2015년: 3,000만원 증여 (공제 5,000만원 중 3,000만원 사용)
2024년: 4,000만원 증여

남은 공제: 2,000만원 (5,000 - 3,000)
과세표준: 2,000만원 (4,000 - 2,000)
산출세액: 2,000만원 × 10% = 200만원
```

## 10년 합산 규정

### 합산 방법
- 동일인으로부터 10년 이내 증여받은 재산 합산
- 증여재산공제도 10년간 합산

### 예시
```
2020년: 부모→자녀 3,000만원 (공제 5,000만원)
2024년: 부모→자녀 5,000만원

합산 증여: 8,000만원
합산 공제: 5,000만원 (1인 한도)
과세표준: 3,000만원
```

## 증여 시기별 공제

### 미성년자 (만 19세 미만)
- **2,000만원** (10년 합산)
- 성인 되면 다시 5,000만원 공제 가능

### 성인 (만 19세 이상)
- **5,000만원** (10년 합산)

### 전략
```
미성년 시: 2,000만원 증여
성인 후: 5,000만원 증여
→ 총 7,000만원 비과세
```

## 증여세 신고·납부

### 신고 기한
- 증여일이 속한 달의 **말일로부터 3개월 이내**
- 예: 1월 15일 증여 → 4월 30일까지

### 납부 방법
1. 홈택스 전자신고
2. 세무서 방문 신고
3. 세무대리인 (세무사)

### 분납
- 2,000만원 초과 시 분납 가능
- 연부연납 가능 (5년 이내)

## 증여재산 평가

### 부동산
- 기준시가 또는 감정가액
- 매매사례가액 참고

### 현금
- 증여일 기준 금액

### 주식
- 상장주식: 증여일 전후 2개월 평균가
- 비상장주식: 평가 방법 복잡 (세무사 상담)

## 증여세 절세 전략

### 공제 한도 활용
```
부모 각각 5,000만원
→ 총 1억원 비과세
```

### 10년 주기 증여
```
2024년: 5,000만원
2034년: 5,000만원
→ 총 1억원 비과세
```

### 배우자 경유 증여
```
부모→배우자: 6억원 (비과세)
배우자→자녀: 6억원 (비과세)
→ 총 12억원 (단, 부당행위 주의)
```

### 미성년자·성인 분할
```
미성년 시: 2,000만원
성인 후: 5,000만원
→ 7,000만원 비과세
```

## 주의사항

### 부당행위계산 부인
- 조세 회피 목적 증여
- 배우자 경유 증여 (단기간)
- 재산 형식적 이전

### 증여추정
- 배우자·직계존비속 간 거래
- 시가보다 낮은 가격
- 차액은 증여로 추정

### 명의신탁
- 실제 소유자와 명의자 다름
- 증여세 부과 대상

## 증여세 vs 상속세

### 증여세
- 생전에 재산 이전
- 10년 주기 공제 활용
- 계획적 절세 가능

### 상속세
- 사망 후 재산 이전
- 공제 한도 크지만 일회성
- 세율 높을 수 있음

### 전략
- 미리미리 증여로 재산 분산
- 상속세 부담 경감

## 실무 체크리스트

### 증여 전 확인
- [ ] 10년 내 증여 이력 확인
- [ ] 증여재산공제 한도 확인
- [ ] 세금 계산 및 자금 준비

### 증여 시
- [ ] 증여계약서 작성
- [ ] 재산 이전 (등기·명의변경)
- [ ] 대가 지급 증빙 (매매 시)

### 증여 후
- [ ] 3개월 이내 신고
- [ ] 세금 납부
- [ ] 증빙 서류 보관
