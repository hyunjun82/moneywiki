# 주거용 오피스텔

오피스텔의 용도에 따른 세금과 규제를 정리합니다.

## 오피스텔 분류

### 1.1 용도별 분류

| 용도 | 특징 |
|------|------|
| 업무용 | 사무실 용도 |
| 주거용 | 주택으로 사용 |

### 1.2 주거용 판단 기준

| 기준 | 주거용 |
|------|--------|
| 주민등록 |  전입신고 |
| 실제 거주 |  생활 |
| 시설 | 취사·화장실 구비 |

## 주택 수 포함

### 2.1 세금별 주택 수

| 세금 | 주거용 오피스텔 |
|------|----------------|
| 취득세 |  포함 |
| 종부세 |  포함 |
| 양도세 |  포함 |

### 2.2 주택 수 산정 시점

| 시점 | 기준 |
|------|------|
| 취득세 | 취득 시점 |
| 종부세 | 6월 1일 기준 |
| 양도세 | 양도 시점 |

## 취득세

### 3.1 업무용 오피스텔

| 항목 | 세율 |
|------|------|
| 취득세 | **4%** |
| 지방교육세 | **0.4%** |
| 농어촌특별세 | **0.2%** |
| 합계 | **4.6%** |

### 3.2 주거용 오피스텔

| 주택 수 | 세율 |
|---------|------|
| 1주택 | **1~3%** |
| 2주택 (조정) | **8%** |
| 3주택 이상 (조정) | **12%** |

### 3.3 세율 비교 예시

**5억원 오피스텔 취득**

| 용도 | 세율 | 취득세 |
|------|------|--------|
| 업무용 | 4.6% | 2,300만원 |
| 주거용 1주택 | 1.1% | 550만원 |
| 주거용 2주택 (조정) | 8.4% | 4,200만원 |

## 종합부동산세

### 4.1 과세 여부

| 용도 | 종부세 |
|------|--------|
| 업무용 |  (상업용) |
| 주거용 |  (주택분) |

### 4.2 합산 기준

| 항목 | 내용 |
|------|------|
| 과세 기준일 | 6월 1일 |
| 합산 대상 | 주택 공시가 합계 |
| 공제 | 9억원 (일반) |

### 4.3 주거용 판단

| 방법 | 기준 |
|------|------|
| 재산세 과세대장 | 주택 분류 |
| 용도 변경 | 신고 기준 |

## 양도소득세

### 5.1 주택 수 산정

| 용도 | 양도세 |
|------|--------|
| 업무용 | 일반 양도세 |
| 주거용 | 주택 양도세 |

### 5.2 주거용 중과

| 주택 수 | 중과 |
|---------|------|
| 2주택 | +20%p |
| 3주택 이상 | +30%p |

### 5.3 비과세 적용

| 조건 | 비과세 |
|------|--------|
| 1세대 1주택 |  (조건 충족 시) |
| 오피스텔 포함 2주택 |  |

## 전입신고와 세금

### 6.1 전입신고 효과

| 효과 | 내용 |
|------|------|
| 주거용 확정 | 주택 수 포함 |
| 임차인 보호 | 대항력 발생 |

### 6.2 전입신고 없을 때

| 상황 | 주택 수 |
|------|---------|
| 전입신고 없음 | 업무용 간주 가능 |
| 실제 거주 | 주거용 판단 가능 |

## 임대사업자 등록

### 7.1 등록 효과

| 효과 | 내용 |
|------|------|
| 취득세 감면 | 전용 40㎡ 이하 면제 |
| 종부세 | 합산 배제 (조건) |
| 양도세 | 중과 배제 (조건) |

### 7.2 등록 요건

| 요건 | 기준 |
|------|------|
| 임대 기간 | 8년 또는 10년 |
| 임대료 인상 | 5% 이내 |
| 등록 유지 | 의무 기간 |

## 재산세

### 8.1 용도별 재산세

| 용도 | 세율 |
|------|------|
| 업무용 | 0.25% |
| 주거용 | 0.1~0.4% (주택) |

### 8.2 과세 기준

| 항목 | 기준 |
|------|------|
| 기준일 | 6월 1일 |
| 과세표준 | 시가표준액 × 공정시장가액비율 |

## 분양권·입주권

### 9.1 분양권

| 항목 | 주택 수 |
|------|---------|
| 오피스텔 분양권 |  포함 |
| 전매 시 | 양도세 과세 |

### 9.2 입주권

| 항목 | 주택 수 |
|------|---------|
| 조합원입주권 |  포함 |

## 투자 시 고려사항

### 10.1 세금 비교

| 항목 | 업무용 | 주거용 |
|------|--------|--------|
| 취득세 | 4.6% | 1~12% |
| 종부세 |  |  |
| 양도세 | 일반 | 주택 기준 |

### 10.2 장단점

| 구분 | 업무용 | 주거용 |
|------|--------|--------|
| 취득세 | 4.6% 고정 | 변동 (1~12%) |
| 주택 수 | 미포함 | 포함 |
| 임대 수익 | 사업소득세 | 임대소득세 |

## 용도 변경

### 11.1 변경 절차

| 절차 | 내용 |
|------|------|
| 1단계 | 용도변경 신고 |
| 2단계 | 건축물대장 변경 |
| 3단계 | 세금 변경 적용 |

### 11.2 변경 시 세금

| 변경 | 세금 영향 |
|------|----------|
| 업무→주거 | 주택 수 증가 |
| 주거→업무 | 주택 수 감소 |

## 주의사항

### 12.1 세금 확인

| 주의 | 내용 |
|------|------|
| 주택 수 | 사전 확인 필수 |
| 중과 여부 | 지역 확인 |

### 12.2 실거주 증명

| 증빙 | 내용 |
|------|------|
| 전입신고 | 주민등록 |
| 공과금 | 사용 내역 |
