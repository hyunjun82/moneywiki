# 연말정산 자녀 세액공제

자녀 세액공제는 자녀를 양육하는 근로자의 **세금을 직접 줄여주는** 제도입니다. [국세청 자녀 세액공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6591&cntntsId=7870)에 따른 조건을 정리합니다.

## 1. 자녀 세액공제 금액 (2025년 귀속 기준)

[소득세법 제59조의2](https://www.law.go.kr/법령/소득세법)에 따른 자녀 수별 공제액입니다. **2025년 귀속부터 1명당 10만원씩 상향**되었습니다.

| 자녀 수 | 세액공제액 | 종전 대비 |
|---------|-----------|----------|
| 1명 | **25만원** | +10만원 |
| 2명 | **55만원** | +20만원 |
| 3명 | **95만원** | +30만원 |
| 4명 이상 | 95만원 + (추가 1명당 30만원) | +40만원~ |

## 2. 공제 대상 자녀

### 2.1 나이 요건

| 요건 | 조건 |
|------|------|
| 대상 | **만 7세 이상** 기본공제 대상 자녀 |
| 제외 | 만 7세 미만 (아동수당 수령) |

### 2.2 기본공제 대상자 요건

| 요건 | 조건 |
|------|------|
| 나이 | 만 20세 이하 |
| 소득 | 연간 소득금액 100만원 이하 |
| 관계 | 직계비속, 입양자, 위탁아동 |

## 3. 공제 계산 구조

### 3.1 기본 공제액 (2025년 귀속)

| 자녀 수 | 계산 | 공제액 |
|---------|------|--------|
| 1명 | 25만원 | **25만원** |
| 2명 | 25만원 + 30만원 | **55만원** |
| 3명 | 55만원 + 40만원 | **95만원** |
| 4명 | 95만원 + 40만원 | **135만원** |
| 5명 | 135만원 + 40만원 | **175만원** |

### 3.2 계산 예시

**만 8세, 만 12세 자녀 2명**

- 자녀 수: 2명
- 세액공제: **35만원**

## 4. 출산·입양 세액공제

### 4.1 추가 공제액

해당 연도에 출산하거나 입양한 자녀에 대해 **추가 공제**

| 순서 | 추가 공제액 |
|------|-----------|
| 첫째 | **30만원** |
| 둘째 | **50만원** |
| 셋째 이상 | **70만원** |

### 4.2 순서 판단

- 해당 연도 출산·입양 자녀가 **몇째인지** 기준
- 기존 자녀 수 포함하여 판단

### 4.3 계산 예시

**기존 자녀 1명, 올해 둘째 출산**

| 공제 항목 | 금액 |
|----------|------|
| 자녀 세액공제 (2명) | 35만원 |
| 출산 세액공제 (둘째) | 50만원 |
| **총 세액공제** | **85만원** |

## 5. 아동수당과 관계

### 5.1 중복 배제

| 구분 | 대상 |
|------|------|
| 아동수당 | 만 7세 미만 |
| 자녀 세액공제 | 만 7세 이상 |

### 5.2 7세 도달 연도

- 해당 연도에 만 7세가 되면 **자녀 세액공제 적용**
- 같은 해 아동수당과 자녀 세액공제 **동시 수령 가능**

## 6. 맞벌이 부부 공제

### 6.1 원칙

자녀를 기본공제 받는 **한 쪽 배우자만** 자녀 세액공제

| 자녀 | 기본공제 | 자녀 세액공제 |
|------|---------|--------------|
| 첫째 | 남편 | 남편 |
| 둘째 | 아내 | 아내 |

### 6.2 주의사항

- 기본공제와 자녀 세액공제는 **같은 사람**이 받아야 함
- 분리 공제 불가

## 7. 손자녀 공제

### 7.1 공제 가능

손자녀도 기본공제 대상자 요건 충족 시 **자녀 세액공제 적용**

| 요건 | 조건 |
|------|------|
| 나이 | 만 20세 이하 |
| 소득 | 연간 소득금액 100만원 이하 |
| 동거 | 생계를 같이 함 |

### 7.2 조부모가 양육

부모가 양육하지 않아 조부모가 양육하는 경우:
- 조부모가 기본공제 + 자녀 세액공제 가능

## 8. 자녀 기본공제와 구분

### 8.1 기본공제 (소득공제)

| 항목 | 내용 |
|------|------|
| 공제 유형 | 소득공제 |
| 금액 | 1인당 **150만원** |
| 대상 | 만 20세 이하 직계비속 |

### 8.2 자녀 세액공제 (세액공제)

| 항목 | 내용 |
|------|------|
| 공제 유형 | 세액공제 |
| 금액 | 1명 15만원, 2명 35만원... |
| 대상 | 만 7세 이상 기본공제 대상자 |

### 8.3 동시 적용

자녀 1명에 대해:
- 기본공제 **150만원** (소득공제)
- 자녀 세액공제 **15만원** (세액공제)
- **둘 다 적용 가능**

## 9. 주의사항

### 9.1 중복 공제 불가

- 부부가 같은 자녀에 대해 **각각 공제 불가**
- 한 명만 기본공제 + 자녀 세액공제

### 9.2 소득 있는 자녀

- 자녀 소득금액 100만원 초과 → 기본공제 대상 제외
- 자녀 세액공제도 **불가**

### 9.3 만 7세 기준

- 12월 31일 기준 만 7세 이상
- 생일 기준이 아닌 **과세연도 말 기준**
