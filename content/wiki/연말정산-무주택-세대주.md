# 연말정산 무주택 세대주

무주택 세대주 공제 혜택을 정리합니다.

## 1. 무주택 세대주 공제

| 공제 | 한도 |
|------|------|
| 주택청약 | **300만원** (소득공제) |
| 월세 | **1,000만원** (세액공제) |
| 전세대출 이자 | **400만원** (소득공제) |

## 2. 주택청약 공제

| 요건 | 기준 |
|------|------|
| 총급여 | 7천만원 이하 |
| 주택 | 무주택 세대주 |
| 공제율 | 40% |

## 3. 월세 세액공제

| 요건 | 기준 |
|------|------|
| 총급여 | 8천만원 이하 |
| 주택 | 무주택 세대주 |
| 공제율 | 15~17% |

## 4. 전세대출 공제

| 요건 | 기준 |
|------|------|
| 주택 | 무주택 세대주 |
| 대상 | 원리금 상환액 |
| 공제율 | 40% |

## 5. 무주택 확인

| 확인 방법 | 내용 |
|----------|------|
| 세대 전원 | 무주택이어야 함 |
| 분양권 | 주택 수 포함 |
| 입주권 | 주택 수 포함 |
