# 전세권 설정등기

전세권 설정등기는 등기부등본에 전세권을 기재하는 것입니다. 대항력보다 강력한 보호를 받지만 비용과 임대인 동의가 필요합니다.

## 1. 전세권이란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 정의 | 전세금을 지급하고 부동산을 사용하는 권리 |
| 성질 | **물권** (등기부에 등기) |
| 근거 | 민법 제303조~제319조 |

### 1.2 전세권 vs 임차권

| 구분 | 전세권 | 임차권 (확정일자) |
|------|--------|-----------------|
| 성질 | **물권** | **채권** |
| 등기 | 등기부 갑구 또는 을구 | 확정일자부 |
| 비용 | 등기비용 발생 | 수수료 600원 |
| 동의 | 임대인 동의 필요 | 불필요 |
| 경매권 | ✅ 직접 경매 가능 | ❌ 불가 |

### 1.3 장단점

| 장점 | 단점 |
|------|------|
| 강력한 물권 보호 | 임대인 동의 필요 |
| 직접 경매 신청 가능 | 등기 비용 발생 |
| 대항력 자동 취득 | 절차 복잡 |
| 이사해도 권리 유지 | 말소 절차 필요 |

## 2. 전세권 vs 확정일자

### 2.1 권리 비교

| 항목 | 전세권 | 확정일자 |
|------|--------|---------|
| 성립 | 등기 | 전입+확정일자 |
| 공시 | 등기부등본 | 확정일자부 |
| 대항력 | ✅ | ✅ (전입 필요) |
| 우선변제권 | ✅ | ✅ |
| 경매신청권 | ✅ | ❌ |

### 2.2 보호 차이

| 상황 | 전세권 | 확정일자 |
|------|--------|---------|
| 이사 후 | 권리 유지 | 대항력 상실 |
| 경매 시 | 직접 경매 가능 | 지급명령/소송 필요 |
| 제3자 대항 | 누구에게나 대항 | 전입 유지 필요 |

### 2.3 어떤 것을 선택?

| 상황 | 권장 |
|------|------|
| 일반적인 경우 | **확정일자** (간편, 저비용) |
| 고액 보증금 | 전세권 검토 |
| 이사 가능성 | 전세권 유리 |
| 임대인 동의 어려움 | 확정일자 |

## 3. 전세권 설정 절차

### 3.1 설정 요건

| 요건 | 내용 |
|------|------|
| 합의 | 임대인·임차인 합의 |
| 등기 | 설정등기 신청 |
| 전세금 | 전세금 지급 |

### 3.2 필요 서류

| 서류 | 발급처 |
|------|--------|
| 등기신청서 | 인터넷등기소 |
| 등기원인증서 | 전세권설정계약서 |
| 임대인 인감증명서 | 주민센터 |
| 임대인 주민등록초본 | 주민센터 |
| 임차인 주민등록등본 | 주민센터 |
| 등기권리증 | 소유자 보관 |
| 취득세 영수증 | 시군구청 |

### 3.3 설정 절차

| 단계 | 내용 |
|------|------|
| 1 | 전세권 설정 합의 |
| 2 | 서류 준비 |
| 3 | 취득세 납부 |
| 4 | 등기소 신청 (온라인/방문) |
| 5 | 등기 완료 (3~5일) |

## 4. 등기 비용

### 4.1 비용 구성

| 항목 | 금액 |
|------|------|
| 등록면허세 | 전세금의 **0.2%** |
| 지방교육세 | 등록면허세의 20% |
| 등기수수료 | 약 15,000원 |
| 증지대 | 약 3,000원 |

### 4.2 비용 계산 예시

**보증금 3억원 기준**

| 항목 | 계산 | 금액 |
|------|------|------|
| 등록면허세 | 3억 × 0.2% | 600,000원 |
| 지방교육세 | 60만 × 20% | 120,000원 |
| 등기수수료 | - | 15,000원 |
| 합계 | - | **약 73.5만원** |

### 4.3 누가 부담?

| 부담 | 통상 |
|------|------|
| 설정 비용 | **임차인** |
| 말소 비용 | **임대인** |
| 협의 | 특약으로 조정 가능 |

## 5. 전세권 효력

### 5.1 기본 효력

| 효력 | 내용 |
|------|------|
| 사용·수익권 | 부동산 사용 권리 |
| 대항력 | 제3자에게 대항 |
| 우선변제권 | 경매 시 우선 배당 |
| 경매신청권 | 직접 경매 신청 |

### 5.2 경매신청권

| 항목 | 내용 |
|------|------|
| 권리 | 전세금 미반환 시 직접 경매 신청 |
| 장점 | 소송 없이 경매 진행 |
| 방법 | 법원에 경매 신청 |

### 5.3 존속 기간

| 기간 | 내용 |
|------|------|
| 약정 기간 | 계약서 기재 기간 |
| 최단 | 규정 없음 |
| 최장 | 10년 (갱신 가능) |

## 6. 전세권 변동

### 6.1 양도

| 항목 | 내용 |
|------|------|
| 가능 | 전세권 양도 가능 |
| 동의 | 임대인 동의 불필요 (약정 없으면) |
| 등기 | 이전등기 필요 |

### 6.2 전전세

| 항목 | 내용 |

| 가능 | 전전세 가능 (범위 내) |

### 6.3 소멸

| 사유 | 설명 |
|------|------|
| 기간 만료 | 존속 기간 종료 |
| 합의 | 당사자 합의 해지 |
| 소멸 시효 | 전세금반환청구권 10년 |

## 8. 전세권과 대항력 병행

| 답변 | 설명 |
|------|------|
| ✅ 가능 | 중복 취득 가능 |
| 효과 | 더 강한 보호 |

## 9. 자주 묻는 질문

### 9.1 임대인이 꼭 동의해야?

| 답변 | 설명 |
|------|------|
| ✅ 필수 | 소유자 동의 없이 불가 |
| 거부 시 | 확정일자로 대체 |

### 9.2 전세권 있으면 전입신고 필요 없나요?

| 답변 | 설명 |
|------|------|
| 주거 목적 | 전입신고 권장 |
| 소액임차인 | 전입 있어야 최우선변제 |
| 주민등록 | 생활 편의상 필요 |

| 절차 | 내용 |
|------|------|
| 1 | 전세금 반환 청구 (내용증명) |
| 2 | 반환 거절 또는 미이행 |
| 3 | 법원에 경매 신청 |
| 4 | 경매 진행, 배당 |
