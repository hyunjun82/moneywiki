# 출산휴가

출산휴가(출산전후휴가)는 임신과 출산으로 인한 휴가를 보장하는 제도입니다. 조건과 급여를 정리합니다.

## 1. 출산전후휴가 개요

### 1.1 휴가 기간

| 구분 | 기간 |
|------|------|
| 단태아 | **90일** |
| 다태아 (쌍둥이 이상) | **120일** |

### 1.2 출산 후 사용 의무

| 구분 | 출산 후 최소 기간 |
|------|-----------------|
| 단태아 | **45일** 이상 |
| 다태아 | **60일** 이상 |

### 1.3 분할 사용

| 조건 | 분할 |
|------|------|
| 유산·사산 위험 | 출산 전 44일 한도 분할 가능 |
| 일반 | 연속 사용 원칙 |

## 2. 출산휴가 급여

### 2.1 급여 지급 주체

| 기간 | 지급 주체 |
|------|----------|
| 최초 60일 | 사업주 (우선지원대상기업은 고용보험) |
| 이후 30일 | 고용보험 |

### 2.2 우선지원대상기업

| 기간 | 지급 주체 |
|------|----------|
| 전체 90일 | **고용보험** |

### 2.3 급여 금액

| 항목 | 금액 |
|------|------|
| 지급액 | 통상임금 **100%** |
| 상한액 | 월 **210만원** |
| 하한액 | 최저임금 |

### 2.4 급여 계산 예시

| 통상임금 | 급여 (월) |
|----------|----------|
| 200만원 | 200만원 |
| 250만원 | 210만원 (상한) |
| 180만원 | 180만원 |

## 3. 신청 방법

### 3.1 신청 절차

| 단계 | 내용 |
|------|------|
| 1 | 사업주에게 휴가 신청 |
| 2 | 휴가 시작 후 급여 신청 |
| 3 | 고용센터 또는 온라인 신청 |

### 3.2 신청 기한

| 항목 | 기한 |
|------|------|
| 급여 신청 | 휴가 종료 후 **12개월 이내** |

### 3.3 필요 서류

| 서류 | 용도 |
|------|------|
| 출산전후휴가 확인서 | 사업주 발급 |
| 통상임금 확인 서류 | 급여 산정 |
| 통장 사본 | 급여 지급 |

## 4. 유산·사산 휴가

### 4.1 휴가 기간

| 임신 기간 | 휴가 |
|----------|------|
| 11주 이내 | 5일 |
| 12~15주 | 10일 |
| 16~21주 | 30일 |
| 22~27주 | 60일 |
| 28주 이후 | **90일** |

### 4.2 급여

| 항목 | 내용 |
|------|------|
| 지급 | 출산휴가 급여와 동일 |
| 상한액 | 월 210만원 |

## 5. 배우자출산휴가

### 5.1 휴가 기간

| 항목 | 기간 |
|------|------|
| 기간 | **10일** |
| 유급 | ✅ |

### 5.2 사용 기한

| 항목 | 기한 |
|------|------|
| 사용 기한 | 출산일로부터 **90일** 이내 |
| 분할 사용 | 1회 분할 가능 |

### 5.3 급여

| 항목 | 내용 |
|------|------|
| 최초 5일 | 유급 (사업주 지급) |
| 이후 5일 | 고용보험 지급 |
| 지급액 | 통상임금 100% |
| 상한액 | 월 210만원 |

## 6. 우선지원대상기업

### 6.1 기준

| 업종 | 상시 근로자 수 |
|------|--------------|
| 제조업 | 500인 이하 |
| 건설업 | 300인 이하 |
| 운수·창고업 | 300인 이하 |
| 기타 | 100인 이하 |

### 6.2 혜택

| 혜택 | 내용 |
|------|------|
| 출산휴가 급여 | 90일 전액 고용보험 지급 |
| 사업주 부담 | 없음 |

## 7. 임신기 근로시간 단축

### 7.1 대상

| 대상 | 단축 |
|------|------|
| 임신 12주 이내 | 2시간 단축 |
| 임신 36주 이후 | 2시간 단축 |

### 7.2 조건

| 항목 | 내용 |
|------|------|
| 임금 | 삭감 금지 |
| 신청 | 근로자 신청 시 |

## 8. 불이익 금지

### 8.1 금지 사항

| 금지 | 내용 |
|------|------|
| 해고 | ❌ 휴가 기간 + 30일 |
| 불리한 처우 | ❌ |
| 임금 삭감 | ❌ |

### 8.2 복직 보장

| 항목 | 내용 |
|------|------|
| 복직 | 원직 복귀 보장 |
| 동등 수준 | 같은 수준 업무 |

## 9. 고용보험 미가입자

### 9.1 비적용 대상

| 대상 | 급여 |
|------|------|
| 고용보험 미가입 | 사업주 지급 |
| 자영업자 | 별도 제도 없음 |

### 9.2 대안

| 대안 | 내용 |
|------|------|
| 지자체 지원 | 지역별 출산 지원금 |
| 국민행복카드 | 임신·출산 의료비 |

## 10. 주의사항

### 10.1 신청 시 유의

| 주의 | 내용 |
|------|------|
| 기한 엄수 | 12개월 이내 신청 |
| 서류 확인 | 사업주 확인서 필수 |

### 10.2 급여 관련

| 주의 | 내용 |
|------|------|
| 중복 수급 | 실업급여와 중복 불가 |
| 상한액 | 월 210만원 한도 |
