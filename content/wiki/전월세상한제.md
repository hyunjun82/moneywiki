# 전월세상한제

전월세상한제는 세입자를 급격한 임대료 인상으로부터 보호하는 제도입니다. 2020년 임대차3법의 핵심 내용입니다.

## 1. 전월세상한제란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 제도 | 임대료 인상을 **5%**로 제한 |
| 대상 | 계약 **갱신** 시 |
| 근거 | 주택임대차보호법 제7조 |

### 1.2 시행 배경

| 항목 | 내용 |
|------|------|
| 시행일 | 2020년 7월 31일 |
| 목적 | 세입자 주거 안정 |
| 명칭 | 임대차 3법 중 하나 |

## 2. 적용 범위

### 2.1 적용 대상

| 대상 | 적용 |
|------|------|
| 계약갱신청구권 행사 | ✅ 적용 |
| 합의 갱신 | ✅ 적용 |
| 묵시적 갱신 | ⚠️ 미적용 (논란) |
| **신규 계약** | ❌ 미적용 |

### 2.2 적용 시기

| 시기 | 적용 |
|------|------|
| 계약 갱신 시점 | ✅ |
| 최초 계약 | ❌ |
| 재계약 (새 계약) | ❌ |

### 2.3 적용 대상 주택

| 주택 | 적용 |
|------|------|
| 아파트 | ✅ |
| 다세대·빌라 | ✅ |
| 단독주택 | ✅ |
| 오피스텔 (주거용) | ✅ |
| 상가 | ❌ (상가임대차법 적용) |

## 3. 인상 한도

### 3.1 5% 상한

| 항목 | 한도 |
|------|------|
| 보증금 인상 | **5% 이내** |
| 월세 인상 | **5% 이내** |
| 각각 적용 | 보증금, 월세 별도 계산 |

### 3.2 계산 기준

| 기준 | 내용 |
|------|------|
| 기준 금액 | 직전 계약 금액 |
| 계산 방식 | 직전 금액 × 5% |

## 4. 인상 한도 계산

### 4.1 보증금 계산

**현재 보증금 3억원**

| 항목 | 금액 |
|------|------|
| 현재 보증금 | 3억원 |
| 5% 상한 | 3억 × 5% = 1,500만원 |
| 최대 갱신 보증금 | **3억 1,500만원** |

### 4.2 월세 계산

**현재 월세 100만원**

| 항목 | 금액 |
|------|------|
| 현재 월세 | 100만원 |
| 5% 상한 | 100만 × 5% = 5만원 |
| 최대 갱신 월세 | **105만원** |

### 4.3 보증금 + 월세 계산

**보증금 1억, 월세 80만원**

| 항목 | 계산 |
|------|------|
| 보증금 5% | 1억 × 5% = 500만원 |
| 월세 5% | 80만 × 5% = 4만원 |
| 최대 | 보증금 **1억 500만원**, 월세 **84만원** |

## 5. 전월세 전환 시

### 5.1 전환율 제한

| 항목 | 기준 |
|------|------|
| 법정 전환율 | 한국은행 기준금리 + 2% |
| 상한 | 연 **10%** |
| 2026년 기준금리 | 약 3% (변동) |

### 5.2 전환 계산

**보증금 3억원을 월세로 전환 (전환율 5%)**

| 계산 | 금액 |
|------|------|
| 보증금 1억 전환 | 1억 × 5% ÷ 12 = **약 42만원/월** |
| 결과 | 보증금 2억 + 월세 42만원 |

### 5.3 전환 시 5% 상한

| 적용 | 내용 |
|------|------|
| 총액 기준 | 환산보증금 기준 5% |
| 환산 | 보증금 + (월세 × 100) |

## 6. 5% 초과 시 효과

### 6.1 초과분 무효

| 결과 | 내용 |
|------|------|
| 효력 | 5% 초과분 **무효** |
| 유효 금액 | 5%까지만 유효 |

### 6.2 반환 청구

| 상황 | 조치 |
|------|------|
| 이미 지급 | 초과분 반환 청구 |
| 청구 기간 | 계약 기간 중 |

### 6.3 계산 예시

**5% 초과 인상 시**

| 항목 | 금액 |
|------|------|
| 현재 보증금 | 3억원 |
| 요구 보증금 | 3억 3천만원 (10% 인상) |
| 유효 금액 | 3억 1,500만원 (5%) |
| 무효 금액 | **1,500만원** |

## 7. 예외 상황

### 7.1 상한제 미적용

| 상황 | 미적용 이유 |
|------|-----------|
| 신규 계약 | 갱신이 아님 |
| 임차인 변경 | 새로운 계약 |
| 임대인 실거주 후 재임대 | 새로운 계약 |

### 7.2 묵시적 갱신

| 논란 | 내용 |
|------|------|
| 적용 여부 | 해석 논란 있음 |
| 판례 | 미적용으로 보는 견해 |
| 실무 | 협의로 결정 |

### 7.3 특약으로 배제 가능?

| 결론 | 내용 |
|------|------|
| 불가 | 강행규정으로 배제 불가 |
| 특약 효력 | 무효 |

## 8. 상한제 계산기

### 8.1 보증금 상한 계산

```
최대 보증금 = 현재 보증금 × 1.05

예) 2억원 → 2억 1,000만원
```

### 8.2 월세 상한 계산

```
최대 월세 = 현재 월세 × 1.05

예) 50만원 → 52.5만원
```

### 8.3 환산보증금 계산

```
환산보증금 = 보증금 + (월세 × 100)

예) 1억 + (50만 × 100) = 1억 5천만원
```

## 9. 분쟁 해결

### 9.1 협의 안 될 때

| 방법 | 기관 |
|------|------|
| 조정 신청 | 임대차분쟁조정위원회 |
| 연락처 | 132 (대한법률구조공단) |

### 9.2 분쟁 예방

| 방법 | 내용 |
|------|------|
| 서면 명시 | 갱신 조건 계약서에 |
| 증거 보관 | 협의 내용 기록 |

## 10. 자주 묻는 질문

### 10.1 매년 5%씩 올릴 수 있나요?

| 답변 | 설명 |
|------|------|
| ⚠️ 아니요 | 계약 갱신 시만 적용 |
| 갱신 주기 | 보통 2년마다 |
| 계산 | 2년간 총 5% 한도 |

### 10.2 5% 인상을 거부할 수 있나요?

| 답변 | 설명 |
|------|------|
| ⚠️ 협의 필요 | 5% 이내면 정당한 인상 |
| 동결 요구 | 법적 근거 없음 |
| 협상 | 가능하나 강제 불가 |

### 10.3 인하도 가능한가요?

| 답변 | 설명 |
|------|------|
| ✅ 가능 | 하한 제한 없음 |
| 방법 | 협의로 인하 |
