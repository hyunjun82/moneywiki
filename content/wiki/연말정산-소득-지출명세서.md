소득·지출명세서는 연말정산에 필요한 공제 자료를 확인할 수 있는 문서예요. [국세청 간소화서비스](https://www.hometax.go.kr)에서 조회할 수 있어요.

## 조회 경로

[소득세법 제165조](https://www.law.go.kr/법령/소득세법)에 따라 간소화서비스가 제공돼요.

| 방법 | 경로 |
|------|------|
| 홈택스 | 연말정산 → 간소화서비스 |
| 손택스 | 조회/발급 → 연말정산 |

PC와 모바일 앱 모두 이용할 수 있어요.

## 조회 항목

간소화서비스에서 조회 가능한 항목이에요.

| 항목 | 내용 |
|------|------|
| 보험료 | 건강, 고용, 보장성보험 |
| 의료비 | 병원, 약국, 안경 |
| 교육비 | 학교, 유치원, 학원 |
| 신용카드 | 카드, 현금, 직불 |
| 주택자금 | 청약, 대출 이자 |
| 기부금 | 기부금 영수증 |
| 연금저축 | 연금저축, IRP |

## 조회 기간

간소화서비스 일정이에요.

| 시기 | 내용 |
|------|------|
| 1월 15일 | 간소화서비스 오픈 |
| 1월 15일~17일 | 자료 수정 요청 기간 |
| 1월 18일 | 수정 자료 반영 |
| 1월 20일 이후 | 안정적 조회 권장 |

초반에는 자료 누락이나 오류가 있을 수 있어서 1월 20일 이후에 조회하는 게 좋아요.

## 누락 자료 처리

간소화서비스에 누락된 자료가 있을 수 있어요.

| 누락 항목 | 처리 방법 |
|----------|----------|
| 의료비 | 병원에 직접 요청 |
| 기부금 | 기부처에 영수증 요청 |
| 월세 | 계약서, 이체내역 제출 |
| 교육비 | 학교에 납입증명서 요청 |

1월 17일까지 간소화서비스에서 수정 요청을 할 수도 있어요.

## 자료 다운로드

간소화 자료를 PDF로 다운로드해서 회사에 제출해요.

| 방법 | 내용 |
|------|------|
| PDF 다운로드 | 홈택스에서 직접 저장 |
| 회사 시스템 연동 | 일부 회사는 자동 연동 |
| 출력 제출 | 종이로 인쇄해서 제출 |

회사에서 요구하는 제출 방식을 확인하세요.

## 주의사항

간소화 자료가 모든 공제 항목을 포함하지 않아요. 누락된 자료는 직접 챙겨야 해요.

부양가족 자료를 조회하려면 미리 동의를 받아야 해요. 홈택스에서 부양가족 동의 신청을 하면 돼요.

이직한 경우 전 직장 원천징수영수증을 별도로 발급받아야 해요.
