# 연말정산 소득 지출명세서

소득·지출명세서 확인 방법을 정리합니다.

## 1. 조회 경로

| 방법 | 경로 |
|------|------|
| 홈택스 | 연말정산 → 간소화서비스 |
| 손택스 | 조회/발급 → 연말정산 |

## 2. 조회 항목

| 항목 | 내용 |
|------|------|
| 보험료 | 건강, 고용, 보장성보험 |
| 의료비 | 병원, 약국, 안경 |
| 교육비 | 학교, 유치원, 학원 |
| 신용카드 | 카드, 현금, 직불 |
| 주택자금 | 청약, 대출 이자 |
| 기부금 | 기부금 영수증 |

## 3. 조회 기간

| 시기 | 내용 |
|------|------|
| 1월 15일~ | 간소화서비스 오픈 |
| 1월 18일~ | 수정 자료 반영 |

## 4. 누락 자료

| 누락 | 처리 |
|------|------|
| 의료비 | 병원에 직접 요청 |
| 기부금 | 기부처에 영수증 요청 |
| 월세 | 계약서, 이체내역 제출 |
