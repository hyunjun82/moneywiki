# 임대차보호법

주택임대차보호법은 주거용 건물 세입자를 보호하는 법률입니다. [주택임대차보호법](https://www.law.go.kr/법령/주택임대차보호법)의 핵심 내용을 정리합니다.

## 1. 법의 적용 범위

### 1.1 적용 대상

| 대상 | 적용 |
|------|------|
| 아파트 | ✅ |
| 다세대·연립 | ✅ |
| 단독주택 | ✅ |
| 오피스텔 (주거용) | ✅ |
| 고시원 | ⚠️ 일부 |
| 상가 | ❌ (상가임대차보호법) |

### 1.2 적용 조건

| 조건 | 내용 |
|------|------|
| 용도 | 주거 목적 |
| 계약 | 임대차 계약 |

## 2. 대항력

### 2.1 대항력 요건

| 요건 | 내용 |
|------|------|
| 주택 인도 | 실제 거주 (입주) |
| 전입신고 | 주민등록 이전 |
| 효력 발생 | **다음 날 0시** |

### 2.2 대항력의 효과

| 상황 | 보호 |
|------|------|
| 집주인 변경 | 새 집주인에게 권리 주장 |
| 경매 | 낙찰자에게 권리 주장 |
| 가압류 | 대항력 있으면 보호 |

### 2.3 전입신고 시점

```
계약 → 잔금 지급 → 입주 → 전입신고 → 다음 날 0시 대항력 발생
```

⚠️ **주의**: 잔금일 당일 전입신고 권장

## 3. 우선변제권

### 3.1 요건

| 요건 | 내용 |
|------|------|
| 대항력 | 주택 인도 + 전입신고 |
| 확정일자 | 계약서에 확정일자 |

### 3.2 확정일자 받는 곳

| 장소 | 방법 |
|------|------|
| 주민센터 | 계약서 지참 방문 |
| 등기소 | 방문 신청 |
| 공인중개사 | 전자계약 시 자동 |

### 3.3 효과

| 상황 | 보호 |
|------|------|
| 경매 | 보증금 우선 변제 |
| 순위 | 확정일자 선후 기준 |

## 4. 소액임차인 최우선변제권

### 4.1 보증금 기준 (2026년)

| 지역 | 보증금 기준 | 최우선변제액 |
|------|-----------|-------------|
| 서울 | 1.65억원 이하 | **5,500만원** |
| 수도권 과밀억제 | 1.45억원 이하 | **4,800만원** |
| 광역시 | 8,500만원 이하 | **2,800만원** |
| 기타 지역 | 7,500만원 이하 | **2,500만원** |

### 4.2 효과

| 특징 | 내용 |
|------|------|
| 최우선 | 다른 채권자보다 먼저 |
| 한도 | 낙찰가의 1/2 범위 |

## 5. 계약갱신청구권

### 5.1 제도 개요

| 항목 | 내용 |
|------|------|
| 사용 횟수 | **1회** |
| 갱신 기간 | **2년** |
| 총 거주 기간 | 최초 2년 + 갱신 2년 = **4년** |

### 5.2 행사 방법

| 시기 | 방법 |
|------|------|
| 기간 | 계약 만료 **6개월~1개월 전** |
| 방법 | 서면 또는 구두 |

### 5.3 임대인 거절 사유

| 사유 | 거절 가능 |
|------|----------|
| 임대인 실거주 | ✅ |
| 재건축·재개발 | ✅ |
| 2기 차임 연체 | ✅ |
| 임차인 동의 하 전대 | ✅ |
| 무단 전대 | ✅ |
| 고의 파손 | ✅ |

## 6. 전월세상한제

### 6.1 적용 시기

| 적용 | 시기 |
|------|------|
| 계약 갱신 시 | ✅ |
| 신규 계약 | ❌ |

### 6.2 인상 한도

| 항목 | 한도 |
|------|------|
| 보증금 | **5%** 이내 |
| 월세 | **5%** 이내 |

### 6.3 계산 예시

**보증금 3억원 → 갱신**

| 항목 | 금액 |
|------|------|
| 현재 보증금 | 3억원 |
| 5% 상한 | 1,500만원 |
| **최대 갱신 보증금** | **3억1,500만원** |

## 7. 묵시적 갱신

### 7.1 요건

| 조건 | 내용 |
|------|------|
| 임대인 | 만료 6개월~1개월 전 갱신 거절 없음 |
| 임차인 | 만료 1개월 전까지 이사 의사 없음 |

### 7.2 효과

| 항목 | 내용 |
|------|------|
| 갱신 기간 | **2년** |
| 조건 | 동일 조건 |
| 임차인 해지권 | **3개월** 전 통지로 해지 가능 |

## 8. 임차권등기명령

### 8.1 제도 개요

| 항목 | 내용 |
|------|------|
| 목적 | 이사 후에도 대항력 유지 |
| 조건 | 보증금 미반환 상태 |

### 8.2 신청 방법

| 방법 | 내용 |
|------|------|
| 신청처 | 관할 법원 |
| 비용 | 등록면허세, 수수료 |

### 8.3 효과

| 효과 | 내용 |
|------|------|
| 대항력 유지 | 이사해도 권리 유지 |
| 우선변제권 | 그대로 유지 |

## 9. 전월세신고제

### 9.1 신고 대상

| 조건 | 신고 |
|------|------|
| 보증금 6,000만원 초과 | ✅ |
| 월세 30만원 초과 | ✅ |

### 9.2 신고 기한

| 항목 | 기한 |
|------|------|
| 신규 계약 | **30일 이내** |
| 변경 계약 | **30일 이내** |

### 9.3 신고 방법

| 방법 | 내용 |
|------|------|
| 주민센터 | 방문 신고 |
| 온라인 | 부동산거래관리시스템 |
