# 연말정산 신용카드 체크카드 비교

카드 종류별 공제율과 효율적인 사용 전략을 정리합니다.

## 1. 공제율 비교

| 결제수단 | 공제율 |
|----------|--------|
| 신용카드 | **15%** |
| 체크카드 | **30%** |
| 현금영수증 | **30%** |
| 전통시장 | **40%** |
| 대중교통 | **40%** |
| 도서·공연 | **30%** |

## 2. 공제 계산 순서

| 순서 | 내용 |
|------|------|
| 1 | 총급여의 25% 최저사용금액 |
| 2 | 신용카드 사용분 먼저 차감 |
| 3 | 체크카드·현금 사용분 공제 |

## 3. 효율적인 전략

| 시기 | 전략 |
|------|------|
| 1~6월 | 신용카드로 25% 채우기 |
| 7~12월 | 체크카드·현금 사용 |

## 4. 공제한도

| 총급여 | 기본한도 |
|--------|----------|
| 7천만원 이하 | 300만원 |
| 7천만원~1.2억 | 250만원 |
| 1.2억 초과 | 200만원 |

## 5. 추가한도

| 항목 | 추가한도 |
|------|----------|
| 전통시장 | +100만원 |
| 대중교통 | +100만원 |
| 도서·공연 | +100만원 |
