# 연말정산 맞벌이 절세 전략

맞벌이 부부는 공제 항목을 **어떻게 분담**하느냐에 따라 환급액이 달라집니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 절세 전략을 정리합니다.

## 1. 기본 원칙

### 1.1 중복 공제 불가

| 항목 | 중복 공제 |
|------|----------|
| 같은 부양가족 | ❌ 불가 |
| 같은 의료비 영수증 | ❌ 불가 |
| 배우자 기본공제 | ❌ 불가 (맞벌이) |

### 1.2 배우자 공제

맞벌이 부부는 서로를 **기본공제 대상자로 등록 불가**
(소득금액 100만원 초과)

## 2. 부양가족 공제 전략

### 2.1 원칙

**소득이 높은 배우자**가 부양가족 공제 → 절세 효과 큼

| 세율 | 기본공제 150만원 효과 |
|------|---------------------|
| 15% | 22.5만원 |
| 24% | 36만원 |
| 35% | 52.5만원 |

### 2.2 자녀 분담

자녀가 여러 명이면 **나눠서 공제** 가능

| 자녀 | 아빠 | 엄마 |
|------|------|------|
| 첫째 | ✅ | - |
| 둘째 | - | ✅ |

### 2.3 주의사항

- 자녀 세액공제도 **기본공제 받는 사람**이 적용
- 교육비도 기본공제 받는 사람이 공제

## 3. 의료비 공제 전략

### 3.1 원칙

**소득이 낮은 배우자**가 의료비 공제 → 3% 기준 넘기 쉬움

| 총급여 | 3% 기준 |
|--------|---------|
| 3,000만원 | 90만원 |
| 5,000만원 | 150만원 |
| 7,000만원 | 210만원 |

### 3.2 특례

의료비는 **부양가족 요건 무관**하게 공제 가능

| 대상 | 조건 |
|------|------|
| 맞벌이 배우자 의료비 | ✅ 가능 |
| 소득 있는 부모님 | ✅ 가능 |

### 3.3 전략

- 부부 중 **한 명에게 몰아서** 의료비 지출
- 3% 초과분만 공제되므로 한 명이 많이 쓰는 게 유리

## 4. 신용카드 공제 전략

### 4.1 원칙

**소득이 낮은 배우자** 카드로 지출 → 25% 기준 넘기 쉬움

| 총급여 | 25% 기준 |
|--------|----------|
| 3,000만원 | 750만원 |
| 5,000만원 | 1,250만원 |
| 7,000만원 | 1,750만원 |

### 4.2 배우자 카드

**본인 명의 카드**로 결제한 금액만 공제

| 상황 | 공제 |
|------|------|
| 남편 카드로 남편 명의 결제 | 남편 공제 |
| 아내 카드로 아내 명의 결제 | 아내 공제 |

### 4.3 전략

- 가계 지출을 **한 명 카드**로 집중
- 25% 기준을 넘긴 후에는 **체크카드** 사용 (30% 공제)

## 5. 항목별 분담 요약

### 5.1 소득 높은 배우자

| 항목 | 이유 |
|------|------|
| 부양가족 기본공제 | 소득공제 효과 큼 |
| 자녀 세액공제 | 기본공제와 연동 |
| 교육비 세액공제 | 기본공제와 연동 |
| 보험료 세액공제 | 피보험자 기본공제 필요 |

### 5.2 소득 낮은 배우자

| 항목 | 이유 |
|------|------|
| 의료비 | 3% 기준 넘기 쉬움 |
| 신용카드 | 25% 기준 넘기 쉬움 |
| 연금저축 | 공제율 15% (5,500만원 이하) |

## 6. 계산 예시

### 6.1 상황

- 남편: 총급여 7,000만원
- 아내: 총급여 4,000만원
- 자녀 2명, 시부모 2명 (70세)

### 6.2 최적 분담

| 항목 | 남편 | 아내 |
|------|------|------|
| 자녀 2명 기본공제 | ✅ | - |
| 시부모 기본공제 | ✅ | - |
| 자녀 교육비 | ✅ | - |
| 의료비 | - | ✅ |
| 신용카드 | - | ✅ |

### 6.3 절세 효과

| 항목 | 절세액 |
|------|--------|
| 부양가족 4명 (남편, 24%) | 약 144만원 |
| 의료비 200만원 (아내, 3%↓) | 약 22만원 |
| 카드 공제 (아내, 25%↓) | 추가 공제 |

## 7. 주의사항

### 7.1 기본공제 대상자 확인

- 부양가족을 공제받으면 **관련 세액공제도** 그 사람이 받음
- 기본공제 없이 의료비·교육비만 공제 불가 (의료비 예외)

### 7.2 연도 중 변동

- 연도 중 출산, 이직 등 변동 시 조정 필요
- 12월 31일 기준으로 판단

### 7.3 경정청구

- 분담을 잘못한 경우 **5년 내 경정청구**로 정정 가능
