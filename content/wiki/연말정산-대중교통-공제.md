# 연말정산 대중교통 공제

대중교통 이용금액은 **40% 공제율**로 높은 혜택을 받습니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 공제 조건을 정리합니다.

## 1. 대중교통 공제율

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제율입니다.

| 사용처 | 공제율 |
|--------|--------|
| **대중교통** | **40%** |
| 전통시장 | 40% |
| 체크카드/현금영수증 | 30% |
| 신용카드 | 15% |

**결제 수단 무관**: 신용카드, 체크카드, 현금영수증 모두 **40%** 적용

## 2. 대중교통 공제 대상

### 2.1 포함 대상

| 교통수단 | 공제 여부 |
|----------|----------|
| 시내버스 | ✅ 가능 |
| 마을버스 | ✅ 가능 |
| 지하철 | ✅ 가능 |
| 고속버스 | ✅ 가능 |
| 시외버스 | ✅ 가능 |
| KTX, SRT | ✅ 가능 |
| ITX, 무궁화호 | ✅ 가능 |

### 2.2 제외 대상

| 교통수단 | 공제 여부 |
|----------|----------|
| 택시 | ❌ 불가 |
| 항공기 | ❌ 불가 |
| 자가용 주유비 | ❌ 불가 |
| 고속도로 통행료 | ❌ 불가 |

## 3. 공제 한도

### 3.1 기본 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

### 3.2 추가 한도

기본 한도 외 **추가 공제** 가능:

| 항목 | 추가 한도 |
|------|-----------|
| 대중교통 | **100만원** |
| 전통시장 | 100만원 |
| 도서·공연·영화 | 100만원 |

**총 합계 한도**: 추가 공제 항목 합산 최대 **300만원**

## 4. 공제 조건

### 4.1 최저 사용 금액

**총급여의 25% 초과 사용분**부터 공제

| 총급여 | 25% 기준액 |
|--------|------------|
| 4,000만원 | 1,000만원 초과분 |
| 5,000만원 | 1,250만원 초과분 |
| 6,000만원 | 1,500만원 초과분 |

## 5. 공제 계산 예시

**연봉 5,000만원, 대중교통 300만원 사용**

- 전체 카드 사용: 1,750만원
- 25% 기준: 1,250만원
- 공제 대상: 500만원 (1,750만원 - 1,250만원)
- 대중교통 300만원 × 40% = **120만원 소득공제**

## 6. 대중교통비 조회

### 6.1 연말정산 간소화

1월 15일부터 [홈택스 간소화 서비스](https://www.hometax.go.kr)에서:
- 대중교통 사용금액 **자동 구분** 조회

### 6.2 교통카드 사용내역

- 티머니, 캐시비 등 교통카드 홈페이지에서 조회 가능
- 신용카드 명세서에서도 확인 가능

## 7. 절세 전략

### 7.1 대중교통 적극 활용

출퇴근, 출장 시 대중교통 이용:
- 100만원 × 40% = **40만원 공제** (신용카드 대비 25만원 추가)

### 7.2 추가 한도 활용

기본 한도 300만원 채운 후:
- 대중교통 추가 사용 → **별도 100만원 한도**

### 7.3 교통카드 등록

교통카드 번호를 홈택스에 등록하면 **자동 집계**
