# 연말정산 투잡

투잡 근로자 연말정산을 정리합니다.

## 1. 투잡 신고 방법

| 방법 | 내용 |
|------|------|
| 방법 1 | 주된 직장에서 합산 |
| 방법 2 | 5월 종합소득세 신고 |

## 2. 주된 직장 합산

| 절차 | 내용 |
|------|------|
| 1 | 부업 원천징수영수증 수령 |
| 2 | 주된 직장에 제출 |
| 3 | 합산 연말정산 |

## 3. 5월 종소세 신고

| 절차 | 내용 |
|------|------|
| 1 | 양쪽 원천징수영수증 확보 |
| 2 | 홈택스 종소세 신고 |
| 3 | 합산 세금 정산 |

## 4. 세금 영향

| 항목 | 내용 |
|------|------|
| 합산 소득 | 세율 상승 가능 |
| 추가 납부 | 발생 가능성 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 미신고 | 가산세 부과 |
| 공제 중복 | 한 곳에서만 가능 |
