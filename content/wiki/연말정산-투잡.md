투잡 근로자 연말정산을 정리해 볼게요. 두 군데 이상에서 근무하는 경우 신고 방법이 달라요.

## 투잡 신고 방법

두 가지 방법이 있어요.

| 방법 | 내용 |
|------|------|
| 방법 1 | 주된 직장에서 합산 |
| 방법 2 | 5월 종합소득세 신고 |

## 주된 직장 합산

주된 직장에서 합산하는 절차예요.

| 절차 | 내용 |
|------|------|
| 1 | 부업 원천징수영수증 수령 |
| 2 | 주된 직장에 제출 |
| 3 | 합산 연말정산 |

## 5월 종소세 신고

5월 종합소득세 신고 절차예요.

| 절차 | 내용 |
|------|------|
| 1 | 양쪽 원천징수영수증 확보 |
| 2 | 홈택스 종소세 신고 |
| 3 | 합산 세금 정산 |

## 세금 영향

투잡이 세금에 미치는 영향이에요.

| 항목 | 내용 |
|------|------|
| 합산 소득 | 세율 상승 가능 |
| 추가 납부 | 발생 가능성 |

## 주의사항

주의할 점이에요.

| 주의 | 내용 |
|------|------|
| 미신고 | 가산세 부과 |
| 공제 중복 | 한 곳에서만 가능 |
