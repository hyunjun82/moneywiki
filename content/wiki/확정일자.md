# 확정일자

확정일자는 임대차계약서에 공적 기관이 날짜를 확인해주는 제도입니다. 보증금 보호를 위한 필수 절차입니다.

## 1. 확정일자 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 확정일자 | 계약서에 공적 기관이 찍는 날짜 도장 |
| 목적 | 우선변제권 확보 |
| 효력 | 보증금 우선 회수 권리 |

### 1.2 필요성

| 상황 | 보호 |
|------|------|
| 집주인 파산 | 보증금 우선 회수 |
| 경매 | 배당 순위 확보 |
| 가압류 | 다른 채권자보다 우선 |

## 2. 전입신고와 확정일자

### 2.1 차이점

| 구분 | 전입신고 | 확정일자 |
|------|----------|----------|
| 효력 | **대항력** | **우선변제권** |
| 의미 | 임대차 주장 권리 | 보증금 우선 회수 |
| 발급처 | 주민센터 | 주민센터, 등기소 |
| 비용 | 무료 | **600원** |

### 2.2 대항력

| 조건 | 효력 |
|------|------|
| 전입신고 + 점유 | 새 집주인에게 임대차 주장 |
| 효력 발생 | 전입신고 다음 날 **0시** |

### 2.3 우선변제권

| 조건 | 효력 |
|------|------|
| 전입신고 + 확정일자 + 점유 | 보증금 우선 변제 |
| 배당 순위 | 확정일자 기준 |

## 3. 확정일자 받는 법

### 3.1 발급 장소

| 장소 | 내용 |
|------|------|
| 주민센터 | 가장 일반적 |
| 등기소 | 방문 발급 |
| 공증사무소 | 공증과 함께 |
| 온라인 | 전월세신고 시 자동 |

### 3.2 필요 서류

| 서류 | 용도 |
|------|------|
| 임대차계약서 원본 | 확정일자 날인 |
| 신분증 | 본인 확인 |

### 3.3 비용

| 항목 | 비용 |
|------|------|
| 확정일자 | **600원** |
| 전월세신고 시 | **무료** (자동 부여) |

### 3.4 처리 시간

| 방법 | 시간 |
|------|------|
| 방문 | 즉시 |
| 전월세신고 | 신고 완료 시 |

## 4. 효력 발생

### 4.1 요건

| 요건 | 설명 |
|------|------|
| 전입신고 | 주민등록 이전 |
| 확정일자 | 계약서에 날인 |
| 점유 | 실제 거주 |

### 4.2 효력 발생 시점

| 항목 | 시점 |
|------|------|
| 대항력 | 전입신고 다음 날 0시 |
| 우선변제권 | 전입신고 다음 날 0시 (확정일자 이후) |

### 4.3 선순위 확인

| 순위 | 확인 |
|------|------|
| 저당권 | 등기부등본 확인 |
| 다른 임차인 | 확정일자 부여 현황 |
| 국세·지방세 | 체납 여부 |

## 5. 경매 시 보호

### 5.1 배당 순위

| 순위 | 채권자 |
|------|--------|
| 1순위 | 임금채권, 소액임차인 |
| 2순위 | 당해세 (재산세 등) |
| 3순위 | 저당권·확정일자 (선순위) |
| 4순위 | 기타 채권자 |

### 5.2 소액임차인 보호

| 지역 | 보증금 기준 | 최우선변제 |
|------|------------|----------|
| 서울 | 1억 6,500만원 이하 | 5,500만원 |
| 수도권 과밀 | 1억 4,500만원 이하 | 4,800만원 |
| 광역시 등 | 8,500만원 이하 | 2,800만원 |
| 기타 | 7,500만원 이하 | 2,500만원 |

### 5.3 최우선변제권

| 조건 | 보호 |
|------|------|
| 소액임차인 | 경매 시 최우선 배당 |
| 저당권보다 후순위라도 | 일정 금액 보호 |

## 6. 전월세신고와 확정일자

### 6.1 자동 부여

| 조건 | 확정일자 |
|------|----------|
| 전월세신고 완료 | **자동 부여** |
| 별도 신청 | 불필요 |
| 비용 | 무료 |

### 6.2 신고 대상

| 조건 | 신고 의무 |
|------|----------|
| 보증금 6천만원 초과 | ✅ |
| 월세 30만원 초과 | ✅ |

## 7. 주의사항

### 7.1 계약서 보관

| 주의 | 내용 |
|------|------|
| 원본 보관 | 확정일자 받은 원본 보관 |
| 분실 시 | 확정일자부 열람으로 확인 |

### 7.2 계약 갱신 시

| 상황 | 조치 |
|------|------|
| 보증금 변경 | **새로 확정일자** 필요 |
| 묵시적 갱신 | 기존 확정일자 유효 |
| 계약서 재작성 | 새 확정일자 필요 |

### 7.3 전입신고 먼저

| 순서 | 권장 |
|------|------|
| 1순위 | 전입신고 |
| 2순위 | 확정일자 |
| 같은 날 | 동시 처리 권장 |

## 8. 확정일자부 열람

### 8.1 열람 내용

| 내용 | 확인 |
|------|------|
| 확정일자 현황 | 해당 주소의 확정일자 |
| 선순위 임차인 | 먼저 확정일자 받은 임차인 |

### 8.2 열람 방법

| 방법 | 장소 |
|------|------|
| 방문 | 주민센터, 등기소 |
| 온라인 | 대법원 인터넷등기소 |

### 8.3 열람 비용

| 항목 | 비용 |
|------|------|
| 열람 | 300원 |

## 9. 임차권등기명령

### 9.1 용도

| 상황 | 용도 |
|------|------|
| 보증금 미반환 | 이사 후에도 대항력 유지 |
| 다른 곳 이사 | 전입신고 말소 대비 |

### 9.2 신청 방법

| 절차 | 내용 |
|------|------|
| 관할 법원 | 신청서 제출 |
| 비용 | 인지대 + 송달료 |
| 효력 | 등기 후 대항력 유지 |
