# 다단계, 보험설계사도 실업급여 받을 수 있나요?

"보험설계사는 개인사업자라서 실업급여 못 받는 거 아니야?" 많이들 궁금해하시죠. **2021년 7월부터 특수형태근로종사자(특고)도 고용보험에 가입할 수 있어요!**

## 핵심 요약

- 보험설계사, 학습지교사, 골프장 캐디 등 **12개 직종** 고용보험 가입 가능
- 2021년 7월부터 단계적으로 적용 확대
- 고용보험 가입 180일 이상이면 **실업급여 수급 가능**
- 본인 부담 보험료: 소득의 **0.8%**

## 비자발적 이직 사유 (특고)

### 실업급여 받을 수 있는 경우

| 사유 | 설명 |
|------|------|
| **계약 해지** | 회사가 일방적으로 계약 종료 |
| **소득 50% 감소** | 이직 전 대비 소득 절반 이하 |
| **업무 변경** | 본질적 업무 내용 변경 |
| **근로조건 악화** | 수수료율 인하 등 |
| **질병·부상** | 업무 수행 곤란 (의사 소견서) |
| **임신·출산·육아** | 육아휴직 불가 시 |
| **가족 간호** | 장기간 간호 필요 |

### 실업급여 못 받는 경우

| 사유 | 설명 |
|------|------|
| **단순 자진 해지** | 개인 사정으로 스스로 그만둔 경우 |
| **다른 회사로 이직** | 더 좋은 조건으로 옮긴 경우 |
| **중대 귀책사유** | 회사에 손해를 끼친 경우 |

## 실업급여 신청 절차

### 1단계: 사전 준비

| 서류 | 설명 |
|------|------|
| 위탁계약서 | 계약 내용 확인 |
| 해지통보서 | 회사가 발송 |
| 소득증빙 | 수수료 내역서 등 |
| 신분증 | 본인 확인용 |

### 2단계: 신청 절차

1. **워크넷 구직등록**
   - [워크넷](https://www.work.go.kr) 접속
   - 특고 구직등록

2. **수급자격 신청**
   - 고용센터 방문 or 온라인
   - 이직확인서 제출 (회사)

3. **수급자격 교육**
   - 온라인 또는 방문 교육 이수

4. **실업인정**
   - 1~4주마다 구직활동 보고

5. **실업급여 수령**
   - 지정 계좌로 입금
