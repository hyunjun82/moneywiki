# 원천징수 영수증 발급

## 기본 개념
근로소득을 지급하는 회사가 원천징수한 세금 내역을 증명하는 서류입니다.

## 원천징수 영수증 종류

### 근로소득 원천징수영수증
- 가장 일반적인 형태
- 연말정산 결과 반영
- 모든 근로자 대상

### 사업소득 원천징수영수증
- 프리랜서, 사업자
- 3.3% 원천징수 내역

### 기타소득 원천징수영수증
- 강연료, 원고료 등
- 단발성 소득

## 발급 의무자

### 회사 (원천징수의무자)
- 근로소득 지급하는 모든 사업자
- 연말정산 완료 후 발급
- **매년 3월 10일까지** 근로자에게 교부

### 발급 방법
- 종이 문서
- 이메일 (PDF)
- 회사 인트라넷

## 발급 시기

### 정기 발급
- **매년 3월 10일까지**
- 전년도 연말정산 결과
- 2월에 퇴사해도 3월에 발급

### 중도 퇴사 시
- 퇴사일로부터 **1개월 이내**
- 퇴사 정산 결과 반영
- 다음 직장 연말정산용

### 수시 발급
- 근로자 요청 시
- 대출, 비자 등 용도
- 수수료 없음

## 영수증 내용

### 기본 정보
- 근로자 인적사항 (성명, 주민번호)
- 회사 정보 (상호, 사업자번호)
- 근무기간

### 소득 정보
- 총급여액
- 비과세 소득
- 과세 소득

### 공제 정보
- 소득공제 (인적공제, 연금보험료 등)
- 세액공제 (보험료, 의료비, 교육비 등)
- 기납부세액

### 결정세액
- 산출세액
- 기납부세액
- 차감징수세액 (환급액)

## 활용 용도

### 이직 시 연말정산
- 새 직장에 제출
- 전 직장 소득과 합산
- 연말정산 정산

### 대출 신청
- 소득 증빙
- 재직 증명
- 신용평가

### 비자 신청
- 해외 비자 소득 증명
- 번역본 필요 시 있음

### 종합소득세 신고
- 프리랜서 겸업
- 2곳 이상 근무
- 5월 신고 시 사용

### 정부 지원금 신청
- 근로장려금
- 자녀장려금
- 각종 복지 혜택

## 미발급 시 대처

### 회사에 요청
- 유선, 이메일로 요청
- 발급 의무 안내
- 1주일 이내 발급 권장

### 홈택스 발급
- 국세청 홈택스 접속
- My 홈택스 → 연말정산·지급명세서
- PDF 다운로드

### 관할 세무서 문의
- 회사가 불응 시
- 세무서에 신고
- 과태료 부과 요청 가능

## 홈택스 발급 방법

### 단계별 절차
1. **홈택스 접속** (www.hometax.go.kr)
2. **로그인** (공동인증서, 간편인증)
3. **My 홈택스** 클릭
4. **연말정산·지급명세서** 선택
5. **지급명세서 등 제출내역** 클릭
6. **귀속연도** 선택
7. **PDF 다운로드**

### 온라인 발급 장점
- 24시간 발급 가능
- 회사 협조 불필요
- 무료
- 즉시 출력 가능

## 중도 퇴사자 처리

### 퇴사 즉시 발급
```
1월 입사 → 6월 퇴사
7월까지 영수증 발급
새 직장 연말정산 시 제출
```

### 연말정산 미완료 퇴사
```
1월 입사 → 1월 퇴사 (연말정산 전)
→ 회사에서 퇴사 정산 후 발급
→ 새 직장 또는 5월 종합소득세 신고 시 사용
```

## 영수증 확인 사항

### 필수 확인
- [ ] 인적사항 정확성
- [ ] 총급여액 확인
- [ ] 공제 항목 확인
- [ ] 결정세액 확인
- [ ] 회사 직인 확인 (종이 문서)

### 오류 발견 시
1. 회사에 정정 요청
2. 수정 영수증 재발급
3. 국세청 홈택스 확인

## 분실 시 재발급

### 재발급 방법
1. **회사 요청**: 회사에 재발급 요청
2. **홈택스 발급**: 온라인 즉시 발급
3. **세무서 방문**: 관할 세무서에서 확인

### 재발급 수수료
- 일반적으로 무료
- 회사 내규에 따라 다름
- 홈택스는 항상 무료

## 주의사항

### 개인정보 보호
- 주민번호 등 민감정보 포함
- 보안 주의
- 필요 시 마스킹 처리

### 위변조 주의
- 공식 경로로만 발급
- 홈택스 발급 권장
- 회사 직인 확인

### 보관 기한
- **5년간 보관** 권장
- 세무조사 대비
- 소득 증빙용

## 실무 팁

1. **미리 받기**: 3월 초 회사에 요청
2. **홈택스 활용**: 즉시 필요 시 온라인 발급
3. **사본 보관**: 원본 및 사본 여러 부 보관
4. **PDF 저장**: 디지털 파일로 보관
