# 연말정산 추가공제 종류

인적공제의 추가공제 종류를 정리합니다.

## 1. 추가공제 금액

| 구분 | 공제액 |
|------|--------|
| 경로우대 | **100만원** |
| 장애인 | **200만원** |
| 부녀자 | **50만원** |
| 한부모 | **100만원** |

## 2. 경로우대 공제

| 요건 | 내용 |
|------|------|
| 나이 | 만 70세 이상 |
| 대상 | 기본공제 대상자 |
| 금액 | 100만원 |

## 3. 장애인 공제

| 요건 | 내용 |
|------|------|
| 대상 | 장애인복지법상 장애인 |
| 범위 | 중증질환자 포함 |
| 금액 | 200만원 |

## 4. 부녀자 공제

| 요건 | 내용 |
|------|------|
| 대상 | 배우자 있는 여성 |
| 또는 | 부양가족 있는 여성 세대주 |
| 소득 | 종합소득 3천만원 이하 |
| 금액 | 50만원 |

## 5. 한부모 공제

| 요건 | 내용 |
|------|------|
| 대상 | 배우자 없이 자녀 부양 |
| 금액 | 100만원 |
| 주의 | 부녀자 공제와 중복 불가 |
