인적공제의 추가공제 종류를 정리해 볼게요. [소득세법 제51조](https://www.law.go.kr/법령/소득세법)에 따른 내용이에요.

## 추가공제 금액

대상별 공제 금액이에요.

| 구분 | 공제액 |
|------|--------|
| 경로우대 | 100만원 |
| 장애인 | 200만원 |
| 부녀자 | 50만원 |
| 한부모 | 100만원 |

## 경로우대 공제

만 70세 이상 부양가족에게 적용돼요.

| 요건 | 내용 |
|------|------|
| 나이 | 만 70세 이상 |
| 대상 | 기본공제 대상자 |
| 금액 | 100만원 |

## 장애인 공제

장애인 부양가족에게 적용돼요.

| 요건 | 내용 |
|------|------|
| 대상 | 장애인복지법상 장애인 |
| 범위 | 중증질환자 포함 |
| 금액 | 200만원 |

## 부녀자 공제

근로소득이 있는 여성에게 적용돼요.

| 요건 | 내용 |
|------|------|
| 대상 | 배우자 있는 여성 |
| 또는 | 부양가족 있는 여성 세대주 |
| 소득 | 종합소득 3천만원 이하 |
| 금액 | 50만원 |

## 한부모 공제

배우자 없이 자녀를 부양하는 경우 적용돼요.

| 요건 | 내용 |
|------|------|
| 대상 | 배우자 없이 자녀 부양 |
| 금액 | 100만원 |
| 주의 | 부녀자 공제와 중복 불가 |

## 중복 적용

추가공제 중복 적용 여부예요.

| 구분 | 중복 가능 여부 |
|------|---------------|
| 경로우대 + 장애인 | 가능 |
| 경로우대 + 부녀자 | 가능 |
| 장애인 + 부녀자 | 가능 |
| 부녀자 + 한부모 | 불가 (한부모 우선 적용) |
