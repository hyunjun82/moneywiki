# 주택연금

주택연금은 집을 담보로 맡기고 평생 연금을 받는 제도입니다. 가입 조건과 수령액을 정리합니다.

## 1. 주택연금 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 주택 담보로 평생 연금 수령 |
| 다른 이름 | 역모기지론 |
| 운영 기관 | 한국주택금융공사 |

### 1.2 특징

| 특징 | 내용 |
|------|------|
| 평생 수령 | 사망 시까지 |
| 거주 보장 | 계속 거주 가능 |
| 배우자 승계 | 배우자도 계속 수령 |

## 2. 가입 조건

### 2.1 연령 조건

| 대상 | 조건 |
|------|------|
| 주택 소유자 | **55세 이상** |
| 배우자 | 연령 제한 없음 |

### 2.2 주택 조건

| 조건 | 기준 |
|------|------|
| 주택 가격 | 공시가격 **12억원 이하** |
| 주택 유형 | 주택, 노인복지주택, 주거용 오피스텔 |
| 소유 형태 | 단독 소유 또는 부부 공동 |

### 2.3 기타 조건

| 조건 | 내용 |
|------|------|
| 우선변제권 설정 | 주택에 1순위 근저당 |
| 다주택자 | 합산 12억 이하 시 가능 |
| 거주 | 실제 거주 필요 (일부 예외) |

## 3. 연금 수령 방식

### 3.1 수령 유형

| 유형 | 내용 |
|------|------|
| 종신지급 | 평생 동일 금액 |
| 종신혼합 | 일부 목돈 + 평생 연금 |
| 확정기간 | 정해진 기간 수령 |
| 대출상환 | 기존 대출 상환 후 연금 |

### 3.2 종신지급 방식

| 방식 | 내용 |
|------|------|
| 정액형 | 매월 동일 금액 |
| 증가형 | 매년 3% 증가 |
| 감소형 | 매년 3% 감소 |

### 3.3 우대형

| 대상 | 우대 |
|------|------|
| 기초연금 수급자 | 월 지급액 **최대 20%** 추가 |
| 조건 | 1.5억원 이하 주택 |

## 4. 월 수령액

### 4.1 종신지급 정액형 예시 (2026년)

| 주택가격 | 60세 | 65세 | 70세 | 75세 | 80세 |
|----------|------|------|------|------|------|
| 2억원 | 42만 | 52만 | 65만 | 83만 | 109만 |
| 3억원 | 63만 | 78만 | 98만 | 124만 | 164만 |
| 5억원 | 105만 | 130만 | 163만 | 207만 | 273만 |
| 7억원 | 147만 | 182만 | 228만 | 290만 | 382만 |
| 10억원 | 210만 | 260만 | 327만 | 414만 | 546만 |

### 4.2 수령액 영향 요인

| 요인 | 영향 |
|------|------|
| 나이 | 고령일수록 많음 |
| 주택가격 | 비쌀수록 많음 |
| 수령 방식 | 유형별 차이 |

## 5. 신청 방법

### 5.1 신청 절차

| 단계 | 내용 |
|------|------|
| 1 | 상담 (콜센터 1688-8114) |
| 2 | 주택금융공사 지사 방문 |
| 3 | 신청서 작성·제출 |
| 4 | 주택 감정평가 |
| 5 | 보증서 발급 |
| 6 | 금융기관 대출 계약 |
| 7 | 연금 수령 시작 |

### 5.2 필요 서류

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 신원 확인 |
| 주택 등기부등본 | 소유권 확인 |
| 신분증 | 본인 확인 |
| 가족관계증명서 | 배우자 확인 |

### 5.3 처리 기간

| 단계 | 기간 |
|------|------|
| 심사 | 약 2주 |
| 전체 과정 | 약 1개월 |

## 6. 비용

### 6.1 초기 비용

| 비용 | 금액 |
|------|------|
| 감정평가 수수료 | 30~50만원 |
| 등록면허세 | 주택가격의 0.2% |
| 인지세 | 주택가격에 따라 |

### 6.2 이용 비용

| 비용 | 금액 |
|------|------|
| 보증료 | 연 0.05% (대출잔액 기준) |
| 이자 | 약 연 3%대 (변동) |

### 6.3 비용 납부 방식

| 방식 | 내용 |
|------|------|
| 월 수령액에서 차감 | 별도 납부 불필요 |

## 7. 세금 혜택

### 7.1 세금 감면

| 세금 | 혜택 |
|------|------|
| 재산세 | **25%** 감면 |
| 등록면허세 | **75%** 감면 |
| 대출이자 | 소득공제 가능 |

### 7.2 연금소득세

| 항목 | 세금 |
|------|------|
| 연금 수령액 | **비과세** |

## 8. 해지와 상환

### 8.1 중도 해지

| 조건 | 가능 |
|------|------|
| 본인 의사 | ✅ 가능 |
| 상환 | 수령액 + 이자 일시 상환 |

### 8.2 사망 시 처리

| 상황 | 처리 |
|------|------|
| 가입자 사망 | 배우자 계속 수령 |
| 모두 사망 | 주택 매각 정산 |
| 잔여 금액 | 상속인에게 반환 |

### 8.3 정산 방식

```
주택 매각가 > 총 수령액 + 이자 → 차액 상속
주택 매각가 < 총 수령액 + 이자 → 차액 면제 (비소구 방식)
```

## 9. 장단점

### 9.1 장점

| 장점 | 내용 |
|------|------|
| 평생 연금 | 사망 시까지 수령 |
| 거주 보장 | 집에서 계속 생활 |
| 상속 가능 | 잔여액 상속 |
| 비소구 | 부족분 상속인 부담 없음 |

### 9.2 단점

| 단점 | 내용 |
|------|------|
| 주택 처분 제한 | 가입 중 매도 불가 |
| 이자 누적 | 시간이 지날수록 증가 |
| 집값 하락 리스크 | 정산 시 반영 |

## 10. 주의사항

### 10.1 가입 전 확인

| 확인 | 내용 |
|------|------|
| 다른 선택지 | 주택 매각 후 월세 비교 |
| 장기 거주 계획 | 이사 계획 여부 |
| 상속 계획 | 자녀에게 상속 의사 |

### 10.2 계약 주의

| 주의 | 내용 |
|------|------|
| 수령방식 선택 | 신중히 결정 |
| 이자율 확인 | 변동 이자율 |
| 배우자 동의 | 필수 |
