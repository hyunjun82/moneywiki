# 사직서 제출해도 실업급여 받을 수 있나요?

"사직서 냈으면 실업급여 못 받는 거 아니야?" 많이들 이렇게 알고 계시죠. **결론부터 말하면, 사직서를 제출해도 실업급여를 받을 수 있는 경우가 있어요.**

## 핵심 요약

- 사직서 제출 = 무조건 실업급여 불가 ❌
- **정당한 이직 사유**가 있으면 수급 가능
- 권고사직, 계약만료, 근로조건 위반 등이 해당
- 이직확인서의 **이직 사유 코드**가 중요!

---

## 실업급여 수급 조건 (기본)

| 조건 | 내용 |
|------|------|
| 고용보험 가입 | 퇴직 전 18개월 중 **180일 이상** |
| 비자발적 퇴사 | 회사 사정 or 정당한 이직 사유 |
| 구직 의사 | 적극적인 재취업 활동 필요 |
| 근로 능력 | 즉시 취업 가능한 상태 |

## 사직서 제출해도 실업급여 받는 경우

### 1. 권고사직 (회사가 퇴사를 권유)

| 상황 | 실업급여 |
|------|----------|
| 구조조정으로 권고사직 | ✅ 수급 가능 |
| 희망퇴직 신청 | ✅ 수급 가능 |
| 명예퇴직 | ✅ 수급 가능 |
| 실적 부진으로 퇴사 권유 | ✅ 수급 가능 |

> 💡 **포인트**: 회사가 먼저 퇴사를 권유했다면, 사직서 양식을 썼더라도 **비자발적 퇴사**로 인정돼요.

### 2. 계약 만료

| 상황 | 실업급여 |
|------|----------|
| 계약직 기간 만료 | ✅ 수급 가능 |
| 파견직 계약 종료 | ✅ 수급 가능 |
| 촉탁직 계약 만료 | ✅ 수급 가능 |

> 본인이 연장을 원했으나 회사가 거부한 경우도 포함

### 3. 정당한 이직 사유 (자발적이어도 OK!)

**근로조건 관련**:

| 사유 | 실업급여 | 증빙 예시 |
|------|----------|----------|
| **임금 체불** | ✅ 수급 가능 | 체불확인서, 진정서 |
| **최저임금 미달** | ✅ 수급 가능 | 급여명세서 |
| **근로시간 과다** (주 52시간 초과) | ✅ 수급 가능 | 출퇴근 기록 |
| **연장근로 미지급** | ✅ 수급 가능 | 급여명세서 |

**사업장 문제**:

| 사유 | 실업급여 | 증빙 예시 |
|------|----------|----------|
| 사업장 이전 (통근 3시간+) | ✅ 수급 가능 | 이전 공고문 |
| 휴업·휴직 3개월 이상 | ✅ 수급 가능 | 휴업 통보서 |
| 사업주 법 위반 | ✅ 수급 가능 | 고발장, 진정서 |

**개인 사정 (일부 인정)**:

| 사유 | 실업급여 | 조건 |
|------|----------|------|
| 질병·부상 (본인) | ✅ 수급 가능 | 의사 소견서 필요 |
| 가족 간호 | ✅ 수급 가능 | 진단서 + 가족관계증명 |
| 임신·출산·육아 | ✅ 수급 가능 | 일정 조건 충족 시 |
| 배우자 전근 | ✅ 수급 가능 | 인사발령문 |

## 이직확인서의 중요성

### 이직 사유 코드

회사가 고용보험에 신고하는 **이직확인서**에 기재된 코드가 핵심이에요.

| 코드 | 이직 사유 | 실업급여 |
|------|----------|----------|
| 11 | 폐업·도산 | ✅ 가능 |
| 12 | 경영상 이유 권고사직 | ✅ 가능 |
| 22 | 계약기간 만료 | ✅ 가능 |
| 23 | 근로조건 변동 | ✅ 가능 |
| 26 | 피보험자의 귀책 없는 사유 | ✅ 가능 |
| 31 | 개인 사정 (단순 자진퇴사) | ❌ 불가 |
| 32 | 자진퇴사 (정당 사유) | ✅ 가능 |

### 이직확인서 확인 방법

1. **고용보험 홈페이지** → 개인서비스 → 이직확인서 조회
2. 이직 사유 코드 확인
3. 사실과 다르면 **정정 요청** 가능

> "일신상의 사유로 사직합니다."

### ✅ 이렇게 써야 해요

> "권고사직에 동의하여 퇴사합니다."
> "회사의 구조조정으로 인해 퇴사합니다."
> "임금 체불로 인해 부득이 퇴사합니다."

→ **퇴사 사유를 명확히** 기재해야 실업급여 수급에 유리해요.
