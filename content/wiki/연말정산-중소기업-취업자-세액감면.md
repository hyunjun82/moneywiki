# 중소기업 취업자 세액감면

## 기본 개념
청년, 고령자, 장애인 등이 중소기업에 취업하면 소득세를 **최대 90%** 감면해주는 제도입니다.

## 감면 대상자

### 1. 청년 (15~34세)
- **나이**: 만 15세 이상 ~ 만 34세 이하
- **군 복무**: 최대 6년 추가 인정 (최대 만 40세)
- **감면율**: 소득세의 **90%**
- **기간**: 최대 **5년**

### 2. 고령자 (60세 이상)
- **나이**: 만 60세 이상
- **감면율**: 소득세의 **70%**
- **기간**: 최대 **3년**

### 3. 장애인
- **대상**: 장애인복지법상 장애인
- **감면율**: 소득세의 **70%**
- **기간**: 최대 **3년**

### 4. 경력단절여성
- **대상**: 퇴직 후 3~10년 이내 재취업
- **감면율**: 소득세의 **70%**
- **기간**: 최대 **3년**

## 대상 기업

### 중소기업 기준
- 중소기업기본법상 중소기업
- 업종별 매출액·자산 기준 충족
- 벤처기업, 사회적기업 포함

### 제외 기업
- 전문직 사업 (변호사, 의사 등)
- 비영리법인
- 학원, PC방 등 특정 업종

## 감면 한도

### 연간 감면 한도
| 구분 | 감면율 | 연간 한도 |
|------|-------|---------|
| 청년 (90%) | 90% | **200만원** |
| 고령자 (70%) | 70% | **200만원** |
| 장애인 (70%) | 70% | **200만원** |
| 경력단절여성 (70%) | 70% | **200만원** |

### 계산 예시
```
연 소득세 300만원인 청년
감면액: 300만원 × 90% = 270만원
→ 한도 적용: 200만원 감면
실제 납부세액: 100만원
```

## 적용 기간

### 청년 (5년)
- 취업일부터 5년간
- 중간에 퇴사·재취업 시 통산

### 고령자·장애인·경력단절여성 (3년)
- 취업일부터 3년간
- 중간에 퇴사·재취업 시 통산

### 기간 계산
- 취업일이 속한 달부터 계산
- 예: 2024년 6월 1일 취업 → 2029년 5월까지 (청년)

## 신청 방법

### 연말정산 시
1. 회사에 감면 신청서 제출
2. 중소기업 확인서 제출 (회사 발급)
3. 감면 대상 증명서류 제출
   - 청년: 주민등록등본
   - 고령자: 주민등록등본
   - 장애인: 장애인증명서
   - 경력단절여성: 경력단절 증명서

### 중도 입사 시
- 입사 시점부터 감면 적용
- 다음 연말정산 때 정산

## 주의사항

### 중복 감면 불가
- 다른 세액감면과 중복 적용 불가
- 유리한 하나만 선택

### 감면 배제 대상
- 최대주주 및 특수관계인
- 대기업 근무 후 중소기업 이직 (일부 제외)
- 동일 기업 재취업 (3년 이내)

### 중간 퇴사 시
- 감면 기간 통산
- 잔여 기간만큼 다음 직장에서 감면 가능

### 감면 적용 배제 기간
- 감면 종료 후 재취업 시 새로 적용 불가
- 한 번 5년(또는 3년) 감면받으면 종료

## 실무 적용 사례

### 사례 1: 25세 청년 취업
```
취업일: 2024년 1월
연 소득세: 150만원
감면율: 90%
감면액: 150만원 × 90% = 135만원
실납부: 15만원
```

### 사례 2: 62세 고령자 취업
```
취업일: 2024년 3월
연 소득세: 250만원
감면율: 70%
감면액: 250만원 × 70% = 175만원
실납부: 75만원
```

### 사례 3: 청년 중간 퇴사
```
1차 취업: 2022년 ~ 2024년 (2년)
2차 취업: 2024년 ~ 2027년 (3년)
→ 총 5년 감면 가능
```

## 다른 감면과의 비교

### vs 중소기업 창업 감면
- 창업자는 창업 감면 (5년, 50%)
- 일반 취업자는 취업자 감면 (5년, 90%)
- 중복 불가

### vs 연구개발 감면
- 연구개발 종사자 별도 감면 가능
- 유리한 것 선택

## 실무 팁

1. **나이 확인**: 군 복무 기간 최대 6년 추가 인정
2. **중소기업 확인**: 회사에서 중소기업 확인서 발급 요청
3. **감면 신청**: 매년 연말정산 시 신청 필수
4. **잔여 기간 확인**: 퇴사 시 잔여 감면 기간 확인
