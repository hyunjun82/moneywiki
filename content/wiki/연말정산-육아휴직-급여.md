육아휴직급여의 세금 처리와 연말정산 방법을 정리해 볼게요. [소득세법 제12조](https://www.law.go.kr/법령/소득세법)에 따른 내용이에요.

## 육아휴직급여 세금 처리

육아휴직급여와 출산휴가 급여는 비과세 소득이에요.

| 항목 | 과세 여부 |
|------|----------|
| 육아휴직급여 | 비과세 |
| 출산전후휴가 급여 | 비과세 |
| 배우자 출산휴가 급여 (회사 지급) | 과세 |

고용보험에서 지급하는 급여는 사회보험 급여 성격으로 비과세돼요.

## 연말정산 처리

육아휴직급여는 총급여에 포함되지 않아요.

| 항목 | 연말정산 |
|------|----------|
| 육아휴직급여 | 총급여 미포함 |
| 휴직 전 회사 급여 | 포함 |
| 복직 후 회사 급여 | 포함 |
| 휴직 중 회사 추가 지원금 | 포함 (과세) |

휴직 전후 회사에서 받은 급여만 연말정산 대상이에요.

## 연말정산 공제

육아휴직 중에도 연말정산 공제를 받을 수 있어요.

| 항목 | 공제 가능 기간 |
|------|---------------|
| 인적공제 | 전체 연도 |
| 신용카드 | 전체 연도 |
| 의료비 | 전체 연도 |
| 보험료 | 전체 연도 |

신용카드 공제 25% 기준은 과세 근로소득 기준으로 계산해요.

휴직 전 급여 2,000만원인 경우 25% = 500만원이고, 500만원 초과 사용분이 공제 대상이에요.

## 육아휴직 기간별 처리

### 1월~12월 전체 육아휴직

연간 과세 급여가 없으면 연말정산이 불필요해요.

| 항목 | 처리 |
|------|------|
| 과세 급여 | 없음 |
| 연말정산 | 불필요 |
| 공제 | 배우자 연말정산에 반영 가능 |

### 일부 기간 육아휴직

1~6월 근무, 7~12월 육아휴직인 경우예요.

| 항목 | 처리 |
|------|------|
| 1~6월 급여 | 연말정산 대상 |
| 7~12월 육아휴직급여 | 비과세 (제외) |
| 공제 항목 | 1~12월 전체 사용분 |

## 부양가족 공제

육아휴직자의 과세 소득에 따라 공제 방법이 달라요.

| 상황 | 공제 |
|------|------|
| 과세 소득 있음 | 본인이 공제 가능 |
| 과세 소득 없음 | 배우자가 공제 |

연간 소득 100만원 이하면 배우자의 부양가족 공제 대상이 될 수 있어요.

## 출산전후휴가 급여

지급 주체에 따라 과세 여부가 달라요.

| 지급 주체 | 세금 |
|----------|------|
| 고용보험 | 비과세 |
| 회사 추가 지급 | 과세 |

최초 60일은 회사에서 지급하고 과세돼요. 이후 30일은 고용보험에서 지급하고 비과세예요.

## 배우자 출산휴가

회사 지급분과 고용보험 지급분의 과세가 다르게 적용돼요.

| 항목 | 세금 |
|------|------|
| 회사 급여 | 과세 |
| 고용보험 급여 | 비과세 |

## 실무 예시

1~6월 급여 2,500만원, 7~12월 육아휴직인 경우예요.

| 항목 | 금액/처리 |
|------|----------|
| 총급여 | 2,500만원 (1~6월만) |
| 육아휴직급여 | 비과세 (별도) |
| 신용카드 25% | 625만원 |
| 신용카드 사용 (연간) | 1,000만원 |
| 공제 대상 | 375만원 |

## 주의사항

육아휴직급여는 비과세이지만 부업 소득이 있으면 종합소득세 신고가 필요해요.

육아휴직 중 건강보험료는 경감되거나 면제될 수 있어요.

국민연금은 납부 유예가 가능하고, 고용보험은 면제돼요.
