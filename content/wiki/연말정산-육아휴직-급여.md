# 연말정산 육아휴직 급여

육아휴직급여의 세금 처리와 연말정산 방법을 알아봅니다. [소득세법 제12조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 육아휴직급여 세금 처리

### 1.1 비과세 소득

| 항목 | 내용 |
|------|------|
| 육아휴직급여 | **비과세** |
| 출산전후휴가 급여 | **비과세** |
| 배우자 출산휴가 급여 | 과세 (회사 지급) |

### 1.2 비과세 이유

| 항목 | 내용 |
|------|------|
| 지급 주체 | 고용보험 (정부) |
| 성격 | 사회보험 급여 |
| 근거 | 소득세법 제12조 |

## 2. 연말정산 처리

### 2.1 육아휴직급여

| 항목 | 연말정산 |
|------|----------|
| 총급여 포함 | ❌ 미포함 |
| 세금 | 없음 |
| 신고 | 불필요 |

### 2.2 회사 급여 (휴직 전후)

| 기간 | 연말정산 |
|------|----------|
| 휴직 전 근무 기간 | ✅ 포함 |
| 휴직 후 복직 기간 | ✅ 포함 |
| 휴직 중 회사 추가 지원금 | ✅ 포함 (과세) |

## 3. 연말정산 공제

### 3.1 공제 가능 항목

| 항목 | 공제 가능 기간 |
|------|---------------|
| 인적공제 | 전체 연도 |
| 신용카드 | 전체 연도 |
| 의료비 | 전체 연도 |
| 보험료 | 전체 연도 |

### 3.2 총급여 25% 기준

신용카드 공제 25% 기준은 **과세 근로소득 기준**

| 예시 | 계산 |
|------|------|
| 휴직 전 급여 2,000만원 | 25% = 500만원 |
| 500만원 초과 사용분 | 공제 대상 |

## 4. 육아휴직 기간별 처리

### 4.1 1월~12월 전체 육아휴직

| 항목 | 처리 |
|------|------|
| 과세 급여 | 없음 |
| 연말정산 | 불필요 (소득 없음) |
| 공제 | 배우자 연말정산에 반영 가능 |

### 4.2 일부 기간 육아휴직

**예: 1~6월 근무, 7~12월 육아휴직**

| 항목 | 처리 |
|------|------|
| 1~6월 급여 | 연말정산 대상 |
| 7~12월 육아휴직급여 | 비과세 (제외) |
| 공제 항목 | 1~12월 전체 사용분 |

## 5. 부양가족 공제

### 5.1 육아휴직자의 부양가족

| 상황 | 공제 |
|------|------|
| 과세 소득 있음 | 본인이 공제 가능 |
| 과세 소득 없음 | 배우자가 공제 |

### 5.2 육아휴직자 본인 공제

| 상황 | 배우자의 공제 |
|------|--------------|
| 연간 소득 100만원 이하 | ✅ 배우자 공제 대상 |
| 연간 소득 100만원 초과 | ❌ 배우자 공제 불가 |

## 6. 출산전후휴가 급여

### 6.1 세금 처리

| 지급 주체 | 세금 |
|----------|------|
| 고용보험 | 비과세 |
| 회사 추가 지급 | 과세 |

### 6.2 기간별 구분

| 기간 | 급여 | 세금 |
|------|------|------|
| 최초 60일 | 회사 지급 | 과세 |
| 이후 30일 | 고용보험 | 비과세 |

## 7. 배우자 출산휴가

### 7.1 급여 세금 처리

| 항목 | 세금 |
|------|------|
| 회사 급여 | **과세** |
| 고용보험 급여 | **비과세** |

### 7.2 연말정산

- 회사 지급분: 총급여에 포함
- 고용보험분: 비과세로 제외

## 8. 실무 예시

### 8.1 육아휴직 6개월 사례

**1~6월 급여 2,500만원, 7~12월 육아휴직**

| 항목 | 금액/처리 |
|------|----------|
| 총급여 | 2,500만원 (1~6월만) |
| 육아휴직급여 | 비과세 (별도) |
| 신용카드 25% | 625만원 |
| 신용카드 사용 (연간) | 1,000만원 |
| 공제 대상 | 375만원 |

### 8.2 맞벌이 공제 분배

**아내 육아휴직, 남편 근무**

| 항목 | 누가 공제 |
|------|----------|
| 자녀 기본공제 | 남편 (소득 높음) |
| 자녀 의료비 | 아내도 가능 (지출자) |
| 보험료 | 계약자가 공제 |

## 9. 주의사항

### 9.1 소득 합산

| 소득 유형 | 연말정산 |
|----------|----------|
| 회사 급여 | ✅ 포함 |
| 육아휴직급여 | ❌ 제외 |
| 부업 소득 | 종합소득세 별도 |

### 9.2 건강보험료

| 상황 | 보험료 |
|------|--------|
| 육아휴직 중 | 경감 또는 면제 |
| 복직 후 | 정상 부과 |

### 9.3 4대 보험

- 육아휴직 중 국민연금: 납부 유예 가능
- 고용보험: 면제
- 건강보험: 경감
