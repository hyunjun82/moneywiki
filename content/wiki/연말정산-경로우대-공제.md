# 연말정산 경로우대 공제

경로우대 공제는 만 70세 이상 부양가족에 대해 **추가 100만원 소득공제**를 받는 제도입니다. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6591&cntntsId=7870)에 따른 조건을 정리합니다.

## 1. 공제 기본 조건

[소득세법 제51조](https://www.law.go.kr/법령/소득세법)에 따른 경로우대 공제 요건입니다.

| 요건 | 조건 |
|------|------|
| 나이 | **만 70세 이상** |
| 대상 | **기본공제 대상자** |
| 공제액 | 1인당 **100만원** |

## 2. 나이 기준 (2025년 귀속)

| 기준 | 출생일 |
|------|--------|
| 만 70세 이상 | **1955년 12월 31일 이전** 출생 |

## 3. 공제 대상자

### 3.1 경로우대 대상

| 대상 | 나이 요건 | 소득 요건 |
|------|----------|----------|
| 본인 | 70세 이상 | - |
| 배우자 | 70세 이상 | 100만원 이하 |
| 부모님 | 70세 이상 | 100만원 이하 |
| 조부모님 | 70세 이상 | 100만원 이하 |
| 배우자 부모 | 70세 이상 | 100만원 이하 |

### 3.2 기본공제 대상자 요건

경로우대 공제를 받으려면 먼저 **기본공제 대상자**여야 함

| 요건 | 조건 |
|------|------|
| 직계존속 나이 | 60세 이상 |
| 소득 | 연간 소득금액 100만원 이하 |
| 동거 | 별거해도 실제 부양 시 가능 |

## 4. 공제 금액 계산

### 4.1 기본공제 + 경로우대

| 공제 유형 | 금액 |
|----------|------|
| 기본공제 | 150만원 |
| 경로우대 추가공제 | **100만원** |
| **합계** | **250만원** |

### 4.2 계산 예시

**부모님 2명 (75세, 72세) 부양**

| 대상 | 기본공제 | 경로우대 | 합계 |
|------|---------|---------|------|
| 아버지 (75세) | 150만원 | 100만원 | 250만원 |
| 어머니 (72세) | 150만원 | 100만원 | 250만원 |
| **총 공제** | 300만원 | 200만원 | **500만원** |

## 5. 다른 추가공제와 중복

### 5.1 중복 가능

| 추가공제 | 중복 |
|---------|------|
| 경로우대 + **장애인** | ✅ 중복 가능 |
| 경로우대 + 부녀자 | ✅ 중복 가능 |

### 5.2 중복 계산 예시

**어머니 (75세, 장애인)**

| 공제 | 금액 |
|------|------|
| 기본공제 | 150만원 |
| 경로우대 | 100만원 |
| 장애인 | 200만원 |
| **총 공제** | **450만원** |

## 6. 65세 이상과 구분

### 6.1 나이별 혜택

| 나이 | 기본공제 대상 | 경로우대 |
|------|--------------|---------|
| 60~64세 | ✅ | ❌ |
| 65~69세 | ✅ | ❌ |
| **70세 이상** | ✅ | **✅ 100만원** |

### 6.2 의료비 한도

| 나이 | 의료비 공제 한도 |
|------|-----------------|
| 65세 이상 | **한도 없음** |
| 70세 이상 | **한도 없음** + 경로우대 |

## 7. 부양 판정

### 7.1 실제 부양

| 기준 | 인정 |
|------|------|
| 동거 | ✅ |
| 별거 + 생활비 송금 | ✅ |
| 별거 + 의료비 지급 | ✅ |
| 별거 + 부양 없음 | ❌ |

### 7.2 형제간 중복 공제

- 부모님 1명 = **한 자녀만** 공제 가능
- 형제가 모두 신청 시 → **실제 부양자** 판정

## 8. 증빙서류

### 8.1 연말정산 간소화

인적공제 대상자는 **간소화 서비스에서 조회**

### 8.2 별도 서류

| 상황 | 서류 |
|------|------|
| 별거 부모님 | 가족관계증명서 |
| 배우자 부모 | 가족관계증명서 |

## 9. 주의사항

### 9.1 나이 판정일

- **12월 31일** 기준
- 연도 중 70세 도달 → 해당 연도 공제 가능

### 9.2 소득 확인

- 부양가족의 **실제 소득** 확인 필수
- 연금소득, 사업소득 등 포함

### 9.3 중복 공제 불가

- 형제·자매 중 **1명만** 부모님 공제 가능
- 중복 신청 시 추후 가산세 부과
