# 원상복구 범위

원상복구는 임대차 종료 시 중요한 쟁점입니다. 책임 범위를 정확히 알고 분쟁을 예방하세요.

## 원상복구란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 원상복구 | 임차물을 **인도받은 상태**로 반환 |
| 근거 | 민법 제654조, 제615조 |
| 시기 | 임대차 종료 시 |

### 1.2 기본 원칙

| 원칙 | 설명 |
|------|------|
| 원상 회복 의무 | 임차인의 기본 의무 |
| 통상 손모 제외 | 정상 사용 손모는 제외 |
| 자연 노후 제외 | 시간 경과 노후는 제외 |

## 책임 구분

### 2.1 임대인 책임

| 항목 | 설명 |
|------|------|
| 자연 노후 | 시간 경과에 따른 노화 |
| 통상 손모 | 정상적 사용으로 인한 마모 |
| 시설 노후 | 설비의 경년 열화 |

### 2.2 임차인 책임

| 항목 | 설명 |
|------|------|
| 고의·과실 | 부주의로 인한 훼손 |
| 비정상 사용 | 용도 외 사용으로 인한 손상 |
| 관리 소홀 | 관리 부족으로 인한 훼손 |

### 2.3 책임 구분 기준

| 기준 | 설명 |
|------|------|
| 사용 기간 | 장기 사용 시 감가상각 고려 |
| 훼손 원인 | 과실 여부 |
| 손상 정도 | 통상 범위 초과 여부 |

## 항목별 책임

### 3.1 도배

| 상황 | 책임 |
|------|------|
| 통상 변색, 낡음 | **임대인** |
| 담배 냄새, 황변 | **임차인** |
| 낙서, 찢어짐 | **임차인** |
| 곰팡이 (결로) | ️ 원인에 따라 |
| 곰팡이 (관리 소홀) | **임차인** |

### 3.2 장판·바닥재

| 상황 | 책임 |
|------|------|
| 통상 마모, 색 바램 | **임대인** |
| 가구 자국 (통상) | **임대인** |
| 찢어짐, 파손 | **임차인** |
| 화학물질 손상 | **임차인** |
| 반려동물 손상 | **임차인** |

### 3.3 시설물

| 상황 | 책임 |
|------|------|
| 보일러 노후 | **임대인** |
| 에어컨 노후 | **임대인** |
| 고의·과실 고장 | **임차인** |
| 청소 부족 고장 | **임차인** |

### 3.4 주방·욕실

| 상황 | 책임 |
|------|------|
| 코킹 노후 | **임대인** |
| 배수구 막힘 (관리 소홀) | **임차인** |
| 타일 균열 (노후) | **임대인** |
| 타일 파손 (과실) | **임차인** |

### 3.5 창호·유리

| 상황 | 책임 |
|------|------|
| 창틀 노후 | **임대인** |
| 유리 파손 | **임차인** |
| 방충망 노후 | **임대인** |
| 방충망 찢어짐 (과실) | **임차인** |

## 감가상각

### 4.1 내용연수 기준

| 항목 | 내용연수 |
|------|---------|
| 도배 | 3~5년 |
| 장판 | 5~8년 |
| 보일러 | 10~15년 |
| 에어컨 | 7~10년 |

### 4.2 감가상각 계산

**예시: 장판 5년 사용, 임차인 과실로 교체**

| 항목 | 계산 |
|------|------|
| 내용연수 | 8년 |
| 사용 기간 | 5년 |
| 잔존가치 | (8-5)/8 = **37.5%** |
| 교체 비용 50만원 | 50만 × 37.5% = **18.75만원** |

### 4.3 감가상각 적용

| 원칙 | 내용 |
|------|------|
| 전액 배상  | 신품 교체 비용 전액은 부당 |
| 잔존가치 | 남은 가치만 배상 |
| 협의 | 양측 협의로 결정 |

## 특약 효력

### 5.1 유효한 특약

| 특약 | 효력 |
|------|------|
| 반려동물 원상복구 |  유효 |
| 흡연 원상복구 |  유효 |
| 특정 시설 관리 |  유효 |

### 5.2 무효 가능 특약

| 특약 | 효력 |
|------|------|
| 모든 원상복구 임차인 | ️ 무효 가능 |
| 통상 손모도 배상 | ️ 무효 가능 |
| 신품 교체 의무 | ️ 무효 가능 |

### 5.3 불공정약관

| 기준 | 내용 |
|------|------|
| 약관규제법 | 불공정약관 무효 |
| 통상 손모 | 임대인 책임이 원칙 |
| 과도한 책임 | 임차인에게 불리하면 무효 |
