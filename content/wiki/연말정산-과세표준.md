# 연말정산 과세표준

과세표준 계산과 세율을 정리합니다.

## 1. 과세표준 계산

```
총급여
- 근로소득공제 = 근로소득금액
- 소득공제 = 과세표준
```

## 2. 세율 구간 (2026년)

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 1,400만원 이하 | **6%** | 0 |
| 1,400~5,000만원 | **15%** | 126만원 |
| 5,000~8,800만원 | **24%** | 576만원 |
| 8,800~1.5억 | **35%** | 1,544만원 |
| 1.5억~3억 | **38%** | 1,994만원 |
| 3억~5억 | **40%** | 2,594만원 |
| 5억~10억 | **42%** | 3,594만원 |
| 10억 초과 | **45%** | 6,594만원 |

## 3. 산출세액 계산

| 과세표준 | 계산 |
|----------|------|
| 4,000만원 | 4,000×15% - 126 = 474만원 |
| 7,000만원 | 7,000×24% - 576 = 1,104만원 |

## 4. 결정세액

```
산출세액 - 세액공제 - 세액감면 = 결정세액
```
