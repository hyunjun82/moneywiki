# 고용보험 서식 모음

실업급여 신청할 때 어떤 서류가 필요한지 헷갈리시죠? **필요한 서식들을 한 곳에 모아봤어요.**

## 핵심 요약

- 실업급여 신청에 필요한 **핵심 서식 5가지**
- 모든 서식은 **고용보험 홈페이지**에서 다운로드 가능
- 회사 제출용 vs 본인 제출용 구분

---

## 실업급여 신청 필수 서식

### 1. 이직확인서 (회사 제출)

| 항목 | 내용 |
|------|------|
| **작성 주체** | 회사 (사업주) |
| **제출처** | 고용보험 EDI |
| **제출 기한** | 퇴사일로부터 **10일 이내** |
| **핵심 기재사항** | 이직 사유, 퇴직일, 평균임금 |

#### 이직확인서에 꼭 확인해야 할 것

| 항목 | 왜 중요한가 |
|------|------------|
| **이직 사유 코드** | 실업급여 수급 가능 여부 결정 |
| **평균임금** | 실업급여 금액 계산 기준 |
| **피보험 기간** | 수급 기간 결정 |

> 💡 이직확인서는 회사가 제출하지만, **본인이 조회**할 수 있어요.
> 고용보험 홈페이지 → 개인서비스 → 이직확인서 조회

### 2. 수급자격인정 신청서

| 항목 | 내용 |
|------|------|
| **작성 주체** | 본인 |
| **제출처** | 고용센터 (방문 or 온라인) |
| **제출 시기** | 워크넷 구직등록 후 |
| **필요 서류** | 신분증, 통장사본 |

#### 온라인 신청 방법

1. [고용보험 홈페이지](https://www.ei.go.kr) 접속
2. 개인서비스 → 실업급여 → 수급자격 신청
3. 본인인증 (공동인증서/간편인증)
4. 신청서 작성 및 제출

### 3. 실업인정 신청서

| 항목 | 내용 |
|------|------|
| **작성 주체** | 본인 |
| **제출처** | 고용센터 (방문 or 온라인) |
| **제출 주기** | 1~4주마다 (실업인정일) |
| **핵심 기재사항** | 구직활동 내역 |

#### 실업인정 시 필요한 증빙

| 구직활동 유형 | 증빙 서류 |
|--------------|----------|
| 온라인 입사지원 | 지원확인서 또는 캡처 |
| 면접 참석 | 면접확인서 |
| 직업훈련 | 수강증 또는 수료증 |
| 취업특강 | 참석확인서 |

---

## 특수 상황별 서식

### 4. 취업촉진수당 신청서

| 수당 종류 | 신청 조건 | 금액 |
|----------|----------|------|
| **조기재취업수당** | 수급일수 1/2 이상 남기고 재취업 | 남은 급여의 1/2 |
| **직업능력개발수당** | 직업훈련 수강 시 | 1일 1만원 한도 |
| **광역구직활동비** | 거주지 50km+ 거리 면접 | 교통비 실비 |
| **이주비** | 타지역 취업 시 | 이사비용 지원 |

### 5. 연장급여 신청서

| 연장급여 종류 | 조건 | 연장 기간 |
|--------------|------|----------|
| **훈련연장급여** | 직업훈련 수강 중 | 최대 2년 |
| **개별연장급여** | 취업 곤란 인정 | 60일 |
| **특별연장급여** | 경기침체 시 | 60일 |

## 서식 다운로드 방법

### 고용보험 홈페이지

1. [고용보험 홈페이지](https://www.ei.go.kr) 접속
2. **자료실** → **서식자료실**
3. 필요한 서식 검색 및 다운로드

### 주요 서식 목록

| 서식명 | 용도 | 다운로드 |
|--------|------|----------|
| 이직확인서 | 퇴사 증명 (회사용) | [고용보험 EDI](https://edi.ei.go.kr) |
| 수급자격인정 신청서 | 실업급여 신청 | [고용보험](https://www.ei.go.kr) |
| 실업인정 신청서 | 구직활동 증명 | [고용보험](https://www.ei.go.kr) |
| 조기재취업수당 신청서 | 조기취업 시 | [고용보험](https://www.ei.go.kr) |
| 이의신청서 | 불승인 시 이의제기 | [고용보험](https://www.ei.go.kr) |

## 신청 절차 요약

```
1. 퇴사
   ↓
2. 회사: 이직확인서 제출 (10일 내)
   ↓
3. 본인: 워크넷 구직등록
   ↓
4. 본인: 수급자격인정 신청 (고용센터/온라인)
   ↓
5. 수급자격 교육 이수
   ↓
6. 실업인정일마다 구직활동 보고
   ↓
7. 실업급여 수령
```
