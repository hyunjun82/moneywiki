원천징수영수증은 회사에서 원천징수한 세금 내역을 확인할 수 있는 서류예요. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 발급 방법을 정리해 볼게요.

## 발급 방법

원천징수영수증을 발급받는 방법이에요.

| 방법 | 절차 |
|------|------|
| 회사 | 인사팀에 요청 |
| 홈택스 | My홈택스 → 지급명세서 조회 |
| 손택스 | 조회/발급 → 지급명세서 |

재직 중이면 회사에 요청하는 게 빨라요.

## 홈택스 조회 방법

홈택스에서 원천징수영수증을 조회하는 순서예요.

| 단계 | 내용 |
|------|------|
| 1 | 홈택스 로그인 |
| 2 | My홈택스 클릭 |
| 3 | 지급명세서 등 제출내역 |
| 4 | 근로소득 지급명세서 조회 |

공인인증서나 간편인증으로 로그인해야 해요.

## 조회 가능 시점

홈택스에서 조회할 수 있는 시점이에요.

| 구분 | 시점 |
|------|------|
| 당해연도 | 3월 10일 이후 |
| 과거연도 | 즉시 조회 |

당해연도 영수증은 회사가 국세청에 제출한 후에 조회돼요.

## 기재 내용

원천징수영수증에 기재되는 주요 항목이에요.

| 항목 | 내용 |
|------|------|
| 총급여 | 연간 급여 합계 |
| 비과세 | 비과세 소득 |
| 원천징수세액 | 납부한 세금 |
| 공제 내역 | 각종 공제 금액 |
| 결정세액 | 최종 납부 세액 |

## 용도

원천징수영수증이 필요한 상황이에요.

| 용도 | 설명 |
|------|------|
| 이직 | 새 회사 제출 (합산 정산) |
| 대출 | 소득증명 (은행 제출) |
| 종합소득세 | 5월 신고 시 참고 |
| 기타 | 각종 공적 서류 제출 |

## 이직 시 처리

연내에 이직한 경우 전 직장 원천징수영수증을 새 회사에 제출해야 해요.

| 상황 | 처리 |
|------|------|
| 연내 이직 | 전 직장 영수증 제출 |
| 제출 시 | 새 회사에서 합산 정산 |
| 미제출 시 | 5월 종합소득세 신고 |

전 직장에서 퇴사 시 받거나 홈택스에서 조회할 수 있어요.

## 주의사항

전 직장 원천징수영수증을 새 회사에 제출하지 않으면 합산 정산이 안 돼요. 5월에 종합소득세 신고를 직접 해야 해요.

홈택스 조회는 회사가 국세청에 제출한 후에 가능해요. 급하면 전 직장에 직접 요청하세요.

원천징수영수증 내용이 실제와 다르면 회사에 수정을 요청하거나 경정청구를 할 수 있어요.
