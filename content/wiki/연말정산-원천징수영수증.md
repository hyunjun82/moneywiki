# 연말정산 원천징수영수증

원천징수영수증 발급 방법을 정리합니다.

## 1. 발급 방법

| 방법 | 절차 |
|------|------|
| 회사 | 인사팀에 요청 |
| 홈택스 | My홈택스 → 지급명세서 조회 |
| 손택스 | 조회/발급 → 지급명세서 |

## 2. 홈택스 조회

| 단계 | 내용 |
|------|------|
| 1 | 홈택스 로그인 |
| 2 | My홈택스 클릭 |
| 3 | 지급명세서 등 제출내역 |
| 4 | 근로소득 지급명세서 조회 |

## 3. 조회 가능 시점

| 구분 | 시점 |
|------|------|
| 당해연도 | 3월 10일 이후 |
| 과거연도 | 즉시 조회 |

## 4. 기재 내용

| 항목 | 내용 |
|------|------|
| 총급여 | 연간 급여 합계 |
| 비과세 | 비과세 소득 |
| 원천징수 | 납부한 세금 |
| 공제 내역 | 각종 공제 금액 |

## 5. 용도

| 용도 | 설명 |
|------|------|
| 이직 | 새 회사 제출 |
| 대출 | 소득증명 |
| 종소세 | 5월 신고 시 |
