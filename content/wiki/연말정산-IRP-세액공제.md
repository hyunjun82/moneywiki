# 연말정산 IRP 세액공제

IRP(개인형퇴직연금) 세액공제는 노후 대비를 위한 연금계좌 납입에 대해 **세금을 줄여주는** 제도입니다. [국세청 연금계좌 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6596&cntntsId=7875)에 따른 조건을 정리합니다.

## 1. 공제 한도

[조세특례제한법 제86조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제 한도입니다.

### 1.1 IRP 단독

| 항목 | 내용 |
|------|------|
| 연간 한도 | **900만원** |
| 공제율 | 12~15% |
| 최대 공제액 | 900만원 × 15% = **135만원** |

### 1.2 연금저축과 합산

| 구분 | 한도 |
|------|------|
| 연금저축 | 600만원 |
| IRP (추가) | 300만원 |
| **합계** | **900만원** |

**조합 예시**:
- 연금저축 600만원 + IRP 300만원 = 900만원
- 연금저축 400만원 + IRP 500만원 = 900만원
- IRP 900만원 (연금저축 없음) = 900만원

## 2. 공제율

### 2.1 소득별 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **15%** |
| 5,500만원 초과 ~ 1.2억원 이하 | **12%** |
| 1.2억원 초과 | **12%** |

### 2.2 최대 공제액

| 총급여 | 납입 한도 | 공제율 | 최대 공제액 |
|--------|----------|--------|-------------|
| 5,500만원 이하 | 900만원 | 15% | **135만원** |
| 5,500만원 초과 | 900만원 | 12% | **108만원** |

## 3. 공제 대상 납입금

### 3.1 공제 가능

| 항목 | 공제 여부 |
|------|----------|
| **추가 납입금** | ✅ 공제 가능 |
| 자기부담 DC형 납입 | ✅ 공제 가능 |

### 3.2 공제 불가

| 항목 | 공제 여부 |
|------|----------|
| 퇴직금 이전액 | ❌ 공제 불가 |
| 회사 부담 DC형 납입 | ❌ 공제 불가 |

## 4. 연금저축과 비교

### 4.1 공통점

| 항목 | 연금저축 | IRP |
|------|---------|-----|
| 공제 유형 | 세액공제 | 세액공제 |
| 공제율 | 12~15% | 12~15% |
| 합산 한도 | 900만원 | 900만원 |

### 4.2 차이점

| 항목 | 연금저축 | IRP |
|------|---------|-----|
| 단독 한도 | 600만원 | 900만원 |
| 위험자산 투자 | 제한 없음 | 70% 제한 |
| 중도 인출 | 상대적 유연 | 법적 사유만 |
| 수수료 | 낮음 | 상대적 높음 |

## 5. 공제 계산 예시

### 5.1 연금저축 + IRP

**총급여 5,000만원, 연금저축 400만원 + IRP 300만원 납입**

| 항목 | 금액 |
|------|------|
| 총 납입액 | 700만원 |
| 공제율 | 15% |
| 세액공제 | 700만원 × 15% = **105만원** |

### 5.2 IRP만

**총급여 7,000만원, IRP 600만원 납입**

| 항목 | 금액 |
|------|------|
| 납입액 | 600만원 |
| 공제율 | 12% |
| 세액공제 | 600만원 × 12% = **72만원** |

## 6. 중도 인출

### 6.1 중도 인출 사유

IRP는 **법정 사유**에만 중도 인출 가능

| 사유 | 중도 인출 |
|------|----------|
| 무주택자 주택 구입 | ✅ 가능 |
| 전세보증금 | ✅ 가능 |
| 6개월 이상 요양 | ✅ 가능 |
| 파산·개인회생 | ✅ 가능 |
| 천재지변 | ✅ 가능 |

### 6.2 중도 해지 시 세금

| 상황 | 세율 |
|------|------|
| 중도 해지 | 기타소득세 **16.5%** |
| 연금 외 수령 | 기타소득세 **16.5%** |

## 7. 연금 수령 시

### 7.1 연금 수령 조건

| 조건 | 내용 |
|------|------|
| 나이 | 만 55세 이상 |
| 가입 기간 | 5년 이상 |
| 수령 기간 | 10년 이상 |

### 7.2 연금소득세

| 수령 나이 | 세율 |
|----------|------|
| 55~69세 | 5.5% |
| 70~79세 | 4.4% |
| 80세 이상 | 3.3% |

## 8. 절세 전략

### 8.1 최적 조합

| 전략 | 내용 |
|------|------|
| 연금저축 600만원 우선 | 수수료 낮고 운용 자유 |
| IRP 300만원 추가 | 최대 공제 한도 달성 |

### 8.2 소득별 전략

| 총급여 | 권장 |
|--------|------|
| 5,500만원 이하 | 900만원 납입 (15% 공제) |
| 5,500만원 초과 | 투자수익률과 비교 후 결정 |

## 9. 주의사항

### 9.1 납입 한도 초과

- 900만원 초과 납입 시 **초과분 공제 불가**
- 초과분은 수령 시 비과세

### 9.2 퇴직금과 분리

- 퇴직금 이전액 ≠ 세액공제 대상
- 추가 납입금만 공제

### 9.3 위험자산 제한

- IRP는 위험자산 **70% 제한**
- 안정적 투자 선호 시 유리
