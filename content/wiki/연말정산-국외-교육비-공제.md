# 연말정산 국외 교육비 공제

해외 유학 중인 자녀의 학비도 연말정산 교육비 세액공제를 받을 수 있습니다. [국세청 교육비 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 대상

### 1.1 교육기관 요건

[소득세법 시행령 제118조의6](https://www.law.go.kr/법령/소득세법시행령)에 따른 대상입니다.

| 교육기관 | 공제 여부 |
|----------|----------|
| 해외 초등학교 | ✅ 가능 |
| 해외 중학교 | ✅ 가능 |
| 해외 고등학교 | ✅ 가능 |
| 해외 대학교 (학위과정) | ✅ 가능 |
| 해외 대학원 (본인만) | ✅ 가능 |
| 어학연수 | ❌ 불가 |
| 비학위 과정 | ❌ 불가 |

### 1.2 유학 요건

| 조건 | 내용 |
|------|------|
| **본인 유학** | 정규 학위과정 |
| **자녀 유학** | 법정 조기유학 또는 재외국민 교육 |

## 2. 공제 한도

### 2.1 학교급별 한도

| 학교급 | 연간 한도 |
|--------|-----------|
| 본인 (대학·대학원) | **한도 없음** |
| 초·중·고등학교 | **300만원** |
| 대학교 | **900만원** |

### 2.2 공제율

모든 교육비: **15%** 세액공제

| 납부액 | 공제액 (초·중·고) | 공제액 (대학) |
|--------|-----------------|--------------|
| 300만원 | 45만원 | 45만원 |
| 500만원 | 45만원 (한도) | 75만원 |
| 900만원 | 45만원 (한도) | 135만원 |

## 3. 유학 유형별 공제

### 3.1 자비 유학

| 조건 | 내용 |
|------|------|
| 대상 | 중학교 졸업 이상 |
| 기간 | 6개월 이상 |
| 학력 인정 | 국내법에 따른 학력 인정 |

### 3.2 조기 유학 (초·중등)

| 조건 | 공제 여부 |
|------|----------|
| 부모 동반 유학 | ✅ 가능 |
| 부모 해외 근무 | ✅ 가능 |
| 부모 없이 단독 유학 | ❌ 불가 (예외 있음) |

### 3.3 재외국민 교육

| 상황 | 공제 |
|------|------|
| 해외 주재원 자녀 | ✅ 가능 |
| 해외 영주권자 자녀 | ✅ 가능 |
| 해외 체류 중 교육 | ✅ 가능 |

## 4. 공제 대상 비용

### 4.1 공제 가능 항목

| 항목 | 공제 여부 |
|------|----------|
| 수업료 (Tuition) | ✅ 가능 |
| 입학금 | ✅ 가능 |
| 기숙사비 | ✅ 가능 |
| 등록비 | ✅ 가능 |

### 4.2 공제 불가 항목

| 항목 | 공제 여부 |
|------|----------|
| 어학연수비 | ❌ 불가 |
| 생활비 | ❌ 불가 |
| 항공료 | ❌ 불가 |
| 보험료 | ❌ 불가 |
| 교재비 | ❌ 불가 |

## 5. 증빙서류

### 5.1 필수 서류

| 서류 | 발급처 |
|------|--------|
| 재학증명서 (영문) | 해외 학교 |
| 등록금 납부 영수증 | 해외 학교 |
| 가족관계증명서 | 대한민국 관공서 |

### 5.2 추가 서류

| 상황 | 서류 |
|------|------|
| 부모 동반 유학 | 부모 출입국 증명 |
| 해외 주재 | 재직증명서 |
| 영주권자 | 영주권 사본 |

## 6. 환율 적용

### 6.1 환율 기준

| 시점 | 환율 |
|------|------|
| 납부일 기준 | 해당일 기준환율 또는 재정환율 |
| 연간 합산 | 각 납부일 환율 적용 후 합산 |

### 6.2 환율 증빙

- 외환은행 환율표 또는
- 신용카드 결제 시 카드사 청구 금액

## 7. 공제 계산 예시

### 7.1 해외 고등학교

**미국 고등학교 학비 $25,000 (환율 1,300원)**

1. 원화 환산: $25,000 × 1,300 = 3,250만원
2. 공제 한도: 300만원
3. 세액공제: 300만원 × 15% = **45만원**

### 7.2 해외 대학교

**영국 대학 학비 £20,000 (환율 1,650원)**

1. 원화 환산: £20,000 × 1,650 = 3,300만원
2. 공제 한도: 900만원
3. 세액공제: 900만원 × 15% = **135만원**

### 7.3 본인 해외 대학원

**미국 대학원 학비 $40,000**

1. 원화 환산: 약 5,200만원
2. 공제 한도: **없음** (본인)
3. 세액공제: 5,200만원 × 15% = **780만원**

## 8. 주의사항

### 8.1 어학연수는 불가

| 교육 유형 | 공제 |
|----------|------|
| 정규 학위과정 | ✅ 가능 |
| 어학연수 | ❌ 불가 |
| 단기 연수 | ❌ 불가 |
| 교환학생 (학점 인정) | ✅ 가능 |

### 8.2 부양가족 요건

| 요건 | 내용 |
|------|------|
| 나이 | 만 20세 이하 (대학생 예외) |
| 소득 | 연 100만원 이하 |

### 8.3 서류 미비 시

- 간소화 서비스에 조회 안 됨
- **반드시 서류 직접 제출** 필요
