# 연말정산 파트타임

파트타임 근로자 연말정산을 정리합니다.

## 1. 파트타임 분류

| 근무기간 | 분류 | 정산 |
|----------|------|------|
| 3개월 미만 | 일용직 | 원천징수 종결 |
| 3개월 이상 | 상용직 | **연말정산** |

## 2. 연말정산 대상

| 조건 | 내용 |
|------|------|
| 근무기간 | 3개월 이상 |
| 고용형태 | 동일 사업장 계속 근무 |

## 3. 공제 가능 항목

| 항목 | 공제 |
|------|------|
| 기본공제 | ✅ |
| 신용카드 | ✅ |
| 보험료 | ✅ |
| 의료비 | ✅ |
| 교육비 | ✅ |

## 4. 주의사항

| 주의 | 내용 |
|------|------|
| 투잡 | 합산 신고 필요 |
| 소득 적음 | 환급 가능성 높음 |

## 5. 투잡 시 처리

| 상황 | 처리 |
|------|------|
| 동시 근무 | 주된 회사에서 합산 |
| 연내 이직 | 새 회사에서 합산 |
