# 연말정산 기본공제 대상

기본공제 대상자와 요건을 정리합니다.

## 1. 공제 금액

| 항목 | 금액 |
|------|------|
| 1인당 공제 | **150만원** |
| 공제 유형 | 소득공제 |

## 2. 공제 대상

| 대상 | 나이 요건 | 소득 요건 |
|------|----------|----------|
| 본인 | 없음 | 없음 |
| 배우자 | 없음 | 100만원 이하 |
| 직계존속 (부모) | 만 60세 이상 | 100만원 이하 |
| 직계비속 (자녀) | 만 20세 이하 | 100만원 이하 |
| 형제자매 | 만 20세 이하 또는 60세 이상 | 100만원 이하 |

## 3. 소득 요건 상세

| 소득 유형 | 기준 |
|----------|------|
| 소득금액 | 연 100만원 이하 |
| 근로소득만 | 총급여 500만원 이하 |

## 4. 동거 요건

| 대상 | 동거 |
|------|------|
| 배우자 | 별거 가능 |
| 직계존속 | 별거 가능 |
| 직계비속 | 별거 가능 |
| 형제자매 | 동거 필요 |
