# 월세 세액공제

월세 세액공제는 월세 지출액의 일부를 세금에서 돌려받는 제도입니다. 조건과 공제 방법을 정리합니다.

## 1. 월세 세액공제 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 월세 지출액에 대한 세액공제 |
| 공제율 | 15~17% |
| 한도 | 연 **1,000만원** |

### 1.2 공제 효과

| 공제율 | 월세 500만원 시 |
|--------|----------------|
| 17% | **85만원** 환급 |
| 15% | **75만원** 환급 |

## 2. 공제 조건

### 2.1 소득 조건

| 조건 | 기준 |
|------|------|
| 총급여 | **8,000만원 이하** |
| 종합소득 | **7,000만원 이하** |

### 2.2 주택 조건

| 조건 | 기준 |
|------|------|
| 주택 규모 | 국민주택규모 이하 (**85㎡**) |
| 기준시가 | **4억원 이하** |
| 주택 유형 | 주택, 오피스텔, 고시원 등 |

### 2.3 세대 조건

| 조건 | 기준 |
|------|------|
| 무주택 | 세대 전원 무주택 |
| 세대주 | 세대주 또는 세대원 |
| 전입신고 | 필수 |

### 2.4 임대차 조건

| 조건 | 기준 |
|------|------|
| 임대차계약서 | 주민등록등본 주소지 일치 |
| 계약자 | 본인 또는 기본공제대상 배우자 |

## 3. 공제율과 한도

### 3.1 소득별 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **17%** |
| 5,500만~8,000만원 | **15%** |

### 3.2 공제 한도

| 항목 | 한도 |
|------|------|
| 연간 월세액 | **1,000만원** |
| 최대 공제액 (17%) | **170만원** |
| 최대 공제액 (15%) | **150만원** |

### 3.3 계산 예시

**월세 80만원, 총급여 4,000만원**

| 항목 | 금액 |
|------|------|
| 연간 월세 | 960만원 |
| 공제율 | 17% |
| **세액공제** | **163.2만원** |

**월세 100만원, 총급여 6,000만원**

| 항목 | 금액 |
|------|------|
| 연간 월세 | 1,200만원 |
| 한도 적용 | 1,000만원 |
| 공제율 | 15% |
| **세액공제** | **150만원** |

## 4. 신청 방법

### 4.1 연말정산 신청

| 절차 | 내용 |
|------|------|
| 1 | 필요 서류 준비 |
| 2 | 회사에 제출 |
| 3 | 연말정산 반영 |

### 4.2 필요 서류

| 서류 | 용도 |
|------|------|
| 임대차계약서 사본 | 계약 확인 |
| 주민등록등본 | 주소 확인 |
| 월세 납입 증명 | 이체내역, 현금영수증 |

### 4.3 납입 증명 방법

| 방법 | 증빙 |
|------|------|
| 계좌이체 | 이체확인서 |
| 현금영수증 | 홈택스 조회 |
| 카드 납부 | 카드내역서 |

## 5. 현금영수증과 비교

### 5.1 월세 공제 방법

| 방법 | 공제 유형 |
|------|----------|
| 월세 세액공제 | **세액공제** |
| 현금영수증 | 소득공제 |

### 5.2 효과 비교

| 방법 | 월세 600만원, 총급여 4천만원 |
|------|---------------------------|
| 세액공제 (17%) | **102만원** 절감 |
| 현금영수증 (15%) | 약 25만원 절감 |

### 5.3 선택 기준

| 조건 | 권장 방법 |
|------|----------|
| 조건 충족 시 | **세액공제** |
| 조건 미충족 시 | 현금영수증 |
| 중복 불가 | 택1 |

## 6. 월세 현금영수증

### 6.1 발급 방법

| 방법 | 절차 |
|------|------|
| 홈택스 | 현금영수증 발급 신청 |
| 세무서 | 임대인 사업자등록 확인 |

### 6.2 임대인 동의 불필요

| 항목 | 내용 |
|------|------|
| 임대인 동의 | 불필요 |
| 임대인 번호 | 사업자번호 또는 주민번호 |

### 6.3 공제 효과

| 소득공제율 | 월세 600만원 |
|------------|--------------|
| 30% (최대) | 180만원 소득공제 |
| 세금 절감 | 약 25~30만원 |

## 7. 제외 대상

### 7.1 공제 불가

| 상황 | 공제 |
|------|------|
| 총급여 8,000만원 초과 | ❌ |
| 주택 소유자 | ❌ |
| 국민주택규모 초과 | ❌ |
| 기준시가 4억 초과 | ❌ |
| 사글세 (보증금만) | ❌ |

### 7.2 주의사항

| 주의 | 내용 |
|------|------|
| 전입신고 | 필수 |
| 계약자 일치 | 본인 또는 배우자 |
| 주소 일치 | 등본과 계약서 |

## 8. 특수 상황

### 8.1 오피스텔

| 조건 | 공제 |
|------|------|
| 주거용 오피스텔 | ✅ |
| 전입신고 필수 | ✅ |

### 8.2 고시원

| 조건 | 공제 |
|------|------|
| 사업자등록 고시원 | ✅ |
| 전입신고 | 필수 |

### 8.3 다가구

| 조건 | 공제 |
|------|------|
| 계약서 보유 | ✅ |
| 전입신고 | 필수 |

## 9. 경정청구

### 9.1 과거분 신청

| 기간 | 경정청구 |
|------|----------|
| 5년 이내 | 가능 |
| 방법 | 홈택스 경정청구 |

### 9.2 절차

| 단계 | 내용 |
|------|------|
| 1 | 홈택스 로그인 |
| 2 | 경정청구 메뉴 |
| 3 | 월세 세액공제 추가 |
| 4 | 서류 첨부 |
| 5 | 제출 |

## 10. 자주 묻는 질문

### 10.1 임대인이 거부하면?

| 상황 | 대응 |
|------|------|
| 세액공제 | 임대인 동의 불필요 |
| 현금영수증 | 직접 발급 신청 |

### 10.2 부모님 집에 월세 내면?

| 상황 | 공제 |
|------|------|
| 부모님께 월세 | ❌ 불가 |
| 직계존비속 제외 | ❌ 불가 |

### 10.3 보증금은 공제되나요?

| 항목 | 공제 |
|------|------|
