# 연말정산 해외교육비

해외교육비 교육비 공제를 정리합니다.

## 1. 공제 조건

| 조건 | 공제 |
|------|------|
| 근무 관련 유학 | ✅ |
| 유치원~고교 | ✅ |
| 자비유학 대학 | ❌ |

## 2. 근무 관련 유학

| 요건 | 내용 |
|------|------|
| 해외파견 | 근로자 해외파견 |
| 해외주재 | 해외주재원 |
| 기간 | 근무기간 중 |

## 3. 공제 한도

| 대상 | 한도 |
|------|------|
| 유치원~고교 | 연 300만원 |
| 본인 대학 | 전액 (근무관련) |

## 4. 공제 불가

| 불가 | 이유 |
|------|------|
| 자비유학 대학 | 근무 무관 |
| 어학연수 | 정규과정 아님 |

## 5. 필요서류

| 서류 | 내용 |
|------|------|
| 재학증명서 | 해외학교 발급 |
| 교육비 납입증명 | 해외학교 발급 |
| 파견증명 | 회사 발급 |
