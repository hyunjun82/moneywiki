외국납부세액공제는 해외에서 발생한 소득에 대해 이중으로 세금을 내는 것을 방지하는 제도예요. [소득세법 제57조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리해 볼게요.

## 제도 개요

해외 소득에 대해 외국과 한국 양쪽에서 과세되는 문제를 해결해요.

| 개념 | 내용 |
|------|------|
| 문제 | 해외 소득에 대해 외국과 한국 양쪽에서 과세 |
| 해결 | 외국 납부세액을 한국 세금에서 공제 |
| 효과 | 실질적으로 한 번만 과세 |

## 적용 대상 소득

외국납부세액공제를 받을 수 있는 소득 유형이에요.

| 소득 유형 | 공제 가능 |
|----------|----------|
| 해외 근로소득 | 가능 |
| 해외 배당소득 | 가능 |
| 해외 이자소득 | 가능 |
| 해외 사업소득 | 가능 |
| 해외 부동산 임대소득 | 가능 |

## 공제 요건

대한민국 거주자가 국외에서 발생한 소득에 대해 외국 정부에 세금을 납부한 경우 공제받을 수 있어요.

| 요건 | 내용 |
|------|------|
| 거주자 | 대한민국 거주자 |
| 국외 소득 | 국외에서 발생한 소득 |
| 외국 세금 | 해당 소득에 대해 외국에 납부 |

조세조약 체결국이든 미체결국이든 한도 내에서 공제받을 수 있어요.

## 공제 한도

공제 한도는 다음 산식으로 계산해요.

공제 한도 = 산출세액 x (국외원천소득 / 종합소득금액)

| 항목 | 설명 |
|------|------|
| 국외원천소득 | 외국에서 발생한 소득 |
| 종합소득금액 | 국내외 전체 소득 |
| 산출세액 | 종합소득세 산출세액 |

### 한도 계산 예시

종합소득 1억원, 국외소득 2,000만원, 산출세액 1,500만원인 경우예요.

| 항목 | 금액 |
|------|------|
| 공제 한도 | 1,500만원 x (2,000만원 / 1억원) = 300만원 |
| 외국 납부세액 300만원 이하 | 전액 공제 |
| 300만원 초과 | 초과분 이월 공제 |

## 공제 방법

세액공제 방법과 필요경비 산입 방법 중 선택할 수 있어요.

### 세액공제 방법 (원칙)

| 항목 | 내용 |
|------|------|
| 적용 | 산출세액에서 직접 차감 |
| 한도 | 국외원천소득 비율 적용 |
| 초과분 | 10년간 이월 공제 |

### 필요경비 산입 방법 (선택)

| 항목 | 내용 |
|------|------|
| 적용 | 외국 납부세액을 경비로 처리 |
| 효과 | 소득금액 감소 |
| 선택 | 세액공제 대신 선택 가능 |

일반적으로 세액공제 방법이 유리해요.

## 해외주식 배당금 공제

미국 주식 배당금의 경우 원천징수와 공제 방법이에요.

| 항목 | 내용 |
|------|------|
| 미국 원천징수 | 15% (조세조약) |
| 한국 과세 | 14% (분리과세) 또는 종합과세 |
| 외국납부세액공제 | 미국 15% 중 14%까지 공제 |

미국 주식 배당금 100만원(세전)인 경우예요.

| 항목 | 금액 |
|------|------|
| 미국 원천징수 | 100만원 x 15% = 15만원 |
| 실수령 | 85만원 |
| 한국 과세 | 100만원 x 14% = 14만원 |
| 외국납부세액공제 | 14만원 (한도) |
| 추가 납부세액 | 0원 |

## 이월공제

공제 한도를 초과하거나 세액이 부족한 경우 10년간 이월 공제받을 수 있어요.

| 연도 | 처리 |
|------|------|
| 당해 연도 | 한도 내 공제 |
| 차년도 이후 | 초과분 이월 공제 |
| 10년 후 | 미공제액 소멸 |

## 증빙서류

외국납부세액 증명서와 원천징수영수증 등을 준비해야 해요.

| 서류 | 용도 |
|------|------|
| 외국납부세액 증명서 | 세금 납부 확인 |
| 원천징수영수증 | 배당, 이자 등 |
| 세금계산서 | 해당 시 |

미국의 경우 IRS 또는 증권사에서 증명서를 발급받을 수 있어요.

## 주의사항

각 국가별 조세조약 내용을 확인해야 해요. 세율과 면제 조건 등이 다를 수 있어요.

외국납부세액 증빙은 5년간 보관해야 해요. 경정청구 시 필요해요.

납부일 기준 기준환율을 적용해요. 연간 여러 번 납부 시 각각 환산해야 해요.
