# 연말정산 외국납부세액공제

외국납부세액공제는 해외에서 발생한 소득에 대해 이중으로 세금을 내는 것을 방지하는 제도입니다. [소득세법 제57조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 제도 개요

### 1.1 이중과세 방지

| 개념 | 내용 |
|------|------|
| 문제 | 해외 소득에 대해 외국과 한국 양쪽에서 과세 |
| 해결 | 외국 납부세액을 한국 세금에서 공제 |
| 효과 | 실질적으로 한 번만 과세 |

### 1.2 적용 대상 소득

| 소득 유형 | 공제 대상 |
|----------|----------|
| 해외 근로소득 | ✅ 가능 |
| 해외 배당소득 | ✅ 가능 |
| 해외 이자소득 | ✅ 가능 |
| 해외 사업소득 | ✅ 가능 |
| 해외 부동산 임대소득 | ✅ 가능 |

## 2. 공제 요건

### 2.1 기본 요건

| 요건 | 내용 |
|------|------|
| 거주자 | 대한민국 거주자 |
| 국외 소득 | 국외에서 발생한 소득 |
| 외국 세금 | 해당 소득에 대해 외국에 납부 |

### 2.2 납부세액 요건

| 조건 | 공제 가능 |
|------|----------|
| 외국 정부에 납부 | ✅ 가능 |
| 조세조약 체결국 | ✅ 가능 |
| 미체결국 | ✅ 가능 (한도 내) |

## 3. 공제 한도

### 3.1 한도 계산

**공제 한도 = 산출세액 × (국외원천소득 / 종합소득금액)**

| 항목 | 산식 |
|------|------|
| 국외원천소득 | 외국에서 발생한 소득 |
| 종합소득금액 | 국내외 전체 소득 |
| 산출세액 | 종합소득세 산출세액 |

### 3.2 한도 계산 예시

**종합소득 1억원, 국외소득 2,000만원, 산출세액 1,500만원**

- 공제 한도: 1,500만원 × (2,000만원 / 1억원) = **300만원**
- 외국 납부세액이 300만원 이하면 전액 공제
- 300만원 초과하면 초과분은 이월 공제

## 4. 공제 방법

### 4.1 세액공제 방법 (원칙)

| 항목 | 내용 |
|------|------|
| 적용 | 산출세액에서 직접 차감 |
| 한도 | 국외원천소득 비율 적용 |
| 초과분 | 10년간 이월 공제 |

### 4.2 필요경비 산입 방법 (선택)

| 항목 | 내용 |
|------|------|
| 적용 | 외국 납부세액을 경비로 처리 |
| 효과 | 소득금액 감소 |
| 선택 | 세액공제 대신 선택 가능 |

### 4.3 방법 비교

| 구분 | 세액공제 | 필요경비 산입 |
|------|---------|--------------|
| 공제 효과 | 세금에서 직접 차감 | 소득에서 차감 |
| 유리한 경우 | 일반적으로 유리 | 소득이 낮은 경우 |
| 선택 | 원칙 | 선택 가능 |

## 5. 해외주식 배당금 공제

### 5.1 미국 주식 배당금

| 항목 | 내용 |
|------|------|
| 미국 원천징수 | 15% (조세조약) |
| 한국 과세 | 14% (분리과세) 또는 종합과세 |
| 외국납부세액공제 | 미국 15% 중 14%까지 공제 |

### 5.2 계산 예시

**미국 주식 배당금 100만원 (세전)**

- 미국 원천징수: 100만원 × 15% = 15만원
- 실수령: 85만원
- 한국 과세: 100만원 × 14% = 14만원
- 외국납부세액공제: 14만원 (한도)
- 추가 납부세액: 0원

## 6. 해외 근로소득 공제

### 6.1 해외 파견 근로자

| 상황 | 공제 |
|------|------|
| 외국에서 근로소득세 납부 | ✅ 공제 가능 |
| 조세조약 면제 | 외국 납부세액 없음 |

### 6.2 주재원 세금 처리

| 국가 | 조세조약 |
|------|---------|
| 미국 | 체결 |
| 중국 | 체결 |
| 일본 | 체결 |
| 베트남 | 체결 |

## 7. 증빙서류

### 7.1 필수 서류

| 서류 | 용도 |
|------|------|
| 외국납부세액 증명서 | 세금 납부 확인 |
| 원천징수영수증 | 배당, 이자 등 |
| 세금계산서 | 해당 시 |

### 7.2 증명서 발급

| 국가 | 발급처 |
|------|--------|
| 미국 | IRS 또는 증권사 |
| 기타 | 해당국 세무당국 |

## 8. 이월공제

### 8.1 이월 요건

| 상황 | 이월 |
|------|------|
| 공제 한도 초과 | 10년간 이월 공제 |
| 세액 부족 | 10년간 이월 공제 |

### 8.2 이월공제 적용

| 연도 | 처리 |
|------|------|
| 당해 연도 | 한도 내 공제 |
| 차년도 이후 | 초과분 이월 공제 |
| 10년 후 | 미공제액 소멸 |

## 9. 주의사항

### 9.1 조세조약 확인

- 각 국가별 **조세조약** 내용 확인 필요
- 세율, 면제 조건 등 다를 수 있음

### 9.2 증빙 보관

- 외국납부세액 증빙 **5년** 보관
- 경정청구 시 필요

### 9.3 환율 적용

- 납부일 기준 **기준환율** 적용
- 연간 여러 번 납부 시 각각 환산
