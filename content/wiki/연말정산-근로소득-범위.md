"월급만 근로소득이에요?"

아니요. 회사에서 받는 혜택 중에도 근로소득으로 세금이 붙는 게 있어요.

## 근로소득이 뭔가요

고용관계에 따라 근로를 제공하고 받는 모든 대가예요. 정규직, 계약직, 일용직 상관없이 회사에 고용되어 받는 돈은 기본적으로 근로소득이에요.

| 구분 | 내용 |
|------|------|
| 발생 원인 | 근로 제공 |
| 지급 주체 | 고용주 |
| 과세 방식 | 원천징수 후 [연말정산](/w/연말정산) |

## 근로소득에 포함되는 것들

**금전급여**

| 항목 | 포함 여부 |
|------|----------|
| 기본급 | 포함 |
| 상여금, 성과급 | 포함 |
| 직책수당, 직무수당 | 포함 |
| 야근수당, 휴일수당 | 포함 |
| 퇴직위로금 | 포함 |

**현물급여**

| 항목 | 포함 여부 |
|------|----------|
| [사택 제공 이익](/w/연말정산-사택-제공-이익) | 포함 (일부 비과세) |
| 회사 차량 사적 이용 | 포함 |
| [자사 제품 할인](/w/연말정산-종업원-할인-비과세) | 포함 (240만원 한도 비과세) |
| 학자금 지원 | 포함 (일부 비과세) |

회사가 직접 돈을 주지 않아도 경제적 이익이 있으면 근로소득이 될 수 있어요.

## 비과세 근로소득도 있어요

근로소득이지만 세금이 안 붙는 항목도 있어요.

| 항목 | 비과세 한도 |
|------|------------|
| 식대 | 월 20만원 |
| 출산·보육수당 | 월 20만원 |
| [자가운전보조금](/w/연말정산-자가운전보조금) | 월 20만원 |
| 야간근로수당 (생산직) | 연 240만원 |
| [국외근로소득](/w/연말정산-국외근로소득-비과세) | 월 300만원 |

[비과세 한도](/w/연말정산-비과세-소득)를 초과하면 초과분은 과세돼요.

## 근로소득이 아닌 것들

고용관계가 아니면 근로소득이 아니에요.

| 항목 | 소득 유형 |
|------|----------|
| 프리랜서 용역비 | 사업소득 또는 기타소득 |
| 일시적 강연료 | 기타소득 |
| 임대료 | 부동산임대소득 |
| 주식배당금 | 배당소득 |
| 퇴직금 | 퇴직소득 |

퇴직금은 근로소득이 아니라 퇴직소득으로 별도 과세돼요.

## 연말정산에서 어떻게 적용돼요

총급여에서 [비과세 소득](/w/연말정산-비과세-소득)을 뺀 금액이 과세 대상이에요.

| 항목 | 금액 |
|------|------|
| 연봉 | 5,000만원 |
| 비과세 (식대 등) | 240만원 |
| 과세 대상 총급여 | 4,760만원 |

급여명세서에서 비과세 항목과 과세 항목이 구분돼 있으니 확인해 보세요.
