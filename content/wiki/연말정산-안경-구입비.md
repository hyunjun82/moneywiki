# 연말정산 안경 구입비

안경 구입비 의료비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 1인당 **50만원** |

## 2. 공제 대상

| 대상 | 공제 |
|------|------|
| 시력보정용 안경 | ✅ |
| 콘택트렌즈 | ✅ |
| 선글라스 | ❌ |
| 도수 없는 렌즈 | ❌ |

## 3. 공제 예시

| 구입비 | 한도 적용 | 공제액(15%) |
|--------|----------|-------------|
| 30만원 | 30만원 | 4.5만원 |
| 60만원 | 50만원 | 7.5만원 |
| 100만원 | 50만원 | 7.5만원 |

## 4. 필요서류

| 서류 | 발급처 |
|------|--------|
| 시력보정용 안경 구입영수증 | 안경점 |
| 간소화 자료 | 일부 자동 반영 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 영수증 | 안경사 확인 필요 |
| 가족 | 기본공제 대상자별 50만원 |
