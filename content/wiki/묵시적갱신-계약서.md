"묵시적갱신 됐는데 계약서 새로 써야 해요?"

**안 써도 돼요.** 기존 계약서 그대로 유지돼요.

## 묵시적갱신 계약서 필요 여부

### 새 계약서 불필요

[묵시적갱신](/w/묵시적갱신)은 **아무 말 없이 자동 연장**되는 거예요. 새로운 서류 필요 없어요.

- 새 계약서: X
- 새 확정일자: X
- 새 전입신고: X

기존 것 그대로 유효해요.

### 왜 안 써도 되나요

묵시적갱신은 법에서 정한 **자동 연장** 제도예요. 양쪽 다 아무 말 안 하면 "기존 조건 그대로 2년 더 계약한다"는 걸 법이 인정해주는 거예요.

새로 합의한 게 아니니까 새 계약서가 필요 없어요.

## 묵시적갱신 확정일자

### 기존 확정일자 유효

"[확정일자](/w/확정일자) 다시 받아야 하나요?" 아니요. 기존 것 그대로 유효해요.

- 기존 확정일자: **유효**
- 새로 받을 필요: 없음
- 우선변제권: 유지

묵시적갱신은 기존 계약이 **동일 조건으로 연장**되는 거라서 확정일자도 그대로 유지돼요.

### 언제 새로 받아야 하나요

**조건이 바뀔 때**만 새로 받으세요.

- 묵시적갱신 (조건 동일) → 기존 것 유효
- 보증금 인상 합의 → **새로 받기 권장**
- 월세 인상 합의 → **새로 받기 권장**
- 완전 새 계약 → **반드시 새로 받기**

조건 바뀌면 새 계약서 쓰고 확정일자 새로 받는 게 안전해요.

## 묵시적갱신 전입신고

### 기존 전입신고 유지

묵시적갱신됐다고 전입신고 다시 할 필요 없어요. 주소 안 바뀌었잖아요.

- 기존 전입신고: **유효**
- 새로 할 필요: 없음
- 대항력: 유지

[대항력](/w/대항력-발생-시점)도 기존대로 유지돼요.

### 이사하면 다시 해야

같은 집에서 묵시적갱신되는 거면 전입신고 필요 없어요. 근데 **다른 집으로 이사**하면 당연히 새로 해야죠.

## 집주인이 새 계약서 쓰자고 하면

### 쓰지 않아도 돼요

"묵시적갱신 됐으니까 계약서 새로 써요." 집주인이 이러면요?

**안 써도 돼요.** 묵시적갱신은 기존 계약서로 유지되거든요.

### 쓰면 어떻게 되나요

새 계약서 쓰면 묵시적갱신이 아니라 **합의 갱신**이나 **재계약**이 돼요.

- 계약서 새로 씀 → **새 계약**이 됨
- 묵시적갱신 유지 → 기존 계약 연장

이게 뭐가 다르냐면요:
- 묵시적갱신: 세입자가 **3개월 전 통보로 언제든 나갈 수 있음**
- 새 계약: 2년 묶일 수 있음

묵시적갱신의 장점(유연한 해지)을 유지하고 싶으면 새 계약서 안 쓰는 게 좋아요.

### 조건 바꾸고 싶으면

"보증금 좀 올려주세요." 집주인이 이러면요?

조건 바꾸려면 합의가 필요해요. 합의하면 묵시적갱신이 아니라 **합의 갱신**이 돼요.

**묵시적갱신**
- 조건: 동일
- 계약서: 불필요
- 해지: 3개월 전 통보

**합의 갱신**
- 조건: 변경 가능
- 계약서: 새로 작성
- 해지: 계약 기간 지켜야

## 묵시적갱신 증빙

### 증빙 서류

"묵시적갱신 됐다는 걸 어떻게 증명해요?"

별도 서류 없이 **기존 계약서 + 계속 거주**로 증명돼요.

- 기존 계약서: 만료일 확인
- 전입신고 내역: 계속 거주 증명
- 월세 납부 기록: 계약 유지 증명

### 분쟁 대비

혹시 분쟁 생기면 이런 거 남겨두세요.

- 기존 계약서 사본: 만료일 확인
- 월세 이체 내역: 계약 유지 증명
- 관리비 납부 내역: 계속 거주 증명
- 집주인과 대화 기록: 묵시적갱신 인정 증거

## 묵시적갱신 vs 재계약 서류

묵시적갱신이 서류 측면에서 훨씬 간편해요.

**묵시적갱신**
- 계약서: 기존 것 유지
- 확정일자: 기존 것 유효
- 전입신고: 기존 것 유지
- 복비: 없음

**재계약**
- 계약서: 새로 작성
- 확정일자: 새로 받기
- 전입신고: 변동 없음
- 복비: 있을 수 있음

## 지금 당장 할 일

**1. 현재 상태 확인**

- 계약 만료일 지났는지
- 집주인이 갱신 거절 통보 안 했는지
- 내가 이사 통보 안 했는지

셋 다 해당하면 묵시적갱신 됐어요.

**2. 서류 확인**

- 기존 계약서: 잘 보관하고 있는지
- 확정일자: 도장 있는지
- 전입신고: 현재 주소로 돼 있는지

기존 것 그대로 유효해요. 새로 받을 필요 없어요.

**3. 집주인이 새 계약서 요구하면**
- 묵시적갱신이면 안 써도 된다고 말하기
- 조건 동일하면 기존 계약서로 유지
- 조건 바꾸려면 새 계약서 + 확정일자 새로
