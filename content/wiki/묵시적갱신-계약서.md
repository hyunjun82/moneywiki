계약 만료됐는데 집주인이랑 아무 말 안 했어요. 그냥 계속 살고 있어요. 계약서 새로 써야 하나요? 확정일자 다시 받아야 하나요?

**안 써도 돼요. 다시 안 받아도 돼요.** 기존 것 그대로 유효해요.

## 왜 새 계약서가 필요 없어요?

[묵시적갱신](/w/묵시적갱신)은 법에서 정한 자동 연장 제도예요.

양쪽 다 아무 말 안 하면 "기존 조건 그대로 2년 더 계약한다"는 걸 법이 인정해줘요. 새로 합의한 게 아니니까 새 계약서가 필요 없어요. [주택임대차보호법](https://www.law.go.kr/법령/주택임대차보호법) 제6조에 나와있어요.

**정리하면요:**

- 새 계약서: 필요 없어요
- 새 확정일자: 필요 없어요
- 새 전입신고: 필요 없어요

기존 것 그대로 유효해요.

## 확정일자, 다시 받아야 하나요?

**아니요.** 기존 확정일자 그대로 유효해요.

묵시적갱신은 기존 계약이 동일 조건으로 연장되는 거예요. 그래서 확정일자도 그대로 유지돼요. 우선변제권도 그대로예요.

**언제 새로 받아야 하냐면요:**

조건이 바뀔 때만 새로 받으세요.

- 묵시적갱신 (조건 동일) → 기존 것 유효
- 보증금 인상 합의 → 새로 받기 권장
- 월세 인상 합의 → 새로 받기 권장
- 완전 새 계약 → 반드시 새로 받기

조건 바뀌면 새 계약서 쓰고 확정일자 새로 받는 게 안전해요.

## 전입신고, 다시 해야 하나요?

**아니요.** 주소 안 바뀌었잖아요.

묵시적갱신됐다고 전입신고 다시 할 필요 없어요. 기존 전입신고 그대로 유효해요. 대항력도 유지돼요.

같은 집에서 묵시적갱신되는 거면 전입신고 필요 없어요. 다른 집으로 이사하면 당연히 새로 해야 하고요.

## 집주인이 새 계약서 쓰자고 하면요?

**안 써도 돼요.** 묵시적갱신이면 기존 계약서로 유지되거든요.

근데 주의할 게 있어요. 새 계약서 쓰면 묵시적갱신이 아니라 **합의 갱신**이나 **재계약**이 돼요.

이게 뭐가 다르냐면요:

**묵시적갱신**

- 세입자가 3개월 전 통보로 언제든 나갈 수 있어요
- 기존 계약서 유지
- 확정일자 그대로

**합의 갱신/재계약**

- 2년 동안 묶일 수 있어요
- 새 계약서 작성
- 확정일자 새로 받아야

묵시적갱신의 가장 큰 장점이 유연한 해지예요. 3개월 전 통보로 언제든 나갈 수 있어요. 새 계약서 쓰면 이 장점이 사라질 수 있어요.

조건 그대로면 새 계약서 안 쓰는 게 좋아요.

## 조건 바꾸고 싶으면요?

집주인이 "보증금 좀 올려주세요"라고 하면요?

조건 바꾸려면 합의가 필요해요. 합의하면 묵시적갱신이 아니라 **합의 갱신**이 돼요.

합의 갱신하면요:

- 새 계약서 작성해야 해요
- 확정일자 새로 받는 게 좋아요
- 3개월 해지권이 사라질 수 있어요

조건 바꾸면서 3개월 해지권 유지하고 싶으면 계약서에 명시하세요. "묵시적갱신 조건 준용"이라고요.

## 묵시적갱신 증빙은 어떻게 해요?

"묵시적갱신 됐다는 걸 어떻게 증명해요?"

별도 서류 없이 **기존 계약서 + 계속 거주**로 증명돼요.

- 기존 계약서: 만료일 확인
- 전입신고 내역: 계속 거주 증명
- 월세 납부 기록: 계약 유지 증명

혹시 분쟁 생기면 이런 거 남겨두세요.

- 기존 계약서 사본
- 월세 이체 내역
- 관리비 납부 내역
- 집주인과 대화 기록

## 지금 당장 확인할 것

**1. 내 상태 확인하기**

- 계약 만료일 지났나요?
- 집주인이 갱신 거절 통보 안 했나요?
- 내가 이사 통보 안 했나요?

셋 다 해당하면 묵시적갱신 된 거예요.

**2. 기존 서류 확인하기**

- 기존 계약서: 잘 보관하고 있는지
- 확정일자: 도장 있는지
- 전입신고: 현재 주소로 돼 있는지

기존 것 그대로 유효해요. 새로 받을 필요 없어요.

**3. 집주인이 새 계약서 요구하면**

- 묵시적갱신이면 안 써도 된다고 말하세요
- 조건 동일하면 기존 계약서로 유지하세요
- 조건 바꾸려면 새 계약서 + 확정일자 새로 받으세요
