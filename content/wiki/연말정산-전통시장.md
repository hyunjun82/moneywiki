# 연말정산 전통시장

전통시장 사용분 소득공제를 정리합니다.

## 1. 공제율 비교

| 사용처 | 공제율 |
|--------|--------|
| 전통시장 | **40%** |
| 대중교통 | 40% |
| 체크카드 | 30% |
| 신용카드 | 15% |

## 2. 추가한도

| 항목 | 한도 |
|------|------|
| 기본한도 | 200~300만원 |
| 전통시장 추가 | **+100만원** |
| 대중교통 추가 | +100만원 |

## 3. 결제 방법

| 방법 | 인정 |
|------|------|
| 신용카드 | ✅ |
| 체크카드 | ✅ |
| 현금영수증 | ✅ |
| 제로페이 | ✅ |

## 4. 전통시장 확인

| 확인 | 방법 |
|------|------|
| 간소화 자료 | 전통시장 별도 표시 |
| 가맹점 | 전통시장 등록 가맹점 |

## 5. 절세 팁

| 팁 | 내용 |
|-----|------|
| 최저사용금액 | 신용카드로 채우기 |
| 추가사용 | 전통시장에서 사용 |
| 공제 최대화 | 40% 공제율 활용 |
