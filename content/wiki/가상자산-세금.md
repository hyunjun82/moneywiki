# 가상자산 세금

가상자산(암호화폐) 거래 수익에 대한 과세 기준과 신고 방법을 정리합니다.

## 1. 가상자산 과세 개요

### 1.1 과세 시행

| 항목 | 내용 |
|------|------|
| 시행일 | **2025년 1월 1일** |
| 과세 대상 | 가상자산 양도·대여 소득 |
| 소득 유형 | 기타소득 |

### 1.2 과세 대상 가상자산

| 종류 | 과세 |
|------|------|
| 비트코인 (BTC) | ✅ |
| 이더리움 (ETH) | ✅ |
| 리플 (XRP) | ✅ |
| 기타 암호화폐 | ✅ |
| NFT | ✅ (일부) |

## 2. 세율 및 공제

### 2.1 세율

| 항목 | 세율 |
|------|------|
| 소득세 | **20%** |
| 지방세 | **2%** |
| 합계 | **22%** |

### 2.2 기본공제

| 항목 | 금액 |
|------|------|
| 기본공제 | 연 **250만원** |
| 적용 | 양도차익에서 공제 |

### 2.3 계산 공식

```
가상자산 세금 = (양도차익 - 250만원) × 22%
```

## 3. 양도차익 계산

### 3.1 계산 방법

| 항목 | 계산 |
|------|------|
| 양도가액 | 매도 금액 |
| (-) 취득가액 | 매수 금액 |
| (-) 부대비용 | 수수료 등 |
| = 양도차익 | 과세 대상 |

### 3.2 취득가액 산정

| 방법 | 설명 |
|------|------|
| 이동평균법 | 평균 취득단가 계산 |
| 선입선출법 | 먼저 산 것 먼저 매도 |
| 선택 | **이동평균법** (원칙) |

### 3.3 취득가액 의제

| 상황 | 취득가액 |
|------|----------|
| 2024년 이전 취득 | 2024.12.31 시가 또는 실제 취득가 중 큰 금액 |
| 증여받은 가상자산 | 증여 시 시가 |
| 상속받은 가상자산 | 상속 시 시가 |

## 4. 계산 예시

### 4.1 기본 예시

**비트코인 매도차익 1,000만원**

| 항목 | 금액 |
|------|------|
| 양도차익 | 1,000만원 |
| (-) 기본공제 | 250만원 |
| 과세표준 | 750만원 |
| 세율 | 22% |
| **세금** | **165만원** |

### 4.2 손익통산 예시

**비트코인 +800만원, 이더리움 -300만원**

| 항목 | 금액 |
|------|------|
| 비트코인 이익 | +800만원 |
| 이더리움 손실 | -300만원 |
| 순이익 | 500만원 |
| (-) 기본공제 | 250만원 |
| 과세표준 | 250만원 |
| **세금** | **55만원** |

## 5. 손익통산

### 5.1 통산 가능

| 항목 | 통산 |
|------|------|
| 같은 해 가상자산 손익 | ✅ |
| 다른 가상자산 간 | ✅ |
| 국내·해외 거래소 합산 | ✅ |

### 5.2 통산 불가

| 항목 | 통산 |
|------|------|
| 다른 소득과 통산 | ❌ |
| 주식 손익과 통산 | ❌ |
| 다음 해로 이월 | ❌ |
