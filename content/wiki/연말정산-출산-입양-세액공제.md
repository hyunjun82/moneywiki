# 연말정산 출산 입양 세액공제

출산·입양 세액공제는 과세연도 중 자녀를 출산하거나 입양한 경우 받을 수 있는 일회성 세액공제입니다. [국세청 자녀 세액공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 금액

### 1.1 출생 순서별 공제액

[소득세법 제59조의2](https://www.law.go.kr/법령/소득세법)에 따른 공제 금액입니다.

| 출생 순서 | 세액공제 |
|----------|----------|
| 첫째 | **30만원** |
| 둘째 | **50만원** |
| 셋째 이상 | **70만원** |

### 1.2 다태아 출산 시

| 상황 | 공제액 |
|------|--------|
| 쌍둥이 (첫째+둘째) | 30만원 + 50만원 = **80만원** |
| 쌍둥이 (둘째+셋째) | 50만원 + 70만원 = **120만원** |
| 세쌍둥이 (첫째+둘째+셋째) | 30만원 + 50만원 + 70만원 = **150만원** |

## 2. 공제 조건

### 2.1 대상자 요건

| 조건 | 내용 |
|------|------|
| 출산·입양 연도 | 해당 과세연도 내 |
| 기본공제 대상 | 해당 자녀가 기본공제 대상자 |
| 입양 유형 | 민법 또는 입양특례법에 따른 입양 |

### 2.2 기본공제 대상 자녀

| 요건 | 기준 |
|------|------|
| 나이 | 만 20세 이하 |
| 소득 | 연간 소득금액 100만원 이하 |
| 동거 | 주거 형편상 별거 인정 |

## 3. 자녀 세액공제와 비교

### 3.1 두 공제의 차이

| 구분 | 출산·입양 세액공제 | 자녀 세액공제 |
|------|------------------|--------------|
| 적용 시점 | **일회성** (출산·입양 연도만) | **매년** (자녀 있는 동안) |
| 첫째 | 30만원 | 15만원/년 |
| 둘째 | 50만원 | 20만원/년 |
| 셋째 이상 | 70만원 | 30만원/년 |

### 3.2 중복 적용

출산·입양 세액공제와 자녀 세액공제는 **별도 적용**

| 항목 | 출산 연도 공제액 |
|------|-----------------|
| 첫째 출산 시 | 출산공제 30만원 + 자녀공제 15만원 = **45만원** |
| 둘째 출산 시 | 출산공제 50만원 + 자녀공제 20만원 = **70만원** |
| 셋째 출산 시 | 출산공제 70만원 + 자녀공제 30만원 = **100만원** |

## 4. 공제 계산 예시

### 4.1 첫째 출산

**2025년 첫째 자녀 출산**

| 공제 항목 | 금액 |
|----------|------|
| 기본공제 (자녀) | 150만원 소득공제 |
| 자녀 세액공제 | 15만원 |
| **출산 세액공제** | **30만원** |
| 합계 세액공제 | 45만원 |

### 4.2 둘째 출산

**첫째 자녀 있는 상태에서 둘째 출산**

| 공제 항목 | 금액 |
|----------|------|
| 기본공제 (자녀 2명) | 300만원 소득공제 |
| 자녀 세액공제 (2명) | 35만원 (15만원+20만원) |
| **출산 세액공제** | **50만원** |
| 합계 세액공제 | 85만원 |

### 4.3 쌍둥이 출산 (첫째+둘째)

**쌍둥이 첫 출산**

| 공제 항목 | 금액 |
|----------|------|
| 기본공제 (자녀 2명) | 300만원 소득공제 |
| 자녀 세액공제 (2명) | 35만원 |
| **출산 세액공제** | **80만원** (30만원+50만원) |
| 합계 세액공제 | 115만원 |

## 5. 입양의 경우

### 5.1 입양 대상

| 입양 유형 | 공제 대상 |
|----------|----------|
| 민법상 입양 | ✅ 가능 |
| 입양특례법 입양 | ✅ 가능 |
| 친양자 입양 | ✅ 가능 |

### 5.2 입양 순서 판단

기존 자녀 수에 따라 **출생 순서** 결정

| 상황 | 순서 |
|------|------|
| 자녀 없이 입양 | 첫째 (30만원) |
| 첫째 있고 입양 | 둘째 (50만원) |
| 둘째까지 있고 입양 | 셋째 (70만원) |

## 6. 증빙서류

### 6.1 출산의 경우

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 출생 확인 |
| 가족관계증명서 | 자녀 관계 확인 |

### 6.2 입양의 경우

| 서류 | 용도 |
|------|------|
| 입양관계증명서 | 입양 사실 확인 |
| 가족관계증명서 | 양자 관계 확인 |

## 7. 주의사항

### 7.1 공제 불가 상황

| 상황 | 공제 여부 |
|------|----------|
| 기본공제 대상 아님 | ❌ 불가 |
| 전년도 출산 | ❌ 불가 (해당 연도만) |
| 사실혼 관계 자녀 (미인지) | ❌ 불가 |

### 7.2 맞벌이 부부

- 부부 중 **1명만** 공제 가능
- 자녀 기본공제를 받는 사람이 출산공제도 받음

### 7.3 이혼 후 출산

- **양육권** 있는 부모가 공제
- 실제 부양하는 쪽에서 공제

## 8. 다른 출산 관련 공제

### 8.1 출산 관련 추가 혜택

| 공제 항목 | 내용 |
|----------|------|
| 산후조리원 공제 | 출산 1회당 200만원 한도 |
| 난임시술비 공제 | 20% 세액공제 (한도 없음) |
| 미숙아 의료비 | 20% 세액공제 (한도 없음) |

### 8.2 총 출산 관련 공제 예시

**첫째 출산, 산후조리원 180만원 이용**

| 항목 | 공제액 |
|------|--------|
| 출산 세액공제 | 30만원 |
| 자녀 세액공제 | 15만원 |
| 산후조리원 | 27만원 (180만원×15%) |
| **합계** | **72만원** |
