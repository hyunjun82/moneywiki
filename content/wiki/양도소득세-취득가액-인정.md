# 양도소득세 취득가액 인정

## 기본 개념
양도소득세 계산 시 공제할 수 있는 **취득가액**의 범위와 증명 방법입니다.

## 취득가액의 범위

### 1. 기본 취득가액
- 실제 매수가액
- 매매계약서상 금액

### 2. 취득 부대비용
- 중개수수료
- 법무사 수수료
- 취득세, 등록세
- 국민주택채권 할인료

### 3. 자본적 지출
- 증축, 개축 비용
- 대수선 비용
- 엘리베이터 설치 등

### 4. 제외 항목
- 수선비 (자본적 지출 아님)
- 재산세, 관리비
- 대출 이자

## 취득가액 증명 방법

### 1. 매매계약서
- **가장 확실한 증명**
- 계약서 원본 또는 사본
- 매도인·매수인 서명

### 2. 등기부등본
- 거래가액 기재 시
- 증빙으로 활용 가능

### 3. 금융거래 내역
- 계좌이체 내역
- 송금증
- 은행 거래 확인서

### 4. 영수증
- 부대비용 영수증
- 자본적 지출 영수증
- 중개수수료 영수증

## 증빙 서류가 없는 경우

### 환산취득가액 적용
- 증빙 불가 시 강제 적용
- **양도가액 × 환산취득가액 비율**

### 환산취득가액 비율 (2024년 기준)
| 취득 시기 | 환산비율 |
|----------|---------|
| ~ 1975년 | 3% |
| 1976~1977년 | 4% |
| 1978~1980년 | 5% |
| 1981~1982년 | 7% |
| 1983~1984년 | 9% |
| 1985~1986년 | 12% |
| 1987~1988년 | 15% |
| 1989~1990년 | 18% |
| 1991~1994년 | 25% |
| 1995~1998년 | 30% |
| 1999~2002년 | 40% |
| 2003~2005년 | 50% |
| 2006~2012년 | 60% |
| 2013년 이후 | 70% |

### 환산취득가액 계산 예시
```
양도가액: 10억원
취득 시기: 2000년
환산비율: 40%
환산취득가액: 10억 × 40% = 4억원
양도차익: 10억 - 4억 = 6억원
```

## 실제 취득가액 vs 환산취득가액

### 유리한 것 선택 가능
- 실제 취득가액 증명 → 실제 금액 적용
- 증명 불가 → 환산취득가액 적용

### 비교 사례
```
양도가액: 10억원
실제 취득가액: 3억원 (증빙 있음)
환산취득가액: 4억원 (2000년 취득)
→ 환산취득가액(4억) 선택 가능 (더 유리)
```

## 상속·증여 주택

### 상속 주택
- **피상속인의 취득가액** 승계
- 상속세 과세가액 아님

### 증여 주택
- **증여 당시 시가** (증여세 과세표준)
- 증여자 취득가액 아님

### 예시: 상속
```
아버지 취득가액: 2억원 (1990년)
상속 시 시가: 5억원 (2020년)
자녀 양도가액: 8억원 (2024년)
→ 취득가액: 2억원 (아버지 취득가액)
→ 양도차익: 6억원
```

### 예시: 증여
```
부모 취득가액: 2억원
증여 시 시가: 5억원
자녀 양도가액: 8억원
→ 취득가액: 5억원 (증여 시 시가)
→ 양도차익: 3억원
```

## 자본적 지출

### 인정되는 지출
- **증축**: 건물 면적 증가
- **개축**: 건물 구조 변경
- **대수선**: 내구성 증가
- **엘리베이터 설치**

### 인정 안 되는 지출
- 도배, 장판 교체
- 수도꼭지 교체
- 전구 교체
- 일반 수선비

### 증빙 필수
- 공사 계약서
- 세금계산서
- 계좌이체 내역
- 사진 자료

## 실무 주의사항

### 1. 계약서 보관
- **최소 10년** 보관 권장
- 원본 또는 사본
- 전자문서도 가능

### 2. 부대비용 영수증
- 중개수수료 영수증
- 취득세 납부 영수증
- 법무사 비용 영수증

### 3. 자본적 지출 증빙
- 공사 계약서
- 세금계산서
- 이체 증빙

### 4. 금융거래 내역
- 계좌이체 내역
- 잔금 지급 증빙

## 절세 전략

### 1. 취득가액 최대화
- 모든 부대비용 영수증 보관
- 자본적 지출 증빙 확보

### 2. 환산취득가액 활용
- 오래된 주택: 환산취득가액 유리
- 증빙 없을 때 활용

### 3. 증여 시점 선택
- 시가 높을 때 증여
- 자녀 양도차익 최소화

## 실무 체크리스트

### 양도 전 준비
- [ ] 매매계약서 확인
- [ ] 취득 부대비용 영수증 정리
- [ ] 자본적 지출 증빙 정리
- [ ] 금융거래 내역 확인
- [ ] 환산취득가액 계산 (증빙 없을 경우)

### 양도 시 제출 서류
- [ ] 매매계약서 사본
- [ ] 부대비용 영수증
- [ ] 자본적 지출 증빙
- [ ] 금융거래 확인서
