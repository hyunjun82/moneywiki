# 자녀 세액공제 조건

## 기본 개념
자녀를 양육하는 근로자에게 세액을 직접 감면해주는 제도입니다.

## 공제 대상 자녀

### 나이 요건
- **만 7세 이상 ~ 만 20세 이하**
- 출생연도 기준으로 판단
- 예: 2024년 기준 → 2004년생 ~ 2017년생

### 소득 요건
- **연간 소득금액 100만원 이하**
- 근로소득만 있는 경우: 총급여 500만원 이하

### 기본공제 대상자 요건
- 반드시 기본공제 대상자여야 함
- 나이·소득 요건 모두 충족 필요

## 공제 금액

### 자녀 수별 세액공제
| 자녀 수 | 연간 세액공제 | 누적 공제액 |
|--------|------------|-----------|
| 1명 | 15만원 | 15만원 |
| 2명 | 30만원 | 45만원 |
| 3명 | 30만원 | 75만원 |
| 4명 이상 | 각 30만원 | - |

### 계산 예시
```
1자녀: 15만원
2자녀: 15 + 30 = 45만원
3자녀: 15 + 30 + 30 = 75만원
4자녀: 15 + 30 + 30 + 30 = 105만원
```

## 출산·입양 세액공제

### 첫째 자녀
- **출산·입양 연도**: 30만원
- 자녀 세액공제와 별도

### 둘째 자녀
- **출산·입양 연도**: 50만원

### 셋째 이상
- **출산·입양 연도**: 70만원

### 적용 기간
- 출생일 또는 입양일이 속한 과세연도
- 1회만 적용

## 실무 적용 사례

### 사례 1: 자녀 2명 (8세, 12세)
```
자녀 세액공제: 45만원 (15 + 30)
출산 공제: 없음 (출생연도 아님)
총 공제: 45만원
```

### 사례 2: 2024년 둘째 출산
```
기존 자녀(5세): 공제 대상 아님 (만 7세 미만)
둘째 출산: 50만원
총 공제: 50만원
```

### 사례 3: 자녀 3명 (9세, 14세, 18세)
```
자녀 세액공제: 75만원 (15 + 30 + 30)
출산 공제: 없음
총 공제: 75만원
```

## 주의사항

### 맞벌이 부부
- **중복 공제 불가**: 부부 중 1명만 공제
- **전략**: 소득 많은 쪽이 공제받는 것이 유리
- 자녀별로 나눠 공제 불가 (전체 자녀를 1명이 공제)

### 나이 계산
- 과세기간 종료일(12월 31일) 기준
- 생일이 지나지 않아도 연도 기준으로 판단

### 소득 확인
- 아르바이트 소득 주의
- 총급여 500만원 초과 시 공제 불가

### 기본공제와의 관계
- 기본공제 대상이 아니면 자녀 세액공제도 불가
- 소득·나이 요건 재확인 필수

## 공제 제외 사례

### 1. 나이 미달
- 만 6세 이하 (2018년생 이후)
- 만 21세 이상 (2003년생 이전)

### 2. 소득 초과
- 총급여 500만원 초과
- 사업소득 100만원 초과

### 3. 기본공제 미대상
- 배우자가 공제받는 경우
- 소득 요건 미충족

## 신청 방법

### 연말정산 시
1. 회사에서 제공하는 공제신고서 작성
2. 자녀 인적사항 기재
3. 주민등록등본 제출 (회사 요구 시)

### 간소화 서비스
- 국세청 홈택스에서 자동 조회
- 부양가족 정보 확인
- 공제 가능 여부 자동 계산
