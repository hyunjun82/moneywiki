2026년부터 국민연금 보험료가 오른다는데, 얼마나 오르는지 궁금하셨죠?

**2026년 국민연금 보험료율이 9%에서 9.5%로 인상됐어요. 월급 300만원 기준으로 본인 부담이 135,000원에서 142,500원으로 7,500원 늘었어요.**

## 2026년 국민연금 요약

| 구분 | 2025년 | 2026년 | 변동 |
|------|--------|--------|------|
| 총 보험료율 | 9% | **9.5%** | +0.5%p |
| 근로자 부담 | 4.5% | **4.75%** | +0.25%p |
| 사업주 부담 | 4.5% | **4.75%** | +0.25%p |
| 소득대체율 | 41.5% | **43%** | +1.5%p |

## 2026년 연금개혁이 뭔

## 2026년 # 왜 바뀌었나

국민연금은 내는 돈보다 받는 돈이 더 많아서 **연금 기금이 고갈될 위험**이 있었어요. 2055년경 기금이 바닥날 것으로 예상됐죠.

연금개혁은 이 문제를 해결하기 위해:
1. **보험료율 인상**: 더 내고
2. **소득대체율 인상**: 더 받는

방향으로 개편한 거예요.

### 보험료율 인상 일정

| 연도 | 보험료율 | 본인 부담 |
|------|----------|----------|
| 2025년 | 9% | 4.5% |
| **2026년** | **9.5%** | **4.75%** |
| 2027년 | 10% | 5% |
| 2028년 | 10.5% | 5.25% |
| 2029년 | 11% | 5.5% |
| 2030년 | 11.5% | 5.75% |
| 2031년 | 12% | 6% |
| 2032년 | 12.5% | 6.25% |
| 2033년~ | **13%** | **6.5%** |

매년 0.5%p씩 올라서 2033년에 13%가 돼요.

## 월급별 국민연금 보험료

### 2026년 본인 부담액

| 월급 | 2025년 | 2026년 | 인상액 |
|------|--------|--------|--------|
| 200만원 | 90,000원 | **95,000원** | +5,000원 |
| 250만원 | 112,500원 | **118,750원** | +6,250원 |
| 300만원 | 135,000원 | **142,500원** | +7,500원 |
| 350만원 | 157,500원 | **166,250원** | +8,750원 |
| 400만원 | 180,000원 | **190,000원** | +10,000원 |
| 500만원 | 225,000원 | **237,500원** | +12,500원 |
| 617만원(상한) | 277,650원 | **293,075원** | +15,425원 |

### 계산 방법

```
본인 부담 = 월급 × 4.75%

예: 월급 300만원
   = 3,000,000원 × 4.75%
   = 142,500원
```

## 기준소득월액 상한·하한

### 2026년 기준

| 구분 | 금액 |
|------|------|
| 하한 | 37만원 |
| 상한 | **617만원** |

- 월급이 37만원 미만이면 37만원 기준으로 계산
- 월급이 617만원 초과여도 617만원 기준으로 계산

### 상한 적용 예시

월급 700만원이어도 보험료는 617만원 기준으로 계산돼요.

```
본인 부담 = 617만원 × 4.75% = 293,075원
(700만원이 아니라 617만원 기준)
```

## 소득대체율 인상

## 2026년 # 소득대체율이란

은퇴 전 평균 소득 대비 연금으로 받는 비율이에요.

예를 들어 평균 월급이 300만원이었고 소득대체율이 43%라면:
```
예상 연금 = 300만원 × 43% = 약 129만원
```

### 소득대체율 변화

| 연도 | 소득대체율 |
|------|-----------|
| 2025년 | 41.5% |
| **2026년** | **43%** |
| 이후 | 점진적 인상 |

더 내는 만큼 더 받게 됐어요.
