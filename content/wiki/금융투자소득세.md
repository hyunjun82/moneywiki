# 금융투자소득세

금융투자소득세는 금융상품 투자로 발생한 소득에 부과되는 세금입니다. 과세 기준과 세율을 정리합니다.

## 1. 금융투자소득세 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 금융상품 투자 소득에 대한 세금 |
| 대상 | 주식, 펀드, 채권, 파생상품 등 |
| 시행 | 2025년 1월 1일 |

### 1.2 과세 대상

| 금융상품 | 과세 |
|----------|------|
| 국내 상장주식 | ✅ |
| 해외주식 | ✅ |
| 펀드 | ✅ |
| 채권 | ✅ |
| 파생상품 | ✅ |
| ETF | ✅ |

## 2. 기본공제

### 2.1 공제 한도

| 구분 | 공제 한도 |
|------|----------|
| 국내 상장주식 | **5,000만원** |
| 기타 금융상품 | **250만원** |

### 2.2 공제 적용

| 항목 | 내용 |
|------|------|
| 적용 | 연간 합산 소득에서 공제 |
| 이월 | ❌ 불가 |
| 손익통산 | ✅ 가능 |

## 3. 세율

### 3.1 세율 구조

| 과세표준 | 세율 |
|----------|------|
| 3억원 이하 | **22%** (20% + 지방세 2%) |
| 3억원 초과 | **27.5%** (25% + 지방세 2.5%) |

### 3.2 계산 공식

```
금융투자소득세 = (금융투자소득 - 기본공제) × 세율
```

## 4. 손익통산

### 4.1 통산 범위

| 항목 | 통산 |
|------|------|
| 같은 해 손익 | ✅ 통산 가능 |
| 국내+해외주식 | ✅ 통산 가능 |
| 주식+펀드 | ✅ 통산 가능 |
| 파생상품 손익 | ✅ 통산 가능 |

### 4.2 손실 이월

| 항목 | 내용 |
|------|------|
| 이월 기간 | **5년** |
| 적용 | 다음 연도 소득에서 공제 |

### 4.3 손익통산 예시

**A주식 +3,000만원, B주식 -1,000만원**

| 항목 | 금액 |
|------|------|
| A주식 이익 | +3,000만원 |
| B주식 손실 | -1,000만원 |
| 순이익 | 2,000만원 |
| (-) 기본공제 | 5,000만원 |
| 과세표준 | **0원** |
| **세금** | **0원** |

## 5. 계산 예시

### 5.1 기본 예시

**국내 상장주식 양도차익 8,000만원**

| 항목 | 금액 |
|------|------|
| 양도차익 | 8,000만원 |
| (-) 기본공제 | 5,000만원 |
| 과세표준 | 3,000만원 |
| 세율 | 22% |
| **금융투자소득세** | **660만원** |

### 5.2 고액 예시

**금융투자소득 5억원**

| 항목 | 금액 |
|------|------|
| 금융투자소득 | 5억원 |
| (-) 기본공제 | 5,000만원 |
| 과세표준 | 4.5억원 |
| 3억원 × 22% | 6,600만원 |
| 1.5억원 × 27.5% | 4,125만원 |
| **금융투자소득세** | **1억 725만원** |

## 6. 신고·납부

### 6.1 신고 기간

| 항목 | 기간 |
|------|------|
| 반기 예정신고 | 8월 1일 ~ 31일 |
| 확정신고 | **5월 1일 ~ 31일** |

### 6.2 신고 방법

| 방법 | 내용 |
|------|------|
| 홈택스 | 전자신고 |
| 세무서 | 방문 신고 |
| 증권사 | 원천징수 (일부) |

## 7. 원천징수

### 7.1 원천징수 대상

| 대상 | 원천징수 |
|------|----------|
| 펀드 환매 | ✅ |
| ETF 매도 | ✅ |
| 채권 매도 | ✅ |
| 상장주식 | ❌ (확정신고) |

### 7.2 원천징수 세율

| 항목 | 세율 |
|------|------|
| 원천징수 | 22% |
| 정산 | 확정신고 시 |

## 8. 비과세·감면

### 8.1 비과세 상품

| 상품 | 비과세 한도 |
|------|------------|
| ISA | 200~400만원 |
| 연금저축 | 납입 시 비과세 |
| IRP | 납입 시 비과세 |

### 8.2 분리과세

| 상품 | 세율 |
|------|------|
| ISA 초과분 | 9.9% |

## 9. 기존 제도와 비교

### 9.1 변경 전후

| 항목 | 변경 전 | 변경 후 |
|------|---------|---------|
| 상장주식 | 대주주만 과세 | **전원 과세** |
| 공제 | 250만원 | **5,000만원** |
| 손익통산 | 제한적 | **전면 확대** |
| 손실이월 | 불가 | **5년** |

### 9.2 대주주 기준 폐지

| 항목 | 내용 |
|------|------|
| 기존 | 10억원 이상 대주주만 과세 |
| 변경 | 대주주 기준 **폐지** |
| 결과 | 소액주주도 과세 |

## 10. 절세 전략

### 10.1 기본공제 활용

| 전략 | 내용 |
|------|------|
| 5,000만원 내 | 매년 5,000만원 내 실현 |
| 분산 매도 | 여러 해에 나누어 매도 |

### 10.2 손익통산 활용

| 전략 | 내용 |
|------|------|
| 손실 확정 | 이익 발생 시 손실 종목 매도 |
| 연말 정리 | 12월에 손익 조절 |

### 10.3 비과세 계좌 활용

| 계좌 | 효과 |
|------|------|
| ISA | 비과세 + 분리과세 |
| 연금저축 | 과세이연 |
| IRP | 과세이연 |

## 11. 주의사항

### 11.1 신고 의무

| 주의 | 내용 |
|------|------|
| 확정신고 | 5월 필수 |
| 예정신고 | 반기별 선택 |
| 미신고 | 가산세 부과 |

### 11.2 증빙 보관

| 보관 | 기간 |
|------|------|
| 거래내역 | 5년 이상 |
| 손실 증빙 | 이월 기간 동안 |
