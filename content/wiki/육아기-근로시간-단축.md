육아휴직 말고 근무시간만 줄이면서 일하고 싶은데 어떻게 하는지 궁금하셨죠?

**육아기 근로시간 단축은 아이를 돌보면서 일도 계속할 수 있는 제도예요. 2026년에 급여 상한액이 올랐어요.**

## 2026년 육아기 근로시간 단축 급여 요약

| 구분 | 2025년 | 2026년 | 변동 |
|------|--------|--------|------|
| 매주 10시간 단축분 상한 | 월 220만원 | **월 250만원** | +30만원 |
| 나머지 단축분 상한 | 월 150만원 | **월 160만원** | +10만원 |

## 육아기 근로시간 단축이란?

### 제도 개요

육아휴직처럼 완전히 쉬는 게 아니라, **근무시간만 줄이면서 일을 계속**하는 제도예요.

| 항목 | 내용 |
|------|------|
| 대상 | 만 8세 이하 또는 초등 2학년 이하 자녀 |
| 단축 범위 | 주 15~35시간 (최소 주 15시간 근무) |
| 사용 기간 | 최대 **2년** |
| 분할 사용 | 가능 |

### 육아휴직과 비교

| 구분 | 육아휴직 | 육아기 근로시간 단축 |
|------|---------|-------------------|
| 근무 | 완전 휴직 | 시간 단축 근무 |
| 급여 | 통상임금 80% | 단축시간 비례 |
| 최대 기간 | 1년 | 2년 |
| 커리어 | 단절 우려 | 유지 가능 |

## 급여 계산 방법

### 급여 구성

급여는 두 부분으로 나눠요.

```
총 급여 = ① 매주 10시간 단축분 + ② 나머지 단축분
```

### ① 매주 10시간 단축분

| 항목 | 내용 |
|------|------|
| 지급률 | 통상임금의 **100%** |
| 상한액 | 월 **250만원** (2026년) |

### ② 나머지 단축분

| 항목 | 내용 |
|------|------|
| 지급률 | 통상임금의 **80%** |
| 상한액 | 월 **160만원** (2026년) |

### 계산 예시

**월급 300만원, 주 40시간 → 주 25시간으로 단축 (15시간 단축)**

```
① 10시간 단축분
   = 300만원 × (10/40) × 100%
   = 75만원

② 나머지 5시간 단축분
   = 300만원 × (5/40) × 80%
   = 30만원

총 급여 = 75만원 + 30만원 = 105만원

단축 후 회사 급여 = 300만원 × (25/40) = 187.5만원

총 수입 = 187.5만원 + 105만원 = 292.5만원
```

## 사용 기간

### 기본 기간

자녀 1명당 최대 **2년**이에요.

### 육아휴직과 합산

육아휴직 미사용 기간을 단축근무로 **2배** 전환할 수 있어요.

```
예: 육아휴직 6개월만 사용

미사용 육아휴직 = 6개월
단축근무 전환 = 6개월 × 2 = 12개월 추가

총 단축근무 = 2년 + 12개월 = 3년
```

## 신청 조건

### 근로자 조건

| 조건 | 내용 |
|------|------|
| 자녀 나이 | 만 8세 이하 또는 초등 2학년 이하 |
| 고용보험 | 가입 필수 |
| 근속 기간 | 6개월 이상 (예외 있음) |

### 신청 방법

1. **회사에 신청**: 단축 시작 **30일 전** 서면 신청
2. **회사 승인**: 정당한 사유 없으면 거부 불가
3. **급여 신청**: 단축 시작 후 고용센터에 신청

### 회사가 거부할 수 있는 경우

- 근속 6개월 미만
- 같은 자녀에 대해 배우자가 육아휴직 중
- 대체인력 채용이 불가능한 경우 (증명 필요)
