# 연말정산 중도퇴사

중도퇴사 시 연말정산 방법을 정리합니다.

## 1. 퇴사 시 정산

| 항목 | 내용 |
|------|------|
| 정산 시점 | 퇴사월 급여 지급 시 |
| 적용 공제 | 기본공제만 |
| 추가공제 | 5월 종소세 신고 |

## 2. 상황별 처리

| 상황 | 처리 |
|------|------|
| 연내 재취업 | 새 회사에서 합산 정산 |
| 연내 미취업 | 5월 종소세 신고 |
| 프리랜서 전환 | 5월 종소세 신고 |

## 3. 5월 종소세 신고

| 항목 | 내용 |
|------|------|
| 기간 | 5월 1일 ~ 31일 |
| 방법 | 홈택스 전자신고 |
| 필요서류 | 원천징수영수증, 공제증빙 |

## 4. 환급 가능 항목

| 공제 | 환급 |
|------|------|
| 신용카드 | ✅ |
| 의료비 | ✅ |
| 교육비 | ✅ |
| 기부금 | ✅ |
