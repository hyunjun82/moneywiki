# 등록면허세

등록면허세는 재산권 등기·등록 또는 면허 취득 시 부과되는 지방세입니다. 세율과 납부 방법을 정리합니다.

## 등록면허세 개요

### 1.1 정의

| 구분 | 내용 |
|------|------|
| 등록분 | 재산권 등기·등록 시 |
| 면허분 | 면허·허가·인가 시 |

### 1.2 과세 주체

| 항목 | 내용 |
|------|------|
| 과세권자 | 지방자치단체 |
| 세목 | 지방세 |

## 등록분 (부동산)

### 2.1 소유권 이전

| 취득 원인 | 세율 |
|----------|------|
| 유상거래 (매매) | **2%** |
| 상속 | **0.8%** |
| 증여 | **1.5%** |
| 무상취득 (기타) | **1.5%** |

### 2.2 기타 등기

| 등기 종류 | 세율 |
|----------|------|
| 소유권 보존 | **0.8%** |
| 지상권 설정 | 0.2% |
| 저당권 설정 | **0.2%** |
| 전세권 설정 | **0.2%** |
| 임차권 설정 | 0.2% |
| 가등기 | 건당 6,000원 |

### 2.3 대도시 중과

| 지역 | 중과율 |
|------|--------|
| 대도시 내 법인 본점 | **3배** |
| 수도권 과밀억제권역 | 3배 |

### 2.4 계산 예시

**3억원 아파트 매매 시**

| 항목 | 금액 |
|------|------|
| 과세표준 | 3억원 |
| 세율 | 2% |
| 등록면허세 | **600만원** |

## 등록분 (기타 재산)

### 3.1 자동차

| 등록 종류 | 세율 |
|----------|------|
| 비영업용 승용 | **5%** |
| 비영업용 그 외 | 3% |
| 영업용 | 2% |

### 3.2 선박

| 등록 종류 | 세율 |
|----------|------|
| 소유권 보존 | 0.02% |
| 소유권 이전 (매매) | 0.02% |
| 저당권 설정 | 0.02% |

### 3.3 항공기

| 등록 종류 | 세율 |
|----------|------|
| 소유권 보존 | 0.01% |
| 소유권 이전 | 0.02% |

### 3.4 법인 등기

| 등기 종류 | 세율/금액 |
|----------|----------|
| 법인 설립 | 자본금의 **0.4%** |
| 본점 이전 | 112,500원 |
| 지점 설치 | 40,200원 |
| 임원 변경 | 40,200원 |

## 면허분

### 4.1 정기분 (매년)

| 종별 | 세액 |
|------|------|
| 제1종 | 67,500원 |
| 제2종 | 54,000원 |
| 제3종 | 40,500원 |
| 제4종 | 27,000원 |
| 제5종 | 18,000원 |

### 4.2 면허 종류 예시

| 종별 | 면허 예시 |
|------|----------|
| 제1종 | 도매업, 은행업 |
| 제2종 | 소매업, 음식점 |
| 제3종 | 유흥업소, 학원 |
| 제4종 | 이·미용업 |
| 제5종 | 기타 |

### 4.3 납부 시기

| 항목 | 시기 |
|------|------|
| 정기분 | 매년 **1월** |
| 수시분 | 면허 취득 시 |

## 납부 방법

### 5.1 납부 장소

| 방법 | 내용 |
|------|------|
| 온라인 | 위택스 (www.wetax.go.kr) |
| 은행 | 납부서 지참 |
| 지자체 | 시·군·구청 |

### 5.2 납부 시기

| 구분 | 시기 |
|------|------|
| 등록분 | 등기·등록 **전** |
| 정기 면허분 | 1월 중 |
| 수시 면허분 | 면허 취득 시 |

### 5.3 필요 서류

| 서류 | 용도 |
|------|------|
| 등록면허세 납부확인서 | 등기 신청 시 첨부 |
| 취득 관련 서류 | 과세표준 확인 |
