# 연말정산 임플란트

임플란트 비용 의료비 공제를 정리합니다.

## 1. 공제 여부

| 구분 | 공제 |
|------|------|
| 치료 목적 | ✅ 가능 |
| 미용 목적 | ❌ 불가 |

## 2. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 의료비 세액공제 |
| 공제율 | **15%** |
| 한도 | 대상자별 상이 |

## 3. 공제 한도

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 | **한도 없음** |
| 65세 이상 | **한도 없음** |
| 그 외 | 700만원 |

## 4. 공제 예시

| 임플란트 비용 | 대상 | 공제액(15%) |
|--------------|------|-------------|
| 300만원 | 본인 | 45만원 |
| 500만원 | 65세 부모 | 75만원 |

## 5. 필요서류

| 서류 | 발급처 |
|------|--------|
| 의료비 영수증 | 치과 |
| 간소화 자료 | 자동 반영 |
