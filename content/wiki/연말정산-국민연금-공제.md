# 연말정산 국민연금 공제

국민연금 보험료 소득공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 소득공제 |
| 공제율 | **100%** |
| 한도 | **없음** |

## 2. 공제 대상

| 대상 | 공제 |
|------|------|
| 본인 부담분 | ✅ 전액 |
| 회사 부담분 | ❌ |

## 3. 국민연금 요율

| 항목 | 요율 |
|------|------|
| 총 요율 | 9% |
| 본인 부담 | **4.5%** |
| 회사 부담 | 4.5% |

## 4. 공제 예시

| 월급여 | 본인 부담 | 연간 공제액 |
|--------|----------|-------------|
| 300만원 | 13.5만원 | 162만원 |
| 400만원 | 18만원 | 216만원 |
| 500만원 | 22.5만원 | 270만원 |

## 5. 추가 공제 항목

| 항목 | 공제 |
|------|------|
| 건강보험료 | ✅ 전액 |
| 고용보험료 | ✅ 전액 |
| 장기요양보험료 | ✅ 전액 |
