# 연말정산 주택마련저축 공제

주택마련저축 공제는 무주택자의 내 집 마련을 지원하기 위한 소득공제 제도입니다. [조세특례제한법 제87조](https://www.law.go.kr/법령/조세특례제한법)에 따른 조건을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 공제 유형 | **소득공제** |
| 공제율 | 납입액의 **40%** |
| 연간 한도 | **300만원** |
| 연간 최대 공제 | 납입 750만원 × 40% = 300만원 |

### 1.2 대상 저축

| 저축 상품 | 공제 대상 |
|----------|----------|
| 주택청약종합저축 | ✅ 가능 |
| 청약저축 (기존 가입자) | ✅ 가능 |
| 청약예금 | ❌ 불가 |
| 청약부금 | ❌ 불가 |

## 2. 공제 조건

### 2.1 소득 요건

| 조건 | 기준 |
|------|------|
| 총급여 | **7,000만원 이하** |
| 종합소득금액 | **6,000만원 이하** |

### 2.2 주택 요건

| 조건 | 기준 |
|------|------|
| 무주택 | 과세연도 종료일 기준 **무주택** |
| 세대 전체 | **세대 전원 무주택** |

### 2.3 세대주 요건

| 조건 | 기준 |
|------|------|
| 세대주 | **세대주**여야 함 |
| **배우자 (2025년~)** | **무주택 세대주의 배우자도 공제 가능** |
| 예외 | 세대주가 주택 관련 공제 미신청 시 세대원도 가능 |

> **2025년 변경사항**: 종전에는 무주택 세대주만 공제 가능했으나, 2025년부터 무주택 세대주의 배우자도 각각 청약저축 납입액에 대해 소득공제를 받을 수 있습니다.

## 3. 공제 계산

### 3.1 공제 산식

**소득공제액 = 연간 납입액 × 40%** (최대 300만원)

### 3.2 계산 예시

| 연간 납입액 | 공제율 | 소득공제액 |
|------------|--------|-----------|
| 240만원 | 40% | 96만원 |
| 500만원 | 40% | 200만원 |
| 750만원 | 40% | **300만원** (한도) |
| 1,000만원 | 40% | **300만원** (한도 적용) |

### 3.3 절세 효과

**연간 750만원 납입 시** (한도 적용)

| 과세표준 구간 | 세율 | 절감 세액 |
|--------------|------|----------|
| 1,400만원 이하 | 6% | 18만원 |
| 5,000만원 이하 | 15% | 45만원 |
| 8,800만원 이하 | 24% | 72만원 |

## 4. 주택청약종합저축 특징

### 4.1 상품 정보

| 항목 | 내용 |
|------|------|
| 가입 대상 | 주민등록등본상 1인 1계좌 |
| 월 납입 | 2만원 ~ 50만원 |
| 이자 | 비과세 (무주택 세대주) |

### 4.2 이자 비과세

| 조건 | 비과세 한도 |
|------|-------------|
| 무주택 세대주 | 이자소득 **비과세** |
| 가입 기간 | 2년 이상 |
| 총급여 | 7,000만원 이하 |

## 5. 다른 주택 공제와 비교

### 5.1 주택 관련 소득공제

| 공제 항목 | 공제율/한도 |
|----------|-------------|
| **주택마련저축** | 40% / 300만원 |
| 주택임차차입금 원리금 | 40% / 400만원 |
| 장기주택저당차입금 이자 | 이자 전액 / 300~1,800만원 |

### 5.2 중복 적용

주택마련저축 공제와 다른 주택 공제 **중복 가능**

| 조합 | 가능 여부 |
|------|----------|
| 주택마련저축 + 전세대출 원리금 | ✅ 가능 |
| 주택마련저축 + 월세 세액공제 | ✅ 가능 |

## 6. 증빙서류

### 6.1 연말정산 간소화

- 납입액 **자동 조회** (1월 15일부터)

### 6.2 추가 서류

| 서류 | 용도 |
|------|------|
| 무주택 확인서 | 무주택 요건 충족 확인 |
| 주민등록등본 | 세대주 확인 |

## 7. 주의사항

### 7.1 무주택 판정 시점

| 시점 | 기준 |
|------|------|
| 납입 시점 | 무주택 필요 |
| 연말 기준 | 12월 31일 무주택 |

### 7.2 주택 취득 시

| 상황 | 공제 |
|------|------|
| 연중 주택 취득 | 취득 전 납입액만 공제 |
| 연말 기준 유주택 | 해당 연도 공제 불가 |

### 7.3 청약 당첨 시

| 상황 | 결과 |
|------|------|
| 청약 당첨 | 기존 공제 유지 |
| 해약 | 기존 공제 유지 |
| 당첨 후 무주택 | 계속 공제 가능 (입주 전) |

## 8. 절세 전략

### 8.1 납입 최적화

- 연 750만원 납입 시 최대 공제 (300만원)
- 월 62.5만원 납입 권장

### 8.2 이자 비과세와 병행

- 소득공제 + 이자 비과세 **이중 혜택**
- 무주택 세대주 필수

### 8.3 장기 가입 유리

- 청약 가점 축적
- 세제 혜택 지속
