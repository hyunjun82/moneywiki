# 연말정산 중소기업 청년

중소기업 취업 청년 소득세 감면을 정리합니다.

## 1. 감면 개요

| 항목 | 내용 |
|------|------|
| 감면율 | **90%** |
| 기간 | **5년간** |
| 한도 | 연 **200만원** |

## 2. 대상 요건

| 요건 | 기준 |
|------|------|
| 나이 | 만 15~34세 |
| 기업 | 중소기업 |
| 취업일 | 2023.12.31까지 취업 |

## 3. 추가 감면 대상

| 대상 | 감면율 | 기간 |
|------|--------|------|
| 60세 이상 | 70% | 3년 |
| 장애인 | 70% | 3년 |
| 경력단절여성 | 70% | 3년 |

## 4. 제외 업종

| 업종 | 제외 |
|------|------|
| 전문서비스업 | ❌ |
| 금융보험업 | ❌ |
| 부동산업 | ❌ |

## 5. 신청 방법

| 절차 | 내용 |
|------|------|
| 서류 | 감면신청서 |
| 제출처 | 회사 |
| 시기 | 취업 후 |
