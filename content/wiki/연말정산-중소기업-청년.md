중소기업 취업 청년 소득세 감면을 정리해 볼게요. [국세청 중소기업 취업자 감면 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6598&cntntsId=7877)에 따른 내용이에요.

## 감면 개요

[조세특례제한법 제30조](https://www.law.go.kr/법령/조세특례제한법)에 따른 감면 내용이에요.

| 항목 | 내용 |
|------|------|
| 감면율 | 90% |
| 기간 | 5년간 |
| 한도 | 연 200만원 |

## 대상 요건

청년 감면 대상 요건이에요.

| 요건 | 기준 |
|------|------|
| 나이 | 만 15~34세 |
| 기업 | 중소기업 |
| 군복무 | 복무기간 최대 6년까지 나이에서 차감 |

## 추가 감면 대상

청년 외 추가 감면 대상이에요.

| 대상 | 감면율 | 기간 |
|------|--------|------|
| 60세 이상 | 70% | 3년 |
| 장애인 | 70% | 3년 |
| 경력단절여성 | 70% | 3년 |

## 제외 업종

감면 제외 업종이에요.

| 업종 | 감면 |
|------|------|
| 전문서비스업 | 제외 |
| 금융보험업 | 제외 |
| 부동산업 | 제외 |

## 신청 방법

신청 절차예요.

| 절차 | 내용 |
|------|------|
| 서류 | 감면신청서 |
| 제출처 | 회사 |
| 시기 | 취업 후 |

## 감면 계산 예시

연봉 4,000만원, 결정세액 200만원인 청년의 경우예요.

| 구분 | 금액 |
|------|------|
| 결정세액 | 200만원 |
| 감면액 (90%) | 180만원 |
| 납부세액 | 20만원 |
