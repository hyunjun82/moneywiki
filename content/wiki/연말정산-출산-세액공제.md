# 출산 세액공제

## 기본 개념
자녀를 출산하거나 입양하면 해당 연도에 세액을 직접 공제해주는 제도입니다.

## 공제 대상

### 출산
- 본인 또는 배우자의 출산
- 사산은 제외 (출생신고 완료 기준)
- 쌍둥이는 각각 공제

### 입양
- 18세 미만 입양 자녀
- 입양신고 완료 기준
- 파양 시 추징

## 공제 금액

| 구분 | 세액공제 | 비고 |
|------|---------|------|
| **첫째** | **30만원** | 1회 |
| **둘째** | **50만원** | 1회 |
| **셋째 이상** | **70만원** | 1회 |

### 쌍둥이 출산
```
쌍둥이(첫 출산): 30만원 + 50만원 = 80만원
쌍둥이(둘째·셋째): 50만원 + 70만원 = 120만원
```

## 적용 사례

### 사례 1: 첫째 출산
```
2024년 1월 출산
출산 세액공제: 30만원
자녀 세액공제: 없음 (만 7세 미만)
총 공제: 30만원
```

### 사례 2: 둘째 출산
```
2024년 6월 출산
출산 세액공제: 50만원
기존 자녀(만 8세): 15만원
총 공제: 65만원
```

### 사례 3: 셋째 출산
```
2024년 출산
출산 세액공제: 70만원
기존 자녀 2명(만 10세, 8세): 45만원
총 공제: 115만원
```

### 사례 4: 쌍둥이 첫 출산
```
2024년 쌍둥이 출산 (첫 자녀)
첫째: 30만원
둘째: 50만원
총 공제: 80만원
```

## 자녀 세액공제와의 관계

### 출산 연도
```
출산 세액공제만 적용 (30/50/70만원)
자녀 세액공제는 만 7세부터
```

### 만 7세 이후
```
자녀 세액공제 적용
- 1자녀: 15만원
- 2자녀: 45만원 (15+30)
- 3자녀: 75만원 (15+30+30)
```

### 중복 공제 불가
- 출산 연도: 출산 공제만
- 이후 연도: 자녀 공제만
- 동시 적용 불가

## 맞벌이 부부 처리

### 원칙
- 부부 중 **1명만** 공제 가능
- 중복 공제 불가

### 전략
- 소득 많은 배우자가 공제받는 것이 유리
- 세율 높은 쪽이 절세 효과 큰

### 예시
```
남편 총급여: 8,000만원 (세율 24%)
아내 총급여: 3,000만원 (세율 15%)
→ 남편이 공제받는 것이 유리
  남편: 30만원 공제 시 실질 절세
  아내: 30만원 공제해도 절세 효과 작음
```

## 입양 세액공제

### 대상
- 18세 미만 입양 자녀
- 입양신고 완료 기준
- 입양 특례법상 입양

### 공제액
- 출산과 동일 (30/50/70만원)
- 몇 번째 자녀인지에 따라 결정

### 주의사항
- 파양 시 공제받은 세액 추징
- 입양일이 속한 과세연도에 공제

## 신청 방법

### 연말정산 시
1. 출산 자녀 인적사항 기재
2. 가족관계증명서 제출 (필요 시)
3. 자녀 수 확인
4. 자동 공제 (회사에서 처리)

### 준비 서류
- 출생신고 완료 확인
- 주민등록등본
- 가족관계증명서 (필요 시)

## 주의사항

### 출산일 기준
- 과세기간 중 출산 (1월 1일 ~ 12월 31일)
- 연도 말 출산도 해당 연도 공제

### 자녀 수 계산
- 이전 출산 자녀 포함
- 사망 자녀도 포함
- 입양 자녀도 포함

### 사산 제외
- 출생신고 미완료 시 공제 불가
- 사산증명서만으로는 공제 안 됨

### 해외 출산
- 국내·해외 구분 없음
- 출생신고만 완료되면 공제 가능

## 다자녀 추가 혜택

### 출산 세액공제 외
- 자녀 세액공제 (만 7세 이상)
- 다자녀 추가공제 (3자녀 이상)
- 교육비 세액공제
- 의료비 세액공제

### 기타 정부 지원
- 출산장려금 (지자체별)
- 아동수당
- 육아휴직 급여
- 양육수당

## 실무 팁

1. **출산 시기 선택**: 연말 출산도 해당 연도 공제 가능
2. **맞벌이 전략**: 소득 높은 배우자 공제
3. **자녀 수 확인**: 입양·사망 자녀 포함
4. **자동 공제**: 회사에서 주민등록등본 확인 후 자동 처리
