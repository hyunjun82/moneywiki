# 우선변제권 요건

우선변제권은 세입자가 경매에서 보증금을 먼저 받을 수 있는 권리입니다. 대항력과 확정일자가 모두 필요합니다.

## 1. 우선변제권이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 우선변제권 | 경매 시 다른 채권자보다 **먼저** 보증금 받는 권리 |
| 순위 | 확정일자 순서에 따라 배당 |
| 근거 | 주택임대차보호법 제3조의2 |

### 1.2 효과

| 상황 | 보호 |
|------|------|
| 경매 | 배당 순위에 따라 보증금 회수 |
| 공매 | 동일 보호 |
| 임의 매각 | 우선변제 주장 가능 |

## 2. 우선변제권 요건

### 2.1 3가지 요건

| 요건 | 내용 |
|------|------|
| 1. 주택 인도 | 실제 **입주** (점유) |
| 2. 전입신고 | 주민등록 **이전** |
| 3. 확정일자 | 계약서에 **확정일자** |

### 2.2 요건 상세

**주택 인도 (점유)**

| 내용 | 설명 |
|------|------|
| 의미 | 실제 거주 시작 |
| 인정 | 짐 이동, 생활 |
| 불인정 | 짐만 넣어둠 |

**전입신고**

| 내용 | 설명 |
|------|------|
| 의미 | 주민등록 이전 |
| 장소 | 주민센터, 정부24 |
| 효력 | 다음 날 0시 발생 |

**확정일자**

| 내용 | 설명 |
|------|------|
| 의미 | 계약서에 날짜 확인 도장 |
| 장소 | 주민센터, 등기소 |
| 비용 | 600원 (전월세신고 시 무료) |

## 3. 우선변제권 vs 대항력

### 3.1 비교

| 구분 | 대항력 | 우선변제권 |
|------|--------|-----------|
| 요건 | 입주+전입신고 | 입주+전입신고+**확정일자** |
| 효과 | 임대차 주장 | **보증금 우선 변제** |
| 대상 | 새 집주인 | 경매 배당 |
| 발생 시점 | 다음 날 0시 | 다음 날 0시 |

### 3.2 관계

| 관계 | 설명 |
|------|------|
| 대항력 ⊂ 우선변제권 | 우선변제권은 대항력 포함 |
| 대항력만 | 임대차 주장만 가능 |
| 우선변제권 | 임대차 주장 + 보증금 우선 |

## 4. 우선변제권 발생 시점

### 4.1 발생 시점 결정

| 기준 | 내용 |
|------|------|
| 대항력 취득 | 전입신고 다음 날 0시 |
| 확정일자 | 확정일자 받은 날 |
| **우선변제 순위** | **둘 중 늦은 시점** |

### 4.2 시점 예시

**예시 1: 전입 먼저**

| 날짜 | 사건 | 우선변제 순위 |
|------|------|--------------|
| 1월 15일 | 전입신고 | |
| 1월 16일 | 대항력 발생 (0시) | |
| 1월 18일 | 확정일자 | **1월 18일** |

**예시 2: 확정일자 먼저**

| 날짜 | 사건 | 우선변제 순위 |
|------|------|--------------|
| 1월 15일 | 확정일자 | |
| 1월 17일 | 전입신고 | |
| 1월 18일 | 대항력 발생 (0시) | **1월 18일** |

**예시 3: 같은 날**

| 날짜 | 사건 | 우선변제 순위 |
|------|------|--------------|
| 1월 15일 | 전입신고 + 확정일자 | |
| 1월 16일 | 대항력 발생 (0시) | **1월 16일** |

## 5. 우선변제 순위

### 5.1 경매 배당 순위

| 순위 | 채권자 |
|------|--------|
| 1순위 | 집행비용 |
| 2순위 | 소액임차인 최우선변제 |
| 3순위 | 당해세 |
| 4순위 | **확정일자·근저당 (선순위)** |
| 5순위 | 일반 채권 |

### 5.2 확정일자 vs 근저당

| 비교 | 선순위 결정 |
|------|-----------|
| 확정일자 A | 1월 10일 |
| 근저당권 | 1월 15일 |
| **순위** | 확정일자 A **선순위** |

### 5.3 순위 계산 예시

**경매 배당 예시**

| 채권자 | 순위일 | 채권액 |
|--------|--------|--------|
| 근저당 | 2024.1.10 | 2억원 |
| 임차인 A | 2024.1.5 확정 | 3억원 |
| 임차인 B | 2024.3.15 확정 | 2억원 |

**낙찰가 6억원 배당**

| 순위 | 배당액 |
|------|--------|
| 임차인 A | 3억원 (선순위) |
| 근저당 | 2억원 |
| 임차인 B | 1억원 (잔액) |

## 6. 확정일자 받는 방법

### 6.1 발급 장소

| 장소 | 내용 |
|------|------|
| 주민센터 | 가장 일반적 |
| 등기소 | 방문 발급 |
| 공증사무소 | 공증과 함께 |
| 전월세신고 | **자동 부여** |

### 6.2 필요 서류

| 서류 | 용도 |
|------|------|
| 임대차계약서 원본 | 확정일자 날인 |
| 신분증 | 본인 확인 |

### 6.3 비용

| 방법 | 비용 |
|------|------|
| 주민센터 | 600원 |
| 전월세신고 | **무료** |

## 7. 전월세신고와 확정일자

### 7.1 자동 부여

| 조건 | 확정일자 |
|------|----------|
| 전월세신고 완료 | **자동 부여** |
| 별도 신청 | 불필요 |

### 7.2 신고 대상

| 조건 | 신고 의무 |
|------|----------|
| 보증금 6천만원 초과 | ✅ |
| 월세 30만원 초과 | ✅ |

## 8. 우선변제권 유지·상실

### 8.1 유지 조건

| 조건 | 유지 |
|------|------|
| 계속 거주 | ✅ |
| 전입 유지 | ✅ |
| 확정일자 보관 | ✅ |

### 8.2 상실 사유

| 사유 | 결과 |
|------|------|
| 전출신고 | 대항력·우선변제권 **상실** |
| 점유 상실 | 대항력·우선변제권 **상실** |
| 계약서 분실 | ⚠️ 확정일자부 열람으로 증명 |

### 8.3 상실 방지

| 방법 | 내용 |
|------|------|
| 임차권등기명령 | 이사해도 권리 유지 |
| 적용 | 보증금 미반환 시 |

## 9. 확정일자부 열람

### 9.1 열람 목적

| 목적 | 내용 |
|------|------|
| 선순위 확인 | 나보다 먼저 확정일자 받은 임차인 |
| 계약 판단 | 안전한 계약인지 확인 |

### 9.2 열람 방법

| 방법 | 장소 |
|------|------|
| 방문 | 주민센터, 등기소 |
| 온라인 | 대법원 인터넷등기소 |
| 비용 | 300원 |

### 9.3 열람 내용

| 내용 | 확인 |
|------|------|
| 확정일자 현황 | 해당 주소의 모든 확정일자 |
| 선순위 임차인 | 나보다 앞선 임차인 |
| 보증금 합계 | 선순위 보증금 총액 |

## 10. 실무 체크리스트

### 10.1 계약 시

| 순서 | 확인 |
|------|------|
| ☐ 등기부등본 확인 | 근저당 등 |
| ☐ 확정일자부 열람 | 선순위 임차인 |
| ☐ 시세 확인 | 보증금 적정성 |

### 10.2 입주 시

| 순서 | 확인 |
|------|------|
| ☐ 잔금 지급 | |
| ☐ 입주 (점유) | |
| ☐ 전입신고 | 당일 필수 |
| ☐ 확정일자 | 당일 필수 |

### 10.3 배당 시

| 순서 | 확인 |
|------|------|
| ☐ 배당요구 신청 | 경매법원 |
| ☐ 배당표 확인 | 순위 확인 |
