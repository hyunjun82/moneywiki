# 연말정산 한부모 공제

한부모 공제는 배우자 없이 자녀를 부양하는 근로자에게 추가로 소득공제를 해주는 제도입니다. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6594&cntntsId=7873)에 따른 조건을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 공제 유형 | **추가 소득공제** |
| 공제 금액 | 연 **100만원** |
| 적용 대상 | 배우자 없이 직계비속 부양 |

### 1.2 세금 절감 효과

| 과세표준 구간 | 세율 | 절감 세액 |
|--------------|------|----------|
| 1,400만원 이하 | 6% | 6만원 |
| 5,000만원 이하 | 15% | 15만원 |
| 8,800만원 이하 | 24% | 24만원 |
| 1.5억원 이하 | 35% | 35만원 |

## 2. 공제 조건

[소득세법 제51조](https://www.law.go.kr/법령/소득세법)에 따른 조건입니다.

### 2.1 대상자 조건

| 조건 | 내용 |
|------|------|
| **배우자** | 없어야 함 (미혼, 이혼, 사별) |
| **부양가족** | 기본공제 대상 직계비속 있음 |
| **동거** | 주민등록상 동일 세대 |

### 2.2 직계비속 요건

기본공제 대상인 직계비속:

| 대상 | 나이 요건 | 소득 요건 |
|------|----------|----------|
| 자녀 | 만 20세 이하 | 100만원 이하 |
| 손자녀 | 만 20세 이하 | 100만원 이하 |
| 장애인 직계비속 | 나이 제한 없음 | 100만원 이하 |

### 2.3 "배우자 없음"의 의미

| 상황 | 공제 가능 |
|------|----------|
| 미혼 (자녀 있음) | ✅ 가능 |
| 이혼 (자녀 양육) | ✅ 가능 |
| 사별 (자녀 양육) | ✅ 가능 |
| 별거 (법적 혼인 유지) | ❌ 불가 |
| 사실혼 | ❌ 불가 |

## 3. 부녀자 공제와 비교

### 3.1 공제 비교

| 구분 | 한부모 공제 | 부녀자 공제 |
|------|------------|------------|
| 공제 금액 | **100만원** | 50만원 |
| 대상 | 남녀 모두 | 여성만 |
| 조건 | 배우자 없음 + 직계비속 | 배우자 있는 여성 또는 세대주 여성 |

### 3.2 중복 적용

**한부모 공제**와 **부녀자 공제**가 모두 해당되면:
→ **한부모 공제(100만원)만 적용** (더 큰 금액)

## 4. 공제 대상 예시

### 4.1 공제 가능

| 사례 | 적용 |
|------|------|
| 미혼모가 자녀 양육 | ✅ 한부모 공제 |
| 이혼 후 자녀 양육하는 아버지 | ✅ 한부모 공제 |
| 배우자 사망 후 자녀 양육 | ✅ 한부모 공제 |
| 미혼부가 자녀 양육 | ✅ 한부모 공제 |
| 손자녀만 양육 (자녀 없음) | ✅ 한부모 공제 |

### 4.2 공제 불가

| 사례 | 적용 |
|------|------|
| 별거 중이나 법적 혼인 유지 | ❌ 불가 |
| 자녀가 만 20세 초과 (비장애인) | ❌ 불가 |
| 자녀 소득 100만원 초과 | ❌ 불가 |
| 사실혼 관계 | ❌ 불가 |

## 5. 공제 계산 예시

### 5.1 한부모 공제 적용

**이혼 후 자녀 1명 양육, 연봉 4,000만원**

| 공제 항목 | 금액 |
|----------|------|
| 본인 기본공제 | 150만원 |
| 자녀 기본공제 | 150만원 |
| **한부모 공제** | **100만원** |
| 자녀 세액공제 | 15만원 |

### 5.2 부녀자 공제와 비교

**미혼모, 자녀 1명 양육**

| 공제 | 금액 |
|------|------|
| 부녀자 공제 자격 | ✅ 해당 |
| 한부모 공제 자격 | ✅ 해당 |
| **실제 적용** | 한부모 공제 100만원만 |

## 6. 다른 공제와 병행

### 6.1 함께 적용 가능

| 공제 | 병행 가능 |
|------|----------|
| 자녀 세액공제 | ✅ 가능 |
| 자녀 기본공제 | ✅ 가능 |
| 양육비 공제 | ✅ 가능 |
| 교육비 공제 | ✅ 가능 |

### 6.2 중복 불가

| 공제 | 병행 |
|------|------|
| 부녀자 공제 | ❌ 한부모 공제만 적용 |

## 7. 증빙서류

### 7.1 기본 서류

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 세대 구성 확인 |
| 가족관계증명서 | 직계비속 관계 확인 |

### 7.2 상황별 추가 서류

| 상황 | 서류 |
|------|------|
| 이혼 | 이혼 증명 (판결문 등) |
| 사별 | 사망진단서 또는 제적등본 |
| 미혼 | 별도 서류 불필요 |

## 8. 주의사항

### 8.1 재혼 시

- 과세연도 중 **재혼**: 한부모 공제 불가
- 재혼 전까지 기간만 해당 시: 공제 불가

### 8.2 자녀 양육권

- **양육권**이 있고 실제 부양해야 공제 가능
- 양육권이 상대방에게 있으면 공제 불가

### 8.3 소득 있는 자녀

- 자녀 소득이 **100만원 초과**: 기본공제 불가 → 한부모 공제도 불가
