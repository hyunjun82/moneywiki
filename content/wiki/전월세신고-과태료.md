# 전월세신고 과태료

전월세신고를 하지 않으면 과태료가 부과됩니다. 신고 대상과 과태료 기준을 정확히 알고 대비하세요.

## 전월세신고 의무

### 1.1 신고 대상

| 조건 | 기준 |
|------|------|
| 보증금 | **6,000만원 초과** |
| 월세 | **30만원 초과** |
| 적용 | 둘 중 하나만 해당해도 |

### 1.2 신고 기한

| 계약 유형 | 신고 기한 |
|----------|----------|
| 신규 계약 | 계약일로부터 **30일** |
| 갱신 계약 | 계약일로부터 **30일** |
| 금액 변경 | 변경일로부터 **30일** |

### 1.3 신고 제외

| 대상 | 제외 사유 |
|------|----------|
| 보증금 6천만원 이하 + 월세 30만원 이하 | 기준 미달 |
| 친족 간 무상 임대 | 무상 |
| 묵시적 갱신 (조건 동일) | 변동 없음 |

## 과태료 부과 기준

### 2.1 시행 시기

| 시기 | 내용 |
|------|------|
| 계도기간 | 2021.6.1 ~ 2024.5.31 |
| 과태료 부과 | **2024.6.1부터** |

### 2.2 과태료 기준

| 위반 유형 | 과태료 |
|----------|--------|
| 미신고 | 최대 **100만원** |
| 거짓 신고 | 최대 **100만원** |

### 2.3 지연 기간별 과태료

| 지연 기간 | 과태료 (임대인) | 과태료 (임차인) |
|----------|----------------|----------------|
| 3개월 이하 | 4~10만원 | 4~10만원 |
| 3~6개월 | 10~30만원 | 10~30만원 |
| 6개월~1년 | 20~50만원 | 20~50만원 |
| 1년 초과 | 30~100만원 | 30~100만원 |

## 과태료 세부 기준

### 3.1 1차 위반

| 지연 기간 | 과태료 |
|----------|--------|
| 3개월 이하 | **4만원** |
| 3~6개월 | **10만원** |
| 6개월~1년 | **20만원** |
| 1년 초과 | **30만원** |

### 3.2 2차 위반

| 지연 기간 | 과태료 |
|----------|--------|
| 3개월 이하 | **7만원** |
| 3~6개월 | **20만원** |
| 6개월~1년 | **35만원** |
| 1년 초과 | **60만원** |

### 3.3 3차 이상 위반

| 지연 기간 | 과태료 |
|----------|--------|
| 3개월 이하 | **10만원** |
| 3~6개월 | **30만원** |
| 6개월~1년 | **50만원** |
| 1년 초과 | **100만원** |

## 거짓 신고 과태료

### 4.1 거짓 신고 기준

| 유형 | 설명 |
|------|------|
| 보증금 허위 | 실제와 다른 금액 신고 |
| 월세 허위 | 실제와 다른 금액 신고 |
| 계약 조건 허위 | 기타 조건 허위 기재 |

### 4.2 거짓 신고 과태료

| 위반 횟수 | 과태료 |
|----------|--------|
| 1차 | **30만원** |
| 2차 | **60만원** |
| 3차 이상 | **100만원** |

## 과태료 감경

### 5.1 감경 사유

| 사유 | 감경률 |
|------|--------|
| 자진신고 | 최대 50% |
| 경제적 어려움 | 최대 50% |
| 경미한 위반 | 최대 50% |
| 기타 정당한 사유 | 협의 |

### 5.2 감경 신청

| 절차 | 내용 |
|------|------|
| 의견 제출 | 과태료 통지 후 의견서 |
| 심의 | 지자체 심의 |
| 결정 | 감경 여부 결정 |

### 5.3 자진신고 감경

| 조건 | 감경 |
|------|------|
| 적발 전 자진 신고 | **50%** 감경 |
| 적발 후 신고 | 감경 없음 |

## 과태료 부과·납부

### 6.1 부과 주체

| 주체 | 내용 |
|------|------|
| 부과 기관 | 시·군·구청장 |
| 통지 | 과태료 부과 통지서 |

### 6.2 납부 기한

| 항목 | 기한 |
|------|------|
| 납부 기한 | 통지 후 **60일** |
| 이의 신청 | 통지 후 **60일** |

### 6.3 납부 방법

| 방법 | 내용 |
|------|------|
| 가상계좌 | 통지서 기재 계좌 |

| 인터넷 | 위택스 |

| 절차 | 내용 |
|------|------|
| 관할 | 지방법원 |
