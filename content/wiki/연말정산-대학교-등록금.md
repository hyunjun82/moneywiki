# 연말정산 대학교 등록금

대학교 등록금 교육비 공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 교육비 세액공제 |
| 공제율 | **15%** |
| 한도 | 연 **900만원** |

## 2. 공제 대상

| 대상 | 한도 |
|------|------|
| 본인 대학 | **전액** |
| 자녀 대학 | **900만원** |
| 본인 대학원 | **전액** |
| 자녀 대학원 | ❌ 불가 |

## 3. 공제 항목

| 항목 | 공제 |
|------|------|
| 입학금 | ✅ |
| 수업료 | ✅ |
| 기성회비 | ✅ |
| 기숙사비 | ❌ |

## 4. 공제 예시

| 등록금 | 한도 | 공제액(15%) |
|--------|------|-------------|
| 600만원 | 600만원 | 90만원 |
| 1,000만원 | 900만원 | 135만원 |

## 5. 장학금 처리

| 구분 | 처리 |
|------|------|
| 등록금 - 장학금 | 차액만 공제 |
