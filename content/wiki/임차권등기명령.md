# 임차권등기명령

임차권등기명령은 보증금을 돌려받지 못한 세입자가 이사할 수 있게 해주는 제도입니다. 이사 후에도 권리를 유지할 수 있습니다.

## 임차권등기명령이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 임차권등기명령 | 법원이 임차권을 등기하도록 명령 |
| 목적 | 이사 후에도 **대항력·우선변제권 유지** |
| 근거 | 주택임대차보호법 제3조의3 |

### 1.2 필요한 상황

| 상황 | 필요성 |
|------|--------|
| 보증금 미반환 | 이사해야 하는데 돈 못 받음 |
| 새 전세 계약 | 다른 곳 계약 필요 |
| 취업·이직 | 지역 이동 필요 |

## 신청 요건

### 2.1 필수 요건

| 요건 | 내용 |
|------|------|
| 임대차 종료 | 계약 만료 또는 해지 |
| 보증금 미반환 | 전부 또는 일부 |
| 대항력 보유 | 전입신고 + 점유 상태 |

### 2.2 임대차 종료 사유

| 사유 | 인정 |
|------|------|
| 계약 기간 만료 |  |
| 합의 해지 |  |
| 묵시적 갱신 후 해지 |  |
| 임대인 귀책 해지 |  |

### 2.3 보증금 미반환

| 상태 | 신청 가능 |
|------|----------|
| 전액 미반환 |  |
| 일부 미반환 |  (미반환분) |
| 전액 반환 |  (필요 없음) |

## 신청 절차

### 3.1 관할 법원

| 관할 | 기준 |
|------|------|
| 법원 | 주택 소재지 관할 지방법원 |
| 지원·시군법원 | 소재지에 따라 |

### 3.2 신청 방법

| 방법 | 내용 |
|------|------|
| 방문 | 법원 민원실 |
| 온라인 | 전자소송 (ecfs.scourt.go.kr) |

### 3.3 절차 흐름

```
신청서 작성 → 서류 준비 → 법원 접수 →
비용 납부 → 법원 심사 → 결정 →
등기촉탁 → 등기 완료
```

### 3.4 소요 기간

| 단계 | 기간 |
|------|------|
| 신청~결정 | 1-2주 |
| 결정~등기 | 3~7일 |
| 총 소요 | **약 2-3주** |

## 필요 서류

### 4.1 필수 서류

| 서류 | 용도 |
|------|------|
| 신청서 | 법원 양식 |
| 임대차계약서 사본 | 계약 증명 |
| 주민등록등본 | 전입 증명 |
| 등기부등본 | 부동산 정보 |
| 계약 종료 증빙 | 만료 또는 해지 |

### 4.2 계약 종료 증빙

| 종류 | 용도 |
|------|------|
| 계약서 (만료일 확인) | 기간 만료 |
| 해지 통보 내용증명 | 해지 |
| 합의 해지서 | 합의 해지 |

### 4.3 추가 서류 (상황별)

| 상황 | 추가 서류 |
|------|----------|
| 확정일자 | 확정일자 증명 |
| 보증금 일부 수령 | 영수증 |

## 비용

### 5.1 비용 내역

| 항목 | 금액 |
|------|------|
| 인지대 | 2,000원 |
| 송달료 | 약 5,000원 |
| 등록면허세 | 6,000원 |
| 등기신청 수수료 | 3,000원 |
| **합계** | **약 15,000원** |

### 5.2 납부 방법

| 항목 | 납부처 |
|------|--------|
| 인지대 | 수입인지 구매 |
| 송달료 | 법원 수납 |
| 등록면허세 | 위택스 또는 구청 |

## 결정 및 등기

### 6.1 법원 결정

| 결과 | 처리 |
|------|------|
| 인용 | 등기 촉탁 |
| 기각 | 항고 가능 |

### 6.2 등기 촉탁

| 절차 | 내용 |
|------|------|
| 촉탁 | 법원이 등기소에 촉탁 |
| 등기 | 등기부 갑구에 기재 |

### 6.3 등기 내용

| 기재 사항 | 내용 |
|----------|------|
| 임차인 | 성명, 주민등록번호 |
| 보증금 | 금액 |
| 임대차 기간 | 시작~종료일 |
| 확정일자 | 있는 경우 |

## 임차권등기 효과

### 7.1 대항력 유지

| 효과 | 내용 |
|------|------|
| 전출 가능 | 다른 곳으로 이사 가능 |
| 대항력 | 그대로 유지 |
| 새 전세 | 다른 곳 계약 가능 |

### 7.2 우선변제권 유지

| 효과 | 내용 |
|------|------|
| 경매 배당 | 순위 유지 |
| 확정일자 | 기존 확정일자 효력 유지 |

### 7.3 새 임차인 제한

| 효과 | 내용 |
|------|------|
| 등기 공시 | 등기부등본에 표시 |
| 새 임차인 | 후순위가 됨 |
| 경고 효과 | 문제 있는 물건임을 알림 |

## 등기 후 절차

### 8.1 이사

| 절차 | 내용 |
|------|------|
| 등기 확인 | 등기부등본 열람 |
| 이사 | 전출신고 가능 |
| 권리 유지 | 대항력·우선변제권 유지 |

### 8.2 보증금 회수

| 방법 | 내용 |
|------|------|
| 임대인 청구 | 직접 청구 |
| 지급명령 | 법원 신청 |
| 강제집행 | 경매 등 |

### 8.3 등기 말소

| 시점 | 절차 |
|------|------|
| 보증금 반환 후 | 말소 신청 |
| 신청인 | 임대인 또는 임차인 |

## 주의사항

### 9.1 신청 시점

| 주의 | 내용 |
|------|------|
| 대항력 있을 때 | 전입 유지 상태에서 신청 |
| 전출 후 신청 |  불가 (대항력 상실) |

### 9.2 등기 전 전출

| 주의 | 내용 |
|------|------|
| 등기 완료 전 | 전출하면 안 됨 |
| 등기 완료 후 | 전출 가능 |

### 9.3 새 임대차 제한

| 주의 | 내용 |
|------|------|
| 임차권등기 물건 | 새 임차인 보호 약함 |
| 소액임차인 | 최우선변제 불가 |

## 자주 묻는 질문

## # 10.1 임대인 동의 필요한

| 답변 | 설명 |
|------|------|
|  불필요 | 일방적 신청 가능 |
| 법원 결정 | 임대인 의견 없이 |

## # 10.2 등기비용은 누가 부담

| 답변 | 설명 |
|------|------|
| 일단 임차인 | 신청 시 임차인 부담 |
| 청구 가능 | 나중에 임대인에게 청구 |

## # 10.3 등기 후 경매되면

| 답변 | 설명 |
|------|------|
| 권리 유지 | 배당요구 가능 |
| 순위 | 기존 순위 유지 |

## # 10.4 여러 채 임대차에도 가능

| 답변 | 설명 |
|------|------|
|  가능 | 각각 신청 |
