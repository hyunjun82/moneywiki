# 이자소득 세금 계산

## 기본 개념
예금, 적금, 채권 등에서 발생하는 이자소득에 대한 세금 계산 방법입니다.

## 세율

### 원천징수세율
- **이자소득세**: 14%
- **지방소득세**: 1.4%
- **합계**: **15.4%**

### 계산 예시
```
이자소득: 100만원
이자소득세: 14만원 (100만원 × 14%)
지방소득세: 1.4만원 (100만원 × 1.4%)
실수령액: 84.6만원
```

## 과세 방법

### 원천징수 (분리과세)
- **기본 원칙**
- 금융기관에서 자동 원천징수
- 이자 지급 시 세금 차감
- 별도 신고 불필요

### 종합과세
- **연간 이자·배당소득 2,000만원 초과**
- 다음 연도 5월 종합소득세 신고
- 금융소득종합과세 적용

## 금융소득종합과세

### 기준 금액
- 이자소득 + 배당소득 **2,000만원 초과**

### 과세 방법
```
2,000만원 이하: 14% 원천징수로 종결
2,000만원 초과: 종합소득세율 적용 (6.6%~49.5%)
```

### 계산 예시
```
이자소득: 3,000만원
배당소득: 1,000만원
금융소득 합계: 4,000만원

1단계: 2,000만원까지 → 14% 분리과세
2단계: 초과분 2,000만원 → 종합소득세율 적용
```

## 이자소득 종류별 세금

### 예금·적금 이자
- 원천징수 15.4%
- 2,000만원 초과 시 종합과세

### 채권 이자
- 일반 채권: 15.4%
- 국공채 이자: 14% (지방소득세 없음)

### 저축성보험
- 보험차익: 15.4%
- 10년 이상 비과세 상품 있음

### 환매조건부채권(RP)
- 원천징수 15.4%

### MMF·CMA
- 원천징수 15.4%
- 매일 이자 발생

## 비과세 이자소득

### 비과세 종합저축
- 대상: 65세 이상, 장애인, 독립유공자 등
- 한도: **1인당 5,000만원**
- 이자 전액 비과세

### 장기저축성보험
- 10년 이상 유지
- 월 보험료 150만원 이하
- 이자 비과세

### 세금우대 저축 (폐지)
- 2014년 12월 31일 폐지
- 기존 가입자만 유지

### 청년 우대 저축
- 만 19~34세
- 일부 이자 비과세
- 한도 및 조건 있음

## 실무 계산 사례

### 사례 1: 일반 예금 이자
```
정기예금 1억원
연 이자율: 4%
연 이자: 400만원
원천징수세: 61.6만원 (400만원 × 15.4%)
실수령액: 338.4만원
```

### 사례 2: 금융소득종합과세 대상
```
이자소득: 3,000만원
원천징수: 462만원 (3,000 × 15.4%)

5월 종합소득세 신고:
- 다른 소득과 합산
- 종합소득세율 적용 (예: 35%)
- 추가 납부세액 발생 가능
```

### 사례 3: 비과세 종합저축
```
65세 이상 어르신
비과세 종합저축 5,000만원
연 이자율: 4%
연 이자: 200만원
원천징수세: 0원
실수령액: 200만원 (전액)
```

## 이자소득 신고

### 원천징수만 하는 경우
- 금융소득 2,000만원 이하
- 별도 신고 불필요
- 금융기관에서 자동 처리

### 종합소득세 신고 필요
- 금융소득 2,000만원 초과
- 다음 연도 5월 1일~5월 31일
- 홈택스 또는 세무대리인

## 세금 절감 전략

### 분산 투자
- 가족에게 분산
- 각자 2,000만원 이하 유지
- 종합과세 회피

### 비과세 상품 활용
- 비과세 종합저축 (해당자)
- 장기저축성보험
- 청년 우대 저축

### 과세이연 상품
- 연금저축
- IRP (개인형 퇴직연금)
- 연금 수령 시 과세 이연

### 세금우대 상품
- ISA (개인종합자산관리계좌)
- 200~400만원 비과세
- 초과분 9.9% 분리과세

## 국외 이자소득

### 해외 예금 이자
- 국내와 동일하게 과세
- 종합소득세 신고 필요
- 외국납부세액공제 가능

### 신고 의무
- 금액 무관 모두 신고
- 5월 종합소득세 신고
- 미신고 시 가산세

## 주의사항

### 2,000만원 계산
- 이자소득 + 배당소득 합산
- 원천징수 전 금액 기준

### 원천징수 확인
- 금융기관 명세서 확인
- 이자소득 합산 관리
- 2,000만원 근접 시 주의

### 가족 분산 시
- 실소유자 과세 원칙
- 명의신탁 주의
- 증여세 문제 발생 가능
