# 연말정산 주택담보대출 이자

장기주택저당차입금 이자 소득공제를 정리합니다.

## 1. 공제 한도

| 상환기간 | 금리 | 한도 |
|----------|------|------|
| 15년 이상 | 고정+비거치 | **1,800만원** |
| 15년 이상 | 고정 또는 비거치 | **1,500만원** |
| 15년 이상 | 기타 | **500만원** |
| 10~15년 | - | **300만원** |

## 2. 공제 조건

| 요건 | 기준 |
|------|------|
| 세대주 | 무주택 또는 1주택 |
| 기준시가 | 취득 당시 **5억원 이하** |
| 상환기간 | 10년 이상 |

## 3. 주택 요건

| 항목 | 기준 |
|------|------|
| 주택 규모 | 제한 없음 |
| 기준시가 | 5억원 이하 |
| 취득 시기 | 차입일 3개월 내 |

## 4. 차입금 요건

| 항목 | 기준 |
|------|------|
| 대출처 | 금융기관 |
| 담보 | 해당 주택 |
| 상환방식 | 원리금 또는 원금 |
