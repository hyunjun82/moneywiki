장기주택저당차입금 이자 소득공제를 정리해 볼게요. 주택담보대출 이자도 조건을 충족하면 소득공제받을 수 있어요.

## 공제 한도

상환기간과 금리 유형에 따른 공제 한도예요.

| 상환기간 | 금리 | 한도 |
|----------|------|------|
| 15년 이상 | 고정+비거치 | 1,800만원 |
| 15년 이상 | 고정 또는 비거치 | 1,500만원 |
| 15년 이상 | 기타 | 500만원 |
| 10~15년 | - | 300만원 |

## 공제 조건

공제받으려면 충족해야 하는 조건이에요.

| 요건 | 기준 |
|------|------|
| 세대주 | 무주택 또는 1주택 |
| 기준시가 | 취득 당시 5억원 이하 |
| 상환기간 | 10년 이상 |

## 주택 요건

주택에 대한 요건이에요.

| 항목 | 기준 |
|------|------|
| 주택 규모 | 제한 없음 |
| 기준시가 | 5억원 이하 |
| 취득 시기 | 차입일 3개월 내 |

## 차입금 요건

대출에 대한 요건이에요.

| 항목 | 기준 |
|------|------|
| 대출처 | 금융기관 |
| 담보 | 해당 주택 |
| 상환방식 | 원리금 또는 원금 |
