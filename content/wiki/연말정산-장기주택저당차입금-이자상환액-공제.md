# 장기주택저당차입금 이자상환액 공제

## 기본 개념
주택 구입을 위해 금융기관에서 대출받은 차입금의 이자 상환액에 대한 소득공제입니다.

## 공제 대상 주택

### 주택 요건
- **주택 가격**: 취득 당시 기준시가 **5억원 이하**
- **주택 수**: 무주택 또는 **1주택자**
- **주택 유형**: 아파트, 다가구, 단독주택 등

### 차입금 요건
- **대출기관**: 은행, 보험사, 주택금융공사 등 금융기관
- **상환기간**: **15년 이상** (고정금리·비거치식)
- **상환기간**: **10년 이상** (기타 차입금)
- **대출목적**: 주택 취득 또는 임차보증금 용도

## 공제 한도 및 공제율

### 상환기간 15년 이상 (고정금리·비거치식)
| 구분 | 공제율 | 한도 |
|------|-------|------|
| 상환기간 15~29년 | 1,500만원 | 연 1,800만원 |
| 상환기간 30년 이상 | 1,800만원 | (다른 공제 포함) |

### 상환기간 10년 이상
- **공제 한도**: 연 **300만원**
- **공제율**: 상환액의 일정 비율

### 고정금리·비거치식 요건
- **고정금리**: 대출기간 동안 금리 변동 없음
- **비거치식**: 원금과 이자를 매월 함께 상환

## 무주택 세대주 우대

### 우대 요건
- 무주택 세대주가 주택 구입
- 대출 실행일로부터 3개월 이내 주택 취득
- 취득 후 3개월 이내 전입

### 우대 내용
- 공제 한도 상향
- 공제율 우대

## 공제 제외 사례

### 주택 요건 미달
- 취득 당시 기준시가 5억원 초과
- 2주택 이상 보유 (일시적 2주택 제외)

### 차입금 요건 미달
- 개인 간 차입금
- 상환기간 10년 미만
- 회사 대출금 (복리후생 차원)

### 기타 제외
- 전세자금 대출 (별도 공제 있음)
- 주택 리모델링 대출
- 담보 대출 중 주택 외 용도

## 실무 적용 사례

### 사례 1: 15년 고정금리 대출
```
대출금: 3억원
연이자: 1,200만원
상환기간: 15년 (고정금리·비거치식)
→ 공제액: 1,200만원 (한도 1,500만원 이내)
```

### 사례 2: 10년 대출
```
대출금: 2억원
연이자: 800만원
상환기간: 10년
→ 공제액: 300만원 (한도)
```

### 사례 3: 일시적 2주택
```
기존 주택 보유 중 신규 주택 구입
→ 3년 이내 기존 주택 처분 시 공제 가능
→ 처분하지 않으면 공제 불가
```

## 주의사항

### 한도 통합 관리
- 주택임차차입금 원리금 상환액 (연 300만원)
- 장기주택저당차입금 이자상환액 (연 300~1,800만원)
- **합산 한도**: 연 **1,800만원**

### 맞벌이 부부
- 대출자 명의인만 공제 가능
- 공동명의 시 지분율대로 안분
- 한 사람이 전액 공제 불가

### 주택 가격 확인
- 아파트: 국세청 기준시가
- 단독·다가구: 개별주택가격
- 취득 당시 기준으로 판단

## 신청 방법

### 준비 서류
1. 주택취득 차입금 이자상환증명서 (금융기관 발급)
2. 주민등록등본
3. 건물등기부등본
4. 주택 가격 확인 서류

### 연말정산 시
1. 회사에 증명서 제출
2. 간소화 서비스에서 자동 조회
3. 공제 금액 확인
