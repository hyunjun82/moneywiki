# 청년도약계좌

청년도약계좌는 청년의 자산 형성을 지원하는 정책 금융상품입니다. 가입 조건과 혜택을 정리합니다.

## 상품 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 상품명 | 청년도약계좌 |
| 운영 | 11개 은행 |
| 가입 기간 | **5년** (60개월) |
| 목표 금액 | 약 **5,000만원** |

### 1.2 지원 내용

| 지원 | 내용 |
|------|------|
| 정부 기여금 | 소득별 차등 지원 |
| 비과세 | 이자소득 비과세 |
| 금리 | 시중 금리 + 우대금리 |

## 가입 조건

### 2.1 연령 조건

| 항목 | 기준 |
|------|------|
| 연령 | 만 **19세 ~ 34세** |
| 기준일 | 가입 신청일 |
| 병역 | 최대 6년 추가 인정 |

### 2.2 소득 조건

| 항목 | 기준 |
|------|------|
| 총급여 | **7,500만원 이하** |
| 종합소득 | **6,300만원 이하** |
| 기준 연도 | 직전 과세기간 |

### 2.3 가구소득 조건

| 항목 | 기준 |
|------|------|
| 가구소득 | 기준 중위소득 **180% 이하** |

### 2.4 금융소득 제한

| 항목 | 기준 |
|------|------|
| 금융소득종합과세 | 대상자 **제외** |
| 기준 | 이자·배당 2,000만원 초과 |

## 납입 조건

### 3.1 납입 한도

| 항목 | 금액 |
|------|------|
| 월 납입 | **40~70만원** |
| 자유납입 | 범위 내 자유 |

### 3.2 의무 납입

| 항목 | 조건 |
|------|------|
| 최소 납입 | 월 40만원 이상 |
| 납입 중단 | 6개월 이상 시 해지 가능 |

## 정부 기여금

### 4.1 소득별 기여금

| 개인소득 (총급여) | 월 기여금 |
|------------------|----------|
| 2,400만원 이하 | **2.4만원** |
| 2,400만~3,600만원 | **2.3만원** |
| 3,600만~4,800만원 | **2.2만원** |
| 4,800만~6,000만원 | **2.1만원** |
| 6,000만~7,500만원 | **2.0만원** |

### 4.2 5년간 총 기여금

| 개인소득 | 5년 기여금 |
|----------|-----------|
| 2,400만원 이하 | **144만원** |
| 2,400만~3,600만원 | **138만원** |
| 3,600만~4,800만원 | **132만원** |
| 4,800만~6,000만원 | **126만원** |
| 6,000만~7,500만원 | **120만원** |

### 4.3 기여금 매칭 방식

| 방식 | 내용 |
|------|------|
| 매칭 | 본인 납입액에 비례 |
| 70만원 납입 시 | 100% 기여금 |
| 40만원 납입 시 | 비례 축소 |

## 금리

### 5.1 금리 구조

| 구분 | 금리 |
|------|------|
| 기본금리 | 연 **3.5%** 내외 |
| 우대금리 | 최대 **+0.5%** |
| 합계 | 최대 연 **4.0%** |

### 5.2 우대금리 조건

| 조건 | 우대 |
|------|------|
| 급여이체 | +0.2~0.3% |
| 카드 사용 | +0.1~0.2% |
| 마케팅 동의 | +0.1% |

## 비과세 혜택

### 6.1 세금 혜택

| 구분 | 일반 저축 | 청년도약계좌 |
|------|----------|-------------|
| 이자소득세 | 15.4% | **0%** |
| 비과세 조건 | - | 만기 유지 |

### 6.2 혜택 계산

**5년간 70만원 납입 시**

| 항목 | 금액 |
|------|------|
| 본인 납입 | 4,200만원 |
| 정부 기여금 | 최대 144만원 |
| 이자 (4% 가정) | 약 450만원 |
| 비과세 절감 | 약 70만원 |
| **만기 수령액** | **약 4,800만원** |
