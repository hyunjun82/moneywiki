외벌이 가구는 배우자와 부양가족 공제를 최대한 활용할 수 있어요. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6591&cntntsId=7870)에 따른 절세 전략을 정리해 볼게요.

## 활용 가능 공제

[소득세법 제50조](https://www.law.go.kr/법령/소득세법)에 따른 인적공제 항목이에요.

| 공제 | 금액 |
|------|------|
| 배우자 기본공제 | 150만원 |
| 부양가족 (1인당) | 150만원 |
| 경로우대 (70세 이상) | 100만원 추가 |
| 장애인 | 200만원 추가 |

부양가족이 많을수록 공제 금액이 커져요.

## 배우자 공제 조건

배우자 공제를 받으려면 소득 요건을 충족해야 해요.

| 요건 | 기준 |
|------|------|
| 소득금액 | 100만원 이하 |
| 근로소득만 있는 경우 | 총급여 500만원 이하 |

배우자가 무소득이거나 소득이 적으면 150만원 공제를 받을 수 있어요.

## 카드 사용 전략

외벌이 가구는 카드 사용을 소득자에게 몰아주는 게 유리해요.

| 전략 | 내용 |
|------|------|
| 소득자 명의 카드 | 공제 대상 |
| 배우자 사용분 | 소득자가 공제 가능 |
| 가족카드 | 명의자가 공제 |

배우자가 사용한 카드도 소득자가 공제받을 수 있어요.

## 의료비 전략

가족 전체 의료비를 소득자가 공제받을 수 있어요.

| 전략 | 내용 |
|------|------|
| 가족 의료비 | 소득자가 공제 |
| 배우자 의료비 | 소득자가 공제 |
| 부모님 의료비 | 소득자가 공제 |
| 자녀 의료비 | 소득자가 공제 |

의료비는 총급여 3% 초과분부터 공제되므로 한 사람에게 몰아주면 기준을 빨리 넘겨요.

## 보험료 전략

보험료 공제는 계약자와 피보험자 조건을 확인해야 해요.

| 전략 | 내용 |
|------|------|
| 가족 보험 | 피보험자가 부양가족이면 공제 가능 |
| 계약자 | 본인 또는 배우자 |

보험 계약자를 소득자로 하면 공제받기 편해요.

## 교육비 전략

자녀 교육비도 소득자가 공제받을 수 있어요.

| 대상 | 공제 한도 |
|------|----------|
| 취학전 아동 | 300만원 |
| 초중고 | 300만원 |
| 대학생 | 900만원 |

## 주의사항

배우자가 소득금액 100만원을 초과하면 배우자 공제를 받을 수 없어요.

부양가족이 다른 형제자매와 중복으로 등록되지 않도록 확인하세요.

가족카드 사용분은 카드 명의자만 공제받을 수 있으므로 소득자 명의로 발급받는 게 좋아요.
