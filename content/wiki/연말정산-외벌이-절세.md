# 연말정산 외벌이 절세

외벌이 가구 절세 전략을 정리합니다.

## 1. 활용 가능 공제

| 공제 | 금액 |
|------|------|
| 배우자 기본공제 | 150만원 |
| 부양가족 (1인당) | 150만원 |
| 경로우대 (70세 이상) | 100만원 |
| 장애인 | 200만원 |

## 2. 배우자 공제 조건

| 요건 | 기준 |
|------|------|
| 소득금액 | 100만원 이하 |
| 근로소득만 | 총급여 500만원 이하 |

## 3. 카드 사용 전략

| 전략 | 내용 |
|------|------|
| 소득자 명의 | 공제 대상 |
| 배우자 사용 | 소득자가 공제 |
| 가족카드 | 명의자가 공제 |

## 4. 의료비 전략

| 전략 | 내용 |
|------|------|
| 가족 의료비 | 소득자가 공제 |
| 배우자 의료비 | 소득자가 공제 |
| 부모님 의료비 | 소득자가 공제 |

## 5. 보험료 전략

| 전략 | 내용 |
|------|------|
| 가족 보험 | 피보험자가 부양가족이면 공제 |
| 계약자 | 본인 또는 배우자 |
