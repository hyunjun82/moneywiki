# 계약갱신청구권 행사방법

계약갱신청구권은 세입자가 안정적으로 거주할 수 있도록 보장하는 제도입니다. 2020년 7월 시행된 임대차3법의 핵심입니다.

## 1. 계약갱신청구권이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 권리 | 임차인이 계약 갱신을 **요구**하는 권리 |
| 효력 | 임대인은 정당한 사유 없이 거절 불가 |
| 근거 | 주택임대차보호법 제6조의3 |

### 1.2 제도 개요

| 항목 | 내용 |
|------|------|
| 사용 횟수 | **1회** |
| 갱신 기간 | **2년** |
| 총 거주 기간 | 최초 2년 + 갱신 2년 = **4년** |
| 시행일 | 2020년 7월 31일 |

## 2. 행사 요건

### 2.1 기본 요건

| 요건 | 내용 |
|------|------|
| 임대차 계약 | 주거용 주택 임대차 |
| 사용 횟수 | 1회 미사용 |
| 행사 시기 | 만료 6개월~1개월 전 |

### 2.2 대상 임대차

| 대상 | 적용 |
|------|------|
| 아파트 | ✅ |
| 다세대·빌라 | ✅ |
| 단독주택 | ✅ |
| 오피스텔 (주거용) | ✅ |
| 상가 | ❌ (상가임대차법 적용) |

### 2.3 적용 대상 계약

| 계약 시점 | 적용 |
|----------|------|
| 2020.7.31 이전 체결, 갱신 | ✅ 적용 |
| 2020.7.31 이후 체결 | ✅ 적용 |

## 3. 행사 시기

### 3.1 행사 기간

| 기간 | 내용 |
|------|------|
| 시작 | 계약 만료 **6개월 전** |
| 종료 | 계약 만료 **1개월 전** |

### 3.2 기간 계산 예시

**계약 만료일: 2026년 6월 30일**

| 시점 | 날짜 | 행사 가능 |
|------|------|----------|
| 6개월 전 | 2025.12.31 | ✅ 시작 |
| 3개월 전 | 2026.3.31 | ✅ 가능 |
| 1개월 전 | 2026.5.31 | ✅ 마지막 |
| 1개월 미만 | 2026.6.15 | ❌ 불가 |

### 3.3 기간 도과 시

| 상황 | 결과 |
|------|------|
| 6개월 전 이전 | 유효하지 않음 |
| 1개월 전 이후 | 권리 상실 |
| 기간 내 미행사 | 묵시적 갱신 가능 |

## 4. 행사 방법

### 4.1 통지 방법

| 방법 | 효력 | 권장 |
|------|------|------|
| 내용증명 | ✅ 확실한 증거 | ⭐ 권장 |
| 문자·카톡 | ✅ 가능 | ○ |
| 구두 | ✅ 가능 | △ 증거 확보 어려움 |

### 4.2 내용증명 작성

**필수 기재 사항**

| 항목 | 내용 |
|------|------|
| 제목 | 계약갱신청구권 행사 통지 |
| 계약 내용 | 주소, 계약기간, 보증금 |
| 행사 의사 | 계약갱신청구권 행사 명시 |
| 갱신 조건 | 동일 조건 또는 5% 이내 인상 |

### 4.3 내용증명 예시

```
제목: 계약갱신청구권 행사 통지

수신: 임대인 홍길동 귀하

본인은 귀하와 아래 주택에 대해 임대차계약을 체결한
임차인입니다.

- 주소: 서울시 강남구 테헤란로 123, 456동 789호
- 계약기간: 2024.7.1. ~ 2026.6.30.
- 보증금: 금 300,000,000원

주택임대차보호법 제6조의3에 따라 계약갱신청구권을
행사합니다.

갱신되는 임대차는 전 임대차와 동일한 조건으로 하되,
보증금은 5% 범위 내에서 협의할 수 있습니다.

통지일: 2026년 1월 15일
임차인: 김철수
```

## 5. 갱신 조건

### 5.1 갱신 기간

| 항목 | 기간 |
|------|------|
| 갱신 기간 | **2년** |
| 단축 | 불가 |
| 연장 | 합의 시 가능 |

### 5.2 전월세상한제

| 항목 | 한도 |
|------|------|
| 보증금 인상 | **5% 이내** |
| 월세 인상 | **5% 이내** |
| 초과 인상 | 무효 |

### 5.3 5% 인상 계산

**보증금 3억원 갱신 시**

| 항목 | 금액 |
|------|------|
| 현재 보증금 | 3억원 |
| 5% 인상 한도 | 1,500만원 |
| 최대 갱신 보증금 | **3억 1,500만원** |

### 5.4 월세 환산

**전월세 전환 시**

| 항목 | 기준 |
|------|------|
| 전환율 | 한국은행 기준금리 + 2% |
| 상한 | 연 10% |

## 6. 임대인 거절 사유

### 6.1 정당한 거절 사유

| 사유 | 설명 |
|------|------|
| 임대인 **실거주** | 임대인 또는 직계존비속 거주 |
| 2기 차임 **연체** | 2개월분 이상 연체 |
| 임차인 동의 **전대** | 동의 후 전대 |
| 무단 **전대** | 무단으로 제3자에게 전대 |
| 고의·중과실 **파손** | 주택 훼손 |
| 거짓·부정 **계약** | 허위 정보로 계약 |
| 철거·재건축 | 주택 철거 또는 재건축 |
| 상당한 보상 | 상당한 보상 제공 |

### 6.2 임대인 실거주 요건

| 요건 | 내용 |
|------|------|
| 대상 | 임대인 본인 또는 직계존비속 |
| 시기 | 계약 종료 후 즉시 |
| 기간 | 2년 이상 실거주 의무 |

### 6.3 허위 실거주 시 제재

| 제재 | 내용 |
|------|------|
| 손해배상 | 임차인에게 손해배상 |
| 배상액 | 갱신거절로 인한 손해 |
| 청구 기간 | 3년 내 |

## 8. 묵시적 갱신과 차이

### 8.1 비교

| 구분 | 계약갱신청구권 | 묵시적 갱신 |
|------|--------------|------------|
| 방식 | 임차인 **청구** | 자동 갱신 |
| 요건 | 6개월~1개월 전 행사 | 쌍방 갱신 거절 없음 |
| 기간 | 2년 | 2년 |
| 상한제 | 5% 적용 | 적용 안 됨 |

### 8.2 선택 기준

| 상황 | 권장 |
|------|------|
| 보증금 인상 억제 | 계약갱신청구권 |
| 5% 초과 인상 협의 | 신규 계약 |
| 자동 갱신 원할 때 | 묵시적 갱신 |

## 9. 실무 체크리스트

### 9.1 행사 전 확인

| 확인 | 내용 |
|------|------|
| ☐ 1회 미사용 확인 | 이전에 행사한 적 없는지 |
| ☐ 행사 기간 확인 | 6개월~1개월 전인지 |
| ☐ 거절 사유 검토 | 연체 등 사유 없는지 |

### 9.2 행사 시

| 절차 | 확인 |
|------|------|
| ☐ 내용증명 작성 | 행사 의사 명확히 |
| ☐ 내용증명 발송 | 기간 내 도달 |
| ☐ 수령 확인 | 배달 증명 보관 |

### 9.3 갱신 계약

| 절차 | 확인 |
|------|------|
| ☐ 갱신 조건 협의 | 5% 이내 |
| ☐ 계약서 작성 | 갱신 계약서 |
| ☐ 확정일자 | 필요 시 재취득 |

## 10. 자주 묻는 질문

### 10.1 2회 이상 행사 가능한가요?

| 답변 | 설명 |
|------|------|
| ❌ 불가 | 1회만 가능 |
| 이후 | 합의 갱신 또는 묵시적 갱신 |

### 10.2 갱신 후 중도 퇴거 가능한가요?

| 답변 | 설명 |
|------|------|
| ⚠️ 원칙적 불가 | 계약 기간 준수 |
| 예외 | 임대인 동의 시 중도 해지 |

### 10.3 갱신 시 확정일자 다시 받아야 하나요?

| 상황 | 필요 여부 |
|------|----------|

| 보증금 증액 | **새로 받아야** |
