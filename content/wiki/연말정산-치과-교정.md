# 연말정산 치과 교정

치과 교정비 의료비 공제를 정리합니다.

## 1. 공제 기준

| 목적 | 공제 |
|------|------|
| 치료 목적 | ✅ 가능 |
| 미용 목적 | ❌ 불가 |

## 2. 치료 목적 예시

| 치료 목적 | 공제 |
|----------|------|
| 저작기능 장애 | ✅ |
| 부정교합 치료 | ✅ |
| 선천성 기형 | ✅ |
| 단순 심미 교정 | ❌ |

## 3. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제율 | **15%** |
| 한도 | 700만원 (본인·장애인·65세 이상은 무제한) |

## 4. 필요서류

| 서류 | 발급처 |
|------|--------|
| 의료비 영수증 | 치과 |
| 진단서 | 치과 (치료 목적 증명) |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 진단서 | 치료 필요성 명시 권장 |
| 미용 | 미용 목적 시 공제 거부 가능 |
