# 연말정산 의료비 본인부담금 공제

의료비 본인부담금 공제는 건강보험 적용 후 **본인이 실제 부담한 금액**에 대해 세액공제를 받는 제도입니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 조건을 정리합니다.

## 1. 공제 대상 금액

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 공제 대상입니다.

### 1.1 공제 대상

| 항목 | 공제 여부 |
|------|----------|
| 건강보험 본인부담금 | ✅ 공제 |
| 비급여 진료비 | ✅ 공제 |
| 건강보험 공단부담금 | ❌ 대상 아님 |

### 1.2 계산 구조

| 순서 | 내용 |
|------|------|
| 총 진료비 | 100% |
| - 건강보험 공단부담금 | 약 70% |
| = **본인부담금** | 약 30% (공제 대상) |

## 2. 비급여 항목

### 2.1 공제 가능

| 항목 | 공제 여부 |
|------|----------|
| MRI, CT 검사 (비급여) | ✅ 가능 |
| 선택 진료비 | ✅ 가능 |
| 상급병실료 | ✅ 가능 |
| 비급여 주사제 | ✅ 가능 |
| 비급여 약제비 | ✅ 가능 |

### 2.2 공제 불가

| 항목 | 공제 여부 |
|------|----------|
| 미용·성형 수술 | ❌ 불가 |
| 미용 목적 피부과 시술 | ❌ 불가 |
| 건강검진 (대부분) | ❌ 불가 |

## 3. 보전받은 금액 제외

### 3.1 공제 제외 항목

| 항목 | 공제 |
|------|------|
| 실손의료보험 환급금 | ❌ 제외 |
| 본인부담상한제 환급금 | ❌ 제외 |
| 회사 의료비 지원금 | ❌ 제외 |

### 3.2 계산 예시

| 항목 | 금액 |
|------|------|
| 총 의료비 지출 | 500만원 |
| 실손보험 환급 | -200만원 |
| **공제 대상** | **300만원** |

## 4. 본인부담상한제

### 4.1 개념

연간 의료비 본인부담금이 상한액을 초과하면 **초과분 환급**

| 소득분위 | 상한액 (2025년) |
|---------|----------------|
| 1분위 | 87만원 |
| 2~3분위 | 108만원 |
| 4~5분위 | 162만원 |
| 6~7분위 | 303만원 |
| 8분위 | 414만원 |
| 9분위 | 497만원 |
| 10분위 | 780만원 |

### 4.2 연말정산 처리

| 상황 | 처리 |
|------|------|
| 연도 중 환급 | 공제 대상 금액에서 차감 |
| 연말정산 후 환급 | 수정신고 또는 다음 해 조정 |

## 5. 공제 계산

### 5.1 기본 공식

```
공제 대상 = 의료비 지출 - 보전금액 - (총급여 × 3%)
세액공제 = 공제 대상 × 15%
```

### 5.2 계산 예시

**총급여 5,000만원, 의료비 400만원, 실손보험 100만원**

| 항목 | 금액 |
|------|------|
| 의료비 지출 | 400만원 |
| 실손보험 환급 | -100만원 |
| 실제 부담금 | 300만원 |
| 3% 기준액 | -150만원 |
| 공제 대상 | 150만원 |
| **세액공제** | 150만원 × 15% = **22.5만원** |

## 6. 증빙서류

### 6.1 연말정산 간소화

| 항목 | 조회 |
|------|------|
| 병원 진료비 | ✅ 자동 조회 |
| 약국 약제비 | ✅ 자동 조회 |
| 실손보험 환급액 | ✅ 자동 반영 |

### 6.2 실손보험 처리

- 간소화 서비스에서 **실손보험 환급액 자동 차감**
- 별도 계산 불필요

## 7. 부양가족 의료비

### 7.1 특례

의료비는 부양가족의 **나이·소득 제한 없이** 공제 가능

| 대상 | 공제 |
|------|------|
| 소득 있는 배우자 | ✅ 가능 |
| 소득 있는 부모님 | ✅ 가능 |
| 60세 미만 부모님 | ✅ 가능 |

### 7.2 조건

- 의료비를 **실제 지출한 본인**이 공제
- 다른 가족이 기본공제하면 그 가족이 의료비 공제

## 8. 주의사항

### 8.1 이중 공제 불가

- 같은 의료비를 **부부 각각 공제 불가**
- 한 명만 공제

### 8.2 연도 기준

- **지출 연도** 기준으로 공제
- 진료는 작년, 결제는 올해 → 올해 공제

### 8.3 해외 의료비

- 해외 의료기관 진료비도 공제 가능
- **영수증** 직접 제출 필요
