육아휴직급여는 자녀 양육을 위해 휴직하는 근로자에게 지급되는 급여예요. [고용보험법](https://www.law.go.kr/법령/고용보험법)에 따른 내용을 정리해드릴게요.

## 육아휴직 개요

### 1.1 대상

| 조건 | 기준 |
|------|------|
| 자녀 연령 | 만 **8세 이하** (초등 2학년 이하) |
| 고용보험 가입 | **180일 이상** |
| 성별 | 남녀 모두 가능 |

### 1.2 휴직 기간

| 항목 | 기간 |
|------|------|
| 최대 기간 | 자녀 1명당 **1년** |
| 부모 합산 | 각각 1년씩 (총 2년) |
| 분할 사용 | 가능 |

## 육아휴직급여 금액

### 2.1 기본 급여

| 기간 | 지급액 |
|------|--------|
| 전 기간 | 통상임금의 **80%** |
| 상한액 | **150만원** |
| 하한액 | **70만원** |

### 2.2 월급별 예시

| 통상임금 | 80% 계산 | 실 지급액 |
|----------|----------|----------|
| 200만원 | 160만원 | **150만원** (상한) |
| 150만원 | 120만원 | **120만원** |
| 80만원 | 64만원 | **70만원** (하한) |

### 2.3 지급 방식

| 구분 | 비율 |
|------|------|
| 휴직 중 | 75% |
| 복직 후 6개월 | 25% |

## 6+6 부모육아휴직제

### 3.1 제도 개요

| 조건 | 내용 |
|------|------|
| 대상 | 생후 18개월 이내 자녀 |
| 조건 | 부모 **모두 육아휴직** 사용 |
| 혜택 | 첫 6개월 상한 상향 |

### 3.2 급여 상향

| 개월 | 상한액 |
|------|--------|
| 1개월 | **250만원** |
| 2개월 | **250만원** |
| 3개월 | **300만원** |
| 4개월 | **350만원** |
| 5개월 | **400만원** |
| 6개월 | **450만원** |

### 3.3 예시

**부부 모두 6개월 육아휴직 (통상임금 400만원)**

| 기간 | 아빠 | 엄마 |
|------|------|------|
| 1개월 | 250만원 | 250만원 |
| 2개월 | 250만원 | 250만원 |
| 3개월 | 300만원 | 300만원 |
| 4개월 | 350만원 | 350만원 |
| 5개월 | 400만원 | 400만원 |
| 6개월 | 450만원 | 450만원 |
| **합계** | **2,000만원** | **2,000만원** |

## 신청 방법

### 4.1 신청 절차

```
사업주에 휴직 신청 → 휴직 개시 → 고용센터에 급여 신청 → 매월 지급
```

### 4.2 신청 시기

| 시기 | 내용 |
|------|------|
| 휴직 신청 | 휴직 시작 **30일 전** (사업주에) |
| 급여 신청 | 휴직 시작 후 **1개월~12개월** 내 |

### 4.3 신청 방법

| 방법 | 내용 |
|------|------|
| 온라인 | 고용보험 홈페이지 (ei.go.kr) |
| 방문 | 고용센터 |

### 4.4 필요 서류

| 서류 | 용도 |
|------|------|
| 육아휴직 확인서 | 사업주 확인 |
| 통장 사본 | 급여 입금 |
| 가족관계증명서 | 자녀 확인 |

## 육아기 근로시간 단축

### 5.1 제도 개요

| 항목 | 내용 |
|------|------|
| 대상 | 육아휴직 대신 선택 |
| 단축 | 주 15~35시간 |
| 기간 | 최대 2년 |

### 5.2 급여

| 항목 | 내용 |
|------|------|
| 단축 급여 | 단축 시간분 × 통상임금 80% |
| 상한 | 150만원 |

## 특수 상황

### 6.1 한부모

| 혜택 | 내용 |
|------|------|
| 급여 | 통상임금의 **100%** |
| 상한 | **250만원** |

### 6.2 쌍둥이

| 내용 | 기간 |
|------|------|
| 각 자녀별 | 1년씩 사용 가능 |
| 동시 사용 | 가능 |

### 6.3 배우자 출산휴가

| 항목 | 내용 |
|------|------|
| 기간 | **10일** 유급 |
| 급여 | 통상임금 전액 |

## 복직 보장

### 7.1 복직 보장 의무

| 의무 | 내용 |
|------|------|
| 원직 복귀 | 동일 업무 또는 동등 수준 |
| 해고 금지 | 휴직 기간 중 해고 불가 |
| 불이익 금지 | 승진·인사 불이익 금지 |

### 7.2 위반 시

| 위반 | 제재 |
|------|------|
| 복직 거부 | 3년 이하 징역 또는 3천만원 벌금 |
| 불이익 처우 | 노동위원회 구제 |

## 주의사항

### 8.1 급여 제한

| 사유 | 제한 |
|------|------|
| 취업 | 급여 지급 중지 |
| 자영업 | 급여 지급 중지 |
| 주 10시간 초과 근무 | 50% 감액 |

### 8.2 사후지급금

| 항목 | 내용 |
|------|------|
| 25% 사후지급 | 복직 후 **6개월** 계속 근무 시 |
| 미복직 시 | 사후지급금 미지급 |
