# 연말정산 치과 공제

치과 치료비는 의료비 세액공제 항목으로, 치료 목적 시술은 **15% 공제**됩니다. [국세청 의료비 공제 안내](https://call.nts.go.kr/call/qna/selectQnaInfo.do?mi=1318&ctgId=CTG11909)에 따른 공제 기준을 정리합니다.

## 1. 공제 대상 치과 시술

### 1.1 공제 가능 (치료 목적)

| 시술 | 공제 여부 | 비고 |
|------|----------|------|
| 충치 치료 | ✅ 가능 | |
| 신경 치료 | ✅ 가능 | |
| 발치 | ✅ 가능 | |
| 잇몸 치료 | ✅ 가능 | 치주 질환 |
| **임플란트** | ✅ 가능 | 전액 공제 |
| **틀니** | ✅ 가능 | 전액 공제 |
| 브릿지 | ✅ 가능 | |
| 크라운 | ✅ 가능 | |
| 스케일링 | ✅ 가능 | 예방 치료 |
| 레진/아말감 충전 | ✅ 가능 | |

### 1.2 조건부 공제

| 시술 | 조건 |
|------|------|
| **치열교정** | 저작기능장애 진단서 필요 |
| **교합 교정** | 의학적 필요성 증명 필요 |

### 1.3 공제 불가 (미용 목적)

| 시술 | 공제 여부 |
|------|----------|
| 치아 미백 | ❌ 불가 |
| 라미네이트 | ❌ 불가 |
| 미용 목적 치열교정 | ❌ 불가 |
| 치아 성형 | ❌ 불가 |

## 2. 치열교정 공제 조건

### 2.1 공제 가능한 경우

[국세청 세법해석사례](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2443&cntntsId=7786)에 따르면:

| 조건 | 내용 |
|------|------|
| 진단 | **저작기능장애** 진단서 필요 |
| 발급처 | 치과 전문의 |
| 기재 내용 | 부정교합으로 인한 기능 장애 명시 |

### 2.2 저작기능장애 예시

- 음식물 저작 곤란
- 발음 장애
- 턱관절 장애
- 심한 부정교합으로 인한 구강 기능 이상

### 2.3 미용 목적 (공제 불가)

- 외모 개선 목적
- 단순 치아 배열 정리
- 기능 장애 없는 교정

## 3. 공제율과 한도

### 3.1 공제 기준

| 항목 | 내용 |
|------|------|
| 공제율 | **15%** |
| 기준 | 총급여 3% 초과분 |

### 3.2 한도 적용

| 대상 | 한도 |
|------|------|
| 본인 | **한도 없음** |
| 장애인 | **한도 없음** |
| 65세 이상 | **한도 없음** |
| 그 외 부양가족 | **700만원** |

## 4. 공제 계산 예시

### 4.1 본인 임플란트

**연봉 5,000만원, 본인 임플란트 500만원 지출**

1. 3% 기준액: 150만원
2. 공제 대상: 500만원 - 150만원 = 350만원
3. 세액공제: 350만원 × 15% = **52만 5천원**

### 4.2 부모님 틀니

**연봉 5,000만원, 65세 부모님 틀니 300만원 지출**

1. 3% 기준액: 150만원
2. 공제 대상: 300만원 - 150만원 = 150만원
3. 세액공제: 150만원 × 15% = **22만 5천원**

※ 65세 이상은 한도 없음

## 5. 증빙서류

### 5.1 연말정산 간소화

치과 진료비는 **간소화 서비스에서 자동 조회**

### 5.2 치열교정 시 추가 서류

| 서류 | 용도 |
|------|------|
| 저작기능장애 진단서 | 치료 목적 증명 |
| 치료 계획서 | 의학적 필요성 확인 |

## 6. 실손보험과 관계

### 6.1 실손보험 보전금액

실손보험으로 보전받은 금액은 **공제 대상에서 제외**

| 구분 | 금액 |
|------|------|
| 임플란트 비용 | 500만원 |
| 실손보험 환급 | -200만원 |
| **공제 대상** | **300만원** |

### 6.2 비급여 항목

- 임플란트, 틀니 일부 항목은 **건강보험 비급여**
- 전액 본인 부담 → **전액 공제 대상**

## 7. 자주 묻는 질문

### Q. 치과 정기검진도 공제되나요?
**A.** 예방 목적의 검진, 스케일링은 공제됩니다.

### Q. 자녀 치열교정 비용도 공제되나요?
**A.** 저작기능장애 진단서가 있으면 공제됩니다. 미용 목적은 불가합니다.

### Q. 해외에서 받은 치과 치료도 공제되나요?
**A.** 네. 외국 의료기관 영수증을 직접 제출하면 공제 가능합니다.
