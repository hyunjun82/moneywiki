"묵시적갱신 됐는데 부동산에서 복비 달라고 해요."

**안 내도 돼요.** 부동산 안 거쳤잖아요.

## 묵시적갱신 복비 없는 이유

### 중개 서비스 안 받았으니까

[묵시적갱신](/w/묵시적갱신)은 **자동 연장**이에요.

- 부동산 중개: X
- 새 계약서 작성: X
- 당사자 협의: X

양쪽 다 아무 말 안 하면 자동으로 2년 연장되는 거예요. 부동산이 뭘 해준 게 없어요.

**복비는 중개 서비스에 대한 대가**예요. 서비스 안 받았으면 낼 이유 없어요.

### 법적 근거

[공인중개사법 시행규칙](https://www.law.go.kr/법령/공인중개사법시행규칙)에서 중개수수료는 **중개가 성립**됐을 때 받을 수 있다고 돼 있어요.

묵시적갱신은 중개가 성립된 게 아니에요. 법적으로 자동 연장된 거예요.

## 부동산에서 복비 달라고 하면

### 거부하세요

묵시적갱신은 **부동산 중개 없이** 된 거예요. 뭐라고 해도 안 내도 돼요.

- "묵시적갱신도 갱신이니까 복비 내세요" → 거부
- "처음 계약할 때 중개했으니까요" → 거부
- "계약서 정리해줬으니까요" → 거부

### 뭐라고 말해요?

> "묵시적갱신은 법적으로 자동 연장된 거라 중개 서비스 받은 게 없어요. 복비 낼 의무 없어요."

이렇게 말하면 돼요.

### 처음 계약 복비는 별개

"처음 계약할 때 복비 냈잖아요."

그건 **처음 계약 중개비**예요. 묵시적갱신이랑 별개예요.

- 처음 계약: 복비 O (이미 냄)
- 묵시적갱신: 복비 X (새로운 중개 없음)

한 번 냈다고 갱신될 때마다 내는 거 아니에요.

## 묵시적갱신 vs 재계약 복비

### 묵시적갱신

- 복비: **없음**
- 이유: 자동 연장, 중개 없음

### 재계약 (부동산 통해서)

- 복비: **있음**
- 이유: 중개 서비스 받음

새로 계약서 쓰는데 부동산 통하면 복비 나와요. 직접 쓰면 안 나오고요.

### 계약갱신청구권 복비

[계약갱신청구권](/w/계약갱신청구권-복비)도 직접 집주인한테 통보하면 복비 없어요. 부동산 끼면 나올 수 있고요.

- 직접 행사: 복비 없음
- 부동산 통해서: 복비 있을 수 있음

## 복비 비교표

**핵심:** 부동산 안 거치면 복비 없어요.

- 묵시적갱신: 복비 X
- 계약갱신청구권 (직접): 복비 X
- 계약갱신청구권 (부동산 통해서): 복비 O
- 재계약 (직접): 복비 X
- 재계약 (부동산 통해서): 복비 O
- 신규 계약: 복비 O

## 복비 아끼는 방법

### 직접 하기

부동산 안 거치면 복비 0원이에요.

- 묵시적갱신 기다리기: 0원
- 집주인한테 직접 갱신 통보: 0원
- 계약서 직접 쓰기: 0원

### 계약서만 부탁하기

"협의는 다 됐는데 계약서만 작성해주세요."

이러면 복비 대신 **계약서 작성 비용**만 낼 수도 있어요. 보통 10~20만원 정도. 부동산마다 달라요.

## 주의사항

### 새 계약서 쓰면 달라질 수 있어요

묵시적갱신 됐는데 **새 계약서 쓰면** 어떻게 되나요?

- 묵시적갱신 후 그냥 살기: 복비 X
- 묵시적갱신 후 직접 계약서: 복비 X
- 묵시적갱신 후 부동산 통해 계약서: 복비 O

부동산 통해서 새 계약서 쓰면 복비 나올 수 있어요.

### 집주인이 부동산 통하자고 하면

"부동산 통해서 갱신하자"고 집주인이 하면요?

**거부해도 돼요.** 묵시적갱신 됐으면 이미 갱신된 거예요. 새로 뭘 할 필요 없어요.

## 지금 당장 할 일

**1. 현재 상태 확인**

- 묵시적갱신 됨 → 낼 필요 없음
- 아직 계약 중 → 만료 후 묵시적갱신 기다리기

**2. 부동산에서 연락 오면**

- "갱신 복비 내세요" → "묵시적갱신이라 중개 서비스 없어요"
- "계약서 정리해줄게요" → "안 해도 돼요. 기존 계약서로 유지돼요"

**3. 복비 피하려면**
- 부동산 안 거치고 직접 하기
- 묵시적갱신 상태 유지하기
- 새 계약서 쓸 때도 직접 쓰기
