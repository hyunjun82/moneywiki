계약 끝났는데 집주인한테 연락 안 했어요. 나도 아무 말 안 했고요. 그냥 계속 살고 있어요. 그런데 갑자기 부동산에서 연락이 왔어요.

"갱신됐으니까 복비 내세요."

안 내도 돼요. 부동산이 뭘 해준 게 없잖아요.

## 복비, 왜 안 내도 되는 거예요?

복비는 중개 서비스에 대한 대가예요.

[묵시적갱신](/w/묵시적갱신)이 뭔지 생각해보세요. 양쪽 다 아무 말 안 하면 자동으로 2년 연장되는 거예요. 부동산이 끼어들 틈이 없어요.

부동산이 집 소개해줬나요? 아니요. 계약서 써줬나요? 아니요. 협상 도와줬나요? 아니요. 서비스 안 받았으면 돈 낼 이유 없어요. [공인중개사법 시행규칙](https://www.law.go.kr/법령/공인중개사법시행규칙)에서도 중개수수료는 중개가 성립됐을 때만 받을 수 있다고 돼 있어요.

## 부동산에서 복비 달라고 하면요?

거부하세요.

뭐라고 하든 안 내도 돼요. 부동산에서 흔히 하는 말 있어요. "묵시적갱신도 갱신이니까 복비 내세요"라고 하면 아니라고 하세요. "처음 계약할 때 중개했으니까요"라고 하면 그건 그때 낸 거라고 하세요. "계약서 정리해줄게요"라고 하면 필요 없다고 하세요.

뭐라고 말하면 되냐고요?

"묵시적갱신은 법적으로 자동 연장된 거예요. 중개 서비스 받은 게 없어서 복비 낼 의무 없어요."

이렇게 말하면 돼요.

## 처음 계약 때 복비 냈는데요?

그건 별개예요.

처음 계약할 때 낸 복비는 그때 중개해준 대가예요. 묵시적갱신이랑 아무 상관 없어요. 처음 계약 때 복비는 이미 끝난 얘기고, 묵시적갱신은 새로운 중개가 없어서 복비도 없어요. 한 번 냈다고 갱신될 때마다 내는 거 아니에요.

## 언제 복비 내야 해요?

부동산 거치면 내야 해요.

묵시적갱신은 부동산 안 거쳐요. 그래서 복비 없어요. 근데 부동산 통해서 뭔가 하면 달라져요.

| 상황 | 복비 |
|------|------|
| 묵시적갱신 (자동 연장) | 없음 |
| 계약갱신청구권 직접 행사 | 없음 |
| 집주인이랑 직접 재계약 | 없음 |
| 부동산 통해서 재계약 | 있음 |
| 부동산 통해서 새 계약서 작성 | 있음 |
| 신규 계약 | 있음 |

핵심이에요. 부동산 안 거치면 복비 없어요.

## 복비 아끼려면요?

직접 하세요. 묵시적갱신 기다리면 0원, 집주인한테 직접 갱신 통보해도 0원, 계약서 직접 써도 0원이에요. 부동산 안 끼면 복비 0원이에요.

계약서만 도움받고 싶으면요? "협의는 다 됐는데 계약서만 작성해주세요"라고 하면 돼요. 복비 대신 계약서 작성 비용만 낼 수도 있어요. 보통 10~20만원 정도예요. 부동산마다 달라요.

## 집주인이 부동산 통하자고 하면요?

거부해도 돼요.

묵시적갱신 됐으면 이미 갱신된 거예요. 새로 뭘 할 필요 없어요.

"이미 묵시적갱신 됐어요. 부동산 안 거쳐도 돼요."

이렇게 말하면 돼요.

## 지금 당장 확인할 것

**내 상태 확인하기**

묵시적갱신 됐으면 복비 낼 필요 없어요. 아직 계약 중이면 만료 후 아무 말 안 하면 묵시적갱신 돼요.

**부동산에서 연락 오면**

"갱신 복비 내세요"라고 하면 "묵시적갱신이라 중개 서비스 없어요"라고 하세요. "계약서 정리해줄게요"라고 하면 "안 해도 돼요. 기존 계약서로 유지돼요"라고 하세요.

**복비 피하려면**

부동산 안 거치고 직접 하세요. 묵시적갱신 상태 유지하거나, 새 계약서 쓸 때도 직접 쓰면 복비 0원이에요.
