# 연말정산 교육비 세액공제

교육비 세액공제는 본인과 부양가족의 교육비에 대해 **15% 세금을 줄여주는** 제도입니다. [국세청 교육비 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 대상자별 한도

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따른 교육비 공제 한도입니다.

| 대상 | 연간 한도 | 공제율 |
|------|-----------|--------|
| **본인** | **한도 없음** | 15% |
| 취학 전 아동 | **300만원** | 15% |
| 초·중·고등학생 | **300만원** | 15% |
| **대학생** | **900만원** | 15% |
| 장애인 특수교육비 | **한도 없음** | 15% |

## 2. 공제 대상 교육비

### 2.1 본인 교육비

| 항목 | 공제 여부 |
|------|----------|
| 대학교 등록금 | ✅ 가능 |
| 대학원 등록금 | ✅ 가능 |
| 직업훈련비 | ✅ 가능 |
| 학점은행제 | ✅ 가능 |
| 독학학위 | ✅ 가능 |
| 사이버대학 | ✅ 가능 |

### 2.2 취학 전 아동 (유아)

| 항목 | 공제 여부 |
|------|----------|
| 유치원비 | ✅ 가능 |
| 어린이집 보육료 | ✅ 가능 |
| **학원비** | ✅ 가능 (태권도, 미술, 영어 등) |
| 체육시설 이용료 | ✅ 가능 |

### 2.3 초·중·고등학생

| 항목 | 공제 여부 |
|------|----------|
| 수업료·입학금 | ✅ 가능 |
| 급식비 | ✅ 가능 |
| 교과서 대금 | ✅ 가능 |
| 방과후학교 수업료 | ✅ 가능 |
| 교복 구입비 | ✅ 가능 (50만원 한도) |
| **학원비** | ❌ 불가 |

### 2.4 대학생

| 항목 | 공제 여부 |
|------|----------|
| 등록금 | ✅ 가능 |
| 입학금 | ✅ 가능 |
| 기성회비 | ✅ 가능 |
| 기숙사비 | ❌ 불가 |

## 3. 공제 불가 항목

| 항목 | 사유 |
|------|------|
| 초중고 학원비 | 사교육비 제외 |
| 대학 기숙사비 | 교육비 아님 |
| 해외유학 어학연수 | 정규 학위과정 아님 |
| 학자금 대출 상환액 | 별도 공제 (학자금대출 이자) |
| 수익자 부담금 일부 | 현장학습비 등 제외 |

## 4. 교복 구입비 공제

### 4.1 공제 조건

| 항목 | 내용 |
|------|------|
| 대상 | 중학생, 고등학생 |
| 한도 | **1인당 연간 50만원** |
| 항목 | 교복, 체육복 |

### 4.2 증빙

교복 구입 영수증 필요 (간소화 서비스 조회)

## 5. 공제 계산 예시

### 5.1 본인 대학원

**본인 대학원 등록금 800만원 지출**

- 공제 대상: 800만원 (한도 없음)
- 세액공제: 800만원 × 15% = **120만원**

### 5.2 자녀 교육비

**유치원생 자녀 1명, 고등학생 자녀 1명**

| 자녀 | 지출액 | 공제 대상 | 세액공제 |
|------|--------|----------|----------|
| 유치원 | 400만원 | 300만원 | 45만원 |
| 고등학교 | 200만원 | 200만원 | 30만원 |
| **합계** | 600만원 | 500만원 | **75만원** |

## 6. 부양가족 요건

### 6.1 기본공제 대상자

교육비 공제를 받으려면 해당 부양가족이 **기본공제 대상자**여야 함

| 요건 | 내용 |
|------|------|
| 나이 | 제한 없음 (교육비 특례) |
| 소득 | 연간 소득금액 100만원 이하 |
| 동거 | 별거해도 가능 (생계 같이) |

### 6.2 예외: 소득 있는 대학생 자녀

- 자녀 소득이 100만원 초과 → 기본공제 대상 제외
- 교육비 공제도 **불가**

## 7. 증빙서류

### 7.1 연말정산 간소화

대부분의 교육비는 **간소화 서비스에서 자동 조회**

### 7.2 별도 제출 필요

| 항목 | 제출 서류 |
|------|----------|
| 교복 구입비 | 구입 영수증 |
| 해외 교육기관 | 재학증명서, 납입증명서 |
| 장애인 특수교육비 | 특수교육비 납입증명서 |

## 8. 맞벌이 부부 공제

### 8.1 원칙

- 자녀를 기본공제 받는 배우자가 교육비 공제
- 부부가 나눠서 공제 불가

### 8.2 유리한 선택

- 소득이 높은 배우자가 공제 → 절세 효과 큼
- 단, 기본공제 대상자 등록 필요
