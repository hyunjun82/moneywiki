# 중소기업 취업자 소득세 감면

중소기업에 취업한 일정 대상자는 소득세를 감면받을 수 있습니다. 대상과 감면율을 정리합니다.

## 제도 개요

### 1.1 목적

| 목적 | 내용 |
|------|------|
| 중소기업 인력난 해소 | 취업 유인 제공 |
| 청년 고용 촉진 | 청년 취업 지원 |
| 취약계층 지원 | 경단녀, 장애인 등 |

### 1.2 법적 근거

| 항목 | 내용 |
|------|------|
| 근거 법령 | 조세특례제한법 제30조 |
| 시행 | 2012년~ (계속 연장) |

## 감면 대상자

### 2.1 대상자별 감면율

| 대상 | 감면율 | 기간 |
|------|--------|------|
| 청년 (15~34세) | **90%** | **5년** |
| 60세 이상 | **70%** | 3년 |
| 장애인 | **70%** | 3년 |
| 경력단절여성 | **70%** | 3년 |

### 2.2 청년 연령 기준

| 항목 | 기준 |
|------|------|
| 연령 | 15세 이상 ~ 34세 이하 |
| 기준일 | 취업 시점 |
| 군복무 | 최대 6년 차감 (40세까지) |

### 2.3 경력단절여성 요건

| 요건 | 내용 |
|------|------|
| 경력단절 사유 | 임신, 출산, 육아, 결혼 |
| 단절 기간 | 2년 이상 ~ 15년 이내 |
| 이전 취업 | 1년 이상 근무 경력 |

## 감면 요건

### 3.1 중소기업 요건

| 요건 | 내용 |
|------|------|
| 기업 규모 | 중소기업기본법상 중소기업 |
| 상시 근로자 | 해당 없음 (모든 중소기업) |

### 3.2 업종 제한

| 제외 업종 | 해당 여부 |
|----------|----------|
| 전문/과학/기술 서비스업 (일부) |  제외 |
| 금융/보험업 |  제외 |
| 부동산업 |  제외 |
| 유흥주점업 |  제외 |

### 3.3 고용 형태

| 형태 | 감면 |
|------|------|
| 정규직 |  |
| 계약직 |  |
| 파트타임 |  |
| 일용직 |  |

## 감면 한도

### 4.1 연간 한도

| 대상 | 연간 한도 |
|------|----------|
| 청년 (90%) | **200만원** |
| 기타 (70%) | **150만원** |

### 4.2 감면 기간

| 대상 | 기간 |
|------|------|
| 청년 | 취업일로부터 **5년** |
| 60세 이상, 장애인, 경단녀 | 취업일로부터 **3년** |

### 4.3 적용 기한

| 항목 | 기한 |
|------|------|
| 현행 적용 기한 | 2026년 12월 31일까지 취업 |
| 연장 가능성 | 매년 검토 |

## 감면액 계산

### 5.1 계산 방법

```
감면세액 = 근로소득 산출세액 × 감면율 (90% 또는 70%)
단, 연간 한도 내
```

### 5.2 계산 예시 (청년)

**연봉 4,000만원 청년**

| 항목 | 금액 |
|------|------|
| 산출세액 (예시) | 300만원 |
| 감면율 | 90% |
| 감면 가능액 | 270만원 |
| 한도 적용 | **200만원** |
| **실제 감면액** | **200만원** |

### 5.3 월 환산 효과

| 연봉 | 연 감면액 | 월 효과 |
|------|----------|---------|
| 3,000만원 | 약 100만원 | 약 8만원 |
| 4,000만원 | 약 150만원 | 약 12만원 |
| 5,000만원 | 약 200만원 | 약 17만원 |

## 신청 방법

### 6.1 신청 절차

| 단계 | 내용 |
|------|------|
| 1 | 감면 대상 확인 |
| 2 | 감면신청서 작성 |
| 3 | 회사에 제출 |
| 4 | 회사가 원천징수 시 적용 |

### 6.2 필요 서류

| 서류 | 용도 |
|------|------|
| 중소기업 취업자 소득세 감면신청서 | 기본 서류 |
| 주민등록등본 | 연령 확인 |
| 병적증명서 | 군복무 확인 (해당 시) |
| 장애인증명서 | 장애인 (해당 시) |
| 경력단절 증빙 | 경단녀 (해당 시) |

### 6.3 신청 시기

| 시기 | 권장 |
|------|------|
| 입사 시 | 즉시 신청 권장 |
| 연말정산 | 늦어도 연말정산 전 |
| 소급 적용 | 경정청구로 가능 (5년 내) |

## 감면 적용

### 7.1 원천징수 적용

| 방법 | 내용 |
|------|------|
| 매월 | 간이세액에서 감면 적용 |
| 연말정산 | 최종 정산 |

### 7.2 연말정산 반영

| 항목 | 내용 |
|------|------|
| 세액감면란 | 감면액 기재 |
| 코드 | 중소기업취업자 감면 |

## 주의사항

### 8.1 중복 감면 불가

| 상황 | 적용 |
|------|------|
| 다른 감면과 중복 | 선택 적용 |
| 유리한 감면 선택 | 권장 |

### 8.2 이직 시

| 상황 | 처리 |
|------|------|
| 중소기업 → 중소기업 | 계속 적용 (잔여 기간) |
| 중소기업 → 대기업 | 감면 종료 |
| 대기업 → 중소기업 | 적용 불가 |

### 8.3 퇴사 후 재취업

| 상황 | 적용 |
|------|------|
| 감면 기간 내 재취업 | 잔여 기간만 적용 |
| 감면 기간 후 재취업 | 적용 불가 |

## 소급 신청

### 9.1 과거분 환급

| 방법 | 내용 |
|------|------|
| 경정청구 | 5년 이내 |
| 장소 | 홈택스 또는 세무서 |

### 9.2 필요 서류

| 서류 | 용도 |
|------|------|
| 경정청구서 | 기본 서류 |
| 감면신청서 | 감면 요건 증명 |
| 원천징수영수증 | 기납부세액 확인 |
