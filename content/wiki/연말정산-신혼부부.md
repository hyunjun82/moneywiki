# 연말정산 신혼부부

신혼부부 연말정산 전략을 정리합니다.

## 1. 혼인 첫해 주의사항

| 항목 | 내용 |
|------|------|
| 기준일 | 12월 31일 |
| 배우자 공제 | 혼인신고 완료 시 가능 |
| 부양가족 | 양가 부모님 분배 가능 |

## 2. 배우자 공제

| 조건 | 공제 |
|------|------|
| 맞벌이 | 각자 본인만 |
| 외벌이 | 배우자 150만원 |
| 소득 100만원 이하 | 공제 가능 |

## 3. 몰아주기 전략

| 항목 | 전략 |
|------|------|
| 의료비 | 소득 낮은 쪽 |
| 신용카드 | 소득 높은 쪽 |
| 보험료 | 소득 높은 쪽 |
| 교육비 | 소득 높은 쪽 |

## 4. 주택자금 공제

| 항목 | 전략 |
|------|------|
| 주택청약 | 무주택 세대주 |
| 월세 | 무주택 세대주 |
| 주담대 이자 | 1주택 또는 무주택 |

## 5. 첫해 체크리스트

| 항목 | 확인 |
|------|------|
| 혼인신고 | ✅ 완료 |
| 주소 통합 | ✅ 확인 |
| 부양가족 분배 | ✅ 협의 |
