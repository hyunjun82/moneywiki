"실업급여 받던 분이 돌아가셨는데, 남은 급여는요?" **유족이 미지급 실업급여를 받을 수 있어요.**


## 미지급 실업급여란?

### 사망한 분의 남은 실업급여

실업급여를 받던 분이 **사망**하면, 아직 받지 못한 급여가 남을 수 있어요.

이 남은 급여를 **유족이 대신 받는 제도**가 미지급 실업급여예요.


### 미지급 실업급여 수급 대상

아래 경우에 해당하면 유족이 미지급 실업급여를 신청할 수 있어요.

1. **수급자격자가 사망**한 경우
2. 사망 전에 **지급받지 못한 구직급여**가 있는 경우
3. **유족**이 있는 경우


## 미지급 실업급여 수급 순위

### 유족의 범위와 순위

미지급 실업급여를 받을 수 있는 **유족의 순위**예요.

| 순위 | 유족 | 비고 |
|------|------|------|
| 1순위 | **배우자** | 사실혼 포함 |
| 2순위 | **자녀** | - |
| 3순위 | **부모** | - |
| 4순위 | **손자녀** | - |
| 5순위 | **조부모** | - |
| 6순위 | **형제자매** | - |


### 생계를 같이한 유족 우선

같은 순위 안에서는 **사망 당시 생계를 같이한 유족**이 우선해요.

예: 부모가 2명이면, 사망자와 함께 살던 부모가 우선


### 같은 순위가 여러 명이면?

같은 순위의 유족이 **여러 명**이면 **똑같이 나눠서** 받아요.

예: 자녀 2명이면 → 50%씩 나눠 받아요


## 미지급 실업급여 신청 방법

### 신청 절차

**1단계**: 관할 고용센터 방문 또는 온라인

**2단계**: 미지급 급여 청구서 작성

**3단계**: 사망 증빙서류 제출

**4단계**: 유족 관계 증빙서류 제출


### 필요 서류

| 서류 | 비고 |
|------|------|
| 미지급 급여 청구서 | 고용센터 비치 또는 온라인 |
| 사망진단서 또는 사망확인서 | 사망 증빙 |
| 가족관계증명서 | 유족 관계 증빙 |
| 주민등록등본 | 생계 같이 증빙 (필요시) |
| 통장 사본 | 급여 수령 계좌 |


### 신청 장소

**관할 고용센터**에 직접 방문하거나 **고용24**(ei.go.kr)에서 온라인으로 신청해요.

고용센터 전화: **1350**


## 미지급 실업급여 수급권의 소멸시효

### 3년 안에 신청해야

미지급 실업급여 수급권의 **소멸시효는 3년**이에요.

사망일로부터 **3년이 지나면** 청구할 수 없어요.


### 예시

2026년 1월 15일 사망 → **2029년 1월 14일까지** 신청 가능

늦지 않게 신청하세요!


## 미지급 실업급여 금액

### 사망 전까지 받을 금액

미지급 실업급여는 **사망일 전날까지** 받을 수 있었던 급여예요.

사망일 이후의 실업급여는 지급되지 않아요.


### 계산 예시

소정급여일수 150일 중 100일 수급 후 사망

→ **남은 50일분**이 미지급 실업급여

→ 50일 × 68,100원(상한액) = 약 **340만원**


## 주의사항

### 실업인정을 받아야

사망 전에 **실업인정을 받은 기간**에 대해서만 미지급 급여가 발생해요.

실업인정을 받지 않은 기간은 급여가 없어요.


### 사망일 이후는 불가

사망일 **이후**의 구직급여는 지급되지 않아요.

소정급여일수가 남아있어도 사망일까지만 계산돼요.
