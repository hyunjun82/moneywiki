# 인지세

인지세는 각종 계약서와 증서 작성 시 부과되는 국세입니다. 과세 대상과 세율을 정리합니다.

## 인지세 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 계약서·증서 작성 시 부과 |
| 세목 | **국세** |
| 납부 방법 | 수입인지 첩부 |

### 1.2 과세 근거

| 항목 | 내용 |
|------|------|
| 법률 | 인지세법 |
| 과세 대상 | 법정 문서 |

## 인지세액

### 3.1 부동산 매매계약서

| 기재금액 | 인지세 |
|----------|--------|
| 1억원 이하 | **비과세** |
| 1억 초과 ~ 10억 이하 | **15만원** |
| 10억 초과 | **35만원** |

### 3.2 금전소비대차계약서

| 기재금액 | 인지세 |
|----------|--------|
| 1천만원 이하 | 비과세 |
| 1천만 초과 ~ 3천만 이하 | **2만원** |
| 3천만 초과 ~ 5천만 이하 | **4만원** |
| 5천만 초과 ~ 1억 이하 | **7만원** |
| 1억 초과 ~ 10억 이하 | **15만원** |
| 10억 초과 | **35만원** |

### 3.3 공사도급계약서

| 기재금액 | 인지세 |
|----------|--------|
| 1천만원 이하 | 비과세 |
| 1천만 초과 ~ 3천만 이하 | 2만원 |
| 3천만 초과 ~ 5천만 이하 | 4만원 |
| 5천만 초과 ~ 1억 이하 | 7만원 |
| 1억 초과 ~ 10억 이하 | 15만원 |
| 10억 초과 | 35만원 |

## 납부 방법

### 4.1 수입인지 종류

| 종류 | 용도 |
|------|------|
| 종이 수입인지 | 문서에 직접 부착 |
| 전자수입인지 | 온라인 발급·출력 |

### 4.2 구입처

| 방법 | 장소 |
|------|------|
| 종이 수입인지 | 은행, 우체국, 편의점 |
| 전자수입인지 | e-revenuestamp.or.kr |

### 4.3 부착 방법

| 절차 | 내용 |
|------|------|
| 1 | 수입인지 구입 |
| 2 | 계약서에 부착 |
| 3 | 소인 (도장 또는 서명) |

### 4.4 전자수입인지

| 장점 | 내용 |
|------|------|
| 편리성 | 온라인 구매·출력 |
| 위변조 방지 | QR코드 검증 |
| 환불 가능 | 미사용 시 환불 |
