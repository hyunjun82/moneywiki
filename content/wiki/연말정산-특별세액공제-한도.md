# 특별세액공제 한도

## 기본 개념
보험료, 의료비, 교육비, 기부금 등에 대한 세액공제의 개별 한도를 정리한 문서입니다.

## 특별세액공제 항목별 한도

### 1. 보험료 세액공제
| 구분 | 공제율 | 연간 한도 |
|------|-------|---------|
| 보장성 보험료 | 12% | **100만원** |
| 장애인 전용 보험료 | 15% | **100만원** |

### 2. 의료비 세액공제
| 구분 | 공제율 | 연간 한도 |
|------|-------|---------|
| 본인·65세 이상·장애인 | 15% | **한도 없음** |
| 그 외 부양가족 | 15% | **700만원** |
| 난임시술비 | 20% | **한도 없음** |

**최저사용액**: 총급여의 3% 초과분

### 3. 교육비 세액공제
| 구분 | 공제율 | 연간 한도 |
|------|-------|---------|
| 본인 (대학원 포함) | 15% | **한도 없음** |
| 장애인 특수교육비 | 15% | **한도 없음** |
| 취학전 아동 | 15% | **300만원** |
| 초·중·고등학생 | 15% | **300만원** |
| 대학생 | 15% | **900만원** |

### 4. 기부금 세액공제
| 구분 | 공제율 | 한도 |
|------|-------|------|
| 정치자금 (10만원 이하) | 100/110 | 소득금액 100% |
| 정치자금 (10만원 초과) | 15% | 소득금액 100% |
| 법정기부금 | 15% | 소득금액 100% |
| 지정기부금 (1천만원 이하) | 15% | 소득금액 30% |
| 지정기부금 (1천만원 초과) | 30% | 소득금액 30% |
| 종교단체 기부금 | 15%/30% | 소득금액 10% |

### 5. 주택자금 소득공제 (특별세액공제 아님)
| 구분 | 공제율 | 연간 한도 |
|------|-------|---------|
| 주택청약종합저축 | 40% | **240만원** (300만원) |
| 주택임차차입금 원리금 | 40% | **300만원** |
| 장기주택저당차입금 이자 | - | **300~1,800만원** |

## 종합 한도

### 표준세액공제 vs 특별세액공제
- **표준세액공제**: 13만원 (무조건)
- **특별세액공제**: 항목별 합계
- 둘 중 **유리한 것 선택**

### 선택 기준
```
특별세액공제 합계 > 13만원
→ 특별세액공제 선택

특별세액공제 합계 < 13만원
→ 표준세액공제 선택
```

## 실무 계산 사례

### 사례 1: 기본 케이스
```
보험료: 100만원 → 공제 12만원 (한도)
의료비: 500만원 → 공제 75만원
교육비: 600만원 (대학생 2명) → 공제 90만원
기부금: 100만원 → 공제 15만원
총 공제: 192만원
```

### 사례 2: 한도 초과
```
보험료: 150만원 → 공제 12만원 (한도 100만원)
의료비: 1,000만원 → 공제 105만원 (한도 700만원)
교육비: 1,200만원 → 공제 135만원 (한도 900만원)
총 공제: 252만원
```

### 사례 3: 최저사용액 미달
```
총급여: 5,000만원
의료비: 100만원
최저사용액: 150만원 (5,000×3%)
→ 공제액: 0원 (최저사용액 미달)
```

## 항목별 주의사항

### 보험료
- 본인·배우자·부양가족 합산 100만원
- 장애인 전용은 별도 100만원

### 의료비
- 총급여의 3% 초과분만 공제
- 본인·노인·장애인은 한도 없음
- 기타 가족은 700만원 한도

### 교육비
- 학생 1인당 한도 적용
- 본인 교육비는 한도 없음
- 직계존속 교육비는 공제 불가

### 기부금
- 소득금액 대비 비율 한도
- 이월공제 가능 (10년)
- 종교단체 10% 별도 한도

## 맞벌이 부부 전략

### 원칙
- 각자 지출한 것만 공제
- 의료비·교육비는 지출자 기준

### 전략
```
소득 높은 배우자: 교육비, 기부금 집중
소득 낮은 배우자: 의료비 (3% 기준 낮음)
```

### 예시
```
남편 총급여: 8,000만원 (3% = 240만원)
아내 총급여: 3,000만원 (3% = 90만원)
의료비 500만원
→ 아내가 공제받는 것이 유리
  아내: (500-90) × 15% = 61.5만원
  남편: (500-240) × 15% = 39만원
```
