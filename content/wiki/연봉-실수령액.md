# 연봉 실수령액

연봉에서 공제 항목을 제외하면 실수령액이 됩니다. 연봉별 실수령액과 계산 방법을 정리합니다.

## 1. 실수령액 개요

### 1.1 공제 항목

| 공제 항목 | 부담률 |
|----------|--------|
| 국민연금 | 4.75% |
| 건강보험 | 3.595% |
| 장기요양보험 | 건보의 13.14% |
| 고용보험 | 0.9% |
| 소득세 | 과세표준별 |
| 지방소득세 | 소득세의 10% |

### 1.2 계산 공식

```
실수령액 = 월급여 - 4대보험 - 소득세 - 지방소득세
```

## 6. 실수령액 영향 요인
