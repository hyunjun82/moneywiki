# 연말정산 세액공제 순서

세액공제 적용 순서를 정리합니다.

## 1. 적용 순서

| 순서 | 세액공제 |
|------|----------|
| 1 | 근로소득세액공제 |
| 2 | 자녀세액공제 |
| 3 | 연금계좌세액공제 |
| 4 | 특별세액공제 (보험, 의료, 교육, 기부) |
| 5 | 월세세액공제 |
| 6 | 세액감면 |

## 2. 특별세액공제 순서

| 순서 | 항목 |
|------|------|
| 1 | 보험료 |
| 2 | 의료비 |
| 3 | 교육비 |
| 4 | 기부금 |

## 3. 결정세액 0원

| 상황 | 결과 |
|------|------|
| 결정세액 0원 | 추가 공제 불가 |
| 세액공제 남음 | 환급 없음 |

## 4. 전략

| 전략 | 내용 |
|------|------|
| 소득공제 우선 | 과세표준 낮추기 |
| 세액공제 | 결정세액 0원까지 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 이월 | 일부 기부금만 이월 가능 |
| 소멸 | 대부분 당해연도 소멸 |
