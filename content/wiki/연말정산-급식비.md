# 연말정산 급식비

급식비 교육비 공제를 정리합니다.

## 1. 공제 구분

| 대상 | 급식비 공제 |
|------|-------------|
| 어린이집 | ✅ |
| 유치원 | ✅ |
| 초등학교 | ❌ |
| 중학교 | ❌ |
| 고등학교 | ❌ |

## 2. 취학 전 아동 공제

| 항목 | 내용 |
|------|------|
| 한도 | 연 300만원 (교육비 전체) |
| 공제율 | 15% |
| 포함 | 급식비, 차량운행비 등 |

## 3. 초중고 급식비

| 구분 | 공제 |
|------|------|
| 무상급식 | - |
| 유상급식 | ❌ 교육비 불가 |
| 카드결제 | ✅ 신용카드 공제 |

## 4. 신용카드 공제

| 항목 | 내용 |
|------|------|
| 급식비 카드결제 | 신용카드 공제 가능 |
| 공제율 | 15~30% |
