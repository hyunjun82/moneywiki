# 연말정산 종교단체 기부금

종교단체 기부금은 지정기부금으로 **세액공제**를 받을 수 있습니다. [국세청 기부금 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6597&cntntsId=7876)에 따른 조건을 정리합니다.

## 1. 공제 대상

### 1.1 종교단체 범위

| 종교 | 대상 |
|------|------|
| 기독교 | 교회, 성당 |
| 불교 | 절, 사찰 |
| 기타 | 종교법인으로 등록된 단체 |

### 1.2 공제 대상 기부금

| 항목 | 공제 여부 |
|------|----------|
| 십일조, 헌금 | ✅ 가능 |
| 봉헌금, 보시금 | ✅ 가능 |
| 건축헌금 | ✅ 가능 |
| 특별헌금 | ✅ 가능 |

## 2. 공제율과 한도

### 2.1 공제율

| 기부금액 | 공제율 |
|----------|--------|
| 1천만원 이하 | **15%** |
| 1천만원 초과 | **30%** |

### 2.2 한도

| 구분 | 한도 |
|------|------|
| 종교단체만 | 근로소득금액의 **10%** |
| + 다른 지정기부금 | 합산 **30%** |

### 2.3 계산 예시

**근로소득금액 4,000만원, 종교기부금 500만원**

1. 한도: 4,000만원 × 10% = 400만원
2. 공제 대상: 400만원 (한도)
3. 세액공제: 400만원 × 15% = **60만원**

※ 100만원은 한도 초과로 이월공제

## 3. 기부금영수증

### 3.1 발급 방법

| 방법 | 내용 |
|------|------|
| 종교단체 직접 발급 | 연말에 일괄 발급 |
| 연말정산 간소화 | 일부 단체 조회 가능 |

### 3.2 필수 기재 사항

- 기부자 성명, 주민등록번호
- 기부금액, 기부일자
- 기부금 유형 (지정기부금)
- 종교단체명, 사업자번호
- 고유번호 (종교단체 고유번호)

### 3.3 간소화 미제공 시

종교단체에서 직접 기부금영수증 발급받아 회사에 제출

## 4. 다른 기부금과 합산

### 4.1 지정기부금 합산

| 기부금 | 한도 |
|--------|------|
| 종교단체만 | 10% |
| + 사회복지단체 | **합산 30%** |
| + 교육기관 | **합산 30%** |

### 4.2 계산 예시

**근로소득금액 4,000만원**
- 종교단체: 500만원
- 사회복지단체: 200만원

| 계산 | 금액 |
|------|------|
| 종교단체 한도 | 400만원 (10%) |
| 사회복지 한도 | 800만원 (30% - 종교) |
| 공제 대상 | 600만원 |

## 5. 이월공제

### 5.1 한도 초과 시

당해 한도를 초과한 기부금은 **10년간 이월공제** 가능

### 5.2 이월 순서

1. 먼저 발생한 기부금부터 공제
2. 이월 기부금 → 당해 기부금 순서

## 6. 부양가족 기부금

### 6.1 공제 가능

기본공제 대상 부양가족의 종교기부금도 공제 가능

| 대상 | 공제 |
|------|------|
| 배우자 기부금 | ✅ |
| 부모님 기부금 | ✅ |
| 자녀 기부금 | ✅ |

### 6.2 요건

- 부양가족이 **기본공제 대상자**여야 함
- 소득금액 100만원 이하

## 7. 주의사항

### 7.1 허위 기부금영수증

- 실제 기부하지 않은 금액 기재 → **가산세 40%**
- 세액공제 환수 + 가산세

### 7.2 사업자등록 확인

- 종교단체가 **고유번호** 또는 **사업자번호** 보유 필수
- 미등록 단체 기부금 → 공제 불가

### 7.3 영수증 보관

- 기부금영수증 **5년간** 보관 권장
- 세무조사 대비
