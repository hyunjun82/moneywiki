# 연말정산 환급금 계산

연말정산 환급금은 **이미 납부한 세금**과 **실제 내야 할 세금**의 차이입니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 계산 방법을 정리합니다.

## 1. 환급금 계산 공식

### 1.1 기본 공식

**환급금 = 기납부세액 - 결정세액**

| 결과 | 의미 |
|------|------|
| **양수(+)** | **환급** (돌려받음) |
| **음수(-)** | **추가 납부** (더 냄) |

### 1.2 용어 설명

| 용어 | 의미 |
|------|------|
| 기납부세액 | 매월 급여에서 **이미 납부한** 세금 합계 |
| 결정세액 | 연말정산으로 계산된 **실제 세금** |

## 2. 세금 계산 과정

### 2.1 계산 순서

```
① 총급여
② - 근로소득공제
③ = 근로소득금액
④ - 소득공제
⑤ = 과세표준
⑥ × 세율
⑦ = 산출세액
⑧ - 세액공제·감면
⑨ = 결정세액
⑩ 기납부세액 - 결정세액 = 환급/납부
```

### 2.2 예시

| 항목 | 금액 |
|------|------|
| 총급여 | 5,000만원 |
| 근로소득금액 | 4,000만원 |
| 소득공제 | 1,000만원 |
| 과세표준 | 3,000만원 |
| 산출세액 | 400만원 |
| 세액공제 | 150만원 |
| **결정세액** | **250만원** |
| 기납부세액 | 300만원 |
| **환급금** | **50만원** |

## 3. 기납부세액 확인

### 3.1 급여명세서

매월 급여명세서의 **소득세** 항목 확인

| 월 | 소득세 |
|---|--------|
| 1월 | 25만원 |
| 2월 | 25만원 |
| ... | ... |
| 12월 | 25만원 |
| **연간 합계** | **300만원** |

### 3.2 원천징수영수증

연말정산 후 발급되는 **근로소득 원천징수영수증**에서 확인

## 4. 환급/납부 사례

### 4.1 환급 발생 (돌려받음)

| 상황 | 원인 |
|------|------|
| 부양가족 증가 | 인적공제 증가 |
| 의료비·교육비 지출 | 세액공제 증가 |
| 신용카드 사용 증가 | 소득공제 증가 |
| 연금저축 납입 | 세액공제 증가 |

### 4.2 추가 납부 발생 (더 냄)

| 상황 | 원인 |
|------|------|
| 부양가족 감소 | 인적공제 감소 |
| 급여 인상 | 세율 구간 상승 |
| 상여금·성과급 | 기납부세액보다 세금 많음 |
| 퇴직 후 이직 | 합산소득 증가 |

## 5. 환급금 지급

### 5.1 지급 시기

| 시기 | 내용 |
|------|------|
| 2~3월 | 대부분 회사에서 급여와 함께 지급 |
| 4~5월 | 일부 회사는 별도 지급 |

### 5.2 지급 방법

| 방법 | 내용 |
|------|------|
| 급여 합산 | 2~3월 급여에 포함 |
| 별도 지급 | 환급금만 따로 입금 |

## 6. 추가 납부 시

### 6.1 납부 방법

| 방법 | 내용 |
|------|------|
| 급여 차감 | 2~3월 급여에서 공제 |
| 분할 납부 | 회사와 협의하여 분할 |

### 6.2 분할 납부

추가 납부세액이 **10만원 초과** 시 **3개월 분할** 가능

## 7. 환급금 예상 계산

### 7.1 간이 계산

국세청 [연말정산 미리보기](https://www.hometax.go.kr)에서 예상 환급금 조회

### 7.2 확인 시기

| 시기 | 내용 |
|------|------|
| 10~11월 | 미리보기 서비스 오픈 |
| 1월 | 간소화 자료 반영 |
| 2월 | 최종 확정 |

## 8. 환급금 늘리는 방법

### 8.1 공제 항목 확인

| 항목 | 내용 |
|------|------|
| 부양가족 | 누락 여부 확인 |
| 의료비 | 간소화 누락 자료 추가 |
| 교육비 | 학원비, 교복비 등 |
| 기부금 | 종교단체, 공익단체 |

### 8.2 절세 상품

| 상품 | 효과 |
|------|------|
| 연금저축 | 최대 135만원 환급 |
| IRP | 추가 108만원 환급 |
| 주택청약 | 소득공제 효과 |

## 9. 주의사항

### 9.1 허위 공제

- 허위 부양가족, 허위 기부금 → **가산세 부과**
- 과다 공제 발견 시 **환급금 반환 + 가산세**

### 9.2 경정청구

- 공제 누락 발견 시 → **5년 내 경정청구** 가능
- 추가 환급 가능
