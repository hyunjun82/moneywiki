# 해외주식 세금

해외주식 투자 시 발생하는 양도소득세와 배당세를 정리합니다.

## 1. 해외주식 세금 개요

### 1.1 과세 대상

| 소득 | 세금 |
|------|------|
| 양도차익 | 양도소득세 |
| 배당금 | 배당소득세 |

### 1.2 국내주식과 차이

| 구분 | 국내주식 | 해외주식 |
|------|----------|----------|
| 양도세 | 대주주만 | **전원 과세** |
| 배당세 | 15.4% | 현지세 + 국내세 |

## 2. 양도소득세

### 2.1 과세 기준

| 항목 | 내용 |
|------|------|
| 대상 | 매도차익 전액 |
| 공제 | 연 **250만원** |
| 세율 | **22%** (20% + 지방세 2%) |

### 2.2 계산 공식

```
양도소득세 = (양도차익 - 250만원) × 22%
```

### 2.3 계산 예시

**양도차익 1,000만원**

| 항목 | 금액 |
|------|------|
| 양도차익 | 1,000만원 |
| (-) 기본공제 | 250만원 |
| 과세표준 | 750만원 |
| 세율 | 22% |
| **양도소득세** | **165만원** |

### 2.4 손익통산

| 항목 | 통산 |
|------|------|
| 같은 해 해외주식 손실 | ✅ 통산 가능 |
| 국내주식 손실 | ❌ 통산 불가 |
| 다음해 이월 | ❌ 불가 |

### 2.5 손익통산 예시

**A주식 +500만원, B주식 -200만원**

| 항목 | 금액 |
|------|------|
| A주식 이익 | +500만원 |
| B주식 손실 | -200만원 |
| 순이익 | 300만원 |
| (-) 기본공제 | 250만원 |
| 과세표준 | 50만원 |
| **양도소득세** | **11만원** |

## 3. 배당소득세

### 3.1 과세 방식

| 단계 | 세금 |
|------|------|
| 현지 원천징수 | 국가별 상이 |
| 국내 과세 | 종합과세 또는 분리과세 |

### 3.2 주요국 원천징수율

| 국가 | 원천징수율 |
|------|----------|
| 미국 | **15%** |
| 중국 | 10% |
| 일본 | 15% |
| 홍콩 | 0% |

### 3.3 국내 과세

| 금융소득 | 과세 방식 |
|----------|----------|
| 2,000만원 이하 | 분리과세 (15.4%) |
| 2,000만원 초과 | 종합과세 |

### 3.4 미국 배당 예시

**배당금 $1,000 (환율 1,300원)**

| 항목 | 금액 |
|------|------|
| 배당금 | 130만원 |
| 미국 원천징수 (15%) | -19.5만원 |
| 실수령 | 110.5만원 |
| 국내 추가세 | 외납세액공제 적용 |

### 3.5 외국납부세액공제

| 항목 | 내용 |
|------|------|
| 공제 | 외국에서 낸 세금 |
| 한도 | 국내 세액 범위 |
| 효과 | 이중과세 방지 |

## 4. 환율 적용

### 4.1 환율 기준

| 시점 | 적용 환율 |
|------|----------|
| 매수 시 | 매수일 환율 |
| 매도 시 | 매도일 환율 |

### 4.2 환차익 과세

| 항목 | 과세 |
|------|------|
| 주식 환차익 | 양도소득에 포함 |
| 예금 환차익 | 비과세 |

### 4.3 환율 영향 예시

**$10,000 매수 (환율 1,200원) → $10,000 매도 (환율 1,400원)**

| 항목 | 금액 |
|------|------|
| 매수금액 | 1,200만원 |
| 매도금액 | 1,400만원 |
| 환차익 | 200만원 |
| **과세 대상** | **200만원** |

## 5. 신고 방법

### 5.1 신고 기간

| 항목 | 기간 |
|------|------|
| 신고 | **5월 1일 ~ 31일** |
| 대상 | 전년도 양도차익 |

### 5.2 신고 채널

| 방법 | 내용 |
|------|------|
| 홈택스 | 전자신고 |
| 세무서 | 방문 신고 |
| 증권사 | 대행 서비스 (일부) |

### 5.3 필요 서류

| 서류 | 용도 |
|------|------|
| 거래내역서 | 매수·매도 확인 |
| 환율 자료 | 원화 환산 |
| 수수료 내역 | 필요경비 |

### 5.4 미신고 시

| 가산세 | 세율 |
|--------|------|
| 무신고 | 20% |
| 과소신고 | 10% |
| 납부지연 | 1일 0.022% |

## 6. 절세 전략

### 6.1 기본공제 활용

| 전략 | 내용 |
|------|------|
| 매년 250만원 활용 | 연말 손실 확정 매도 |
| 부부 분산 | 각자 250만원 공제 |

### 6.2 손익통산

| 전략 | 내용 |
|------|------|
| 손실 확정 | 이익과 상계 |
| 연내 매도 | 같은 해에 정리 |

### 6.3 계좌 활용

| 계좌 | 효과 |
|------|------|
| ISA | 배당 비과세 (한도 내) |
| 연금저축 | 과세이연 |

### 6.4 연말 전략

| 시기 | 전략 |
|------|------|
| 12월 | 손실 종목 정리 |
| 이익 조절 | 250만원에 맞추기 |

## 7. 주요 투자국

### 7.1 미국

| 항목 | 내용 |
|------|------|
| 거래시간 | 한국시간 23:30~06:00 |
| 배당세 | 15% 원천징수 |
| 양도세 | 22% (한국) |

### 7.2 중국 (홍콩 상장)

| 항목 | 내용 |
|------|------|
| 배당세 | 10% 원천징수 |
| 양도세 | 22% (한국) |

### 7.3 일본

| 항목 | 내용 |
|------|------|
| 배당세 | 15% 원천징수 |
| 양도세 | 22% (한국) |

## 8. 주의사항

### 8.1 신고 필수

| 주의 | 내용 |
|------|------|
| 자진신고 | 증권사가 대신 안 해줌 |
| 기한 엄수 | 5월 31일까지 |
| 모든 계좌 | 합산 신고 |

### 8.2 기록 보관

| 보관 | 기간 |
|------|------|
| 거래내역 | 5년 이상 |
| 환율 기록 | 5년 이상 |

### 8.3 금융소득 관리

| 주의 | 내용 |
|------|------|
| 2,000만원 기준 | 배당+이자 합산 |
| 종합과세 | 건보료 영향 |
