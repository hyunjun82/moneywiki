# 국민기초생활보장

국민기초생활보장은 최저 생활을 보장하는 사회보장제도입니다. [국민기초생활보장법](https://www.law.go.kr/법령/국민기초생활보장법)에 따른 내용을 정리합니다.

## 1. 제도 개요

### 1.1 급여 종류

| 급여 | 목적 |
|------|------|
| 생계급여 | 일상 생활비 |
| 의료급여 | 의료비 지원 |
| 주거급여 | 주거비 지원 |
| 교육급여 | 교육비 지원 |

### 1.2 급여별 기준 (2026년)

| 급여 | 기준 중위소득 |
|------|-------------|
| 생계급여 | **32%** 이하 |
| 의료급여 | **40%** 이하 |
| 주거급여 | **48%** 이하 |
| 교육급여 | **50%** 이하 |

## 2. 기준 중위소득 (2026년)

### 2.1 가구원수별 중위소득

| 가구원 | 기준 중위소득 |
|--------|-------------|
| 1인 | 약 228만원 |
| 2인 | 약 377만원 |
| 3인 | 약 482만원 |
| 4인 | 약 584만원 |
| 5인 | 약 679만원 |

### 2.2 급여별 선정 기준 (4인 가구)

| 급여 | 소득인정액 |
|------|-----------|
| 생계급여 (32%) | 약 187만원 |
| 의료급여 (40%) | 약 234만원 |
| 주거급여 (48%) | 약 280만원 |
| 교육급여 (50%) | 약 292만원 |

## 3. 소득인정액 계산

### 3.1 계산 공식

```
소득인정액 = 소득평가액 + 재산의 소득환산액
```

### 3.2 소득평가액

| 소득 | 산정 |
|------|------|
| 근로소득 | 30% 공제 후 산입 |
| 사업소득 | 전액 산입 |
| 재산소득 | 전액 산입 |
| 공적이전소득 | 전액 산입 |

### 3.3 재산의 소득환산

| 재산 | 환산율 |
|------|--------|
| 일반재산 | 연 4.17% |
| 금융재산 | 연 6.26% |
| 자동차 | 연 100% |

## 4. 생계급여

### 4.1 지급액

```
생계급여 = 생계급여 선정기준액 - 소득인정액
```

### 4.2 지급 예시 (4인 가구)

| 소득인정액 | 생계급여 |
|-----------|---------|
| 100만원 | 약 87만원 |
| 150만원 | 약 37만원 |
| 0원 | 약 187만원 (최대) |

### 4.3 지급일

| 항목 | 내용 |
|------|------|
| 지급일 | 매월 **20일** |
| 지급 방법 | 계좌이체 |

## 5. 의료급여

### 5.1 의료급여 종류

| 종별 | 대상 |
|------|------|
| 1종 | 근로능력 없는 가구 |
| 2종 | 근로능력 있는 가구 |

### 5.2 본인부담금

| 종별 | 입원 | 외래 |
|------|------|------|
| 1종 | 면제 | 1,000원 |
| 2종 | 10% | 1,000원 |

### 5.3 혜택

| 혜택 | 내용 |
|------|------|
| 건강보험료 | 면제 |
| 본인부담 | 최소화 |

## 6. 주거급여

### 6.1 지급 방식

| 유형 | 지급 |
|------|------|
| 임차가구 | **임차료** 지원 |
| 자가가구 | **수선비** 지원 |

### 6.2 임차급여 (기준임대료)

| 가구원 | 서울 | 경기·인천 | 광역시 | 기타 |
|--------|------|----------|--------|------|
| 1인 | 34.1만원 | 26.8만원 | 21.6만원 | 17.8만원 |
| 2인 | 38.2만원 | 30.0만원 | 24.0만원 | 20.1만원 |
| 3인 | 45.5만원 | 35.8만원 | 28.7만원 | 23.9만원 |
| 4인 | 52.7만원 | 41.4만원 | 33.3만원 | 27.7만원 |

### 6.3 자가가구 수선비

| 수선 유형 | 금액 | 주기 |
|----------|------|------|
| 경보수 | 457만원 | 3년 |
| 중보수 | 849만원 | 5년 |
| 대보수 | 1,241만원 | 7년 |

## 7. 교육급여

### 7.1 지원 대상

| 대상 | 지원 |
|------|------|
| 초등학생 | ✅ |
| 중학생 | ✅ |
| 고등학생 | ✅ |

### 7.2 지원 내용 (2026년)

| 항목 | 초등 | 중등 | 고등 |

## 8. 신청 방법

### 8.1 신청 장소

| 방법 | 장소 |
|------|------|
| 방문 | 주민센터 |
| 온라인 | 복지로 (www.bokjiro.go.kr) |

### 8.2 필요 서류

| 서류 | 용도 |
|------|------|
| 신청서 | 기본 서류 |
| 신분증 | 본인 확인 |
| 소득·재산 증빙 | 자격 확인 |
| 금융정보 동의서 | 재산 조회 |

| 단계 | 기간 |
|------|------|
| 조사 | 30일 |
| 결정 통보 | 14일 |

## 9. 부양의무자 기준

### 9.1 급여별 적용

| 급여 | 부양의무자 기준 |
|------|----------------|
| 생계급여 | 완화 (고소득 제외) |
| 의료급여 | 적용 |
| 주거급여 | **폐지** |
| 교육급여 | **폐지** |

### 9.2 생계급여 예외

| 조건 | 부양의무자 기준 미적용 |
|------|---------------------|
| 노인 가구 | ✅ |
| 한부모 가구 | ✅ |
| 장애인 가구 | ✅ |
