# 연말정산 중소기업 취업자 감면

중소기업 취업자 소득세 감면은 중소기업에 취업한 근로자의 **소득세를 감면**해주는 제도입니다. [국세청 중소기업 취업자 감면 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6598&cntntsId=7877)에 따른 조건을 정리합니다.

## 1. 감면 대상자와 감면율

[조세특례제한법 제30조](https://www.law.go.kr/법령/조세특례제한법)에 따른 대상별 감면율입니다.

| 대상 | 감면율 | 감면 기간 | 연간 한도 |
|------|--------|----------|----------|
| **청년 (15~34세)** | **90%** | **5년** | 200만원 |
| 고령자 (60세 이상) | 70% | 3년 | 200만원 |
| 장애인 | 70% | 3년 | 200만원 |
| 경력단절여성 | 70% | 3년 | 200만원 |
| **재취직 남성 (2025년~)** | **70%** | **3년** | 200만원 |

## 2. 청년 감면 조건

### 2.1 나이 요건

| 요건 | 기준 |
|------|------|
| 연령 | 취업일 현재 **15세 이상 34세 이하** |
| 군복무 | 복무기간 최대 6년까지 나이에서 차감 |

### 2.2 군복무 기간 차감

**예시**: 만 36세, 군복무 2년

- 실제 나이: 36세
- 군복무 차감: 36 - 2 = 34세
- **청년 감면 적용 가능**

### 2.3 감면 내용

| 항목 | 내용 |
|------|------|
| 감면율 | **90%** |
| 감면 기간 | 취업일부터 **5년** |
| 연간 한도 | **200만원** |

## 3. 기타 대상자 조건

### 3.1 고령자

| 요건 | 기준 |
|------|------|
| 연령 | 취업일 현재 **60세 이상** |
| 감면율 | 70% |
| 감면 기간 | 3년 |

### 3.2 장애인

| 요건 | 기준 |
|------|------|
| 대상 | 장애인복지법상 장애인 |
| 감면율 | 70% |
| 감면 기간 | 3년 |

### 3.3 경력단절여성

| 요건 | 기준 |
|------|------|
| 퇴직 사유 | 임신, 출산, 육아, 결혼 |
| 경력단절 기간 | 퇴직 후 2~15년 이내 재취업 |
| 감면율 | 70% |
| 감면 기간 | 3년 |

### 3.4 재취직 남성 (2025년 신설)

| 요건 | 기준 |
|------|------|
| 적용 시기 | **2025년 3월 14일 이후** 중소기업 재취직 |
| 대상 | 경력단절 후 중소기업 재취직한 남성 |
| 감면율 | **70%** |
| 감면 기간 | **3년** |

> 기존에는 경력단절여성만 해당되었으나, 2025년부터 남성도 동일한 감면 혜택을 받을 수 있습니다.

## 4. 중소기업 요건

### 4.1 중소기업 범위

| 업종 | 매출액 기준 |
|------|------------|
| 제조업 | 1,500억원 이하 |
| 건설업 | 1,000억원 이하 |
| 도소매업 | 1,000억원 이하 |
| 서비스업 | 600억원 이하 |

### 4.2 제외 업종

| 업종 | 제외 여부 |
|------|----------|
| 전문·과학·기술서비스업 | ✅ 감면 가능 |
| 금융·보험업 | ❌ 제외 |
| 부동산업 | ❌ 제외 |
| 유흥주점업 | ❌ 제외 |

## 5. 감면 신청 방법

### 5.1 신청 절차

1. **감면신청서 작성**: 입사 시 작성
2. **회사 제출**: 인사/경리 부서에 제출
3. **원천징수 반영**: 회사가 매월 급여에서 감면 적용
4. **연말정산 확인**: 감면 내역 확인

### 5.2 제출 서류

| 서류 | 용도 |
|------|------|
| 중소기업 취업자 소득세 감면신청서 | 감면 신청 |
| 병적증명서 (청년 군복무자) | 나이 차감 |
| 장애인증명서 (장애인) | 자격 확인 |
| 경력단절 확인서 (경력단절여성) | 자격 확인 |

## 6. 감면 계산 예시

### 6.1 청년 (90% 감면)

**연봉 4,000만원, 결정세액 200만원인 청년**

| 구분 | 금액 |
|------|------|
| 결정세액 | 200만원 |
| 감면액 (90%) | 180만원 |
| **납부세액** | **20만원** |

### 6.2 고령자 (70% 감면)

**연봉 3,500만원, 결정세액 150만원인 고령자**

| 구분 | 금액 |
|------|------|
| 결정세액 | 150만원 |
| 감면액 (70%) | 105만원 |
| **납부세액** | **45만원** |

## 7. 감면 기간 계산

### 7.1 시작일

| 상황 | 감면 시작일 |
|------|------------|
| 신규 취업 | **취업일** |
| 이직 (다른 중소기업) | **새 취업일** |

### 7.2 종료일

| 대상 | 종료일 |
|------|--------|
| 청년 | 취업일로부터 **5년** |
| 기타 | 취업일로부터 **3년** |

### 7.3 이직 시

- 다른 중소기업으로 이직 → **잔여 기간** 감면
- 대기업으로 이직 → 감면 종료

## 8. 주의사항

### 8.1 소급 신청

- 입사 후 신청하지 않았다면 **경정청구**로 환급 가능
- 5년 이내 경정청구 가능

### 8.2 중복 적용 불가

- 청년 + 장애인 → **청년(90%)** 적용
- 청년 + 경력단절여성 → **청년(90%)** 적용

### 8.3 연간 한도

- 감면액이 **200만원 초과** 시 200만원까지만 감면
- 초과분은 다음 해로 이월되지 않음

### 8.4 퇴직 시

- 퇴직 시점까지 감면 적용
- 중도 퇴직해도 감면받은 세금 반환 불필요
