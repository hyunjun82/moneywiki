# 정치자금 기부금 공제

## 기본 개념
정당이나 후보자에게 기부한 정치자금에 대해 세액을 공제해주는 제도입니다.

## 공제 대상 기부금

### 해당 기부금
- 정당 후원금
- 선거 후보자 기부금
- 중앙선거관리위원회 경유 기부금
- 정치자금법에 따른 기부금

### 제외 대상
- 정치자금법 위반 기부금
- 익명 기부금
- 한도 초과 기부금

## 공제 한도

### 연간 한도
- 근로소득자: **종합소득금액의 100%**
- 실질적 한도: **10만원** (세액공제 효과 고려)

### 기부금 한도
| 기부 대상 | 연간 한도 |
|---------|---------|
| 정당 후원금 | 2천만원 |
| 후보자 기부금 | 2천만원 |
| 합계 | 각각 적용 |

## 세액공제율

### 10만원 이하
- **공제율**: 100/110 (약 **90.9%**)
- 실질적으로 기부 금액 전액 환급

### 10만원 초과분
- **공제율**: 15% (또는 25%)
- 3천만원 초과 고소득자는 25%

### 계산 예시
```
기부금 15만원인 경우:
- 10만원: 10만원 × 100/110 = 90,909원
- 5만원: 5만원 × 15% = 7,500원
- 총 공제: 98,409원
```

## 실무 적용 사례

### 사례 1: 정당 후원금 10만원
```
기부금: 10만원
세액공제: 10만원 × 100/110 = 90,909원
실질 부담: 9,091원
```

### 사례 2: 정당 후원금 30만원
```
10만원 이하: 90,909원
20만원 초과분: 20만원 × 15% = 30,000원
총 공제: 120,909원
실질 부담: 179,091원
```

### 사례 3: 정당 + 후보자 기부
```
정당 후원금: 10만원 → 공제 90,909원
후보자 기부: 10만원 → 공제 90,909원
총 공제: 181,818원
```

## 신청 방법

### 기부금 납부
- 정당 후원금: 정당 계좌 이체
- 후보자 기부금: 선거관리위원회 경유
- 반드시 본인 명의로 기부

### 영수증 발급
- 정당: 정치자금영수증 발급
- 선관위: 선관위에서 발급
- 연말정산 간소화 서비스 조회 가능

### 연말정산 제출
- 회사에 영수증 제출
- 또는 간소화 서비스 자료 활용
- 정치자금 항목 별도 표시

## 주의사항

### 본인 명의 필수
- 배우자, 자녀 명의 기부금 공제 불가
- 반드시 본인 계좌에서 이체

### 한도 초과 기부
- 2천만원 초과 기부 시 초과분 무효
- 정치자금법 위반으로 과태료 부과

### 익명 기부 불가
- 반드시 실명으로 기부
- 익명 기부는 공제 불가

### 영수증 보관
- 정치자금영수증 5년간 보관
- 세무조사 시 증빙 자료

## 다른 기부금과의 관계

### 공제 순서
1. 정치자금 기부금 (우선 공제)
2. 법정기부금
3. 지정기부금

### 한도 계산
- 정치자금 기부금: 별도 한도
- 다른 기부금: 각각 별도 계산
- 합산하지 않음

## 정당 후원회 가입

### 가입 방법
1. 정당 홈페이지 방문
2. 후원회 가입 신청
3. 자동이체 설정

### 정기 후원
- 월 1만원 이상 가능
- 자동이체 시 영수증 자동 발급
- 연말정산 시 자동 조회

### 혜택
- 세액공제로 실질 부담 최소화
- 정치 참여 기회

## 선거 기부금

### 선거 기간
- 예비후보자 등록일부터 가능
- 투표일 전일까지

### 기부 방법
- 선거관리위원회 경유 필수
- 후보자 직접 기부 금지

### 한도
- 후보자당 2천만원
- 여러 후보자 가능 (각각 2천만원)

## 실무 팁

1. **10만원 기부**: 실질 부담 약 9천원으로 최대 효과
2. **정기 후원**: 자동이체 설정으로 편리한 관리
3. **영수증 확인**: 간소화 서비스 자동 조회 확인
4. **가족 합산 불가**: 각자 명의로 기부해야 공제
