# 법인세

법인세는 법인의 소득에 대해 부과되는 세금입니다. 세율과 신고 방법을 정리합니다.

## 1. 법인세 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 법인 소득에 대한 세금 |
| 납세 의무자 | 국내 법인, 외국 법인 |
| 과세 대상 | 각 사업연도 소득 |

### 1.2 과세 기간

| 항목 | 기간 |
|------|------|
| 사업연도 | 1년 (정관에 따름) |
| 일반적 | 1월 1일 ~ 12월 31일 |

## 2. 법인세율 (2026년)

### 2.1 일반 세율

| 과세표준 | 세율 | 누진공제 |
|----------|------|----------|
| 2억원 이하 | **9%** | - |
| 2억~200억원 | **19%** | 2,000만원 |
| 200억~3,000억원 | **21%** | 4.2억원 |
| 3,000억원 초과 | **24%** | 13.2억원 |

### 2.2 세액 계산 예시

**과세표준 10억원**

| 항목 | 금액 |
|------|------|
| 과세표준 | 10억원 |
| 세율 | 19% |
| 산출세액 | 1.9억원 |
| (-) 누진공제 | 2,000만원 |
| **법인세** | **1.7억원** |

### 2.3 지방소득세

| 항목 | 세율 |
|------|------|
| 지방소득세 | 법인세의 **10%** |

## 3. 중소기업 특례

### 3.1 중소기업 세율

| 과세표준 | 일반 | 중소기업 |
|----------|------|----------|
| 2억 이하 | 9% | **9%** |
| 2억~200억 | 19% | 19% |

### 3.2 중소기업 기준

| 업종 | 매출액 기준 |
|------|------------|
| 제조업 | 1,500억원 이하 |
| 도소매 | 1,500억원 이하 |
| 서비스업 | 1,000억원 이하 |

### 3.3 중소기업 감면

| 감면 | 내용 |
|------|------|
| 창업 감면 | 5년간 50~100% |
| 고용증가 | 1인당 세액공제 |
| R&D | 연구개발비 세액공제 |

## 4. 과세표준 계산

### 4.1 계산 구조

```
각 사업연도 소득
= 익금 - 손금

익금: 매출, 이자수익 등
손금: 비용, 감가상각 등

과세표준
= 각 사업연도 소득 - 이월결손금 - 비과세소득 - 소득공제
```

### 4.2 익금 항목

| 익금 | 예시 |
|------|------|
| 매출액 | 상품·용역 판매 |
| 이자수익 | 예금 이자 |
| 배당수익 | 투자 배당 |
| 자산처분익 | 자산 매각 이익 |

### 4.3 손금 항목

| 손금 | 예시 |
|------|------|
| 매출원가 | 상품 구입비 |
| 판매관리비 | 인건비, 임차료 |
| 감가상각비 | 자산 감가상각 |
| 대손충당금 | 대손 예상액 |

## 5. 세액공제·감면

### 5.1 주요 세액공제

| 공제 | 내용 |
|------|------|
| R&D 세액공제 | 연구개발비의 25~50% |
| 고용증대 | 1인당 700~1,550만원 |
| 투자세액공제 | 투자금액의 3~12% |
| 중소기업투자 | 투자금액의 10% |

### 5.2 창업 중소기업 감면

| 지역 | 감면율 | 기간 |
|------|--------|------|
| 수도권 과밀억제권역 외 | 50% | 5년 |
| 청년 창업 | 100% | 5년 |

### 5.3 공제 한도

| 한도 | 기준 |
|------|------|
| 최저한세 | 과세표준의 7~17% |
| 중소기업 | 7% |
| 일반기업 | 10~17% |

## 6. 신고·납부

### 6.1 신고 기한

| 항목 | 기한 |
|------|------|
| 법인세 신고 | 사업연도 종료 후 **3개월** |
| 12월 결산 법인 | **3월 31일** |

### 6.2 중간예납

| 항목 | 내용 |
|------|------|
| 대상 | 사업연도 6개월 초과 법인 |
| 기한 | 사업연도 개시 후 6개월 + 2개월 |
| 금액 | 직전년도 법인세의 50% |

### 6.3 신고 방법

| 방법 | 내용 |
|------|------|
| 홈택스 | 전자신고 |
| 세무대리인 | 세무사 위임 |

### 6.4 필요 서류

| 서류 | 용도 |
|------|------|
| 재무제표 | 손익계산서, 재무상태표 |
| 세무조정계산서 | 익금·손금 조정 |
| 법인세 과세표준 신고서 | 기본 신고서 |

## 7. 가산세

### 7.1 신고 가산세

| 위반 | 가산세 |
|------|--------|
| 무신고 | **20%** (부정 40%) |
| 과소신고 | **10%** (부정 40%) |

### 7.2 납부 가산세

| 위반 | 가산세 |
|------|--------|
| 미납·과소납부 | 1일 **0.022%** |

### 7.3 기타 가산세

| 위반 | 가산세 |
|------|--------|
| 지급명세서 미제출 | 1% |
| 계산서 미발급 | 2% |

## 8. 이월결손금

### 8.1 공제 요건

| 요건 | 내용 |
|------|------|
| 이월 기간 | **15년** |
| 공제 한도 | 과세표준의 60% (중소기업 100%) |

### 8.2 활용

| 상황 | 효과 |
|------|------|
| 전년도 손실 | 당해 소득에서 공제 |
| 법인세 절감 | 과세표준 감소 |

## 9. 법인지방소득세

### 9.1 세율

| 항목 | 세율 |
|------|------|
| 법인지방소득세 | 법인세의 **10%** |

### 9.2 신고

| 항목 | 내용 |
|------|------|
| 신고처 | 지방자치단체 (위택스) |
| 기한 | 법인세 신고기한과 동일 |

## 10. 주의사항

### 10.1 세무조정

| 주의 | 내용 |
|------|------|
| 손금불산입 | 인정되지 않는 비용 |
| 익금불산입 | 과세 제외 수익 |
| 전문가 상담 | 세무사 활용 권장 |

### 10.2 접대비 한도

| 기본 한도 | 추가 한도 |
|----------|----------|
| 1,200만원 | 매출액의 0.03~0.2% |
