# 지방소득세 신고·납부

## 기본 개념
국세인 소득세와 별도로 지방자치단체에 납부하는 지방세입니다.

## 세율

### 소득세의 10%
- 지방소득세 = 소득세 × **10%**

### 예시
```
소득세: 100만원
지방소득세: 10만원 (100만원 × 10%)
총 납부세액: 110만원
```

## 신고·납부 대상

### 1. 종합소득 지방소득세
- 종합소득세 납부자
- 프리랜서, 사업자, 임대소득자 등

### 2. 퇴직소득 지방소득세
- 퇴직금 수령자
- 회사에서 원천징수

### 3. 양도소득 지방소득세
- 부동산 양도자
- 양도소득세 납부자

### 4. 특별징수 (원천징수)
- 근로소득: 회사에서 원천징수
- 이자·배당: 금융기관에서 원천징수

## 신고 기한

### 종합소득 지방소득세
- **5월 1일 ~ 5월 31일**
- 종합소득세와 동일

### 양도소득 지방소득세
- 예정신고: 양도일이 속한 달의 **말일로부터 2개월**
- 확정신고: 다음 연도 **5월 1일~5월 31일**

### 특별징수 (원천징수)
- 원천징수 다음 달 **10일까지**
- 회사·금융기관에서 자동 납부

## 신고 방법

### 위택스 (www.wetax.go.kr)
1. 위택스 접속
2. 로그인 (공동인증서, 간편인증)
3. 신고하기 → 지방소득세 (종합소득)
4. 자동 계산 (종합소득세 자료 연동)
5. 신고·납부

### 자동 연동
- 국세청 종합소득세 신고 자료 자동 연동
- 소득세 × 10% 자동 계산
- 별도 입력 불필요

### 방문 신고
- 주소지 관할 구청·시청
- 신고서 작성 및 제출

## 납부 방법

### 전자납부
1. 위택스 전자납부
2. 가상계좌 이체
3. 신용카드 납부

### 금융기관
- 은행 방문 납부
- 지방소득세 납부서

### 자동이체
- 계좌 자동이체 등록
- 납부 기한에 자동 출금

## 실무 사례

### 사례 1: 종합소득세 100만원
```
소득세: 100만원
지방소득세: 10만원
총 납부: 110만원

신고:
- 5월 31일까지
- 국세청: 소득세 100만원
- 위택스: 지방소득세 10만원
```

### 사례 2: 양도소득세 500만원
```
양도소득세: 500만원
양도소득 지방소득세: 50만원

신고:
- 예정신고: 양도일로부터 2개월
- 국세청: 양도세 500만원
- 위택스: 지방소득세 50만원
```

### 사례 3: 근로소득 (원천징수)
```
월급: 300만원
소득세: 5만원
지방소득세: 5,000원

처리:
- 회사에서 자동 원천징수
- 별도 신고 불필요
```

## 가산세

### 무신고 가산세
- 납부세액의 **20%**
- 부정 무신고: **40%**

### 과소신고 가산세
- 과소 납부액의 **10%**
- 부정 과소신고: **40%**

### 납부지연 가산세
- 미납세액 × 일수 × **0.022%**
- 연 약 8% 수준

## 주의사항

### 별도 신고 필수
- 소득세 신고와 별도
- 위택스에서 따로 신고

### 자동 연동 확인
- 국세청 자료 연동되었는지 확인
- 금액 일치 여부 점검

### 신고 기한
- 소득세와 동일한 기한
- 동시에 신고·납부 권장

### 환급
- 소득세 환급 시 지방소득세도 환급
- 위택스에서 별도 환급 신청

## 연말정산과 지방소득세

### 원천징수
- 회사에서 매월 원천징수
- 소득세의 10%

### 연말정산
- 연말정산 시 함께 정산
- 환급·추징 함께 처리

### 별도 신고 불필요
- 근로소득만 있으면 신고 안 해도 됨
- 회사에서 자동 처리

## 개인사업자·프리랜서

### 종합소득세 신고 시
1. **5월 국세청**: 종합소득세 신고
2. **5월 위택스**: 지방소득세 신고 (자동 연동)

### 자동 계산
- 소득세 × 10%
- 별도 계산 불필요

### 납부
- 신고와 동시에 납부
- 또는 납부 기한 내 납부

## 양도소득 지방소득세

### 예정신고 (필수)
- 양도일로부터 2개월 이내
- 양도세와 함께 신고

### 확정신고
- 다음 연도 5월
- 연간 양도 내역 확정

### 납부
- 예정신고: 양도일로부터 2개월
- 확정신고: 다음 연도 5월

## 실무 팁

### 1. 동시 신고
- 소득세와 지방소득세 같은 날 신고
- 신고 누락 방지

### 2. 자동 연동 활용
- 국세청 자료 자동 연동
- 입력 오류 방지

### 3. 전자납부 활용
- 위택스 전자납부
- 영수증 자동 발급

### 4. 환급 확인
- 소득세 환급 시 지방소득세도 확인
- 위택스에서 환급 신청
