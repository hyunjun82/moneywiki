# 연말정산 현금영수증 공제

현금영수증은 30% 공제율로 신용카드보다 유리합니다. [국세청 현금영수증 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2469&cntntsId=7794)에 따른 발급 방법과 공제 조건을 정리합니다.

## 1. 현금영수증 공제율

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제율입니다.

| 결제 수단 | 공제율 |
|-----------|--------|
| 현금영수증 | **30%** |
| 체크카드/직불카드 | 30% |
| 신용카드 | 15% |
| 전통시장 사용분 | 40% |
| 대중교통 사용분 | 40% |

## 2. 공제 조건

### 2.1 공제 시작점

**총급여의 25% 초과 사용분**부터 공제 적용

| 총급여 | 25% 기준액 |
|--------|------------|
| 3,000만원 | 750만원 초과분부터 |
| 4,000만원 | 1,000만원 초과분부터 |
| 5,000만원 | 1,250만원 초과분부터 |

### 2.2 공제 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

## 3. 현금영수증 발급 방법

### 3.1 발급수단 등록 (사전 등록)

[국세청 홈택스](https://www.hometax.go.kr)에서 등록:
1. 홈택스 로그인
2. 계산서·영수증·카드 → 현금영수증
3. 소비자 발급수단·전용카드
4. **휴대전화번호** 또는 **카드번호** 등록

등록 후 현금 결제 시 휴대전화번호만 말하면 **자동 발급**됩니다.

### 3.2 현장 발급 요청

- 결제 시 "현금영수증 발급해주세요" 요청
- 휴대전화번호 또는 현금영수증 카드 제시

### 3.3 사후 발급 요청

발급받지 못한 경우:
- **거래일로부터 3년 이내** 홈택스에서 발급 요청 가능
- 거래 증빙 서류 필요 (영수증 등)

## 4. 현금영수증 조회

### 4.1 홈택스 조회

1. [홈택스](https://www.hometax.go.kr) 로그인
2. 계산서·영수증·카드 → 현금영수증
3. 사용내역 조회

### 4.2 연말정산 간소화

1월 15일부터 **연말정산 간소화 서비스**에서 일괄 조회 가능

## 5. 현금영수증 미발급 신고

### 5.1 의무발행 사업자

**건당 10만원 이상** 현금 거래 시 사업자는 반드시 현금영수증을 발급해야 합니다.

### 5.2 신고 방법

미발급 시 홈택스에서 신고 가능:
- 상담·불복·제보 → 현금영수증·신용카드 미발급/발급거부

### 5.3 신고 포상금

미발급 금액의 **20%** 포상금 지급 (건당 최대 50만원)

## 6. 공제 제외 항목

현금영수증을 발급받아도 **공제되지 않는 항목**:

- 세금, 공과금
- 상품권 구매 (상품권으로 물건 구입 시는 가능)
- 면세점 구매
- 보험료
- 학교 등록금
- 아파트 관리비

## 7. 월세 현금영수증

### 7.1 발급 방법

1. 홈택스 → 상담·불복·제보
2. 현금영수증·신용카드 미발급/발급거부
3. **주택임차료(월세) 현금영수증 발급신청**
4. 임대차계약서 첨부 (임대인 동의 불필요)

### 7.2 주의사항

월세 현금영수증과 **월세 세액공제는 중복 불가**
- 둘 중 유리한 것 하나만 선택
- 일반적으로 **월세 세액공제**(15~17%)가 더 유리
