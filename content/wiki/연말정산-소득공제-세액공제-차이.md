# 연말정산 소득공제 세액공제 차이

소득공제와 세액공제의 차이를 정리합니다.

## 1. 개념 비교

| 구분 | 소득공제 | 세액공제 |
|------|----------|----------|
| 정의 | 과세표준 감소 | 세액 직접 차감 |
| 효과 | 세율에 따라 다름 | 일정 금액 차감 |

## 2. 계산 순서

```
총급여
- 근로소득공제 = 근로소득금액
- 소득공제 = 과세표준
× 세율 = 산출세액
- 세액공제 = 결정세액
```

## 3. 주요 소득공제

| 항목 | 내용 |
|------|------|
| 인적공제 | 1인당 150만원 |
| 신용카드 | 사용액의 15~40% |
| 연금보험료 | 전액 |
| 주택자금 | 최대 1,800만원 |

## 4. 주요 세액공제

| 항목 | 공제율 |
|------|--------|
| 자녀 | 1인당 15만원 |
| 의료비 | 15% |
| 교육비 | 15% |
| 연금저축 | 13.2~16.5% |

## 5. 효과 비교

| 총급여 | 소득공제 100만원 | 세액공제 100만원 |
|--------|-----------------|-----------------|
| 3천만원 | 세금 15만원 감소 | 세금 100만원 감소 |
| 8천만원 | 세금 24만원 감소 | 세금 100만원 감소 |
