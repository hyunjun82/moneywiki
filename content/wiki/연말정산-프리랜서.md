# 연말정산 프리랜서

프리랜서 세금 정산을 정리합니다.

## 1. 프리랜서 과세 방식

| 항목 | 내용 |
|------|------|
| 연말정산 | ❌ 대상 아님 |
| 종소세 신고 | ✅ 5월 신고 |
| 원천징수 | 3.3% |

## 2. 3.3% 원천징수

| 구성 | 세율 |
|------|------|
| 소득세 | 3% |
| 지방소득세 | 0.3% |
| 합계 | **3.3%** |

## 3. 종합소득세 신고

| 항목 | 내용 |
|------|------|
| 신고 기간 | 5월 1~31일 |
| 신고 방법 | 홈택스 전자신고 |
| 납부 | 5월 31일까지 |

## 4. 필요경비

| 방법 | 내용 |
|------|------|
| 단순경비율 | 수입 적을 때 유리 |
| 기준경비율 | 수입 많을 때 |
| 장부기장 | 경비 많을 때 |

## 5. 환급 가능성

| 상황 | 환급 |
|------|------|
| 경비 많음 | 3.3% 환급 가능 |
| 소득 적음 | 환급 가능성 높음 |
