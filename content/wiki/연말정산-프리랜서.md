프리랜서 세금 정산을 정리해 볼게요. 프리랜서는 연말정산이 아닌 종합소득세 신고를 해요.

## 프리랜서 과세 방식

프리랜서 세금 처리 방식이에요.

| 항목 | 내용 |
|------|------|
| 연말정산 | 대상 아님 |
| 종소세 신고 | 5월 신고 |
| 원천징수 | 3.3% |

## 3.3% 원천징수

원천징수 세율 구성이에요.

| 구성 | 세율 |
|------|------|
| 소득세 | 3% |
| 지방소득세 | 0.3% |
| 합계 | 3.3% |

## 종합소득세 신고

종소세 신고 방법이에요.

| 항목 | 내용 |
|------|------|
| 신고 기간 | 5월 1~31일 |
| 신고 방법 | 홈택스 전자신고 |
| 납부 | 5월 31일까지 |

## 필요경비

경비 처리 방법이에요.

| 방법 | 내용 |
|------|------|
| 단순경비율 | 수입 적을 때 유리 |
| 기준경비율 | 수입 많을 때 |
| 장부기장 | 경비 많을 때 |

## 환급 가능성

환급받을 수 있는 경우예요.

| 상황 | 환급 |
|------|------|
| 경비 많음 | 3.3% 환급 가능 |
| 소득 적음 | 환급 가능성 높음 |
