# 임대차분쟁조정 신청

임대차 분쟁이 생겼을 때 소송 전 조정을 통해 해결할 수 있습니다. 무료이고 빠른 분쟁 해결 방법입니다.

## 1. 임대차분쟁조정이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 조정 | 제3자가 분쟁 해결을 중재 |
| 기관 | 임대차분쟁조정위원회 |
| 근거 | 주택임대차보호법 제14조 |

### 1.2 장점

| 장점 | 설명 |
|------|------|
| 무료 | 신청료 없음 |
| 빠름 | 60일 이내 |
| 전문성 | 전문 위원 조정 |
| 비공개 | 비공개 진행 |

### 1.3 운영 기관

| 기관 | 연락처 |
|------|--------|
| 대한법률구조공단 | **132** |
| 온라인 | klac.or.kr |

## 2. 조정 대상

### 2.1 조정 가능 분쟁

| 분쟁 유형 | 예시 |
|----------|------|
| 보증금 반환 | 보증금 미반환 |
| 임대료 | 인상 분쟁 |
| 계약갱신 | 갱신청구권 분쟁 |
| 수리 | 수리의무 분쟁 |
| 원상복구 | 퇴거 시 분쟁 |
| 계약 해지 | 해지 분쟁 |

### 2.2 조정 대상 주택

| 주택 | 대상 |
|------|------|
| 아파트 | ✅ |
| 다세대·빌라 | ✅ |
| 단독주택 | ✅ |
| 오피스텔 (주거용) | ✅ |
| 상가 | ❌ (별도 기관) |

### 2.3 조정 제외

| 제외 | 이유 |
|------|------|
| 상가 | 상가건물임대차분쟁조정위원회 |
| 형사 | 형사 사건 |
| 소송 중 | 법원에서 진행 중 |

## 3. 신청 자격

### 3.1 신청인

| 신청인 | 가능 |
|--------|------|
| 임차인 | ✅ |
| 임대인 | ✅ |
| 대리인 | ✅ (위임장) |

### 3.2 신청 시기

| 시기 | 신청 |
|------|------|
| 분쟁 발생 후 | ✅ |
| 소송 전 | ✅ |
| 소송 중 | ❌ |

## 4. 신청 방법

### 4.1 온라인 신청

| 절차 | 내용 |
|------|------|
| 사이트 | klac.or.kr |
| 메뉴 | 임대차분쟁조정 신청 |
| 입력 | 신청서 작성 |
| 서류 | 첨부 업로드 |

### 4.2 방문 신청

| 절차 | 내용 |
|------|------|
| 장소 | 대한법률구조공단 지부 |
| 준비 | 신청서, 서류 |
| 접수 | 직접 제출 |

### 4.3 전화 상담

| 절차 | 내용 |
|------|------|
| 전화 | 132 |
| 상담 | 분쟁 내용 상담 |
| 안내 | 신청 방법 안내 |

## 5. 필요 서류

### 5.1 필수 서류

| 서류 | 용도 |
|------|------|
| 조정신청서 | 신청 양식 |
| 임대차계약서 | 계약 증명 |
| 신분증 사본 | 신청인 확인 |
| 등기부등본 | 부동산 정보 |

### 5.2 분쟁별 추가 서류

| 분쟁 | 추가 서류 |
|------|----------|
| 보증금 반환 | 내용증명, 입금 내역 |
| 수리 | 수리 요청 내역, 견적서 |
| 원상복구 | 인수인계서, 사진 |
| 계약갱신 | 갱신 요청 내역 |

### 5.3 대리인 신청

| 서류 | 내용 |
|------|------|
| 위임장 | 대리권 증명 |
| 대리인 신분증 | 대리인 확인 |

## 6. 조정 절차

### 6.1 절차 흐름

```
신청 접수 → 요건 심사 → 조정 개시 →
조정 기일 → 조정안 제시 → 수락/거부 →
조정 성립/불성립
```

### 6.2 단계별 내용

| 단계 | 내용 | 기간 |
|------|------|------|
| 접수 | 신청서 접수 | 즉시 |
| 심사 | 요건 검토 | 7일 |
| 통지 | 상대방 통지 | 14일 |
| 조정 | 조정 기일 진행 | 30~60일 |
| 종결 | 조정 성립/불성립 | 즉시 |

### 6.3 조정 기일

| 항목 | 내용 |
|------|------|
| 출석 | 당사자 출석 |
| 진술 | 각자 주장 |
| 협의 | 조정 위원 중재 |
| 조정안 | 해결방안 제시 |

## 7. 조정 효력

### 7.1 조정 성립

| 효력 | 내용 |
|------|------|
| 재판상 화해 | 판결과 같은 효력 |
| 집행력 | 강제집행 가능 |
| 기판력 | 같은 분쟁 소송 불가 |

### 7.2 조정 불성립

| 상황 | 결과 |
|------|------|
| 거부 | 조정 불성립 |
| 불출석 | 조정 불성립 |
| 이후 | 소송 진행 가능 |

### 7.3 수락과 거부

| 선택 | 효과 |
|------|------|
| 양측 수락 | 조정 성립, 법적 효력 |
| 일방 거부 | 조정 불성립 |
| 이의 | 14일 내 이의 가능 |

## 8. 조정 기간

### 8.1 처리 기간

| 기간 | 내용 |
|------|------|
| 원칙 | **60일** 이내 |
| 연장 | 30일 1회 연장 가능 |
| 최대 | **90일** |

### 8.2 기간 단축

| 방법 | 효과 |
|------|------|
| 서류 완비 | 신속 처리 |
| 협조 | 조정 기일 조율 |

## 9. 비용

### 9.1 신청 비용

| 항목 | 비용 |
|------|------|
| 신청료 | **무료** |
| 조정 비용 | **무료** |
| 서류 발급 | 별도 (등기부등본 등) |

### 9.2 소송과 비교

| 구분 | 조정 | 소송 |
|------|------|------|
| 비용 | 무료 | 인지대, 변호사비 |
| 기간 | 60일 | 3~6개월 이상 |
| 공개 | 비공개 | 공개 |

## 10. 조정 결과 대응

### 10.1 성립 시

| 조치 | 내용 |
|------|------|
| 조정조서 수령 | 보관 |
| 이행 | 조정 내용 이행 |
| 미이행 시 | 강제집행 신청 |

### 10.2 불성립 시

| 재협의 | 당사자 협의 |

### 10.3 이의 신청
