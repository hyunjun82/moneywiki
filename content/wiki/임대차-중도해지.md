# 임대차 중도해지

계약 기간 중 집을 비워야 하거나 세입자를 내보내야 할 때가 있습니다. 중도해지 조건과 절차를 정확히 알아야 합니다.

## 1. 중도해지란

### 1.1 기본 개념

| 항목 | 내용 |
|------|------|
| 중도해지 | 계약 기간 중 계약 종료 |
| 요건 | 합의 또는 법정 사유 |
| 효과 | 계약 종료, 원상회복 |

### 1.2 해지 유형

| 유형 | 설명 |
|------|------|
| 합의해지 | 쌍방 합의로 종료 |
| 약정해지 | 계약서 해지조항에 따라 |
| 법정해지 | 법률상 해지사유 발생 |

## 2. 임차인 중도해지

### 2.1 합의해지

| 절차 | 내용 |
|------|------|
| 협의 | 임대인과 해지 협의 |
| 합의 | 해지 일자, 조건 합의 |
| 서면 | 합의서 작성 권장 |
| 정산 | 보증금 반환, 위약금 등 |

### 2.2 약정해지 (특약)

| 특약 예시 | 효력 |
|----------|------|
| "임차인은 2개월 전 통지로 해지" | ✅ 유효 |
| "위약금 월세 1개월분" | ✅ 유효 |
| "중도해지 불가" | ⚠️ 효력 제한적 |

### 2.3 법정 해지사유 (임차인)

| 사유 | 근거 |
|------|------|
| 목적물 하자 | 거주 불가 수준 하자 |
| 임대인 의무 불이행 | 수리의무 불이행 등 |
| 건물 멸실 | 화재 등 |

## 3. 임대인 중도해지

### 3.1 합의해지

| 절차 | 내용 |
|------|------|
| 협의 | 임차인과 해지 협의 |
| 보상 | 이사비, 조기 반환 등 |
| 합의 | 조건 합의 후 해지 |

### 3.2 법정 해지사유 (임대인)

| 사유 | 요건 |
|------|------|
| 2기 월세 연체 | **2회 이상 연체** |
| 무단 전대 | 임대인 동의 없이 전대 |
| 용도 위반 | 주거 외 용도 사용 |
| 고의 훼손 | 고의로 시설 파손 |

### 3.3 2기 연체 해지

| 항목 | 내용 |
|------|------|
| 요건 | 월세 2기분 이상 연체 |
| 통지 | 해지 통지 필요 |
| 효력 | 통지 도달 시 해지 |

## 4. 묵시적 갱신 후 해지

### 4.1 임차인 해지

| 항목 | 내용 |
|------|------|
| 통지 | 해지 의사 통지 |
| 기간 | 통지 후 **3개월** 경과 |
| 효력 | 3개월 후 해지 효력 |

### 4.2 임대인 해지

| 항목 | 내용 |
|------|------|
| 제한 | 법정 사유 필요 |
| 사유 | 2기 연체, 무단 전대 등 |
| 통지 | 해지 통지 필요 |

### 4.3 해지 통지 예시

```
[계약해지 통지서]

임대인 ○○○ 귀하

○○시 ○○구 ○○동 ○○호 임대차계약에 대하여
주택임대차보호법 제6조의3에 따라 계약 해지를
통지합니다.

해지 효력 발생일: 본 통지일로부터 3개월 후

2026년 ○월 ○일
임차인: ○○○ (서명)
```

## 5. 위약금

### 5.1 위약금 기준

| 유형 | 통상 기준 |
|------|----------|
| 월세 | 월세 1-2개월분 |
| 전세 | 보증금의 10% 이내 |
| 약정 | 계약서 특약 우선 |

### 5.2 위약금 발생 상황

| 상황 | 위약금 |
|------|--------|
| 합의해지 (위약금 합의) | 합의 금액 |
| 약정해지 (특약 있음) | 특약 금액 |
| 일방적 해지 | 손해배상 청구 가능 |

### 5.3 위약금 면제 사유

| 사유 | 설명 |
|------|------|
| 임대인 동의 | 위약금 없이 합의 |
| 법정 해지사유 | 상대방 귀책 |
| 불가항력 | 재난, 질병 등 |

## 6. 해지 절차

### 6.1 임차인 해지 절차

| 단계 | 내용 |
|------|------|
| 1 | 임대인에게 해지 의사 통지 |
| 2 | 해지 조건 협의 (위약금 등) |
| 3 | 합의서 작성 |
| 4 | 퇴거일 확정 |
| 5 | 보증금 반환 정산 |

### 6.2 통지 방법

| 방법 | 권장 |
|------|------|
| 내용증명 | ⭐ 증거력 확보 |
| 문자·카톡 | ○ 증거 남김 |
| 구두 | △ 증거 어려움 |

### 6.3 합의서 내용

| 항목 | 내용 |
|------|------|
| 해지 일자 | 계약 종료일 |
| 위약금 | 금액 또는 면제 |
| 보증금 | 반환 일자, 방법 |
| 원상복구 | 범위 합의 |

## 7. 보증금 반환

### 7.1 반환 시기

| 상황 | 반환 시기 |
|------|----------|
| 합의해지 | 합의한 날짜 |
| 통상 | 퇴거 시 동시이행 |
| 특약 | 특약에 따름 |

### 7.2 공제 항목

| 공제 | 내용 |
|------|------|
| 위약금 | 합의한 위약금 |
| 연체 월세 | 미납 월세 |
| 원상복구 | 임차인 과실 부분 |
| 공과금 | 미납 공과금 |

### 7.3 반환 지연 시

| 대응 | 방법 |
|------|------|
| 내용증명 | 반환 촉구 |
| 임차권등기 | 대항력 유지 |
| 지급명령 | 법원 신청 |

## 8. 중도해지 특약

### 8.1 유효한 특약

| 특약 | 효력 |
|------|------|
| 해지 통지 기간 | ✅ 유효 |
| 합리적 위약금 | ✅ 유효 |
| 후임자 구해오면 면제 | ✅ 유효 |

### 8.2 효력 제한 특약

| 특약 | 효력 |
|------|------|
| 과도한 위약금 | ⚠️ 감액 가능 |
| 중도해지 절대 불가 | ⚠️ 효력 제한 |
| 보증금 전액 몰수 | ❌ 무효 가능 |

### 8.3 특약 예시

```
[중도해지 특약]

1. 임차인은 2개월 전 서면 통지로 본 계약을
   해지할 수 있다.
2. 중도해지 시 임차인은 월 임대료 1개월분을
   위약금으로 지급한다.
3. 임차인이 동일 조건의 후임 임차인을 주선한
   경우 위약금을 면제한다.
```

## 10. 자주 묻는 질문

### 10.1 전세도 중도해지 가능?

| 답변 | 설명 |
|------|------|
| ✅ 가능 | 합의 또는 법정사유 |
| 위약금 | 통상 보증금 10% 이내 |
| 대안 | 전전세, 임차권 양도 |

### 10.2 월세 2개월만 살고 나가면?

| 답변 | 설명 |
|------|------|
| 가능 | 해지 가능 |
| 위약금 | 특약 또는 손해배상 |
| 협의 | 후임자 구하면 면제 협의 |

### 10.3 임대인이 급하게 나가라고 하면?

| 답변 | 설명 |
|------|------|
| 거부 가능 | 법정 사유 없으면 |
| 예외 | 2기 연체 등 해지사유 시 |
| 대응 | 법률상담 권장 |
