연말정산은 정해진 일정에 따라 진행돼요. [소득세법 제137조](https://www.law.go.kr/법령/소득세법)에 따른 일정을 정리해 볼게요.

## 연말정산 전체 일정

주요 일정 (2026년 귀속 기준)이에요.

| 시기 | 내용 |
|------|------|
| 1월 15일 | 간소화 서비스 오픈 |
| 1월 15일~17일 | 간소화 자료 수정 요청 기간 |
| 1월 18일 | 수정 자료 반영 |
| 1월 말 | 공제 신고서 회사 제출 |
| 2월 말 | 회사 → 국세청 신고 |
| 3월 급여 | 환급 또는 추가 징수 |

일정별 담당자예요.

| 시기 | 담당 |
|------|------|
| 1월 | 근로자: 자료 조회, 서류 준비 |
| 2월 | 회사: 세금 계산, 국세청 신고 |
| 3월 | 회사: 환급/추징 처리 |

## 간소화 서비스 일정

자료 조회 일정이에요.

| 일정 | 내용 |
|------|------|
| 1월 15일 | 간소화 서비스 오픈 |
| 1월 15~17일 | 자료 확인 및 수정 요청 |
| 1월 18일 | 수정 자료 최종 반영 |
| 1월 20일 이후 | 안정적 조회 권장 |

누락 자료 처리 방법이에요.

| 방법 | 내용 |
|------|------|
| 간소화 수정 요청 | 1월 17일까지 |
| 직접 증빙 제출 | 영수증 등 회사에 제출 |
| 경정청구 | 연말정산 후 5년 이내 |

## 회사 제출 기한

공제 신고서 제출 기한이에요.

| 서류 | 제출 기한 |
|------|----------|
| 소득공제신고서 | 회사 지정 기한 (보통 1월 말) |
| 간소화 자료 PDF | 신고서와 함께 |
| 별도 증빙서류 | 신고서와 함께 |

회사별로 제출 방식이 달라요.

| 회사 유형 | 특징 |
|----------|------|
| 대기업 | 온라인 시스템 사용, 기한 엄격 |
| 중소기업 | 종이 제출 가능, 기한 유동적 |
| 외국계 | 별도 일정 운영 가능 |

## 국세청 신고 기한

회사가 국세청에 신고하는 기한이에요.

| 항목 | 기한 |
|------|------|
| 지급명세서 제출 | 3월 10일 |
| 원천세 신고·납부 | 매월 10일 |

미신고 시 가산세가 부과돼요.

## 환급/추징 시점

일반적인 시점이에요.

| 결과 | 시점 |
|------|------|
| 환급 | 3월 급여에 포함 |
| 추가 납부 | 3월 급여에서 공제 |

회사별로 환급 시점이 다를 수 있어요.

| 회사 | 환급 시점 |
|------|----------|
| 2월 급여 지급 | 2월 환급 가능 |
| 3월 급여 지급 | 3월 환급 |
| 분할 추징 | 3~5월 분할 공제 |

## 기한 후 처리

연말정산 기한이 지났을 때 처리 방법이에요.

| 방법 | 내용 |
|------|------|
| 5월 종합소득세 신고 | 직접 신고 |
| 경정청구 | 5년 이내 |

종합소득세 신고 일정이에요.

| 항목 | 기한 |
|------|------|
| 신고 기간 | 5월 1일~31일 |
| 납부 기한 | 5월 31일 |
| 환급 | 신고 후 2~3개월 |

경정청구 안내예요.

| 항목 | 내용 |
|------|------|
| 기한 | 법정신고기한 5년 이내 |
| 방법 | 홈택스 온라인 |
| 용도 | 누락 공제 추가 신청 |

## 주의사항

간소화 자료 초기 오류에 주의하세요. 1월 15~17일에는 일부 자료가 누락될 수 있어서 1월 20일 이후에 최종 확인하는 게 좋아요.

회사 기한을 꼭 지켜야 해요. 기한 내 미제출 시 수정이 어려울 수 있어요.

이직자는 전 직장 원천징수영수증을 미리 발급받아야 해요.
