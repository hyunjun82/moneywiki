# 연말정산 신고 기한

연말정산은 정해진 일정에 따라 진행됩니다. [소득세법 제137조](https://www.law.go.kr/법령/소득세법)에 따른 일정을 정리합니다.

## 1. 연말정산 전체 일정

### 1.1 주요 일정 (2026년 귀속 기준)

| 시기 | 내용 |
|------|------|
| **1월 15일** | 간소화 서비스 오픈 |
| **1월 15일~17일** | 간소화 자료 수정 요청 기간 |
| **1월 18일** | 수정 자료 반영 |
| **1월 말** | 공제 신고서 회사 제출 |
| **2월 말** | 회사 → 국세청 신고 |
| **3월 급여** | 환급 또는 추가 징수 |

### 1.2 일정별 담당자

| 시기 | 담당 |
|------|------|
| 1월 | **근로자**: 자료 조회, 서류 준비 |
| 2월 | **회사**: 세금 계산, 국세청 신고 |
| 3월 | **회사**: 환급/추징 처리 |

## 2. 간소화 서비스 일정

### 2.1 자료 조회

| 일정 | 내용 |
|------|------|
| 1월 15일 | 간소화 서비스 오픈 |
| 1월 15~17일 | 자료 확인 및 수정 요청 |
| 1월 18일 | 수정 자료 최종 반영 |
| 1월 20일 이후 | 안정적 조회 권장 |

### 2.2 누락 자료 처리

| 방법 | 내용 |
|------|------|
| 간소화 수정 요청 | 1월 17일까지 |
| 직접 증빙 제출 | 영수증 등 회사에 제출 |
| 경정청구 | 연말정산 후 5년 이내 |

## 3. 회사 제출 기한

### 3.1 공제 신고서

| 서류 | 제출 기한 |
|------|----------|
| 소득공제신고서 | **회사 지정 기한** (보통 1월 말) |
| 간소화 자료 PDF | 신고서와 함께 |
| 별도 증빙서류 | 신고서와 함께 |

### 3.2 회사별 차이

| 회사 유형 | 특징 |
|----------|------|
| 대기업 | 온라인 시스템 사용, 기한 엄격 |
| 중소기업 | 종이 제출 가능, 기한 유동적 |
| 외국계 | 별도 일정 운영 가능 |

## 4. 국세청 신고 기한

### 4.1 회사 신고

| 항목 | 기한 |
|------|------|
| 지급명세서 제출 | **3월 10일** |
| 원천세 신고·납부 | 매월 10일 |

### 4.2 미신고 시

| 상황 | 불이익 |
|------|--------|
| 지급명세서 미제출 | 가산세 부과 |
| 원천세 미납부 | 가산세 부과 |

## 5. 환급/추징 시점

### 5.1 일반적인 시점

| 결과 | 시점 |
|------|------|
| 환급 | **3월 급여**에 포함 |
| 추가 납부 | **3월 급여**에서 공제 |

### 5.2 회사별 차이

| 회사 | 환급 시점 |
|------|----------|
| 2월 급여 지급 | 2월 환급 가능 |
| 3월 급여 지급 | 3월 환급 |
| 분할 추징 | 3~5월 분할 공제 |

## 6. 기한 후 처리

### 6.1 연말정산 기한 경과

| 방법 | 내용 |
|------|------|
| 5월 종합소득세 신고 | 직접 신고 |
| 경정청구 | 5년 이내 |

### 6.2 종합소득세 신고

| 항목 | 기한 |
|------|------|
| 신고 기간 | **5월 1일~31일** |
| 납부 기한 | **5월 31일** |
| 환급 | 신고 후 2~3개월 |

### 6.3 경정청구

| 항목 | 내용 |
|------|------|
| 기한 | 법정신고기한 **5년 이내** |
| 방법 | 홈택스 온라인 |
| 용도 | 누락 공제 추가 신청 |

## 7. 주의사항

### 7.1 간소화 자료 초기 오류

| 시기 | 주의 |
|------|------|
| 1월 15~17일 | 일부 자료 누락 가능 |
| 1월 18일 이후 | 안정적 조회 |
| 1월 20일 이후 | 최종 확인 권장 |

### 7.2 회사 기한 준수

| 상황 | 결과 |
|------|------|
| 기한 내 미제출 | 회사 정책에 따름 |
| 늦은 제출 | 수정 어려울 수 있음 |

### 7.3 이직자 특별 주의

| 상황 | 처리 |
|------|------|
| 전 직장 원천징수영수증 | 미리 발급받기 |
| 새 직장 기한 확인 | 일찍 제출 |
