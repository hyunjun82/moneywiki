# 오피스텔 임대차

오피스텔은 주거용으로 사용하면 주택임대차보호법이 적용됩니다. 보호받기 위한 조건과 주의사항을 알아봅니다.

## 1. 오피스텔 임대차 기본

### 1.1 오피스텔 유형

| 유형 | 용도 | 법 적용 |
|------|------|---------|
| 주거용 | 실제 거주 | **주택임대차보호법** |
| 업무용 | 사무실 사용 | 상가임대차보호법 또는 민법 |
| 겸용 | 주거+업무 | 주된 용도에 따라 |

### 1.2 주거용 판단 기준

| 기준 | 내용 |
|------|------|
| 실제 사용 | 실제로 거주하는지 |
| 전입신고 | 전입신고 완료 여부 |
| 시설 | 주거 시설 구비 (취사, 화장실) |

### 1.3 주택임대차보호법 적용

| 항목 | 적용 여부 |
|------|----------|
| 대항력 | ✅ 적용 |
| 우선변제권 | ✅ 적용 |
| 최우선변제권 | ✅ 적용 |
| 계약갱신청구권 | ✅ 적용 |
| 묵시적 갱신 | ✅ 적용 |
| 전월세상한제 | ✅ 적용 |

## 2. 계약 전 확인사항

### 2.1 건축물대장 확인

| 확인 | 내용 |
|------|------|
| 용도 | 오피스텔, 근린생활시설 등 |
| 위반건축물 | 위반 여부 확인 |
| 전용면적 | 실제 면적 확인 |

### 2.2 등기부등본 확인

| 확인 | 내용 |
|------|------|
| 소유자 | 임대인과 일치 |
| 근저당 | 채권최고액 확인 |
| 가압류 | 권리 침해 여부 |

### 2.3 전입신고 가능 여부

| 확인 | 방법 |
|------|------|
| 사전 확인 | 관할 주민센터 문의 |
| 건축물대장 | 용도 확인 |
| 위반건축물 | 위반 시 전입 불가 |

## 3. 대항력 확보

### 3.1 대항력 요건

| 요건 | 내용 |
|------|------|
| 인도 | 실제 입주 |
| 전입신고 | 주민센터에 전입신고 |
| 확정일자 | 확정일자 받기 (우선변제권) |

### 3.2 전입신고 절차

| 절차 | 내용 |
|------|------|
| 방문 | 관할 주민센터 |
| 온라인 | 정부24 (gov.kr) |
| 준비물 | 신분증, 임대차계약서 |

### 3.3 대항력 발생 시점

| 시점 | 내용 |
|------|------|
| 전입신고 | 신고일 **다음날 0시** |
| 주의 | 근저당보다 선순위 필요 |

## 4. 우선변제권

### 4.1 요건

| 요건 | 내용 |
|------|------|
| 대항요건 | 인도 + 전입신고 |
| 확정일자 | 확정일자 받기 |
| 순위 | 확정일자 기준 |

### 4.2 확정일자 받기

| 방법 | 내용 |
|------|------|
| 주민센터 | 계약서 지참 |
| 등기소 | 등기소에서 가능 |
| 공증사무소 | 공증 시 포함 |

### 4.3 소액임차인 최우선변제

| 지역 | 보증금 기준 | 최우선변제금 |
|------|-----------|------------|
| 서울 | 1.65억 이하 | 5,500만원 |
| 수도권 과밀억제권역 | 1.45억 이하 | 4,800만원 |
| 광역시 등 | 8,500만원 이하 | 2,800만원 |
| 그 외 | 7,500만원 이하 | 2,500만원 |

## 5. 계약갱신청구권

### 5.1 적용 여부

| 유형 | 적용 |
|------|------|
| 주거용 오피스텔 | ✅ 적용 |
| 업무용 오피스텔 | ❌ 미적용 |

### 5.2 행사 방법

| 항목 | 내용 |
|------|------|
| 기간 | 계약 종료 6개월~1개월 전 |
| 방법 | 임대인에게 갱신 요청 |
| 효과 | 동일 조건 2년 연장 |
| 횟수 | 1회 한정 |

### 5.3 갱신 거절 사유

| 사유 | 설명 |
|------|------|
| 2기 연체 | 월세 2회 이상 연체 |
| 무단 전대 | 임대인 동의 없이 전대 |
| 고의 훼손 | 시설물 훼손 |
| 실거주 | 임대인 본인 또는 직계존비속 실거주 |

## 6. 전월세신고

### 6.1 신고 의무

| 지역 | 신고 |
|------|------|
| 신고 의무 지역 | **30일 내 신고** |
| 서울 전역 | ✅ 해당 |
| 수도권 대부분 | ✅ 해당 |

### 6.2 신고 대상

| 조건 | 신고 |
|------|------|
| 보증금 6천만원 초과 | ✅ 의무 |
| 월세 30만원 초과 | ✅ 의무 |
| 그 외 | 선택적 |

### 6.3 신고 방법

| 방법 | 내용 |
|------|------|
| 온라인 | 부동산거래관리시스템 |
| 방문 | 주민센터 |
| 공동신고 | 임대인·임차인 공동 |

## 7. 오피스텔 특수 사항

### 7.1 관리비

| 확인 | 내용 |
|------|------|
| 관리비 구성 | 포함 항목 확인 |
| 개별 계량 | 전기, 수도, 가스 |
| 공용 관리비 | 관리사무소 확인 |

### 7.2 주차

| 확인 | 내용 |
|------|------|
| 주차 가능 | 세대당 배정 확인 |
| 추가 비용 | 주차비 별도 여부 |
| 기계식 주차 | 차량 규격 제한 |

### 7.3 시설물

| 확인 | 내용 |
|------|------|
| 빌트인 가전 | 옵션 상태 확인 |
| 냉난방 | 개별 난방 확인 |
| 인터넷 | 단지 인터넷 확인 |

## 8. 주의사항

### 8.1 전입신고 불가 위험

| 상황 | 위험 |
|------|------|
| 위반건축물 | 전입신고 불가 |
| 업무용 전용 | 주거용 사용 불가 |
| 불법 용도변경 | 권리 보호 안 됨 |

### 8.2 근저당 확인

| 확인 | 내용 |
|------|------|
| 채권최고액 | 보증금 안전 여부 |
| 선순위 채권 | 경매 시 배당 순위 |
| 안전 기준 | 시세 70% 이하 |

### 8.3 전세보증보험

| 가입 | 내용 |
|------|------|
| 가능 | 주거용 오피스텔 가능 |
| 조건 | HUG, SGI 조건 충족 |
| 권장 | **강력 권장** |

## 9. 계약서 작성

### 9.1 필수 기재 사항

| 항목 | 내용 |
|------|------|
| 목적물 | 호수, 전용면적 |
| 용도 | **주거용** 명시 |
| 보증금/월세 | 금액 명시 |
| 계약 기간 | 2년 이상 권장 |

### 9.2 특약 예시

```
[오피스텔 임대차 특약]

1. 본 오피스텔은 주거용으로 사용하며,
   임대인은 임차인의 전입신고에 협조한다.
2. 빌트인 가전(에어컨, 세탁기, 냉장고)은
   현 상태로 제공하며, 노후 고장 시
   임대인이 수리한다.
3. 관리비는 월 ○만원이며, 전기·가스·수도는
   임차인이 별도 부담한다.
```

### 9.3 인수인계서

| 확인 | 기재 |
|------|------|
| 시설물 상태 | 가전, 가구 상태 |
| 하자 | 기존 하자 기록 |
| 사진 첨부 | 증빙 사진 |

## 10. 자주 묻는 질문

### 10.1 전입신고 안 하면?

| 답변 | 설명 |
|------|------|
| 대항력 없음 | 경매 시 보호 안 됨 |
| 우선변제 없음 | 배당 순위 없음 |
| 반드시 | **전입신고 필수** |

### 10.2 도시형생활주택과 차이?

| 구분 | 오피스텔 | 도시형생활주택 |
|------|---------|--------------|
| 용도 | 업무시설 | 주택 |
| 전입신고 | 주거용만 가능 | 자동 가능 |
| 보호 | 주거용만 적용 | 당연 적용 |

### 10.3 오피스텔도 중개수수료?

| 답변 | 설명 |
|------|------|
| 적용 | 주거용은 주택 요율 |
| 요율 | 거래금액에 따라 0.3~0.5% |
