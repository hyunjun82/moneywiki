# 연금저축

연금저축은 노후 대비와 세금 혜택을 동시에 받을 수 있는 금융상품입니다. 종류와 혜택을 정리합니다.

## 1. 연금저축 종류

### 1.1 상품 유형

| 종류 | 운용 주체 | 특징 |
|------|----------|------|
| 연금저축펀드 | 자산운용사 | 펀드 투자 |
| 연금저축보험 | 보험사 | 원금 보장 |
| 연금저축신탁 | 은행 | 원금 보장 (판매 중단) |

### 1.2 상품별 비교

| 구분 | 연금저축펀드 | 연금저축보험 |
|------|------------|-------------|
| 원금 보장 | ❌ | ✅ |
| 수익률 | 변동 (높은 수익 가능) | 안정적 (낮은 수익) |
| 이전 | 타사 이전 가능 | 가능 |
| 적합 대상 | 적극적 투자자 | 안정 추구자 |

## 2. 가입 조건

### 2.1 가입 자격

| 조건 | 내용 |
|------|------|
| 나이 | 제한 없음 |
| 소득 | 제한 없음 |
| 가입 개수 | 제한 없음 (합산 한도) |

### 2.2 납입 한도

| 구분 | 한도 |
|------|------|
| 연간 납입 한도 | **1,800만원** |
| 세액공제 한도 (연금저축) | **600만원** |
| 세액공제 한도 (IRP 포함) | **900만원** |

## 3. 세액공제

### 3.1 공제율

| 총급여 | 공제율 |
|--------|--------|
| 5,500만원 이하 | **16.5%** |
| 5,500만원 초과 | **13.2%** |

### 3.2 최대 공제액

| 납입액 | 공제율 16.5% | 공제율 13.2% |
|--------|-------------|-------------|
| 400만원 | 66만원 | 52.8만원 |
| 600만원 | **99만원** | **79.2만원** |

### 3.3 IRP 포함 공제

| 상품 | 한도 | 공제율 16.5% |
|------|------|-------------|
| 연금저축만 | 600만원 | 99만원 |
| 연금저축 + IRP | **900만원** | **148.5만원** |

## 4. 연금 수령

### 4.1 수령 조건

| 조건 | 기준 |
|------|------|
| 수령 개시 연령 | **55세 이후** |
| 가입 기간 | **5년 이상** |
| 수령 기간 | **10년 이상** 권장 |

### 4.2 연금소득세

| 수령 나이 | 세율 |
|----------|------|
| 55~69세 | **5.5%** |
| 70~79세 | **4.4%** |
| 80세 이상 | **3.3%** |

### 4.3 연간 수령 한도

| 항목 | 내용 |
|------|------|
| 연금 수령 한도 | 연간 **1,200만원** 이하 |
| 초과 시 | 종합소득 합산 과세 |

## 5. 중도 해지

### 5.1 해지 시 과세

| 구분 | 세율 |
|------|------|
| 기타소득세 | **16.5%** |
| 부정 해지 | 세액공제액 추징 |

### 5.2 해지 예외 (저율 과세)

| 사유 | 세율 |
|------|------|
| 천재지변 | 3.3~5.5% |
| 사망 | 3.3~5.5% |
| 해외 이주 | 3.3~5.5% |
| 파산 | 3.3~5.5% |

## 6. 연금저축펀드

### 6.1 특징

| 항목 | 내용 |
|------|------|
| 투자 대상 | 국내외 주식, 채권, ETF |
| 리밸런싱 | 자유로운 변경 가능 |
| 수수료 | 펀드별 상이 |

### 6.2 투자 전략

| 연령대 | 권장 포트폴리오 |
|--------|----------------|
| 20~30대 | 주식형 70~80% |
| 40~50대 | 혼합형 50~60% |
| 60대 이상 | 채권형 60~70% |

### 6.3 주요 ETF

| 유형 | 예시 |
|------|------|
| 국내 주식 | KODEX 200 |
| 해외 주식 | TIGER 미국S&P500 |
| 채권 | KOSEF 국고채 |
| TDF | 타겟데이트펀드 |

## 7. 연금저축보험

### 7.1 특징

| 항목 | 내용 |
|------|------|
| 원금 보장 | ✅ |
| 공시이율 | 연 2~3% 수준 |
| 사업비 | 초기 납입금의 일부 |

### 7.2 유의사항

| 주의 | 내용 |
|------|------|
| 초기 사업비 | 초기 몇 년간 수수료 높음 |
| 해지 환급금 | 초기 해지 시 원금 손실 가능 |
| 이전 | 타사 연금저축으로 이전 가능 |

## 8. IRP와의 비교

### 8.1 차이점

| 구분 | 연금저축 | IRP |
|------|---------|-----|
| 퇴직금 수령 | ❌ | ✅ |
| 세액공제 한도 | 600만원 | 900만원 (합산) |
| 중도 인출 | 불가 | 조건부 가능 |

### 8.2 조합 전략

| 전략 | 내용 |
|------|------|
| 연금저축 600만원 | 기본 세액공제 |
| IRP 추가 300만원 | 추가 세액공제 |
| **합계** | **900만원** |

## 9. 주의사항

### 9.1 장기 유지

| 주의 | 내용 |
|------|------|
| 최소 5년 | 연금 수령 조건 |
| 55세 이후 | 수령 개시 |
| 중도 해지 불이익 | 세금 추징 |

### 9.2 세액공제 확인

| 확인 | 내용 |
|------|------|
| 공제 한도 | 600만원 (IRP 포함 900만원) |
| 다른 상품 합산 | 한도 초과 시 공제 불가 |
