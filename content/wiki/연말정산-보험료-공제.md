# 연말정산 보험료 공제

보장성보험료 세액공제를 정리합니다.

## 1. 공제 개요

| 항목 | 내용 |
|------|------|
| 공제 유형 | 세액공제 |
| 공제율 | **12%** |
| 기본 한도 | **100만원** |

## 2. 공제 대상

| 보험 종류 | 공제 |
|----------|------|
| 보장성보험 | ✅ |
| 장애인전용보험 | ✅ (별도 100만원) |
| 저축성보험 | ❌ |
| 변액보험 | 보장 부분만 ✅ |

## 3. 보장성보험 종류

| 종류 | 예시 |
|------|------|
| 생명보험 | 종신보험, 정기보험 |
| 손해보험 | 자동차보험, 화재보험 |
| 상해보험 | 실손보험, 상해보험 |

## 4. 공제 한도

| 구분 | 한도 | 공제액 |
|------|------|--------|
| 일반 | 100만원 | 최대 12만원 |
| 장애인전용 | 100만원 | 최대 12만원 |
| 합계 | 200만원 | 최대 24만원 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 피보험자 | 기본공제 대상자여야 함 |
| 계약자 | 본인 또는 기본공제 대상자 |
