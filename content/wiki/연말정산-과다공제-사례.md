연말정산 많이 돌려받고 싶은 마음은 알지만, 잘못하면 나중에 더 큰 돈을 토해내야 해요.

✨핵심 요약!

1. 소득 있는 부양가족 공제가 가장 흔한 실수
2. 중복 공제, 허위 증빙 주의
3. 적발 시 가산세 포함 추징

## 목차

## 과다공제란

받을 수 없는 공제를 받거나 한도를 초과해서 받는 거예요.

국세청이 자료를 대조하면 다 나와요. 적발되면 세금 + 가산세를 내야 해요.

*과다공제 유형

| 유형 | 예시 |
|------|------|
| 자격 미달 공제 | 소득 초과 부양가족 |
| 중복 공제 | 부부가 같은 자녀 공제 |
| 허위 증빙 | 가짜 기부금 영수증 |
| 한도 초과 | 공제 한도 초과 신청 |

## 흔한 과다공제 사례

많이 적발되는 사례들이에요.

*부양가족 관련

| 사례 | 문제점 |
|------|--------|
| 소득 있는 배우자 공제 | 연소득 100만원 초과 시 불가 |
| 아르바이트하는 자녀 공제 | 총급여 500만원 초과 시 불가 |
| 형제자매 중복 공제 | 한 명만 공제 가능 |
| 별거 부모 공제 | 실제 부양 입증 필요 |

소득이 있는 가족을 공제받는 게 가장 흔한 실수예요.

*의료비·교육비 관련

| 사례 | 문제점 |
|------|--------|
| 실손보험 수령 후 의료비 공제 | 보험금 뺀 금액만 가능 |
| 미취학 자녀 학원비 | 유치원·체육시설만 가능 |
| 해외 대학 학원비 | 정규 학위과정만 가능 |

*기부금 관련

| 사례 | 문제점 |
|------|--------|
| 허위 기부금 영수증 | 사기죄 해당 가능 |
| 종교단체 한도 초과 | 소득 10% 한도 |

## 적발 방법

국세청이 자료를 대조해서 찾아내요.

| 자료 | 대조 내용 |
|------|----------|
| 소득자료 | 부양가족 소득 확인 |
| 의료비 자료 | 실손보험 수령 내역 |
| 기부금 자료 | 기부 단체 신고 내역 |

건강보험공단, 보험회사, 기부단체 자료를 국세청이 다 받아요.

## 추징 절차

과다공제가 확인되면 이렇게 진행돼요.

| 단계 | 내용 |
|------|------|
| 1단계 | 국세청 소명 요청 |
| 2단계 | 소명 자료 제출 |
| 3단계 | 추징 결정 |
| 4단계 | 세금 + 가산세 납부 |

소명 기회가 주어지니까 증빙 자료가 있으면 제출하세요.

## 가산세

과다공제 추징 시 가산세가 붙어요.

| 유형 | 가산세율 |
|------|---------|
| 일반 과소신고 | 10% |
| 부당 과소신고 | 40% |
| 납부지연 | 0.022%/일 |

허위 증빙 제출은 부당 과소신고로 40% 가산세가 붙어요.

## 예방 방법

과다공제를 피하려면 확인이 필요해요.

- 부양가족 소득 확인 (연 100만원, 총급여 500만원 기준)
- 실손보험 수령액 확인
- 공제 한도 확인
- 증빙 서류 보관

잘 모르겠으면 간소화 서비스 자료만 공제받는 게 안전해요.
