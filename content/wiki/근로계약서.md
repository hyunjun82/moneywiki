# 근로계약서

근로계약서는 근로조건을 정하는 서면 계약입니다. [근로기준법](https://www.law.go.kr/법령/근로기준법)에 따른 작성 의무와 내용을 정리합니다.

## 1. 근로계약서 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 근로계약 | 근로 제공, 임금 지급 약정 |
| 서면 명시 | 법적 의무 |
| 교부 | 근로자에게 1부 교부 |

### 1.2 작성 의무

| 대상 | 의무 |
|------|------|
| 정규직 | ✅ |
| 계약직 | ✅ |
| 파트타임 | ✅ |
| 아르바이트 | ✅ |
| 일용직 | ✅ |

## 2. 필수 기재사항

### 2.1 모든 근로자 공통

| 항목 | 내용 |
|------|------|
| 임금 | 금액, 지급방법, 지급일 |
| 근로시간 | 소정근로시간, 시작·종료 시간 |
| 휴일 | 주휴일, 공휴일 |
| 연차휴가 | 연차 부여 기준 |
| 업무내용 | 담당 업무 |
| 취업장소 | 근무 장소 |

### 2.2 기간제·단시간 추가 사항

| 항목 | 내용 |
|------|------|
| 계약기간 | 시작일, 종료일 |
| 근로일 | 주 근무 요일 |
| 근로시간 | 주당, 일당 근로시간 |

### 2.3 연소자 (18세 미만) 추가

| 항목 | 내용 |
|------|------|
| 친권자 동의서 | 법정대리인 동의 |
| 연령 증명 | 가족관계증명서 |

## 3. 임금 명시

### 3.1 기재 내용

| 항목 | 예시 |
|------|------|
| 기본급 | 월 2,500,000원 |
| 수당 | 직책수당, 식대 등 |
| 지급일 | 매월 10일 |
| 지급방법 | 은행 계좌이체 |

### 3.2 시급제 기재

| 항목 | 예시 |
|------|------|
| 시급 | 10,030원 |
| 주휴수당 | 포함/별도 명시 |
| 계산 방법 | 근무시간 × 시급 |

## 4. 근로시간 명시

### 4.1 표준 기재

| 항목 | 예시 |
|------|------|
| 소정근로시간 | 주 40시간 |
| 근무시간 | 09:00~18:00 |
| 휴게시간 | 12:00~13:00 (1시간) |
| 근무일 | 월~금 |

### 4.2 교대제 기재

| 항목 | 예시 |
|------|------|
| 교대 유형 | 2교대, 3교대 |
| 각 근무 시간 | 명시 |

## 5. 계약 기간

### 5.1 기간제 근로자

| 항목 | 내용 |
|------|------|
| 최대 기간 | 원칙 **2년** |
| 초과 시 | 무기계약 전환 |

### 5.2 기재 예시

| 항목 | 예시 |
|------|------|
| 계약기간 | 2026.01.01~2026.12.31 |
| 갱신 조건 | 명시 가능 |

## 6. 작성·교부

### 6.1 작성 시기

| 시기 | 권장 |
|------|------|
| 입사 전 | 권장 |
| 입사일 | 늦어도 입사일 |

### 6.2 교부 방법

| 방법 | 효력 |
|------|------|
| 서면 교부 | ✅ |
| 이메일 | ✅ (근로자 요청 시) |
| 문자 | ✅ (근로자 요청 시) |

### 6.3 보관

| 주체 | 보관 |
|------|------|
| 사용자 | 3년 보관 의무 |
| 근로자 | 1부 보관 |

|------|--------|
