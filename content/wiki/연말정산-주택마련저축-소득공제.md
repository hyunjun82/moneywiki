# 주택마련저축 소득공제

## 기본 개념
무주택 근로자가 주택 마련을 위해 저축하는 금액에 대한 소득공제입니다.

## 공제 대상 저축

### 주택청약종합저축
- 주택도시기금 주택청약종합저축
- 매월 2만원 ~ 50만원 납입 가능
- **공제율**: 40%
- **한도**: 연 **240만원** (납입액 600만원 기준)

### 청년 우대형 청약통장
- 만 19세 ~ 34세 무주택자
- 소득 요건: 연 3천만원 이하
- **공제율**: 40%
- **한도**: 연 **300만원** (납입액 750만원 기준)

## 가입 자격

### 공통 요건
- **무주택 세대주** (또는 세대원)
- **총급여 7천만원 이하** (종합소득 6천만원 이하)

### 세대주 요건
- 세대주가 주택 관련 공제 받지 않는 경우
- 세대원도 공제 가능 (2명 이상 불가)

### 무주택 판단
- 본인 및 배우자 무주택
- 주민등록표상 동거 가족 무주택

## 공제 한도 및 공제율

### 일반 주택청약종합저축
```
연 납입액: 600만원 (월 50만원)
공제율: 40%
최대 공제액: 240만원
실제 세금 절감: 약 36만원 (15% 세율 기준)
```

### 청년 우대형 (19~34세)
```
연 납입액: 750만원
공제율: 40%
최대 공제액: 300만원
실제 세금 절감: 약 45만원 (15% 세율 기준)
```

## 실무 적용 사례

### 사례 1: 일반 직장인 (30세, 무주택)
```
월 납입: 50만원
연 납입: 600만원
소득공제: 600만원 × 40% = 240만원
예상 절세: 240만원 × 15% = 36만원
```

### 사례 2: 청년 우대형 (25세)
```
월 납입: 50만원
연 납입: 600만원
소득공제: 600만원 × 40% = 240만원
(750만원까지 가능하나 600만원만 납입)
예상 절세: 36만원
```

### 사례 3: 맞벌이 부부
```
남편: 유주택 → 공제 불가
아내: 무주택 세대원 → 공제 가능
(단, 남편이 주택 관련 공제 받지 않는 경우만)
```

## 주의사항

### 중복 공제 제한
- 주택청약종합저축 + 주택임차차입금 → 합산 **300만원** 한도
- 주택청약종합저축 + 장기주택저당차입금 → 각각 별도 한도

### 소득 요건
- 총급여 7천만원 초과 시 공제 불가
- 종합소득금액 6천만원 초과 시 불가

### 무주택 확인
- 연말정산 시점 기준
- 주택 취득 시 해당 연도는 공제 가능
- 다음 연도부터 공제 불가

### 해지 시 추징
- 다음 연도 3월 말 이전 해지 시 공제받은 세액 추징
- 주택 구입 목적 해지는 추징 제외

## 공제 신청 방법

### 연말정산 시
1. 주택청약종합저축 납입증명서 발급 (은행)
2. 무주택 확인서 (필요 시)
3. 회사에 제출
4. 간소화 서비스에서 자동 조회 가능

### 준비 서류
- 주택청약종합저축 통장 사본
- 납입증명서 (은행 발급)
- 주민등록등본 (무주택 확인)

## 다른 주택 공제와의 관계

### 주택임차차입금 원리금 상환액
- 전세자금 대출 원리금 공제
- 합산 한도: **300만원**

### 장기주택저당차입금 이자상환액
- 주택 구입 대출 이자 공제
- 별도 한도: **300~1,800만원**

### 월세 세액공제
- 월세액 세액공제
- 중복 불가 (하나만 선택)

## 실무 팁

### 최대 납입 활용
- 가능하면 월 50만원 납입 권장
- 소득공제 효과 최대화

### 청년 우대형 가입
- 19~34세면 우대형 가입 필수
- 300만원 한도 활용

### 해지 시기 주의
- 다음 연도 4월 이후 해지 권장
- 추징 방지

### 맞벌이 전략
- 한 사람이 주택 관련 공제 받으면
- 다른 사람이 청약저축 공제 받기
