# 묵시적갱신

묵시적갱신은 별도 의사표시 없이 임대차가 자동으로 연장되는 제도입니다. 임차인에게 유연한 거주 기간을 보장합니다.

## 1. 묵시적갱신이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 묵시적갱신 | 별도 통지 없이 **자동 갱신** |
| 조건 | 양측 모두 갱신 거절 안 함 |
| 근거 | 주택임대차보호법 제6조 |

### 1.2 발생 조건

| 조건 | 내용 |
|------|------|
| 임대인 | 만료 6개월~1개월 전 갱신 거절 **없음** |
| 임차인 | 만료 1개월 전까지 이사 의사 표시 **없음** |

## 2. 성립 요건

### 2.1 임대인 측

| 요건 | 내용 |
|------|------|
| 갱신 거절 기간 | 만료 **6개월~1개월 전** |
| 갱신 거절 없으면 | 묵시적 갱신 성립 |

### 2.2 임차인 측

| 요건 | 내용 |
|------|------|
| 이사 통보 기간 | 만료 **1개월 전**까지 |
| 이사 통보 없으면 | 묵시적 갱신 성립 |

### 2.3 성립 시점

| 시점 | 내용 |
|------|------|
| 기존 계약 만료일 | 묵시적 갱신 효력 발생 |
| 갱신 기간 | 만료일 다음 날부터 2년 |

## 3. 묵시적갱신 효과

### 3.1 갱신 조건

| 항목 | 내용 |
|------|------|
| 보증금 | **동일** |
| 월세 | **동일** |
| 기간 | **2년** |
| 기타 조건 | 동일 |

### 3.2 계약 기간

| 항목 | 기간 |
|------|------|
| 갱신 기간 | **2년** |
| 법정 기간 | 기간 정함 없어도 2년 |

### 3.3 대항력·우선변제권

| 권리 | 효력 |
|------|------|
| 대항력 | 유지 |
| 우선변제권 | 유지 |
| 확정일자 | 기존 것 유효 |

## 4. 임차인 해지권

### 4.1 3개월 해지

| 항목 | 내용 |
|------|------|
| 해지 방법 | **3개월 전** 통지 |
| 적용 시기 | 묵시적 갱신 후 **언제든** |
| 효력 | 통지 후 3개월 뒤 해지 |

### 4.2 해지 통지 방법

| 방법 | 권장 |
|------|------|
| 내용증명 | ⭐ 권장 |
| 문자·카톡 | ○ |
| 구두 | △ 증거 어려움 |

### 4.3 해지 예시

**묵시적 갱신 후 이사 원할 때**

| 날짜 | 사건 |
|------|------|
| 1월 15일 | 임대인에게 해지 통지 |
| 4월 15일 | 해지 효력 발생 |
| 4월 15일까지 | 보증금 반환, 이사 |

## 5. 임대인 해지권

### 5.1 임대인의 해지

| 항목 | 내용 |
|------|------|
| 해지 가능 여부 | **불가** |
| 이유 | 임대인은 일방 해지 불가 |
| 예외 | 임차인 귀책사유 시 해지 |

### 5.2 임대인이 해지 가능한 경우

| 사유 | 해지 가능 |
|------|----------|
| 2기 차임 연체 | ✅ |
| 임차인 무단 전대 | ✅ |
| 임차인 고의 파손 | ✅ |
| 기타 중대 위반 | ✅ |

## 6. 계약갱신청구권과 비교

### 6.1 차이점

| 구분 | 묵시적 갱신 | 계약갱신청구권 |
|------|-----------|--------------|
| 방식 | **자동** | 임차인 **청구** |
| 조건 | 양측 갱신거절 없음 | 임차인 행사 |
| 임대인 거절 | 못함 (자동 성립) | 정당 사유 시 가능 |
| 임차인 해지 | **3개월** 전 통지 | 어려움 |
| 5% 상한 | ⚠️ 미적용 (논란) | ✅ 적용 |

### 6.2 선택 기준

| 상황 | 권장 |
|------|------|
| 유연한 퇴거 원함 | 묵시적 갱신 |
| 5% 상한 적용 원함 | 계약갱신청구권 |
| 4년 보장 원함 | 계약갱신청구권 |

### 6.3 중복 사용

| 상황 | 가능 여부 |
|------|----------|
| 계약갱신청구권 후 묵시적 갱신 | ✅ 가능 |
| 묵시적 갱신 후 계약갱신청구권 | ⚠️ 계수 논란 |

## 7. 전월세상한제 적용

### 7.1 논란

| 견해 | 내용 |
|------|------|
| 적용 견해 | 갱신이므로 적용 |
| 미적용 견해 | 법률상 명시 없음 |
| 실무 | 합의로 결정 |

### 7.2 보증금 인상 시

| 상황 | 조치 |
|------|------|
| 동일 조건 유지 | 묵시적 갱신 |
| 인상 협의 원할 때 | 별도 합의 필요 |

## 8. 실무 상황

### 8.1 묵시적 갱신 후 매매

| 상황 | 효과 |
|------|------|
| 주택 매매 | 새 집주인에게 대항 |
| 계약 유지 | 2년간 거주 가능 |
| 임차인 해지 | 3개월 전 통지로 가능 |

### 8.2 묵시적 갱신 후 재갱신

| 상황 | 처리 |
|------|------|
| 2년 후 다시 | 또 묵시적 갱신 가능 |
| 계속 거주 | 반복 묵시적 갱신 |

### 8.3 확정일자

| 상황 | 필요 여부 |
|------|----------|
| 묵시적 갱신 | 기존 확정일자 유효 |
| 새로 받을 필요 | ❌ 없음 |

## 9. 주의사항

### 9.1 갱신 거절 시점

| 주의 | 내용 |
|------|------|
| 임대인 | 6개월~1개월 전 통보해야 |
| 늦은 통보 | 묵시적 갱신 성립 |

### 9.2 임차인 이사 의사

| 주의 | 내용 |
|------|------|
| 통보 기한 | 만료 1개월 전까지 |
| 늦은 통보 | 묵시적 갱신 성립 |

### 9.3 보증금 반환

| 상황 | 처리 |
|------|------|
| 묵시적 갱신 후 퇴거 | 3개월 전 통지 후 반환 |
| 동시이행 | 보증금↔주택 인도 동시 |

## 10. 자주 묻는 질문

### 10.1 묵시적 갱신 후 바로 퇴거 가능?

| 답변 | 설명 |
|------|------|
| 3개월 | 통지 후 3개월 대기 |
| 즉시 불가 | 기다려야 함 |

### 10.2 임대인이 3개월 전 통지 안 하면?

| 답변 | 설명 |
|------|------|
| 자동 갱신 | 묵시적 갱신 성립 |
| 임대인 | 강제 퇴거 불가 |

### 10.3 묵시적 갱신 시 계약서 새로 써야?

| 답변 | 설명 |
|------|------|
| ❌ 불필요 | 기존 계약서 유효 |
| 선택 | 원하면 새로 작성 가능 |
