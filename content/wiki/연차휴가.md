# 연차휴가

연차휴가는 근로자에게 보장되는 유급휴가입니다. 발생 기준과 수당 계산을 정리합니다.

## 1. 연차휴가 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 근로자의 유급휴가 |
| 법적 근거 | 근로기준법 제60조 |
| 적용 대상 | 상시 5인 이상 사업장 |

### 1.2 연차 발생 조건

| 조건 | 연차 |
|------|------|
| 1년 80% 이상 출근 | **15일** |
| 1년 미만 근무 | 월 **1일** |

## 2. 연차 일수

### 2.1 입사 1년 미만

| 조건 | 연차 |
|------|------|
| 1개월 개근 시 | **1일** 발생 |
| 1년간 최대 | **11일** |

### 2.2 입사 1년 이상

| 조건 | 연차 |
|------|------|
| 80% 이상 출근 | **15일** |
| 80% 미만 출근 | 0일 |

### 2.3 장기근속 가산

| 근속년수 | 연차일수 |
|----------|----------|
| 1~2년 | 15일 |
| 3~4년 | 16일 |
| 5~6년 | 17일 |
| 7~8년 | 18일 |
| 9~10년 | 19일 |
| 21년 이상 | **25일** (최대) |

### 2.4 계산 공식

```
연차일수 = 15일 + ((근속년수 - 1) ÷ 2)
※ 소수점 이하 버림, 최대 25일
```

## 3. 출근율 계산

### 3.1 출근율 공식

```
출근율 = 출근일수 ÷ 소정근로일수 × 100
```

### 3.2 출근 인정 일수

| 사유 | 출근 인정 |
|------|----------|
| 업무상 부상·질병 | ✅ |
| 육아휴직 | ✅ |
| 출산휴가 | ✅ |
| 연차휴가 | ✅ |

### 3.3 출근 불인정

| 사유 | 인정 |
|------|------|
| 개인 질병 (업무 외) | ❌ |
| 무단결근 | ❌ |
| 정직 | ❌ |

## 4. 연차 사용

### 4.1 사용 원칙

| 원칙 | 내용 |
|------|------|
| 근로자 청구 | 원하는 시기에 사용 |
| 사업주 의무 | 유급휴가 보장 |
| 시기변경권 | 사업 운영상 필요시 변경 요청 |

### 4.2 사용 기한

| 항목 | 기한 |
|------|------|
| 사용 기한 | 발생일로부터 **1년** |
| 미사용 시 | 소멸 또는 수당 지급 |

### 4.3 연차 사용 촉진

| 단계 | 시기 | 내용 |
|------|------|------|
| 1차 | 만료 6개월 전 | 미사용 연차 통보 |
| 2차 | 만료 2개월 전 | 사용 시기 지정 요청 |
| 미지정 시 | 만료 1개월 전 | 회사가 지정 |

## 5. 연차수당

### 5.1 수당 계산

```
연차수당 = 1일 통상임금 × 미사용 연차일수
```

### 5.2 1일 통상임금

```
월 통상임금 ÷ 월 소정근로시간 × 8시간
또는
월 통상임금 ÷ 209시간 × 8시간
```

### 5.3 계산 예시

**월급 300만원, 미사용 연차 10일**

| 항목 | 금액 |
|------|------|
| 월 통상임금 | 3,000,000원 |
| 1일 통상임금 | 약 114,832원 |
| 미사용 연차 | 10일 |
| **연차수당** | **약 1,148,320원** |

### 5.4 수당 지급 시기

| 상황 | 지급 시기 |
|------|----------|
| 연차 소멸 시 | 소멸 후 임금 지급일 |
| 퇴직 시 | 퇴직 시 정산 |

## 6. 연차촉진제도

### 6.1 촉진 요건

| 요건 | 내용 |
|------|------|
| 1차 통보 | 사용 기한 6개월 전 |
| 2차 통보 | 사용 기한 2개월 전 |
| 서면 통보 | 문서 또는 이메일 |

### 6.2 촉진 효과

| 상황 | 수당 지급 |
|------|----------|
| 촉진 O + 미사용 | 수당 **미지급** |
| 촉진 X + 미사용 | 수당 **지급** |

### 6.3 촉진 절차

| 단계 | 시기 | 내용 |
|------|------|------|
| 1 | 6개월 전 | 미사용 연차 일수 통보, 사용 시기 지정 요청 |
| 2 | 10일 내 | 근로자 사용 시기 지정 |
| 3 | 2개월 전 | 미지정 시 재촉구 |
| 4 | 10일 내 | 근로자 사용 시기 지정 |
| 5 | 미지정 시 | 사용자가 사용 시기 지정 |

| 연간 최대 11일 | 2년차 15일과 별도 |

| 비례 적용 | 근로시간 비례 |

## 8. 연차와 퇴직
