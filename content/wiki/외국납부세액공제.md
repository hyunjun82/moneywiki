# 외국납부세액공제

## 기본 개념
해외에서 발생한 소득에 대해 외국에 납부한 세금을 국내 세금에서 공제해주는 제도입니다.

## 목적
- **이중과세 방지**
- 같은 소득에 대해 국내·외국 모두 과세하는 것 방지
- 국제적 조세 형평성 확보

## 공제 대상 소득

### 해외 소득
- 외국에서 발생한 근로소득
- 외국 주식 배당금
- 해외 부동산 임대소득
- 외국 사업소득
- 해외 금융소득

### 제외 소득
- 국내 원천소득
- 외국에 세금 납부하지 않은 소득

## 공제 방식

### 외국납부세액공제 (원칙)
```
공제액 = MIN(외국납부세액, 국내세액 × 해외소득/총소득)
```

### 필요경비 산입 (선택)
- 외국납부세액을 필요경비로 처리
- 공제보다 유리한 경우 선택

## 공제 한도

### 한도 계산식
```
공제 한도 = 국내 산출세액 × (해외소득금액 / 종합소득금액)
```

### 예시
```
국내 산출세액: 1,000만원
해외소득: 2,000만원
국내소득: 8,000만원
총소득: 1억원
공제 한도: 1,000만원 × (2,000/10,000) = 200만원
```

## 실무 적용 사례

### 사례 1: 해외 배당금
```
해외 배당금: 1,000만원
외국 원천세: 150만원 (15%)
국내 산출세액: 500만원
국내소득: 3,000만원
총소득: 4,000만원

공제 한도: 500만원 × (1,000/4,000) = 125만원
외국납부세액: 150만원
→ 공제액: 125만원 (한도 적용)
```

### 사례 2: 해외 근로소득
```
해외 근로소득: 5,000만원
외국 납부세액: 1,200만원 (24%)
국내 산출세액: 2,000만원
국내소득: 3,000만원
총소득: 8,000만원

공제 한도: 2,000만원 × (5,000/8,000) = 1,250만원
외국납부세액: 1,200만원
→ 공제액: 1,200만원 (전액 공제)
```

### 사례 3: 필요경비 산입 선택
```
해외 사업소득: 3,000만원
외국 납부세액: 600만원
필요경비: 1,500만원

방법1 (세액공제): 600만원 중 일부 공제
방법2 (필요경비): 600만원 전액 필요경비 산입
→ 유리한 것 선택
```

## 증명 서류

### 필수 서류
1. **외국납부세액증명서**
   - 외국 정부 발급
   - 영문 또는 번역본

2. **원천징수영수증**
   - 배당금, 이자 소득 증명

3. **납세확인서**
   - 외국 세무당국 발급

### 추가 서류
- 소득 증빙 자료
- 송금 내역
- 계약서 (해외 근로)

## 국가별 세율

### 주요 국가 원천징수세율
| 국가 | 배당 | 이자 | 특징 |
|------|------|------|------|
| 미국 | 15% | 0% | 조세조약 |
| 일본 | 15% | 0% | 조세조약 |
| 중국 | 10% | 10% | 조세조약 |
| 홍콩 | 0% | 0% | 원천징수 없음 |
| 싱가포르 | 0% | 15% | 조세조약 |

### 조세조약
- 대부분 국가와 체결
- 낮은 세율 적용
- 이중과세 방지

## 주의사항

### 공제 vs 필요경비
- 공제: 세액에서 직접 차감
- 필요경비: 소득에서 차감
- **일반적으로 공제가 유리**

### 환율 적용
- 외화 소득: 환율 적용 필요
- 송금일 기준환율 또는 재정환율
- 일관성 있게 적용

### 신고 기한
- 종합소득세 신고 시 함께 신고
- 매년 5월 1일 ~ 5월 31일

### 이월 불가
- 공제 한도 초과분 이월 안 됨
- 필요경비 산입 검토 필요

## 미국 주식 배당금 사례

### 배당금 수령 과정
```
1. 미국 주식 배당금 발생
2. 미국에서 15% 원천징수 (조세조약)
3. 국내 계좌로 입금
4. 다음 해 5월 종합소득세 신고
5. 외국납부세액공제 신청
```

### 계산 예시
```
애플 배당금: $1,000
환율: 1,300원
배당금: 130만원
미국 원천세: 19.5만원 (15%)
실수령액: 110.5만원

국내 신고:
배당소득: 130만원
외국납부세액공제: 19.5만원 (한도 내)
```

## 실무 팁

### 증빙 서류 보관
- 외국납부세액증명서 반드시 발급
- 영문 원본 보관
- 번역본 준비 (필요 시)

### 한도 계산
- 공제 한도 미리 계산
- 초과분은 필요경비 검토

### 조세조약 확인
- 국가별 세율 확인
- 낮은 세율 적용 가능 여부

### 환율 적용
- 일관된 환율 적용
- 재정환율 또는 송금일 환율

## 신고 방법

### 홈택스 신고
1. 종합소득세 신고
2. 외국납부세액공제 항목 선택
3. 외국납부세액 입력
4. 증빙 서류 첨부
5. 신고 완료

### 증빙 첨부
- PDF 스캔본
- 원본 보관 (5년)
