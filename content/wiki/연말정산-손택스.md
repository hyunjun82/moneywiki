손택스 모바일 연말정산을 정리해 볼게요. 손택스는 국세청 공식 앱이에요.

## 손택스 다운로드

앱스토어에서 다운로드해요.

| 스토어 | 앱 이름 |
|--------|---------|
| 앱스토어 | 손택스 |
| 플레이스토어 | 손택스 |

## 로그인 방법

로그인 방법이에요.

| 방법 | 설명 |
|------|------|
| 공동인증서 | 금융인증서, 공동인증서 |
| 간편인증 | 카카오, 네이버 등 |
| 생체인증 | 지문, Face ID |

## 연말정산 기능

손택스에서 이용 가능한 연말정산 기능이에요.

| 기능 | 경로 |
|------|------|
| 간소화자료 조회 | 조회/발급 → 연말정산 |
| PDF 다운로드 | 간소화자료 → PDF |
| 자료 제출 | 회사에 직접 전송 |

## 이용 시간

이용 가능 시간이에요.

| 기간 | 시간 |
|------|------|
| 간소화 서비스 | 1월 15일~ |
| 이용 시간 | 06:00~24:00 |

## 주의사항

이용 시 주의사항이에요.

| 주의 | 내용 |
|------|------|
| 로그인 오류 | 인증서 확인 |
| PDF 저장 | 내 폰에 저장 |
