"K-tech Pass가 있으면 세금 혜택이 있어요?"

네, 소득세 50% 감면을 최대 5년간 받을 수 있어요.

## K-tech Pass가 뭔가요

우수한 외국인 기술인력임을 정부가 인증하는 제도예요. 과학기술정보통신부에서 발급해요.

| 구분 | 내용 |
|------|------|
| 발급 기관 | 과학기술정보통신부 |
| 대상 | 우수 외국인 기술인력 |
| 분야 | IT, 과학, 엔지니어링 등 |

## 세금 감면이 얼마예요

소득세 50%를 최대 5년간 감면받아요.

| 연간 소득 | 일반 세금 | 감면 후 세금 |
|----------|----------|-------------|
| 5,000만원 | 약 600만원 | 약 300만원 |
| 8,000만원 | 약 1,200만원 | 약 600만원 |
| 1억원 | 약 1,800만원 | 약 900만원 |

5년간 상당한 절세 효과가 있어요.

## 적용 조건이 어떻게 돼요

외국인이면서 K-tech Pass를 보유해야 해요.

| 요건 | 내용 |
|------|------|
| 국적 | 외국인 (비거주자 포함) |
| 자격 | K-tech Pass 보유 |
| 근무 | 국내 기업 취업 |
| 기간 | 최초 근무 시작일부터 5년 |

내국인이나 이중국적자는 해당 안 돼요.

## 신청은 어떻게 해요

회사에 K-tech Pass 사본을 제출해요.

**신청 절차**

1. K-tech Pass 발급 (과기정통부)
2. 회사에 Pass 사본 제출
3. 원천징수 시 감면 적용
4. [연말정산](/w/연말정산) 때 정산

회사에서 급여 지급 시 자동으로 감면 적용해 줘요.

## 중소·중견기업 추가 감면

중소·중견기업에 취업하면 추가 감면이 있어요.

| 기업 유형 | 감면율 |
|----------|--------|
| 대기업 | 50% |
| 중소·중견기업 | 70% |

[중소기업 취업자 감면](/w/연말정산-중소기업-취업자-감면)과 중복 적용은 안 되고, 더 유리한 쪽으로 적용해요.

## 주의할 점이 있어요

5년 한도와 갱신에 유의하세요.

| 주의사항 | 내용 |
|----------|------|
| 기간 한도 | 최초 근무일부터 5년 |
| Pass 유효기간 | 갱신 필요 |
| 퇴직 후 재취업 | 잔여 기간 내 적용 |

5년이 지나면 감면이 종료돼요. Pass 유효기간도 별도로 관리해야 해요.
