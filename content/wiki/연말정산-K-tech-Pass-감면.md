첨단 기술 분야에서 일하시나요? K-tech Pass로 세금을 크게 줄일 수 있어요.

✨핵심 요약!

1. K-tech Pass 참여 인력 소득세 50% 감면
2. 연구개발특구 입주기업 대상
3. 5년간 감면 혜택

## 목차

## K-tech Pass란

연구개발특구 입주기업에서 일하는 첨단 기술 인력을 위한 지원 프로그램이에요.

외국 인재 유치를 위해 비자와 세제 혜택을 패키지로 제공해요. 한국에서 첨단 기술 분야 일자리를 만드는 게 목적이에요.

*프로그램 구조

| 혜택 | 내용 |
|------|------|
| 비자 | 간소화된 비자 발급 |
| 세금 | 소득세 50% 감면 |
| 기간 | 5년간 |

## 감면 대상

연구개발특구 입주기업에서 연구개발 업무를 하는 인력이 대상이에요.

| 요건 | 내용 |
|------|------|
| 근무지 | 연구개발특구 입주기업 |
| 업무 | 연구개발 관련 업무 |
| 자격 | 첨단 기술 분야 인력 |

*연구개발특구

- 대덕연구개발특구
- 광주연구개발특구
- 대구연구개발특구
- 부산연구개발특구
- 전북연구개발특구

## 감면 혜택

소득세 50%가 감면돼요.

| 연봉 | 일반 세금 | 감면 후 세금 |
|------|----------|-------------|
| 5,000만원 | 약 400만원 | 약 200만원 |
| 7,000만원 | 약 700만원 | 약 350만원 |
| 1억원 | 약 1,500만원 | 약 750만원 |

연봉이 높을수록 감면 효과가 커요.

*적용 기간

| 항목 | 기간 |
|------|------|
| 감면 기간 | 취업일로부터 5년 |
| 감면율 | 전 기간 50% |

## 신청 방법

회사에서 원천징수 시 감면을 적용해요.

| 절차 | 내용 |
|------|------|
| 자격 확인 | 연구개발특구진흥재단 확인 |
| 서류 제출 | 감면 신청서 제출 |
| 원천징수 | 50% 감면 적용 |

K-tech Pass 대상자로 확인되면 회사에서 원천징수 때 자동으로 50% 감면해 줘요.

## 다른 감면과 중복

다른 소득세 감면과 중복 적용이 안 될 수 있어요.

| 감면 제도 | 중복 |
|----------|------|
| 중소기업 취업자 감면 | 선택 적용 |
| 외국인 기술자 감면 | 선택 적용 |

더 유리한 감면을 선택해서 적용받을 수 있어요.

## 주의사항

연구개발특구를 벗어나거나 연구개발 업무가 아닌 업무로 전환하면 감면이 끝나요.

| 상황 | 감면 |
|------|------|
| 특구 외 이직 | 감면 종료 |
| 비연구개발 업무 전환 | 감면 종료 |
| 5년 경과 | 감면 종료 |

감면 조건을 계속 유지해야 혜택을 받을 수 있어요.
