# 자녀장려금

자녀장려금(CTC)은 저소득 가구의 자녀 양육을 지원하는 제도입니다. [조세특례제한법](https://www.law.go.kr/법령/조세특례제한법)에 따른 내용을 정리합니다.

## 자녀장려금 개요

### 1.1 지원 목적

- 저소득 가구 자녀 양육비 지원
- 출산 장려
- 자녀 양육 부담 경감

### 1.2 지급 대상

| 조건 | 기준 |
|------|------|
| 부양자녀 | 만 **18세 미만** |
| 소득 | 가구 유형별 기준 이하 |
| 재산 | **2.4억원** 미만 |

## 신청 자격

### 2.1 소득 기준 (2026년)

| 가구 유형 | 연 소득 기준 |
|----------|-------------|
| 홑벌이가구 | **7,000만원** 미만 |
| 맞벌이가구 | **7,000만원** 미만 |

※ 단독가구는 자녀장려금 대상 아님 (자녀 없음)

### 2.2 재산 기준

| 항목 | 기준 |
|------|------|
| 재산 합계 | **2.4억원** 미만 |
| 기준일 | 전년도 6월 1일 |

※ 재산 1.7억원 이상 시 지급액의 50% 감액

### 2.3 부양자녀 요건

| 요건 | 기준 |
|------|------|
| 나이 | 만 **18세 미만** |
| 소득 | 연 소득 100만원 이하 |
| 동거 | 주민등록상 동거 |

## 지급액

### 3.1 자녀 1인당 지급액

| 연 소득 | 지급액 |
|--------|--------|
| 2,100만원 미만 | **100만원** (최대) |
| 2,100만~7,000만원 | 점차 감소 |

### 3.2 소득별 지급액 예시

| 연 소득 | 자녀 1인당 |
|--------|-----------|
| 1,500만원 | **100만원** |
| 3,000만원 | 약 80만원 |
| 5,000만원 | 약 45만원 |
| 6,500만원 | 약 15만원 |

### 3.3 총 지급액 예시

**자녀 2명, 연 소득 2,500만원**

| 항목 | 금액 |
|------|------|
| 자녀 1인당 | 약 90만원 |
| 자녀 수 | 2명 |
| **총 지급액** | **약 180만원** |

## 제외 대상

### 4.1 신청 불가

| 사유 | 제외 |
|------|------|
| 부양자녀 없음 |  |
| 재산 2.4억원 이상 |  |
| 대한민국 국적 없음 |  |
| 전문직 사업자 |  |

### 4.2 자녀 제외 사유

| 사유 | 제외 |
|------|------|
| 만 18세 이상 |  |
| 연 소득 100만원 초과 |  |
| 별도 가구 |  |

## 신청 방법

### 5.1 신청 기간

| 유형 | 기간 |
|------|------|
| 정기신청 | **5월 1일~31일** |
| 기한 후 신청 | 6월 1일~11월 30일 (10% 감액) |

### 5.2 신청 방법

| 방법 | 내용 |
|------|------|
| 홈택스 | 온라인 신청 |
| 손택스 | 모바일 앱 |
| ARS | 1544-9944 |
| 세무서 | 방문 신청 |

### 5.3 필요 서류

| 서류 | 용도 |
|------|------|
| 신분증 | 본인 확인 |
| 가족관계증명서 | 부양자녀 확인 |
| 계좌번호 | 지급용 |

## 지급

### 6.1 지급 일정

| 신청 유형 | 지급 시기 |
|----------|----------|
| 정기신청 | **9월** |
| 기한 후 신청 | 신청 후 3-4개월 |

### 6.2 지급 방법

| 방법 | 내용 |
|------|------|
| 계좌이체 | 신청 시 등록 계좌 |
| 현금 수령 | 우체국 방문 |

## 감액·환수

### 7.1 감액 사유

| 사유 | 감액 |
|------|------|
| 재산 1.7억~2.4억원 | **50%** 감액 |
| 기한 후 신청 | **10%** 감액 |
| 체납액 있음 | 체납액 충당 |

### 7.2 환수

| 사유 | 조치 |
|------|------|
| 부정 수급 | 환수 + 가산금 |
| 자녀 나이 오류 | 환수 |

## 근로장려금과의 관계

### 8.1 중복 수급

| 항목 | 가능 여부 |
|------|----------|
| 근로장려금 + 자녀장려금 |  동시 가능 |

### 8.2 비교

| 구분 | 근로장려금 | 자녀장려금 |
|------|----------|-----------|
| 대상 | 저소득 근로자 | 부양자녀 있는 가구 |
| 소득 기준 | 2,200~3,800만원 | 7,000만원 |
| 최대 금액 | 330만원 | 100만원/자녀 |

### 8.3 동시 수급 예시

**맞벌이, 연 소득 2,000만원, 자녀 2명**

| 장려금 | 금액 |
|--------|------|
| 근로장려금 | 약 300만원 |
| 자녀장려금 | 약 200만원 |
| **합계** | **약 500만원** |

## 주의사항

### 9.1 자녀 나이

| 주의 | 내용 |
|------|------|
| 기준일 | 전년도 12월 31일 |
| 만 18세 | 해당 연도 1월 1일 기준 |

### 9.2 신청 누락

| 상황 | 조치 |
|------|------|
| 신청 안 함 | 지급 불가 |
| 5년 이내 | 기한 후 신청 가능 (감액) |
