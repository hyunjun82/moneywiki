# 기부금공제

기부금 세액공제는 기부금 지출에 대해 세금을 감면받는 제도입니다. 종류별 공제 방법을 정리합니다.

## 기부금공제 개요

### 1.1 공제 방식

| 항목 | 내용 |
|------|------|
| 공제 유형 | **세액공제** |
| 공제율 | 15~30% |
| 이월공제 | **10년** |

### 1.2 기부금 종류

| 종류 | 예시 |
|------|------|
| 법정기부금 | 국가, 지자체, 국방헌금 |
| 정치자금기부금 | 정당, 후원회 |
| 지정기부금 | 종교단체, 사회복지단체 |
| 우리사주조합기부금 | 우리사주조합 |

## 공제율

### 2.1 일반 공제율

| 기부금액 | 공제율 |
|----------|--------|
| 1,000만원 이하 | **15%** |
| 1,000만원 초과 | **30%** |

### 2.2 정치자금기부금

| 기부금액 | 공제 |
|----------|------|
| 10만원 이하 | **100/110** (전액) |
| 10만원 초과 | 15%/30% |

## 한도

### 3.1 기부금별 한도

| 종류 | 한도 |
|------|------|
| 법정기부금 | 소득금액의 **100%** |
| 정치자금기부금 | 소득금액의 **100%** |
| 지정기부금 (종교) | 소득금액의 **10%** |
| 지정기부금 (비종교) | 소득금액의 **30%** |
| 우리사주조합 | 소득금액의 **30%** |

### 3.2 공제 순서

| 순서 | 기부금 |
|------|--------|
| 1순위 | 정치자금기부금 |
| 2순위 | 법정기부금 |
| 3순위 | 우리사주조합기부금 |
| 4순위 | 지정기부금 |

## 법정기부금

### 4.1 대상 단체

| 단체 | 예시 |
|------|------|
| 국가·지자체 | 국고, 지방자치단체 |
| 국방헌금 | 국방부 |
| 천재지변 이재민 | 재해구호 |
| 사립학교 (시설비) | 학교법인 |
| 병원 (시설비) | 의료법인 |

### 4.2 공제 한도

| 항목 | 한도 |
|------|------|
| 한도 | 소득금액 **100%** |
| 초과분 | 10년 이월공제 |

## 정치자금기부금

### 5.1 대상

| 단체 | 예시 |
|------|------|
| 정당 | 당비, 후원금 |
| 후원회 | 국회의원 후원회 |
| 선거관리위원회 | 기탁금 |

### 5.2 공제 방법

| 금액 | 공제 |
|------|------|
| 10만원 이하 | **전액** (100/110) |
| 10만원 초과 | 15%/30% |

### 5.3 계산 예시

**정치자금기부금 50만원**

| 구간 | 기부금 | 공제 |
|------|--------|------|
| 10만원 이하 | 10만원 | 10만원 |
| 10만원 초과 | 40만원 | 6만원 (15%) |
| **합계** | 50만원 | **16만원** |

## 지정기부금

### 6.1 종교단체

| 항목 | 내용 |
|------|------|
| 대상 | 종교단체 |
| 한도 | 소득금액의 **10%** |
| 공제율 | 15%/30% |

### 6.2 비종교단체

| 단체 | 한도 |
|------|------|
| 사회복지법인 | 30% |
| 학술·장학단체 | 30% |
| 문화·예술단체 | 30% |
| 환경보전단체 | 30% |
| 국제구호단체 | 30% |

### 6.3 계산 예시

**근로소득금액 5,000만원, 종교기부금 800만원**

| 항목 | 금액 |
|------|------|
| 근로소득금액 | 5,000만원 |
| 한도 (10%) | 500만원 |
| 기부금 | 800만원 |
| 공제 가능액 | **500만원** |
| 이월분 | 300만원 |

## 고액기부금 공제

### 7.1 3,000만원 초과

| 금액 | 공제율 |
|------|--------|
| 1,000만원 이하 | 15% |
| 1,000~3,000만원 | 30% |
| 3,000만원 초과 | **40%** |

### 7.2 계산 예시

**기부금 5,000만원**

| 구간 | 기부금 | 공제율 | 공제액 |
|------|--------|--------|--------|
| 1,000만원 이하 | 1,000만원 | 15% | 150만원 |
| 1,000~3,000만원 | 2,000만원 | 30% | 600만원 |
| 3,000만원 초과 | 2,000만원 | 40% | 800만원 |
| **합계** | 5,000만원 | - | **1,550만원** |

## 이월공제

### 8.1 이월공제 기간

| 항목 | 기간 |
|------|------|
| 이월 기간 | **10년** |
| 적용 | 한도 초과분 |

### 8.2 이월공제 순서

| 순서 | 적용 |
|------|------|
| 1순위 | 이월된 기부금 (먼저 발생분) |
| 2순위 | 당해연도 기부금 |

## 증빙서류

### 9.1 필수 서류

| 서류 | 발급처 |
|------|--------|
| 기부금영수증 | 기부단체 |
| 법인설립허가증 사본 | 필요시 |

### 9.2 간소화 서비스

| 방법 | 내용 |
|------|------|
| 연말정산 간소화 | 자동 수집 (일부) |
| 직접 제출 | 기부금영수증 |

### 9.3 기부금영수증 필수 기재

| 항목 | 내용 |
|------|------|
| 기부자 인적사항 | 성명, 주민번호 |
| 기부금 유형 | 법정/지정 구분 |
| 기부금액 | 금액, 일자 |
| 단체 정보 | 단체명, 사업자번호 |

## 부양가족 기부금

### 10.1 공제 가능

| 대상 | 조건 |
|------|------|
| 배우자 | 소득요건 충족 |
| 직계존속 | 나이+소득요건 |
| 직계비속 | 나이+소득요건 |
| 형제자매 | 나이+소득요건 |

### 10.2 요건

| 요건 | 기준 |
|------|------|
| 소득 | 연 100만원 이하 |
| 나이 | 기본공제 대상 |
