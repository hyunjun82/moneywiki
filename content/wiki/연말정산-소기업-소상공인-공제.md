# 연말정산 소기업 소상공인 공제 (노란우산공제)

노란우산공제는 소기업·소상공인 대표자를 위한 퇴직금 마련 제도로, 납입금에 대해 소득공제를 받을 수 있습니다. [조세특례제한법 제86조의3](https://www.law.go.kr/법령/조세특례제한법)에 따른 조건을 정리합니다.

## 1. 제도 개요

### 1.1 노란우산공제란

| 항목 | 내용 |
|------|------|
| 운영 기관 | 중소기업중앙회 |
| 목적 | 소상공인 폐업 시 생활 안정 |
| 성격 | 사업주 퇴직금 |
| 세제 혜택 | **소득공제** |

### 1.2 가입 대상

| 대상 | 기준 |
|------|------|
| 소기업 | 업종별 매출액 기준 |
| 소상공인 | 상시근로자 5인 미만 (제조업 10인) |
| 프리랜서 | 사업소득 있는 개인 |
| 법인 대표이사 | 소기업 요건 충족 시 |

## 2. 소득공제 한도

### 2.1 사업소득 기준 한도

[조세특례제한법]에 따른 공제 한도입니다.

| 사업(근로)소득 | 연간 공제 한도 |
|---------------|---------------|
| 4,000만원 이하 | **500만원** |
| 4,000만원 초과 ~ 1억원 이하 | **300만원** |
| 1억원 초과 | **200만원** |

### 2.2 납입 한도

| 항목 | 금액 |
|------|------|
| 월 납입 최소 | 5만원 |
| 월 납입 최대 | 100만원 |
| 연 최대 납입 | 1,200만원 |

## 3. 공제 계산 예시

### 3.1 소득 4,000만원 이하

**연간 사업소득 3,000만원, 연 500만원 납입**

- 공제 한도: 500만원
- 소득공제: **500만원 전액**
- 세금 절감 (15% 구간): 약 75만원

### 3.2 소득 1억원 초과

**연간 사업소득 1.5억원, 연 500만원 납입**

- 공제 한도: 200만원
- 소득공제: **200만원** (한도 적용)
- 세금 절감 (35% 구간): 약 70만원

## 4. 적립금 수령

### 4.1 공제금 수령 사유

| 사유 | 수령 가능 |
|------|----------|
| 폐업 | ✅ 가능 |
| 사망 | ✅ 가능 |
| 퇴임 (법인 대표) | ✅ 가능 |
| 노령 (60세 이상 + 10년 납입) | ✅ 가능 |
| 임의 해지 | ✅ 가능 (불이익) |

### 4.2 수령 시 과세

| 수령 사유 | 과세 방식 |
|----------|----------|
| 폐업, 사망, 퇴임 | **퇴직소득세** (유리) |
| 임의 해지 | **기타소득세 16.5%** (불리) |

## 5. 퇴직소득세 vs 기타소득세

### 5.1 퇴직소득세 (정상 수령)

- 근속연수에 따른 공제
- 분리과세로 세부담 낮음
- 소득공제 혜택 유지

### 5.2 기타소득세 (임의 해지)

- 16.5% 세율 (지방세 포함)
- 소득공제 혜택 환수 효과
- 가능하면 **임의 해지 피해야** 함

## 6. 가입 절차

### 6.1 가입 방법

| 방법 | 절차 |
|------|------|
| 온라인 | [노란우산공제 홈페이지](https://www.8899.or.kr) |
| 오프라인 | 중소기업중앙회 지부 |
| 금융기관 | 제휴 은행 창구 |

### 6.2 필요 서류

| 서류 | 용도 |
|------|------|
| 사업자등록증 | 사업자 확인 |
| 신분증 | 본인 확인 |
| 통장 사본 | 납입 계좌 |

## 7. 압류 방지 혜택

### 7.1 법적 보호

| 항목 | 내용 |
|------|------|
| 압류 금지 | 공제금 압류 불가 |
| 양도 금지 | 담보 제공 불가 |
| 법적 근거 | 중소기업협동조합법 |

### 7.2 보호 한도

- 소기업·소상공인 공제부금: **전액 보호**

## 8. 주의사항

### 8.1 가입 자격 상실

| 상황 | 결과 |
|------|------|
| 폐업 후 재창업 | 재가입 가능 |
| 소기업 기준 초과 | 유지 가능 (신규 불가) |

### 8.2 납입 중단

- 3회 이상 연체 시 계약 해지 가능
- 납입 유예 신청 가능 (일시적 어려움)

### 8.3 연말정산 반영

- 간소화 서비스에서 **자동 조회**
- 사업소득자는 종합소득세 신고 시 공제

## 9. 근로소득자와 비교

### 9.1 유사 제도 비교

| 구분 | 노란우산공제 | 연금저축 |
|------|------------|---------|
| 대상 | 사업자 | 누구나 |
| 공제 유형 | 소득공제 | 세액공제 |
| 최대 공제 | 500만원 | 600만원 |
| 수령 시 | 퇴직소득세 | 연금소득세 |

### 9.2 병행 가입

노란우산공제와 연금저축 **동시 가입 가능**
