# 전월세신고

전월세신고제는 임대차 계약 정보를 신고하는 제도입니다. 신고 방법과 대상을 정리합니다.

## 전월세신고제 개요

### 1.1 시행 목적

| 목적 | 내용 |
|------|------|
| 시장 투명성 | 임대차 가격 정보 공개 |
| 세입자 보호 | 확정일자 자동 부여 |
| 정책 자료 | 주거 정책 수립 활용 |

### 1.2 시행 시기

| 항목 | 시기 |
|------|------|
| 시행 | 2021년 6월 1일 |
| 계도기간 종료 | 2024년 5월 31일 |
| 과태료 부과 | 2024년 6월 1일~ |

## 신고 대상

### 2.1 신고 기준

| 조건 | 기준 |
|------|------|
| 보증금 | **6,000만원 초과** |
| 월세 | **30만원 초과** |
| 적용 | 둘 중 하나만 해당해도 |

### 2.2 신고 대상 계약

| 계약 유형 | 신고 |
|----------|------|
| 신규 계약 |  |
| 갱신 계약 |  |
| 금액 변경 |  |

### 2.3 신고 제외

| 대상 | 제외 |
|------|------|
| 보증금 6천만원 이하 + 월세 30만원 이하 |  |
| 친족 간 무상 임대 |  |

## 신고 방법

### 3.1 신고 기한

| 항목 | 기한 |
|------|------|
| 신규·갱신 | 계약일로부터 **30일** |

### 3.2 신고 방법

| 방법 | 내용 |
|------|------|
| 온라인 | 부동산거래관리시스템 (rtms.molit.go.kr) |
| 방문 | 주민센터, 구청 |
| 공인중개사 | 대행 신고 가능 |

### 3.3 신고자

| 신고자 | 신고 가능 |
|--------|----------|
| 임대인 + 임차인 공동 |  (원칙) |
| 임대인 단독 |  |
| 임차인 단독 |  |
| 공인중개사 |  (대행) |

## 신고 내용

### 4.1 필수 신고 항목

| 항목 | 내용 |
|------|------|
| 계약 당사자 | 임대인·임차인 인적사항 |
| 주택 정보 | 소재지, 면적, 유형 |
| 계약 조건 | 보증금, 월세, 기간 |
| 계약일 | 계약 체결일 |

### 4.2 필요 서류

| 서류 | 용도 |
|------|------|
| 임대차계약서 | 계약 내용 확인 |
| 신분증 | 신고인 확인 |

## 확정일자 자동 부여

### 5.1 제도 효과

| 항목 | 내용 |
|------|------|
| 확정일자 | 신고 완료 시 **자동 부여** |
| 우선변제권 | 확정일자로 우선변제권 확보 |

### 5.2 기존 방식과 비교

| 구분 | 기존 | 전월세신고 |
|------|------|-----------|
| 확정일자 | 주민센터 방문 | 자동 부여 |
| 비용 | 600원 | 무료 |

## 과태료

### 6.1 과태료 기준

| 위반 | 과태료 |
|------|--------|
| 미신고 | **최대 100만원** |
| 거짓 신고 | **최대 100만원** |

### 6.2 과태료 산정

| 지연 기간 | 과태료 |
|----------|--------|
| 3개월 이하 | 4~30만원 |
| 3~6개월 | 10~50만원 |
| 6개월 초과 | 20~100만원 |
