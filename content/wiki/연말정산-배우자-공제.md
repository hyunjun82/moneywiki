# 연말정산 배우자 공제

배우자 기본공제 조건과 금액을 정리합니다.

## 1. 공제 금액

| 항목 | 금액 |
|------|------|
| 기본공제 | **150만원** |
| 공제 유형 | 소득공제 |

## 2. 소득 요건

| 구분 | 기준 |
|------|------|
| 소득금액 | 연 **100만원 이하** |
| 근로소득만 | 총급여 **500만원 이하** |

## 3. 소득금액 계산

| 소득 | 계산 |
|------|------|
| 근로소득 | 총급여 - 근로소득공제 |
| 사업소득 | 수입 - 필요경비 |
| 기타소득 | 수입 - 필요경비(60%) |

## 4. 맞벌이 부부

| 상황 | 공제 |
|------|------|
| 배우자 소득 100만원 초과 | ❌ 불가 |
| 배우자 소득 100만원 이하 | ✅ 가능 |

## 5. 주의사항

| 주의 | 내용 |
|------|------|
| 이혼 | 12월 31일 기준 혼인 상태 |
| 사실혼 | 공제 불가 |
| 별거 | 법적 혼인이면 가능 |
