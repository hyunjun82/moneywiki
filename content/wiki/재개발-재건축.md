# 재개발 재건축

재개발과 재건축은 노후 주거지를 정비하는 사업입니다. 차이점과 투자 시 주의사항을 정리합니다.

## 정비사업 개요

### 1.1 정의

| 사업 | 정의 |
|------|------|
| 재개발 | 기반시설 + 주택 정비 |
| 재건축 | 노후 공동주택 정비 |

### 1.2 대상 지역

| 사업 | 대상 |
|------|------|
| 재개발 | 불량 주거지, 노후 단독주택 밀집 |
| 재건축 | 노후 아파트, 연립주택 |

## 재개발 vs 재건축

### 2.1 주요 차이

| 구분 | 재개발 | 재건축 |
|------|--------|--------|
| 대상 | 단독·다세대 밀집 | 공동주택 (아파트 등) |
| 기반시설 | 함께 정비 | 기존 활용 |
| 세입자 | 세입자 보상 | 보상 없음 |
| 임대주택 | 의무 건설 | 일부 의무 |
| 초과이익환수 | 없음 | **있음** |

### 2.2 조합원 자격

| 사업 | 자격 |
|------|------|
| 재개발 | 토지·건물 소유자 |
| 재건축 | 구분소유자 (아파트 소유자) |

### 2.3 세입자 보상

| 구분 | 재개발 | 재건축 |
|------|--------|--------|
| 주거이전비 |  |  |
| 이사비 |  |  |
| 영업보상 |  |  |

## 사업 절차

### 3.1 진행 단계

| 단계 | 내용 |
|------|------|
| 1. 정비구역 지정 | 지자체 지정 |
| 2. 추진위 구성 | 주민 동의 |
| 3. 조합설립인가 | 조합 설립 |
| 4. 사업시행인가 | 사업 확정 |
| 5. 관리처분인가 | 분양 계획 |
| 6. 이주·철거 | 기존 건물 철거 |
| 7. 착공·분양 | 건설 시작 |
| 8. 준공·입주 | 완공 후 입주 |

### 3.2 소요 기간

| 단계 | 기간 |
|------|------|
| 구역지정 ~ 조합설립 | 2~5년 |
| 조합설립 ~ 사업시행 | 1~3년 |
| 사업시행 ~ 착공 | 1~3년 |
| 착공 ~ 입주 | 2~4년 |
| **총 기간** | **6~15년** |

## 조합원 권리

### 4.1 조합원 자격

| 요건 | 내용 |
|------|------|
| 소유권 | 토지·건물 소유 |
| 동의 | 조합 가입 동의 |
| 분양권 | 신축 주택 분양 |

### 4.2 분양권

| 항목 | 내용 |
|------|------|
| 조합원 분양 | 일반분양보다 저렴 |
| 평형 선택 | 권리가액에 따라 |
| 추가분담금 | 초과분 납부 |

### 4.3 현금청산

| 상황 | 처리 |
|------|------|
| 분양 미신청 | 현금으로 정산 |
| 소수 지분 | 현금청산 가능 |

## 분담금

### 5.1 분담금 계산

```
분담금 = 분양가 - 권리가액
권리가액 = 종전자산평가액 × 비례율
```

### 5.2 비례율

| 비례율 | 의미 |
|--------|------|
| 100% 이상 | 조합원 이익 |
| 100% 미만 | 추가 분담금 |

### 5.3 분담금 예시

**종전자산 3억, 비례율 110%, 분양가 8억**

| 항목 | 금액 |
|------|------|
| 권리가액 | 3.3억원 |
| 분양가 | 8억원 |
| **분담금** | **4.7억원** |

### 5.4 분담금 변동

| 요인 | 영향 |
|------|------|
| 분양가 상승 | 분담금 증가 |
| 공사비 증가 | 분담금 증가 |
| 일반분양 성공 | 분담금 감소 |

## 투자 위험

### 6.1 주요 리스크

| 리스크 | 내용 |
|--------|------|
| 사업 지연 | 수년~10년 이상 |
| 분담금 변동 | 예상보다 증가 |
| 사업 무산 | 투자금 묶임 |
| 조합 분쟁 | 사업 지연 |

### 6.2 입주권 프리미엄

| 단계 | 프리미엄 |
|------|----------|
| 초기 | 낮음 (리스크 높음) |
| 관리처분 후 | 상승 |
| 착공 후 | 높음 |
| 입주 임박 | 최고 |

## 재건축 초과이익환수

### 7.1 부담금 개요

| 항목 | 내용 |
|------|------|
| 대상 | 재건축 사업 |
| 기준 | 초과이익 3,000만원 초과 |
| 부과 | 준공 시 |

### 7.2 부담률

| 초과이익 | 부담률 |
|----------|--------|
| 3,000만원 이하 | 면제 |
| 3,000만~5,000만원 | 10% |
| 5,000만~7,000만원 | 20% |
| 7,000만~9,000만원 | 30% |
| 9,000만~1.1억원 | 40% |
| 1.1억원 초과 | 50% |

### 7.3 계산 예시

**초과이익 1억원**

| 구간 | 금액 | 부담금 |
|------|------|--------|
| 3,000만 이하 | 면제 | 0 |
| 3,000~5,000만 | 2,000만 × 10% | 200만 |
| 5,000~7,000만 | 2,000만 × 20% | 400만 |
| 7,000~9,000만 | 2,000만 × 30% | 600만 |
| 9,000만~1억 | 1,000만 × 40% | 400만 |
| **합계** | | **1,600만원** |

## 세금

### 8.1 취득세

| 상황 | 세율 |
|------|------|
| 조합원 분양 | 1~3% (주택 취득세) |
| 입주권 매수 | 4% (승계취득) |

### 8.2 양도소득세

| 상황 | 과세 |
|------|------|
| 입주권 양도 | 양도세 과세 |
| 완공 후 양도 | 주택 양도세 |
| 1세대 1주택 | 비과세 요건 적용 |

### 8.3 종합부동산세

| 상황 | 과세 |
|------|------|
| 멸실 전 | 기존 주택 기준 |
| 멸실 후 | 입주권은 제외 |
| 입주 후 | 신규 주택 기준 |
