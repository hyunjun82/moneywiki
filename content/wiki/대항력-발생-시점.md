# 대항력 발생 시점

대항력은 세입자가 제3자에게 임대차를 주장할 수 있는 권리입니다. 발생 시점이 중요한 이유와 정확한 기준을 정리합니다.

## 1. 대항력이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 대항력 | 제3자에게 임대차를 주장하는 권리 |
| 효과 | 집주인 바뀌어도 계약 유지 |
| 근거 | 주택임대차보호법 제3조 |

### 1.2 대항력의 효과

| 상황 | 보호 내용 |
|------|----------|
| 집주인 변경 | 새 집주인에게 계약 주장 |
| 경매 낙찰 | 낙찰자에게 계약 주장 |
| 가압류 후 매각 | 매수인에게 계약 주장 |

## 2. 대항력 발생 요건

### 2.1 2가지 요건

| 요건 | 설명 |
|------|------|
| 주택 인도 | 실제 **입주** (점유) |
| 전입신고 | 주민등록 **이전** |

### 2.2 요건 상세

**주택 인도 (점유)**

| 내용 | 설명 |
|------|------|
| 실제 거주 | 짐 옮기고 생활 시작 |
| 점유 인정 | 가족 거주도 인정 |
| 점유 불인정 | 짐만 넣어둔 상태 |

**전입신고**

| 내용 | 설명 |
|------|------|
| 신고처 | 주민센터, 정부24 |
| 신고 내용 | 새 주소로 주민등록 이전 |
| 처리 | 즉시 또는 당일 |

## 3. 대항력 발생 시점

### 3.1 핵심 규정

| 항목 | 내용 |
|------|------|
| 발생 시점 | **다음 날 0시** |
| 조건 | 입주 + 전입신고 완료 후 |

### 3.2 시점 계산

**예시: 1월 15일 입주 및 전입신고**

| 시점 | 대항력 |
|------|--------|
| 1월 15일 23:59 | ❌ 없음 |
| 1월 16일 00:00 | ✅ 발생 |

### 3.3 다음 날 0시 의미

| 구분 | 설명 |
|------|------|
| 당일 | 입주+전입신고 완료일 |
| 다음 날 | 그 다음 날 |
| 0시 | 자정 (00:00) |

```
예시 타임라인:

1월 15일 10:00 - 잔금 지급
1월 15일 14:00 - 입주 (짐 이동, 열쇠 수령)
1월 15일 16:00 - 전입신고 완료
           ↓
1월 16일 00:00 - ★ 대항력 발생 ★
```

## 4. 왜 다음 날 0시인가

### 4.1 법적 취지

| 이유 | 설명 |
|------|------|
| 권리 충돌 방지 | 같은 날 설정된 권리와 순위 구분 |
| 공시 제도 | 등기부등본 반영 시차 고려 |
| 거래 안전 | 제3자 보호 |

### 4.2 실무적 의미

| 상황 | 결과 |
|------|------|
| 같은 날 근저당 설정 | 근저당이 **선순위** |
| 하루 전 전입신고 | 임차인이 **선순위** |

## 5. 잔금일 전입신고가 중요한 이유

### 5.1 위험 시나리오

**잔금일 전입신고 안 한 경우**

| 날짜 | 사건 | 결과 |
|------|------|------|
| 1월 15일 | 잔금 지급, 입주 | |
| 1월 16일 | 근저당 설정 | 근저당 선순위 |
| 1월 17일 | 전입신고 | |
| 1월 18일 | 대항력 발생 | 근저당보다 **후순위** |

### 5.2 안전 시나리오

**잔금일 전입신고 한 경우**

| 날짜 | 사건 | 결과 |
|------|------|------|
| 1월 15일 | 잔금, 입주, 전입신고 | |
| 1월 16일 | 대항력 발생 (0시) | |
| 1월 16일 | 근저당 설정 | 임차인 **선순위** |

### 5.3 순위 비교

| 전입신고 시점 | 대항력 발생 | 근저당(1/16 설정) 대비 |
|--------------|------------|---------------------|
| 1/15 | 1/16 00:00 | **선순위** |
| 1/16 | 1/17 00:00 | **후순위** |

## 6. 대항력 vs 우선변제권

### 6.1 차이점

| 구분 | 대항력 | 우선변제권 |
|------|--------|-----------|
| 요건 | 입주+전입신고 | 입주+전입신고+**확정일자** |
| 효과 | 임대차 주장 | 보증금 우선 회수 |
| 발생 시점 | 다음 날 0시 | 다음 날 0시 (확정일자 이후) |

### 6.2 확정일자 추가

| 상황 | 권리 |
|------|------|
| 입주+전입신고만 | 대항력만 |
| 입주+전입신고+확정일자 | 대항력 + 우선변제권 |

## 7. 전입신고 방법

### 7.1 방문 신고

| 항목 | 내용 |
|------|------|
| 장소 | 주민센터 |
| 구비서류 | 신분증, (임대차계약서) |
| 처리 | 즉시 |

### 7.2 온라인 신고

| 항목 | 내용 |
|------|------|
| 사이트 | 정부24 (gov.kr) |
| 방법 | 전입신고 메뉴 |
| 처리 | 당일~익일 |

### 7.3 주의사항

| 주의 | 내용 |
|------|------|
| 동·호수 정확히 | 틀리면 대항력 없음 |
| 가족 전원 | 함께 전입 권장 |
| 확인 | 전입 완료 확인 필수 |

## 8. 자주 묻는 상황

### 8.1 잔금일에 전입신고 못 한 경우

| 대응 | 내용 |
|------|------|
| 즉시 신고 | 다음 날이라도 빨리 |
| 등기부 확인 | 권리 변동 확인 |
| 위험 인지 | 공백 기간 위험 인식 |

### 8.2 점유 없이 전입신고만 한 경우

| 결과 | 내용 |
|------|------|
| 대항력 | ❌ 불발생 |
| 이유 | 점유 요건 미충족 |
| 해결 | 실제 입주 필요 |

### 8.3 전입신고 후 일시 퇴거

| 상황 | 대항력 |
|------|--------|
| 단기 출장 | ✅ 유지 (점유 의사) |
| 장기 부재 | ⚠️ 위험 (점유 상실 가능) |
| 다른 곳 전입 | ❌ 상실 |

### 8.4 가족만 거주하는 경우

| 상황 | 대항력 |
|------|--------|
| 배우자·직계가족 거주 | ✅ 인정 |
| 임차인 미거주 | ⚠️ 판례에 따라 다름 |

## 9. 대항력 상실 사유

### 9.1 상실 원인

| 원인 | 설명 |
|------|------|
| 전출신고 | 다른 곳으로 주민등록 이전 |
| 점유 상실 | 실제 거주 중단 |
| 임대차 종료 | 계약 해지 후 퇴거 |

### 9.2 대항력 유지 방법

| 방법 | 내용 |
|------|------|
| 계속 거주 | 점유 유지 |
| 전입 유지 | 전출하지 않음 |
| 임차권등기 | 보증금 미반환 시 등기 |

## 10. 실무 체크리스트

### 10.1 잔금일 순서

| 순서 | 체크 | 내용 |
|------|------|------|
| 1 | ☐ | 등기부등본 재확인 |
| 2 | ☐ | 잔금 지급 |
| 3 | ☐ | 열쇠 수령 |
| 4 | ☐ | **입주 (점유 시작)** |
| 5 | ☐ | **전입신고** |
| 6 | ☐ | **확정일자** |

### 10.2 대항력 발생 확인

| 확인 | 방법 |
|------|------|
| 전입신고 완료 | 주민등록등본 발급 |
| 주소 정확 | 동·호수 확인 |
| 효력 발생일 | 다음 날 0시 |
