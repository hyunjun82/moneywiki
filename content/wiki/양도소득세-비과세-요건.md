# 양도소득세 비과세 요건

## 기본 개념
1세대 1주택자가 일정 요건을 충족하면 양도소득세를 **전액 비과세**받는 제도입니다.

## 1세대 1주택 비과세 요건

### 1. 보유기간 요건
- **2년 이상** 보유 필수
- 계산: 취득일부터 양도일까지

### 2. 거주기간 요건
- **2년 이상** 거주 필수 (조정대상지역)
- 비조정대상지역: 거주 요건 없음

### 3. 1세대 1주택 요건
- 양도일 현재 1주택만 보유
- 세대 전체 기준 (배우자 포함)

### 4. 주택 가액 요건
- **12억원 이하**: 전액 비과세
- **12억원 초과**: 초과분에 대해 과세

## 세대 구성

### 세대의 범위
- 거주자 및 배우자
- 동거 가족 (생계를 같이하는 경우)
- 미혼 30세 미만 자녀

### 1세대 인정
- 배우자와 생계를 같이하는 가족 전체
- 따로 사는 부모님은 별도 세대

## 보유·거주기간 계산

### 보유기간
```
취득일: 2020년 1월 1일
양도일: 2024년 6월 30일
보유기간: 4년 6개월 → 2년 충족
```

### 거주기간 (조정대상지역)
```
전입신고일: 2020년 3월 1일
전출신고일: 2024년 6월 30일
거주기간: 4년 4개월 → 2년 충족
```

### 주의사항
- 전입신고 필수
- 실제 거주 확인 가능
- 세무조사 시 검증

## 조정대상지역 특례

### 일반 지역
- 보유 2년만 충족하면 비과세
- 거주 요건 없음

### 조정대상지역
- 보유 2년 + 거주 2년 모두 충족
- 서울, 과천, 성남(분당), 광명 등

### 현재 조정대상지역 (2024년 기준)
- 서울 전역
- 경기 과천, 성남(분당·수정), 광명, 하남
- 기타 일부 지역

## 12억원 초과 주택

### 양도가액 12억원 이하
```
양도가액: 10억원
양도세: 0원 (전액 비과세)
```

### 양도가액 12억원 초과
```
양도가액: 15억원
비과세: 12억원
과세: 3억원
→ 3억원에 대해 양도세 부과
```

### 계산 방법
```
양도차익 = 양도가액 - 취득가액 - 필요경비
과세 양도차익 = 양도차익 × (15억-12억) / 15억
= 양도차익 × 20%
```

## 일시적 2주택 특례

### 요건
1. 기존 주택 보유 중 신규 주택 취득
2. 기존 주택을 **3년 이내** 양도
3. 신규 주택으로 전입

### 비과세 적용
- 기존 주택 양도 시 비과세 적용
- 보유·거주기간 계산은 기존 주택 기준

### 예시
```
2020년: A주택 취득·거주
2023년: B주택 취득·전입
2024년: A주택 양도
→ A주택 비과세 가능 (3년 이내 처분)
```

## 주의사항

### 증여·상속 주택
- 증여받은 주택: 증여자 보유기간 합산 가능
- 상속받은 주택: 피상속인 보유기간 합산 가능

### 농어촌주택 특례
- 9억원 이하 농어촌주택
- 1세대 1주택과 별도
- 합산 2주택 보유 가능

### 고가주택 과세
- 12억원 초과분은 과세
- 장기보유특별공제 적용 가능

### 분양권·입주권
- 분양권은 주택으로 미인정
- 입주 후부터 주택으로 인정

## 비과세 배제 사례

### 2년 미만 보유·거주
```
보유: 1년 6개월
→ 비과세 불가, 전액 과세
```

### 2주택 이상 보유
```
양도일 현재 2주택
→ 비과세 불가
→ 일시적 2주택 특례 확인 필요
```

### 조정지역 거주 미달
```
조정지역 주택
보유: 3년
거주: 1년 6개월
→ 비과세 불가
```

## 실무 체크리스트

### 양도 전 확인사항
- [ ] 보유기간 2년 이상
- [ ] 거주기간 2년 이상 (조정지역)
- [ ] 1세대 1주택 확인
- [ ] 주택 가액 12억원 기준 확인
- [ ] 일시적 2주택 해당 여부

### 증빙 서류
- [ ] 매매계약서
- [ ] 주민등록등본 (거주 확인)
- [ ] 전입세대 열람 내역
- [ ] 취득·양도 시기 증빙
