# 연말정산 기숙사비

기숙사비 교육비 공제를 정리합니다.

## 1. 교육비 공제

| 항목 | 공제 |
|------|------|
| 기숙사비 | ❌ 불가 |
| 등록금 | ✅ 가능 |
| 입학금 | ✅ 가능 |

## 2. 공제 불가 이유

| 이유 | 설명 |
|------|------|
| 교육비 정의 | 수업료, 입학금 등 |
| 숙식비 | 교육비에 미포함 |

## 3. 대안

| 방법 | 공제 |
|------|------|
| 카드결제 | 신용카드 공제 |
| 현금영수증 | 현금영수증 공제 |

## 4. 유사 공제 불가 항목

| 항목 | 공제 |
|------|------|
| 기숙사비 | ❌ |
| 급식비(초중고) | ❌ |
| 통학버스비(초중고) | ❌ |
