# 전세보증보험

전세보증보험은 보증금을 지켜주는 안전장치입니다. 전세사기 피해 예방을 위해 반드시 가입을 권장합니다.

## 1. 전세보증보험이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 보증 내용 | 임대인이 보증금 못 돌려주면 **보증기관이 대신 지급** |
| 보증기관 | HUG, SGI, HF |
| 청구 시기 | 계약 종료 후 보증금 미반환 시 |

### 1.2 필요한 이유

| 이유 | 설명 |
|------|------|
| 전세사기 예방 | 깡통전세 피해 방지 |
| 안심 거주 | 보증금 걱정 없이 거주 |
| 이사 자유 | 미반환 시에도 이사 가능 |

## 2. 보증 기관

### 2.1 주요 보증기관

| 기관 | 상품명 |
|------|--------|
| 주택도시보증공사(HUG) | 전세보증금반환보증 |
| SGI서울보증 | 전세금보장신용보험 |
| 한국주택금융공사(HF) | 전세자금보증 |

### 2.2 기관별 특징

| 기관 | 특징 |
|------|------|
| HUG | 가장 많이 이용, 심사 엄격 |
| SGI | 가입 조건 완화, 보증료 높음 |
| HF | 전세대출 연계 |

### 2.3 연락처

| 기관 | 연락처 |
|------|--------|
| HUG | 1566-9009 |
| SGI | 1670-7000 |
| HF | 1688-8114 |

## 3. HUG 전세보증금반환보증

### 3.1 가입 조건

| 조건 | 기준 |
|------|------|
| 대항력 | 전입신고 + 확정일자 |
| 보증금 비율 | 시세의 **90% 이하** |
| 선순위 합산 | (근저당+선순위보증금+보증금) ≤ 시세 90% |

### 3.2 보증 한도

| 지역 | 한도 |
|------|------|
| 수도권 | **7억원** |
| 지방 | **5억원** |

### 3.3 보증료율 (2026년)

| 유형 | 연 보증료율 |
|------|-----------|
| 일반 | **0.128%** |
| 우대 (청년 등) | **0.115%** |
| 할증 | **0.154%** |

### 3.4 보증료 계산

**보증금 3억원, 2년 계약 (일반)**

| 항목 | 금액 |
|------|------|
| 보증금 | 3억원 |
| 보증료율 | 0.128%/년 |
| 2년 보증료 | 3억 × 0.128% × 2 = **76.8만원** |

## 4. SGI 전세금보장신용보험

### 4.1 가입 조건

| 조건 | 기준 |
|------|------|
| 대항력 | 전입신고 + 확정일자 |
| 보증금 비율 | 시세의 **90% 이하** |
| 심사 | HUG보다 완화 |

### 4.2 보증 한도

| 한도 | 금액 |
|------|------|
| 최대 | **5억원** |

### 4.3 보증료율

| 유형 | 연 보증료율 |
|------|-----------|
| 일반 | **0.183~0.208%** |

### 4.4 HUG vs SGI 비교

| 항목 | HUG | SGI |
|------|-----|-----|
| 한도 | 7억(수도권) | 5억 |
| 보증료 | 저렴 | 높음 |
| 심사 | 엄격 | 완화 |
| 가입 용이성 | 어려움 | 쉬움 |

## 5. 가입 절차

### 5.1 가입 시기

| 시기 | 권장 |
|------|------|
| 계약 전 | 가입 가능 여부 확인 |
| 잔금 전 | **가입 완료** 권장 |
| 입주 후 | 가입 가능 (대항력 취득 후) |

### 5.2 가입 방법

| 방법 | 내용 |
|------|------|
| 온라인 | HUG 홈페이지, 은행 앱 |
| 은행 | 전세대출 연계 가입 |
| 방문 | HUG 지사 |

### 5.3 필요 서류

| 서류 | 용도 |
|------|------|
| 임대차계약서 | 계약 내용 확인 |
| 등기부등본 | 권리관계 확인 |
| 주민등록등본 | 전입 확인 |
| 신분증 | 본인 확인 |
| 확정일자 증명 | 확정일자 확인 |

### 5.4 심사 기간

| 단계 | 기간 |
|------|------|
| 접수~심사 | 3~7일 |
| 승인~발급 | 1~3일 |
| 총 소요 | **약 1~2주** |

## 6. 가입 조건 상세

### 6.1 시세 확인

| 확인 방법 | 내용 |
|----------|------|
| HUG 시세 | HUG 자체 시세 적용 |
| KB시세 | 참고 |
| 감정평가 | 필요 시 |

### 6.2 90% 계산

**시세 4억원, 근저당 1억원**

| 항목 | 금액 |
|------|------|
| 시세 | 4억원 |
| 90% 한도 | 3.6억원 |
| 근저당 | 1억원 |
| 가입 가능 보증금 | 3.6억 - 1억 = **2.6억원** |

### 6.3 다가구 특례

| 항목 | 내용 |
|------|------|
| 다가구 | 선순위 보증금 합산 확인 |
| 확인 방법 | 확정일자부 열람 |
| 합산 | (근저당+선순위+내 보증금) ≤ 90% |

## 7. 가입 거절 사유

### 7.1 주요 거절 사유

| 사유 | 설명 |
|------|------|
| 보증금 > 시세 90% | 과다 보증금 |
| 선순위 과다 | 근저당+선순위 많음 |
| 권리 하자 | 가압류, 압류 등 |
| 신탁 물건 | 신탁 동의서 없음 |
| 위반건축물 | 불법 건축물 |

### 7.2 거절 시 대응

| 대응 | 방법 |
|------|------|
| 계약 재검토 | 보증금 낮추기 협의 |
| SGI 시도 | HUG 거절 시 SGI |
| 계약 철회 | 위험한 물건 피하기 |

## 8. 보증금 청구

### 8.1 청구 조건

| 조건 | 내용 |
|------|------|
| 계약 종료 | 임대차 만료 또는 해지 |
| 보증금 미반환 | 반환 기한 경과 |
| 보증 유효 | 보증 기간 내 |

### 8.2 청구 절차

| 단계 | 내용 |
|------|------|
| 1 | 임대인에게 반환 청구 |
| 2 | 내용증명 발송 |
| 3 | 보증기관에 청구 |
| 4 | 심사 (2~4주) |
| 5 | 보증금 지급 |

### 8.3 필요 서류

| 서류 | 용도 |
|------|------|
| 보증서 원본 | 보험 증빙 |
| 임대차계약서 | 계약 증빙 |
| 내용증명 사본 | 청구 증빙 |
| 주민등록등본 | 전입 확인 |
| 통장 사본 | 지급 계좌 |

### 8.4 지급 기간

| 단계 | 기간 |
|------|------|
| 접수~심사 | 2~3주 |
| 심사~지급 | 1주 |
| 총 소요 | **약 3~4주** |

## 9. 주의사항

### 9.1 가입 전 확인

| 확인 | 내용 |
|------|------|
| 가입 가능성 | 계약 전 미리 확인 |
| 시세 대비 | 보증금 적정성 |
| 권리관계 | 등기부등본 확인 |

### 9.2 가입 후 주의

| 주의 | 내용 |
|------|------|
| 보증서 보관 | 원본 안전 보관 |
| 전입 유지 | 전출하면 보증 무효 가능 |
| 갱신 시 | 보증 갱신 필요 여부 확인 |

### 9.3 보증 면책 사유

| 사유 | 면책 |
|------|------|
| 고의 허위 | 허위 서류 제출 |
| 대항력 상실 | 전출로 대항력 없음 |
| 보증료 미납 | 보증료 연체 |

## 10. 자주 묻는 질문

### 10.1 전세대출 받으면 자동 가입?

| 답변 | 설명 |
|------|------|
| ⚠️ 다를 수 있음 | 전세대출 보증 ≠ 전세보증보험 |
| 확인 필요 | 별도 가입 여부 확인 |

### 10.2 갱신 시 다시 가입?

| 답변 | 설명 |
|------|------|
| ⚠️ 확인 필요 | 기존 보증 유효기간 확인 |
| 보증금 증액 시 | 추가 가입 필요 |

### 10.3 집주인이 거부하면?

| 답변 | 설명 |
|------|------|
| 가입 가능 | 임차인 단독 가입 가능 |
| 협조 필요 | 일부 서류 협조 필요 |
