# 연말정산 세액공제 소득공제 차이

연말정산에서 **세액공제**와 **소득공제**는 세금을 줄이는 방식이 다릅니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 차이를 정리합니다.

## 1. 기본 개념

### 1.1 소득공제

| 항목 | 내용 |
|------|------|
| 정의 | **과세표준**을 줄이는 공제 |
| 효과 | 세금 계산 기준금액 감소 |
| 실제 절세 | 공제액 × **세율** |

### 1.2 세액공제

| 항목 | 내용 |
|------|------|
| 정의 | **세금**에서 직접 빼는 공제 |
| 효과 | 계산된 세금 직접 감소 |
| 실제 절세 | **공제액 그대로** |

## 2. 세금 계산 구조

### 2.1 계산 순서

```
총급여
 - 근로소득공제
= 근로소득금액
 - 소득공제 ← [여기서 적용]
= 과세표준
 × 세율
= 산출세액
 - 세액공제 ← [여기서 적용]
= 결정세액
```

### 2.2 흐름도

| 단계 | 내용 | 공제 유형 |
|------|------|----------|
| 1 | 총급여 | - |
| 2 | 근로소득금액 | - |
| 3 | 과세표준 | **소득공제** |
| 4 | 산출세액 | - |
| 5 | 결정세액 | **세액공제** |

## 3. 절세 효과 비교

### 3.1 소득공제 100만원

| 세율 | 절세 효과 |
|------|----------|
| 6% | 6만원 |
| 15% | 15만원 |
| 24% | 24만원 |
| 35% | 35만원 |

**고소득자에게 유리**

### 3.2 세액공제 100만원

| 세율 | 절세 효과 |
|------|----------|
| 6% | **100만원** |
| 15% | **100만원** |
| 24% | **100만원** |
| 35% | **100만원** |

**모든 소득수준에 동일**

## 4. 주요 소득공제 항목

### 4.1 인적공제

| 항목 | 공제액 |
|------|--------|
| 기본공제 (1인당) | 150만원 |
| 경로우대 | 100만원 |
| 장애인 | 200만원 |
| 부녀자 | 50만원 |
| 한부모 | 100만원 |

### 4.2 특별소득공제

| 항목 | 공제율/한도 |
|------|------------|
| 신용카드 | 15% |
| 체크카드·현금영수증 | 30% |
| 전통시장·대중교통 | 40% |
| 주택청약 | 40% (240만원 한도) |
| 주택임차차입금 | 40% (400만원 한도) |

## 5. 주요 세액공제 항목

### 5.1 특별세액공제

| 항목 | 공제율 | 한도 |
|------|--------|------|
| 보험료 | 12% | 100만원 |
| 의료비 | 15% | 한도 있음 |
| 교육비 | 15% | 한도 있음 |
| 기부금 | 15~30% | 소득 기준 |

### 5.2 기타 세액공제

| 항목 | 공제율 | 한도 |
|------|--------|------|
| 연금저축 | 12~15% | 600만원 |
| IRP | 12~15% | 900만원 |
| 월세 | 15~17% | 1,000만원 |
| 자녀 세액공제 | - | 자녀 수 기준 |

## 6. 계산 예시

### 6.1 소득공제 효과

**연봉 5,000만원, 신용카드 공제 100만원**

1. 과세표준: 3,000만원 → 2,900만원
2. 세율 15% 적용
3. 절세 효과: 100만원 × 15% = **15만원**

### 6.2 세액공제 효과

**연봉 5,000만원, 의료비 세액공제 15만원**

1. 산출세액: 300만원
2. 세액공제 적용: 300만원 - 15만원
3. 절세 효과: **15만원 (그대로)**

## 7. 공제 유형별 분류

### 7.1 소득공제 항목

| 구분 | 항목 |
|------|------|
| 인적공제 | 기본공제, 추가공제 |
| 소비 | 신용카드, 체크카드, 현금영수증 |
| 주거 | 주택청약, 주택임차차입금 |

### 7.2 세액공제 항목

| 구분 | 항목 |
|------|------|
| 특별 | 보험료, 의료비, 교육비, 기부금 |
| 연금 | 연금저축, IRP |
| 주거 | 월세 |
| 가족 | 자녀, 출산·입양 |

## 8. 절세 전략

### 8.1 소득공제 활용

| 전략 | 내용 |
|------|------|
| 고소득자 | 소득공제 효과 큼 |
| 카드 사용 | 체크카드·현금영수증 위주 |
| 전통시장 | 40% 공제율 활용 |

### 8.2 세액공제 활용

| 전략 | 내용 |
|------|------|
| 모든 소득 | 동일한 효과 |
| 연금저축 | 최대 납입 권장 |
| 월세 | 소득 기준 확인 |

## 9. 주의사항

### 9.1 공제 한도

- 각 항목별 **한도** 확인
- 한도 초과분은 공제 불가

### 9.2 결정세액 0원

- 세액공제가 산출세액보다 많으면 → **환급**
- 소득공제는 마이너스 과세표준 불가

### 9.3 공제 순서

1. 소득공제 먼저 적용
2. 세액공제 나중에 적용
3. 순서에 따라 결과 달라질 수 있음
