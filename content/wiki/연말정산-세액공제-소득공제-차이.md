연말정산에서 세액공제와 소득공제는 세금을 줄이는 방식이 달라요. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 차이를 정리해 볼게요.

## 기본 개념

[소득세법 제59조](https://www.law.go.kr/법령/소득세법)에 따른 공제 방식이에요.

소득공제는 과세표준을 줄이는 공제예요.

| 항목 | 내용 |
|------|------|
| 정의 | 과세표준을 줄이는 공제 |
| 효과 | 세금 계산 기준금액 감소 |
| 실제 절세 | 공제액 x 세율 |

세액공제는 세금에서 직접 빼는 공제예요.

| 항목 | 내용 |
|------|------|
| 정의 | 세금에서 직접 빼는 공제 |
| 효과 | 계산된 세금 직접 감소 |
| 실제 절세 | 공제액 그대로 |

## 세금 계산 구조

총급여에서 세금이 계산되는 순서예요.

| 단계 | 내용 | 공제 유형 |
|------|------|----------|
| 1 | 총급여 | - |
| 2 | 근로소득금액 | - |
| 3 | 과세표준 | 소득공제 |
| 4 | 산출세액 | - |
| 5 | 결정세액 | 세액공제 |

소득공제는 과세표준 계산 단계에서, 세액공제는 결정세액 계산 단계에서 적용돼요.

## 절세 효과 비교

소득공제 100만원의 절세 효과는 세율에 따라 달라요.

| 세율 | 절세 효과 |
|------|----------|
| 6% | 6만원 |
| 15% | 15만원 |
| 24% | 24만원 |
| 35% | 35만원 |

고소득자에게 유리해요.

세액공제 100만원의 절세 효과는 모든 소득에 동일해요.

| 세율 | 절세 효과 |
|------|----------|
| 6% | 100만원 |
| 15% | 100만원 |
| 24% | 100만원 |
| 35% | 100만원 |

모든 소득수준에 동일한 효과예요.

## 주요 소득공제 항목

인적공제 항목이에요.

| 항목 | 공제액 |
|------|--------|
| 기본공제 (1인당) | 150만원 |
| 경로우대 | 100만원 |
| 장애인 | 200만원 |
| 부녀자 | 50만원 |
| 한부모 | 100만원 |

특별소득공제 항목이에요.

| 항목 | 공제율/한도 |
|------|------------|
| 신용카드 | 15% |
| 체크카드·현금영수증 | 30% |
| 전통시장·대중교통 | 40% |
| 주택청약 | 40% (240만원 한도) |
| 주택임차차입금 | 40% (400만원 한도) |

## 주요 세액공제 항목

특별세액공제 항목이에요.

| 항목 | 공제율 | 한도 |
|------|--------|------|
| 보험료 | 12% | 100만원 |
| 의료비 | 15% | 한도 있음 |
| 교육비 | 15% | 한도 있음 |
| 기부금 | 15~30% | 소득 기준 |

기타 세액공제 항목이에요.

| 항목 | 공제율 | 한도 |
|------|--------|------|
| 연금저축 | 12~15% | 600만원 |
| IRP | 12~15% | 900만원 |
| 월세 | 15~17% | 1,000만원 |
| 자녀 세액공제 | - | 자녀 수 기준 |

## 계산 예시

연봉 5,000만원, 신용카드 소득공제 100만원인 경우예요.

| 항목 | 내용 |
|------|------|
| 과세표준 | 3,000만원 → 2,900만원 |
| 세율 | 15% |
| 절세 효과 | 100만원 x 15% = 15만원 |

연봉 5,000만원, 의료비 세액공제 15만원인 경우예요.

| 항목 | 내용 |
|------|------|
| 산출세액 | 300만원 |
| 세액공제 | 15만원 |
| 절세 효과 | 15만원 (그대로) |

## 공제 유형별 분류

소득공제 항목이에요.

| 구분 | 항목 |
|------|------|
| 인적공제 | 기본공제, 추가공제 |
| 소비 | 신용카드, 체크카드, 현금영수증 |
| 주거 | 주택청약, 주택임차차입금 |

세액공제 항목이에요.

| 구분 | 항목 |
|------|------|
| 특별 | 보험료, 의료비, 교육비, 기부금 |
| 연금 | 연금저축, IRP |
| 주거 | 월세 |
| 가족 | 자녀, 출산·입양 |

## 절세 전략

소득공제 활용 전략이에요.

| 전략 | 내용 |
|------|------|
| 고소득자 | 소득공제 효과가 커요 |
| 카드 사용 | 체크카드·현금영수증 위주로 |
| 전통시장 | 40% 공제율 활용 |

세액공제 활용 전략이에요.

| 전략 | 내용 |
|------|------|
| 모든 소득 | 동일한 효과 |
| 연금저축 | 최대 납입 권장 |
| 월세 | 소득 기준 확인 |

## 주의사항

각 항목별 한도를 확인해야 해요. 한도 초과분은 공제받을 수 없어요.

세액공제가 산출세액보다 많으면 환급받을 수 있어요. 소득공제는 마이너스 과세표준이 불가능해요.

소득공제가 먼저 적용되고 세액공제가 나중에 적용돼요.
