# 연말정산 이직자

이직한 경우 연말정산 방법을 정리합니다.

## 1. 기본 원칙

| 항목 | 내용 |
|------|------|
| 신고 회사 | 12월 31일 재직 회사 |
| 합산 | 전 직장 + 현 직장 소득 |

## 2. 필요 서류

| 서류 | 발급처 |
|------|--------|
| 근로소득 원천징수영수증 | 전 직장 |
| 소득자료 | 홈택스 |

## 3. 제출 방법

| 방법 | 절차 |
|------|------|
| 전 직장 요청 | 원천징수영수증 발급 요청 |
| 홈택스 | 지급명세서 조회 |
| 현 직장 제출 | 서류 제출 |

## 4. 주의사항

| 주의 | 내용 |
|------|------|
| 미제출 시 | 5월 종합소득세 신고 필요 |
| 중복공제 | 전·현 직장 중복 확인 |
