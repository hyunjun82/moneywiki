"의료비 200만원 썼는데 실손보험으로 150만원 받았어요. 공제 얼마예요?"

실손보험금을 빼고 50만원만 공제 대상이에요.

## 실손보험금 차감이 뭔가요

실손의료보험으로 보상받은 금액은 [의료비 공제](/w/연말정산-의료비-공제)에서 빼야 해요. 본인이 실제 부담한 금액만 공제 대상이에요.

| 항목 | 금액 |
|------|------|
| 의료비 지출 | 200만원 |
| 실손보험금 | 150만원 |
| 공제 대상 | 50만원 |

## 왜 빼야 해요

의료비 공제는 본인 부담금에 대한 공제예요.

| 구분 | 공제 |
|------|------|
| 본인 부담금 | 공제 대상 |
| 보험 보상금 | 공제 대상 X |

보험으로 보상받았으면 본인이 부담한 게 아니니까 공제 못 받아요.

## 간소화에 반영돼요

2023년부터 실손보험금이 간소화 자료에 자동 반영돼요.

| 연도 | 반영 |
|------|------|
| 2022년 이전 | 직접 확인 필요 |
| 2023년 이후 | 자동 반영 |

다만 금액이 정확한지 직접 확인하세요. 보험사 청구 시기에 따라 차이가 있을 수 있어요.

## 계산은 어떻게 해요

지출한 의료비에서 보상받은 금액을 빼요.

| 예시 | 금액 |
|------|------|
| 총 의료비 | 500만원 |
| 실손보험금 수령 | 300만원 |
| 공제 대상 의료비 | 200만원 |

[연말정산](/w/연말정산) 간소화에서 차감 후 금액을 확인하세요.

## 차감 안 하면 어떻게 돼요

[과다공제](/w/연말정산-과다공제-사례)로 추징당해요.

| 결과 | 내용 |
|------|------|
| 추징 | 과다공제분 세금 납부 |
| 가산세 | 과소신고 [가산세](/w/연말정산-가산세) 10% |

국세청이 보험사 자료와 대조해서 자동으로 적발해요.

## 확인하는 방법이요

간소화 서비스에서 확인하세요.

**확인 절차**

1. 홈택스 → 연말정산 간소화
2. 의료비 항목 조회
3. 실손보험금 수령액 확인
4. 차감 후 금액 확인

보험사 앱에서도 실손보험금 수령 내역을 확인할 수 있어요.
