의료비 많이 썼으니까 공제 많이 받겠지? 잠깐, 실손보험 받은 건 빼야 해요.

✨핵심 요약!

1. 실손보험 수령액은 의료비 공제에서 차감
2. 간소화 서비스에서 자동 반영
3. 실제 부담액만 공제 가능

## 목차

## 실손보험금 차감이란

의료비 공제는 본인이 실제 부담한 금액만 대상이에요.

실손보험으로 돌려받았으면 그건 내가 부담한 게 아니잖아요. 그래서 공제 대상에서 빼야 해요.

*차감 원리

| 항목 | 금액 |
|------|------|
| 지출 의료비 | 500만원 |
| 실손보험 수령 | 300만원 |
| 공제 대상 | 200만원 |

실제 내 돈으로 부담한 200만원만 공제돼요.

## 자동 반영 시스템

간소화 서비스에서 실손보험금이 자동으로 차감돼요.

| 연도 | 반영 방식 |
|------|----------|
| 2019년 이후 | 자동 반영 |
| 2018년 이전 | 수동 입력 |

보험회사가 국세청에 지급 내역을 통보해요. 간소화 자료에 이미 실손보험금이 빠진 금액이 표시돼요.

## 공제 계산 예시

실손보험금 차감 후 공제액을 계산해 볼게요.

*예시 1: 총급여 5,000만원

| 항목 | 금액 |
|------|------|
| 지출 의료비 | 400만원 |
| 실손보험 수령 | 200만원 |
| 공제 대상 의료비 | 200만원 |
| 총급여 3% 기준액 | 150만원 |
| 실제 공제 대상 | 50만원 |
| 공제율 | 15% |
| 세액공제액 | 7.5만원 |

총급여의 3%를 넘는 부분만 공제되니까 실손보험을 빼면 공제액이 크게 줄어요.

## 실손보험 미청구 시

실손보험 청구를 안 해서 수령을 안 했다면 전액 공제돼요.

| 상황 | 공제 |
|------|------|
| 실손보험 수령 | 수령액 차감 |
| 실손보험 미청구 | 전액 공제 |

단, 연말정산 후에 실손보험을 청구하면 수정신고해야 할 수 있어요.

## 과다공제 주의

실손보험금을 안 빼고 공제받으면 과다공제예요.

| 상황 | 결과 |
|------|------|
| 실손보험금 미차감 | 과다공제 |
| 적발 시 | 세금 + 가산세 추징 |

국세청이 보험회사 자료를 갖고 있어서 대조하면 바로 나와요.

## 확인 방법

간소화 서비스에서 의료비 자료를 보면 실손보험 차감 내역이 나와요.

| 확인 항목 | 위치 |
|----------|------|
| 지출 의료비 | 의료비 항목 |
| 실손보험 수령액 | 보험금 수령 항목 |
| 공제 대상 의료비 | 차감 후 금액 |

자동 반영이 안 된 경우 직접 차감해서 신고해야 해요.
