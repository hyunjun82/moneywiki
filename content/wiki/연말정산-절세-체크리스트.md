연말정산 절세 체크리스트로 놓치기 쉬운 공제 항목을 확인해 보세요. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 내용을 정리해 볼게요.

## 인적공제 확인

인적공제 확인 항목이에요.

| 항목 | 확인 사항 |
|------|----------|
| 부양가족 소득 | 소득금액 100만원 이하 확인 |
| 부양가족 나이 | 직계존속 60세 이상, 직계비속 20세 이하 확인 |
| 경로우대 대상 | 70세 이상 부양가족 확인 |
| 장애인 증명서 | 장애인 부양가족 증명서 준비 |

## 놓치기 쉬운 공제

놓치기 쉬운 공제 항목이에요.

| 항목 | 한도 | 비고 |
|------|------|------|
| 안경 구입비 | 50만원 | 의료비 공제 |
| 교복 구입비 | 50만원 | 교육비 공제 |
| 월세 | 별도 서류 제출 | 세액공제 |
| 기부금 영수증 | 종류별 다름 | 영수증 확인 |

## 서류 준비

필요한 서류 목록이에요.

| 서류 | 용도 |
|------|------|
| 주민등록등본 | 부양가족 확인 |
| 가족관계증명서 | 가족 관계 확인 |
| 월세 계약서·이체내역 | 월세 세액공제 |
| 장애인증명서 | 장애인 공제 |

## 간소화 자료 확인

간소화 자료 확인 항목이에요.

| 항목 | 내용 |
|------|------|
| 조회 시점 | 1월 15일 이후 |
| 누락 항목 | 직접 영수증 제출 |
| 가족 자료 | 제공 동의 확인 |

## 제출 전 최종 확인

제출 전 확인할 항목이에요.

| 항목 | 확인 사항 |
|------|----------|
| 부양가족 중복 | 다른 가족과 중복 공제 확인 |
| 공제 증빙 | 필요 서류 첨부 확인 |
| 제출 기한 | 회사별 제출 기한 확인 |

## 주의사항

부양가족은 한 명만 공제받을 수 있어요. 가족 간에 누가 공제받을지 미리 정하세요.

소득이 있는 부양가족은 소득금액 100만원(총급여 500만원) 이하인지 확인하세요.

간소화 서비스에서 누락된 항목은 직접 영수증을 발급받아 제출해야 해요.
