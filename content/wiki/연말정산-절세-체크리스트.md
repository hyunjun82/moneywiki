# 연말정산 절세 체크리스트

연말정산 절세 체크리스트를 정리합니다.

## 1. 인적공제 확인

| 체크 | 항목 |
|------|------|
| ☐ | 부양가족 소득 요건 확인 |
| ☐ | 나이 요건 확인 |
| ☐ | 경로우대 대상 확인 |
| ☐ | 장애인 증명서 준비 |

## 2. 놓치기 쉬운 공제

| 체크 | 항목 |
|------|------|
| ☐ | 안경 구입비 (50만원 한도) |
| ☐ | 교복 구입비 (50만원 한도) |
| ☐ | 월세 (별도 서류 제출) |
| ☐ | 기부금 영수증 |

## 3. 서류 준비

| 체크 | 항목 |
|------|------|
| ☐ | 주민등록등본 |
| ☐ | 가족관계증명서 |
| ☐ | 월세 계약서·이체내역 |
| ☐ | 장애인증명서 |

## 4. 간소화 자료 확인

| 체크 | 항목 |
|------|------|
| ☐ | 1월 15일 이후 조회 |
| ☐ | 누락 항목 확인 |
| ☐ | 가족 자료 제공 동의 |

## 5. 제출 전 최종 확인

| 체크 | 항목 |
|------|------|
| ☐ | 부양가족 중복 확인 |
| ☐ | 공제 증빙 첨부 |
| ☐ | 서류 제출 기한 확인 |
