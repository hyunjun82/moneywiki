# 연말정산 취학전 아동 교육비

취학전 아동 교육비는 초등학교 입학 전 자녀의 교육비에 대해 **15% 세액공제**를 받는 제도입니다. [국세청 교육비 공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 대상자

[소득세법 시행령 제118조의6](https://www.law.go.kr/법령/소득세법시행령)에 따른 대상자입니다.

| 요건 | 조건 |
|------|------|
| 대상 | **초등학교 입학 전** 자녀 |
| 나이 | 만 6세 이하 (입학 전까지) |
| 기본공제 | 기본공제 대상자여야 함 |

## 2. 공제율과 한도

| 항목 | 내용 |
|------|------|
| 공제율 | **15%** |
| 연간 한도 | **300만원** |
| 최대 공제액 | 300만원 × 15% = **45만원** |

## 3. 공제 대상 교육비

### 3.1 공제 가능 항목

| 항목 | 공제 여부 |
|------|----------|
| 유치원 교육비 | ✅ 가능 |
| 어린이집 보육료 | ✅ 가능 |
| **학원비** | ✅ 가능 (취학전 특례) |
| 체육시설 수강료 | ✅ 가능 |
| 방문학습지 | ❌ 불가 |

### 3.2 학원비 상세

| 학원 종류 | 공제 여부 |
|----------|----------|
| 태권도 | ✅ 가능 |
| 미술 | ✅ 가능 |
| 영어 | ✅ 가능 |
| 피아노 | ✅ 가능 |
| 수영 | ✅ 가능 |
| 발레 | ✅ 가능 |

**주의**: 초중고 학생 학원비는 공제 불가

## 4. 정부지원금과 관계

### 4.1 누리과정 지원

| 구분 | 공제 대상 |
|------|----------|
| 정부지원금 | ❌ 공제 불가 |
| **본인부담분** | ✅ 공제 가능 |

### 4.2 계산 예시

| 항목 | 금액 |
|------|------|
| 유치원비 총액 | 월 40만원 |
| 정부지원금 | 월 30만원 |
| **본인부담분** | 월 10만원 |
| 연간 공제 대상 | **120만원** |

## 5. 공제 계산 예시

### 5.1 유치원 + 학원

**자녀 1명 (만 5세)**
- 유치원 본인부담: 연 120만원
- 태권도 학원: 연 100만원
- 영어 학원: 연 150만원

| 항목 | 금액 |
|------|------|
| 총 교육비 | 370만원 |
| 공제 한도 | 300만원 |
| 세액공제 | 300만원 × 15% = **45만원** |

### 5.2 어린이집

**자녀 1명 (만 3세)**
- 어린이집 보육료 본인부담: 연 80만원
- 체육시설: 연 60만원

| 항목 | 금액 |
|------|------|
| 총 교육비 | 140만원 |
| 세액공제 | 140만원 × 15% = **21만원** |

## 6. 증빙서류

### 6.1 연말정산 간소화

| 교육기관 | 조회 |
|----------|------|
| 유치원 | ✅ 자동 조회 |
| 어린이집 | ✅ 자동 조회 |
| 학원 | 일부 자동, 일부 직접 제출 |

### 6.2 직접 제출 필요

| 상황 | 서류 |
|------|------|
| 간소화 누락 | 교육비납입증명서 |
| 학원비 | 학원 수강료 영수증 |

### 6.3 학원 영수증 요건

- 학원명, 사업자번호
- 수강자 성명
- 수강 기간, 금액
- 학원 직인

## 7. 자녀가 여러 명인 경우

### 7.1 한도 적용

| 자녀 | 한도 |
|------|------|
| 자녀 1명 | 300만원 |
| 자녀 2명 | **각각** 300만원 |

### 7.2 계산 예시

**취학전 자녀 2명**
- 첫째 교육비: 400만원 → 300만원 (한도)
- 둘째 교육비: 200만원 → 200만원

| 항목 | 금액 |
|------|------|
| 공제 대상 | 500만원 |
| 세액공제 | 500만원 × 15% = **75만원** |

## 8. 초등학교 입학 연도

### 8.1 입학 전까지 공제

| 시기 | 공제 |
|------|------|
| 1~2월 유치원비 | ✅ 취학전 아동 (300만원 한도, 학원비 포함) |
| 3월~ 초등학교비 | ✅ 초중고 (300만원 한도, 학원비 제외) |

### 8.2 같은 해 두 가지 적용

입학 연도에는 **입학 전/후** 교육비 각각 공제

## 9. 주의사항

### 9.1 기본공제 대상자

- 자녀가 **기본공제 대상자**여야 교육비 공제 가능
- 소득금액 100만원 초과 자녀 → 교육비 공제 불가

### 9.2 맞벌이 부부

- 기본공제 받는 배우자가 교육비도 공제
- 분리 공제 불가

### 9.3 학원비 조건

- 학원이 **사업자등록**된 학원이어야 함
- 개인과외는 공제 불가
