1인 가구 싱글 절세 전략을 정리해 볼게요. 부양가족 공제가 없어도 잘 활용하면 환급받을 수 있어요.

## 핵심 절세 항목

주요 절세 항목이에요.

| 항목 | 최대 공제 |
|------|----------|
| 연금저축+IRP | 900만원 (세액공제) |
| 신용카드 | 300만원 (소득공제) |
| 의료비 | 한도 없음 |
| 보장성보험 | 100만원 |

## 연금저축 전략

연금저축 활용법이에요.

| 전략 | 내용 |
|------|------|
| 최대 납입 | 연 600만원 |
| IRP 추가 | 연 300만원 |
| 공제율 | 13.2~16.5% |
| 최대 환급 | 148.5만원 |

## 신용카드 전략

신용카드 공제 활용법이에요.

| 전략 | 내용 |
|------|------|
| 최저사용 | 총급여 25% 채우기 |
| 체크카드 | 25% 초과분은 체크카드 |
| 전통시장 | 40% 공제율 |

## 의료비 전략

의료비 공제 활용법이에요.

| 전략 | 내용 |
|------|------|
| 본인 의료비 | 한도 없이 공제 |
| 문턱 | 총급여 3% 초과분 |
| 미용·성형 | 공제 불가 |

## 추가 팁

추가 절세 팁이에요.

| 팁 | 내용 |
|-----|------|
| 월세 | 세액공제 가능 |
| 기부금 | 15~30% 세액공제 |
| 교육비 | 본인 학원비 불가 |
