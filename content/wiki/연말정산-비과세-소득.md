# 연말정산 비과세 소득

비과세 소득은 세금이 부과되지 않는 소득으로, 총급여에서 제외됩니다. [소득세법 제12조](https://www.law.go.kr/법령/소득세법)에 따른 주요 항목을 정리합니다.

## 1. 비과세 소득이란

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 의미 | **세금이 부과되지 않는** 소득 |
| 효과 | 총급여에서 제외 → 과세표준 감소 |
| 혜택 | 실수령액 증가 |

### 1.2 총급여 계산

```
총급여 = 연간 급여 - 비과세 소득
```

## 2. 주요 비과세 항목

### 2.1 식대

| 항목 | 내용 |
|------|------|
| 한도 | 월 **20만원** |
| 조건 | 식사 제공 없이 현금으로 지급 |
| 초과분 | 과세 (20만원 초과분만) |

### 2.2 자가운전보조금

| 항목 | 내용 |
|------|------|
| 한도 | 월 **20만원** |
| 조건 | 본인 소유 차량으로 업무 수행 |
| 대상 | 업무용 차량 미지급 근로자 |

### 2.3 출산·보육수당

| 항목 | 내용 |
|------|------|
| 한도 | 월 **20만원** (2024년부터) |
| 대상 | 6세 이하 자녀 |
| 조건 | 회사에서 지급하는 보육 관련 수당 |

### 2.4 연구보조비

| 항목 | 내용 |
|------|------|
| 한도 | 월 **20만원** |
| 대상 | 기업부설연구소 연구원 |
| 조건 | 연구활동비 명목 지급 |

## 3. 실비변상적 비과세

### 3.1 일직료·숙직료

| 항목 | 비과세 |
|------|--------|
| 일직료 | 실비 범위 내 |
| 숙직료 | 실비 범위 내 |
| 야근식대 | 10만원 이내 |

### 3.2 여비

| 항목 | 비과세 |
|------|--------|
| 출장비 | 실비 범위 내 |
| 교통비 | 실비 정산 |
| 숙박비 | 실비 정산 |

## 4. 기타 비과세

### 4.1 생산직 근로자 초과근무수당

| 항목 | 내용 |
|------|------|
| 대상 | 월정액급여 210만원 이하 |
| 한도 | 연 **240만원** |
| 조건 | 생산직, 광업, 어업 등 |

### 4.2 국외근로소득

| 항목 | 내용 |
|------|------|
| 한도 | 월 **100~300만원** |
| 조건 | 국외에서 근로 제공 |
| 차등 | 원양어업 300만원, 국외건설 300만원, 기타 100만원 |

### 4.3 벽지수당

| 항목 | 내용 |
|------|------|
| 한도 | 월 **20만원** |
| 대상 | 도서·벽지 근무자 |

## 5. 비과세 한도 정리

### 5.1 월 한도 비과세

| 항목 | 월 한도 | 연 한도 |
|------|---------|---------|
| 식대 | 20만원 | 240만원 |
| 자가운전보조금 | 20만원 | 240만원 |
| 출산·보육수당 | 20만원 | 240만원 |
| 연구보조비 | 20만원 | 240만원 |

### 5.2 연 한도 비과세

| 항목 | 연 한도 |
|------|---------|
| 생산직 야간근로 | 240만원 |
| 학자금 | 비과세 (한도 없음) |

## 6. 비과세 효과 계산

### 6.1 식대 비과세 효과

**식대 20만원 비과세 적용 시** (연간)

| 과세표준 구간 | 세율 | 절감액 (연) |
|--------------|------|-------------|
| 1,400만원 이하 | 6% | 14.4만원 |
| 5,000만원 이하 | 15% | 36만원 |
| 8,800만원 이하 | 24% | 57.6만원 |

### 6.2 복합 비과세 효과

**식대 20만원 + 자가운전 20만원** (연간)

- 연간 비과세: 480만원
- 세율 15% 기준: **72만원 절감**

## 7. 주의사항

### 7.1 한도 초과 시

| 상황 | 처리 |
|------|------|
| 식대 25만원 | 20만원 비과세, 5만원 과세 |
| 자가운전 30만원 | 20만원 비과세, 10만원 과세 |

### 7.2 증빙 요구

| 항목 | 증빙 |
|------|------|
| 자가운전보조금 | 차량등록증, 운행기록 |
| 연구보조비 | 연구활동 증빙 |
| 출장비 | 영수증, 출장보고서 |

### 7.3 회사 정책 확인

- 비과세 항목은 **회사가 지급해야** 적용
- 급여명세서에서 비과세 항목 확인

## 8. 비과세 vs 소득공제

### 8.1 차이점

| 구분 | 비과세 | 소득공제 |
|------|--------|---------|
| 적용 시점 | 총급여 계산 전 | 총급여 계산 후 |
| 효과 | 총급여 자체 감소 | 과세표준 감소 |
| 예시 | 식대 20만원 | 신용카드 공제 |

### 8.2 비과세의 장점

- 총급여 기준 공제 (카드 25% 등)에도 유리
- 건강보험료 산정에도 유리
