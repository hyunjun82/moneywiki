# 연말정산 근로소득세액공제

근로소득세액공제는 근로소득이 있는 모든 근로자에게 자동으로 적용되는 세액공제입니다. [소득세법 제59조](https://www.law.go.kr/법령/소득세법)에 따른 내용을 정리합니다.

## 1. 공제 개요

### 1.1 기본 정보

| 항목 | 내용 |
|------|------|
| 적용 대상 | 근로소득 있는 모든 거주자 |
| 신청 필요 | ❌ 자동 적용 |
| 공제 방식 | 산출세액에서 직접 차감 |

### 1.2 공제의 의미

- 근로소득자에 대한 **기본적인 세금 경감**
- 사업소득자에 비해 경비 공제가 제한적인 점 보완
- **별도 신청 없이** 자동 적용

## 2. 공제율과 한도

### 2.1 공제 계산 구조

[소득세법 제59조]에 따른 공제율입니다.

| 산출세액 구간 | 공제율 |
|--------------|--------|
| 130만원 이하 | **55%** |
| 130만원 초과 | 71.5만원 + 130만원 초과분 × 30% |

### 2.2 총급여별 한도

| 총급여 | 공제 한도 |
|--------|-----------|
| 3,300만원 이하 | **74만원** |
| 3,300만원 초과 ~ 7,000만원 | 74만원 - (총급여-3,300만원)×0.8% |
| 7,000만원 초과 | 66만원 - (총급여-7,000만원)×50% |
| 1.2억원 초과 | **50만원** |

### 2.3 한도 계산 예시

| 총급여 | 공제 한도 |
|--------|-----------|
| 3,000만원 | 74만원 |
| 5,000만원 | 74만원 - 136,000원 = **60.4만원** |
| 7,000만원 | 66만원 |
| 1억원 | 66만원 - 150만원 = **51만원** |
| 1.5억원 | 50만원 (최저 한도) |

## 3. 공제 계산 예시

### 3.1 산출세액 100만원인 경우

**산출세액 100만원 (130만원 이하)**

- 공제율: 55%
- 공제액: 100만원 × 55% = **55만원**

### 3.2 산출세액 200만원인 경우

**산출세액 200만원 (130만원 초과)**

- 기본 공제: 71.5만원
- 초과분 공제: (200만원 - 130만원) × 30% = 21만원
- 합계: 71.5만원 + 21만원 = **92.5만원**
- 한도 적용 후: 총급여에 따라 다름

### 3.3 실제 적용 예시

**총급여 5,000만원, 산출세액 150만원**

1. 공제액 계산:
   - 71.5만원 + (150만원-130만원) × 30% = 77.5만원

2. 한도 확인:
   - 74만원 - (5,000만원-3,300만원) × 0.8% = 60.4만원

3. 최종 공제: **60.4만원** (한도 적용)

## 4. 다른 세액공제와 순서

### 4.1 세액공제 적용 순서

```
산출세액
  ↓
① 근로소득세액공제
② 자녀세액공제
③ 연금계좌세액공제
④ 특별세액공제 (보험료, 의료비, 교육비, 기부금)
⑤ 월세세액공제
⑥ 외국납부세액공제
  ↓
결정세액
```

### 4.2 세액공제 한도

- 산출세액을 초과하는 세액공제는 **환급되지 않음**
- 단, 자녀세액공제 등 일부는 환급 가능

## 5. 특수한 경우

### 5.1 일용근로자

| 구분 | 적용 |
|------|------|
| 일용근로자 | 근로소득세액공제 미적용 |
| 적용 세율 | 일률 6% (원천징수) |

### 5.2 중도 퇴사자

| 상황 | 적용 |
|------|------|
| 중도 퇴사 | 재직 기간 근로소득에 적용 |
| 이직 | 합산하여 연말정산 시 적용 |

### 5.3 휴직자

| 상황 | 적용 |
|------|------|
| 유급 휴직 | 근로소득 있으면 적용 |
| 무급 휴직 | 근로소득 없으면 미적용 |

## 6. 근로소득공제와 구분

### 6.1 두 공제의 차이

| 구분 | 근로소득공제 | 근로소득세액공제 |
|------|-------------|-----------------|
| 공제 유형 | **소득공제** | **세액공제** |
| 적용 단계 | 총급여 → 근로소득금액 | 산출세액 → 결정세액 |
| 공제 대상 | 총급여 | 산출세액 |

### 6.2 세금 계산 흐름

```
총급여
  ↓ 근로소득공제 (소득공제)
근로소득금액
  ↓ 인적공제, 특별소득공제 등
과세표준
  ↓ 세율 적용
산출세액
  ↓ 근로소득세액공제 (세액공제)
결정세액
```

## 7. 주의사항

### 7.1 자동 적용

- 별도 신청 **불필요**
- 회사 연말정산 시 **자동 계산**

### 7.2 환급 불가

- 근로소득세액공제는 산출세액 **한도 내**에서만 적용
- 초과분은 환급되지 않음

### 7.3 다른 소득과 합산

- 근로소득 외 다른 소득 있으면 종합소득세 신고
- 근로소득에 대해서만 공제 적용
