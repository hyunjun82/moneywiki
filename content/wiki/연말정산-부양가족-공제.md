# 연말정산 부양가족 공제

부양가족 기본공제는 가족 1인당 **150만원 소득공제**를 받는 제도입니다. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6591&cntntsId=7870)에 따른 조건을 정리합니다.

## 1. 기본공제 금액

[소득세법 제50조](https://www.law.go.kr/법령/소득세법)에 따른 기본공제 금액입니다.

| 구분 | 공제 금액 |
|------|-----------|
| 본인 | **150만원** |
| 기본공제 대상자 1인당 | **150만원** |

## 2. 기본공제 대상자

### 2.1 배우자

| 요건 | 조건 |
|------|------|
| 혼인 | 법률혼 배우자 |
| 나이 | **제한 없음** |
| 소득 | 연간 소득금액 100만원 이하 |

### 2.2 직계존속 (부모, 조부모)

| 요건 | 조건 |
|------|------|
| 나이 | **60세 이상** (1966년 12월 31일 이전 출생) |
| 소득 | 연간 소득금액 100만원 이하 |
| 동거 | 별거해도 가능 (실제 부양) |
| 배우자 부모 | 공제 가능 (시부모, 장인·장모) |

### 2.3 직계비속 (자녀, 손자녀)

| 요건 | 조건 |
|------|------|
| 나이 | **20세 이하** (2006년 1월 1일 이후 출생) |
| 소득 | 연간 소득금액 100만원 이하 |
| 입양자 | 공제 가능 |
| 위탁아동 | 공제 가능 |

### 2.4 형제자매

| 요건 | 조건 |
|------|------|
| 나이 | **20세 이하** 또는 **60세 이상** |
| 소득 | 연간 소득금액 100만원 이하 |
| 배우자 형제 | 공제 가능 (처남, 시누이 등) |

## 3. 소득 요건

### 3.1 연간 소득금액 100만원 이하

| 소득 유형 | 100만원 이하 기준 |
|-----------|------------------|
| 근로소득만 있는 경우 | 총급여 **500만원 이하** |
| 사업소득만 있는 경우 | 수입금액 - 필요경비 = 100만원 이하 |
| 기타소득만 있는 경우 | 기타소득금액 100만원 이하 |
| 연금소득만 있는 경우 | 연금소득금액 100만원 이하 |

### 3.2 분리과세 소득

다음 소득은 소득금액 계산에서 **제외**:

- 분리과세 이자·배당소득 (2천만원 이하)
- 분리과세 연금소득 (1,500만원 이하)
- 분리과세 기타소득 (300만원 이하)

## 4. 나이 요건 (2025년 귀속 기준)

| 대상 | 나이 요건 | 출생일 기준 |
|------|----------|------------|
| 배우자 | 제한 없음 | - |
| 직계존속 | 60세 이상 | 1965년 12월 31일 이전 |
| 직계비속 | 20세 이하 | 2005년 1월 1일 이후 |
| 형제자매 | 20세 이하 또는 60세 이상 | 위와 동일 |

**장애인**인 경우 **나이 제한 없음**

## 5. 추가공제

기본공제 대상자 중 다음에 해당하면 **추가 공제**:

| 구분 | 추가공제 금액 |
|------|--------------|
| 경로우대 (70세 이상) | **100만원** |
| 장애인 | **200만원** |
| 부녀자 (종합소득 3천만원 이하) | **50만원** |
| 한부모 | **100만원** |

## 6. 중복 공제 불가

### 6.1 원칙

한 명의 부양가족은 **1명의 근로자만** 공제 가능

### 6.2 맞벌이 부부

| 자녀 공제 | 가능 여부 |
|----------|----------|
| 남편이 자녀 기본공제 | ✅ |
| 동시에 아내도 자녀 기본공제 | ❌ 불가 |

### 6.3 형제가 부모님 공제

| 상황 | 처리 |
|------|------|
| 형과 동생 모두 공제 신청 | **한 명만** 가능 |
| 실제 부양자 판단 | 주민등록, 송금 내역 등 |

## 7. 공제 계산 예시

**근로자 본인, 배우자, 자녀 2명, 60세 이상 부모님 2명 부양**

| 대상 | 기본공제 | 추가공제 | 합계 |
|------|---------|---------|------|
| 본인 | 150만원 | - | 150만원 |
| 배우자 | 150만원 | - | 150만원 |
| 자녀 2명 | 300만원 | - | 300만원 |
| 부모님 2명 (70세) | 300만원 | 200만원 | 500만원 |
| **총 인적공제** | | | **1,100만원** |

## 8. 주의사항

### 8.1 소득금액 확인

- 부양가족의 **실제 소득** 확인 필수
- 소득금액 100만원 초과 시 공제 불가
- 추후 가산세 부과 가능

### 8.2 별거 부양가족

- 주민등록상 별거해도 **실제 부양** 시 공제 가능
- 송금 내역, 생활비 지원 등 증빙 권장

### 8.3 연도 중 사망·출생

- 사망 전일까지 부양 → 해당 연도 공제 가능
- 연도 중 출생 자녀 → 해당 연도 공제 가능
