# 최저임금

최저임금은 국가가 정한 최소한의 임금 수준입니다. [최저임금법](https://www.law.go.kr/법령/최저임금법)에 따른 내용을 정리합니다.

## 1. 2026년 최저임금

### 1.1 기본 금액

| 항목 | 금액 |
|------|------|
| 시급 | **10,030원** |
| 일급 (8시간) | **80,240원** |
| 주급 (40시간+주휴) | **481,440원** |
| 월급 (209시간) | **2,096,270원** |

### 1.2 월급 환산 기준

```
월 환산액 = 시급 × 209시간
209시간 = (주 40시간 + 주휴 8시간) × 4.345주
```

## 2. 적용 범위

| 대상 | 적용 |
|------|------|
| 정규직 | ✅ |
| 계약직 | ✅ |
| 파트타임 | ✅ |
| 아르바이트 | ✅ |
| 일용직 | ✅ |
| 외국인 근로자 | ✅ |

### 2.2 사업장 규모

| 규모 | 적용 |
|------|------|
| 대기업 | ✅ |
| 중소기업 | ✅ |
| 소규모 사업장 | ✅ |
| 1인 사업장 | ✅ |

## 3. 수습 감액

### 3.1 감액 조건

| 조건 | 내용 |
|------|------|
| 계약 기간 | **1년 이상** |
| 수습 기간 | 최초 **3개월** |
| 감액률 | **10%** |
| 감액 시급 | **9,027원** |

### 3.2 감액 제외

| 대상 | 감액 불가 |
|------|----------|
| 단순노무직 | ❌ |
| 1년 미만 계약 | ❌ |

### 3.3 단순노무직 종류

- 청소원
- 경비원
- 주차관리원
- 배달원
- 음식 서빙원

## 4. 최저임금 산입 범위

### 4.1 포함 항목

| 항목 | 포함 |
|------|------|
| 기본급 | ✅ |
| 고정수당 | ✅ |
| 상여금 (월 환산) | ⚠️ 일부 |
| 복리후생비 (월 환산) | ⚠️ 일부 |

### 4.2 제외 항목

| 항목 | 제외 |
|------|------|
| 연장·야간·휴일수당 | ❌ |
| 연차수당 | ❌ |
| 식대 (비과세) | ⚠️ 산입 비율 적용 |
| 숙박비 | ⚠️ 산입 비율 적용 |

### 4.3 상여금·복리후생 산입

| 항목 | 산입 기준 (2026년) |
|------|-------------------|
| 상여금 | 월 최저임금의 **5%** 초과분 |
| 복리후생 | 월 최저임금의 **1%** 초과분 |

## 5. 위반 시 처벌

### 5.1 형사 처벌

| 위반 | 처벌 |
|------|------|
| 최저임금 미달 지급 | 3년 이하 징역 또는 2,000만원 이하 벌금 |
| 최저임금 게시 위반 | 100만원 이하 과태료 |

### 5.2 민사 청구

| 권리 | 내용 |
|------|------|
| 임금 차액 청구 | 최저임금과 실제 임금 차액 |
| 지연이자 | 퇴직 후 연 20% |

## 7. 주휴수당

### 7.1 주휴수당 조건

| 조건 | 기준 |
|------|------|
| 주 근무시간 | **15시간 이상** |
| 근무일 출근 | 소정근로일 개근 |

### 7.2 계산 방법

```
주휴수당 = 1일 근로시간 × 시급
```

**예시**: 주 5일, 일 8시간 근무

| 항목 | 금액 |
|------|------|
| 주휴수당 | 8시간 × 10,030원 = **80,240원/주** |
| 월 주휴수당 | 약 **34.8만원** |
