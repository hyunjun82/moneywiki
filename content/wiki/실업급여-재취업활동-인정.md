# 실업급여 재취업활동 인정 기준

실업급여를 받으려면 **적극적인 재취업활동**을 해야 해요. 그런데 어떤 활동이 인정되고, 어떤 활동은 인정 안 될까요?

## 핵심 요약

- 실업인정일마다 **1회 이상** 재취업활동 필요
- 온라인 입사지원, 면접, 직업훈련 등이 인정됨
- 단순 채용공고 검색만으로는 인정 안 됨

---

## 재취업활동으로 인정되는 경우

### 1. 구직활동

| 활동 | 인정 여부 | 비고 |
|------|----------|------|
| **온라인 입사지원** | ✅ 인정 | 워크넷, 사람인, 잡코리아 등 |
| **이력서 제출** | ✅ 인정 | 방문, 우편, 이메일 모두 가능 |
| **면접 참석** | ✅ 인정 | 화상면접도 인정 |
| **채용설명회 참가** | ✅ 인정 | 온라인 설명회 포함 |
| **직업박람회 참가** | ✅ 인정 | |

### 2. 직업훈련 및 교육

| 활동 | 인정 여부 | 비고 |
|------|----------|------|
| **내일배움카드 훈련** | ✅ 인정 | 온라인, 오프라인 모두 |
| **국비지원 교육** | ✅ 인정 | |
| **직업상담** | ✅ 인정 | 고용센터 방문 상담 |
| **취업특강 수강** | ✅ 인정 | 고용센터 제공 프로그램 |

### 3. 창업 준비 활동

| 활동 | 인정 여부 | 비고 |
|------|----------|------|
| **창업교육 수강** | ✅ 인정 | 소상공인진흥공단 등 |
| **사업계획서 작성** | ✅ 인정 | 증빙 필요 |
| **창업박람회 참가** | ✅ 인정 | |

## 재취업활동으로 불인정되는 경우

### 절대 인정 안 되는 활동

| 활동 | 인정 여부 | 이유 |
|------|----------|------|
| **채용공고 검색만** | ❌ 불인정 | 실제 지원 필요 |
| **이력서 수정만** | ❌ 불인정 | 제출해야 인정 |
| **취업사이트 가입** | ❌ 불인정 | 구직활동 아님 |
| **친구에게 취업 부탁** | ❌ 불인정 | 증빙 불가 |
| **구인광고 스크랩** | ❌ 불인정 | 실제 지원 필요 |

### 조건부 불인정

| 활동 | 인정 여부 | 조건 |
|------|----------|------|
| **같은 회사 반복 지원** | ❌ 불인정 | 동일 회사 연속 지원 불가 |
| **허위 입사지원** | ❌ 불인정 | 부정수급으로 처벌 |
| **자격 미달 회사 지원** | ⚠️ 주의 | 합리적 범위 내 인정 |

## 구직활동 증빙 방법

### 온라인 입사지원
1. **워크넷**: 지원내역 자동 연동
2. **사람인/잡코리아**: 지원확인서 출력
3. **기업 채용사이트**: 지원완료 화면 캡처

### 오프라인 활동
- 면접확인서 (회사 직인)
- 직업훈련 수료증
- 채용박람회 참가확인서
