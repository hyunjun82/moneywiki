# 연말정산 특별재난지역

특별재난지역 기부금 공제를 정리합니다.

## 1. 기부금 분류

| 분류 | 종류 |
|------|------|
| 법정기부금 | 특별재난지역 기부 |
| 한도 | 소득금액 **100%** |

## 2. 공제율

| 금액 | 공제율 |
|------|--------|
| 1천만원 이하 | **15%** |
| 1천만원 초과 | **30%** |

## 3. 대상 기부

| 대상 | 공제 |
|------|------|
| 이재민 구호금 | ✅ |
| 재해복구 성금 | ✅ |
| 적십자 재해성금 | ✅ |

## 4. 기부처

| 기부처 | 인정 |
|--------|------|
| 대한적십자사 | ✅ |
| 지자체 | ✅ |
| 사회복지공동모금회 | ✅ |

## 5. 필요서류

| 서류 | 발급처 |
|------|--------|
| 기부금영수증 | 기부처 |
| 간소화 자료 | 일부 자동 반영 |
