# 연말정산 간소화서비스 이용방법

연말정산 간소화서비스는 국세청이 의료비, 교육비, 카드 사용액 등 **공제 자료를 한 번에 제공**하는 서비스입니다. [홈택스](https://www.hometax.go.kr)에서 이용합니다.

## 1. 서비스 개요

### 1.1 개통 일정

| 일정 | 내용 |
|------|------|
| **1월 15일** | 간소화서비스 **개통** |
| **1월 18일** | 1차 자료 확정 |
| **1월 20일 이후** | 추가·수정 자료 반영 |

### 1.2 조회 가능 자료

| 자료 | 제공 기관 |
|------|----------|
| 의료비 | 병원, 약국 |
| 교육비 | 학교, 학원 (취학전) |
| 신용카드 | 카드사 |
| 체크카드·현금영수증 | 국세청 |
| 보험료 | 보험사 |
| 기부금 | 기부금단체 |
| 연금저축 | 금융기관 |

## 2. 이용 방법

### 2.1 접속 방법

1. [홈택스](https://www.hometax.go.kr) 접속
2. **장려금·연말정산·전자기부금** 메뉴
3. **연말정산 간소화** 선택
4. 공인인증서 또는 간편인증 로그인

### 2.2 자료 조회

| 단계 | 내용 |
|------|------|
| 1단계 | 귀속연도 선택 (2025년) |
| 2단계 | 조회 항목 선택 (전체 또는 개별) |
| 3단계 | **조회하기** 클릭 |
| 4단계 | 자료 확인 |

### 2.3 PDF 다운로드

| 방법 | 내용 |
|------|------|
| 한꺼번에 내려받기 | 전체 자료 PDF 다운로드 |
| 항목별 내려받기 | 개별 항목 다운로드 |
| 회사 전송 | 회사에 직접 전송 (회사 동의 시) |

## 3. 부양가족 자료 조회

### 3.1 자료제공동의

부양가족 자료를 조회하려면 **동의 절차** 필요

| 방법 | 대상 |
|------|------|
| 온라인 동의 | 성인 부양가족 |
| 팩스·방문 | 미성년자, 노인 등 |

### 3.2 온라인 동의 절차

1. 홈택스 로그인 (부양가족 본인)
2. **조회/발급 → 연말정산 → 부양가족 자료제공 동의**
3. 신청 또는 동의

### 3.3 미성년자녀 동의

| 방법 | 내용 |
|------|------|
| 온라인 | 부모가 자녀 명의로 동의 신청 |
| 세무서 방문 | 가족관계증명서 제출 |

## 4. 조회 안 되는 자료

### 4.1 직접 준비 필요

| 자료 | 발급처 |
|------|--------|
| 안경·콘택트렌즈 | 안경점 영수증 |
| 교복 구입비 | 교복점 영수증 |
| 장애인증명서 | 의료기관 |
| 해외 교육비 | 해외 교육기관 |
| 일부 기부금 | 기부금단체 영수증 |

### 4.2 누락 자료 확인

1월 15일 이후 조회되지 않는 자료:
1. 해당 기관에 **등록 여부 확인**
2. 영수증 **직접 발급** 요청
3. **1월 18일** 이후 재조회

## 5. 자료 오류 수정

### 5.1 오류 발견 시

| 오류 유형 | 처리 방법 |
|----------|----------|
| 금액 오류 | 해당 기관에 정정 요청 |
| 누락 | 해당 기관에 등록 요청 |
| 중복 | 삭제 요청 후 재조회 |

### 5.2 수정 반영

- 수정 요청 후 **1~3일** 내 반영
- 1월 18일까지 수정된 내용은 **확정 자료**에 반영

## 6. 일정별 체크리스트

### 6.1 1월 15일 이전

| 항목 | 내용 |
|------|------|
| 부양가족 동의 | 자료제공동의 신청 |
| 영수증 준비 | 간소화 미제공 자료 준비 |
| 홈택스 회원가입 | 미가입자 가입 |

### 6.2 1월 15일~18일

| 항목 | 내용 |
|------|------|
| 1차 조회 | 자료 누락 여부 확인 |
| 누락 자료 | 해당 기관에 등록 요청 |

### 6.3 1월 20일 이후

| 항목 | 내용 |
|------|------|
| 최종 조회 | 추가 자료 반영 확인 |
| PDF 다운로드 | 최종 자료 저장 |
| 회사 제출 | 제출 기한 내 제출 |

## 7. 모바일 이용

### 7.1 손택스 앱

| 기능 | 지원 |
|------|------|
| 간소화 자료 조회 | ✅ |
| PDF 다운로드 | ✅ |
| 자료제공동의 | ✅ |

### 7.2 이용 방법

1. **손택스** 앱 설치 (국세청 공식)
2. 간편인증 로그인
3. **연말정산 간소화** 메뉴

## 8. 주의사항

### 8.1 조회 시점

- **1월 15일**: 자료 조회 가능, 일부 누락 가능
- **1월 20일 이후**: 추가 자료 반영 완료

### 8.2 개인정보 보호

- 타인 자료 조회 불가
- 부양가족은 **동의 필수**
- 로그인 정보 관리 주의

### 8.3 자료 보관

- PDF 파일 **5년간** 보관 권장
- 경정청구 시 필요할 수 있음
