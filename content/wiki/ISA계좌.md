# ISA계좌

ISA(Individual Savings Account)는 다양한 금융상품을 한 계좌에서 운용하며 세금 혜택을 받는 계좌입니다.

## 1. ISA 개요

### 1.1 정의

| 항목 | 내용 |
|------|------|
| 정의 | 개인종합자산관리계좌 |
| 목적 | 자산 형성 + 세제 혜택 |
| 도입 | 2016년 |

### 1.2 핵심 혜택

| 혜택 | 내용 |
|------|------|
| 비과세 | 순이익 200~400만원 |
| 저율과세 | 초과분 9.9% |
| 손익통산 | 이익과 손실 상계 |

## 2. ISA 유형

### 2.1 운용 방식별

| 유형 | 특징 |
|------|------|
| 일임형 | 금융사가 운용 |
| 신탁형 | 본인이 운용 (펀드·예금) |
| 중개형 | 본인이 운용 (주식 포함) |

### 2.2 세제 유형별

| 유형 | 비과세 한도 | 대상 |
|------|------------|------|
| 일반형 | **200만원** | 일반 가입자 |
| 서민형 | **400만원** | 총급여 5,000만원 이하 |
| 농어민형 | **400만원** | 농어민 |

## 3. 가입 조건

### 3.1 연령·자격

| 조건 | 기준 |
|------|------|
| 연령 | 만 **19세 이상** |
| 근로소득자 | 만 **15세 이상** |
| 계좌 수 | 1인 **1계좌** |

### 3.2 서민형 조건

| 조건 | 기준 |
|------|------|
| 총급여 | **5,000만원 이하** |
| 종합소득 | **3,800만원 이하** |
| 기준 | 직전 과세기간 |

### 3.3 금융소득 제한

| 조건 | 제한 |
|------|------|
| 금융소득종합과세 | 대상자 가입 불가 |
| 기준 | 이자·배당 2,000만원 초과 |

## 4. 납입 한도

### 4.1 연간 한도

| 항목 | 한도 |
|------|------|
| 연간 납입 | **2,000만원** |
| 총 납입 | **1억원** (5년) |

### 4.2 이월 가능

| 상황 | 처리 |
|------|------|
| 연간 미납분 | 다음해 이월 가능 |
| 예시 | 올해 1,000만원 → 내년 3,000만원 가능 |

## 5. 투자 가능 상품

### 5.1 일임형·신탁형

| 상품 | 투자 |
|------|------|
| 예·적금 | ✅ |
| 펀드 | ✅ |
| ETF | ✅ |
| ELS/DLS | ✅ |
| RP | ✅ |

### 5.2 중개형

| 상품 | 투자 |
|------|------|
| 국내 상장주식 | ✅ |
| ETF/ETN | ✅ |
| 펀드 | ✅ |
| 해외주식 | ❌ |
| 채권 | ✅ (일부) |

### 5.3 중개형 장점

| 장점 | 내용 |
|------|------|
| 직접 매매 | 주식 직접 투자 |
| 수수료 | 저렴 |
| 배당소득세 | 비과세 적용 |

## 6. 세제 혜택

### 6.1 비과세 한도

| 유형 | 비과세 한도 |
|------|------------|
| 일반형 | **200만원** |
| 서민형·농어민형 | **400만원** |

### 6.2 초과분 과세

| 항목 | 세율 |
|------|------|
| 비과세 초과분 | **9.9%** 분리과세 |
| 일반 과세 | 15.4% |
| 절감 효과 | 5.5%p |

### 6.3 손익통산

```
과세 대상 = (주식 이익 + 펀드 이익 + 이자) - (주식 손실 + 펀드 손실)
```

### 6.4 계산 예시

**총 이익 500만원, 손실 100만원**

| 항목 | 금액 |
|------|------|
| 순이익 | 400만원 |
| 비과세 (일반형) | 200만원 |
| 과세 대상 | 200만원 |
| 세금 (9.9%) | 19.8만원 |

**일반 계좌 시**

| 항목 | 금액 |
|------|------|
| 과세 대상 (손익통산 無) | 500만원 |
| 세금 (15.4%) | 77만원 |
| **절감액** | **57.2만원** |

## 7. 의무 보유 기간

### 7.1 기간

| 항목 | 기간 |
|------|------|
| 의무 보유 | **3년** |
| 연장 | 최대 5년 |

### 7.2 중도 해지 시

| 상황 | 결과 |
|------|------|
| 3년 미만 해지 | 일반 과세 (15.4%) |
| 비과세 혜택 | 소멸 |

### 7.3 특별 해지

| 사유 | 혜택 유지 |
|------|----------|
| 사망 | ✅ |
| 해외이주 | ✅ |
| 천재지변 | ✅ |
| 퇴직 | ✅ |

## 8. 가입 방법

### 8.1 가입처

| 유형 | 가입처 |
|------|--------|
| 일임형 | 은행, 증권사 |
| 신탁형 | 은행 |
| 중개형 | 증권사 |

### 8.2 개설 절차

| 단계 | 내용 |
|------|------|
| 1 | 금융사 앱/영업점 방문 |
| 2 | 가입 유형 선택 |
| 3 | 서민형 자격 확인 (해당 시) |
| 4 | 계좌 개설 |
| 5 | 입금·투자 시작 |

### 8.3 유형 변경

| 변경 | 가능 여부 |
|------|----------|
| 일임형 ↔ 신탁형 | ✅ 가능 |
| 일임형 → 중개형 | ✅ 가능 |
| 중개형 → 일임형 | ❌ 불가 |

## 9. 만기 처리

### 9.1 만기 후 선택

| 옵션 | 내용 |
|------|------|
| 연장 | 최대 5년까지 |
| 해지 | 인출 가능 |
| 연금계좌 전환 | 추가 세제 혜택 |

### 9.2 연금계좌 전환

| 전환 | 혜택 |
|------|------|
| IRP/연금저축 전환 | 세액공제 추가 |
| 전환 한도 | 없음 |
| 조건 | 만기 후 60일 이내 |

## 10. 다른 상품 비교

### 10.1 ISA vs 연금저축

| 구분 | ISA | 연금저축 |
|------|-----|---------|
| 목적 | 자산 형성 | 노후 대비 |
| 의무 기간 | 3년 | 5년 (55세~) |
| 비과세 | 200~400만원 | 수령 시 과세 |
| 세액공제 | 없음 | 있음 |

### 10.2 ISA vs 일반 계좌

| 구분 | ISA | 일반 계좌 |
|------|-----|----------|
| 손익통산 | ✅ | ❌ |
| 비과세 | ✅ | ❌ |
| 세율 | 9.9% | 15.4% |
| 납입 한도 | 있음 | 없음 |

## 11. 주의사항

### 11.1 가입 시 유의

| 주의 | 내용 |
|------|------|
| 1인 1계좌 | 중복 불가 |
| 금융사 선택 | 상품·수수료 비교 |
| 유형 선택 | 투자 스타일 고려 |

### 11.2 운용 시 유의

| 주의 | 내용 |
|------|------|
| 손실 가능 | 투자 손실 본인 책임 |
| 3년 유지 | 조기 해지 불이익 |
| 한도 관리 | 연간 2,000만원 |
