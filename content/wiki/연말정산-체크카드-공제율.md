# 체크카드 공제율

## 기본 개념
체크카드, 현금영수증 사용액에 대해 신용카드보다 높은 **30%** 공제율을 적용합니다.

## 공제율 비교

| 결제 수단 | 공제율 | 특징 |
|----------|-------|------|
| 신용카드 | 15% | 기본 |
| 체크카드 | **30%** | 2배 높음 |
| 현금영수증 | **30%** | 체크카드와 동일 |
| 전통시장 | **40%** | 추가 우대 |
| 대중교통 | **40%** | 추가 우대 |

## 공제 한도

### 총급여별 한도
| 총급여 | 공제 한도 |
|--------|---------|
| 7천만원 이하 | **300만원** |
| 7천만원~1.2억원 | **250만원** |
| 1.2억원 초과 | **200만원** |

### 최저 사용액
- **총급여의 25% 초과** 사용분부터 공제
- 예: 총급여 5천만원 → 1,250만원 초과분부터

## 공제액 계산

### 기본 계산식
```
공제액 = (사용액 - 총급여×25%) × 공제율
```

### 계산 예시 1: 신용카드만 사용
```
총급여: 5,000만원
신용카드: 2,000만원
최저사용액: 1,250만원
초과액: 750만원
공제액: 750 × 15% = 112.5만원
```

### 계산 예시 2: 체크카드로 전환
```
총급여: 5,000만원
신용카드: 1,250만원 (최저사용액)
체크카드: 750만원
초과액: 750만원
공제액: 750 × 30% = 225만원
→ 신용카드 대비 2배 효과
```

### 계산 예시 3: 혼합 사용
```
총급여: 5,000만원
신용카드: 1,500만원
체크카드: 1,000만원
최저사용액: 1,250만원
초과액: 1,250만원

계산:
1) 신용카드 250만원 × 15% = 37.5만원
2) 체크카드 1,000만원 × 30% = 300만원
3) 합계: 337.5만원 → 한도 300만원 적용
```

## 전략적 사용 방법

### 최적 전략
1. **최저사용액까지**: 신용카드 (혜택 활용)
2. **최저사용액 이후**: 체크카드, 현금영수증
3. **전통시장·대중교통**: 우선 사용 (40%)

### 맞벌이 부부 전략
- 소득 낮은 배우자: 체크카드 집중 사용
- 소득 높은 배우자: 최저사용액만 충족

### 월별 전략
- 1~9월: 최저사용액 도달 목표
- 10~12월: 체크카드·전통시장 집중

## 공제 제외 항목

### 제외 대상
- 상품권 구매
- 취득세·등록세 납부
- 자동차 구매
- 리스료
- 해외 사용액

### 주의사항
- 통신비, 관리비: 공제 가능
- 교육비, 의료비: 별도 공제와 중복 불가
- 보험료: 공제 제외

## 실무 팁

### 1. 25% 도달 전략
```
총급여 5,000만원 기준:
목표액: 1,250만원
월평균: 약 105만원
→ 모든 생활비를 카드로 결제
```

### 2. 체크카드 전환 시점
- 최저사용액 도달 확인
- 보통 9~10월경
- 이후 체크카드로 전환

### 3. 현금영수증 활용
- 체크카드 없는 곳: 현금영수증 요청
- 공제율 30% 동일

### 4. 가족카드 활용
- 배우자, 자녀 가족카드
- 본인 사용액으로 합산

## 주의사항

### 공제율 적용 순서
1. 전통시장·대중교통 (40%)
2. 체크카드·현금영수증 (30%)
3. 신용카드 (15%)

### 한도 관리
- 공제 한도 300만원 초과 시
- 공제율 높은 것부터 차감

### 간소화 서비스 확인
- 연말정산 시 자동 조회
- 사용액, 공제액 확인
- 오류 시 카드사 확인

## 체크카드 vs 신용카드

### 체크카드 장점
- 공제율 2배 (30% vs 15%)
- 과소비 방지
- 즉시 출금으로 자금 관리

### 신용카드 장점
- 포인트, 마일리지 적립
- 할인 혜택 많음
- 무이자 할부

### 최적 조합
```
신용카드: 최저사용액까지 (혜택 활용)
체크카드: 최저사용액 이후 (공제율 활용)
전통시장·대중교통: 우선 사용 (40%)
```
