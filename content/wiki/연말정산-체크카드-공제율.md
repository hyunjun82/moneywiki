# 연말정산 체크카드 공제율

체크카드는 신용카드보다 2배 높은 공제율이 적용되어 연말정산에서 유리합니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 정확한 공제율과 활용법을 정리합니다.

## 1. 결제 수단별 공제율 비교

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제율입니다.

| 결제 수단 | 공제율 | 비고 |
|-----------|--------|------|
| 신용카드 | **15%** | 기본 |
| 체크카드 | **30%** | 신용카드의 2배 |
| 직불카드 | **30%** | 체크카드와 동일 |
| 선불카드 | **30%** | 체크카드와 동일 |
| 현금영수증 | **30%** | 체크카드와 동일 |
| 전통시장 | **40%** | 결제 수단 무관 |
| 대중교통 | **40%** | 결제 수단 무관 |

## 2. 공제 조건

### 2.1 최저 사용 금액

공제는 **총급여의 25% 초과 사용분**부터 적용됩니다.

| 총급여 | 25% 기준액 | 공제 시작점 |
|--------|------------|-------------|
| 3,000만원 | 750만원 | 750만원 초과분 |
| 4,000만원 | 1,000만원 | 1,000만원 초과분 |
| 5,000만원 | 1,250만원 | 1,250만원 초과분 |

### 2.2 공제 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 ~ 1.2억원 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

## 3. 체크카드 공제 계산 예시

**연봉 5,000만원, 체크카드로 2,000만원 사용한 경우**

1. 25% 기준액: 5,000만원 × 25% = 1,250만원
2. 공제 대상 금액: 2,000만원 - 1,250만원 = 750만원
3. 공제율 적용: 750만원 × 30% = **225만원 소득공제**

**같은 금액을 신용카드로 사용했다면?**
- 750만원 × 15% = 112.5만원 (체크카드 대비 112.5만원 손해)

## 4. 체크카드 vs 신용카드 전략

### 4.1 25% 미달 구간

- 어차피 공제 대상이 아닌 구간
- **신용카드** 사용 추천 (포인트, 캐시백, 할인 혜택)

### 4.2 25% 초과 구간

- 본격적인 공제 구간
- **체크카드** 사용 추천 (공제율 2배)

### 4.3 최적 전략

```
1~6월: 신용카드로 25% 채우기 (혜택 챙기기)
7~12월: 체크카드로 전환 (공제율 극대화)
```

## 5. 체크카드 공제 제외 항목

다음 항목은 체크카드로 결제해도 **공제되지 않습니다**.

- 세금, 공과금
- 통신비, 인터넷 사용료
- 보험료
- 학교 등록금
- 아파트 관리비
- 자동차 구매비, 리스료
- 해외 결제

## 6. 주의사항

### 6.1 체크카드 = 현금영수증 아님

체크카드는 별도 매출전표가 발행되므로 **현금영수증과 별개**입니다. 둘 다 30% 공제율이지만 서로 다른 결제 수단입니다.

### 6.2 중복 공제 불가

같은 결제 건에 대해 신용카드 공제와 현금영수증 공제를 **중복 적용받을 수 없습니다**.
