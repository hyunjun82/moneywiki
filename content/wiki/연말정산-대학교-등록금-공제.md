# 연말정산 대학교 등록금 공제

대학교 등록금은 교육비 세액공제로 절세할 수 있습니다. [국세청 교육비 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따른 조건을 정리합니다.

## 1. 공제 대상자

### 1.1 대상 범위

| 대상 | 한도 | 비고 |
|------|------|------|
| **본인** | 한도 없음 | 대학원 포함 |
| **배우자** | 900만원 | 대학원 불가 |
| **직계비속(자녀)** | 900만원 | 대학원 불가 |
| **형제자매** | 900만원 | 대학원 불가 |

### 1.2 부양가족 요건

| 조건 | 기준 |
|------|------|
| 나이 요건 | **없음** (만 20세 초과해도 가능) |
| 소득 요건 | 연간 소득금액 100만원 이하 |

※ 대학생 자녀는 만 20세 초과해도 소득 요건만 충족하면 공제 가능

## 2. 공제 한도와 공제율

### 2.1 연간 한도

| 대상 | 연간 한도 |
|------|-----------|
| 본인 대학교 | **한도 없음** |
| 본인 대학원 | **한도 없음** |
| 부양가족 대학교 | **1인당 900만원** |
| 부양가족 대학원 | ❌ 공제 불가 |

### 2.2 공제율

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따라 **15%** 세액공제

| 납부 교육비 | 공제율 | 세액공제액 |
|------------|--------|-----------|
| 500만원 | 15% | 75만원 |
| 700만원 | 15% | 105만원 |
| 900만원 | 15% | 135만원 |

## 3. 공제 대상 항목

### 3.1 공제 가능 항목

| 항목 | 공제 여부 |
|------|----------|
| 입학금 | ✅ 가능 |
| 수업료 | ✅ 가능 |
| 기숙사비 | ✅ 가능 |
| 실험실습비 | ✅ 가능 |

### 3.2 공제 불가 항목

| 항목 | 공제 여부 |
|------|----------|
| 학교 외 교재비 | ❌ 불가 |
| 학원비 | ❌ 불가 |
| 어학연수비 | ❌ 불가 (학점 인정 제외) |
| 휴학 중 등록금 | ❌ 불가 |

## 4. 대학 유형별 공제

### 4.1 공제 대상 대학

| 대학 유형 | 공제 여부 |
|----------|----------|
| 4년제 대학교 | ✅ 가능 |
| 전문대학 (2~3년) | ✅ 가능 |
| 사이버대학(원격대학) | ✅ 가능 |
| 방송통신대학 | ✅ 가능 |
| 학점은행제 | ✅ 가능 |
| 대학원 (본인만) | ✅ 가능 |

### 4.2 해외 대학

| 조건 | 공제 여부 |
|------|----------|
| 정규 대학 학위과정 | ✅ 가능 |
| 어학연수 | ❌ 불가 |

## 5. 장학금과 학자금 대출

### 5.1 장학금 차감

실제 납부액에서 **장학금 차감** 후 공제

| 항목 | 공제 대상액 |
|------|-------------|
| 등록금 800만원 | |
| 장학금 300만원 | |
| **공제 대상** | **500만원** |

### 5.2 학자금 대출

| 상황 | 공제 시점 |
|------|----------|
| 대출금으로 등록금 납부 | 납부 연도에 공제 |
| 대출 상환 | 상환 시 공제 불가 |

※ 등록금을 실제 납부한 연도에 공제 (대출 상환 시점 아님)

## 6. 공제 계산 예시

### 6.1 본인 대학원 등록금

**본인 대학원 등록금 1,200만원**

- 한도: 없음
- 공제액: 1,200만원 × 15% = **180만원**

### 6.2 자녀 대학 등록금

**자녀 대학 등록금 850만원, 장학금 200만원**

- 실제 납부액: 650만원
- 공제액: 650만원 × 15% = **97.5만원**

### 6.3 두 자녀 대학 등록금

**첫째 900만원, 둘째 700만원**

- 첫째 공제: 900만원 × 15% = 135만원
- 둘째 공제: 700만원 × 15% = 105만원
- **총 공제: 240만원**

## 7. 증빙서류

### 7.1 연말정산 간소화

대부분 **자동 조회** 가능

### 7.2 별도 제출 필요

| 상황 | 제출 서류 |
|------|----------|
| 간소화 미등록 | 대학 납입증명서 |
| 해외 대학 | 재학증명서 + 납부영수증 |
| 기숙사비 | 기숙사비 납입증명서 |

## 8. 주의사항

### 8.1 공제 불가 상황

- **휴학** 중 납부한 등록금
- **자퇴** 후 반환받은 금액
- **대학원**: 부양가족은 공제 불가
- **재수생**: 학원비 공제 불가

### 8.2 중복 공제 불가

- 본인이 공제받은 교육비를 다른 가족이 중복 공제 불가
- 형제자매 중 1명만 공제 가능
