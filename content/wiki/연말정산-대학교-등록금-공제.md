대학교 등록금은 교육비 세액공제로 절세할 수 있어요. [국세청 교육비 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6595&cntntsId=7874)에 따르면 본인이나 부양가족의 대학 등록금에 15%를 세액공제해줘요.

## 공제 대상자

대학 등록금 공제 대상과 한도예요.

| 대상 | 한도 | 비고 |
|------|------|------|
| 본인 | 한도 없음 | 대학원 포함 |
| 배우자 | 900만원 | 대학원 불가 |
| 직계비속(자녀) | 900만원 | 대학원 불가 |
| 형제자매 | 900만원 | 대학원 불가 |

부양가족 요건도 있어요.

| 조건 | 기준 |
|------|------|
| 나이 요건 | 없음 (만 20세 초과해도 가능) |
| 소득 요건 | 연간 소득금액 100만원 이하 |

대학생 자녀는 만 20세가 넘어도 소득 요건만 충족하면 공제받을 수 있어요. 아르바이트로 연 소득금액 100만원(총급여 500만원)을 넘기면 공제 대상에서 빠지니까 주의하세요.

## 공제 한도와 공제율

[소득세법 제59조의4](https://www.law.go.kr/법령/소득세법)에 따라 15% 세액공제돼요.

| 대상 | 연간 한도 |
|------|-----------|
| 본인 대학교 | 한도 없음 |
| 본인 대학원 | 한도 없음 |
| 부양가족 대학교 | 1인당 900만원 |
| 부양가족 대학원 | 공제 불가 |

실제 공제액을 계산해 볼게요.

| 납부 교육비 | 공제율 | 세액공제액 |
|------------|--------|-----------|
| 500만원 | 15% | 75만원 |
| 700만원 | 15% | 105만원 |
| 900만원 | 15% | 135만원 |

## 공제 대상 항목

공제 가능한 항목이에요.

| 항목 | 공제 여부 |
|------|----------|
| 입학금 | 가능 |
| 수업료 | 가능 |
| 기숙사비 | 가능 |
| 실험실습비 | 가능 |

공제가 안 되는 항목도 있어요.

| 항목 | 공제 여부 |
|------|----------|
| 학교 외 교재비 | 불가 |
| 학원비 | 불가 |
| 어학연수비 | 불가 (학점 인정 제외) |
| 휴학 중 등록금 | 불가 |

## 대학 유형별 공제

다양한 형태의 대학이 공제 대상이에요.

| 대학 유형 | 공제 여부 |
|----------|----------|
| 4년제 대학교 | 가능 |
| 전문대학 (2~3년) | 가능 |
| 사이버대학(원격대학) | 가능 |
| 방송통신대학 | 가능 |
| 학점은행제 | 가능 |
| 대학원 (본인만) | 가능 |

해외 대학도 정규 학위과정이면 공제돼요. 어학연수는 안 돼요.

## 장학금과 학자금 대출

장학금을 받았으면 장학금을 뺀 실제 납부액만 공제돼요.

| 항목 | 금액 |
|------|------|
| 등록금 | 800만원 |
| 장학금 | 300만원 |
| 공제 대상 | 500만원 |

학자금 대출로 등록금을 냈다면 납부한 연도에 공제받아요. 나중에 대출 상환할 때는 공제 안 돼요.

## 공제 계산 예시

본인 대학원 등록금 1,200만원이면 한도가 없어서 전액 공제돼요. 1,200만원 × 15% = 180만원 세액공제예요.

자녀 대학 등록금 850만원인데 장학금 200만원 받았다면 실제 납부액 650만원에 대해 공제돼요. 650만원 × 15% = 97.5만원이에요.

두 자녀가 대학에 다니고 첫째 900만원, 둘째 700만원이면 각각 공제받아요. 첫째 135만원 + 둘째 105만원 = 총 240만원 세액공제예요.

## 증빙서류

대부분 연말정산 간소화 서비스에서 자동 조회돼요.

| 상황 | 제출 서류 |
|------|----------|
| 간소화 미등록 | 대학 납입증명서 |
| 해외 대학 | 재학증명서 + 납부영수증 |
| 기숙사비 | 기숙사비 납입증명서 |

## 주의사항

휴학 중 납부한 등록금이나 자퇴 후 반환받은 금액은 공제 대상이 아니에요. 부양가족의 대학원 등록금도 공제가 안 돼요. 재수생 학원비도 마찬가지로 공제 불가예요.

본인이 공제받은 교육비를 다른 가족이 중복 공제받을 수 없어요. 형제자매 중 1명만 공제 가능해요.
