# 국민연금

국민연금은 노후 소득을 보장하기 위한 공적 연금 제도입니다. [국민연금법](https://www.law.go.kr/법령/국민연금법)에 따른 내용을 정리합니다.

## 1. 국민연금 개요

### 1.1 가입 대상

| 대상 | 가입 유형 |
|------|----------|
| 직장인 | 사업장 가입자 (의무) |
| 자영업자 | 지역 가입자 (의무) |
| 전업주부 등 | 임의 가입자 (선택) |
| 연령 | 만 18세~60세 미만 |

### 1.2 보험료율

| 구분 | 보험료율 | 부담 |
|------|----------|------|
| 직장인 | 9% | 본인 4.5% + 회사 4.5% |
| 자영업자 | 9% | 전액 본인 |
| 임의가입자 | 9% | 전액 본인 |

### 1.3 기준소득월액

| 항목 | 금액 (2026년) |
|------|--------------|
| 하한 | **37만원** |
| 상한 | **617만원** |

## 2. 보험료 계산

### 2.1 계산 공식

```
월 보험료 = 기준소득월액 × 9%
```

### 2.2 월 보험료 예시

| 월 소득 | 기준소득월액 | 월 보험료 | 본인 부담 (직장인) |
|--------|------------|----------|------------------|
| 300만원 | 300만원 | 27만원 | 13.5만원 |
| 400만원 | 400만원 | 36만원 | 18만원 |
| 500만원 | 500만원 | 45만원 | 22.5만원 |
| 700만원 | 617만원 (상한) | 55.5만원 | 27.8만원 |

## 3. 연금 수령 연령

### 3.1 출생연도별 수령 나이

| 출생연도 | 수령 개시 연령 |
|----------|--------------|
| 1952년 이전 | 만 60세 |
| 1953~1956년 | 만 61세 |
| 1957~1960년 | 만 62세 |
| 1961~1964년 | 만 63세 |
| 1965~1968년 | 만 64세 |
| 1969년 이후 | 만 **65세** |

### 3.2 조기연금

| 항목 | 내용 |
|------|------|
| 조기수령 | 정상 수령 연령 **5년 전**부터 가능 |
| 감액 | 1년 조기 시 **6%** 감액 |
| 최대 감액 | 5년 조기 시 **30%** 감액 |

### 3.3 연기연금

| 항목 | 내용 |
|------|------|
| 연기 가능 | 최대 **5년** |
| 증액 | 1년 연기 시 **7.2%** 증액 |
| 최대 증액 | 5년 연기 시 **36%** 증액 |

## 4. 연금 급여 종류

### 4.1 노령연금

| 유형 | 조건 |
|------|------|
| 완전노령연금 | 가입 20년 이상 + 수령 연령 |
| 감액노령연금 | 가입 10~20년 + 수령 연령 |
| 조기노령연금 | 가입 10년 이상 + 조기 수령 |

### 4.2 유족연금

| 수급자 | 지급률 |
|--------|--------|
| 배우자 | 기본연금의 **40~60%** |
| 자녀 (18세 미만) | 기본연금의 40~60% |
| 부모 (60세 이상) | 기본연금의 40~60% |

### 4.3 장애연금

| 장애등급 | 지급률 |
|----------|--------|
| 1급 | 기본연금의 **100%** |
| 2급 | 기본연금의 **80%** |
| 3급 | 기본연금의 **60%** |
| 4급 | 일시보상금 |

## 5. 연금액 계산

### 5.1 기본 공식

```
기본연금액 = 1.2 × (A + B) × (1 + 0.05n/12)
```

- A: 전체 가입자 평균소득
- B: 본인 가입기간 평균소득
- n: 20년 초과 가입개월 수

### 5.2 예상 연금액

| 가입기간 | 평균소득 | 예상 월 연금 |
|----------|----------|-------------|
| 10년 | 250만원 | 약 30~35만원 |
| 20년 | 300만원 | 약 70~80만원 |
| 30년 | 350만원 | 약 100~120만원 |
| 40년 | 400만원 | 약 130~150만원 |

※ 실제 연금액은 개인별 소득, 물가상승률 등에 따라 달라집니다.

### 5.3 소득대체율

| 가입기간 | 소득대체율 |
|----------|-----------|
| 40년 | 약 **40%** |
| 30년 | 약 30% |
| 20년 | 약 20% |

## 6. 반환일시금

### 6.1 지급 조건

| 조건 | 지급 |
|------|------|
| 가입기간 10년 미만 | 60세 도달 시 |
| 국적 상실 | 출국 시 |
| 사망 (유족연금 미해당) | 유족에게 지급 |

### 6.2 계산 방법

```
반환일시금 = 납부한 보험료 + 이자
```

## 7. 크레딧 제도

### 7.1 출산 크레딧

| 자녀 수 | 가입기간 추가 |
|---------|--------------|
| 2명 | **12개월** |
| 3명 | **30개월** |
| 4명 | **48개월** |
| 5명 이상 | **50개월** |

### 7.2 군복무 크레딧

| 항목 | 내용 |
|------|------|
| 대상 | 2008년 이후 군복무자 |
| 인정 기간 | 복무기간 전체 (최대 6개월) |
| 소득 인정 | A값의 50% |

### 7.3 실업 크레딧

| 항목 | 내용 |
|------|------|
| 대상 | 구직급여 수급자 |
| 지원율 | 보험료의 75% |
| 인정 기간 | 최대 12개월 |

## 8. 납부 관련

### 8.1 납부 예외

| 사유 | 예외 인정 |
|------|----------|
| 실업 | ✅ |
| 사업 중단 | ✅ |
| 휴직 | ✅ |
| 학업 | ✅ |

### 8.2 추후 납부

| 항목 | 내용 |
|------|------|
| 대상 | 납부예외 기간, 미가입 기간 |
| 기간 | 최대 10년분 |
| 효과 | 가입기간 인정 |

### 8.3 연체 시

| 상황 | 조치 |
|------|------|
| 1개월 연체 | 납부 독촉 |
| 3개월 이상 | 연체금 부과 |
| 재산 압류 | 강제 징수 가능 |

## 9. 주의사항

### 9.1 중복 수급 제한

| 상황 | 처리 |
|------|------|
| 노령연금 + 유족연금 | 택일 |
| 장애연금 + 노령연금 | 택일 |
| 기초연금 | 별도 수급 가능 |

### 9.2 해외 거주 시

| 상황 | 연금 수급 |
|------|----------|
| 해외 이주 | 수급 가능 |
| 국적 상실 | 반환일시금 청구 가능 |
