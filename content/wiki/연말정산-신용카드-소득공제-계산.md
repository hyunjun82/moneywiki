# 연말정산 신용카드 소득공제 계산

신용카드 소득공제는 **총급여의 25%를 초과**하여 사용한 금액에 대해 공제가 적용됩니다. [국세청 연말정산 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=2304&cntntsId=238938)에 따른 계산 방법을 정리합니다.

## 1. 공제 계산 기본 구조

### 1.1 공제 대상 금액 산정

| 단계 | 내용 |
|------|------|
| 1단계 | 총급여 × 25% = 최저 사용 기준액 |
| 2단계 | 총 카드 사용액 - 최저 기준액 = 공제 대상액 |
| 3단계 | 공제 대상액 × 공제율 = 소득공제액 |

### 1.2 총급여별 25% 기준액

| 총급여 | 25% 기준액 |
|--------|------------|
| 3,000만원 | 750만원 |
| 4,000만원 | 1,000만원 |
| 5,000만원 | 1,250만원 |
| 6,000만원 | 1,500만원 |
| 7,000만원 | 1,750만원 |

## 2. 결제수단별 공제율

[조세특례제한법 제126조의2](https://www.law.go.kr/법령/조세특례제한법)에 따른 공제율입니다.

| 결제수단 | 공제율 |
|----------|--------|
| 신용카드 | **15%** |
| 체크카드 | **30%** |
| 현금영수증 | **30%** |
| 전통시장 | **40%** |
| 대중교통 | **40%** |
| 도서·공연·영화 | **30%** (7천만원 이하) |

## 3. 공제 순서

### 3.1 계산 원리

**공제율이 낮은 것부터** 25% 기준액을 채움

1. 신용카드 (15%) → 먼저 기준액 충당
2. 체크카드/현금영수증 (30%) → 다음 충당
3. 전통시장/대중교통 (40%) → 마지막 충당

### 3.2 공제율 적용 원칙

25% 기준액을 초과한 부분부터 **사용처별 공제율** 적용

## 4. 계산 예시

### 4.1 기본 예시

**연봉 5,000만원, 카드 사용 2,000만원**

- 25% 기준액: 1,250만원
- 공제 대상: 2,000만원 - 1,250만원 = **750만원**

**결제수단별 사용액**:
- 신용카드: 1,500만원
- 체크카드: 500만원

**계산**:
1. 신용카드 1,250만원으로 기준액 충당
2. 신용카드 초과분: 250만원 × 15% = 37.5만원
3. 체크카드 전액: 500만원 × 30% = 150만원
4. **총 소득공제액: 187.5만원**

### 4.2 전통시장 포함 예시

**연봉 4,000만원, 총 사용 1,800만원**

- 25% 기준액: 1,000만원
- 공제 대상: 800만원

**결제수단별 사용액**:
- 신용카드: 1,000만원
- 전통시장: 300만원
- 대중교통: 500만원

**계산**:
1. 신용카드 1,000만원으로 기준액 충당
2. 전통시장: 300만원 × 40% = 120만원
3. 대중교통: 500만원 × 40% = 200만원
4. **총 소득공제액: 320만원**

## 5. 공제 한도

### 5.1 기본 한도

| 총급여 | 기본 한도 |
|--------|-----------|
| 7,000만원 이하 | **최대 300만원** |
| 7,000만원 초과 | **최대 250만원** |
| 1.2억원 초과 | **최대 200만원** |

### 5.2 추가 한도

기본 한도를 채운 후 **추가 공제** 가능:

| 항목 | 추가 한도 |
|------|-----------|
| 전통시장 | +100만원 |
| 대중교통 | +100만원 |
| 도서·공연·영화 (7천만원 이하) | +100만원 |

**최대 공제**: 300만원 + 300만원 = **600만원**

## 6. 공제 제외 항목

### 6.1 공제 대상 아닌 사용처

| 항목 | 공제 여부 |
|------|----------|
| 세금, 공과금 | ❌ 불가 |
| 통신비, 인터넷요금 | ❌ 불가 |
| 보험료 | ❌ 불가 |
| 해외 사용액 | ❌ 불가 |
| 현금서비스 | ❌ 불가 |
| 신차 구입비 | ❌ 불가 |
| 리스료 | ❌ 불가 |
| 아파트 관리비 | ❌ 불가 |

## 7. 절세 전략

### 7.1 결제수단 최적화

**25% 기준액까지**: 신용카드 사용 (카드 혜택 활용)

**25% 초과분부터**: 체크카드/현금영수증 사용 (공제율 2배)

### 7.2 고공제율 사용처 활용

- **전통시장**: 40% 공제 + 추가 한도
- **대중교통**: 40% 공제 + 추가 한도

### 7.3 연간 전략

- 상반기: 신용카드로 25% 채우기
- 하반기: 체크카드/고공제율 사용처 집중
