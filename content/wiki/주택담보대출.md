# 주택담보대출

주택담보대출(주담대)은 주택을 담보로 자금을 빌리는 대출입니다. 규제와 한도 계산 방법을 정리합니다.

## 1. 주택담보대출 개요

### 1.1 대출 유형

| 유형 | 특징 |
|------|------|
| 구입자금 대출 | 주택 매수 시 |
| 기존주택 담보대출 | 보유 주택 담보 |
| 대환대출 | 기존 대출 갈아타기 |

### 1.2 대출 조건

| 조건 | 기준 |
|------|------|
| 담보 | 주택 (아파트, 빌라, 단독 등) |
| 대출 기간 | 10~40년 |
| 상환 방식 | 원리금균등, 원금균등, 체증식 |

## 2. LTV (담보인정비율)

### 2.1 LTV 개념

```
LTV = 대출금액 / 주택가격 × 100
```

### 2.2 지역별 LTV

| 지역 | 규제지역 여부 | LTV |
|------|-------------|-----|
| 투기과열지구 | 규제 | **40%** |
| 조정대상지역 | 규제 | **50%** |
| 비규제지역 | 비규제 | **70%** |

### 2.3 LTV 계산 예시

**비규제지역 10억원 아파트 (LTV 70%)**

| 항목 | 금액 |
|------|------|
| 주택가격 | 10억원 |
| LTV | 70% |
| **최대 대출** | **7억원** |

### 2.4 생애최초 주택 LTV 우대

| 조건 | LTV |
|------|-----|
| 생애최초 + 무주택 | 규제지역 **70%** |
| 일반 | 지역별 기준 적용 |

## 3. DTI (총부채상환비율)

### 3.1 DTI 개념

```
DTI = (주담대 연간 원리금 + 기타대출 연간 이자) / 연소득 × 100
```

### 3.2 DTI 한도

| 지역 | DTI |
|------|-----|
| 투기과열지구 | **40%** |
| 조정대상지역 | **50%** |
| 비규제지역 | **60%** |

## 4. DSR (총부채원리금상환비율)

### 4.1 DSR 개념

```
DSR = 모든 대출의 연간 원리금 / 연소득 × 100
```

### 4.2 DSR 규제 (2026년)

| 대출 금액 | DSR 한도 |
|----------|---------|
| 전 금융권 | **40%** |

### 4.3 DSR 계산 예시

**연소득 6,000만원, DSR 40%**

| 항목 | 계산 |
|------|------|
| 연소득 | 6,000만원 |
| DSR 40% | 2,400만원 |
| 월 상환 한도 | **200만원** |

### 4.4 대출 한도 역산

**연소득 6,000만원, 금리 4%, 30년 상환**

| 항목 | 금액 |
|------|------|
| 연간 원리금 한도 | 2,400만원 |
| 월 원리금 | 200만원 |
| **대출 가능 금액** | **약 4.2억원** |

## 5. 금리

### 5.1 금리 유형

| 유형 | 특징 |
|------|------|
| 고정금리 | 만기까지 금리 고정 |
| 변동금리 | 시장금리에 따라 변동 |
| 혼합금리 | 일정 기간 고정 후 변동 |

### 5.2 금리 수준 (2026년 기준)

| 유형 | 금리 범위 |
|------|----------|
| 시중은행 | 연 3.5~5.5% |
| 정책대출 (디딤돌) | 연 2.15~3.0% |

### 5.3 금리 결정 요소

| 요소 | 영향 |
|------|------|
| 기준금리 | 한국은행 기준금리 |
| 가산금리 | 은행 마진, 리스크 |
| 우대금리 | 급여이체, 카드 실적 등 |

## 6. 상환 방식

### 6.1 상환 방식 비교

| 방식 | 특징 | 초기 부담 | 총 이자 |
|------|------|----------|---------|
| 원리금균등 | 매월 동일 금액 | 중간 | 중간 |
| 원금균등 | 원금 동일, 이자 감소 | 높음 | 낮음 |
| 체증식 | 점점 증가 | 낮음 | 높음 |

### 6.2 상환 예시 (3억원, 4%, 30년)

| 방식 | 초기 월 상환액 | 총 이자 |
|------|--------------|---------|
| 원리금균등 | 약 143만원 | 약 2.2억원 |
| 원금균등 | 약 183만원 | 약 1.8억원 |

## 7. 정책 대출

### 7.1 디딤돌 대출

| 조건 | 기준 |
|------|------|
| 대상 | 무주택 세대주 |
| 소득 | 부부합산 연 6,000만원 이하 |
| 주택가격 | 5억원 이하 |
| 금리 | 연 2.15~3.0% |
| 한도 | 최대 2.5억원 |

### 7.2 보금자리론

| 조건 | 기준 |
|------|------|
| 대상 | 무주택 또는 1주택 |
| 소득 | 부부합산 연 7,000만원 이하 |
| 주택가격 | 6억원 이하 |
| 금리 | 연 4% 내외 (고정) |
| 한도 | 최대 3.6억원 |

## 8. 대출 절차

### 8.1 신청 흐름

```
상담 → 가심사 → 본심사 → 승인 → 담보설정 → 실행
```

### 8.2 필요 서류

| 서류 | 용도 |
|------|------|
| 신분증 | 본인 확인 |
| 소득증빙 | 원천징수영수증, 소득금액증명 |
| 재직증명서 | 재직 확인 |
| 매매계약서 | 구입자금의 경우 |
| 등기부등본 | 담보 확인 |

### 8.3 소요 기간

| 단계 | 기간 |
|------|------|
| 가심사 | 1~2일 |
| 본심사 | 3~7일 |
| 담보설정 | 3~5일 |
| **총 소요** | **약 2주** |

## 9. 주의사항

### 9.1 중도상환수수료

| 시기 | 수수료 |
|------|--------|
| 3년 이내 | 약 1.0~1.5% |
| 3년 이후 | 없음 |

### 9.2 규제 변경

| 주의 | 내용 |
|------|------|
| 규제지역 변경 | LTV, DTI 변동 가능 |
| DSR 강화 | 대출 한도 감소 가능 |
