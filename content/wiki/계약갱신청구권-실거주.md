"저 직접 살 거예요." 집주인이 이렇게 말하면 나가야 할까요?

네, 나가야 해요. **실거주는 정당한 거절 사유**거든요. 근데 진짜 실거주인지가 문제예요.

## 실거주 거절이란

집주인 본인이나 가족이 **직접 거주**하려고 할 때 [계약갱신청구권](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-행사방법)을 거절할 수 있어요.

주택임대차보호법 제6조의3에서 정한 정당한 [거절 사유](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-거절사유) 중 하나예요.

## 실거주 대상자

집주인 본인만 되는 게 아니에요.

| 대상 | 실거주 거절 |
|------|-----------|
| 집주인 본인 | O |
| 집주인 배우자 | O |
| 직계존속 (부모, 조부모) | O |
| 직계비속 (자녀, 손자녀) | O |
| 배우자의 직계존비속 | O |
| 형제자매 | X |
| 사촌 | X |
| 조카 | X |

**직계**만 가능해요. 방계(형제자매, 사촌 등)는 안 돼요.

### 직계존비속 예시

- 집주인 어머니가 살겠다 → O
- 집주인 아들이 살겠다 → O
- 집주인 며느리 부모가 살겠다 → O (배우자의 직계존속)
- 집주인 동생이 살겠다 → X

## 실거주 확인 방법

집주인이 "실거주할 거야"라고 하면 일단 나가야 해요. 근데 진짜인지 의심되면요?

### 확인할 수 있는 것

| 방법 | 확인 가능 시점 |
|------|--------------|
| 전입신고 여부 | 퇴거 후 확인 |
| 실제 거주 여부 | 퇴거 후 확인 |
| 새 세입자 입주 | 퇴거 후 확인 |

사전에 "진짜 실거주할 거예요?"라고 증명해달라고 요구할 법적 권리는 없어요. 다만 나간 후에 확인해서 거짓이면 손해배상 청구할 수 있어요.

### 의심될 때 할 수 있는 것

- "언제부터 거주하실 건가요?" 물어보고 **기록 남기기** (문자, 카톡)
- "가족분 중 누가 거주하시나요?" 물어보고 **기록 남기기**
- 집주인 답변 날짜와 내용 **캡처해두기**

나중에 거짓으로 밝혀지면 이 기록들이 **증거**가 돼요.

## 거짓 실거주 손해배상

실거주한다고 해서 나갔는데, 다른 세입자 들이면요? **손해배상 청구**할 수 있어요.

### 손해배상 항목

| 항목 | 설명 |
|------|------|
| 이사 비용 | 이삿짐센터 비용 |
| 중개수수료 | 새 집 계약 복비 |
| 보증금 차액 | 새 집이 더 비싸면 그 차액 |
| 월세 차액 | 새 집 월세가 더 비싸면 그 차액 |
| 기타 손해 | 입증 가능한 실제 손해 |

### 손해배상 청구 방법

| 순서 | 할 일 |
|------|-------|
| 1 | 거짓 실거주 증거 확보 (새 세입자 입주 확인) |
| 2 | 내용증명으로 손해배상 요청 |
| 3 | 협의 안 되면 임대차분쟁조정위원회 신청 (무료) |
| 4 | 조정 안 되면 민사소송 |

### 증거 확보 방법

- 해당 주소로 **등기부등본** 열람 → 새 전세권 설정 확인
- 해당 주소로 **방문** → 다른 사람 거주 확인
- **부동산** 문의 → 해당 집 세입자 모집 중인지 확인

## 실거주 기간 규정

### 법에 정해진 기간 없음

"얼마나 살아야 실거주예요?" 법에 명시된 기간은 없어요.

다만 판례에서는 **실질적인 거주 의사와 행위**를 봐요.

| 상황 | 실거주 인정 |
|------|-----------|
| 실제로 수개월 이상 거주 | O |
| 전입신고만 하고 안 살음 | X |
| 며칠 살다가 세 놓음 | X |
| 거주하다 사정 변경으로 이사 | 경우에 따라 |

형식적으로 전입신고만 해놓고 안 사는 건 **거짓 실거주**로 볼 수 있어요.

### 사정 변경된 경우

"진짜 살려고 했는데 갑자기 해외 발령 났어요." 이런 경우는요?

진정한 실거주 의사가 있었는데 **예측 못한 사정**으로 못 살게 된 거면 거짓 실거주로 보기 어려워요. 근데 이것도 케이스마다 달라서 분쟁 가능성 있어요.

## 실거주 거절 대응

### 정당한 실거주면

나가야 해요. 다만 협의해볼 수 있는 것들이요.

- 퇴거 시기 조율 (이사 준비 기간)
- 이사 비용 일부 지원 요청
- 보증금 반환 일정 협의

법적 의무는 아니지만 협의해볼 수는 있어요.

### 거짓 실거주 의심되면

일단 나가되, 증거 남겨두세요.

| 보관할 증거 | 용도 |
|-----------|------|
| 집주인과의 대화 기록 | 실거주 주장 증거 |
| 퇴거 전 집 사진 | 상태 증명 |
| 이사 영수증 | 손해배상 청구 |
| 새 집 계약서 | 보증금 차액 증명 |

나간 후 **3개월 정도** 지나서 확인해보세요. 새 세입자 들었는지, 집주인이 실제로 사는지요.

## 자주 묻는 질문

### 실거주 안 하고 빈집으로 두면?

빈집으로 둬도 일단은 문제 없어요. 세입자 안 들이면요. 근데 바로 세입자 들이면 거짓 실거주예요.

### 실거주하다가 세 놓으면?

어느 정도 기간 실거주 후에 세 놓는 건 문제 없어요. "며칠 살다가 바로" 세 놓으면 거짓 실거주로 볼 수 있어요.

### 실거주 거절 통보는 언제까지?

[계약갱신청구권 기간](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-기간)과 같아요. 세입자가 갱신청구하면 집주인은 **1개월 내**에 거절 여부를 통보해야 해요.

더 자세한 거절 사유는 [계약갱신청구권 거절사유](/w/%EB%B6%80%EB%8F%99%EC%82%B0/계약갱신청구권-거절사유)에서 확인하세요.
