# 임대차계약서 작성법

임대차계약서는 임대인과 임차인의 권리와 의무를 명시하는 법적 문서입니다. 정확한 작성이 분쟁 예방의 핵심입니다.

## 계약서 종류

### 1.1 계약서 유형

| 유형 | 용도 |
|------|------|
| 표준임대차계약서 | 국토교통부 권장 양식 |
| 공인중개사 계약서 | 중개 시 사용 |
| 당사자 직접 계약서 | 직거래 시 작성 |
| 전자계약서 | 온라인 계약 |

### 1.2 표준계약서 장점

| 장점 | 내용 |
|------|------|
| 법적 검토 완료 | 필수 조항 포함 |
| 분쟁 예방 | 명확한 권리의무 |
| 확정일자 자동 | 전자계약 시 |

## 필수 기재 사항

### 2.1 당사자 정보

| 항목 | 기재 내용 |
|------|----------|
| 임대인 | 성명, 주민등록번호, 주소, 연락처 |
| 임차인 | 성명, 주민등록번호, 주소, 연락처 |
| 대리인 | 대리인 정보 + 위임장 |

### 2.2 주택 정보

| 항목 | 기재 내용 |
|------|----------|
| 소재지 | 정확한 주소 (동·호수 포함) |
| 면적 | 전용면적, 공급면적 |
| 구조 | 아파트, 다세대, 단독 등 |
| 용도 | 주거용 |

### 2.3 계약 조건

| 항목 | 기재 내용 |
|------|----------|
| 보증금 | 총 보증금 금액 |
| 월세 | 월 임대료 (월세 계약 시) |
| 관리비 | 포함/별도 명시 |
| 계약기간 | 시작일 ~ 종료일 |

### 2.4 금액 지급

| 항목 | 기재 내용 |
|------|----------|
| 계약금 | 금액, 지급일 |
| 중도금 | 금액, 지급일 (있는 경우) |
| 잔금 | 금액, 지급일 |
| 입금계좌 | 임대인 명의 계좌 |

## 계약서 양식 구성

### 3.1 표준계약서 구성

| 순서 | 항목 |
|------|------|
| 1 | 부동산의 표시 |
| 2 | 계약 내용 |
| 3 | 특약사항 |
| 4 | 계약 당사자 |
| 5 | 중개업소 정보 |

### 3.2 부동산 표시

```
[부동산의 표시]

소재지: 서울특별시 강남구 테헤란로 123, 456동 789호
토지: 대지권 비율 표시
건물: 철근콘크리트조
면적: 전용 84㎡ / 공급 110㎡
```

### 3.3 계약 내용

```
[계약 내용]

보증금: 금 삼억원정 (300,000,000원)
월세: 금 오십만원정 (500,000원)
계약기간: 2026년 2월 1일 ~ 2028년 1월 31일

- 계약금: 금 삼천만원정 (계약 시)
- 중도금: 없음
- 잔금: 금 이억칠천만원정 (입주일)
```

## 특약사항 작성

### 4.1 필수 특약

| 특약 | 내용 |
|------|------|
| 확정일자 | 잔금일에 확정일자 취득 |
| 전입신고 | 잔금일에 전입신고 |
| 등기부 확인 | 잔금 전 등기부등본 재확인 |

### 4.2 권장 특약

| 특약 | 내용 |
|------|------|
| 원상복구 | 퇴거 시 원상복구 범위 |
| 시설물 | 에어컨, 가전 포함 여부 |
| 수리 | 시설 고장 시 수리 책임 |
| 반려동물 | 반려동물 가능 여부 |
| 흡연 | 흡연 가능 여부 |

### 4.3 특약 예시

**임차인 보호 특약**
```
1. 임차인은 잔금 지급과 동시에 전입신고 및 확정일자를
   취득하며, 임대인은 이에 협조한다.
2. 잔금 지급 전 등기부등본상 권리변동 발생 시
   임차인은 계약을 해제할 수 있다.
3. 임대차기간 중 매매 시 임대인은 임차인에게
   우선매수권을 부여한다.
```

**시설물 관련 특약**
```
1. 현 시설 상태로 임대하며, 시설목록은 별첨한다.
2. 에어컨, 냉장고, 세탁기는 임대인 소유로 포함한다.
3. 임차인 귀책사유로 인한 고장은 임차인이 수리한다.
4. 자연 노후로 인한 고장은 임대인이 수리한다.
```

## 주의사항

### 5.1 계약 전 확인

| 확인 사항 | 방법 |
|----------|------|
| 등기부등본 | 인터넷등기소 열람 |
| 건축물대장 | 정부24 발급 |
| 소유자 확인 | 신분증 대조 |
| 선순위 확인 | 근저당, 압류 등 |

### 5.2 등기부등본 확인 포인트

| 항목 | 확인 내용 |
|------|----------|
| 갑구 | 소유자, 가압류, 가처분 |
| 을구 | 근저당권, 전세권 |
| 근저당 | 채권최고액 확인 |

### 5.3 위험 신호

| 위험 | 대응 |
|------|------|
| 근저당 > 시세 70% | 계약 재고려 |
| 가압류 설정 | 계약 지양 |
| 다수 권리 설정 | 전문가 상담 |

## 계약 체결 절차

### 6.1 단계별 절차

| 단계 | 내용 |
|------|------|
| 1단계 | 매물 확인, 현장 방문 |
| 2단계 | 등기부등본 열람 |
| 3단계 | 계약서 작성 |
| 4단계 | 계약금 지급 |
| 5단계 | 잔금 지급, 입주 |
| 6단계 | 전입신고, 확정일자 |

### 6.2 잔금일 체크리스트

| 순서 | 확인 사항 |
|------|----------|
| 1 | 등기부등본 재확인 |
| 2 | 잔금 지급 |
| 3 | 열쇠 수령 |
| 4 | 입주 (점유) |
| 5 | 전입신고 |
| 6 | 확정일자 |

## 전자계약

### 7.1 전자계약 장점

| 장점 | 내용 |
|------|------|
| 확정일자 자동 | 계약 완료 시 자동 부여 |
| 안전 거래 | 공인인증 본인 확인 |
| 보관 편의 | 온라인 계약서 보관 |
| 신고 간편 | 전월세신고 연동 |

### 7.2 전자계약 방법

| 방법 | 내용 |
|------|------|
| 부동산거래 전자계약시스템 | 국토부 운영 |
| 공인중개사 앱 | 중개업소 통한 계약 |

### 7.3 전자계약 절차

```
중개의뢰 → 매물확인 → 계약서작성 →
전자서명 → 계약체결 → 확정일자자동부여
```

## 계약 후 처리

### 8.1 필수 처리

| 처리 | 기한 |
|------|------|
| 전입신고 | 입주 후 즉시 |
| 확정일자 | 입주 후 즉시 |
| 전월세신고 | 계약일로부터 30일 |

### 8.2 보관 서류

| 서류 | 보관 기간 |
|------|----------|
| 임대차계약서 원본 | 퇴거 후까지 |
| 등기부등본 | 계약 기간 중 |
| 지급 영수증 | 퇴거 후까지 |
| 인수인계서 | 퇴거 후까지 |

## 분쟁 예방 팁

### 9.1 작성 시 주의

| 주의 | 내용 |
|------|------|
| 금액 명확 | 한글 + 숫자 병기 |
| 날짜 정확 | 년월일 명확히 |
| 특약 상세 | 구체적으로 작성 |
| 서명 확인 | 당사자 직접 서명 |

### 9.2 보관 방법

| 방법 | 권장 |
|------|------|
| 원본 | 안전한 장소 보관 |
| 사본 | 별도 장소 보관 |
| 스캔 | 디지털 백업 |
| 사진 | 계약 현장 촬영 |
