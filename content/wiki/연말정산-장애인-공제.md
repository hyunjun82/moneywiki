# 연말정산 장애인 공제

장애인 추가공제는 장애인 부양가족에 대해 **200만원 추가 소득공제**를 받는 제도입니다. [국세청 인적공제 안내](https://www.nts.go.kr/nts/cm/cntnts/cntntsView.do?mi=6591&cntntsId=7870)에 따른 조건을 정리합니다.

## 1. 공제 기본 조건

[소득세법 제51조](https://www.law.go.kr/법령/소득세법)에 따른 장애인 공제 요건입니다.

| 요건 | 조건 |
|------|------|
| 대상 | 기본공제 대상자 중 **장애인** |
| 나이 | **제한 없음** |
| 공제액 | 1인당 **200만원** |

## 2. 장애인 범위

### 2.1 장애인복지법상 장애인

| 유형 | 대상 |
|------|------|
| 지체장애 | 신체 기능 장애 |
| 뇌병변장애 | 뇌 손상으로 인한 장애 |
| 시각장애 | 시력, 시야 장애 |
| 청각장애 | 청력, 평형 기능 장애 |
| 지적장애 | 지능지수 70 이하 |
| 정신장애 | 정신질환 |
| 기타 | 자폐, 신장, 심장, 호흡기 등 |

### 2.2 국가유공자

| 유형 | 대상 |
|------|------|
| 상이등급 | 1~7급 상이자 |
| 국가유공자 | 상이등급 판정자 |

### 2.3 세법상 장애인 (중증환자)

**장애인등록증 없이도 공제 가능**

| 대상 | 조건 |
|------|------|
| 암환자 | 항시 치료를 요하는 경우 |
| 치매환자 | 항시 치료를 요하는 경우 |
| 중풍환자 | 항시 치료를 요하는 경우 |
| 만성신부전환자 | 투석 등 치료 필요 |
| 희귀난치성질환자 | 지속적 치료 필요 |

## 3. 나이 제한 면제

### 3.1 장애인 특례

| 대상 | 일반 | 장애인 |
|------|------|--------|
| 직계존속 | 60세 이상 | **나이 제한 없음** |
| 직계비속 | 20세 이하 | **나이 제한 없음** |
| 형제자매 | 20세 이하/60세 이상 | **나이 제한 없음** |

### 3.2 예시

**성인 장애인 자녀 (35세)**

| 공제 | 적용 |
|------|------|
| 기본공제 | ✅ (나이 무관) |
| 장애인 추가공제 | ✅ 200만원 |

## 4. 공제 금액 계산

### 4.1 기본공제 + 장애인 추가공제

| 공제 유형 | 금액 |
|----------|------|
| 기본공제 | 150만원 |
| 장애인 추가공제 | **200만원** |
| **합계** | **350만원** |

### 4.2 다른 추가공제와 중복

| 추가공제 | 중복 가능 |
|---------|----------|
| 장애인 + 경로우대 | ✅ |
| 장애인 + 부녀자 | ✅ |
| 장애인 + 한부모 | ❌ (중복 불가) |

### 4.3 중복 계산 예시

**어머니 (75세, 장애인)**

| 공제 | 금액 |
|------|------|
| 기본공제 | 150만원 |
| 경로우대 (70세 이상) | 100만원 |
| 장애인 | 200만원 |
| **총 공제** | **450만원** |

## 5. 증빙서류

### 5.1 장애인복지법상 장애인

| 서류 | 발급처 |
|------|--------|
| 장애인등록증 사본 | 주민센터 |
| 복지카드 사본 | 주민센터 |

### 5.2 국가유공자

| 서류 | 발급처 |
|------|--------|
| 국가유공자증 사본 | 보훈청 |
| 상이등급확인원 | 보훈청 |

### 5.3 세법상 장애인 (중증환자)

| 서류 | 발급처 |
|------|--------|
| **장애인증명서** | 의료기관 |

**장애인증명서 기재 사항**:
- 환자 성명, 주민번호
- 장애 내용 및 원인
- **장애 예상 기간** (영구적 또는 기간 명시)

## 6. 중증환자 장애인증명서

### 6.1 대상 질환 예시

| 질환 | 대상 |
|------|------|
| 암 | 치료 중인 암환자 |
| 치매 | 알츠하이머, 혈관성 치매 |
| 중풍 | 뇌졸중 후유증 |
| 파킨슨병 | 지속 치료 필요 |
| 만성신부전 | 투석 환자 |
| 심장질환 | 심부전, 심장판막증 등 |

### 6.2 발급 방법

1. 담당 의사에게 장애인증명서 발급 요청
2. **"항시 치료를 요하는 중증환자"** 기재
3. 장애 예상 기간 명시

### 6.3 유효기간

| 기재 내용 | 증빙 필요 |
|----------|----------|
| 영구 | 최초 1회 제출 |
| 기간 명시 | 기간 만료 후 재발급 |

## 7. 의료비 공제 특례

### 7.1 장애인 의료비 한도

| 대상 | 한도 |
|------|------|
| 일반 부양가족 | 700만원 |
| **장애인** | **한도 없음** |

### 7.2 장애인보장구

| 항목 | 공제 |
|------|------|
| 휠체어 | ✅ 전액 |
| 보청기 | ✅ 전액 |
| 의수족 | ✅ 전액 |

## 8. 교육비 공제 특례

### 8.1 특수교육비

| 대상 | 한도 |
|------|------|
| 일반 교육비 | 한도 있음 |
| **장애인 특수교육비** | **한도 없음** |

### 8.2 대상 교육비

- 사회복지시설 교육비
- 특수교육기관 교육비
- 재활교육비

## 9. 주의사항

### 9.1 소득 요건

장애인도 **소득금액 100만원 이하** 요건 적용

| 소득 | 공제 |
|------|------|
| 100만원 이하 | ✅ |
| 100만원 초과 | ❌ |

### 9.2 중복 공제 불가

- 한 명의 장애인을 **여러 가족이 중복 공제 불가**
- 실제 부양자 1명만 공제
