# 임대사업자 등록

주택 임대사업자 등록 제도와 세금 혜택을 정리합니다.

## 임대사업자 개요

### 1.1 등록 유형

| 유형 | 임대 기간 |
|------|----------|
| 장기일반민간임대 | **10년** |
| 공공지원민간임대 | **8년** |

### 1.2 등록 대상

| 대상 | 등록 |
|------|------|
| 단독주택 |  |
| 아파트 |  |
| 다세대·다가구 |  |
| 오피스텔 |  |

## 등록 요건

### 2.1 주택 요건

| 요건 | 기준 |
|------|------|
| 주택 수 | 1호 이상 |
| 면적 | 제한 없음 |
| 공시가격 | 제한 없음 (일부 혜택 제한) |

### 2.2 임대료 제한

| 항목 | 기준 |
|------|------|
| 초기 임대료 | 시세 이하 권장 |
| 인상률 | 연 **5% 이내** |
| 인상 주기 | 1년 이상 경과 후 |

## 등록 절차

### 3.1 등록 기관

| 등록 | 기관 |
|------|------|
| 사업자 등록 | 시·군·구청 |
| 세무 등록 | 세무서 |

### 3.2 필요 서류

| 서류 | 용도 |
|------|------|
| 등록신청서 | 사업자 등록 |
| 임대차계약서 | 임대 조건 확인 |
| 건물등기부등본 | 소유권 확인 |
| 주민등록등본 | 신청인 확인 |

### 3.3 등록 절차

| 순서 | 절차 |
|------|------|
| 1 | 시·군·구청 임대사업자 등록 |
| 2 | 세무서 사업자 등록 |
| 3 | 임대차계약 신고 |
| 4 | 표준임대차계약서 사용 |

## 세금 혜택

### 4.1 취득세 감면

| 조건 | 감면 |
|------|------|
| 전용 40㎡ 이하 | **면제** |
| 전용 60㎡ 이하 | **75% 감면** |
| 전용 85㎡ 이하 | **50% 감면** |

### 4.2 재산세 감면

| 조건 | 감면 |
|------|------|
| 전용 40㎡ 이하 | **면제** |
| 전용 60㎡ 이하 | **75% 감면** |
| 전용 85㎡ 이하 | **50% 감면** |

### 4.3 종합부동산세

| 혜택 | 조건 |
|------|------|
| 합산 배제 | 8년 이상 임대 |
| 공시가 기준 | 6억원 이하 (수도권 외 3억원) |

### 4.4 양도소득세

| 혜택 | 조건 |
|------|------|
| 중과 배제 | 10년 이상 임대 |
| 장기보유특별공제 | 최대 **70%** |

## 장기보유특별공제

### 5.1 공제율

| 임대 기간 | 공제율 |
|----------|--------|
| 8년 이상 | 50% |
| 9년 이상 | 60% |
| 10년 이상 | **70%** |

### 5.2 적용 조건

| 조건 | 기준 |
|------|------|
| 임대 기간 | 8년 이상 |
| 임대료 제한 | 5% 준수 |
| 등록 유지 | 의무 기간 유지 |

## 의무 사항

### 6.1 임대 의무

| 의무 | 내용 |
|------|------|
| 임대 기간 | 10년 (장기) / 8년 (공공지원) |
| 임대료 인상 | 5% 이내 |
| 계약 신고 | 30일 이내 |

### 6.2 관리 의무

| 의무 | 내용 |
|------|------|
| 시설 유지 | 주거 기능 유지 |
| 수선 의무 | 임대인 부담 |
| 정보 공개 | 임대 조건 공개 |

## 위반 시 제재

### 7.1 과태료

| 위반 | 과태료 |
|------|--------|
| 신고 의무 위반 | 500만원 |
| 임대료 상한 위반 | 1,000만원 |
| 등록 말소 | 세금 추징 |

### 7.2 세금 추징

| 위반 | 추징 |
|------|------|
| 의무 기간 미준수 | 감면 세금 + 이자 |
| 등록 말소 | 혜택 환수 |

## 등록 말소

### 8.1 자진 말소

| 사유 | 처리 |
|------|------|
| 양도 | 말소 + 세금 정산 |
| 자가 거주 | 의무 위반 (추징) |

### 8.2 직권 말소

| 사유 | 처리 |
|------|------|
| 의무 위반 | 직권 말소 + 제재 |
| 허위 등록 | 직권 말소 + 과태료 |

## 세금 계산 예시

### 9.1 취득세 감면 예시

**전용 50㎡ 주택, 취득가 3억원**

| 항목 | 금액 |
|------|------|
| 일반 취득세 (1%) | 300만원 |
| 감면 (75%) | 225만원 |
| **납부 취득세** | **75만원** |

### 9.2 양도세 절감 예시

**10년 임대 후 양도, 양도차익 2억원**

| 항목 | 일반 | 임대사업자 |
|------|------|-----------|
| 양도차익 | 2억원 | 2억원 |
| 장특공제 | 30% | **70%** |
| 과세표준 | 1.4억원 | 6,000만원 |
| 예상 세금 | 약 4,900만원 | 약 900만원 |

## 임대소득세

### 10.1 과세 기준

| 임대소득 | 과세 |
|----------|------|
| 2,000만원 이하 | 분리과세 (14%) |
| 2,000만원 초과 | 종합과세 |

### 10.2 필요경비

| 경비 | 공제율 |
|------|--------|
| 단순경비율 | 42.6% |
| 기준경비율 | 임대료의 일정 % |
| 실제 경비 | 장부 기장 |

## 등록 vs 미등록

### 11.1 비교

| 항목 | 등록 | 미등록 |
|------|------|--------|
| 취득세 | 감면 | 일반 |
| 종부세 | 합산 배제 | 합산 |
| 양도세 | 중과 배제 | 중과 |
| 의무 | 있음 | 없음 |

### 11.2 선택 기준

| 고려 | 등록 유리 | 미등록 유리 |
|------|----------|-----------|
| 보유 기간 | 10년 이상 | 단기 |
| 주택 수 | 다주택 | 1주택 |
| 임대 의향 | 장기 임대 | 자가 전환 |

## 주의사항

### 12.1 등록 전 확인

| 확인 | 내용 |
|------|------|
| 의무 기간 | 10년 유지 가능성 |
| 임대료 제한 | 5% 준수 가능성 |
| 혜택 조건 | 공시가 기준 확인 |

### 12.2 등록 후 주의

| 주의 | 내용 |
|------|------|
| 계약 신고 | 30일 이내 필수 |
| 인상 제한 | 5% 초과 시 제재 |
| 양도 시 | 의무 승계 또는 추징 |
