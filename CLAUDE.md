## SEO 전략: 롱테일 키워드 ONLY

### 핵심 원칙 (절대 규칙)
1. **신규 도메인 = 단일 키워드 노출 불가능**
2. **모든 문서는 롱테일 키워드**
3. **단일 키워드 허브 문서 = 만들지 마라**

### 왜?
- 나무위키: 도메인 점수 높음 → "퇴직금", "연말정산" 노출됨
- 신규 도메인: 도메인 점수 없음 → 롱테일로만 노출 가능
- 200~300개 롱테일 문서로 조금씩 노출시키는 전략

### 올바른 예시
```
/w/연말정산-신용카드-공제-한도
title: "연말정산 신용카드 공제 한도"

/w/퇴직금-통상임금-계산
title: "퇴직금 통상임금 계산"
```

### 금지 (절대 하지 마라)
```
/w/연말정산 → "연말정산" (단일 키워드 = 노출 안됨)
/w/퇴직금 → "퇴직금" (단일 키워드 = 노출 안됨)
title: "2026년 연말정산 총정리: ..." (블로그식 = 금지)
```

## 콘텐츠 작성 가이드라인 (가장 중요!)

### 0. 핵심 원칙: 간결 + 정확 + 이해쉽게

**불필요한 말 금지. 제목/소제목에 대한 정보만 정확히 전달.**

- 쓸데없는 서론, 부연설명, 반복 금지
- 키워드(제목)에 대한 핵심 정보만 간결하게
- 간결해도 주요 내용이 빠지면 안 됨
- 독자가 한 번 읽고 바로 이해할 수 있게

#### 나쁜 예시 (불필요한 말)
```
전입신고는 정말 중요한 절차입니다. 많은 분들이 이 부분을 놓치시는데요,
사실 전입신고가 왜 중요한지 이해하려면 먼저 대항력이라는 개념을 알아야 합니다.
대항력이란 어쩌구저쩌구...
```

#### 좋은 예시 (핵심만)
```
전입신고하면 대항력이 생겨요. 집주인이 바뀌어도 보증금을 받을 수 있는 힘이죠.
잔금일 당일 바로 신고하세요.
```

### 1. 문체: 구어체 필수

**원칙**: 정보는 정확하게, 말투는 쉽게

#### 좋은 예시 (이렇게 써라)
```
대항력이란 쉽게 말해 '내 보증금을 지킬 수 있는 힘'이에요.
집주인이 바뀌거나 집이 경매로 넘어가도 보증금을 받을 수 있죠.
이 힘을 얻으려면 실제로 이사하고 전입신고를 해야 해요.
```

#### 나쁜 예시 (사용 금지)
```
대항력은 임차인이 제3자에게 임대차의 내용을 주장할 수 있는
법적 효력을 의미합니다. 주택임대차보호법 제3조에 따라...
```

#### 문체 패턴
- "~이에요", "~해요", "~하죠", "~거든요", "~인데요"
- "쉽게 말해", "다시 말해", "예를 들면"
- "~할 수 있어요", "~해야 해요", "~하면 돼요"

# 대항력 발생 시점

대항력이란 쉽게 말해 '내 보증금을 지킬 수 있는 힘'이에요.
집주인이 바뀌거나 집이 경매로 넘어가도 보증금을 받을 수 있죠.

## 1. 대항력이란

대항력은 임차인이 제3자(새 집주인, 경매 낙찰자 등)에게
"나 여기 살고 있고, 보증금 받을 권리 있어요"라고 주장할 수 있는 힘이에요.
이게 없으면 집이 팔렸을 때 보증금을 못 받을 수도 있어요.

| 항목 | 내용 |
|------|------|
| 정의 | 제3자에게 임대차를 주장할 수 있는 힘 |
| 근거 | 주택임대차보호법 제3조 |
| 효과 | 집주인 변경되어도 보증금 보호 |

## 2. 대항력 발생 조건

대항력을 얻으려면 두 가지가 필요해요.
[전입신고](/w/전입신고-방법)하고, 실제로 그 집에서 살아야 해요(점유).

| 조건 | 설명 |
|------|------|
| 전입신고 | 주민센터 또는 정부24에서 |
| 점유 | 실제 거주 (이사) |

**중요!** 대항력은 전입신고한 **다음 날 0시**에 생겨요.
그래서 [잔금일](/w/전세계약-주의사항)에 바로 전입신고하는 게 좋아요.

[확정일자](/w/확정일자)까지 받으면 [우선변제권](/w/우선변제권-요건)도 생기니까
같이 받아두세요.

더 자세한 내용은 [주택임대차보호법](https://www.law.go.kr/법령/주택임대차보호법)을
참고하세요.
