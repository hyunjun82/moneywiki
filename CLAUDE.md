# 머니위키 (wiki-site) 작업 규칙

## SEO 전략: 롱테일 키워드

### 핵심 원칙
- **title = 검색 키워드 그대로** (블로그식 X)
- **slug = title** (하이픈으로 연결)

### 올바른 예시
```
/w/연말정산-신용카드-공제-한도
title: "연말정산 신용카드 공제 한도"

/w/맞벌이-연말정산-절세-방법
title: "맞벌이 연말정산 절세 방법"

/w/연말정산-의료비-공제-계산
title: "연말정산 의료비 공제 계산"
```

### 잘못된 예시 (블로그식 - 절대 금지)
```
title: "2026년 연말정산 총정리: 소득공제부터..."
title: "연말정산 신용카드 공제 한도: 2026년 기준 총정리"
```

### 허브 문서 vs 롱테일 문서
- **허브**: `/w/연말정산` → title: "연말정산" (짧게)
- **롱테일**: `/w/연말정산-신용카드-공제-한도` → title: "연말정산 신용카드 공제 한도"

### 콘텐츠 클러스터 구조
```
연말정산 (허브)
├── 연말정산 신용카드 공제 한도
├── 연말정산 의료비 공제 계산
├── 맞벌이 연말정산 절세 방법
└── 연말정산 교육비 공제 조건
```

## 문서 작성 규칙

### frontmatter 필수 항목
```yaml
title: "롱테일 키워드"  # 검색어 그대로
description: "..."
category: "세금"
keywords: [...]
lastUpdated: "YYYY-MM-DD"
datePublished: "YYYY-MM-DD"
summary: "3줄 요약"
sources:
  - name: "출처명"
    url: "https://..."
```

### 본문 출처 표기
- 출처는 본문 내 인라인으로 표기
- 예: `[국세청](https://nts.go.kr)에 따르면...`

## 광고 (AdSense)
- 본문 상단: HORIZONTAL
- 본문 하단: VERTICAL
- 사이드바: SQUARE
