## 본문 출처 표기 규칙 (필수!)

### 법령 인용 시
```markdown
[주택임대차보호법 제3조](https://www.law.go.kr/법령/주택임대차보호법)에 따르면
대항력은 전입신고 다음 날 0시에 발생해요.
```

### 복잡한 내용 쉽게 풀기
```markdown
❌ "임차인은 주택의 인도와 주민등록을 마친 때에는 그 다음 날부터
   제3자에 대하여 효력이 생긴다. (주택임대차보호법 제3조)"

✅ 쉽게 말하면, 이사하고 전입신고하면 그 다음 날부터 보증금을
   지킬 수 있는 힘이 생겨요. [주택임대차보호법 제3조](https://www.law.go.kr/법령/주택임대차보호법)에서
   이 권리를 보장하고 있어요.
```

### 출처 이중 표기 (본문 + 하단) - 필수!

**위키피디아/나무위키 스타일 = 본문 인라인 + 하단 출처 섹션**

```markdown
## 대항력이란

쉽게 말하면 보증금을 지킬 수 있는 힘이에요.
[주택임대차보호법 제3조](https://www.law.go.kr/법령/주택임대차보호법)에서
이 권리를 보장하고 있어요.  ← 본문에 자연스럽게

(... 글 내용 ...)

---

## 출처

- [주택임대차보호법](https://www.law.go.kr/법령/주택임대차보호법) - 법제처
- [전세보증금 반환보증](https://www.khug.or.kr) - HUG
```

**규칙:**
- 본문 중간에 자연스럽게 출처 링크 삽입 (읽으면서 신뢰)
- 글 하단에 "## 출처" 섹션 추가 (정리된 참고문헌)
- frontmatter sources에도 동일하게 기재

## 작성 전 확인 (매번!)

### 금지 사항
- [ ] 이모지 없음
- [ ] 헤딩에 숫자 없음 (`## 제목` O, `## 1. 제목` X)
- [ ] 본문에 FAQ 섹션 없음 (frontmatter만)
- [ ] "~알아봅니다" description 아님
- [ ] 단일 키워드 제목 아님

### 필수 사항
- [ ] 구어체 (~이에요, ~하죠)
- [ ] summary 배열 3줄
- [ ] 각 섹션 4문장 이상
- [ ] 내부링크 `/w/카테고리/슬러그` 3개 이상
- [ ] 테이블 2개 이하

### URL 구조
```
/w/부동산/임대차보호법
/w/연말정산/신용카드-공제-한도
/w/[category]/[slug]
```
